### **估值方法與計算**

我們將應用折現現金流模型 (DCF)、相對估值法和股息折現模型 (DDM) 進行分析。

#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型通過預測公司未來的自由現金流 (FCF) 並將其折現至現值來估計公司價值。

**估值假設：**

*   **折現率 (WACC - 加權平均資本成本)：**
    *   **無風險利率 (Rf)：** 採用美國10年期國債收益率，約 **4.20%**。
    *   **市場風險溢價 (MRP)：** 採用行業標準，約 **5.50%**。
    *   **Beta (β)：** 根據Yahoo Finance等數據源，INGR 的 Beta 值約為 **1.00** (接近市場平均水平，反映其相對穩定的業務性質)。
    *   **股權成本 (Ke) = Rf + β * MRP** = 4.20% + 1.00 * 5.50% = **9.70%**。
    *   **債務成本 (Kd)：** 根據公司財報和當前利率環境，假設稅前債務成本為 **5.00%**。
    *   **稅率 (Tax Rate)：** 根據公司歷史有效稅率，約 **23%**。
    *   **稅後債務成本 (Kd_after_tax) = Kd * (1 - Tax Rate)** = 5.00% * (1 - 0.23) = **3.85%**。
    *   **股權市值 (E)：** $7.08 Billion (Market Cap)。
    *   **債務市值 (D)：** 根據最新財報，公司長期債務約為25億美元，短期債務較少，總債務約為 **2.5 Billion**。
    *   **WACC = (E / (E+D)) * Ke + (D / (E+D)) * Kd_after_tax**
        *   E+D = 7.08B + 2.5B = 9.58B
        *   WACC = (7.08 / 9.58) * 9.70% + (2.5 / 9.58) * 3.85%
        *   WACC = 0.739 * 9.70% + 0.261 * 3.85% = 7.16% + 1.00% = **8.16%**。
        *   為簡化計算，我們取 **8.20%** 作為折現率。

*   **預測期間：** 5年顯性預測期。
*   **自由現金流 (FCF) 預測：**
    *   **基期 FCF (LTM)：** 根據最新財報，2023年自由現金流為 **6.01億美元**。
    *   **增長率：**
        *   **顯性預測期 (5年)：** 考慮到公司指引的溫和增長和行業趨勢，我們假設：
            *   第1-2年：**4.5%** (略高於EPS指引，反映現金流效率)
            *   第3-5年：**3.5%** (增長趨於穩定)
        *   **永續增長率 (g)：** 採用長期通脹率和全球經濟增長率，約 **2.20%**。

**DCF 計算步驟：**

1.  **預測未來5年自由現金流 (FCF)：**
    *   FCF0 (2023 LTM): $601 Million
    *   FCF1 (2024): $601 * (1 + 0.045) = $628.05 Million
    *   FCF2 (2025): $628.05 * (1 + 0.045) = $656.36 Million
    *   FCF3 (2026): $656.36 * (1 + 0.035) = $679.38 Million
    *   FCF4 (2027): $679.38 * (1 + 0.035) = $703.16 Million
    *   FCF5 (2028): $703.16 * (1 + 0.035) = $727.87 Million

2.  **計算終值 (Terminal Value, TV)：**
    *   TV = FCF5 * (1 + g) / (WACC - g)
    *   TV = $727.87M * (1 + 0.022) / (0.082 - 0.022)
    *   TV = $743.88M / 0.06 = $12,398 Million

3.  **折現所有現金流和終值至現值：**
    *   PV(FCF1) = $628.05M / (1 + 0.082)^1 = $580.45M
    *   PV(FCF2) = $656.36M / (1 + 0.082)^2 = $560.00M
    *   PV(FCF3) = $679.38M / (1 + 0.082)^3 = $537.00M
    *   PV(FCF4) = $703.16M / (1 + 0.082)^4 = $516.00M
    *   PV(FCF5) = $727.87M / (1 + 0.082)^5 = $496.00M
    *   PV(TV) = $12,398M / (1 + 0.082)^5 = $8,480.00M

4.  **計算企業價值 (Enterprise Value, EV)：**
    *   EV = Sum(PV(FCFs)) + PV(TV)
    *   EV = $580.45M + $560.00M + $537.00M + $516.00M + $496.00M + $8,480.00M = $11,169.45 Million

5.  **計算股權價值 (Equity Value) 和每股價值：**
    *   股權價值 = EV + 現金及等價物 - 總債務
    *   根據最新財報，現金及等價物約為 **4億美元**。
    *   總債務約為 **25億美元**。
    *   股權價值 = $11,169.45M + $400M - $2,500M = $9,069.45 Million
    *   流通股數 (Shares Outstanding) = 市值 / 股價 = $7.08B / $111.40 = **63.55 Million**。
    *   **DCF 估值每股價值 = $9,069.45M / 63.55M = $142.71**

**DCF 估值結果：**
*   **合理價位：$142.71**
*   **合理價位區間：$135 - $150** (考慮到WACC和增長率假設的敏感性)

#### **3. 股息折現模型 (Dividend Discount Model, DDM)**

由於 INGR 是一家成熟且穩定派發股息的公司，DDM 適用。

**估值假設：**

*   **當前每股股息 (D0)：** 股價 $111.40 * 股息率 0.0289 = **$3.22**。
*   **股權成本 (Ke)：** 沿用 DCF 計算的 **9.70%**。
*   **股息增長率 (g)：**
    *   INGR 歷史上股息增長穩定。考慮到公司未來溫和的盈利增長預期 (約5.4%) 和其約36%的派息率 ($3.22 / $9.01)，我們假設一個可持續的長期股息增長率為 **4.00%**。此增長率低於股權成本，符合 DDM 模型要求。

**DDM 計算步驟 (戈登增長模型 - Gordon Growth Model)：**

*   **預期下一期股息 (D1) = D0 * (1 + g)** = $3.22 * (1 + 0.04) = $3.3488
*   **DDM 估值 = D1 / (Ke - g)**
*   **DDM 估值 = $3.3488 / (0.097 - 0.04) = $3.3488 / 0.057 = $58.75**

**DDM 估值結果：**
*   **合理價位：$58.75**
*   **合理價位區間：$55 - $65**

**DDM 模型評估：**
DDM 模型給出的估值顯著低於當前股價和DCF、相對估值結果。這可能表明：
1.  **股息增長率假設過於保守：** 如果市場預期更高的股息增長率，DDM 估值會更高。
2.  **公司價值不僅體現在股息：** INGR 是一家成長型與價值型兼具的公司，其保留盈餘用於再投資以實現未來增長，這部分價值在簡單的戈登增長模型中可能未被充分捕捉。對於這類公司，DCF 和相對估值可能更能反映其真實價值。
因此，DDM 在此案例中可能作為輔助參考，但其結果的權重應低於 DCF 和相對估值。
