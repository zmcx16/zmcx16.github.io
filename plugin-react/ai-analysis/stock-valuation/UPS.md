*   **"Better, Not Bigger" 策略：** UPS 正在實施「更好而非更大」的策略，專注於高利潤業務和效率提升，而非單純追求包裹量增長。這可能導致短期內營收增長放緩，但有望改善長期盈利能力。

### II. 估值假設

在進行估值之前，我們需要建立一系列關鍵假設。

#### A. 加權平均資本成本 (WACC) 計算

WACC 是折現現金流模型中的折現率，代表公司為其資產融資的平均成本。

1.  **股權成本 (Cost of Equity, Ke) - 使用資本資產定價模型 (CAPM):**
    *   **無風險利率 (Risk-Free Rate, Rf):** 參考美國 10 年期國債收益率。查詢最新數據，約為 **4.30%**。
    *   **市場風險溢價 (Market Risk Premium, MRP):** 通常在 5.0% - 6.0% 之間。我們採用 **5.50%**。
    *   **Beta (β):** 衡量股票相對於市場的波動性。根據查詢，UPS 的 Beta 值約為 **1.10**。
    *   **Ke = Rf + β * MRP**
        *   Ke = 4.30% + 1.10 * 5.50% = 4.30% + 6.05% = **10.35%**

2.  **債務成本 (Cost of Debt, Kd):**
    *   根據 UPS 2023 年 10-K 報告和近期財報，其總債務約為 400 億美元，年度利息支出約為 18 億美元。
    *   **Kd = 利息支出 / 總債務**
        *   Kd = $1.8B / $40B = **4.50%**

3.  **稅率 (Tax Rate, t):**
    *   根據 UPS 近期財報，其有效稅率約為 **24%**。

4.  **資本結構權重：**
    *   **股權市值 (Market Value of Equity, E):** $84.57 億 (Market Cap)
    *   **債務市值 (Market Value of Debt, D):** $40 億 (總債務)
    *   **總企業價值 (V) = E + D** = $84.57B + $40B = $124.57B
    *   **股權權重 (We) = E / V** = $84.57B / $124.57B = **67.89%**
    *   **債務權重 (Wd) = D / V** = $40B / $124.57B = **32.11%**

5.  **WACC 計算：**
    *   **WACC = (We * Ke) + (Wd * Kd * (1 - t))**
        *   WACC = (0.6789 * 10.35%) + (0.3211 * 4.50% * (1 - 0.24))
        *   WACC = 7.02% + (0.3211 * 4.50% * 0.76)
        *   WACC = 7.02% + 1.10% = **8.12%**
    *   為簡化計算並考慮一定的波動性，我們將使用 **8.0%** 作為 WACC。

#### B. 增長率假設

*   **預測期間：** 5 年 (2024 - 2028)
*   **短期營收增長率 (2024-2025):** 考慮到當前宏觀經濟逆風、包裹量下降以及「更好而非更大」策略的影響，預計短期增長將非常溫和甚至持平。
    *   2024 年：**0.0%** (基於公司指引和市場預期，可能略有下降或持平)
    *   2025 年：**2.0%** (溫和復甦)
    *   2026-2028 年：**3.0%** (隨著經濟復甦和策略效果顯現)
*   **長期永續增長率 (Terminal Growth Rate, g):** 預計公司在預測期後將以一個穩定的速度增長，通常接近長期通脹率或略低於 GDP 增長率。我們採用 **2.5%**。

#### C. 其他假設

*   **營業利潤率 (Operating Margin):** 根據提供的數據為 9.76%。考慮到勞動成本上升和策略調整，我們預計短期內可能略有波動，但長期目標是改善。我們將在預測期內維持在 **9.5% - 10.0%** 之間。
*   **折舊與攤銷 (D&A):** 根據歷史數據和 CapEx 規模，預計每年約 **42 億美元**。
*   **資本支出 (Capital Expenditure, CapEx):** UPS 每年投入大量資金用於網絡現代化和自動化。預計每年約 **50 億美元**。
*   **營運資本變動 (Change in Net Working Capital, NWC):** 假設為營收的 1% 變動，或每年約 **1 億美元** 的小幅增加 (對 FCF 產生負面影響)。
*   **流通股數：** 市值 / 股價 = $84.57B / $99.68 = 約 **8.484 億股**。

#### C. 股息折現模型 (Dividend Discount Model, DDM)

UPS 是一家成熟且股息豐厚的公司，DDM 適用於此類公司。

**1. 基礎數據：**
*   當前股價 (P0): $99.68
*   當前年度股息 (D0): UPS 季度股息為 $1.63，因此年度股息為 $1.63 * 4 = **$6.52**。
*   股權成本 (Ke): 10.35% (來自 WACC 計算)

**2. 股息增長率 (g):**
*   考慮到近期盈利壓力，儘管 UPS 有穩定的股息增長歷史，但未來幾年增長可能放緩。
*   短期 (未來 3 年): 假設股息增長率為 **2.0%**。
*   長期 (永續): 假設股息增長率為 **2.5%** (與 DCF 的永續增長率一致)。

**3. 多階段 DDM 計算：**

*   **預測期股息：**
    *   D1 = D0 * (1 + g1) = $6.52 * (1 + 0.020) = $6.65
    *   D2 = D1 * (1 + g1) = $6.65 * (1 + 0.020) = $6.78
    *   D3 = D2 * (1 + g1) = $6.78 * (1 + 0.020) = $6.92

*   **終端股息價值 (在第 3 年末):**
    *   D4 = D3 * (1 + g_long) = $6.92 * (1 + 0.025) = $7.09
    *   **P3 = D4 / (Ke - g_long)**
        *   P3 = $7.09 / (0.1035 - 0.025) = $7.09 / 0.0785 = **$90.32**

*   **每股價值：**
    *   每股價值 = D1/(1+Ke)^1 + D2/(1+Ke)^2 + D3/(1+Ke)^3 + P3/(1+Ke)^3
    *   每股價值 = $6.65/(1.1035)^1 + $6.78/(1.1035)^2 + $6.92/(1.1035)^3 + $90.32/(1.1035)^3
    *   每股價值 = $6.03 + $5.56 + $5.14 + $67.08 = **$83.81**

**DDM 估值結果：**
*   **每股合理價值：$83.81**
*   **合理價位區間：$75 - $90** (對股息增長率和股權成本非常敏感)

當前股價 $99.68 介於各模型估值區間之間，且與分析師目標價 $103.81 接近。其高達 6.58% 的股息率是其吸引力所在，但這也可能反映了市場對其未來增長前景的擔憂。
