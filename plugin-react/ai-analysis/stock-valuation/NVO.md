### 5. 股息折現模型 (Dividend Discount Model, DDM)

NVO 支付股息 (Dividend %: 0.0294)。我們可以使用 DDM 進行估值。

**假設：**
*   當前股價: 58.81 美元
*   股息率: 2.94%
*   當前每股股息 (D0) = 58.81 \* 0.0294 = 1.728 美元
*   股息增長率 (g)：
    *   短期高增長率 (g1)：考慮到公司強勁的銷售增長和盈利能力，我們假設未來 5 年股息增長率為 10%。
    *   長期永續增長率 (g2)：與 DCF 模型一致，2.5%。
*   股權成本 (Ke)：我們使用 DCF 模型中計算的股權成本 9.48%。

**兩階段 DDM 模型：**

**步驟 1: 計算高增長階段的股息現值**

| 年度 | 股息 (D) = D0 \* (1+g1)^n | 折現因子 (Ke=9.48%) | 股息現值 |
| :--- | :----------------------- | :------------------ | :------- |
| 1    | 1.728 \* (1.10)^1 = 1.901 | 1 / (1.0948)^1 = 0.9134 | 1.736    |
| 2    | 1.728 \* (1.10)^2 = 2.091 | 1 / (1.0948)^2 = 0.8343 | 1.745    |
| 3    | 1.728 \* (1.10)^3 = 2.299 | 1 / (1.0948)^3 = 0.7620 | 1.752    |
| 4    | 1.728 \* (1.10)^4 = 2.529 | 1 / (1.0948)^4 = 0.6960 | 1.759    |
| 5    | 1.728 \* (1.10)^5 = 2.782 | 1 / (1.0948)^5 = 0.6357 | 1.768    |

**高增長階段股息現值總和 = 1.736 + 1.745 + 1.752 + 1.759 + 1.768 = 8.76 美元**

**步驟 2: 計算終值 (Terminal Value) - 第 5 年末的股價**
P5 = D6 / (Ke - g2)
D6 = D5 \* (1 + g2) = 2.782 \* (1 + 0.025) = 2.782 \* 1.025 = 2.852 美元
P5 = 2.852 / (0.0948 - 0.025) = 2.852 / 0.0698 = 40.86 美元

**步驟 3: 計算終值現值**
PV(P5) = P5 / (1 + Ke)^5
PV(P5) = 40.86 / (1.0948)^5 = 40.86 / 1.5776 = 25.90 美元

**步驟 4: 計算每股估值**
每股估值 = 高增長階段股息現值總和 + 終值現值
每股估值 = 8.76 + 25.90 = **34.66 美元**

**DDM 估值結果：**
基於 DDM 模型，NVO 的合理股價約為 **34.66 美元**。

**DDM 估值範圍 (考慮股息增長率變化)：**
*   如果短期股息增長率為 8%，長期為 2.5%，Ke 為 9.48%：估值會略低。
*   如果短期股息增長率為 12%，長期為 2.5%，Ke 為 9.48%：估值會略高。

DDM 估值通常對增長率和折現率非常敏感。NVO 的 DDM 估值顯著低於當前股價和 DCF 估值，這可能表明市場對其未來增長預期遠高於我們在 DDM 中保守假設的股息增長率，或者公司傾向於將更多利潤再投資而非以股息形式分配。

**綜合結論：**

綜合來看，DCF 模型考慮了 NVO 未來的現金流潛力，尤其是在 GLP-1 市場的強勁增長，給出了相對樂觀的估值。相對估值則將 NVO 與同業進行比較，顯示其估值在行業中處於合理水平，但並未被顯著低估。DDM 模型對於像 NVO 這樣處於高增長階段且需要大量再投資的公司可能不太適用。

考慮到 NVO 在 GLP-1 市場的領導地位、強勁的產品線 (Ozempic, Wegovy) 以及持續的研發和產能擴張，其未來增長潛力依然巨大。然而，來自 Eli Lilly 的激烈競爭、潛在的定價壓力以及供應鏈挑戰是需要關注的風險。
