*   TV = $2.0161 / (0.0835 - 0.025) = $2.0161 / 0.0585 = $34.46

**計算現值：**

*   PV (DPS 2026) = $1.7474 / (1 + 0.0835)^1 = $1.6128
*   PV (DPS 2027) = $1.8000 / (1 + 0.0835)^2 = $1.5369
*   PV (DPS 2028) = $1.8540 / (1 + 0.0835)^3 = $1.4649
*   PV (DPS 2029) = $1.9096 / (1 + 0.0835)^4 = $1.3966
*   PV (DPS 2030) = $1.9669 / (1 + 0.0835)^5 = $1.3319
*   PV (TV) = $34.46 / (1 + 0.0835)^5 = $23.19

**DDM 估值結果：**

*   每股估值 = 預測期股息現值之和 + 終值現值
*   每股估值 = $1.6128 + $1.5369 + $1.4649 + $1.3966 + $1.3319 + $23.19 = **$30.54**
*   **合理價位區間：$28.00 - $33.00** (考慮到股息增長率假設的敏感性)
