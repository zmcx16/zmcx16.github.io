### 1. 公司簡介與投資亮點

Novo Nordisk A/S 是一家全球領先的醫療保健公司，總部位於丹麥。公司在糖尿病、肥胖症以及罕見血液和內分泌疾病治療領域擁有強大的地位。近年來，NVO 因其 GLP-1 藥物 Ozempic (糖尿病) 和 Wegovy (肥胖症) 的巨大成功而成為市場焦點，這兩種藥物在市場上取得了爆炸性增長，並預計將在未來幾年繼續驅動公司增長。

**主要投資亮點：**
*   **GLP-1 市場領導者：** Ozempic 和 Wegovy 在快速增長的肥胖症和糖尿病治療市場中佔據主導地位。
*   **強勁的研發管線：** 持續投入研發，以擴大 GLP-1 類藥物的適應症並開發新的治療方案。
*   **高盈利能力：** 憑藉其專利藥品和市場地位，NVO 保持著行業領先的利潤率。
*   **全球擴張潛力：** GLP-1 藥物在全球範圍內仍有巨大的滲透和增長空間。

### 2. 估值假設概覽

#### 2.2 加權平均資本成本 (WACC) 計算

WACC 是 DCF 模型中的折現率，代表了公司為所有資本來源（股權和債務）支付的平均成本。

*   **無風險利率 (Risk-Free Rate, Rf)：** 採用美國10年期國債殖利率，**4.40%**。
*   **市場風險溢酬 (Market Risk Premium, MRP)：** 採用行業標準，**5.50%**。
*   **Beta 值 (β)：** 根據 Yahoo Finance 等數據源，NVO 的五年月度 Beta 值約為 **0.85**。這表明 NVO 的股價波動性低於整體市場。
*   **股權成本 (Cost of Equity, Re) - 採用 CAPM 模型：**
    Re = Rf + β * MRP
    Re = 4.40% + 0.85 * 5.50% = 4.40% + 4.68% = **9.08%**

*   **債務成本 (Cost of Debt, Rd)：** 考慮 NVO 的高信用評級，採用較低的發債成本，預計為 **5.50%**。
*   **有效稅率 (Tax Rate, t)：** 根據 NVO 歷史數據和預期，採用 **21.0%**。
*   **稅後債務成本 (After-Tax Cost of Debt)：**
    Rd (after-tax) = Rd * (1 - t) = 5.50% * (1 - 0.21) = 5.50% * 0.79 = **4.35%**

*   **資本結構 (Capital Structure)：** NVO 的債務水平相對較低，甚至擁有淨現金。為保守起見，我們假設一個極低的債務權重。
    *   市場股權價值 (E)：~$618.24B ($138 * 4.48B shares)
    *   市場債務價值 (D)：假設長期債務約 $5B
    *   總資本 (V) = E + D = $618.24B + $5B = $623.24B
    *   股權權重 (We) = E / V = 618.24B / 623.24B ≈ 0.992
    *   債務權重 (Wd) = D / V = 5B / 623.24B ≈ 0.008

*   **WACC 計算：**
    WACC = (We * Re) + (Wd * Rd (after-tax))
    WACC = (0.992 * 9.08%) + (0.008 * 4.35%)
    WACC = 9.01% + 0.03% = **9.04%**
    為簡化，我們取 WACC 為 **9.00%**。

### 4. 相對估值模型 (Comparable Company Analysis)

### 5. 股息折現模型 (DDM)

NVO 是一家支付股息的公司。然而，由於其處於高速增長階段，公司傾向於將大部分利潤再投資以支持研發和擴張，而不是全部分配股息。因此，簡單的戈登增長模型 (Gordon Growth Model) 可能會嚴重低估其價值。我們將嘗試使用兩階段 DDM 模型。

#### 5.1 DDM 假設

*   **當前年度股息 (D0)：** 2023年每股股息為 DKK 9.40。轉換為美元：9.40 DKK * 0.145 USD/DKK = $1.36 / 股。
*   **股權成本 (Re)：** 採用 WACC 計算中的股權成本，**9.08%**。
*   **第一階段股息增長率 (g1 - 高增長階段)：** 由於 NVO 處於高速增長階段，我們假設未來3年股息保持較快增長。考慮其盈利增長，但低於盈利增長率，因為需大量再投資。假設 **18.0%**。
*   **第二階段股息增長率 (g2 - 永續增長階段)：** 考慮長期可持續性，假設 **4.50%** (高於長期通脹，但低於短期增長)。

#### 5.2 兩階段 DDM 計算

*   **當前股息 (D0)：** $1.36
*   **預測期股息：**
    *   D1 = $1.36 * (1 + 0.18) = $1.60
    *   D2 = $1.60 * (1 + 0.18) = $1.89
    *   D3 = $1.89 * (1 + 0.18) = $2.23
*   **永續期首年股息 (D4)：**
    *   D4 = D3 * (1 + g2) = $2.23 * (1 + 0.045) = $2.33
*   **永續價值 (Terminal Value of Dividends at Year 3)：**
    *   TV_D3 = D4 / (Re - g2) = $2.33 / (0.0908 - 0.0450) = $2.33 / 0.0458 = $50.87
*   **折現至今日的價值：**
    *   PV(D1) = $1.60 / (1.0908)^1 = $1.47
    *   PV(D2) = $1.89 / (1.0908)^2 = $1.59
    *   PV(D3 + TV_D3) = ($2.23 + $50.87) / (1.0908)^3 = $53.10 / 1.298 = $40.91

**DDM 每股估值 = PV(D1) + PV(D2) + PV(D3 + TV_D3) = $1.47 + $1.59 + $40.91 = $43.97 / 股**

**DDM 估值結果：** $43.97 / 股
**合理價位區間：** $35 - $60 / 股 (此區間反映了 DDM 對 NVO 的局限性)

**局限性：** 股息折現模型對於像 NVO 這樣的高增長公司通常會嚴重低估其價值。這是因為 NVO 的股息派發率相對較低，公司將大量現金流再投資於研發和業務擴張，而非以股息形式返還股東。因此，DDM 無法充分捕捉 NVO 的增長潛力，其結果僅供參考，不應作為主要估值依據。

### 6. 綜合結論與比較
