#### 3.1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測公司的自由現金流 (Free Cash Flow, FCF)。由於沒有直接提供歷史FCF數據，我將使用淨利潤和營運現金流的關係來估算。

**估算自由現金流 (FCF):**
*   最新年度淨利 (Net Income): $15.51 billion.
*   營收 (Revenue): $46.80 billion.
*   淨利率 (Net Profit Margin): $15.51B / $46.80B ≈ 33.14%.
*   假設 FCF Margin (FCF/Revenue) 約為 25% (製藥公司通常有較高的利潤率和現金流轉換率，但考慮到研發投入和資本支出，會略低於淨利率)。
*   因此，基期 FCF (2025年) = $46.80B * 25% = $11.70 billion。

**DCF 計算步驟：**

| 年度 | 預測營收 (Billion USD) | 預測FCF (Billion USD) | 折現因子 (1/(1+WACC)^n) | FCF現值 (Billion USD) |
| :--- | :--------------------- | :--------------------- | :----------------------- | :--------------------- |
| 2025 | $46.80 (實際)          | $11.70 (估算)          | -                        | -                      |
| 2026 | $46.80 * (1+0.15) = 53.82 | $11.70 * (1+0.15) = 13.46 | 1/(1.075)^1 = 0.9302     | 12.52                  |
| 2027 | $53.82 * (1+0.15) = 61.89 | $13.46 * (1+0.15) = 15.48 | 1/(1.075)^2 = 0.8653     | 13.39                  |
| 2028 | $61.89 * (1+0.15) = 71.17 | $15.48 * (1+0.15) = 17.80 | 1/(1.075)^3 = 0.8049     | 14.32                  |
| 2029 | $71.17 * (1+0.10) = 78.29 | $17.80 * (1+0.10) = 19.58 | 1/(1.075)^4 = 0.7487     | 14.66                  |
| 2030 | $78.29 * (1+0.10) = 86.12 | $19.58 * (1+0.10) = 21.54 | 1/(1.075)^5 = 0.6965     | 15.00                  |

**預測期內FCF現值總和 (Sum of PV of FCFs):**
$12.52 + $13.39 + $14.32 + $14.66 + $15.00 = $69.89 billion

**終值 (Terminal Value, TV) 計算：**
TV = FCF_2031 / (WACC - g)
FCF_2031 = FCF_2030 * (1 + g) = $21.54 * (1 + 0.025) = $22.08 billion
TV = $22.08 / (0.075 - 0.025) = $22.08 / 0.05 = $441.60 billion

**終值現值 (Present Value of Terminal Value, PV of TV):**
PV of TV = TV / (1 + WACC)^5 = $441.60 / (1.075)^5 = $441.60 * 0.6965 = $307.60 billion

**企業價值 (Enterprise Value, EV):**
EV = 預測期內FCF現值總和 + 終值現值
EV = $69.89 billion + $307.60 billion = $377.49 billion

**股權價值 (Equity Value):**
股權價值 = 企業價值 + 現金及約當現金 - 總債務
由於缺乏詳細的資產負債表數據，我將假設淨債務為零，或將其納入 WACC 計算中。因此，股權價值 ≈ 企業價值。
股權價值 = $377.49 billion

**每股估值 (Intrinsic Value Per Share):**
流通股數 (Shares Outstanding): $161.79B (Market Cap) / $37.74 (Current Price) ≈ 4.287 billion 股 (此為反推計算，實際流通股數可能略有不同，但用於估值分析應合理).

#### 3.3. 股息折現模型 (Dividend Discount Model, DDM)

Novo Nordisk 是一家支付股息的公司，因此 DDM 適用。

**DDM 假設：**
*   **當前每股股息 (Latest Annual Dividend, D0):** 根據 Macrotrends，TTM 股息為 $0.82 (截至 2026年3月2日)。Simply Wall St 顯示當前收益率為 5.05%，如果以 $37.74 股價計算，則 D0 = $37.74 * 0.0505 = $1.90。Investing.com 顯示股息收益率為 3.52%。由於數據來源差異較大，我將使用 Macrotrends 的 TTM 股息 $0.82 作為保守估計，並考慮其歷史增長。
*   **股息增長率 (Dividend Growth Rate, g):** 考慮到公司穩定的盈利能力和市場地位，但同時面臨競爭壓力，我將假設股息長期增長率為 **4%**。
*   **股權成本 (Cost of Equity, Ke):** 我將使用 CAPM 模型估算。
    *   無風險利率 (Risk-Free Rate): 假設為 10年期美國國債收益率，約 **4.5%**。
    *   市場風險溢價 (Market Risk Premium, ERP): 假設為 **5.5%**。
    *   Beta: 由於 NVO 的 Beta 數據在不同來源有差異，且 GuruFocus 提到其 Beta 無法獲得並預設為 1。我將採用行業平均 Beta，或根據其穩定性設定為 **0.9**。
    *   Ke = 無風險利率 + Beta * 市場風險溢價 = 4.5% + 0.9 * 5.5% = 4.5% + 4.95% = **9.45%**

**DDM 計算 (Gordon Growth Model):**
P0 = D1 / (Ke - g)
D1 = D0 * (1 + g) = $0.82 * (1 + 0.04) = $0.8528

P0 = $0.8528 / (0.0945 - 0.04) = $0.8528 / 0.0545 = **$15.65**

**DDM 估值結果：**
*   **合理價位區間:** $14.00 - $18.00
*   **估值結果:** 每股 **$15.65**

**DDM 模型的局限性：** 由於 NVO 的股息收益率和股息增長率數據存在一定差異，且 DDM 模型對增長率和股權成本的敏感度較高，此結果可能較為保守。特別是如果採用 Simply Wall St 的 5.05% 股息收益率，D0 = $1.90，則 P0 = $1.90 * (1+0.04) / (0.0945 - 0.04) = $1.976 / 0.0545 = $36.26。這與當前股價更接近。這表明股息數據的選擇對 DDM 結果影響巨大。考慮到 NVO 的增長潛力，單純的股息折現可能無法完全捕捉其價值。
