### **估值方法與結果**

#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型通過預測公司未來的自由現金流 (FCF) 並將其折現至現值來估計公司價值。

**估值假設：**
*   **預測期間:** 5 年 (2025-2029)
*   **折現率 (WACC):** 7.5% (基於行業平均和公司風險，並參考 GuruFocus 提供的 6.97% 和 Stock Analysis 提供的 5.39% 等數據，取一個較為保守的估計)
*   **永續增長率 (Terminal Growth Rate):** 2.5% (反映長期通脹和經濟增長，適用於成熟的製藥公司)

**自由現金流 (FCF) 估計：**
根據提供的 P/FCF (14.12) 和最新市值 ($165.58B)，估計 LTM FCF 約為 $165.58B / 14.12 = $11.72 億美元。
考慮到公司在 GLP-1 領域的強勁增長和 AI 合作的潛力，儘管有2026年的短期逆風，我們採用以下 FCF 增長率假設：
*   2025年: 18%
*   2026年: -5% (維持短期挑戰)
*   2027年: 15%
*   2028年: 12%
*   2029年: 10%

| 年度 | FCF 增長率 | 估計 FCF (億美元) | 折現因子 (1/(1+WACC)^n) | PV of FCF (億美元) |
| :--- | :--------- | :---------------- | :----------------------- | :------------------- |
| LTM  | -          | 11.72             | -                        | -                    |
| 2025 | 18%        | 13.83             | 0.9302                   | 12.86                |
| 2026 | -5%        | 13.14             | 0.8653                   | 11.37                |
| 2027 | 15%        | 15.11             | 0.8049                   | 12.16                |
| 2028 | 12%        | 16.92             | 0.7487                   | 12.67                |
| 2029 | 10%        | 18.61             | 0.6965                   | 12.96                |

**計算終值 (Terminal Value, TV):**
TV = FCF (2029) * (1 + 永續增長率) / (WACC - 永續增長率)
TV = $18.61 * (1 + 0.025) / (0.075 - 0.025) = $381.505 億美元
PV of TV = $381.505 * 0.6965 = $265.79 億美元

**總企業價值 (Enterprise Value, EV):**
EV = PV of FCFs + PV of TV = ($12.86 + $11.37 + $12.16 + $12.67 + $12.96) + $265.79 = $62.02 + $265.79 = $327.81 億美元

**股權價值 (Equity Value):**
根據，公司有 $4.24B 現金和 $20.59B 債務，淨債務為 $16.35B。
股權價值 = EV - 淨債務 = $327.81B - $16.35B = $311.46 億美元。

**DCF 估值結果：**
*   **估值結果:** $311.46 億美元 (股權價值)
*   **合理價位區間:** 由於此估值遠高於當前市值 ($165.58B)，這表明 DCF 模型在當前假設下可能無法完全捕捉 NVO 的市場價值，或者我的 FCF 估計仍過於保守。這也可能反映了市場對其 GLP-1 藥物的未來潛力有更高的預期。

#### **2. 相對估值 (Comparable Company Analysis)**

相對估值法通過比較 NVO 與其同業公司的估值倍數來評估其價值。

**可比公司：** Eli Lilly and Company (LLY), Novartis (NVS), AstraZeneca (AZN), Sanofi (SNY), Pfizer (PFE), GSK (GSK)。

**NVO 的估值倍數：**
*   P/E (TTM): 10.28
*   P/S: 2.7
*   EV/EBITDA: 7.55
*   P/B: 5.46

**比較分析：**
NVO 的 P/E (10.28) 和 EV/EBITDA (7.55) 相對於一般大型製藥公司而言較低。這可能反映了市場對其2026年展望的擔憂，以及與 Eli Lilly 等競爭對手在 GLP-1 市場的激烈競爭。 然而，NVO 在 Q1 2024 表現強勁，且在肥胖症治療領域擁有領先地位和創新產品線。 如果市場認為其短期挑戰是暫時的，且長期增長潛力巨大，那麼其估值倍數可能被低估。

**相對估值結果：**
*   NVO 的估值倍數 (P/E, EV/EBITDA) 相對於其行業地位和增長潛力可能被低估。
*   **合理價位區間:** 如果 NVO 的 P/E 能夠達到行業平均水平 (例如 15x)，則基於 LTM EPS ($3.69)，股價約為 $55.35。分析師平均目標價 $43.00 - $65.56 也支持其有上漲空間。

#### **3. 股息折現模型 (Dividend Discount Model, DDM)**

DDM 適用於支付穩定股息且股息增長可預測的公司。NVO 支付半年度股息。

**估值假設：**
*   **當前年股息 (D0):** $1.29/股
*   **股息增長率 (g):**
    *   短期 (未來3年): 8.79% (基於歷史股息增長率)
    *   長期 (永續增長率): 2.5% (與 DCF 模型的永續增長率一致)
*   **所需報酬率 (r):** 7.5% (與 WACC 一致)

**計算每股估值：**
*   D1 = $1.29 * (1 + 0.0879) = $1.403
*   D2 = $1.403 * (1 + 0.0879) = $1.526
*   D3 = $1.526 * (1 + 0.0879) = $1.660
*   P3 (終值) = $1.660 * (1 + 0.025) / (0.075 - 0.025) = $34.03
*   股價 = $1.403/(1.075)^1 + $1.526/(1.075)^2 + $1.660/(1.075)^3 + $34.03/(1.075)^3 = $1.305 + $1.320 + $1.336 + $27.393 = $31.354

**DDM 估值結果：**
*   **估值結果:** $31.35/股
*   **合理價位區間:** $30 - $35/股。此估值也低於當前股價 $37.98。

### **綜合結論與建議**

綜合來看，不同估值模型給出了不同的結果：DCF 模型在較為積極的增長假設下給出了高於當前市值的估值，而 DDM 模型則給出了較低的估值。相對估值法顯示 NVO 的估值倍數可能被低估，且分析師目標價普遍高於當前股價。

**綜合分析：**
DCF 和 DDM 模型在保守假設下給出的估值較低，這可能反映了這些模型在捕捉像 Novo Nordisk 這樣具有強大創新能力和高增長潛力（尤其是在 GLP-1 藥物市場）的製藥公司時的局限性。市場可能對其未來增長，特別是其新產品線（如 Wegovy）的市場滲透率和盈利能力，抱有更高的預期。

相對估值法顯示 NVO 的估值倍數相對較低，這可能是一個買入機會，前提是市場對其2026年的短期逆風過於悲觀，而忽略了其長期增長潛力。分析師的目標價也普遍支持其有上漲空間。

**主要考量因素：**
1.  **GLP-1 藥物市場的領導地位:** Novo Nordisk 在糖尿病和肥胖症治療領域的 GLP-1 藥物（Ozempic, Wegovy）是其核心增長動力，市場需求持續強勁。
2.  **創新與研發:** 公司持續投入研發，並積極探索人工智慧在藥物發現中的應用，這為其長期增長提供了潛力。
3.  **2026年展望的挑戰:** 美國定價壓力及部分藥物專利到期帶來的短期負面影響不容忽視。
4.  **競爭加劇:** Eli Lilly 等競爭對手在肥胖症藥物市場的崛起，將對 NVO 的市場份額和定價能力構成挑戰。

**合理價位區間：**
考慮到分析師的目標價範圍 ($43.00 - $65.56) 以及其在相對估值中的潛在低估，我認為 NVO 的合理價位區間可能在 **$45 - $55/股** 之間。這反映了其核心業務的強勁表現和長期增長潛力，同時也考慮到短期內的挑戰和競爭壓力。

**投資建議：**
對於長期投資者而言，如果相信 Novo Nordisk 能夠有效應對競爭並持續創新，其在當前價位可能具有吸引力。然而，投資者應密切關注其2026年的財報表現以及 GLP-1 市場的競爭格局。

**與同產業可比公司的估值比較 (簡要):**
*   **Eli Lilly (LLY):** 作為 NVO 在 GLP-1 市場的主要競爭對手，LLY 通常享有更高的估值倍數，反映了市場對其強勁增長和創新能力的更高預期。如果 NVO 能夠持續證明其在肥胖症市場的領導地位和增長潛力，其估值倍數有望向 LLY 靠攏。
*   **其他大型製藥公司 (NVS, AZN, SNY, PFE, GSK):** 這些公司通常擁有更為多元化的產品線，但可能缺乏 NVO 在 GLP-1 領域的爆發性增長。NVO 相對較低的 P/E 和 EV/EBITDA 倍數，在考慮其增長前景後，可能使其更具吸引力。

總體而言，NVO 是一家具有強大基本面和增長潛力的公司，但其估值受到短期挑戰和激烈競爭的影響。長期來看，其在創新和市場領導地位方面的優勢有望推動其價值增長。
