### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測未來自由現金流 (Free Cash Flow to Firm, FCFE 或 FCFF) 並以 WACC 進行折現。由於我們沒有詳細的財務報表來計算 FCFF，我將基於營收增長預期和利潤率來估計。

**估值假設:**

*   **預測期間:** 5 年 (2026-2030)
*   **折現率 (WACC):** 7.86%
*   **長期增長率 (Terminal Growth Rate):** 2.5% (考慮到製藥產業的成熟性、創新潛力以及通脹水平，這是一個合理的長期增長率)
*   **營收增長率:**
    *   2026 年: -9% (取公司指引 -5% 至 -13% 的中位數)
    *   2027-2028 年: 5% (考慮到 Wegovy HD 和 Awiqli 的上市，以及 GLP-1 市場的持續增長，但競爭加劇)
    *   2029-2030 年: 4% (增長趨於穩定)
*   **營業利潤率 (Operating Margin):** 41.17% (來自提供數據，假設未來保持穩定，儘管有定價壓力，但高效的營運和新產品可能抵消部分影響)
*   **稅率:** 21.53% (來自最新數據)
*   **資本支出 (Capital Expenditure) / 銷售額:** 假設為 10% (製藥公司需要持續投入研發和產能擴張)
*   **折舊與攤銷 (Depreciation & Amortization) / 銷售額:** 假設為 5% (通常低於資本支出)
*   **營運資本變動 (Change in Working Capital) / 銷售額:** 假設為 1% (輕微增長)

**計算步驟:**

1.  **計算預測期內的營收 (Revenue):**
    *   2025 年營收: 46.81 億美元
    *   22026 年: 46.81 * (1 - 0.09) = 42.59 億美元
    *   2027 年: 42.59 * (1 + 0.05) = 44.72 億美元
    *   2028 年: 44.72 * (1 + 0.05) = 46.96 億美元
    *   2029 年: 46.96 * (1 + 0.04) = 48.84 億美元
    *   2030 年: 48.84 * (1 + 0.04) = 50.80 億美元

2.  **計算預測期內的營業利潤 (EBIT):**
    *   EBIT = Revenue * Operating Margin
    *   2026 年: 42.59 * 0.4117 = 17.54 億美元
    *   2027 年: 44.72 * 0.4117 = 18.42 億美元
    *   2028 年: 46.96 * 0.4117 = 19.34 億美元
    *   2029 年: 48.84 * 0.4117 = 20.12 億美元
    *   2030 年: 50.80 * 0.4117 = 20.92 億美元

3.  **計算稅後營業利潤 (NOPAT):**
    *   NOPAT = EBIT * (1 - Tax Rate)
    *   2026 年: 17.54 * (1 - 0.2153) = 13.74 億美元
    *   2027 年: 18.42 * (1 - 0.2153) = 14.43 億美元
    *   2028 年: 19.34 * (1 - 0.2153) = 15.15 億美元
    *   2029 年: 20.12 * (1 - 0.2153) = 15.76 億美元
    *   2030 年: 20.92 * (1 - 0.2153) = 16.39 億美元

4.  **計算自由現金流 (FCFF):**
    *   FCFF = NOPAT + Depreciation & Amortization - Capital Expenditure - Change in Working Capital
    *   假設 D&A = 0.05 * Revenue, CapEx = 0.10 * Revenue, Change in WC = 0.01 * Revenue
    *   2026 年: 13.74 + (42.59 * 0.05) - (42.59 * 0.10) - (42.59 * 0.01) = 13.74 + 2.13 - 4.26 - 0.43 = 11.18 億美元
    *   2027 年: 14.43 + (44.72 * 0.05) - (44.72 * 0.10) - (44.72 * 0.01) = 14.43 + 2.24 - 4.47 - 0.45 = 11.75 億美元
    *   2028 年: 15.15 + (46.96 * 0.05) - (46.96 * 0.10) - (46.96 * 0.01) = 15.15 + 2.35 - 4.70 - 0.47 = 12.33 億美元
    *   2029 年: 15.76 + (48.84 * 0.05) - (48.84 * 0.10) - (48.84 * 0.01) = 15.76 + 2.44 - 4.88 - 0.49 = 12.83 億美元
    *   2030 年: 16.39 + (50.80 * 0.05) - (50.80 * 0.10) - (50.80 * 0.01) = 16.39 + 2.54 - 5.08 - 0.51 = 13.34 億美元

5.  **計算終值 (Terminal Value, TV):**
    *   TV = FCFF (2030) * (1 + Long-term Growth Rate) / (WACC - Long-term Growth Rate)
    *   TV = 13.34 * (1 + 0.025) / (0.0786 - 0.025) = 13.34 * 1.025 / 0.0536 = 255.07 億美元

6.  **計算各期自由現金流的現值 (PV of FCFF) 和終值的現值 (PV of TV):**
    *   PV(FCFF 2026) = 11.18 / (1 + 0.0786)^1 = 10.36 億美元
    *   PV(FCFF 2027) = 11.75 / (1 + 0.0786)^2 = 10.14 億美元
    *   PV(FCFF 2028) = 12.33 / (1 + 0.0786)^3 = 9.94 億美元
    *   PV(FCFF 2029) = 12.83 / (1 + 0.0786)^4 = 9.70 億美元
    *   PV(FCFF 2030) = 13.34 / (1 + 0.0786)^5 = 9.50 億美元
    *   PV(TV) = 255.07 / (1 + 0.0786)^5 = 181.42 億美元

7.  **計算企業價值 (Enterprise Value, EV):**
    *   EV = Sum of PV(FCFF) + PV(TV)
    *   EV = 10.36 + 10.14 + 9.94 + 9.70 + 9.50 + 181.42 = 231.06 億美元

8.  **計算股權價值 (Equity Value):**
    *   Equity Value = EV + Cash - Debt
    *   根據最新數據，公司有 42.4 億美元現金和 205.9 億美元債務。
    *   Equity Value = 231.06 + 4.24 - 20.59 = 214.71 億美元

9.  **計算每股價值 (Intrinsic Value Per Share):**
    *   總股數 = Market Cap / Close Price = 160.92 億美元 / 36.04 美元 ≈ 4.465 億股 (使用最新市值)
    *   Intrinsic Value Per Share = Equity Value / Total Shares Outstanding
    *   Intrinsic Value Per Share = 214.71 億美元 / 4.465 億股 = 48.09 美元

**DCF 估值結果:**

*   **每股合理價位:** 48.09 美元
*   **合理價位區間:** 43.00 - 53.00 美元 (考慮到假設的敏感性)

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

NVO 是一家成熟且支付股息的公司，因此 DDM 適用。

**估值假設:**

*   **當前股息 (Annual Dividend):** 1.84 美元 (最新)

*   **折現率 (Required Rate of Return, Ke):** 7.86% (使用 WACC 作為股權的折現率，假設資本結構穩定且風險與公司整體風險一致)

**計算步驟 (兩階段 DDM):**

1.  **計算預測期內的股息:**
    *   D0 (當前股息) = 1.84 美元
    *   D1 (2026) = 1.84 * (1 + 0.07) = 1.97 美元

2.  **計算預測期後第一年的股息 (用於計算終值):**

3.  **計算終值 (Terminal Value of Dividends) (在預測期結束時):**
