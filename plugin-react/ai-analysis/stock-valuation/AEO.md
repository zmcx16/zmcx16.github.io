#### **3. 股息折現模型 (Dividend Discount Model, DDM)**

AEO 是一家支付股息的公司，因此 DDM 適用。

**估值假設：**

*   **當前股價：** $26.65
*   **股息率 (Dividend %):** 1.88%
*   **當前每股年股息 (D0)：** $26.65 * 0.0188 = **$0.50**
*   **股息增長率 (g)：** 考慮到公司仍有成長空間，但作為成熟零售商，股息增長會較為溫和。我們假設長期股息增長率為 **4.0%**。
*   **所需報酬率 (r)：** 使用股權成本 (Cost of Equity) **11.35%**。

**戈登增長模型 (Gordon Growth Model)：**
P = D1 / (r - g)
其中 D1 = D0 * (1 + g)

*   D1 = $0.50 * (1 + 0.04) = **$0.52**
*   P = $0.52 / (0.1135 - 0.04) = $0.52 / 0.0735 = **$7.07**

**DDM 估值結果與合理價位區間：**
DDM 模型估算的 AEO 每股內在價值約為 **$7.07**。
這個結果非常低，主要原因在於 AEO 的股息率相對不高，且公司仍將部分盈利用於再投資以支持 Aerie 等品牌的成長，而非全部以股息形式發放。因此，DDM 可能低估了 AEO 的價值，或者說，AEO 並非典型的「股息成長型」股票，其價值更多體現在未來的盈利增長而非當前股息。
