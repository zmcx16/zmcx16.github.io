#### **模型三：股息折現模型 (Dividend Discount Model, DDM)**

Salesforce 最近才開始支付股息。
*   年股息：$1.76/股 (或 $1.66/股)
*   季度股息：$0.44/股
*   股息增長：4.00% 或 4.46% (僅增長 1 年)
*   股息支付率：21.28% 或 17% 或 22.56% (健康水平)
*   股息收益率：0.9% 或 0.85%

由於 Salesforce 剛開始支付股息，且股息增長歷史較短，其主要價值驅動因素仍是盈利和自由現金流的增長，而非股息。因此，DDM 在此情況下可能不是最合適或最可靠的估值方法。然而，我們可以嘗試使用戈登增長模型 (Gordon Growth Model) 進行簡化估算，但需謹慎對待結果。

**估值假設：**
1.  **最新年股息 (D0)：** $1.76/股
2.  **股息增長率 (g)：** 4.0% (參考最新增長率，但由於歷史短，存在不確定性)。
3.  **股權成本 (Ke)：** 我們可以使用 CAPM 模型估算。
    *   無風險利率 (Risk-Free Rate)：約 3.95% (美國 10 年期國債收益率)。
    *   Beta：0.97 或 1.30 (我們取 1.1 作為中間值，考慮其波動性)。
    *   市場風險溢價 (Market Risk Premium, ERP)：4.18% 或 6% (我們取 5%)。
    *   Ke = 無風險利率 + Beta * ERP = 3.95% + 1.1 * 5% = 3.95% + 5.5% = 9.45%

**DDM 計算 (戈登增長模型)：**
每股估值 = D0 * (1 + g) / (Ke - g)
每股估值 = 1.76 * (1 + 0.04) / (0.0945 - 0.04)
每股估值 = 1.76 * 1.04 / 0.0545
每股估值 = 1.8304 / 0.0545 = **$33.58**

**DDM 模型估值結果：**
*   **每股估值：$33.58**
*   **合理價位區間：$25 - $45**

**結論：** DDM 估值結果遠低於當前股價，這再次印證了 DDM 對於像 Salesforce 這樣主要依靠增長而非股息來創造價值的公司來說，並不是一個合適的估值工具。其股息支付率較低 (21.28%)，表明公司將大部分收益再投資於業務增長。
