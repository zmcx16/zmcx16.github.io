2.  **現金流預期過於悲觀：** 如果我們對未來 FCF 的預測過於保守，或者 WACC 過高，會導致企業價值被低估。
3.  **市場對未來增長和盈利能力的擔憂：** 儘管疫情後復甦，但航空業仍面臨燃料價格、勞動力成本、地緣政治和經濟衰退的風險，這可能導致市場對其未來盈利能力持謹慎態度。
4.  **數據來源和計算的準確性：** 尤其是淨債務的計算，需要非常精確的資產負債表數據。

**考慮到 UAL 的市值為 361 億美元，市場顯然認為其股權價值為正。因此，我們的模型假設可能存在偏差。**

**調整 DCF 模型的淨債務和 WACC 假設：**

*   **淨債務：** 根據 Forbes 數據，UAL 總債務為 313.2 億美元，總現金為 133.3 億美元。淨債務 = 179.9 億美元。
*   **WACC：** 7.60% 可能過高。讓我們嘗試使用行業平均 WACC 或更低的 WACC。如果我們假設 WACC 為 6.5% (考慮到復甦和低利率環境的潛在影響，但仍高於無風險利率)，長期增長率 2.5%。
    *   **TV = 2.42 / (0.065 - 0.025) = 2.42 / 0.04 = 60.5 Billion USD**
    *   **終值現值** = 60.5 * 0.692 (2030 年折現因子) = 41.87 Billion USD
    *   **企業價值 (EV) = 7.28 + 41.87 = 49.15 Billion USD**
    *   **股權價值 = EV - 淨債務** = 49.15 - 17.99 = 31.16 Billion USD
    *   **每股價值 = 31.16 Billion USD / 323.7 百萬股** (36.10B 市值 / 111.52 股價) = **96.26 美元/股**。

這個調整後的 DCF 結果更為合理，但仍低於當前股價。
