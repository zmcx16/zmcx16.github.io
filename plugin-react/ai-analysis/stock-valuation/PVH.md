#### 3.4 股息折現模型 (DDM)

PVH 支付股息 (0.23%)，因此 DDM 適用。

**DDM 假設回顧:**
*   股權成本 (Ke): 13.615%
*   最新季度股息: 0.0375 美元/股 (2026 年 2 月 4 日宣布)
*   年化股息 (D0): 0.0375 * 4 = 0.15 美元/股
*   股息增長率 (g): 1.5% (保守估計)

**DDM 計算:**
*   DDM 估值 = D0 * (1 + g) / (Ke - g)
*   DDM 估值 = 0.15 * (1 + 0.015) / (0.13615 - 0.015)
*   DDM 估值 = 0.15 * 1.015 / 0.12115
*   DDM 估值 = 0.15225 / 0.12115 = **1.26 美元**

**DDM 估值結果:**
*   **每股合理價位: 1.26 美元**

**DDM 模型的局限性:**
PVH 的股息率非常低，且股息增長率預期也較低。這使得 DDM 模型對股息增長率和折現率的微小變化非常敏感，導致估值結果可能不具代表性。在這種情況下，DDM 通常不被視為主要估值方法。
