2.  **計算預測期內的自由現金流 (FCFF)：**

3.  **計算終值 (Terminal Value, TV)：**
    TV = FCFF_n * (1 + g) / (WACC - g)

4.  **將 FCFF 和 TV 折現至現值。**
5.  **計算企業價值 (Enterprise Value, EV)。**
6.  **計算股權價值 (Equity Value)：**

3.  **WACC 的選擇：** 我使用的 WACC (7.17%) 相對較低，這會導致更高的估值。如果採用 GuruFocus 提供的 9.21% 或 ValueInvesting.io 的 8.7% WACC，估值將會顯著下降。例如，如果 WACC 為 7.67%，DCF 估值範圍的上限會降至 250.10 美元。
4.  **市場情緒和風險溢價：** 當前市場可能對服裝零售業的未來增長和盈利能力持悲觀態度，導致給予較低的估值倍數。PVH 的 Beta 值在不同來源有較大差異 (0.75 到 1.72)，較高的 Beta 值會導致更高的股權成本和 WACC，進而降低估值。

**與同產業可比公司的估值比較：**

**綜合結論：**

基於 DCF 和相對估值模型，PVH 的內在價值似乎遠高於其當前股價。這兩種模型都指向每股 200 美元以上的估值。然而，分析師的共識目標價則更為保守，約在 90-100 美元區間。
