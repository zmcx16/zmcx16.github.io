    *   PVH 的 EV/EBITDA 根據 Finbox 數據為 7.5x - 8.0x (與 Crocs 和 G-III 接近)。
    *   可比公司 EV/EBITDA 範圍從 G-III 的 5.0x 到 Steven Madden 的 19.0x。中位數約在 10-15x 之間。
    *   如果 PVH 的 EV/EBITDA 達到行業中位數 12x。
    *   需要計算 PVH 的 TTM EBITDA。
    *   根據 Stock Analysis，PVH 的 TTM EBITDA 為 9.222 億美元。
    *   企業價值 (EV) = EBITDA \* 倍數 = 9.222 億 \* 12 = 110.664 億美元
    *   股權價值 = EV + 現金 - 債務 = 110.664 億 + 1.582 億 - 23 億 = 89.246 億美元
    *   每股價值 = 89.246 億 / 4670 萬 = **191.11 美元**

### 5. 股息折現模型 (Dividend Discount Model, DDM)

PVH 的股息率為 0.0024 (0.24%)，股息非常低。這表明公司更傾向於將利潤再投資於業務增長，而不是通過股息回報股東。對於股息率如此低的公司，DDM 模型通常不適用，因為股息在估值中佔比極小，且未來股息增長預測難度大。

然而，如果我們嘗試應用戈登增長模型 (Gordon Growth Model) 作為參考：
*   股息 (D0) = 當前股價 \* 股息率 = 61.23 \* 0.0024 = 0.147 美元
*   假設股息增長率 (g) 與長期 FCF 增長率相同，為 2.0%。
*   股權成本 (Ke) = 10.483%
*   每股價值 = D0 \* (1 + g) / (Ke - g)
*   每股價值 = 0.147 \* (1 + 0.02) / (0.10483 - 0.02)
*   每股價值 = 0.147 \* 1.02 / 0.08483
*   每股價值 = 0.14994 / 0.08483 = **1.77 美元**

這個結果遠低於當前股價，再次印證了 DDM 不適用於 PVH 這種低股息公司。
