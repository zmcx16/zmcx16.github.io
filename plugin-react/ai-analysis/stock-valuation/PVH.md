**DCF 計算步驟：**

1.  **計算預測期內的稅後營業利潤 (NOPAT)**

2.  **計算預測期內的自由現金流 (FCF)**
    FCF = NOPAT + D&A - CapEx - ΔNWC
3.  **計算終值 (Terminal Value, TV)**
    TV = (最後一年 FCF × (1 + 長期增長率)) / (WACC - 長期增長率)
4.  **將 FCF 和 TV 折現至現值**
5.  **計算企業價值 (Enterprise Value, EV)**
    EV = 預測期 FCF 現值總和 + TV 現值
6.  **計算股權價值 (Equity Value)**
    Equity Value = EV - 淨負債 (Net Debt)
7.  **計算每股價值 (Intrinsic Value per Share)**
    Intrinsic Value per Share = Equity Value / 流通股數
