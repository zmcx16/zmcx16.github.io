### 估值假設

為了進行估值分析，我們將採用以下關鍵假設：

**1. 加權平均資本成本 (WACC) 計算：**
*   **無風險利率 (Rf):** 採用美國 10 年期國債殖利率，截至 2026 年 3 月 12 日為 4.24%。
*   **股票風險溢價 (ERP):** 採用 5.5%，這是美國市場常用的合理中位數。
*   **Beta (β):** 根據多個來源，PVH 的 Beta 值介於 0.76 至 1.73 之間。為反映公司對市場波動的敏感度，我們將使用兩個 Beta 值進行敏感性分析：
    *   **保守 Beta:** 0.80 (接近多數來源的較低端)。
    *   **較高 Beta:** 1.67 (來自 MarketBeat 的較高估計)。
*   **股權成本 (Ke) = Rf + β * ERP**
    *   保守 Beta (0.80): Ke = 4.24% + 0.80 * 5.5% = 8.64%
    *   較高 Beta (1.67): Ke = 4.24% + 1.67 * 5.5% = 13.425%
*   **債務成本 (Kd):** 根據 PVH 債券殖利率範圍 (3.4% 至 5.2%)，估計為 4.5%。
*   **稅率:** 採用美國企業標準稅率 21% 作為長期正常化稅率。
*   **股權市值 (E):** 28.1 億美元 (提供數據)。
*   **債務市值 (D):** 22 億美元 (截至 2026 年 Q3 的長期債務)。
*   **WACC = (E / (E+D)) * Ke + (D / (E+D)) * Kd * (1 - 稅率)**
    *   **保守 WACC (基於 Beta 0.80):** (2.81 / (2.81 + 2.2)) * 8.64% + (2.2 / (2.81 + 2.2)) * 4.5% * (1 - 0.21) = 0.5609 * 8.64% + 0.4391 * 4.5% * 0.79 = 4.85% + 1.56% = **6.41%**
    *   **較高 WACC (基於 Beta 1.67):** (2.81 / (2.81 + 2.2)) * 13.425% + (2.2 / (2.81 + 2.2)) * 4.5% * (1 - 0.21) = 0.5609 * 13.425% + 0.4391 * 4.5% * 0.79 = 7.54% + 1.56% = **9.10%**

*   **預測期間:** 5 年 (2026-2030)。
*   **終端增長率 (g):** 2.5% (略高於長期通膨，適用於成熟產業)。

    *   2026 年: 1.5%
    *   2027 年: 1.0%
    *   2028-2030 年: 0.5%
*   **營業利潤率 (EBIT Margin):** 採用 2025 財年非 GAAP 營業利潤率 8.5%，並假設在預測期內保持穩定。
*   **折舊與攤銷 (D&A):** 假設為營收的 2%。
*   **資本支出 (CapEx):** 假設為營收的 2.2%。

綜合來看，DCF 模型和相對估值模型均顯示 PVH 目前的股價 61.03 美元被顯著低估。

PVH 的 P/E (9.2) 和 P/S (0.32) 顯著低於同業平均水平 (P/E 16.58，P/S 0.85)。即使考慮到公司面臨的挑戰，這種折價也表明市場可能過度悲觀。PVH 的 EV/EBITDA (7.1) 也低於同業平均 (11.39)，進一步支持了其被低估的結論。

**潛在低估的原因：**
市場可能因以下因素而對 PVH 進行大幅折價：
*   **近期財務挑戰：** 2025 財年第三季度淨利潤下降、自由現金流為負以及關稅影響。

*   **戰略轉型期：** 儘管「PVH+ 計劃」和與 OpenAI 的合作旨在實現長期增長和效率提升，但市場可能仍在等待更具體的成果和持續的正面財務表現。
*   **分析師目標價：** 分析師的平均目標價約為 90.09 美元，雖然較當前股價有顯著上漲空間，但仍低於我們 DCF 和相對估值模型的中位數，這可能意味著分析師也考慮了當前的逆風或採用了更為保守的假設。

**綜合結論與合理價位區間：**
基於全面的估值分析，PVH Corp. 的內在價值顯著高於其當前市場價格。儘管公司面臨短期挑戰，但其強大的品牌組合 (Calvin Klein、Tommy Hilfiger) 和積極的戰略轉型計劃有望在長期內帶來價值。
