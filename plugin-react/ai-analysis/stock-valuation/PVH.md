### 5. 股息折現模型 (DDM)

PVH 最近宣布季度現金股息為每股0.0375美元，年化股息為0.15美元。 股息收益率為0.24%。 考慮到其較低的股息和相對較高的股權成本，DDM 可能不是最主要的估值方法，但我們仍可進行計算。

*   **當前股息 (D0):** 0.15 美元/股
*   **股息增長率 (g):** 假設為 2.5% (介於2.0% - 3.0%之間)
*   **股權成本 (Ke):** 10.8%

**DDM 估值 = D0 * (1 + g) / (Ke - g)**
*   DDM 估值 = 0.15 * (1 + 0.025) / (0.108 - 0.025)
*   DDM 估值 = 0.15 * 1.025 / 0.083
