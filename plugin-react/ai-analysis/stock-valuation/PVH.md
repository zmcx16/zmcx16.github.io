**終值 (Terminal Value) 計算：**
終值 = FCF (2030) * (1 + g) / (WACC - g)

股權價值 = 公司總價值 + 現金及約當現金 - 總債務

股權價值 = 11,114 + 500 - 2,706 = 8,908 百萬美元

**3. 股息折現模型 (DDM)**

由於 PVH 的股息支付率非常低 (約 2.24%-2.32%) 且股息增長不顯著，DDM 可能不是最適合 PVH 的估值方法。然而，我們仍可進行計算。

DDM 估值 = D0 * (1 + g) / (Ke - g)
*   D0 = 0.15 美元
*   g = 0.5% (長期股息增長率)
*   Ke = 7.25% (股權成本)

估值 = 0.15 * (1 + 0.005) / (0.0725 - 0.005) = 0.15 * 1.005 / 0.0675 = 0.15075 / 0.0675 ≈ **2.23 美元/股**

這個結果遠低於當前股價，再次印證了 DDM 對於低股息支付且增長潛力不高的公司可能不適用，或者表明市場對其未來股息增長有更高的預期，但目前數據不支持。因此，我們將主要依賴 DCF 和相對估值。
