4.  **計算終值 (Terminal Value, TV)。**

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

PVH 支付季度股息 $0.0375/股，年化 $0.15/股，股息率 0.23%。股息率較低，且公司主要通過再投資來實現增長，因此 DDM 可能不是最合適的估值方法。然而，我們可以嘗試使用戈登增長模型 (Gordon Growth Model) 進行估算。

**DDM 假設：**

*   **當前股息 (D0):** $0.15 (年化)
*   **股息增長率 (g):** 考慮到 PVH 的增長策略和行業趨勢，以及其較低的派息率 (2.25%)，假設長期股息增長率為 2.0%。
*   **股權成本 (Ke):** 13.715% (來自 DCF 模型)

**DDM 計算：**

*   **每股價值 = D0 * (1 + g) / (Ke - g)**
    *   每股價值 = $0.15 * (1 + 0.02) / (0.13715 - 0.02)
    *   每股價值 = $0.15 * 1.02 / 0.11715
    *   每股價值 = $0.153 / 0.11715 = **$1.31**

**DDM 估值結果與合理價位區間：**

2.  **相對估值 (P/E):** 基於可比公司中位數 P/E，估值為 **$138.26**。這也表明 PVH 可能被低估。
3.  **相對估值 (其他倍數):** P/S, P/B, EV/EBITDA 的估值結果顯著偏高，可能不適用於 PVH，或者反映了市場對其增長和盈利能力的較低預期，導致其自身倍數遠低於行業平均。直接應用行業中位數可能導致過度樂觀。
4.  **DDM 模型:** 估值僅為 **$1.31**。由於 PVH 的股息支付相對較低，且其價值主要來自於再投資和增長，DDM 不適用於此公司。

**最終結論：**

綜合來看，DCF 模型和基於 P/E 倍數的相對估值都表明 PVH 目前的股價 **$66.24** 可能被低估。DCF 模型給出的內在價值為 **$127.02**，而基於可比公司 P/E 中位數的估值為 **$138.26**。分析師的共識目標價 **$92.80** 也高於當前股價。

PVH 擁有強大的品牌 (Calvin Klein 和 Tommy Hilfiger) 和積極的 "PVH+ Plan" 戰略，這有望推動未來的增長和盈利能力。儘管面臨宏觀經濟逆風和 Heritage Brands 的挑戰，但公司在 Q3 2025 的表現超出預期，並上調了全年展望，顯示出韌性。

考慮到不同模型的結果，以及市場對 PVH 的當前情緒和其自身較低的估值倍數，我認為 PVH 的合理價位區間應在 **$90 - $130** 之間。這反映了 DCF 估值的潛在上行空間，同時也考慮了分析師的共識目標價和市場可能存在的謹慎情緒。

**與同產業可比公司的估值比較：**

PVH 的 P/E (9.94) 和 EV/EBITDA (5.70) 倍數遠低於其主要競爭對手 Ralph Lauren (P/E 27.32, EV/EBITDA 17.33)、Levi Strauss (P/E 14.18, EV/EBITDA 9.3) 和 Tapestry (P/E 119.96, EV/EBITDA 18.83)。這表明市場對 PVH 的估值明顯較低。這種較低的估值可能反映了市場對其 Heritage Brands 業務的擔憂、宏觀經濟逆風的影響，或者對其 "PVH+ Plan" 執行效果的觀望態度。然而，如果 PVH 能夠成功執行其戰略並改善盈利能力，其估值倍數有很大的提升空間，從而推動股價上漲。
