## Hasbro (HAS) 股票估值分析

### 一、 公司概覽與近期發展

Hasbro (HAS) 是一家全球性的玩具和遊戲公司，旗下擁有眾多知名品牌，如 NERF、Play-Doh、Monopoly、Magic: The Gathering、Dungeons & Dragons 等。公司業務主要分為三個部門：玩具 (Consumer Products)、威世智與數位遊戲 (Wizards of the Coast and Digital Gaming) 以及娛樂 (Entertainment)。

**近期關鍵發展 (基於網路搜尋)：**

1.  **出售 eOne 娛樂業務：** 2023 年底，Hasbro 完成了對其娛樂內容部門 eOne (Entertainment One) 的出售，將其電影、電視和動畫業務出售給 Lionsgate。此舉旨在簡化業務結構，專注於核心玩具和遊戲品牌，並利用出售所得資金償還債務。這對公司的財務報表產生了重大影響，包括一次性費用和資產減值，導致近期出現虧損。
2.  **「Blueprint 2.0」策略：** 公司正積極執行其「Blueprint 2.0」策略，重點放在少數高潛力品牌上，並透過授權、數位化和全球擴張來提升這些品牌的價值。
3.  **威世智 (Wizards of the Coast) 表現強勁：** 旗下《魔法風雲會》(Magic: The Gathering) 和《龍與地下城》(Dungeons & Dragons) 等桌遊和數位遊戲品牌持續表現出色，是公司重要的成長引擎。
4.  **債務削減：** eOne 出售所得資金主要用於償還債務，這有助於改善公司的資產負債表結構，降低財務風險。

根據提供的數據和網路搜尋結果，我們對 Hasbro 的基本面進行分析：

### 三、 估值假設

在進行估值之前，我們需要設定一些關鍵假設。這些假設將基於提供的數據、網路搜尋結果以及行業常識。

**1. 折現率 (WACC - 加權平均資本成本)**

*   **無風險利率 (Risk-Free Rate, Rf)：** 採用美國 10 年期國債收益率，約 **4.20%** (截至近期)。
*   **市場風險溢酬 (Market Risk Premium, MRP)：** 採用行業標準，約 **5.50%**。
*   **Beta (β)：** 根據網路搜尋，Hasbro 的 Beta 值約在 1.0-1.2 之間，我們取 **1.10**。
*   **股權成本 (Cost of Equity, Ke)：** 根據 CAPM 模型：Ke = Rf + β * MRP
    Ke = 4.20% + 1.10 * 5.50% = **10.25%**
*   **稅率 (Tax Rate, t)：** 考慮到美國企業稅率和公司實際有效稅率，我們採用 **25%**。
*   **債務成本 (Cost of Debt, Kd)：** 考慮到 Hasbro 較高的負債水平和信用評級，我們假設其債務成本為 **7.00%**。
*   **債務與股權權重：**
    *   市值 (Market Cap, E)：$11.59 Billion
    *   總債務 (Total Debt, D)：根據近期財報和 eOne 出售後的債務削減，估計總債務約為 **$3.8 Billion** (此處需參考最新財報數據，若無精確數據則為合理估計)。
    *   總資本 (Total Capital) = E + D = $11.59B + $3.8B = $15.39 Billion
    *   股權權重 (E / (E+D)) = $11.59B / $15.39B = **75.31%**
    *   債務權重 (D / (E+D)) = $3.8B / $15.39B = **24.69%**
*   **WACC 計算：** WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1 - t)
    WACC = (0.7531 * 10.25%) + (0.2469 * 7.00% * (1 - 0.25))
    WACC = 7.71% + 1.29% = **9.00%**

**2. 增長率假設**

*   **短期增長率 (未來 5 年)：** 考慮到公司正從 eOne 出售和重組中恢復，並專注於核心品牌，預計初期增長較為溫和，隨後逐步穩定。
    *   第 1-3 年：**6.0%** (參考 EPS next Y_% 0.0603，並考慮核心品牌增長潛力)
    *   第 4-5 年：**4.5%** (逐步趨於穩定)
*   **長期增長率 (Terminal Growth Rate, g)：** 假設公司進入穩定增長階段，與長期通脹和全球經濟增長率相符，約為 **2.5%**。

**3. 預測期間：** 5 年詳細預測期，之後進入永續增長階段。

### 四、 估值模型計算

#### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**步驟：**
1.  估計當前自由現金流 (FCF)。
2.  預測未來 5 年的自由現金流。
3.  計算永續價值 (Terminal Value, TV)。
4.  將所有現金流折現至現值。
5.  計算股權價值和每股價值。

**a. 估計當前自由現金流 (FCF0)：**
根據 P/FCF 15.26 和 Market Cap $11.59B，我們可以反推出當前 FCF。
FCF0 = Market Cap / P/FCF = $11.59B / 15.26 = **$759.5 百萬**

**b. 預測未來 5 年自由現金流 (FCF)：**

| 年度 | 增長率 | 預測 FCF (百萬美元) | 折現因子 (WACC=9.00%) | 現值 (百萬美元) |
| :--- | :----- | :------------------ | :-------------------- | :-------------- |
| FCF0 |        | $759.5              |                       |                 |
| 1    | 6.0%   | $805.1              | 1 / (1+0.09)^1 = 0.9174 | $738.6          |
| 2    | 6.0%   | $853.4              | 1 / (1+0.09)^2 = 0.8417 | $718.3          |
| 3    | 6.0%   | $904.6              | 1 / (1+0.09)^3 = 0.7722 | $698.6          |
| 4    | 4.5%   | $945.3              | 1 / (1+0.09)^4 = 0.7084 | $669.7          |
| 5    | 4.5%   | $987.8              | 1 / (1+0.09)^5 = 0.6500 | $642.1          |
| **預測期 FCF 現值總和** |        |                     |                       | **$3,467.3**    |

**c. 計算永續價值 (Terminal Value, TV)：**
TV = FCF5 * (1 + g) / (WACC - g)
TV = $987.8 * (1 + 0.025) / (0.0900 - 0.025)
TV = $987.8 * 1.025 / 0.065
TV = $1012.5 / 0.065 = **$15,576.9 百萬**

**d. 永續價值現值 (PV of TV)：**
PV of TV = TV / (1 + WACC)^5
PV of TV = $15,576.9 / (1 + 0.09)^5
PV of TV = $15,576.9 / 1.5386 = **$10,124.1 百萬**

**e. 企業價值 (Enterprise Value, EV)：**
EV = 預測期 FCF 現值總和 + 永續價值現值
EV = $3,467.3 百萬 + $10,124.1 百萬 = **$13,591.4 百萬**

**f. 股權價值 (Equity Value)：**
Equity Value = EV + 現金及約當現金 - 總債務
假設公司持有現金約 $500 百萬 (需參考最新財報，此為合理估計)。
Equity Value = $13,591.4 百萬 + $500 百萬 - $3,800 百萬 = **$10,291.4 百萬**

**g. 每股價值 (Per Share Value)：**
流通股數 (Shares Outstanding) = Market Cap / Close Price = $11.59B / $82.56 ≈ 140.39 百萬股
每股價值 = Equity Value / Shares Outstanding
每股價值 = $10,291.4 百萬 / 140.39 百萬股 = **$73.30**

**DCF 估值結果：**
*   **合理價位：$73.30**
*   **合理價位區間：$68.00 - $78.00** (考慮到 WACC 和增長率假設的敏感性)

#### 3. 股息折現模型 (Dividend Discount Model, DDM)

Hasbro 支付股息，因此 DDM 適用。然而，由於公司近期盈利波動，且正在進行業務重組，股息增長率的預測存在較大不確定性。我們將採用戈登增長模型 (Gordon Growth Model) 進行估算，但需注意其局限性。

**假設：**
*   當前每股股息 (D0) = Close Price * Dividend % = $82.56 * 0.0339 = **$2.80**
*   股息增長率 (g)：考慮到公司需要償還債務和投資核心業務，股息增長可能較為保守。我們假設一個較低的長期穩定增長率，例如 **1.5%**。
*   股權成本 (Ke)：沿用 DCF 中的 10.25%。

**計算：**
DDM = D0 * (1 + g) / (Ke - g)
DDM = $2.80 * (1 + 0.015) / (0.1025 - 0.015)
DDM = $2.80 * 1.015 / 0.0875
DDM = $2.842 / 0.0875 = **$32.48**

**DDM 估值結果：**
*   **合理價位：$32.48**
*   **合理價位區間：$30.00 - $35.00**

**DDM 模型的局限性：**
DDM 估值結果遠低於當前股價和 DCF/相對估值。這可能表明：
1.  市場預期 Hasbro 未來的股息增長率會遠高於我們假設的 1.5%。
2.  市場對 Hasbro 的估值更多基於其未來盈利和自由現金流的增長潛力，而非單純的股息支付。
3.  在公司經歷重組和盈利波動時期，DDM 可能不是最適合的估值模型，因為其對增長率和折現率的微小變化非常敏感。

### 六、 風險與機會

**機會：**
*   **品牌重塑與聚焦：** 專注於核心品牌有助於提升效率和盈利能力。
*   **數位化轉型：** 威世智部門在數位遊戲領域的成功，以及將更多品牌數位化的潛力。
*   **授權業務增長：** 透過授權其 IP 給第三方，創造新的收入來源。
*   **債務削減：** 改善資產負債表，降低利息支出，提升財務靈活性。

**風險：**
*   **執行風險：** 「Blueprint 2.0」策略的執行不及預期，導致增長放緩或成本控制不力。
*   **宏觀經濟逆風：** 消費者可支配收入減少可能影響玩具和遊戲的銷售。
*   **競爭激烈：** 玩具和遊戲市場競爭激烈，來自其他大廠和新興品牌的壓力。
*   **高負債：** 儘管有所削減，但相對較高的負債水平仍可能在利率上升時增加財務壓力。
*   **IP 依賴：** 過度依賴少數核心 IP 可能帶來風險，若這些 IP 失去吸引力，將對公司造成重大打擊。
