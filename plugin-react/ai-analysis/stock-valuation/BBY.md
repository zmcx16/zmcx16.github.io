**2.1 加權平均資本成本 (WACC) - 折現率**

WACC = (E/V) * Re + (D/V) * Rd * (1 - t)

*   **股權成本 (Re) - 使用資本資產定價模型 (CAPM):**

*   **債務成本 (Rd):**
    *   參考公司債務的平均利率或信用評級。假設稅前債務成本為 **5.5%**。
    *   **稅率 (t):** 參考公司有效稅率。假設為 **24%** (根據近期財報或行業平均)。

*   **資本結構 (E/V, D/V):**

    *   **總資本 (V) = E + D = $14.61B + $5.5B = $20.11 Billion**

    *   **債務權重 (D/V) = $5.5B / $20.11B = 0.274**
