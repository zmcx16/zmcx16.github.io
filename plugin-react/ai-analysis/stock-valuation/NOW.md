### 2. 估值方法與假設

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期：** 5 年明確預測期，之後進入永續增長階段。
*   **折現率 (WACC)：** 採用 8.3%。
    *   風險自由利率 (Risk-Free Rate)：約 4.42% (基於政府債券收益率)。
    *   Beta：0.96。
    *   股權風險溢價 (ERP)：4.18%。
    *   股權成本 (Cost of Equity) = 4.42% + 0.96 \* 4.18% = 8.43%。
    *   稅後債務成本 (After-tax Cost of Debt)：ServiceNow 的債務權重約為 1.6%，債務成本約 4.64%。假設稅率為 26.15%，稅後債務成本 = 4.64% \* (1 - 0.2615) = 3.43%。
    *   WACC = (股權權重 \* 股權成本) + (債務權重 \* 稅後債務成本) = (0.984 \* 8.43%) + (0.016 \* 3.43%) = 8.29% + 0.05% = 8.34%。我將使用 8.3% 作為近似值。
*   **收入增長率：**
    *   ServiceNow 預計 2026 年訂閱收入增長 21%-22.5%。
    *   公司目標到 2030 年訂閱收入超過 300 億美元，複合年增長率約 20%。
    *   分析師預計未來幾年收入增長率約為 15.1% 至 20% 之間。
    *   我將假設前 3 年收入增長率為 20%，第 4-5 年降至 18%，之後永續增長率為 3.0%。
*   **營運利潤率：** Q1 2026 非 GAAP 營運利潤率為 32%。 公司預計 2027 年營運利潤率將擴大 100 個基點。 我將假設營運利潤率從 32% 逐步提升至 35%。
*   **稅率：** 26.15%。
*   **資本支出 (CAPEX) 與折舊攤銷 (D&A)：** 假設 CAPEX 佔收入的 3%，D&A 佔收入的 2%。
*   **營運資本變動：** 假設營運資本佔收入的 5%。
*   **永續增長率 (Terminal Growth Rate)：** 3.0%。

**DCF 計算步驟：**

1.  **預測收入：**
    *   2025 年總收入 (TTM) 約為 139.6 億美元。
    *   2026 年預計收入：139.6 億美元 \* (1 + 20%) = 167.52 億美元。
    *   2027 年預計收入：167.52 億美元 \* (1 + 20%) = 201.02 億美元。
    *   2028 年預計收入：201.02 億美元 \* (1 + 20%) = 241.22 億美元。
    *   2029 年預計收入：241.22 億美元 \* (1 + 18%) = 284.64 億美元。
    *   2030 年預計收入：284.64 億美元 \* (1 + 18%) = 335.88 億美元。

2.  **計算 EBIT (息稅前利潤)：** 收入 \* 營運利潤率。
    *   假設營運利潤率從 2026 年的 32% 逐步提升至 2030 年的 35%。
    *   2026 EBIT: 167.52 \* 0.32 = 53.61 億美元
    *   2027 EBIT: 201.02 \* 0.33 = 66.34 億美元
    *   2028 EBIT: 241.22 \* 0.34 = 82.01 億美元
    *   2029 EBIT: 284.64 \* 0.345 = 98.20 億美元
    *   2030 EBIT: 335.88 \* 0.35 = 117.56 億美元

3.  **計算稅後 EBIT (NOPAT)：** EBIT \* (1 - 稅率)。
    *   稅率 26.15%。
    *   2026 NOPAT: 53.61 \* (1 - 0.2615) = 39.59 億美元
    *   2027 NOPAT: 66.34 \* (1 - 0.2615) = 49.00 億美元
    *   2028 NOPAT: 82.01 \* (1 - 0.2615) = 60.56 億美元
    *   2029 NOPAT: 98.20 \* (1 - 0.2615) = 72.54 億美元
    *   2030 NOPAT: 117.56 \* (1 - 0.2615) = 86.83 億美元

4.  **計算自由現金流 (FCFF)：** NOPAT + D&A - CAPEX - 營運資本變動。
    *   D&A = 收入 \* 2%
    *   CAPEX = 收入 \* 3%
    *   營運資本變動 = (當期收入 - 前期收入) \* 5%
    *   2026 FCFF: 39.59 + (167.52\*0.02) - (167.52\*0.03) - ((167.52-139.6)\*0.05) = 39.59 + 3.35 - 5.03 - 1.396 = 36.51 億美元
    *   2027 FCFF: 49.00 + (201.02\*0.02) - (201.02\*0.03) - ((201.02-167.52)\*0.05) = 49.00 + 4.02 - 6.03 - 1.675 = 45.32 億美元
    *   2028 FCFF: 60.56 + (241.22\*0.02) - (241.22\*0.03) - ((241.22-201.02)\*0.05) = 60.56 + 4.82 - 7.24 - 2.01 = 56.13 億美元
    *   2029 FCFF: 72.54 + (284.64\*0.02) - (284.64\*0.03) - ((284.64-241.22)\*0.05) = 72.54 + 5.69 - 8.54 - 2.17 = 67.52 億美元
    *   2030 FCFF: 86.83 + (335.88\*0.02) - (335.88\*0.03) - ((335.88-284.64)\*0.05) = 86.83 + 6.72 - 10.08 - 2.56 = 80.91 億美元

5.  **計算永續價值 (Terminal Value, TV)：**
    *   TV = (2030 NOPAT \* (1 + 永續增長率)) / (WACC - 永續增長率)
    *   TV = (86.83 \* (1 + 0.03)) / (0.083 - 0.03) = (86.83 \* 1.03) / 0.053 = 89.43 / 0.053 = 1687.36 億美元

6.  **折現所有現金流和永續價值：**
    *   折現因子 = 1 / (1 + WACC)^n
    *   2026 FCFF 現值: 36.51 / (1.083)^1 = 33.71 億美元
    *   2027 FCFF 現值: 45.32 / (1.083)^2 = 38.61 億美元
    *   2028 FCFF 現值: 56.13 / (1.083)^3 = 44.29 億美元
    *   2029 FCFF 現值: 67.52 / (1.083)^4 = 49.40 億美元
    *   2030 FCFF 現值: 80.91 / (1.083)^5 = 55.00 億美元
    *   永續價值現值: 1687.36 / (1.083)^5 = 1147.00 億美元

7.  **計算企業價值 (Enterprise Value, EV)：** 所有現值之和 = 33.71 + 38.61 + 44.29 + 49.40 + 55.00 + 1147.00 = 1368.01 億美元。

8.  **計算股權價值 (Equity Value)：** EV + 現金及約當現金 - 總債務。
    *   ServiceNow 截至 2026 年 3 月 31 日的現金及約當現金為 51.8 億美元，總債務為 24.3 億美元。
    *   股權價值 = 1368.01 + 51.8 - 24.3 = 1395.51 億美元。

9.  **計算每股估值：** 股權價值 / 流通股數。
    *   市值約 918.7 億美元，股價 89.05 美元，則流通股數約為 918.7 億 / 89.05 = 10.316 億股。
    *   每股估值 = 1395.51 億美元 / 10.316 億股 = **135.28 美元**。

**DCF 估值結果與合理價位區間：**
根據 DCF 模型，ServiceNow 的每股估值約為 **135.28 美元**。
考慮到假設的敏感性，合理價位區間可能在 **120 美元至 150 美元** 之間。
