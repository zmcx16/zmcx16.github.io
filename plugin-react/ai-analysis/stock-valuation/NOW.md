**DCF 計算步驟：**

1.  **預測未來 5 年的收入。**
2.  **計算預測期內的 EBIT (稅前息前利潤)。**
3.  **計算稅後 EBIT (NOPAT)。**
4.  **計算自由現金流 (FCFF) = NOPAT + 折舊與攤銷 - 資本支出 - 營運資本變動。** (由於我們沒有詳細的折舊與攤銷數據，我們將使用簡化的方法，或假設其與 CapEx 相抵消，並專注於營業利潤率和營運資本變動對現金流的影響)。更精確的做法是從財報中獲取。

5.  **計算終值 (Terminal Value, TV)：** TV = (FCFF_Year5 \* (1 + g)) / (WACC - g)，其中 g 為永續增長率。
6.  **將預測期內的 FCFF 和終值折現至現值。**
7.  **加上現金，減去債務，得到股權價值，再除以流通股數得到每股估值。**
