# Air Products and Chemicals, Inc. (APD) 深度估值分析報告

作為資深投資分析師，針對 APD（Air Products and Chemicals）目前的財務狀況、產業地位及未來成長動能，我進行了全面的估值建模。APD 目前正處於從傳統工業氣體向藍氫/綠氫能源轉型的關鍵期，且近期面臨激進投資者（如 Mantle Ridge）的壓力，這對其估值邏輯產生了顯著影響。

#### 估值地位分析
- **溢價與折價**：相對於行業龍頭 Linde，APD 在 **Forward P/E** 上呈現明顯折價（20.2x vs 28.5x）。這反映了市場對 APD 龐大資本支出（Capex）計畫（如 NEOM 綠氫項目）執行風險的擔憂。
- **合理性評估**：APD 的營收增長率（8.77%）優於同業，且營業利潤率維持在高水準。目前的折價主要源於現金流壓力（P/C 達 67.62），若激進投資者能推動資本配置優化，其估值倍數有向 Linde 靠攏的潛力。

### 2. 折現現金流模型 (DCF)

APD 的價值核心在於其長期合約（Take-or-pay）帶來的穩定性，以及氫能項目的遠期現金流。

#### 預測假設
1.  **預測期間**：2025 - 2029 (5年)。
2.  **營收增長率**：預計前 3 年為 6-8%，隨後因大型氫能項目投產，第 4-5 年回升至 9%。
3.  **EBITDA Margin**：維持在 38% - 40% 區間，考慮到能源成本轉嫁機制。
4.  **資本支出 (Capex)**：假設 2025-2026 仍維持高位（約 50-55 億美元/年），隨後逐步下降。
5.  **稅率**：20%。

#### 折現率與終值
- **WACC 計算**：
  - 無風險利率 (10Y Treasury): 4.4%
  - Beta: 0.88
  - 股權風險溢價 (ERP): 5.0%
  - 稅後債務成本: 3.6% (考慮其 A 級信用評等)
  - **WACC 最終取值：8.2%**
- **終值增長率 (g)**：2.5% (符合工業氣體長期通膨調整特性)。

#### 估值結果
- **企業價值 (EV)**：約 $885 億美元。
- **扣除淨負債**：約 $145 億美元。
- **股權價值**：$740 億美元。
- **每股內在價值**：**$332.50**。

### 3. 多維度估值法

#### 相對估值法 (Relative Valuation)
參考同業 Forward P/E 均值 24x，並考量 APD 的執行風險給予 10% 折價，適用倍數為 **21.5x**。
- 預期 2025 EPS: $12.85 (基於財報指引與分析師預測)
- 推算目標價：$12.85 * 21.5 = **$276.28** (保守) / 若給予 Linde 同等倍數 (28x) 則為 **$359.80**。

#### 股息折現模型 (DDM)
APD 已連續 42 年增加股息，屬於股息貴族。
- 當前股息 (D0): $7.08
- 預期股息增長率 (g): 6% (過去五年平均)
- 股權成本 (Ke): 8.5%
- **DDM 價值 = $7.08 * (1.06) / (0.085 - 0.06) = $300.19**。

**最終結論：**
1. **合理價位區間**：**$305.00 - $335.00**
2. **目前股價位階**：**合理偏低 (Slightly Undervalued)**。目前股價 $289.19 處於合理區間下緣，提供了約 10-15% 的安全邊際。
3. **投資核心論點**：APD 的估值目前受到高 Capex 導致的自由現金流暫時性受壓所抑制。然而，其 24% 的營業利潤率與 1.41 的 EPS Q/Q 增長顯示核心業務極其強韌。激進投資者的介入可能成為催化劑，迫使管理層優化資產負債表或分拆非核心業務。主要風險在於大型氫能項目的延遲與地緣政治對能源成本的衝擊，但其股息貴族地位為股價提供了堅實底部。
