### **估值方法與計算**

#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值，以得出其內在價值。

**估值假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年
*   **永續增長率 (Terminal Growth Rate, g):** 3.5% (略高於長期通膨和全球 GDP 增長，反映其創新能力)
*   **折現率 (WACC - Weighted Average Cost of Capital):**
    *   **無風險利率 (Risk-Free Rate, Rf):** 4.25% (參考美國 10 年期國債收益率)
    *   **市場風險溢酬 (Market Risk Premium, MRP):** 6.0% (行業標準)
    *   **Beta (β):** 1.05 (參考最新數據，Alphabet 規模龐大，波動性接近市場)
    *   **股權成本 (Cost of Equity, Ke) = Rf + β * MRP**
        *   Ke = 4.25% + 1.05 * 6.0% = 4.25% + 6.30% = 10.55%
    *   **稅率 (Tax Rate):** 17% (參考公司近年有效稅率)
    *   **債務成本 (Cost of Debt, Kd):** 5.0% (Alphabet 信用評級高，債務成本低)
    *   **稅後債務成本 (After-tax Kd) = Kd * (1 - Tax Rate)**
        *   After-tax Kd = 5.0% * (1 - 0.17) = 4.15%
    *   **資本結構 (Capital Structure):**
        *   市值 (Equity Value): $2.2 兆美元
        *   總債務 (Total Debt): 約 $140 億美元 (參考最新財報)
        *   股權權重 (We) = $2.2T / ($2.2T + $14B) ≈ 0.9936
        *   債務權重 (Wd) = $14B / ($2.2T + $14B) ≈ 0.0064
    *   **WACC = (We * Ke) + (Wd * After-tax Kd)**
        *   WACC = (0.9936 * 10.55%) + (0.0064 * 4.15%) = 10.48% + 0.026% = 10.51% (約 10.5%)

*   **自由現金流 (Free Cash Flow, FCF) 預測：**
    *   **基期 FCF (TTM, 截至 2024 年 Q1):** 約 $800 億美元 (參考最新財報)
    *   **FCF 增長率 (預測期內，反映 AI 和 Cloud 帶來的增長，但考慮規模效應和競爭)：**
        *   Year 1: 18%
        *   Year 2: 16%
        *   Year 3: 14%
        *   Year 4: 12%
        *   Year 5: 10%

**計算過程：**

1.  **預測期內 FCF：**
    *   Year 0 (TTM): $80.00 B
    *   Year 1: $80.00 B * (1 + 0.18) = $94.40 B
    *   Year 2: $94.40 B * (1 + 0.16) = $109.40 B
    *   Year 3: $109.40 B * (1 + 0.14) = $124.72 B
    *   Year 4: $124.72 B * (1 + 0.12) = $139.69 B
    *   Year 5: $139.69 B * (1 + 0.10) = $153.66 B

2.  **計算終端價值 (Terminal Value, TV)：**
    *   TV = FCF_Year5 * (1 + g) / (WACC - g)
    *   TV = $153.66 B * (1 + 0.035) / (0.105 - 0.035)
    *   TV = $153.66 B * 1.035 / 0.070 = $159.04 B / 0.070 = $2272.00 B

3.  **折現 FCF 和 TV：**
    *   PV(FCF1) = $94.40 B / (1 + 0.105)^1 = $85.43 B
    *   PV(FCF2) = $109.40 B / (1 + 0.105)^2 = $89.40 B
    *   PV(FCF3) = $124.72 B / (1 + 0.105)^3 = $92.80 B
    *   PV(FCF4) = $139.69 B / (1 + 0.105)^4 = $95.30 B
    *   PV(FCF5) = $153.66 B / (1 + 0.105)^5 = $96.80 B
    *   PV(TV) = $2272.00 B / (1 + 0.105)^5 = $1433.00 B

4.  **計算企業價值 (Enterprise Value, EV)：**
    *   EV = Sum(PV FCFs) + PV(TV)
    *   EV = $85.43 B + $89.40 B + $92.80 B + $95.30 B + $96.80 B + $1433.00 B = $1892.73 B

5.  **計算股權價值 (Equity Value)：**
    *   Equity Value = EV + 現金及約當現金 - 總債務
    *   現金及約當現金 (Cash & Equivalents): 約 $1080 億美元 (參考最新財報)
    *   Equity Value = $1892.73 B + $108.00 B - $14.00 B = $1986.73 B

6.  **計算每股內在價值 (Intrinsic Value Per Share)：**
    *   流通股數 (Shares Outstanding): 約 125 億股 (GOOGL + GOOG 合計)
    *   Value Per Share = Equity Value / Shares Outstanding
    *   Value Per Share = $1986.73 B / 12.5 B = $158.94

**DCF 估值結果與合理價位區間：**
*   **DCF 估值：** $158.94
*   **合理價位區間：** 考慮到增長率和 WACC 的敏感性，合理區間約為 **$150 - $175**。

#### **4. 其他合理估值方法 (PEG Ratio)**

PEG Ratio (Price/Earnings to Growth Ratio) 結合了 P/E 倍數和預期盈利增長率，通常認為 PEG 為 1.0 表示合理估值。

*   **GOOGL 當前 Forward P/E:** 23.0x
*   **分析師預期未來 5 年 EPS 增長率 (CAGR):** 約 15% - 20% (取中位數 17.5%)
    *   *注意：提供的 `EPS next Y_%` 0.0576 (5.76%) 似乎是針對單一年度或季度，而非長期增長率，因此採用市場共識的長期增長率。*

**計算 PEG：**
*   PEG = Forward P/E / 預期增長率 (%)
*   PEG = 23.0 / 17.5 = 1.31

**估值結果：**
*   GOOGL 的 PEG 為 1.31，略高於 1.0，但對於一家具有強大護城河和 AI 增長潛力的科技巨頭來說，這通常被認為是合理的。
