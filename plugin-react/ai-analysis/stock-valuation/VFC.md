*   **折舊與攤銷 (Depreciation & Amortization, D&A)**: 假設為營收的 1.5%。
*   **營運資本變動 (Change in Working Capital)**: 假設為營收增長的 10%。

**DCF 計算步驟:**

1.  **預測營收 (Revenue)**
2.  **預測營業利潤 (EBIT)** = 營收 * 營業利潤率
3.  **預測稅後營業利潤 (NOPAT)** = EBIT * (1 - 稅率)
4.  **預測自由現金流 (FCF)** = NOPAT + D&A - CapEx - 營運資本變動
5.  **計算終端價值 (Terminal Value, TV)** = FCF (2031) * (1 + 終端增長率) / (WACC - 終端增長率)
6.  **折現 FCF 和 TV** 至現值
7.  **計算企業價值 (Enterprise Value, EV)** = 預測期 FCF 現值總和 + TV 現值
8.  **計算股權價值 (Equity Value)** = EV - 淨債務 (Net Debt) + 現金
9.  **計算每股價值 (Intrinsic Value per Share)** = 股權價值 / 流通股數
