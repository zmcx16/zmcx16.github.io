### 估值方法與結果

#### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

**估值假設：**
*   **當前股價:** $18.16
*   **加權平均資本成本 (WACC):** 9.5% (參考 ValueInvesting.io 提供的8.5% 和 GuruFocus 提供的9.89%，並考慮公司債務水平取中間值)
*   **預測期間:** 5年 (顯性預測期)
*   **永續增長率 (Terminal Growth Rate):** 2.0% (保守估計，低於長期GDP增長，反映行業成熟及公司轉型挑戰)
*   **流通股數:** 約3.91億股 (市值 $7.10B / 股價 $18.16)
*   **現金及約當現金:** $4.1912億美元
*   **總債務:** $46.3億美元
*   **2025財年自由現金流 (FCF):** $4.4億美元 (基於公司指引)
*   **顯性預測期 FCF 增長率:**
    *   2026財年 (Year 1): 10% (預期營運改善及債務削減效益)
    *   2027財年 (Year 2): 8%
    *   2028財年 (Year 3): 5%
    *   2029財年 (Year 4): 3%
    *   2030財年 (Year 5): 2.5% (趨近永續增長率)

**計算過程：**
1.  **預測未來自由現金流 (FCF):**
    *   FY2026: $440M * (1 + 0.10) = $484.0M
    *   FY2027: $484.0M * (1 + 0.08) = $522.7M
    *   FY2028: $522.7M * (1 + 0.05) = $548.8M
    *   FY2029: $548.8M * (1 + 0.03) = $565.3M
    *   FY2030: $565.3M * (1 + 0.025) = $579.4M
2.  **計算各期 FCF 的現值 (PV of FCF):**
    *   使用 WACC = 9.5% 折現，各期 FCF 現值總和約為 $20.568億美元。
3.  **計算終端價值 (Terminal Value, TV):**
    *   TV = FCF(Year 6) / (WACC - 永續增長率)
    *   FCF(Year 6) = $579.4M * (1 + 0.02) = $591.0M
    *   TV = $591.0M / (0.095 - 0.02) = $78.80億美元
4.  **計算終端價值的現值 (PV of TV):**
    *   PV(TV) = $78.80億美元 / (1 + 0.095)^5 = $50.05億美元
5.  **計算企業價值 (Enterprise Value, EV):**
    *   EV = PV of FCFs 總和 + PV of TV = $20.568億美元 + $50.05億美元 = $70.618億美元
6.  **計算股權價值 (Equity Value):**
    *   股權價值 = EV + 現金 - 總債務 = $70.618億美元 + $4.1912億美元 - $46.3億美元 = $28.5092億美元
7.  **計算每股內在價值:**
    *   每股內在價值 = $28.5092億美元 / 3.91億股 = **$7.29**

**DCF 估值結果:** 每股約 **$7.29**。
**合理價位區間 (敏感性分析):** 考慮 WACC 介於9.0%至10.0%，永續增長率介於1.5%至2.5%，DCF 估值範圍約為 **$4.80 至 $10.60**。

#### 3. 股息折現模型 (Dividend Discount Model, DDM)

**適用性分析:** VFC 近年股息大幅削減，且派息比率高達155.43%，表明當前股息不可持續。因此，傳統的戈登增長模型 (Gordon Growth Model) 不適用。我們採用兩階段 DDM，假設股息在短期內穩定，然後以極低速度增長。

**估值假設：**
*   **當前年度股息 (D0):** $0.36 (2025年)
*   **股權成本 (Required Rate of Return):** 11.1% (參考 WACC 計算中的股權成本)
*   **股息增長率:**
    *   第一階段 (2026-2028財年): 0% (假設股息穩定不變)
    *   第二階段 (2029財年及以後): 1.0% (極度保守的永續增長率)

**計算過程：**
1.  **預測顯性期股息:**
    *   D1 (FY2026): $0.36
    *   D2 (FY2027): $0.36
    *   D3 (FY2028): $0.36
2.  **計算各期股息現值:**
    *   使用股權成本 11.1% 折現，各期股息現值總和約為 $0.878。
3.  **計算終端股息價值 (Terminal Value of Dividends) (2028財年末):**
    *   D4 = D3 * (1 + 0.01) = $0.36 * 1.01 = $0.3636
    *   TV_Div = D4 / (股權成本 - 永續增長率) = $0.3636 / (0.111 - 0.01) = $3.60
4.  **計算終端股息價值的現值:**
    *   PV(TV_Div) = $3.60 / (1 + 0.111)^3 = $2.625
5.  **計算每股內在價值:**
    *   每股內在價值 = 顯性期股息現值總和 + 終端股息價值現值 = $0.878 + $2.625 = **$3.50**

**DDM 估值結果:** 每股約 **$3.50**。

### 綜合結論

**估值結果匯總：**
*   **當前市場價格:** $18.16
*   **折現現金流模型 (DCF):** 約 $7.29 (區間: $4.80 - $10.60)
*   **相對估值 (可比公司分析):**
    *   P/E (調整後 TTM): $3.86
    *   P/E (預期): $5.69
    *   EV/EBITDA: $14.50
    *   P/FCF: $12.51
*   **股息折現模型 (DDM):** 約 $3.50
*   **EV/Sales (額外參考):** 約 $38.67 (被視為異常值)

**分析與結論：**
綜合來看，VF Corporation (VFC) 目前的市場價格 $18.16 似乎被**高估**。

1.  **內在價值模型 (DCF 和 DDM):** 這兩種模型都給出了遠低於當前市場價格的估值。DCF 估值約為 $7.29，主要受高額債務和保守的現金流增長假設影響。DDM 估值最低，僅為 $3.50，這強烈反映了公司近期股息大幅削減以及當前股息支付不可持續的問題，表明股息對公司內在價值的貢獻已大幅縮水。
2.  **相對估值模型:**
    *   基於當前和預期盈利的 P/E 倍數估值非常低（$3.86 - $5.69），這凸顯了 VFC 當前盈利能力極弱的問題。
    *   EV/EBITDA ($14.50) 和 P/FCF ($12.51) 這些更側重於企業整體價值和現金流的倍數，給出的估值雖然高於 P/E 和 DDM，但仍低於當前股價。這些倍數在公司轉型期，盈利波動較大時，通常被認為更具參考價值。
3.  **異常值:** P/S 和 EV/Sales 倍數給出的估值顯著偏高，這可能是因為 VFC 雖然營收規模較大，但利潤率和增長前景不如部分可比公司，導致基於營收的倍數未能充分反映其盈利困境。

**綜合判斷:**
VFC 正在進行轉型，並通過出售資產來削減債務，部分核心品牌表現良好。然而，公司仍面臨高負債、營收增長乏力以及盈利能力不穩定的挑戰。其 ROIC 低於 WACC，表明公司目前仍在破壞價值。分析師的平均目標價也暗示了當前股價存在下跌風險。

考慮到 DCF、EV/EBITDA 和 P/FCF 等較為可靠的估值方法，VFC 的合理內在價值區間應在 **$7.00 至 $15.00** 之間。當前 $18.16 的市場價格超出了這個估值範圍。

**最終建議:**
基於本次估值分析，VFC 的股票目前被高估。投資者應謹慎對待，並充分考慮公司高負債、轉型中的不確定性以及盈利能力恢復的挑戰。市場可能已經過於樂觀地預期了公司的復甦速度。
