### 2. 估值模型計算

#### A. 股息折現模型 (Dividend Discount Model, DDM)
銀行業屬於成熟產業，現金流穩定且有定期派息，DDM 是極具參考價值的模型。

*   **假設：**
    *   **當前股息 (D0)：** $1.04 (年化)
    *   **預期股息增長率 (g)：** 6% (參考過去五年平均及未來資本釋放潛力)
    *   **股權成本 (Ke)：** 9.5% (計算方式：無風險利率 4.3% + Beta 1.35 * 市場風險溢價 4%)
*   **計算：**
    *   $P = D1 / (Ke - g) = (1.04 * 1.06) / (0.095 - 0.06) = 1.1024 / 0.035 = 31.49$
    *   *註：若使用兩階段模型（前 5 年 8% 增長，後續 4% 永續增長），估值約為 **$42 - $45**。*

#### B. 折現現金流模型 (DCF - 採用超額收益法 Excess Return Model)
對於銀行，傳統 FCF 較難定義，通常使用「超額收益法」。

*   **假設：**
    *   **預測期：** 5 年
    *   **折現率 (Cost of Equity)：** 9.5%
    *   **長期增長率：** 3%
    *   **預期 ROE：** 12% (假設隨利率穩定與效率提升，ROE 從目前的 10.2% 回升)
    *   **當前每股淨資產 (BVPS)：** 約 $38.5 (由 P/B 1.38 與股價 53.2 推算)
*   **計算過程：**
    *   超額收益 = (ROE - Ke) * BVPS
    *   估值 = 當前 BVPS + 未來超額收益的現值
    *   **結果：** 估值約為 **$55.80**。

#### C. 相對估值法 (Comparable Company Analysis)
參考同業（JPM, WFC, C）的估值倍數。

| 公司 | Forward P/E | P/B | ROE |
| :--- | :--- | :--- | :--- |
| **BAC (美銀)** | **10.73** | **1.38** | **10.2%** |
| JPM (摩根大通) | 12.5 | 1.8 | 16.5% |
| WFC (富國銀行) | 11.2 | 1.2 | 10.5% |
| C (花旗集團) | 9.8 | 0.6 | 4.5% |

*   **分析：** BAC 的 Forward P/E (10.73) 低於 JPM 與 WFC，且 PEG 僅 0.75，顯示相對於其 EPS 增長預期（14.7%），股價並未過熱。
*   **合理倍數推估：** 給予 BAC 12 倍的 Forward P/E。
    *   $12 * 預期 EPS (約 4.95) = **$59.40**$

#### D. P/B - ROE 估值模型 (Justified P/B)
*   **公式：** $Fair P/B = (ROE - g) / (Ke - g)$
*   **計算：** $(0.12 - 0.03) / (0.095 - 0.03) = 0.09 / 0.065 = 1.38$
*   **合理價：** $1.38 * 38.5 = **$53.13**$ (與目前市價高度吻合)

### 4. 綜合結論與投資建議

**1. 估值定性：合理偏低 (Fairly Valued to Slightly Undervalued)**
目前 BAC 的股價 ($53.20) 非常接近其基本面支撐位。雖然 DDM 顯示價格稍高，但考慮到 BAC 強大的股票回購能力（未包含在簡單 DDM 中）以及 PEG < 1 的成長性，目前的價格具有吸引力。

**2. 關鍵支撐與壓力：**
*   **合理區間：$52.00 - $58.00**
*   **分析師目標價：$61.64** (顯示市場仍有約 15% 的上行預期)。

**3. 投資風險：**
*   **宏觀風險：** 若美國經濟陷入硬著陸，貸款違約撥備將侵蝕利潤。
*   **利率風險：** 降息過快會壓縮淨利差（NIM）。

**4. 總結：**
BAC 是一家管理穩健、資產負債表強大的銀行。對於尋求穩定股息與適度資本增長的投資者，**$53 以下是理想的切入點**。目前的 Forward P/E 僅 10.7 倍，相較於標普 500 指數顯著折價，具備良好的防禦性。
