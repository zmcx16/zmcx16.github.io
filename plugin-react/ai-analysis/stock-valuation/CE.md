### 2. 估值方法應用

鑑於公司目前處於轉型期，過去一年有GAAP虧損但調整後盈利，且預計未來盈利將大幅增長，我們將主要採用折現現金流模型 (DCF) 和相對估值法。由於股息收益率較低 (0.26%) 且公司目前更側重於去槓桿化，股息折現模型 (DDM) 可能不適用或結果不夠穩定。

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間：** 5年 (2026年 - 2030年)
*   **折現率 (WACC)：** 我們採用7.0%作為WACC的估計值，介於搜索結果中的6.75%和7.5%之間。
    *   **計算WACC的簡要說明：**
        *   **股權成本 (Cost of Equity, Ke)：** 根據CAPM模型，Ke = 無風險利率 + Beta * 股權風險溢價。
            *   無風險利率：參考美國10年期國債收益率，約4.15% (截至2026年1月25日，ValueInvesting.io數據)。
            *   Beta：Celanese的Beta約為1.12 - 1.99。我們取中間值1.5。
            *   股權風險溢價 (ERP)：通常在5% - 6%之間，我們取5.5%。
            *   Ke = 4.15% + 1.5 * 5.5% = 4.15% + 8.25% = 12.40%
        *   **債務成本 (Cost of Debt, Kd)：** 參考Celanese債券的到期收益率，例如7.06% (2034年到期債券)。我們取平均值約6.5%。
        *   **稅率 (Tax Rate)：** 由於公司近期有虧損，但長期來看，我們使用其調整後有效稅率，或行業平均稅率。根據2025年Q3調整後稅率為8%。 考慮到未來盈利恢復，我們保守估計為20%。
        *   **債務權益比 (D/E)：** 3.34 (來自您提供數據和最新搜索結果)。
        *   **WACC = (E/(D+E)) * Ke + (D/(D+E)) * Kd * (1 - Tax Rate)**
            *   E/(D+E) = 1 / (3.34 + 1) = 1 / 4.34 = 0.23
            *   D/(D+E) = 3.34 / 4.34 = 0.77
            *   WACC = 0.23 * 12.40% + 0.77 * 6.5% * (1 - 20%) = 2.85% + 0.77 * 6.5% * 0.8 = 2.85% + 4.00% = 6.85%。
            *   我們將使用7.0%作為WACC。
*   **長期增長率 (Terminal Growth Rate)：** 考慮到化學產業的成熟性，我們假設長期增長率為2.0%。
*   **自由現金流 (Free Cash Flow, FCF) 預測：**
    *   2025年FCF預計為7億 - 8億美元。我們取中值7.5億美元作為基期。
    *   **增長率假設：**
        *   2026年：考慮到2025年Q4的疲軟和2026年Q1的挑戰，但預計Q2改善，以及成本削減和Micromax剝離的影響，我們預計FCF增長率為10%。
        *   2027年：隨著市場復甦和公司戰略效益顯現，預計增長率為8%。
        *   2028年：增長率為5%。
        *   2029年：增長率為3%。
        *   2030年：增長率為2.5%。

**DCF 計算步驟：**

1.  **預測自由現金流 (FCF)：**
    *   2025年 (基期): 7.5億美元
    *   2026年: 7.5億 * (1 + 10%) = 8.25億美元
    *   2027年: 8.25億 * (1 + 8%) = 8.91億美元
    *   2028年: 8.91億 * (1 + 5%) = 9.3555億美元
    *   2029年: 9.3555億 * (1 + 3%) = 9.6362億美元
    *   2030年: 9.6362億 * (1 + 2.5%) = 9.8771億美元

2.  **計算終值 (Terminal Value, TV)：**
    *   TV = FCF (2031年) / (WACC - 長期增長率)
    *   FCF (2031年) = FCF (2030年) * (1 + 長期增長率) = 9.8771億 * (1 + 2.0%) = 10.0746億美元
    *   TV = 10.0746億美元 / (7.0% - 2.0%) = 10.0746億美元 / 0.05 = 201.492億美元

3.  **計算各期FCF和TV的現值：**
    *   PV (2026 FCF) = 8.25億 / (1 + 0.07)^1 = 7.71億美元
    *   PV (2027 FCF) = 8.91億 / (1 + 0.07)^2 = 7.79億美元
    *   PV (2028 FCF) = 9.3555億 / (1 + 0.07)^3 = 7.64億美元
    *   PV (2029 FCF) = 9.6362億 / (1 + 0.07)^4 = 7.34億美元
    *   PV (2030 FCF) = 9.8771億 / (1 + 0.07)^5 = 7.04億美元
    *   PV (TV) = 201.492億 / (1 + 0.07)^5 = 143.50億美元

4.  **計算企業價值 (Enterprise Value, EV)：**
    *   EV = 所有FCF現值之和 + TV現值
    *   EV = 7.71 + 7.79 + 7.64 + 7.34 + 7.04 + 143.50 = 171.02億美元

5.  **計算股權價值 (Equity Value)：**
    *   股權價值 = EV + 現金及約當現金 - 總債務
    *   根據最新數據，現金及約當現金約為11.7億美元，總債務約為133.1億美元。
    *   股權價值 = 171.02億 + 11.7億 - 133.1億 = 49.62億美元

6.  **計算每股估值：**
    *   流通股數：約1.095億股
    *   每股估值 = 49.62億美元 / 1.095億股 = **45.31美元/股**

**DCF模型估值結果：**

*   **合理價位區間：** 考慮到假設的敏感性，我們設定一個區間。
    *   如果WACC為6.5%，長期增長率為2.5%，則估值會更高。
    *   如果WACC為7.5%，長期增長率為1.5%，則估值會更低。
    *   基於我們的核心假設，DCF模型給出的每股估值約為 **45.31美元**。
    *   合理價位區間：**42.00美元 - 49.00美元**

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

Celanese的股息收益率為0.26%，相對較低。雖然公司支付股息，但其主要戰略是去槓桿化和成本削減，而非高股息增長。因此，DDM可能不是最適合Celanese的估值方法，但我們仍可進行簡要分析。

**假設：**

*   **當前每股股息 (D0)：** 股價45.66美元 * 0.26% = 0.1187美元 (年化)
*   **股息增長率 (g)：** 考慮到公司目前的去槓桿化重點和產業成熟度，我們假設一個較低的長期股息增長率，例如2%。
*   **股權成本 (Ke)：** 沿用DCF模型中的12.40%。

**DDM 計算：**

*   DDM = D0 * (1 + g) / (Ke - g)
*   DDM = 0.1187 * (1 + 0.02) / (0.1240 - 0.02)
*   DDM = 0.1187 * 1.02 / 0.1040
*   DDM = 0.121074 / 0.1040 = **1.16美元/股**

**DDM模型估值結果：**

DDM模型給出的估值僅為 **1.16美元/股**，遠低於當前股價和其他估值方法。這表明Celanese的價值主要來自其未來的盈利增長和自由現金流，而非當前的股息支付能力。因此，DDM在此案例中不適用於反映公司的真實價值。
