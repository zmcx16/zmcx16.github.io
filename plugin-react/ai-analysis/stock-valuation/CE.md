### **估值模型應用**

#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF模型將預測公司未來自由現金流，並將其折現至現值，以得出公司內在價值。

**估值假設：**

*   **預測期間：** 5年 (2024-2028)
*   **長期增長率 (Terminal Growth Rate, g)：** 2.0% (保守估計，略低於長期GDP增長，適合成熟的週期性行業)
*   **稅率 (Tax Rate)：** 21% (美國企業稅率，用於計算NOPAT)
*   **股份數量 (Shares Outstanding)：** 市值 / 當前股價 = $4.62B / $42.85 = 約 107.8 百萬股。
*   **淨債務 (Net Debt)：** 根據近期財報和市場數據，Celanese的總債務約為$12.5B，現金約$1.5B，因此淨債務約為 **$11.0B**。

**加權平均資本成本 (WACC) 計算：**

*   **無風險利率 (Rf)：** 4.2% (參考美國10年期國債收益率)
*   **市場風險溢價 (MRP)：** 5.5% (行業標準假設)
*   **Beta (β)：** 1.5 (參考金融數據提供商，反映公司股價相對於市場的波動性)
*   **股權成本 (Ke) = Rf + β * MRP**
    *   Ke = 4.2% + 1.5 * 5.5% = 4.2% + 8.25% = **12.45%**
*   **債務成本 (Kd)：** 6.5% (考慮公司高債務水平和當前利率環境的估計)
*   **稅後債務成本 = Kd * (1 - Tax Rate)**
    *   稅後債務成本 = 6.5% * (1 - 0.21) = 6.5% * 0.79 = **5.135%**
*   **股權市值 (E)：** $4.62B (Market Cap)
*   **債務市值 (D)：** $11.0B (Net Debt)
*   **總資本 (V) = E + D** = $4.62B + $11.0B = $15.62B
*   **股權權重 (We) = E / V** = $4.62B / $15.62B = 0.2958
*   **債務權重 (Wd) = D / V** = $11.0B / $15.62B = 0.7042
*   **WACC = (We * Ke) + (Wd * 稅後債務成本)**
    *   WACC = (0.2958 * 12.45%) + (0.7042 * 5.135%) = 3.68% + 3.62% = **7.30%**

**自由現金流 (FCF) 預測：**

*   **TTM 營收 (2023A):** 約 $10.9B (根據近期財報估計)
*   **營收增長率：** 2024E: -2% (受宏觀經濟和整合影響)，2025E: 3%，2026E: 4%，2027E: 3%，2028E: 2.5% (反映復甦和穩定增長)
*   **營業利潤率 (EBIT Margin)：** 2023A: 8.82% (提供數據)。預計隨整合和效率提升而改善。

*   **資本支出 (CapEx)：** 預計為營收的5.5% (根據歷史數據和行業特點)。
*   **營運資本變動 (Change in NWC)：** 預計為營收變化的1%。

**企業價值 (Enterprise Value) 與股權價值 (Equity Value)：**

*   **企業價值 (EV) = 預測期FCF現值總和 + 終值現值**

    *   股權價值 = $15,216M - $11,000M = **$4,216M**
*   **每股內在價值 = 股權價值 / 股份數量**
