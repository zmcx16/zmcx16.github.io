### 2. 估值假設

在進行估值分析之前，我們需要明確以下關鍵假設：

**A. 加權平均資本成本 (WACC) / 折現率**

我們將根據以下數據計算 WACC：

*   **無風險利率 (Risk-Free Rate):** 3.95% (基於政府債券收益率)
*   **市場風險溢價 (Market Risk Premium, ERP):** 4.18%
*   **貝塔值 (Beta):** 1.57
*   **股權成本 (Cost of Equity, Ke):**
    *   Ke = 無風險利率 + 貝塔值 × 市場風險溢價
    *   Ke = 3.95% + 1.57 × 4.18% = 3.95% + 6.56% = **10.51%**
*   **債務成本 (Cost of Debt, Kd):**
    *   2025財年利息支出 (Interest Expense): $4.906 億美元
    *   2025財年總債務 (Total Debt): $57.4 億美元
    *   Kd = 利息支出 / 總債務 = $490.6M / $5.74B = **8.55%**
*   **稅率 (Tax Rate):** 26.88% (2025財年有效稅率)
*   **稅後債務成本 (After-tax Kd):** 8.55% × (1 - 0.2688) = 8.55% × 0.7312 = **6.25%**
*   **股權市值 (Market Value of Equity, E):** $3.12 億美元 (市值)
*   **債務市值 (Market Value of Debt, D):** $57.4 億美元 (總債務)
*   **總資本 (Total Capital, V):** E + D = $3.12B + $5.74B = $8.86 億美元
*   **股權權重 (Weight of Equity, We):** E / V = $3.12B / $8.86B = 0.3521
*   **債務權重 (Weight of Debt, Wd):** D / V = $5.74B / $8.86B = 0.6479
*   **WACC 計算:** (We × Ke) + (Wd × 稅後 Kd)
    *   WACC = (0.3521 × 10.51%) + (0.6479 × 6.25%) = 3.70% + 4.05% = **7.75%**

我們將使用 **7.75%** 作為 DCF 模型中的折現率。

**B. 預測期間與長期增長率**

*   **預測期間 (Explicit Forecast Period):** 5年 (2026年 - 2030年)。
*   **長期增長率 (Terminal Growth Rate):** 考慮到航運業的周期性以及 ZIM 的特定戰略，我們將採用保守的 **0.5%** 作為永續增長率。

### 3. 估值模型與結果

#### 3.1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值步驟與假設：**

1.  **營收預測 (Revenue Projections):**
    *   2025財年營收: $69.0 億美元
    *   考慮到2026-2027年預計的負增長率 (-13.47%) 以及航運業的周期性，我們將採用保守的營收增長率。
    *   2026年: -10% (基於預測的負增長和市場波動)
    *   2027年: -5%

    *   2030年: 2%

    *   2025財年調整後EBIT利潤率為13%。我們將假設在預測期內，公司通過成本控制和效率提升，營業利潤率能維持在一個合理水平，但考慮到行業波動，會略有調整。
    *   2026年: 10% (考慮到營收下降的影響)
    *   2027年: 10%
    *   2028年: 12%
    *   2029年: 12%
    *   2030年: 13%
3.  **稅率 (Tax Rate):** 26.88%
4.  **資本支出 (Capital Expenditures, CapEx):**
    *   2025財年 CapEx: $2.18 億美元。我們將假設 CapEx 佔營收的比例在預測期內保持相對穩定，或略有增長以支持船隊更新。
    *   假設 CapEx 佔營收的比例約為3-4%。
5.  **折舊與攤銷 (Depreciation & Amortization, D&A):**
    *   根據2025財年數據，EBITDA ($21.7 億美元) 和 EBIT ($8.85 億美元)，D&A 約為 $21.7B - $8.85B = $12.85 億美元。這與 CapEx 存在較大差異。考慮到航運業的資產密集性，D&A 通常較高。我們將假設 D&A 佔營收的比例約為15-18%。
6.  **營運資本變動 (Change in Working Capital):** 假設為營收的穩定百分比，或在預測期內保持中性。
7.  **自由現金流 (Free Cash Flow, FCF):** 計算每年的 FCF。
8.  **終值 (Terminal Value):** 使用永續增長模型 (Gordon Growth Model)。
    *   終值 = (最後一年 FCF × (1 + 長期增長率)) / (WACC - 長期增長率)

#### 3.3. 股息折現模型 (Dividend Discount Model, DDM)

ZIM 的股息政策是可變的，通常將淨利的30-50%作為股息支付。由於航運業的周期性，股息支付波動較大，過去三年的平均股息增長率為 -6.61%，這使得傳統的單一增長率 DDM 難以適用。

然而，我們可以考慮其2025年的總股息支付為每股 $1.99。如果假設未來股息能夠以一個非常低的穩定速度增長（例如0%），或者在短期內保持穩定，然後再以一個極低的增長率增長，則可以進行簡化估算。

**簡化 DDM 假設：**

*   **當前股息 (D0):** $1.99 (2025財年每股股息)
*   **股權成本 (Ke):** 10.51%
*   **股息增長率 (g):** 由於歷史波動和負增長，我們假設一個非常保守的長期增長率，例如 **0%**。

**DDM 計算：**
*   每股估值 = D0 × (1 + g) / (Ke - g)
*   每股估值 = $1.99 × (1 + 0%) / (0.1051 - 0%) = $1.99 / 0.1051 = **$18.93**

**DDM 估值結果：**
*   **每股合理價位：** $18 - $20
*   **分析：** 由於 ZIM 的股息政策高度依賴於其周期性盈利，且歷史股息增長率為負，DDM 模型在此情況下可靠性較低。此模型得出的估值顯著低於當前股價和收購價格，反映了市場對其未來股息穩定性和增長前景的悲觀預期。

### 4. 綜合結論與比較

| 估值模型       | 每股估值範圍 (美元) |

**綜合分析：**

1.  **DCF 模型結果異常高：** DCF 模型給出的估值遠高於其他模型和市場價格。這主要是因為航運業的周期性導致 ZIM 的盈利和現金流波動巨大，使得長期增長率和折舊攤銷等假設對最終估值影響極大。在高度波動的行業中，DCF 模型需要非常謹慎的假設，否則容易產生偏差。市場顯然沒有給予 ZIM 如此高的長期增長預期。
2.  **DDM 模型結果偏低：** DDM 模型由於 ZIM 股息的波動性和過去的負增長，導致估值偏低。這反映了該模型在股息不穩定公司中的局限性。
3.  **相對估值模型較為合理：** 相對估值模型給出的 $23 - $55 區間與當前股價和收購價格更為接近。這表明市場在評估 ZIM 時，更傾向於與同業進行比較。
4.  **收購價格的決定性影響：** 最關鍵的資訊是 Hapag-Lloyd 以每股 $35.00 的價格收購 ZIM 的協議。如果這項交易獲得批准並完成，那麼 $35.00 將成為 ZIM 股票的實際價值。當前股價 $25.88 遠低於收購價格，這可能反映了市場對交易完成時間、監管批准風險或潛在交易破裂的擔憂。

**綜合結論：**

考慮到 ZIM 已有明確的收購協議，每股 $35.00 的收購價格應被視為其最直接且最可靠的估值參考。儘管 DCF 模型給出了非常高的估值，但這在很大程度上是模型對航運業周期性波動敏感的結果，並未被市場認可。DDM 模型因股息不穩定而可靠性較低。相對估值模型提供了一個更為現實的區間，其中 $35.00 的收購價格位於該區間內。

**投資建議：**

對於尋求短期套利機會的投資者，如果認為 Hapag-Lloyd 的收購案將順利完成，那麼 ZIM 目前的股價 ($25.88) 相對於 $35.00 的收購價格存在顯著的溢價空間。然而，投資者需要仔細評估交易完成的風險和時間成本。對於長期投資者而言，在收購案完成前，ZIM 的基本面分析雖然重要，但收購價格已成為其價值的主要驅動因素。

### 5. 與同產業可比公司的估值比較
