#### 2.1 估值假設

**通用假設：**

*   **當前股價 (Close)**: $21.53
*   **流通股數 (Shares Outstanding)**: 根據2023年第四季度財報，約1.2029億股。 (更新：2024年第一季度財報顯示稀釋後每股收益為0.75美元，基於約1.226億股計算，我將使用此數字作為參考。)

*   **長期增長率 (Terminal Growth Rate)**: 由於航運業的週期性，長期增長率將設定為較低的水平，例如0%至1%。

**加權平均資本成本 (WACC) / 折現率：**

WACC的計算需要股權成本和債務成本。

*   **債務成本 (Cost of Debt)**:
    *   ZIM的長期債務/股權比率 (LT Debt/Eq) 為1.12。
    *   我將假設一個無風險利率（例如美國10年期國債收益率，約4.2%）。
    *   ZIM的債務負擔較高，且行業風險較大，我將假設一個較高的債務溢價。

    *   企業稅率：以色列企業稅率約為23%。

*   **股權成本 (Cost of Equity, Ke)**: 使用資本資產定價模型 (CAPM)。
    *   **無風險利率 (Risk-Free Rate, Rf)**: 4.2% (參考美國10年期國債收益率)
    *   **市場風險溢價 (Market Risk Premium, MRP)**: 5.5% (行業常用範圍)
    *   **Beta (β)**: 航運業通常具有較高的Beta值，反映其對市場波動的敏感性。我將保守估計Beta為1.5。

    *   Ke = Rf + β * MRP
    *   Ke = 4.2% + 1.5 * 5.5% = 4.2% + 8.25% = 12.45%

*   **WACC 計算** (假設債務成本為7%):
    *   債務權重 (D/V) = Debt / (Debt + Equity)
    *   股權權重 (E/V) = Equity / (Debt + Equity)
    *   ZIM的債務/股權比率 (Debt/Eq) 為1.41。
    *   D/E = 1.41 => D = 1.41E
    *   D/V = 1.41E / (1.41E + E) = 1.41 / 2.41 ≈ 0.585
    *   E/V = 1 / 2.41 ≈ 0.415
    *   WACC = (E/V) * Ke + (D/V) * Kd * (1 - Tax Rate)
    *   WACC = 0.415 * 12.45% + 0.585 * 7% * (1 - 0.23)
    *   WACC = 5.16% + 0.585 * 7% * 0.77
    *   WACC = 5.16% + 3.16% = 8.32%

    我將使用 **WACC = 8.5%** 作為基準折現率，並進行敏感性分析。

#### 2.2 折現現金流模型 (DCF)

DCF模型需要預測未來自由現金流 (FCF)。鑑於ZIM盈利的劇烈波動，預測FCF具有挑戰性。我將基於其2024年最新的指引和歷史FCF數據進行估計。

*   **終端增長率 (Terminal Growth Rate, g)**: 0.5% (保守估計，反映長期行業平均增長)

**DCF 計算步驟：**

1.  **預測未來5年FCF (2025-2029)**
    *   2024 FCF (估計) = 20億美元
    *   2025 FCF = 20億美元 * (1 - 0.10) = 18億美元
    *   2026 FCF = 18億美元 * (1 - 0.05) = 17.1億美元
    *   2027 FCF = 17.1億美元 * (1 + 0%) = 17.1億美元
    *   2028 FCF = 17.1億美元 * (1 + 0%) = 17.1億美元
    *   2029 FCF = 17.1億美元 * (1 + 0%) = 17.1億美元

2.  **計算各年FCF的現值 (PV of FCF)**
    *   PV(FCF_t) = FCF_t / (1 + WACC)^t

3.  **計算終端價值 (Terminal Value, TV)**
    *   TV = FCF_2029 * (1 + g) / (WACC - g)
    *   TV = 17.1億美元 * (1 + 0.005) / (0.085 - 0.005)
    *   TV = 17.1億美元 * 1.005 / 0.08 = 17.1855億美元 / 0.08 = 214.82億美元

4.  **計算終端價值的現值 (PV of TV)**
    *   PV(TV) = TV / (1 + WACC)^5
    *   PV(TV) = 214.82億美元 / (1 + 0.085)^5 = 214.82億美元 / 1.5036 ≈ 142.87億美元

5.  **計算企業價值 (Enterprise Value, EV)**
    *   EV = PV(FCF_2025) + PV(FCF_2026) + PV(FCF_2027) + PV(FCF_2028) + PV(FCF_2029) + PV(TV)

    *   PV(FCF_2025) = 18億 / (1.085)^1 = 16.59億美元
    *   PV(FCF_2026) = 17.1億 / (1.085)^2 = 14.52億美元
    *   PV(FCF_2027) = 17.1億 / (1.085)^3 = 13.38億美元
    *   PV(FCF_2028) = 17.1億 / (1.085)^4 = 12.33億美元
    *   PV(FCF_2029) = 17.1億 / (1.085)^5 = 11.36億美元

    *   EV = 16.59 + 14.52 + 13.38 + 12.33 + 11.36 + 142.87 = 211.05億美元

6.  **計算股權價值 (Equity Value)**
    *   Equity Value = EV + Cash & Equivalents - Total Debt
    *   根據您提供的數據，ZIM的市值為25.7億美元，企業價值為63億美元。
    *   EV = Market Cap + Total Debt - Cash & Equivalents
    *   Total Debt - Cash & Equivalents = EV - Market Cap = 63億 - 25.7億 = 37.3億美元 (淨債務)
    *   Equity Value = 211.05億美元 - 37.3億美元 = 173.75億美元

7.  **計算每股內在價值 (Intrinsic Value Per Share)**
    *   Intrinsic Value Per Share = Equity Value / Shares Outstanding
    *   Intrinsic Value Per Share = 173.75億美元 / 1.226億股 ≈ **$141.72**

ZIM的股息政策是按季度支付淨利潤的30%。 2024年第一季度派發0.23美元，第三季度派發3.65美元（含特別股息）。

**DDM 假設：**

*   **當前股價**: $21.53
*   **股權成本 (Ke)**: 12.45% (來自WACC計算)
*   **股息增長率 (g)**: 這是DDM的關鍵且最具挑戰性的假設。
    *   ZIM的股息支付極不穩定，過去幾年有高額股息，也有暫停。
    *   2024年第一季度EPS為0.75美元，派發0.23美元股息。
    *   2024年第三季度EPS為9.35美元，派發3.65美元股息（含特別股息）。
    *   分析師預計2024年EPS為17.8美元，2025年為3.04美元，2026年為-3.11美元。
    *   考慮到分析師對未來EPS的急劇下降預期，以及行業的週期性，預計未來股息將大幅波動甚至下降。
    *   我將採用一個保守的股息增長率，甚至負增長，或者假設股息在幾年後穩定在一個較低的水平。

**DDM 兩階段模型：**

由於股息波動大，單一增長率模型不適用。我將嘗試一個兩階段模型：
*   **第一階段 (高增長/波動期，例如2025-2026年)**: 假設股息在2025年和2026年大幅下降。
    *   2024年總股息：Q1 $0.23 + Q3 $3.65 = $3.88 (已宣布)
    *   考慮到2024年全年EPS預計為17.8美元，如果按30%派息率，則2024年總股息可能更高。
    *   假設2024年總股息為 $3.88 (已宣布部分)。
    *   預計2025年EPS大幅下降至3.04美元，按30%派息率，2025年股息約為 $0.91。
    *   預計2026年EPS為負值 (-3.11美元)，因此2026年股息可能為0或非常低。我將假設2026年股息為0。

*   **第二階段 (穩定增長期，從2027年開始)**: 假設公司在2027年後恢復穩定但較低的股息支付，並以0.5%的長期增長率增長。
    *   假設2027年股息恢復到一個較低的穩定水平，例如 $0.50。

**DDM 計算步驟：**

1.  **計算第一階段股息的現值 (PV of Dividends)**
    *   D0 (2024年已宣布) = $3.88
    *   D1 (2025年預計) = $0.91
    *   D2 (2026年預計) = $0

    *   PV(D1) = $0.91 / (1 + 0.1245)^1 = $0.81
    *   PV(D2) = $0 / (1 + 0.1245)^2 = $0

2.  **計算第二階段終端價值 (Terminal Value of Dividends)**
    *   D3 (2027年預計) = $0.50
    *   TV_DDM = D3 / (Ke - g)
    *   TV_DDM = $0.50 / (0.1245 - 0.005) = $0.50 / 0.1195 = $4.18

3.  **計算終端價值的現值 (PV of TV_DDM)**
    *   PV(TV_DDM) = TV_DDM / (1 + Ke)^2 (折現到2026年末)
    *   PV(TV_DDM) = $4.18 / (1 + 0.1245)^2 = $4.18 / 1.2645 ≈ $3.31

4.  **計算每股內在價值 (Intrinsic Value Per Share)**
    *   Intrinsic Value Per Share = PV(D1) + PV(D2) + PV(TV_DDM)
    *   Intrinsic Value Per Share = $0.81 + $0 + $3.31 = **$4.12**

**DDM 模型結果：**
基於對未來股息的保守預期，DDM模型得出的每股內在價值約為 **$4.12**。這個結果遠低於當前股價，反映了市場對ZIM未來股息可持續性的擔憂，以及航運業的週期性對股息的影響。

**DDM 模型的局限性：**
DDM模型對股息增長率和股權成本的假設非常敏感。ZIM的股息歷史波動巨大，使得預測未來股息非常困難。如果公司未來能維持較高的盈利和派息，估值會顯著提高。然而，分析師對未來EPS的悲觀預期使得保守估計更為合理。

#### 2.5 其他合理估值方法 (例如：資產淨值法)

**資產淨值法結果：**
基於賬面價值法，ZIM的合理價位約為 **$33.64**。

**綜合結論：**

ZIM的估值分析呈現出一個複雜的局面。一方面，其極低的P/E、P/B、P/FCF和EV/EBITDA倍數，以及近期強勁的盈利和高股息，似乎表明公司被嚴重低估。DCF和相對估值模型在假設未來盈利能夠維持或僅溫和下降的情況下，給出了非常高的內在價值。

**合理價位區間：**

考慮到ZIM業務的週期性和不確定性，以及不同估值模型之間的巨大差異，很難給出一個精確的「合理價位」。
