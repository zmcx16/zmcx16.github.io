### 2. 估值假設

    *   我們使用長期債務權益比 (LT Debt/Eq) 0.70 來估算債務權重，因為這更能代表公司的長期融資結構。

*   **加權平均資本成本 (WACC):**
    *   WACC = (We * Ke) + (Wd * Kd_after_tax)
    *   WACC = (0.588 * 11.35%) + (0.412 * 5.94%) = 6.67% + 2.45% = **9.12%**

#### 3.1. 折現現金流模型 (Discounted Cash Flow, DCF)

**步驟 3: 計算終端價值 (Terminal Value, TV)**

*   FCF_Year6 = FCF_Year5 * (1 + g_long_term) = $1225.28M * (1 + 0.025) = $1255.91M

**步驟 5: 計算企業價值 (Enterprise Value, EV)**

#### 3.3. 股息折現模型 (Dividend Discount Model, DDM)

**步驟 3: 應用戈登增長模型 (Gordon Growth Model)**
