#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2026 - 2030)
*   **收入增長率 (Revenue Growth Rate):**

    *   2026年: 10% (考慮到市場增長和公司戰略)
    *   2027年: 8%
    *   2028年: 7%
    *   2029年: 6%
    *   2030年: 5%
*   **營業利潤率 (Operating Margin):** 根據提供的數據，Oper. Margin 為 0.3624 (36.24%)。考慮到公司在高利率環境下的毛利增長指引下調，我們將在預測期內保持相對穩定或略有下降，然後趨於長期平均水平。
    *   2026-2030年: 35% (略低於當前水平，反映競爭和成本壓力)
*   **稅率 (Tax Rate):** 巴西企業所得稅率約為 34%。
*   **資本支出 (Capital Expenditure, CapEx):** 假設為營收的 3% (與歷史趨勢和行業平均水平一致)。
*   **營運資本變動 (Change in Working Capital):** 假設為營收變動的 5% (反映營運效率)。
*   **折現率 (WACC - Weighted Average Cost of Capital):**
    *   **無風險利率 (Risk-Free Rate):** 假設為 4.5% (參考美國長期國債收益率)。
    *   **市場風險溢價 (Market Risk Premium):** 假設為 6%。
    *   **Beta:** PAGS 的 Beta 值通常較高，反映其新興市場和成長型公司的特性。我們採用 1.5。
    *   **股權成本 (Cost of Equity, Ke):** Ke = 無風險利率 + Beta \* 市場風險溢價 = 4.5% + 1.5 \* 6% = 13.5%。
    *   **債務成本 (Cost of Debt, Kd):** 考慮巴西的高利率環境，假設為 8%。
    *   **債務權益比 (Debt/Equity Ratio):** 2.89 (來自提供數據)。
    *   **WACC 計算:**
        *   權益佔比 = 1 / (1 + 債務權益比) = 1 / (1 + 2.89) = 0.257 (約 25.7%)
        *   債務佔比 = 債務權益比 / (1 + 債務權益比) = 2.89 / (1 + 2.89) = 0.743 (約 74.3%)
        *   WACC = (權益佔比 \* Ke) + (債務佔比 \* Kd \* (1 - 稅率))
        *   WACC = (0.257 \* 13.5%) + (0.743 \* 8% \* (1 - 0.34)) = 3.47% + 3.92% = 7.39%
        *   **WACC 採用 7.5%** (取整並略微保守)。
*   **永續增長率 (Terminal Growth Rate):** 假設為 3% (略高於長期通脹率，反映公司在成熟市場的長期增長潛力)。

1.  **預測營收 (Revenue Forecast):**
    *   2025年營收 (預期): 38.5 億美元

    *   2030年: 51.87 * (1 + 5%) = 54.46 億美元

2.  **計算營業利潤 (Operating Profit = EBIT):** 營收 \* 營業利潤率 (35%)

3.  **計算稅後營業利潤 (NOPAT):** EBIT \* (1 - 稅率 34%)

4.  **計算自由現金流 (Free Cash Flow, FCF):**
    *   FCF = NOPAT + 折舊與攤銷 (假設包含在營業利潤中，或與 CapEx 相抵) - 資本支出 - 營運資本變動

    *   CapEx = 營收 \* 3%
    *   營運資本變動 = (當年營收 - 前一年營收) \* 5%

5.  **計算永續價值 (Terminal Value, TV):**
    *   TV = (2030年 FCF \* (1 + 永續增長率)) / (WACC - 永續增長率)

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

PAGS 支付股息，其 Dividend % 為 1.22%，並且計劃在 2026 年加速股息派發，預計分配 14 億 BRL。這表明 DDM 適用。

**估值假設：**

*   **最新股價:** 10.86 美元
*   **最新股息率:** 1.22%
*   **每股股息 (D0):** 10.86 美元 \* 1.22% = 0.1325 美元 (基於當前股價和股息率)
*   **股息增長率 (g):** 考慮到公司計劃加速派息和 EPS 的增長預期 (2026年 EPS 增長 11.76%)，我們假設短期內股息增長較快，長期趨於穩定。
    *   短期 (未來 3 年): 10% (基於公司加速派息計劃和 EPS 增長)
    *   長期 (永續增長率): 4% (略高於長期通脹，低於公司長期 EPS 增長目標)
*   **股權成本 (Ke):** 13.5% (與 DCF 模型中的股權成本一致)

**DDM 計算步驟 (兩階段 DDM):**

1.  **計算預測期內的股息 (D1, D2, D3):**
    *   D1 (2026年) = D0 \* (1 + 10%) = 0.1325 \* 1.10 = 0.1458 美元
    *   D2 (2027年) = D1 \* (1 + 10%) = 0.1458 \* 1.10 = 0.1604 美元
    *   D3 (2028年) = D2 \* (1 + 10%) = 0.1604 \* 1.10 = 0.1764 美元

2.  **計算預測期末的股票價值 (P3):**
    *   P3 = D4 / (Ke - g_long)
    *   D4 = D3 \* (1 + g_long) = 0.1764 \* (1 + 0.04) = 0.1835 美元
    *   P3 = 0.1835 / (0.135 - 0.04) = 0.1835 / 0.095 = 1.9316 美元

3.  **折現所有股息和 P3 至現值:**
    *   P0 = D1/(1+Ke)^1 + D2/(1+Ke)^2 + D3/(1+Ke)^3 + P3/(1+Ke)^3
    *   P0 = 0.1458/(1.135)^1 + 0.1604/(1.135)^2 + 0.1764/(1.135)^3 + 1.9316/(1.135)^3
    *   P0 = 0.1458/1.135 + 0.1604/1.288 + 0.1764/1.462 + 1.9316/1.462
    *   P0 = 0.1285 + 0.1245 + 0.1207 + 1.3212 = **1.6949 美元**

**DDM 模型估值結果：**
*   **估值結果:** 1.69 美元
*   **合理價位區間:** 1.50 - 2.00 美元

**DDM 模型的局限性:**
DDM 模型在此處的估值結果遠低於當前股價和 DCF/相對估值。這可能表明 PAGS 的價值主要來自於其未來的成長潛力而非當前的股息派發。對於成長型公司，特別是處於高增長市場的金融科技公司，其大部分利潤可能會再投資於業務擴張，而非以股息形式分配。因此，DDM 可能無法完全捕捉 PAGS 的真實價值。公司計劃在 2026 年加速股息派發，這可能在未來提高 DDM 的適用性，但目前來看，其股息增長率和派息比例可能不足以反映其全部價值。
