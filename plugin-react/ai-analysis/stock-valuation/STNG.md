### 3. 估值方法與計算

#### 3.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。由於STNG的EPS在今年預期為負，我們將更側重於自由現金流的預測，並假設其在未來幾年內恢復正增長。

**估值假設：**

1.  **折現率 (WACC - 加權平均資本成本):**
    *   **無風險利率 (Risk-Free Rate):** 採用美國10年期國債收益率，約 **4.5%** (截至近期)。
    *   **市場風險溢價 (Market Risk Premium, MRP):** 採用行業標準，約 **5.5%**。
    *   **Beta (STNG):** 航運業通常波動較大，根據網路搜尋，STNG的Beta值約在 **1.2 - 1.5** 之間。我們取 **1.35**。
    *   **股權成本 (Cost of Equity, Ke):** Ke = 無風險利率 + Beta \* MRP
        *   Ke = 4.5% + 1.35 \* 5.5% = 4.5% + 7.425% = **11.925%**
    *   **債務成本 (Cost of Debt, Kd):** 考慮到STNG的低負債率和當前利率環境，假設其稅前債務成本為 **6.5%**。

    *   **債務權重 (Weight of Debt, Wd):** 根據 `Debt/Eq = 0.29`，則 `Wd = Debt / (Debt + Equity) = 0.29 / (1 + 0.29) = 0.2248`。
    *   **股權權重 (Weight of Equity, We):** `We = 1 - Wd = 1 - 0.2248 = 0.7752`。
    *   **WACC 計算:** WACC = (We \* Ke) + (Wd \* Kd \* (1 - Tax Rate))
        *   WACC = (0.7752 \* 11.925%) + (0.2248 \* 6.5% \* (1 - 0.21))
        *   WACC = 9.24% + (0.2248 \* 6.5% \* 0.79)
        *   WACC = 9.24% + 1.15% = **10.39%** (約 10.4%)

2.  **預測期間:** 5 年。

3.  **自由現金流 (FCF) 預測:**

        *   **第 5 年 (過渡期):** 增長率放緩至 **2.5%**。
    *   **永續增長率 (Terminal Growth Rate, g):** 採用長期通脹率或全球經濟增長率，約 **2.0%**。

**DCF 計算步驟：**

*   **預測期 FCF:**
    *   FCF0 (TTM) = $335.0M
    *   FCF1 = $335.0M \* (1 + 0.05) = $351.75M
    *   FCF2 = $351.75M \* (1 + 0.05) = $369.34M
    *   FCF3 = $369.34M \* (1 + 0.03) = $380.42M
    *   FCF4 = $380.42M \* (1 + 0.03) = $391.83M
    *   FCF5 = $391.83M \* (1 + 0.025) = $401.62M

    *   PV(FCF1) = $351.75M / (1 + 0.104)^1 = $318.62M
    *   PV(FCF2) = $369.34M / (1 + 0.104)^2 = $302.48M
    *   PV(FCF3) = $380.42M / (1 + 0.104)^3 = $283.09M
    *   PV(FCF4) = $391.83M / (1 + 0.104)^4 = $265.01M
    *   PV(FCF5) = $401.62M / (1 + 0.104)^5 = $244.08M
    *   **預測期 FCF 現值總和 = $318.62 + $302.48 + $283.09 + $265.01 + $244.08 = $1,413.28M**

*   **終值 (Terminal Value, TV):**

*   **企業價值 (Enterprise Value, EV):**
    *   EV = 預測期 FCF 現值總和 + 終值現值
    *   EV = $1,413.28M + $2,973.65M = $4,386.93M

*   **股權價值 (Equity Value):**

    *   每股價值 = Equity Value / 流通股數
