**3. 股息折現模型 (Dividend Discount Model, DDM)**

DDM 適用於股息穩定且預計會持續增長的公司。星巴克有穩定的股息支付歷史，因此適用此模型。

**估值假設：**
*   **最新季度股息:** $0.62，年化股息 = $0.62 * 4 = $2.48。
*   **股息增長率 (g):**
    *   考慮到公司正在進行重組和投資，短期內股息增長可能較為溫和。
    *   分析師預計未來 EPS 增長率約為 26.26% (FY26) 和 28.5% (FY27)。
    *   然而，股息增長通常會比 EPS 增長更穩定和保守。
    *   我將假設一個保守的長期股息增長率，例如 4%。
*   **折現率 (r):** 採用 WACC 的成本股權，即 7.18%。

**DDM 計算步驟 (戈登增長模型 - Gordon Growth Model):**
*   合理股價 = D1 / (r - g)
    *   D1 = 下一期預計股息 = 當前年化股息 * (1 + g)
    *   D1 = $2.48 * (1 + 0.04) = $2.48 * 1.04 = $2.5792
    *   合理股價 = $2.5792 / (0.0718 - 0.04)
    *   合理股價 = $2.5792 / 0.0318 = $81.11

**DDM 估值結果：**
*   **每股合理價位:** $81.11
*   **合理價位區間:** $75.00 - $88.00 (考慮到股息增長率和折現率的敏感性)
