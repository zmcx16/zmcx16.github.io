### III. 估值模型分析

#### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型是評估公司內在價值最全面的方法之一，它基於公司未來產生的自由現金流。

**A. 加權平均資本成本 (WACC) 計算**

WACC是DCF模型的關鍵折現率，代表公司所有資金來源的平均成本。

*   **1. 無風險利率 (Rf)：** 採用美國10年期公債殖利率。
    *   假設 Rf = 4.50%
*   **2. 市場風險溢酬 (MRP)：** 歷史平均值。
    *   假設 MRP = 5.50%
*   **3. 槓桿 Beta (Levered Beta)：** 反映股票相對於市場的波動性。
    *   SBUX Beta (參考值) = 1.10
*   **4. 股權成本 (Cost of Equity, Ke) - CAPM 模型：**
    *   Ke = Rf + Beta * MRP
    *   Ke = 4.50% + 1.10 * 5.50% = 4.50% + 6.05% = **10.55%**
*   **5. 債務成本 (Cost of Debt, Kd)：** SBUX發行債券的平均利率或有效利息支出率。
    *   假設 Kd (稅前) = 5.00%
*   **6. 有效稅率 (Tax Rate, T)：** 參考公司最新財報的平均有效稅率。
    *   假設 T = 23.0%
    *   稅後債務成本 = Kd * (1 - T) = 5.00% * (1 - 0.23) = 3.85%
*   **7. 資本結構 (D/E ratio)：**
    *   市值 (E) = $1020 億美元 (基於$90股價)
    *   總債務 (D) (參考最新財報，含租賃負債) = 約 $180 億美元
    *   E/(D+E) = 1020 / (180 + 1020) = 1020 / 1200 = 0.85
    *   D/(D+E) = 180 / 1200 = 0.15
*   **8. WACC 計算：**
    *   WACC = (Ke * E/(D+E)) + (Kd * (1-T) * D/(D+E))
    *   WACC = (10.55% * 0.85) + (5.00% * (1 - 0.23) * 0.15)
    *   WACC = 8.97% + (3.85% * 0.15)
    *   WACC = 8.97% + 0.58% = **9.55%**
    *   **取整 WACC = 9.5%**

    *   每股內在價值 = $442.81 億 / 1.14 億股 = **$38.84 美元/股**

#### 3. 股息折現模型 (Dividend Discount Model, DDM)

SBUX是穩定派發股息的公司，因此DDM模型適用。然而，DDM更適合於股息增長穩定且可預測的成熟公司。SBUX仍有增長潛力，其估值可能不僅僅反映在股息上。

*   **最新每股股息 (D0)：** $2.28 美元/股 (年化)
*   **股權成本 (Ke)：** 10.55% (來自WACC計算)

**假設兩階段股息增長模型：**
*   **階段一 (高增長期，3年)：** SBUX近年股息增長率約10-15%，我們預計未來3年保持在較高的水平，但會逐步放緩。
    *   2024年 (g1): 8.0%
    *   2025年 (g2): 7.0%
    *   2026年 (g3): 6.0%
*   **階段二 (永續增長期)：** 之後股息以較低的永續增長率增長。
    *   長期增長率 (g_LT): 4.0% (高於通脹，但低於Ke)

**計算：**

*   **D1 =** D0 * (1 + g1) = $2.28 * 1.08 = $2.46
*   **D2 =** D1 * (1 + g2) = $2.46 * 1.07 = $2.63
*   **D3 =** D2 * (1 + g3) = $2.63 * 1.06 = $2.79
*   **D4 =** D3 * (1 + g_LT) = $2.79 * 1.04 = $2.90

*   **終值 (Terminal Value) 於 D3 時點：**
    *   TV3 = D4 / (Ke - g_LT) = $2.90 / (0.1055 - 0.04) = $2.90 / 0.0655 = $44.27

*   **現值計算：**
    *   PV(D1) = D1 / (1 + Ke)^1 = $2.46 / (1.1055)^1 = $2.23
    *   PV(D2) = D2 / (1 + Ke)^2 = $2.63 / (1.1055)^2 = $2.16
    *   PV(D3) = D3 / (1 + Ke)^3 = $2.79 / (1.1055)^3 = $2.05
    *   PV(TV3) = TV3 / (1 + Ke)^3 = $44.27 / (1.1055)^3 = $32.54

*   **DDM 估值：** PV(D1) + PV(D2) + PV(D3) + PV(TV3)
    *   DDM 估值 = $2.23 + $2.16 + $2.05 + $32.54 = **$38.98 美元/股**

*   **DDM 估值結果：** $38.98 美元/股
*   **合理價位區間：** 考量DDM對增長率假設的敏感性，DDM估值範圍約在 **$35 - $45 美元/股**。
    *   *評論：* DDM結果與DCF模型結果非常接近，也顯著低於當前市場價格。這再次印證市場可能對SBUX的未來利潤或估值方式有不同的判斷。這可能是因為SBUX的增長來自於再投資而非完全分配股息，DDM可能無法完全捕捉這部分價值。

#### 4. 其他合理估值方法 (簡要說明)

*   **市銷率 (P/S Ratio)：** 簡單且常用於早期或低利潤公司。SBUX相對成熟，P/E和EV/EBITDA更具參考性。不過，若考慮其作為零售品牌，P/S可以反映市場對其營收增長和品牌價值的認可。可比公司平均P/S約5.55x，應用於SBUX營收$360億，市值約$1998億，每股約$175，明顯過高，因為可比公司中包含MCD等利潤率更高的特許經營模式。因此對於SBUX，EV/Sales可能更合理。
    *   可比公司平均 EV/Sales = 5.55x
    *   SBUX EV (估算) = $360億 * 5.55 = $1998億
    *   SBUX 股權價值 = $1998億 - $140億 (淨債務) = $1858億
    *   每股價值 = $1858億 / 1.14億 = $162.98。此值也顯著高估，主要原因為SBUX的營業模式和利潤率與特許經營的MCD、YUM等有差異，且SBUX自身的EV/Sales約2.85x，遠低於可比公司平均，可能因為其直接經營模式對營收貢獻較大但利潤率相對較低。
