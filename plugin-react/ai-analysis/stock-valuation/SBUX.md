由於沒有直接搜尋到星巴克可比公司的估值倍數，我們將參考餐飲業或咖啡連鎖店的平均倍數，並考慮星巴克的市場地位。

**估值計算：**

    *   SBUX 預期 Forward P/E = 27.93。

    *   使用 Forward P/E 估值 = 27.93 * 2.99 美元 = **83.59 美元/股**。

2.  **基於 P/S 倍數：**
    *   SBUX P/S = 2.57。
    *   2026 財年預計營收 = 394 億美元。
    *   市值 = P/S * 營收 = 2.57 * 394 億美元 = 1012.78 億美元。

3.  **基於 EV/EBITDA 倍數：**
    *   由於沒有直接提供 EV/EBITDA，我們需要估算。
    *   2026 財年預計 EBIT = 39.40 億美元。
    *   2026 財年預計 D&A = 11.82 億美元。

#### 3.3 股息折現模型 (DDM)

星巴克有穩定的股息支付歷史，因此 DDM 適用。

**關鍵數據與假設：**
*   當前股息率 (Dividend %)：0.0292 (2.92%)。
*   當前股價：83.97 美元。
*   每股股息 (DPS) = 83.97 美元 * 0.0292 = 2.45 美元。
*   股息增長率：
    *   短期 (5 年)：假設為 8% (考慮到公司面臨挑戰，但仍有增長潛力)。
    *   長期：3% (與 DCF 的長期增長率一致)。
*   股權成本 (Ke)：10.03%。

**DDM 計算：**

1.  **預測期股息：**
    *   DPS (2026) = 2.45 * (1 + 0.08) = 2.65 美元
    *   DPS (2027) = 2.65 * (1 + 0.08) = 2.86 美元
    *   DPS (2028) = 2.86 * (1 + 0.08) = 3.09 美元
    *   DPS (2029) = 3.09 * (1 + 0.08) = 3.34 美元
    *   DPS (2030) = 3.34 * (1 + 0.08) = 3.61 美元

2.  **永續期股息 (2031 年)：**
    *   DPS (2031) = DPS (2030) * (1 + 長期增長率) = 3.61 * (1 + 0.03) = 3.72 美元

3.  **永續價值 (Terminal Value of Dividend)：**
    *   TV_DDM = DPS (2031) / (Ke - 長期增長率)
    *   TV_DDM = 3.72 / (0.1003 - 0.03) = 3.72 / 0.0703 = 52.92 美元

4.  **每股估值：**
    *   每股估值 = 預測期折現股息總和 + 永續價值折現至當前
    *   折現股息：
        *   2026: 2.65 / (1.1003)^1 = 2.41
        *   2027: 2.86 / (1.1003)^2 = 2.36
        *   2028: 3.09 / (1.1003)^3 = 2.32
        *   2029: 3.34 / (1.1003)^4 = 2.28
        *   2030: 3.61 / (1.1003)^5 = 2.24
    *   折現股息總和 = 2.41 + 2.36 + 2.32 + 2.28 + 2.24 = 11.61 美元
    *   永續價值折現至當前 = 52.92 / (1.1003)^5 = 52.92 / 1.6135 = 32.80 美元
    *   每股估值 = 11.61 + 32.80 = **44.41 美元/股**

DDM 估值也顯著低於當前股價。
