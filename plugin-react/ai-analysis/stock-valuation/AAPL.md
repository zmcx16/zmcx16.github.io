##### 3. 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於派發穩定股息且股息增長可預測的公司。蘋果派發股息，因此適用。

**假設：**

*   **當前每股股息 (D0):** 股息率 0.41%，股價 $255.92。
    D0 = $255.92 \* 0.0041 = $1.049
    (Q1 FY2026 宣布股息為 $0.26/股，年化為 $1.04/股)
*   **股息增長率 (g):** 考慮到蘋果的成熟度，我們假設一個穩定的長期股息增長率。分析師預期EPS增長約為10-13%，但股息增長通常會更保守。我們假設股息增長率為 **5%**。
*   **股權成本 (Ke):** 9.75% (來自 WACC 計算)

**計算步驟：**

*   **永續增長 DDM 模型 (Gordon Growth Model):**
    股價 = D1 / (Ke - g)
    D1 = D0 \* (1 + g) = $1.049 \* (1 + 0.05) = $1.101
    股價 = $1.101 / (0.0975 - 0.05) = $1.101 / 0.0475 = **$231.79**

**DDM 模型估值結果：**
*   **估值結果:** $231.79
*   **合理價位區間:** 考慮到股息增長率的敏感性，合理區間可能在 **$210 - $250**。
