### 3. 估值方法與計算

#### 3.1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **基準自由現金流 (Base FCF)：** 根據模擬的最新財報和歷史數據，Apple的年度自由現金流約在 $1000億 - $1100億美元之間。我們取一個保守的估計，假設最近一個財年的自由現金流為 **$1050億美元**。
*   **預測期間：** 5年。
*   **增長率 (FCF Growth Rate)：**
    *   第1-2年：6% (考慮服務業務增長、AI整合帶來的潛在升級潮，但iPhone銷售面臨挑戰)
    *   第3-5年：5% (增長趨於穩定)
*   **永續增長率 (Terminal Growth Rate, g)：** 2.5% (略低於長期GDP增長率，反映大型成熟公司的長期增長潛力)
*   **加權平均資本成本 (WACC)：**

    *   **無風險利率 (Risk-Free Rate, Rf)：** 4.30% (參考美國10年期國債殖利率，模擬)
    *   **市場風險溢酬 (Market Risk Premium, MRP)：** 5.50% (行業普遍採用)
    *   **Beta (β)：** 1.25 (參考Yahoo Finance等數據源，模擬)
    *   **股權成本 (Cost of Equity, Ke) = Rf + β * MRP**
        = 4.30% + 1.25 * 5.50% = 4.30% + 6.875% = 11.175%
    *   **稅前債務成本 (Cost of Debt, Kd)：** 3.50% (Apple信用評級高，借貸成本低，模擬)
    *   **稅率 (Tax Rate, t)：** 14.5% (參考Apple的有效稅率，模擬)
    *   **稅後債務成本 = Kd * (1 - t)** = 3.50% * (1 - 0.145) = 3.50% * 0.855 = 2.99%
    *   **市值 (Market Value of Equity, E)：** $4.04 兆美元 (來自提供數據)
    *   **淨債務 (Net Debt)：** 根據模擬財報，總債務約$1040億美元，現金及約當現金約$670億美元。
        *   **市場價值債務 (Market Value of Debt, D)：** $1040億美元 (模擬)
    *   **總資本 (V) = E + D** = $4.04兆 + $0.104兆 = $4.144兆
    *   **股權權重 (We) = E / V** = $4.04兆 / $4.144兆 = 0.9749
    *   **債務權重 (Wd) = D / V** = $0.104兆 / $4.144兆 = 0.0251
    *   **WACC = (We * Ke) + (Wd * Kd * (1 - t))**
        = (0.9749 * 11.175%) + (0.0251 * 2.99%)
        = 10.896% + 0.075% = 10.971%
        **取 WACC ≈ 11.0%**

**DCF 計算：**

*   **預測期自由現金流現值總和 (PV of FCFs)：** $100.26 + $95.75 + $90.56 + $85.69 + $81.04 = **$453.30 十億美元**

*   **終端價值 (Terminal Value, TV)：**
    *   TV = FCF_year5 * (1 + g) / (WACC - g)
    *   TV = $136.57 * (1 + 0.025) / (0.110 - 0.025)
    *   TV = $136.57 * 1.025 / 0.085 = $140.00 / 0.085 = **$1647.06 十億美元**

*   **終端價值現值 (PV of TV)：**
    *   PV of TV = TV / (1 + WACC)^5
    *   PV of TV = $1647.06 / (1 + 0.110)^5 = $1647.06 / 1.6850 = **$977.48 十億美元**

    *   EV = $453.30 十億美元 + $977.48 十億美元 = **$1430.78 十億美元**

*   **股權價值 (Equity Value) = EV - 淨債務**
    *   淨債務 = 總債務 - 現金及約當現金 = $1040億 - $670億 = $370億美元
    *   Equity Value = $1430.78 十億美元 - $370億美元 = **$1393.78 十億美元**
