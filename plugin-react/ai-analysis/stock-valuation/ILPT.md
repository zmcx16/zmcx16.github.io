### **2.1 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型需要預測自由現金流 (Free Cash Flow, FCF)。對於 REITs，通常會使用調整後的 FFO 或 AFFO 作為現金流的代理，或者直接預測營運現金流並扣除資本支出。由於我們沒有詳細的資本支出預測，且 REITs 的 FFO 已經是衡量其營運現金流的重要指標，我們將基於 FFO 進行簡化估值。

**估值假設：**

*   **預測期：** 5年 (2026-2030)
*   **增長率：**
    *   **短期增長率 (2026-2028)：** 考慮到2025年Normalized FFO的強勁增長（113% YoY）以及2026年Q1的預期 ($0.29-$0.31)，以及產業的積極前景，我們假設未來三年 FFO 增長率為 5%。這比2026年EPS next Y_% (14.29%) 低，但考慮到REITs的穩定性，採取更保守的增長。
    *   **中期增長率 (2029-2030)：** 假設增長放緩至 3%。
    *   **永續增長率 (Terminal Growth Rate)：** 2%。這符合長期通脹和經濟增長率。
*   **折現率 (Discount Rate) - 加權平均資本成本 (WACC)：**
    *   **股權成本 (Cost of Equity, Ke)：** 使用資本資產定價模型 (CAPM)。
        *   無風險利率 (Risk-Free Rate)：參考美國10年期國債收益率，假設為 4.2%。
        *   市場風險溢價 (Market Risk Premium, MRP)：假設為 5.5%。
        *   Beta：由於未提供，我們將搜尋 ILPT 的 Beta 值。
根據搜尋結果：
*   ILPT 的 Beta (5 Year) 為 2.49。
*   ILPT 的 Beta (LTM) 為 1.09x。
*   MarketBeat 報告的 Beta 為 2.46。

由於 Beta 值差異較大，且 2.49 顯著高於行業平均 0.66，這可能表明 ILPT 的波動性非常高。考慮到 REITs 通常被認為是相對穩定的資產，且其業務性質，我將採用一個更為保守且接近行業平均的 Beta 值，或者考慮其 LTM Beta。然而，由於其高負債，高 Beta 可能是合理的。為了反映其較高的風險，我將使用 1.5 作為 Beta 值，這介於 LTM Beta 和 5年 Beta 之間，並考慮到其高負債。

    *   Ke = 無風險利率 + Beta * 市場風險溢價
    *   Ke = 4.2% + 1.5 * 5.5% = 4.2% + 8.25% = 12.45%

    *   **債務成本 (Cost of Debt, Kd)：**
        *   ILPT 的 Debt/Eq 為 8.59 (用戶提供數據)，或 4.66 (Stock Analysis)。這表明公司負債很高。
        *   利息覆蓋率 (Interest Coverage) 為 0.56，這非常低，表明公司償還利息的能力較弱，債務成本可能較高。
        *   我們需要估計其稅前債務成本。考慮到其高負債和低利息覆蓋率，假設稅前債務成本為 7.5%。
        *   稅率 (Tax Rate)：由於 REITs 通常有特殊的稅務處理，其公司層面繳納的所得稅可能較低。然而，為了 WACC 計算，我們需要一個有效的公司稅率。由於 ILPT 報告淨虧損，其有效稅率可能為負或接近零。我們將使用一個名義稅率，例如 21% (美國聯邦公司稅率)。
        *   稅後債務成本 = Kd * (1 - Tax Rate) = 7.5% * (1 - 0.21) = 7.5% * 0.79 = 5.925%

    *   **債務權重 (Weight of Debt, Wd) 和股權權重 (Weight of Equity, We)：**
        *   市值 (Market Cap): $4.0592 億美元 (用戶提供數據)
        *   總債務 (Total Debt): 根據 Stock Analysis，ILPT 有 $4.20 億美元的債務。
        *   總資本 = 市值 + 總債務 = $4.0592 億 + $42.0 億 = $46.0592 億美元
        *   We = 市值 / 總資本 = $4.0592 億 / $46.0592 億 = 0.0881 (約 8.81%)
        *   Wd = 總債務 / 總資本 = $42.0 億 / $46.0592 億 = 0.9119 (約 91.19%)
        *   注意：這個債務權重非常高，反映了公司的高度槓桿。

    *   **WACC = (We * Ke) + (Wd * Kd * (1 - Tax Rate))**
    *   WACC = (0.0881 * 12.45%) + (0.9119 * 7.5% * (1 - 0.21))
    *   WACC = (0.0881 * 0.1245) + (0.9119 * 0.05925)
    *   WACC = 0.01096 + 0.05401 = 0.06497 = 6.50%

**DCF 計算：**

我們將使用 Normalized FFO 作為現金流。
*   2025年 Normalized FFO/share: $1.16 (年化)
*   流通股數 (Shares Outstanding): 66.65 百萬股
*   2025年總 Normalized FFO = $1.16 * 66.65 百萬 = $77.314 百萬

**終值 (Terminal Value, TV) 計算：**
TV = (2030年 FFO * (1 + 永續增長率)) / (WACC - 永續增長率)
TV = ($94.952 百萬 * (1 + 0.02)) / (0.0650 - 0.02)
TV = ($94.952 百萬 * 1.02) / 0.045

**折現現金流計算：**

| :--- | :------------- | :--------------------- | :------------------- |

**總企業價值 (Total Enterprise Value) = 折現 FFO 總和 + 折現終值**
總企業價值 = $76.220 + $75.130 + $74.170 + $71.660 + $69.300 + $1571.800 = $1938.28 百萬

**股權價值 (Equity Value) = 總企業價值 + 現金及約當現金 - 總債務**

**DCF 估值結果：**
由於計算出的股權價值為負值，這表明在這些假設下，公司的債務負擔過重，其未來現金流不足以覆蓋其債務，因此 DCF 模型在此情況下可能不適用於得出正向的股價。這也反映了公司高負債和目前盈利能力較弱的現實。

### **2.3 股息折現模型 (Dividend Discount Model, DDM)**

ILPT 支付股息，因此 DDM 適用。

**估值假設：**

*   **當前股息 (D0)：** $0.20 (年化)
*   **股息增長率 (g)：**
    *   考慮到公司目前盈利能力較弱（淨虧損），但 FFO 增長強勁，且產業前景積極，我們假設一個保守的長期股息增長率。
    *   ILPT 2025年Q4 股息為 $0.05，年化 $0.20。
    *   考慮到其高負債和負盈利，股息增長可能有限。我們假設一個較低的永續增長率，例如 1%。
*   **股權成本 (Ke)：** 12.45% (與 DCF 模型相同)

**DDM 計算 (永續增長模型)：**
P0 = D1 / (Ke - g)
其中 D1 = D0 * (1 + g)

D1 = $0.20 * (1 + 0.01) = $0.202
P0 = $0.202 / (0.1245 - 0.01)
P0 = $0.202 / 0.1145 = $1.76

**DDM 估值結果：**
*   每股估值: $1.76

**合理價位區間 (DDM)：** $1.76 (由於單一模型，且增長率假設較低，區間較窄)
