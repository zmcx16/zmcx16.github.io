### 2. 估值方法應用

由於 ILPT 是一家 REIT，其估值通常會更側重於 FFO 和物業價值，而非傳統的淨利潤。

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

由於 ILPT 的淨利潤為負，且作為 REIT，其自由現金流的計算可能需要調整。我們將使用自由現金流 (Free Cash Flow to Firm, FCFF) 或自由現金流 (Free Cash Flow to Equity, FCFE) 進行估值。鑑於公司的高負債，FCFF 可能更為合適，因為它代表了公司在支付所有營運費用和資本支出後，可供所有資本提供者（債權人和股東）分配的現金。

**估值假設：**

*   **預測期**: 5年。
*   **折現率 (WACC)**:
    *   **股權成本 (Cost of Equity, Ke)**: 使用資本資產定價模型 (CAPM)。
        *   無風險利率 (Rf): 參考美國10年期國債收益率。假設為4.2%。
        *   市場風險溢價 (MRP): 假設為5.5%。
        *   Beta: 根據搜尋結果，24個月 Beta 為0.87。
        *   Ke = Rf + Beta * MRP = 4.2% + 0.87 * 5.5% = 4.2% + 4.785% = 8.985%
    *   **債務成本 (Cost of Debt, Kd)**: 公司加權平均利率為5.47%。
    *   **稅率 (Tax Rate)**: 由於 REIT 的特殊性，通常不繳納企業所得稅，但為了 WACC 計算的完整性，我們可以使用一個名義稅率或行業平均稅率。考慮到其負利潤，實際稅負可能較低。我們假設有效稅率為20% (僅用於計算稅後債務成本)。
    *   **債務權重 (Weight of Debt, Wd)** 和 **股權權重 (Weight of Equity, We)**:
        *   市值 (Market Cap): 3.6829億美元。
        *   總債務 (Total Debt): 42.1億美元。
        *   總資本 = 市值 + 總債務 = 3.6829億 + 42.1億 = 45.7829億美元。
        *   Wd = 42.1億 / 45.7829億 = 91.95%
        *   We = 3.6829億 / 45.7829億 = 8.05%
    *   **WACC** = (We * Ke) + (Wd * Kd * (1 - Tax Rate))
        *   WACC = (0.0805 * 8.985%) + (0.9195 * 5.47% * (1 - 0.20))
        *   WACC = 0.00723 + (0.9195 * 5.47% * 0.80)
        *   WACC = 0.00723 + 0.04023 = 0.04746 = 4.75% (由於債務佔比極高且稅率調整，WACC 較低)

*   **自由現金流增長率 (FCFF Growth Rate)**:
    *   短期增長率 (未來5年): 考慮到 Q1 2024 FFO 增長19.4% 和租金上漲趨勢，以及電子商務帶來的需求，我們假設未來5年 FCFF 增長率為5%。
    *   長期增長率 (永續增長率, Terminal Growth Rate, g): 假設為2% (略高於長期通脹率，符合 REIT 的穩定增長特性)。

*   **起始自由現金流 (Base Year FCFF)**:
    *   根據搜尋結果，過去12個月的自由現金流為3903萬美元。 我們將以此作為基期 FCFF。

**DCF 計算步驟：**

1.  **預測期內的 FCFF**:
    *   Year 1 (2026): 39.03 * (1 + 0.05) = 40.98 百萬美元
    *   Year 2 (2027): 40.98 * (1 + 0.05) = 43.03 百萬美元
    *   Year 3 (2028): 43.03 * (1 + 0.05) = 45.18 百萬美元
    *   Year 4 (2029): 45.18 * (1 + 0.05) = 47.44 百萬美元
    *   Year 5 (2030): 47.44 * (1 + 0.05) = 49.81 百萬美元

2.  **計算預測期內 FCFF 的現值**:
    *   PV(FCFF1) = 40.98 / (1 + 0.0475)^1 = 39.12 百萬美元
    *   PV(FCFF2) = 43.03 / (1 + 0.0475)^2 = 39.02 百萬美元
    *   PV(FCFF3) = 45.18 / (1 + 0.0475)^3 = 38.93 百萬美元
    *   PV(FCFF4) = 47.44 / (1 + 0.0475)^4 = 38.84 百萬美元
    *   PV(FCFF5) = 49.81 / (1 + 0.0475)^5 = 38.75 百萬美元
    *   預測期內 FCFF 現值總和 = 39.12 + 39.02 + 38.93 + 38.84 + 38.75 = 194.66 百萬美元

3.  **計算終值 (Terminal Value, TV)**:
    *   TV = FCFF5 * (1 + g) / (WACC - g)
    *   TV = 49.81 * (1 + 0.02) / (0.0475 - 0.02)
    *   TV = 49.81 * 1.02 / 0.0275
    *   TV = 50.8062 / 0.0275 = 1847.50 百萬美元

4.  **計算終值的現值**:
    *   PV(TV) = TV / (1 + WACC)^5
    *   PV(TV) = 1847.50 / (1 + 0.0475)^5 = 1847.50 / 1.2616 = 1464.47 百萬美元

5.  **計算公司總價值 (Enterprise Value, EV)**:
    *   EV = 預測期內 FCFF 現值總和 + 終值的現值
    *   EV = 194.66 + 1464.47 = 1659.13 百萬美元

6.  **計算股權價值 (Equity Value)**:
    *   股權價值 = EV - 總債務 + 現金及約當現金
    *   總債務: 42.1億美元。
    *   現金及約當現金: 1.28億美元 (截至2024年3月31日)。
    *   股權價值 = 1659.13 - 4210 + 128 = -2422.87 百萬美元

**DCF 估值結果與合理價位區間：**

DCF 模型計算出的股權價值為負值，這表明在當前的假設下，公司的債務負擔過重，以至於其未來自由現金流無法覆蓋其債務。這是一個嚴重的警示信號，可能意味著公司面臨財務困境或需要進行大規模的債務重組。

**重要提示**: 由於 ILPT 的高負債和負淨利潤，傳統的 DCF 模型可能難以直接應用並得出正向的股權價值。REITs 通常更適合使用 FFO 或 AFFO (Adjusted Funds From Operations) 進行估值，或者基於資產淨值 (NAV) 進行估值。負的股權價值強烈暗示公司存在嚴重的財務風險。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

ILPT 支付股息，因此 DDM 適用。然而，其股息歷史顯示出波動性，且在2025年第二季度才大幅提高股息。 這使得預測穩定的股息增長率變得困難。

**估值假設：**

*   **當前股息 (D0)**: 年化股息為0.20美元/股。
*   **股息增長率 (g)**: 考慮到公司近期才大幅提高股息，且未來面臨債務再融資挑戰，我們假設一個保守的長期股息增長率為1%。
*   **股權成本 (Ke)**: 沿用 DCF 模型中的 8.985%。

**DDM 計算步驟 (戈登增長模型, Gordon Growth Model)**:

*   合理股價 = D0 * (1 + g) / (Ke - g)
*   合理股價 = 0.20 * (1 + 0.01) / (0.08985 - 0.01)
*   合理股價 = 0.20 * 1.01 / 0.07985
*   合理股價 = 0.202 / 0.07985 = 2.53 美元

**DDM 估值結果與合理價位區間：**

*   基於 DDM 的估值結果為 $2.53。

這個結果遠低於當前股價 $5.46，也遠低於分析師目標價 $6.85。這可能反映了市場對其股息增長的可持續性持謹慎態度，或者認為當前股息相對於其風險而言不夠吸引。

**綜合分析：**

1.  **DCF 模型結果為負股權價值**：這是一個非常強烈的警示信號。它表明在當前的現金流預期和高負債情況下，公司的未來營運現金流不足以償還其債務，股東價值為負。這可能是由於公司在過去的擴張（例如收購 Monmouth Real Estate）中承擔了過多的債務，導致財務槓桿過高。對於 REITs 而言，高負債是常見的，但如此高的債務水平和負的股權價值需要引起高度關注。
2.  **相對估值顯示潛在低估**：相對於行業平均的 P/FCF 和 P/FFO，ILPT 的當前股價顯得被低估。這表明市場可能因為其高負債和負淨利潤而給予了較低的估值倍數，但如果公司能夠持續改善 FFO 和 FCF，並有效管理債務，則存在價值回歸的潛力。
3.  **DDM 模型結果較低**：DDM 估值結果遠低於當前股價，這反映了市場對其股息增長可持續性的擔憂。儘管公司近期提高了股息，但其歷史股息波動較大，且財務狀況仍面臨挑戰。
4.  **分析師目標價與當前股價存在上漲空間**：分析師的平均目標價為 $6.85，高於當前股價 $5.46，這暗示了約25%的上漲空間。這與相對估值模型的部分結果相符，表明市場對其未來表現仍有一定期待。
5.  **REIT 特性與 FFO 重要性**：作為 REIT，FFO 是衡量其營運表現的關鍵指標。ILPT 的 Normalized FFO 呈現增長趨勢，且租賃活動強勁，出租率高，租金上漲，這些都是正面的營運信號。

**主要風險：**

*   **高負債**: 淨債務與總資產比率高達69.9%，且2026年有14億美元的合資企業債務到期，這是一個重大的再融資挑戰。 利率上升可能進一步增加其債務成本。
*   **負淨利潤**: 儘管 FFO 為正，但持續的淨虧損仍是財務健康的隱憂。
*   **股息可持續性**: 儘管近期股息有所提升，但其歷史波動性以及高負債背景下，股息的長期可持續性仍需觀察。

**綜合結論：**

ILPT 是一家營運表現良好（高出租率、租金上漲、FFO 增長）的工業物流 REIT，受益於電子商務的增長趨勢。然而，其極高的債務水平是其估值面臨的最大挑戰。DCF 模型給出的負股權價值強烈警示了其財務風險。相對估值模型則顯示公司可能被市場低估，特別是考慮到其 FFO 和 FCF 表現。DDM 模型則對其股息增長持謹慎態度。

**投資建議：**

鑑於不同估值模型之間存在顯著差異，且公司面臨高負債的重大風險，對 ILPT 的投資應持謹慎態度。

*   **看好者**可能會關注其強勁的營運基本面、FFO 增長以及相對於同業較低的 P/FCF 和 P/FFO 倍數，認為市場因其債務問題而過度懲罰了公司，存在價值回歸的機會。分析師的目標價也支持這一觀點。
*   **謹慎者**則會強調其極高的債務水平、負的股權價值以及2026年的再融資風險。在債務問題得到有效解決之前，這些風險可能會持續壓制股價。

**合理價位區間：**

考慮到當前股價 $5.46，分析師目標價 $6.85，以及相對估值模型給出的較高區間，但同時也考慮到 DCF 的負值和 DDM 的低值，一個較為保守的合理價位區間可能在 **$4.00 - $7.00** 之間。這個區間反映了市場對其營運潛力與財務風險之間的權衡。投資者應密切關注公司未來的債務管理策略、再融資進展以及 FFO 的持續增長情況。
