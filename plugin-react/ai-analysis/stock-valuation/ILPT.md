### 2. 估值方法與假設

由於 ILPT 是一家 REIT，我將重點關注以下估值方法：

*   **折現現金流模型 (DCF) - 調整為 FFO/AFFO：** 對於 REITs，通常使用 FFO 或 AFFO 作為現金流的替代指標，因為它們更能反映 REITs 的經營現金流。然而，由於缺乏詳細的 AFFO 歷史數據，我將嘗試使用 FFO 進行估算，並輔以傳統的 FCF 概念。
*   **相對估值 (Comparable Company Analysis)：** 使用 P/FFO、P/AFFO、P/B、P/S 等倍數與可比公司進行比較。
*   **股息折現模型 (DDM)：** 由於 REITs 通常派發高額股息，DDM 是一個重要的估值工具。

**估值假設 (基於最新數據和行業趨勢)：**

1.  **預測期間：** 5 年 (2026-2030)。
2.  **折現率 (WACC)：**
    *   **無風險利率 (Rf)：** 使用美國 10 年期國債收益率。目前約為 4.1%。
    *   **市場風險溢價 (MRP)：** 歷史平均約 5.5%。
    *   **Beta：** 由於 REITs 的特殊性，其 Beta 可能與一般股票不同。我將使用 1.0 作為基準，並在敏感性分析中調整。
    *   **股權成本 (Ke) = Rf + Beta * MRP**
    *   **債務成本 (Kd)：** 考慮 ILPT 的高負債率和當前利率環境，假設債務成本為 6.0%。
    *   **稅率 (t)：** REITs 通常不需要繳納企業所得稅，但為了 WACC 計算的完整性，我將使用一個名義稅率，例如 0% 或一個較低的有效稅率，或者直接使用股權成本作為折現率（如果公司幾乎沒有稅盾效應）。考慮到 REIT 的特殊性，我將在計算 WACC 時使用 0% 的有效稅率，因為其大部分利潤會以股息形式分配給股東。
    *   **資本結構：** 根據提供的數據，Debt/Eq 為 8.53。這意味著債務佔總資本的比例非常高。
        *   股權市值 (E) = 3.7263 億美元
        *   債務市值 (D) = 股權市值 * Debt/Eq = 3.7263 億美元 * 8.53 = 31.78 億美元
        *   總資本 (V) = E + D = 3.7263 億美元 + 31.78 億美元 = 35.5063 億美元
        *   股權佔比 (E/V) = 3.7263 / 35.5063 = 0.105
        *   債務佔比 (D/V) = 31.78 / 35.5063 = 0.895
    *   **WACC 計算：**
        *   Ke = 4.1% + 1.0 * 5.5% = 9.6%
        *   WACC = (E/V) * Ke + (D/V) * Kd * (1 - t)
        *   WACC = 0.105 * 9.6% + 0.895 * 6.0% * (1 - 0%) = 1.008% + 5.37% = 6.378%
        *   由於 ILPT 的負債率極高，且其盈利能力不穩定，WACC 可能會更高。我將使用 7.0% 作為基準 WACC，並在敏感性分析中考慮更高範圍。
3.  **長期增長率 (g)：** 考慮到工業 REITs 的穩定性以及 ILPT 的租賃活動，我將假設長期增長率為 2.0% - 2.5%。
4.  **FFO 增長率：** 根據 Seeking Alpha 的預測，2025 年 FFO 增長率為 76.54%，2026 年為 32.17%，2027 年為 16.67%。 之後我將逐步遞減至長期增長率。
5.  **股息增長率：** 考慮到 ILPT 股息歷史的不穩定性，我將在 DDM 中使用不同的增長率假設。

### 3. 估值模型計算

#### 3.1 折現現金流模型 (DCF) - 基於 FFO

由於 REITs 的特殊性，我將使用 FFO 作為現金流的代理。

**步驟 1：預測 FFO**

*   **基準 FFO (2025E)：** 0.95 美元/股
*   **流通股數：** 市值 / 股價 = 372.63 百萬美元 / 5.22 美元 = 71.38 百萬股 (近似值)
*   **總 FFO (2025E)：** 0.95 美元/股 * 71.38 百萬股 = 67.81 百萬美元

| 年度 | FFO 增長率 | 預測 FFO (百萬美元) |
| :--- | :--- | :--- |
| 2025 (基準) | - | 67.81 |
| 2026 | 32.17% | 67.81 * (1 + 0.3217) = 89.63 |
| 2027 | 16.67% | 89.63 * (1 + 0.1667) = 104.59 |
| 2028 | 10.00% (遞減) | 104.59 * (1 + 0.10) = 115.05 |
| 2029 | 6.00% (遞減) | 115.05 * (1 + 0.06) = 122.00 |
| 2030 | 3.00% (遞減至長期增長率) | 122.00 * (1 + 0.03) = 125.66 |

**步驟 2：計算終端價值 (Terminal Value, TV)**

*   假設 2030 年之後 FFO 以長期增長率 (g) 2.5% 永續增長。
*   TV = FFO (2031) / (WACC - g)
*   FFO (2031) = FFO (2030) * (1 + g) = 125.66 * (1 + 0.025) = 128.80 百萬美元
*   TV = 128.80 / (0.070 - 0.025) = 128.80 / 0.045 = 2862.22 百萬美元

**步驟 3：折現 FFO 和終端價值**

| 年度 | 預測 FFO (百萬美元) | 折現因子 (1 / (1 + WACC)^n) | 折現 FFO (百萬美元) |
| :--- | :--- | :--- | :--- |
| 2026 | 89.63 | 1 / (1 + 0.070)^1 = 0.9346 | 83.75 |
| 2027 | 104.59 | 1 / (1 + 0.070)^2 = 0.8734 | 91.36 |
| 2028 | 115.05 | 1 / (1 + 0.070)^3 = 0.8163 | 93.95 |
| 2029 | 122.00 | 1 / (1 + 0.070)^4 = 0.7629 | 93.04 |
| 2030 | 125.66 | 1 / (1 + 0.070)^5 = 0.7130 | 89.56 |
| 終端價值 (2030) | 2862.22 | 1 / (1 + 0.070)^5 = 0.7130 | 2040.80 |

**步驟 4：計算公司總價值和每股價值**

*   公司總價值 (Total Value) = 折現 FFO 總和 + 折現終端價值
*   公司總價值 = (83.75 + 91.36 + 93.95 + 93.04 + 89.56) + 2040.80 = 451.66 + 2040.80 = 2492.46 百萬美元
*   每股估值 = 公司總價值 / 流通股數 = 2492.46 百萬美元 / 71.38 百萬股 = **34.92 美元/股**

**DCF 估值結果：**

*   **合理價位：** 34.92 美元/股
*   **價位區間 (敏感性分析 - WACC 6.5%-7.5%, g 2.0%-3.0%)：**
    *   WACC 6.5%, g 2.5% -> 39.90 美元/股
    *   WACC 7.5%, g 2.5% -> 30.90 美元/股
    *   WACC 7.0%, g 2.0% -> 30.00 美元/股
    *   WACC 7.0%, g 3.0% -> 41.00 美元/股
    *   **合理價位區間：30.00 - 41.00 美元/股**

這個 DCF 估值結果遠高於當前股價和分析師目標價，這可能表明 FFO 預測的樂觀性，或者市場對 ILPT 的風險溢價更高，或者我的 WACC 假設過低。特別是 ILPT 的高負債和負盈利能力，使得傳統 FFO 估值可能存在較大偏差。

#### 3.3 股息折現模型 (Dividend Discount Model, DDM)

ILPT 派發股息，因此 DDM 適用。然而，其股息歷史不穩定，2022 年曾大幅削減股息，這使得預測股息增長率變得困難。

**步驟 1：確定當前股息和股息增長率**

*   **當前年化股息 (D0)：** 0.20 美元/股 (每季度 0.05 美元)
*   **股權成本 (Ke)：** 9.6% (來自 WACC 計算)

由於股息歷史不穩定，我將考慮兩種情景：

**情景 A：零增長模型 (如果股息保持穩定)**

*   估值 = D0 / Ke = 0.20 / 0.096 = **2.08 美元/股**

**情景 B：穩定增長模型 (假設未來股息能穩定增長)**

*   考慮到公司目標是改善財務狀況和租金增長，假設長期股息增長率 (g) 為 1.0% - 2.0%。
*   估值 = D1 / (Ke - g) = D0 * (1 + g) / (Ke - g)
*   如果 g = 1.0%：0.20 * (1 + 0.01) / (0.096 - 0.01) = 0.202 / 0.086 = **2.35 美元/股**
*   如果 g = 2.0%：0.20 * (1 + 0.02) / (0.096 - 0.02) = 0.204 / 0.076 = **2.68 美元/股**

**DDM 估值結果：**

*   **合理價位：** 2.08 - 2.68 美元/股
*   **價位區間：2.00 - 3.00 美元/股**

這個 DDM 估值結果遠低於當前股價，反映了 ILPT 較低的股息支付和不穩定的股息歷史，以及相對較高的股權成本。
