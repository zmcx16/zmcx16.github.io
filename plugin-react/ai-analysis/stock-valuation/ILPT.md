#### **估值假設概述**

在進行具體模型計算之前，我們需要設定一些核心假設：

2.  **折現率 (WACC / 必要報酬率)：**
    *   **無風險利率 (Rf)：** 我們使用美國 10 年期國債收益率，約 4.90%。
    *   **市場風險溢價 (MRP)：** 普遍使用的範圍為 5.0% - 6.0%，我們取 5.5%。

        *   Ke = 4.90% + 1.25 * 5.5% = 4.90% + 6.88% = **11.78%**
    *   **債務成本 (Kd)：** 根據 ILPT 最近的利息支出和總債務，假設約 6.5% - 7.0%。我們取 6.8%。
    *   **稅率 (T)：** 作為 REIT，如果滿足分紅要求，聯邦層面可免除公司所得稅，故有效稅率取 0% 用於 WACC 計算的稅盾部分。
    *   **資本結構 (D/V, E/V)：**
        *   市場股本價值 (E) = $4.50/股 * 6,750 萬股 = 約 $3.04 億
        *   淨債務 (D) = 總債務 - 現金及等價物。假設總債務約 $24 億，現金約 $5000 萬，則淨債務約 $23.5 億。
        *   企業價值 (V) = E + D = $3.04 億 + $23.5 億 = 約 $26.54 億
        *   E/V = $3.04 億 / $26.54 億 = 11.45%
        *   D/V = $23.5 億 / $26.54 億 = 88.55%
    *   **加權平均資本成本 (WACC) = (E/V) * Ke + (D/V) * Kd * (1 - T)**
        *   WACC = (0.1145 * 11.78%) + (0.8855 * 6.8% * (1 - 0%))
        *   WACC = 1.35% + 6.02% = **7.37%** (此 WACC 用於 FCFF 估值，但我們主要採用 FFO-based DCF 和股權折現，會主要使用 Ke)。

3.  **長期增長率 (g)：**

### **估值模型計算**

#### **模型一：淨資產價值 (Net Asset Value, NAV) 估值**

NAV 估值是 REITs 最常用的方法之一，它評估了公司房地產資產的市場價值減去所有負債後的股權價值。

1.  **穩定化淨營業收入 (Stabilized Net Operating Income, NOI)：**
    *   基於最新財報（假設年化 NOI 約 $2.00 億）。我們假設這個 NOI 是相對穩定的，或可實現的。
2.  **資本化率 (Capitalization Rate, Cap Rate)：**
    *   工業物業的 Cap Rate 因地點、物業質量等因素而異，目前市場範圍大約在 4.5% - 6.5%。考慮到 ILPT 的高槓桿和一些公司特定風險，我們採用稍高的 Cap Rate 範圍，例如 **6.0% - 6.5%**。
3.  **計算總房地產價值 (Gross Real Estate Value)：**
    *   低估值：$2.00 億 / 0.065 = $30.77 億
    *   高估值：$2.00 億 / 0.060 = $33.33 億
4.  **減去淨債務及其他負債：**
    *   淨債務：$23.5 億 (總債務 $24 億 - 現金 $0.5 億)
    *   其他淨負債：假設 $0.5 億 (例如：少數股東權益、或其他非核心資產負債淨額)
5.  **計算淨資產價值 (NAV)：**
    *   低估值 NAV = $30.77 億 - $23.5 億 - $0.5 億 = $6.77 億
    *   高估值 NAV = $33.33 億 - $23.5 億 - $0.5 億 = $9.33 億
6.  **NAV 每股價值：**
    *   低估值 NAV/股 = $6.77 億 / 6,750 萬股 = **$10.03**
    *   高估值 NAV/股 = $9.33 億 / 6,750 萬股 = **$13.82**

**NAV 估值結果：** $10.03 - $13.82 /股
**合理價位區間：** $10.00 - $14.00 /股

#### **模型二：股息折現模型 (Dividend Discount Model, DDM)**

ILPT 在 2023 年初大幅削減了股息，從每股 $0.25 下降至每股 $0.01（季度），即年化 $0.04。這使得基於穩定增長的 DDM 模型變得非常困難且敏感。我們將採用一個兩階段 DDM 來嘗試反映其當前困境和未來潛在的復甦。

1.  **假設：**
    *   股權成本 (Ke)：11.78%
    *   當前年化股息 (D0)：$0.04/股
    *   **階段一 (短期，3 年)：** 假設股息在初期保持不變或微幅增長，因為公司需要時間來穩定財務狀況和償還債務。我們假設前 3 年股息增長為 **0%**。
    *   **階段二 (長期，永續)：** 假設從第四年開始，股息以一個非常保守的長期增長率持續增長，例如 **1.5%**。
2.  **計算：**
    *   D1 = D0 * (1+0%) = $0.04
    *   D2 = D1 * (1+0%) = $0.04
    *   D3 = D2 * (1+0%) = $0.04
    *   D4 = D3 * (1+1.5%) = $0.04 * 1.015 = $0.0406
    *   **階段一 PV (PV1)：**
        *   PV(D1) = $0.04 / (1.1178)^1 = $0.0358
        *   PV(D2) = $0.04 / (1.1178)^2 = $0.0320
        *   PV(D3) = $0.04 / (1.1178)^3 = $0.0286
        *   PV1 = $0.0358 + $0.0320 + $0.0286 = $0.0964
    *   **永續價值 (Terminal Value, TV) 在第三年末：**
        *   TV3 = D4 / (Ke - g) = $0.0406 / (0.1178 - 0.015) = $0.0406 / 0.1028 = $0.3950
    *   **永續價值現值 (PV_TV)：**
        *   PV_TV = $0.3950 / (1.1178)^3 = $0.2854
    *   **DDM 估值 = PV1 + PV_TV = $0.0964 + $0.2854 = $0.3818**

**DDM 估值結果：** 約 $0.38 /股
**合理價位區間：** 由於 ILPT 股息剛被大幅削減，且未來不確定性高，此模型會得出極低的估值。這表明市場可能認為其當前股價並非僅由當前股息驅動，而是潛在的復甦或資產價值。因此，此模型在此情境下作為獨立估值方法的參考意義有限。它更傾向於說明如果股息持續低迷且增長緩慢，對股東的回報將非常低。

#### **模型三：折現現金流模型 (Discounted Cash Flow, DCF - FFO Based)**

對於 REITs，我們通常使用 FFO (Funds From Operations) 或 AFFO (Adjusted FFO) 作為現金流的代理，然後將其折現回現在。這裡我們使用 FFO 作為基礎，並假設一部分 FFO 可用於股權分配或再投資，類似於 FCFE。

1.  **假設：**
    *   股權成本 (Ke)：11.78%
    *   長期增長率 (g)：1.8%
    *   **FFO/股預測 (基於公司財報和分析師預期，假設近期復甦)：**
        *   當前 TTM FFO/股：約 $0.50 (考慮最近表現)
        *   第 1 年 (Y1)：$0.55 (溫和復甦)
        *   第 2 年 (Y2)：$0.60
        *   第 3 年 (Y3)：$0.65
        *   第 4 年 (Y4)：$0.68
        *   第 5 年 (Y5)：$0.70
    *   我們假設 FFO 是一個合理的 "Free Cash Flow to Equity" 代理，或者說 FFO 的高比例可以分配給股東。

2.  **計算：**
    *   **預測期內 FFO/股的現值：**
        *   Y1: $0.55 / (1.1178)^1 = $0.492
        *   Y2: $0.60 / (1.1178)^2 = $0.479
        *   Y3: $0.65 / (1.1178)^3 = $0.463
        *   Y4: $0.68 / (1.1178)^4 = $0.443
        *   Y5: $0.70 / (1.1178)^5 = $0.424
        *   **預測期 FFO 現值總和 = $0.492 + $0.479 + $0.463 + $0.443 + $0.424 = $2.301**
    *   **第 5 年末永續價值 (Terminal Value, TV)：**
        *   FFO_Y6 = FFO_Y5 * (1 + g) = $0.70 * (1 + 0.018) = $0.7126
        *   TV5 = FFO_Y6 / (Ke - g) = $0.7126 / (0.1178 - 0.018) = $0.7126 / 0.0998 = $7.14
    *   **永續價值的現值 (PV_TV)：**
        *   PV_TV = $7.14 / (1.1178)^5 = $4.321
    *   **DCF 估值 (每股) = 預測期 FFO 現值總和 + 永續價值的現值**
        *   DCF 估值 = $2.301 + $4.321 = **$6.622**

**DCF 估值結果：** 約 $6.62 /股
**合理價位區間：** $6.00 - $7.50 /股 (考慮增長率和折現率的敏感性)
