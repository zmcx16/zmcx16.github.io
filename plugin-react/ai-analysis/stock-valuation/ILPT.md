### 2. 估值分析

由於 ILPT 是一家 REIT，其盈利能力通常以 FFO (Funds From Operations) 而非傳統的 GAAP EPS 來衡量。因此，在估值模型中，我將更側重於現金流和 FFO。

**估值假設 (基於最新資訊和行業慣例)：**

*   **折現率 (WACC)：**
    *   **無風險利率 (Rf)：** 參考美國 10 年期國債收益率。假設為 4.20%。
    *   **市場風險溢價 (MRP)：** 假設為 5.5%。
    *   **Beta (β)：** 由於未提供，我將搜尋 ILPT 的 Beta 值。
    *   **股權成本 (Ke) = Rf + β * MRP**
    *   **債務成本 (Kd)：** 考慮到高槓桿和近期再融資，假設為 6.5%。
    *   **稅率 (t)：** REITs 通常享有特殊的稅務待遇，但為計算 WACC，我們需要一個有效的公司稅率。假設為 21% (美國公司稅率)，但由於 REITs 的特殊性，實際有效稅率可能較低。然而，由於 REITs 將大部分收入分配給股東以避免公司層面徵稅，因此在計算 WACC 時，通常會使用一個較低的有效稅率或直接使用稅前債務成本。為了簡化，我們將假設有效稅率為 0% (因為 REITs 的稅務結構)。
    *   **債務權重 (D/V) 和股權權重 (E/V)：**
        *   市值 (E) = 3.6063 億美元。
        *   總債務：Simply Wall St 提到淨債務與總資產比率為 69.9%。假設總資產約為市值 / P/B = 360.63M / 0.73 = 494.01M。那麼總債務約為 494.01M * 0.699 = 345.31M。
        *   總資本 (V) = 股權 + 債務 = 360.63M + 345.31M = 705.94M。
        *   E/V = 360.63M / 705.94M = 0.51。
        *   D/V = 345.31M / 705.94M = 0.49。

*   **長期增長率 (g)：** 考慮到產業趨勢和公司歷史表現，預計長期增長率為 2.0% - 2.5%。
*   **預測期間：** 5 年。

**獲取 Beta 值：**

ILPT 的 Beta 值在不同來源中有所差異：
*   iOCharts 顯示為 2.50。
*   MarketBeat 顯示為 2.48。
*   WallStreetZen 顯示為 1.16。

由於 Beta 值對 WACC 影響較大，且存在顯著差異，我將採用一個較為保守且居中的 Beta 值，例如 1.8。

**WACC 計算：**

*   Rf = 4.20%
*   MRP = 5.5%
*   Beta = 1.8
*   Ke = 4.20% + 1.8 * 5.5% = 4.20% + 9.9% = 14.1%
*   Kd = 6.5%
*   稅率 (t) = 0% (REITs 特殊性)
*   E/V = 0.51
*   D/V = 0.49

WACC = (E/V * Ke) + (D/V * Kd * (1 - t))
WACC = (0.51 * 14.1%) + (0.49 * 6.5% * (1 - 0%))
WACC = 7.191% + 3.185% = 10.376%

### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

對於 REITs，通常使用 Funds From Operations (FFO) 或 Adjusted Funds From Operations (AFFO) 作為現金流的替代指標。由於公司目前處於虧損狀態，且 FFO 數據相對穩定，我們將基於 FFO 進行估算。

**DCF 估值假設：**

*   **預測期間：** 5 年。
*   **折現率 (WACC)：** 10.4%。
*   **長期增長率 (g)：** 2.5%。
*   **起始 FFO：** 根據 Q3 2025 Normalized FFO 為 0.26 美元/股，年化約為 0.26 * 4 = 1.04 美元/股。Seeking Alpha 預計 2025 年 FFO 為 0.95 美元/股，2026 年為 1.26 美元/股，2027 年為 1.47 美元/股。我們將採用 Seeking Alpha 的 FFO 預測作為基礎。
*   **FFO 增長率：**
    *   2026 年：(1.26 - 0.95) / 0.95 = 32.63% (基於 Seeking Alpha 預測)
    *   2027 年：(1.47 - 1.26) / 1.26 = 16.67% (基於 Seeking Alpha 預測)
    *   2028-2030 年：假設 FFO 增長率逐漸趨於長期增長率。我們將假設 2028 年為 10%，2029 年為 7%，2030 年為 4%。

**DCF 計算步驟：**

1.  **計算預測期內的 FFO：**
    *   2025 年 FFO (基數) = 0.95 美元/股
    *   2026 年 FFO = 0.95 * (1 + 0.3263) = 1.26 美元/股
    *   2027 年 FFO = 1.26 * (1 + 0.1667) = 1.47 美元/股
    *   2028 年 FFO = 1.47 * (1 + 0.10) = 1.62 美元/股
    *   2029 年 FFO = 1.62 * (1 + 0.07) = 1.73 美元/股
    *   2030 年 FFO = 1.73 * (1 + 0.04) = 1.80 美元/股

2.  **計算預測期內 FFO 的現值 (PV of FFO)：**
    *   2026 年 PV = 1.26 / (1 + 0.104)^1 = 1.14 美元
    *   2027 年 PV = 1.47 / (1 + 0.104)^2 = 1.21 美元
    *   2028 年 PV = 1.62 / (1 + 0.104)^3 = 1.20 美元
    *   2029 年 PV = 1.73 / (1 + 0.104)^4 = 1.17 美元
    *   2030 年 PV = 1.80 / (1 + 0.104)^5 = 1.09 美元
    *   預測期 FFO 現值總和 = 1.14 + 1.21 + 1.20 + 1.17 + 1.09 = 5.81 美元

3.  **計算終值 (Terminal Value, TV)：**
    *   TV = (FFO_2030 * (1 + g)) / (WACC - g)
    *   TV = (1.80 * (1 + 0.025)) / (0.104 - 0.025)
    *   TV = (1.80 * 1.025) / 0.079
    *   TV = 1.845 / 0.079 = 23.35 美元

4.  **計算終值的現值 (PV of TV)：**
    *   PV of TV = TV / (1 + WACC)^5
    *   PV of TV = 23.35 / (1 + 0.104)^5
    *   PV of TV = 23.35 / 1.640 = 14.24 美元

5.  **計算每股內在價值 (Intrinsic Value per Share)：**
    *   內在價值 = 預測期 FFO 現值總和 + 終值的現值
    *   內在價值 = 5.81 美元 + 14.24 美元 = 20.05 美元

**DCF 估值結果：**

根據 DCF 模型，ILPT 的每股合理價值約為 **20.05 美元**。

**合理價位區間：**

考慮到 FFO 增長率和 WACC 的敏感性，我們可以設定一個區間：
*   若長期增長率為 2.0%，WACC 為 11%：內在價值可能較低。
*   若長期增長率為 3.0%，WACC 為 10%：內在價值可能較高。

基於上述計算，DCF 模型的合理價位區間約為 **18.00 - 22.00 美元**。

### 2.3 股息折現模型 (Dividend Discount Model, DDM)

ILPT 是一家支付股息的 REIT，因此 DDM 適用。然而，由於公司目前處於虧損狀態，且股息的可持續性受到質疑，DDM 的結果需要謹慎解讀。

**DDM 估值假設：**

*   **當前股息 (D0)：** 最新年化股息為 0.20 美元/股 (季度 0.05 美元)。
*   **股息增長率 (g)：** 考慮到公司虧損和高槓桿，儘管近期股息有所提升，但長期增長率應保守估計。我們將假設一個較低的長期股息增長率，例如 1.0% - 2.0%。
*   **股權成本 (Ke)：** 14.1% (來自 WACC 計算)。

**DDM 計算步驟 (戈登增長模型)：**

*   合理股價 = D0 * (1 + g) / (Ke - g)

**情境 1：股息增長率 g = 1.0%**
*   合理股價 = 0.20 * (1 + 0.01) / (0.141 - 0.01)
*   合理股價 = 0.20 * 1.01 / 0.131
*   合理股價 = 0.202 / 0.131 = 1.54 美元

**情境 2：股息增長率 g = 2.0%**
*   合理股價 = 0.20 * (1 + 0.02) / (0.141 - 0.02)
*   合理股價 = 0.20 * 1.02 / 0.121
*   合理股價 = 0.204 / 0.121 = 1.69 美元

**DDM 估值結果：**

根據 DDM 模型，ILPT 的每股合理價值約為 **1.54 美元至 1.69 美元**。

**合理價位區間：**

DDM 模型的合理價位區間約為 **1.50 - 1.70 美元**。

**DDM 模型的局限性：**

DDM 模型對增長率和折現率非常敏感。由於 ILPT 目前處於虧損狀態，且股息的可持續性存在不確定性，DDM 估值結果顯著低於當前股價和其他估值模型。這表明市場可能預期公司未來盈利能力將大幅改善，或者股息並非其主要價值驅動因素，或者市場對其高槓桿的風險定價不足。對於 REITs，FFO 通常是比股息更好的現金流衡量指標，因為股息可能來自借款或資產出售。

這種巨大的差異可能源於以下幾個原因：
*   **盈利能力差異：** 可比公司如 Prologis 和 EastGroup Properties 都是盈利且成熟的 REITs，其 FFO 穩定且增長可預期。而 ILPT 雖然 FFO 預計增長強勁，但仍處於 GAAP 虧損狀態，且歷史盈利能力不佳，這使得市場給予其較低的估值倍數。
*   **槓桿風險：** ILPT 的高槓桿是其面臨的主要風險之一，這可能導致投資者要求更高的風險溢價，從而壓低其估值。
*   **市場信心：** 儘管 ILPT 擁有優質的物業組合和穩定的租賃期限，但其過去的表現和財務結構可能影響了市場的信心。

**最終結論：**

儘管 DCF 和相對估值模型顯示 ILPT 的內在價值遠高於當前股價，但 DDM 模型和分析師目標價則相對保守。這種差異主要源於 ILPT 複雜的財務狀況：作為一家工業物流 REIT，它受益於強勁的市場趨勢和穩定的租賃收入，但同時面臨持續虧損和高槓桿的挑戰。

如果 ILPT 能夠成功執行其策略，改善盈利能力，並有效管理其高額債務，那麼其股價確實存在顯著的潛在上漲空間，以達到 DCF 和相對估值模型所暗示的水平。然而，這些模型中的 FFO 增長假設和折現率選擇對結果影響巨大，且公司實現這些增長目標的風險不容忽視。

考慮到 ILPT 的高槓桿、持續虧損的歷史以及股息可持續性的不確定性，我認為市場目前對其估值持謹慎態度是合理的。分析師的目標價 (6.85 美元) 可能更貼近市場對其近期潛力的共識。
