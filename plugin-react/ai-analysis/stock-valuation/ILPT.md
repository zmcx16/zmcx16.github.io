#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF) - 基於 FFO

對於 REITs，DCF 通常基於 FFO 或 AFFO，因為這些指標更能反映其營運現金流。由於我們有 Normalized FFO 的預期，我們將使用 Normalized FFO 作為現金流的代理。

**估值假設：**

    *   **股權成本 (Ke):** Ke = Rf + Beta * ERP

        *   E/V = 511.32 / 4721.32 ≈ 0.11

    *   **WACC 計算:** WACC = (E/V) * Ke + (D/V) * Kd * (1 - t)

    首先，搜尋 ILPT 的 Beta 值。
根據搜尋結果，ILPT 的 Beta 值在 1.19 到 2.47 之間。我們將採用一個相對保守的 Beta 值，例如 1.5。

市場對 ILPT 的高槓桿和過去的負面盈利能力可能給予了過高的風險折價。然而，隨著公司財務狀況的改善和營運表現的提升，這種折價有望收窄。
