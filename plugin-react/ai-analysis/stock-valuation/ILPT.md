### 2. 估值分析

由於 ILPT 是一家 REIT，其盈利能力通常以 FFO (Funds From Operations) 或 AFFO (Adjusted Funds From Operations) 來衡量，而非傳統的淨利潤。因此，P/E 比率為負數是常見現象。我們將重點關注基於 FFO/AFFO 的估值方法，並考慮其股息支付。

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

對於 REITs，DCF 通常基於 FFO 或 AFFO，並考慮資本支出、租戶改善費用等調整。由於缺乏詳細的 FFO/AFFO 預測數據，我們將嘗試基於 FFO 增長進行簡化 DCF 估值。

**估值假設：**

*   **預測期:** 5 年 (2026-2030)
*   **起始 FFO (2025 年):** 根據 Zacks 預期，2025 財年 FFO 為 $0.86/股。
*   **FFO 增長率:**
    *   短期 (2026-2027): 考慮到 2025 年第四季度 FFO 顯著增長 113% 以及產業趨勢預計 2026 年需求正常化，2027 年租金增長恢復通脹水平，我們假設 2026 年 FFO 增長率為 10%，2027 年為 8%。
    *   中期 (2028-2030): 隨著市場趨於成熟和競爭加劇，增長率將放緩。假設為 5%。
    *   長期 (永續增長率): 考慮到工業 REITs 的穩定性，但受限於經濟增長和產業成熟度，設定為 2.5%。
*   **折現率 (WACC):**
    *   **股權成本 (Cost of Equity):** 使用資本資產定價模型 (CAPM)。
        *   無風險利率 (Risk-Free Rate): 參考美國 10 年期國債收益率，假設為 4.5%。
        *   市場風險溢價 (Market Risk Premium): 假設為 5.5%。
        *   Beta: 根據提供的數據，我們沒有 Beta 值。通常 REITs 的 Beta 接近 1。我們將使用 1.0。
        *   股權成本 = 無風險利率 + Beta \* 市場風險溢價 = 4.5% + 1.0 \* 5.5% = 10.0%。
    *   **債務成本 (Cost of Debt):** ILPT 的加權平均利率為 5.47%。 假設稅率為 25%。
        *   稅後債務成本 = 5.47% \* (1 - 0.25) = 4.10%。
    *   **資本結構:**
        *   市值 (Market Cap): $361.93 百萬。
        *   淨債務 (Net Debt): 截至 2025 年第四季度為 40.3 億美元。
        *   總資本 = 市值 + 淨債務 = $361.93 百萬 + $4030 百萬 = $4391.93 百萬。
        *   股權佔比 = $361.93 / $4391.93 = 8.24%。
        *   債務佔比 = $4030 / $4391.93 = 91.76%。
        *   **WACC** = (股權佔比 \* 股權成本) + (債務佔比 \* 稅後債務成本)
            = (0.0824 \* 10.0%) + (0.9176 \* 4.10%)
            = 0.00824 + 0.03762 = 0.04586 = **4.59%**。
            *注意：由於 ILPT 的債務佔比非常高，WACC 會受到顯著影響。這個 WACC 值相對較低，主要是因為債務成本較低且佔比高。*

**DCF 計算 (基於 FFO/股):**

| 年份 | 2025 (基準) | 2026 | 2027 | 2028 | 2029 | 2030 |
| :--- | :---------- | :--- | :--- | :--- | :--- | :--- |
| FFO/股 | $0.86 | $0.95 (0.86 * 1.10) | $1.03 (0.95 * 1.08) | $1.08 (1.03 * 1.05) | $1.13 (1.08 * 1.05) | $1.19 (1.13 * 1.05) |
| 折現因子 (WACC=4.59%) | | 0.9561 | 0.9141 | 0.8739 | 0.8355 | 0.7988 |
| 折現 FFO/股 | | $0.91 | $0.94 | $0.95 | $0.94 | $0.95 |

*   **預測期 FFO 現值總和:** $0.91 + $0.94 + $0.95 + $0.94 + $0.95 = $4.69

*   **永續價值 (Terminal Value) 計算:**
    *   永續價值 FFO (2031 年) = 2030 年 FFO \* (1 + 長期增長率) = $1.19 \* (1 + 0.025) = $1.22
    *   永續價值 = 永續價值 FFO / (WACC - 長期增長率)
        = $1.22 / (0.0459 - 0.025) = $1.22 / 0.0209 = $58.37
    *   永續價值現值 = 永續價值 / (1 + WACC)^5 = $58.37 / (1 + 0.0459)^5 = $58.37 / 1.251 = $46.66

*   **DCF 估值 (每股):** 預測期 FFO 現值總和 + 永續價值現值 = $4.69 + $46.66 = **$51.35**

**DCF 估值結果：**

基於上述假設，DCF 模型得出的 ILPT 每股估值為 **$51.35**。這個結果遠高於當前股價，主要受到高債務佔比導致的低 WACC 和相對樂觀的 FFO 增長假設影響。

**合理價位區間：**

考慮到假設的敏感性，特別是 WACC 和長期增長率，我們將調整這些參數來得出一個區間：

*   **WACC 敏感性:**
    *   WACC 提高 1% (至 5.59%): 永續價值現值會降低，估值會下降。
    *   WACC 降低 1% (至 3.59%): 永續價值現值會提高，估值會上升。
*   **長期增長率敏感性:**
    *   長期增長率提高 0.5% (至 3.0%): 永續價值會提高。
    *   長期增長率降低 0.5% (至 2.0%): 永續價值會降低。

由於 REITs 的 FFO 波動性，以及 ILPT 過去的股息下降歷史，這個 DCF 估值可能過於樂觀。高債務水平也增加了風險。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

ILPT 支付股息，因此 DDM 適用。然而，其股息歷史顯示過去五年下降了 -63.56%，這使得股息增長率的預測變得複雜。公司在 2025 年將年化股息從 $0.04 提高到 $0.20，這是一個積極信號。

**估值假設：**

*   **當前股息 (D0):** $0.20 (年化)
*   **股息增長率 (g):** 考慮到公司最近的股息增長和產業前景，但也要考慮過去的波動性。我們假設一個保守的長期股息增長率為 2%。
*   **股權成本 (Ke):** 沿用 DCF 模型中的 10.0%。

**DDM 計算 (戈登增長模型):**

*   DDM 估值 = D0 \* (1 + g) / (Ke - g)
    = $0.20 \* (1 + 0.02) / (0.10 - 0.02)
    = $0.20 \* 1.02 / 0.08
    = $0.204 / 0.08 = **$2.55**

**DDM 估值結果：**

基於 DDM 模型，ILPT 的每股估值為 **$2.55**。這個結果遠低於當前股價，反映了相對較低的股息支付和保守的增長假設。

**合理價位區間：**

如果股息增長率能達到 3%，估值為 $0.20 * 1.03 / (0.10 - 0.03) = $0.206 / 0.07 = $2.94。
如果股息增長率僅為 1%，估值為 $0.20 * 1.01 / (0.10 - 0.01) = $0.202 / 0.09 = $2.24。

DDM 估值對增長率非常敏感。考慮到 ILPT 過去的股息削減歷史，以及其高槓桿率，DDM 可能反映了市場對其未來股息穩定性和增長潛力的謹慎態度。

3.  **DDM 模型 ($2.55):** 這個估值遠低於當前股價，甚至低於分析師的最低目標價。DDM 模型的結果強調了 ILPT 較低的股息支付和過去股息的不穩定性。對於尋求穩定股息收入的投資者來說，DDM 可能更真實地反映了其價值。
4.  **當前股價 ($5.43) 與分析師目標價 ($7.70):** 當前股價介於 DDM 估值和分析師目標價之間。分析師目標價通常會考慮公司的增長潛力、產業趨勢以及管理層的戰略。

**綜合結論：**

ILPT 的估值分析呈現出顯著的分歧，這反映了其作為一家高槓桿 REIT 的複雜性。

*   **高槓桿風險:** ILPT 擁有非常高的債務權益比 (8.59) 和長期債務權益比 (5.7)，以及高淨債務與總資產比率 (69%)。這使得公司對利率變化和經濟下行非常敏感。雖然公司已將部分浮動利率債務再融資為固定利率，但高額債務仍然是其最大的風險因素。
*   **盈利能力與 FFO 質量:** 儘管 2025 年第四季度 Normalized FFO 實現了顯著增長，但公司仍處於淨虧損狀態，且過去的盈利和股息表現不穩定。REITs 的 FFO 雖然重要，但投資者也應關注其產生可分配現金流的能力。
*   **產業前景:** 工業 REITs 受益於電子商務的增長，長期需求前景良好。然而，短期內供應增加和經濟不確定性帶來了挑戰。ILPT 的高入住率和租金增長是積極信號。
*   **估值差異:**
    *   DCF 和 EV/EBITDA 估值因其高槓桿和對未來 FFO/EBITDA 的樂觀假設而顯得過高，可能未能充分考慮其風險。
    *   DDM 估值則過於悲觀，可能過度強調了其歷史股息不穩定性，而忽略了其資產價值和潛在的 FFO 增長。
    *   P/FFO 估值提供了一個相對合理的參考，表明如果 ILPT 能夠改善其財務狀況並獲得市場對其 FFO 的更高認可，其股價有潛力。

**最終判斷：**

考慮到 ILPT 的高槓桿風險、過去盈利和股息的不穩定性，以及不同估值模型之間的巨大差異，我認為當前股價 $5.43 附近可能反映了市場對其風險和潛力的權衡。

*   **如果公司能夠持續改善其 FFO 表現，有效管理債務，並維持穩定的股息增長，那麼其股價有潛力向分析師目標價 $7.70 靠攏，甚至更高 (接近 P/FFO 估值 $19.65)。**
*   然而，如果宏觀經濟環境惡化，利率進一步上升，或者公司無法有效降低槓桿，那麼其股價可能面臨下行壓力，甚至接近 DDM 估值 $2.55。

**建議：**

對於 ILPT，投資者應密切關注其未來的 FFO 報告、債務管理策略以及宏觀經濟環境對工業房地產市場的影響。由於其高槓桿和歷史波動性，這是一項風險較高的投資。在考慮投資之前，建議進行更深入的盡職調查，特別是關於其債務到期時間表、再融資能力以及 FFO 到可分配現金流的轉換情況。
