    *   P0 = $0.1867 + $0.1744 + $0.1628 + $1.5893
    *   P0 = $2.1132

**DDM 模型估值結果：**

*   **每股合理價位：** $2.11。
*   **合理價位區間：** 考慮到股息增長假設的敏感性，可以設定在 $1.80 - $2.50。

**DDM 模型的注意事項：**
*   DDM 對股息增長率和股權成本的假設非常敏感。
*   ILPT 過去股息增長不穩定，近期大幅提升，使得未來增長率的預測存在較大不確定性。
*   對於虧損公司，股息的可持續性需要密切關注。

*   **DDM 模型 ($2.11)：** 顯著低於當前股價。這可能反映了市場對 ILPT 股息增長的可持續性持謹慎態度，或者模型中使用的長期股息增長率過於保守。儘管近期股息大幅提升，但過去五年平均為負的歷史數據增加了不確定性。

**綜合結論：**

ILPT 的估值結果存在顯著差異，這凸顯了其業務複雜性和高槓桿帶來的挑戰。

2.  **DCF 估值過高，可能需要更精細的 REITs 專用現金流模型 (如 FFO/AFFO) 和更保守的假設。** 簡化的 FCFF 模型可能未能充分捕捉 REITs 的資本結構和現金流特性。

4.  **DDM 估值較低，反映了對股息可持續性和增長前景的擔憂。** 儘管近期股息增加，但其歷史不穩定性以及高槓桿可能限制未來的股息增長。

**投資建議：**

考慮到 ILPT 目前的負盈利能力、高槓桿率以及估值模型之間的顯著差異，投資 ILPT 存在較高風險。

*   **潛在利好：** 工業 REITs 行業前景良好、租金上漲、高入住率、近期股息大幅提升、分析師普遍給予「買入/持有」評級且目標價高於當前股價。
