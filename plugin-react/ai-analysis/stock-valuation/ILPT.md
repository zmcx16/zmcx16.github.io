*   D1 = D0 * (1 + g) = $0.20 * (1 + 0.025) = $0.205

這個 DDM 估值結果遠低於當前股價，這表明市場可能預期更高的股息增長，或者 DDM 模型對於 ILPT 這種有高負債和盈利波動的 REIT 不夠適用，或者我的股息增長率假設過於保守。

**重新審視 DDM：**
如果市場預期 ILPT 的股息增長率更高，例如 5% (與 FFO 增長率一致)，那麼：
*   D1 = $0.20 * (1 + 0.05) = $0.21
*   P = $0.21 / (0.1275 - 0.05) = $0.21 / 0.0775 = $2.71/股

即使是 5% 的增長率，DDM 估值仍然很低。這可能是因為 ILPT 的股息支付率相對較高，且其盈利能力（淨虧損）並不能完全支持股息的持續高增長。此外，高槓桿也增加了股息支付的風險。
