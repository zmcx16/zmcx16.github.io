### **2.1 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型需要預測未來自由現金流 (Free Cash Flow, FCF)。由於缺乏詳細的歷史 FCF 數據和未來營收、成本、資本支出等詳細預測，我們將基於提供的數據和最新資訊進行合理推斷。

    *   之後逐年遞減，2027 年 4%，2028 年 3%，2029 年 2.5%，2030 年 2%。
*   **營業利潤率 (Operating Margin):** 提供的營業利潤率為 26.85%。我們假設在預測期內保持在 25% 左右，反映成本控制和市場波動。

*   **資本支出 (Capital Expenditure, CapEx):** 2025 年資本支出指導為 54-57 億美元。我們假設未來幾年 CapEx 保持在 55 億美元左右，以支持增長和維護。

*   **營運資本變動 (Change in Working Capital):** 假設為營收的 1% (適度增長)。

1.  **計算稅後營業利潤 (NOPAT) = 營業收入 * 營業利潤率 * (1 - 稅率)**

3.  **計算終值 (Terminal Value, TV) = (FCF_last_year * (1 + g)) / (WACC - g)**

### **2.3 股息折現模型 (Dividend Discount Model, DDM)**

VALE 擁有 7.38% 的高股息率，這表明 DDM 是一個適用的估值方法。

**假設：**

*   **當前股息 (D0):** 股息率 7.38% * 當前股價 14.93 美元 = 1.10 美元/股。
*   **股息增長率 (g):** 考慮到礦業的週期性和公司對基礎金屬的投資，我們假設短期內股息增長率較低，長期穩定增長。
    *   第一階段 (2026-2028): 2%
    *   第二階段 (2029 之後): 1.5% (永續增長率)
*   **股權成本 (Ke):** 11.1% (與 DCF 模型相同)。

**計算步驟 (兩階段 DDM):**

1.  **計算預測期內的股息。**
2.  **計算預測期末的終值 (Terminal Value of Dividends)。**
3.  **將所有股息和終值折現回現值。**

| 年度 | D0 | 股息增長率 | 預計股息 (Dt) | 折現因子 (11.1%) | 股息現值 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 2025 | 1.10 | - | - | - | - |
| 2026 | | 2.0% | 1.10 * 1.02 = 1.122 | 0.900 | 1.010 |
| 2027 | | 2.0% | 1.122 * 1.02 = 1.144 | 0.810 | 0.927 |
| 2028 | | 2.0% | 1.144 * 1.02 = 1.167 | 0.729 | 0.851 |

*   **2028 年末的終值 (Terminal Value) 計算 (使用永續增長率 1.5%):**
    *   D2029 = 1.167 * 1.015 = 1.1845
    *   TV_2028 = D2029 / (Ke - g_long_term) = 1.1845 / (0.111 - 0.015) = 1.1845 / 0.096 = 12.3385 美元
*   **終值現值 (Present Value of Terminal Value) = 12.3385 / (1 + 0.111)^3 = 12.3385 / 1.374 = 8.980 美元**

**每股估值 = 股息現值總和 + 終值現值**
每股估值 = (1.010 + 0.927 + 0.851) + 8.980 = 2.788 + 8.980 = **11.768 美元/股**
