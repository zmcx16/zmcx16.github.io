### 3. 股息折現模型 (Dividend Discount Model, DDM)

VALE 是一家支付股息的公司，股息率為 6.62%。我們可以使用 DDM 進行估值。

**關鍵假設：**

*   **當前每股股息 (D0)：** 16.30 美元 \* 0.0662 = 1.079 美元
*   **股息增長率 (g)：** 考慮到鐵礦石價格的波動性和公司未來的資本支出，我們假設股息在短期內保持穩定，長期以較低速度增長。
    *   短期 (未來 3 年)：0% (考慮到市場不確定性和公司可能優先用於資本支出)
    *   長期 (永續增長率)：1.5% (低於終值增長率，更保守)
*   **股權成本 (Ke)：** 10.8% (來自 WACC 計算)

**估算步驟：**

由於我們假設短期股息不增長，我們可以使用戈登增長模型 (Gordon Growth Model) 進行估值。

*   **DDM 估值 = D1 / (Ke - g)**

    *   DDM 估值 = 1.095 美元 / (0.108 - 0.015)
    *   DDM 估值 = 1.095 美元 / 0.093 = **11.77 美元/股**

**DDM 模型估值結果：**

*   **合理價位區間：** 11.00 美元 - 12.50 美元
