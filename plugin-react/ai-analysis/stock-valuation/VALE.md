## VALE (淡水河谷) 股票估值分析報告

**報告日期:** 2024年5月14日
**目標公司:** VALE S.A. (NYSE: VALE)
**當前股價 (截至2024年5月14日收盤):** $12.45 USD

### 1. 公司概覽

淡水河谷 (VALE S.A.) 是一家全球性的多元化礦業公司，總部位於巴西。它是全球最大的鐵礦石和鎳生產商之一，也是重要的銅、煤和鉑族金屬生產商。VALE 的營運高度依賴大宗商品價格，特別是鐵礦石價格，這使其財務表現具有顯著的週期性。

### 2. 估值假設 (共同及一般性假設)

在進行估值之前，我們需要設定一系列關鍵假設，這些假設將在不同模型中應用或作為基礎。

*   **基準數據:** 主要基於VALE截至2024年3月31日的第一季度財報數據，以及2023年全年財務報告。
*   **預測期間 (DCF):** 5年 (2024 - 2028年)，之後進入永續增長階段。
*   **無風險利率 (Risk-Free Rate, Rf):** 採用美國10年期國債殖利率，約 **4.45%** (截至2024年5月14日)。
*   **市場風險溢酬 (Market Risk Premium, MRP):** 採用普遍接受的長期平均值，約 **5.50%**。
*   **VALE Beta 值:** 根據Yahoo Finance及Bloomberg數據，VALE的Beta值約為 **1.35**。這反映了其相對於整體市場較高的波動性，與其週期性行業特性相符。
*   **稅率 (Tax Rate):** 根據VALE近年財報的有效稅率，我們採用平均值 **22%**。
*   **永續增長率 (Terminal Growth Rate, g):** 考慮到VALE的規模和成熟度以及大宗商品行業的週期性，我們假設其長期永續增長率為 **2.0%** (略低於全球長期GDP增長率)。

#### 2.1 加權平均資本成本 (Weighted Average Cost of Capital, WACC) 計算

WACC 是DCF模型中的折現率，代表了公司為其資產融資的平均成本。

**2.1.1 股權成本 (Cost of Equity, Ke)**
採用資本資產定價模型 (CAPM)：
$Ke = Rf + Beta \times MRP$
$Ke = 4.45\% + 1.35 \times 5.50\%$
$Ke = 4.45\% + 7.43\%$
$Ke = 11.88\%$

**2.1.2 債務成本 (Cost of Debt, Kd)**
*   **VALE 債務利率估計:** VALE的平均債務利率約為6.0-7.0%。我們估計其稅前債務成本為 **6.50%**。
*   **稅後債務成本:**
    $Kd (after-tax) = Kd (pre-tax) \times (1 - Tax Rate)$
    $Kd (after-tax) = 6.50\% \times (1 - 0.22)$
    $Kd (after-tax) = 6.50\% \times 0.78$
    $Kd (after-tax) = 5.07\%$

**2.1.3 資本結構 (Capital Structure)**
根據VALE最新財報，我們估計其市場價值債務和股權比例如下：
*   **市場總市值 (Market Cap):** 約 $55.0 B
*   **淨債務 (Net Debt):** 約 $10.0 B (總債務 - 現金及約當現金)
*   **債務佔總資本比重 (Weight of Debt, Wd):** $10.0B / ($10.0B + $55.0B) = 15.38\%$
*   **股權佔總資本比重 (Weight of Equity, We):** $55.0B / ($10.0B + $55.0B) = 84.62\%$

**2.1.4 WACC 計算**
$WACC = (We \times Ke) + (Wd \times Kd (after-tax))$
$WACC = (0.8462 \times 11.88\%) + (0.1538 \times 5.07\%)$
$WACC = 10.05\% + 0.78\%$
$WACC = 10.83\%$

**綜合WACC假設為：10.83%**

### 5. 股息折現模型 (Dividend Discount Model, DDM)

VALE 是一家成熟的、有穩健股息支付歷史的公司。然而，由於其盈利能力與大宗商品價格高度相關，其股息支付的穩定性和增長預測難度較大。因此，我們將採用一個兩階段 DDM 模型來應對這種情況，並強調其敏感性。

**5.1 關鍵假設**

*   **當前股息 (D0):** VALE 過去12個月股息約 $0.80 - $1.00 USD/股 (高度波動，取中間值 $0.90 USD)。
*   **股權成本 (Ke):** 沿用DCF模型中的 Ke = 11.88%。
*   **階段一 (短期高增長):** 預計未來 3 年 (2024-2026) 股息增長率為 **3.0%**。考慮到商品價格的潛在波動和公司再投資需求，這個增長率較為保守。
*   **階段二 (永續增長):** 之後，股息將以永續增長率 **2.0%** 增長 (與DCF的永續增長率一致)。

**5.2 股息預測**

*   $D_0 = $0.90
*   $D_1 = D_0 \times (1 + g_1) = 0.90 \times (1 + 0.03) = 0.927$
*   $D_2 = D_1 \times (1 + g_1) = 0.927 \times (1 + 0.03) = 0.955$
*   $D_3 = D_2 \times (1 + g_1) = 0.955 \times (1 + 0.03) = 0.983$
*   $D_4 = D_3 \times (1 + g_2) = 0.983 \times (1 + 0.02) = 1.003$

**5.3 估值計算**

*   **階段一現值:**
    *   $PV(D_1) = D_1 / (1 + Ke)^1 = 0.927 / (1 + 0.1188)^1 = 0.828$
    *   $PV(D_2) = D_2 / (1 + Ke)^2 = 0.955 / (1 + 0.1188)^2 = 0.761$
    *   $PV(D_3) = D_3 / (1 + Ke)^3 = 0.983 / (1 + 0.1188)^3 = 0.701$
    *   **階段一總現值 = 0.828 + 0.761 + 0.701 = 2.290**

*   **終端價值 (在第三年末):**
    $P_3 = D_4 / (Ke - g_2) = 1.003 / (0.1188 - 0.02) = 1.003 / 0.0988 = 10.152$
    **終端價值現值:**
    $PV(P_3) = P_3 / (1 + Ke)^3 = 10.152 / (1 + 0.1188)^3 = 10.152 / 1.401 = 7.246$

*   **DDM 估值結果 (每股):** $2.290 + 7.246 = **$9.54 USD**

**5.4 DDM 模型的適用性與局限性**
對於VALE這類高度週期性且股息波動大的公司，DDM模型通常較不適用。預測穩定的股息增長率極具挑戰性。此模型的結果應被視為一個相對保守的參考，特別是考慮到其股息支付可能因未來盈餘變化而調整。目前的估值結果遠低於DCF和相對估值，這可能反映了DDM模型對穩定性和成長性假設的嚴格要求。

**DDM 合理價位區間: $9.00 - $11.00 USD** (考慮到對增長率和Ke的敏感性)

*   **相對估值** 提供了市場對同類公司的估值參考。P/E 倍數結果偏高 ($17.53)，這可能部分受BHP較高P/E拉高平均值影響；EV/EBITDA 倍數結果較低 ($13.90)，與DCF結果更接近。中位數EV/EBITDA 倍數通常對礦業公司更具參考價值，因為它消除了資本結構和稅率差異的影響。

**6.2 綜合結論**

考量VALE的業務特性和各估值模型的優缺點：

*   DCF模型提供了較為全面的內在價值評估，其假設中性且合理。
*   相對估值中的EV/EBITDA倍數估值與DCF結果更為吻合，P/E倍數受個別可比公司數據影響較大。
*   DDM模型因VALE股息的波動性，其估值結果應被視為下限或保守情境。

綜合來看，我們認為 **DCF模型和EV/EBITDA相對估值** 更能反映VALE的價值。當前VALE股價為 $12.45 USD，低於大多數模型的估值結果。

**綜合合理價位區間: $14.00 - $16.00 USD**

基於上述分析，VALE的目前股價 ($12.45) 似乎略微被低估。其潛在價值可能在 **$14.00 到 $16.00 USD** 之間。

**6.3 簡要與同產業可比公司的估值比較**

**6.4 風險因素**

*   **大宗商品價格波動:** 鐵礦石、鎳、銅等價格的劇烈波動直接影響VALE的營收和盈利能力。

*   **巴西政治與經濟風險:** 巴西的政治不確定性、貨幣波動和監管變化可能影響VALE的營運。
*   **全球經濟增長:** 特別是中國的需求，對VALE的產品銷售有巨大影響。
*   **營運成本:** 能源、勞動力和物流成本的上升可能侵蝕利潤。
