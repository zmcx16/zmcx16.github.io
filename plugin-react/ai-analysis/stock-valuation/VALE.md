#### 2.3. 股息折現模型 (Dividend Discount Model, DDM)

**估值假設：**

*   **股息收益率 (Dividend %):** 3.67% (來自最新數據)

*   **股息增長率 (g)：**
    *   由於礦業的周期性，以及鐵礦石價格預期下降，我們將採用兩階段DDM。

*   **股權成本 (Ke)：** 6.915% (來自DCF模型計算)。

**計算：**

*   **預測股息：**
    *   D1 = D0 * (1 + g1) = 0.621 * (1 + 0.03) = 0.640 美元
    *   D2 = D1 * (1 + g1) = 0.640 * (1 + 0.03) = 0.659 美元
    *   D3 = D2 * (1 + g1) = 0.659 * (1 + 0.03) = 0.679 美元
