| **預測期股息現值總和** |                      |                      | **2.698 美元**     |

**終值 (Terminal Value) - 永續增長階段：**

*   D6 = D5 \* (1 + 長期增長率) = 0.739 \* (1 + 0.015) = 0.750 美元
*   TV = D6 / (Ke - 長期增長率)
*   TV = 0.750 / (0.0969 - 0.015) = 0.750 / 0.0819 = 9.157 美元
*   **終值現值 (PV of TV):** 9.157 / (1 + 0.0969)^5 = 9.157 / 1.586 = **5.774 美元**

**每股估值 (Intrinsic Value per Share):**

*   每股估值 = 預測期股息現值總和 + 終值現值
*   每股估值 = 2.698 美元 + 5.774 美元 = **8.47 美元**

**DDM 模型估值結果：**

*   **合理價位：8.47 美元**
*   **價位區間：7.50 - 9.50 美元**

**注意：** DDM 估值結果顯著低於當前股價和 DCF/CCA 估值。這可能反映了 DDM 對股息增長率的敏感性，以及 VALE 的股息政策可能不完全反映其全部盈利能力，或者市場對其未來增長和盈利的預期高於我在此模型中使用的保守股息增長假設。特別是，如果 2025 年 16% 的股息收益率是基於特殊股息或一次性派發，那麼使用較低的長期增長率會導致較低的估值。
