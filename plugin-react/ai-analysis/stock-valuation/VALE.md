### 3. 股息折現模型 (Dividend Discount Model, DDM)

VALE 具有較高的股息率 (7.74%)，因此 DDM 適用。

**估值假設:**

*   **當前股息 (D0):**
    *   股價 = $13.90
    *   股息率 = 7.74%
    *   D0 = $13.90 \* 0.0774 = $1.076
*   **股息增長率 (g):** 考慮到鐵礦石價格壓力及公司回購股票的策略，我們預計股息增長將較為溫和。
    *   短期 (未來 3 年): 3%
    *   長期 (永續增長): 1.5% (與 DCF 的長期增長率保持一致，且考慮到行業的成熟性)
*   **所需報酬率 (Required Rate of Return / Cost of Equity, Ke):** 我們使用之前計算的股權成本，並考慮一個範圍。
    *   Ke = 9.5% (基於 CAPM 計算)
    *   我們將使用 9.0% - 10.0% 的範圍。

**DDM 估值計算 (兩階段 DDM):**

**階段一：預測期 (3 年，股息增長率 3%)**

*   D1 = D0 \* (1 + 0.03) = $1.076 \* 1.03 = $1.108
*   D2 = D1 \* (1 + 0.03) = $1.108 \* 1.03 = $1.141
*   D3 = D2 \* (1 + 0.03) = $1.141 \* 1.03 = $1.175

**階段二：終端價值 (從第 4 年開始永續增長 1.5%)**

*   D4 = D3 \* (1 + 0.015) = $1.175 \* 1.015 = $1.193
*   終端價值 (TV) = D4 / (Ke - g_long) = $1.193 / (0.095 - 0.015) = $1.193 / 0.08 = $14.91

**計算現值 (Ke = 9.5%):**

*   D1 現值 = $1.108 / (1 + 0.095)^1 = $1.012
*   D2 現值 = $1.141 / (1 + 0.095)^2 = $0.955
*   D3 現值 = $1.175 / (1 + 0.095)^3 = $0.899
*   TV 現值 = $14.91 / (1 + 0.095)^3 = $14.91 / 1.312 = $11.364

**DDM 估值 = $1.012 + $0.955 + $0.899 + $11.364 = $14.23**
