### **估值方法與計算**

#### **1. 加權平均資本成本 (Weighted Average Cost of Capital, WACC) 計算**

WACC 是折現現金流模型中的關鍵折現率。我們將使用資本資產定價模型 (CAPM) 計算股權成本 (Ke)，然後結合債務成本 (Kd) 和資本結構計算 WACC。

*   **股權成本 (Cost of Equity, Ke):**
    *   無風險利率 (Risk-Free Rate, Rf): 採用巴西 10 年期政府債券收益率 13.7%。
    *   貝塔 (Beta, β): 採用 0.78。
    *   市場風險溢價 (Equity Risk Premium, ERP): 考慮到巴西作為新興市場的風險，採用 6.0% (高於 Alpha Spread 提供的 4.12%)。
    *   **Ke = Rf + β \* ERP** = 13.7% + 0.78 \* 6.0% = 13.7% + 4.68% = **18.38%**

*   **債務成本 (Cost of Debt, Kd):**
    *   Alpha Spread 提供了 VALE3 (巴西上市股票) 的稅後債務成本為 14.97%。我們將採用此數據。
    *   **稅後債務成本 (Kd_after_tax) = 14.97%**

*   **資本結構權重:**
    *   Alpha Spread 提供的債務權重 (Wd) 為 23.66%。
    *   股權權重 (We) = 1 - Wd = 1 - 0.2366 = 0.7634

*   **WACC = (We \* Ke) + (Wd \* Kd_after_tax)**
    *   WACC = (0.7634 \* 18.38%) + (0.2366 \* 14.97%)
    *   WACC = 14.03% + 3.54% = **17.57%**
    *   為簡化計算，我們將 WACC 取整為 **17.5%**。

#### **2. 折現現金流模型 (Discounted Cash Flow, DCF)**

*   **計算過程:**

*   **計算過程:**
    *   D0 = $0.90
    *   **2026 年股息 (D1):** $0.90 \* (1 + 0.05) = $0.945

    *   **2028 年末終端價值 (Terminal Value at end of 2028):**

        *   TV_2028 = $1.042 \* (1 + 0.015) / (0.1838 - 0.015) = $1.058 / 0.1688 = $6.268
    *   **股價 (P0) = PV(D1) + PV(D2) + PV(D3) + PV(TV_2028)**

        *   PV(D2) = $0.992 / (1 + 0.1838)^2 = $0.709
        *   PV(D3) = $1.042 / (1 + 0.1838)^3 = $0.629
        *   PV(TV_2028) = $6.268 / (1 + 0.1838)^3 = $3.785
        *   P0 = $0.798 + $0.709 + $0.629 + $3.785 = **$5.92**
