### 3. 估值模型與結果

#### 3.1. 折現現金流模型 (DCF)

**步驟 3: 計算終值 (Terminal Value, TV)**

*   TV = (最後一年 FCF \* (1 + 長期增長率)) / (WACC - 長期增長率)

*   TV = 138.826 / 0.0843 = 1646.81 億美元

**步驟 4: 計算折現自由現金流 (Present Value of FCF) 和終值現值 (Present Value of TV)**

*   終值現值 = TV / (1 + WACC)^5
*   終值現值 = 1646.81 / (1 + 0.1093)^5
*   終值現值 = 1646.81 / 1.6819 = 979.14 億美元

**步驟 5: 計算企業價值 (Enterprise Value, EV) 和股權價值 (Equity Value)**

*   企業價值 (EV) = 預測期 FCF 現值總和 + 終值現值
*   EV = 437.84 億美元 + 979.14 億美元 = 1416.98 億美元

*   股權價值 (Equity Value) = EV + 現金及約當現金 - 總債務
    *   QCOM 現金及約當現金 (截至 2025 年 9 月)：約 120 億美元 (估計值，需查閱最新財報)。
    *   QCOM 總債務 (截至 2025 年 9 月)：148.11 億美元 (長期債務)。
    *   股權價值 = 1416.98 億美元 + 120 億美元 - 148.11 億美元 = 1388.87 億美元

**步驟 6: 計算每股價值**

*   流通股數 = 市值 / 股價 = 1590.2 億美元 / 148.89 美元 = 10.68 億股 (約)。
*   每股價值 = 股權價值 / 流通股數
*   每股價值 = 1388.87 億美元 / 10.68 億股 = **130.04 美元**

*   **每股估值：130.04 美元**
*   **合理價位區間：115 - 145 美元** (考慮到假設的敏感性)

#### 3.3. 股息折現模型 (DDM)

**假設:**

*   當前股息 (D0) = 3.51 美元。
*   股權成本 (Ke) = 11.59%。
*   長期股息增長率 (g) = 2.5%。

**戈登增長模型 (Gordon Growth Model):**

*   每股價值 = D0 \* (1 + g) / (Ke - g)
*   每股價值 = 3.51 \* (1 + 0.025) / (0.1159 - 0.025)
*   每股價值 = 3.51 \* 1.025 / 0.0909
*   每股價值 = 3.59775 / 0.0909 = **39.58 美元**

這個結果顯著低於當前股價，表明單純的戈登增長模型可能不適用於 QCOM，因為它假設股息以恆定速度永遠增長，且 Ke 遠大於 g。QCOM 雖然支付股息，但其主要價值驅動因素可能來自於未來的資本增值和業務擴張，而非僅僅是股息。此外，其自由現金流增長率遠高於股息增長率，說明公司有能力將更多利潤再投資於業務。

**多階段 DDM (Two-Stage DDM):**

考慮到 QCOM 在汽車和 IoT 領域的成長潛力，我們採用兩階段 DDM：
*   **階段 1 (高增長階段，5 年):** 股息增長率 g1 = 8% (基於 FCF 增長潛力)。
*   **階段 2 (穩定增長階段):** 股息增長率 g2 = 2.5%。
