### 4. 估值模型計算

#### 4.1. 加權平均資本成本 (WACC) 計算

**a. 股權成本 (Cost of Equity, Ke) - 使用 CAPM 模型:**
Ke = Rf + Beta \* MRP
Ke = 4.61% + 1.42 \* 5.5%
Ke = 4.61% + 7.81%
**Ke = 12.42%**

**b. 債務成本 (Cost of Debt, Kd):**
由於沒有提供具體的利息費用和債務總額，我們將使用一個估計值。假設 Adobe 的債務成本約為無風險利率加上一個信用利差，例如 2%。
Kd = Rf + Credit Spread = 4.61% + 2% = 6.61%
**稅後債務成本 (Kd_after_tax) = Kd \* (1 - Tax Rate)**
Kd_after_tax = 6.61% \* (1 - 20%) = 6.61% \* 0.80 = **5.29%**

**c. 權重 (Weights):**
我們使用長期負債權益比 (LT Debt/Eq) 0.50 來估計債務和股權的權重。
股權權重 (We) = 1 / (1 + Debt/Equity) = 1 / (1 + 0.50) = 1 / 1.5 = 0.6667 (約 66.67%)
債務權重 (Wd) = Debt/Equity / (1 + Debt/Equity) = 0.50 / (1 + 0.50) = 0.50 / 1.5 = 0.3333 (約 33.33%)

**d. WACC 計算:**
WACC = (We \* Ke) + (Wd \* Kd_after_tax)
WACC = (0.6667 \* 12.42%) + (0.3333 \* 5.29%)
WACC = 8.28% + 1.76%
**WACC = 10.04%**
