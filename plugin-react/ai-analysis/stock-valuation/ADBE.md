#### 2.1 折現現金流模型 (DCF)

DCF 模型旨在估計公司未來現金流的現值。

**估值假設：**

1.  **預測期間 (Explicit Forecast Period):** 5 年 (2026 年 - 2030 年)。
2.  **折現率 (Discount Rate - WACC):**
    *   **股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM) 計算：`Ke = 無風險利率 + Beta * 市場風險溢價`
        *   **無風險利率 (Risk-Free Rate):** 採用 10 年期美國國債收益率，假設為 4.20% (接近 Alpha Spread 的 4.18%)。
        *   **Beta:** 考慮到不同來源的差異，我們採用一個較為保守且接近市場平均的 Beta 值，例如 Alpha Spread 的 0.96。
        *   **市場風險溢價 (Market Risk Premium, ERP):** 考慮到當前市場環境和歷史數據，我們採用 5.5% (介於 Oppenheimer 的低值和常見的 5-6% 之間)。
        *   `Ke = 4.20% + 0.96 * 5.50% = 4.20% + 5.28% = 9.48%`
    *   **債務成本 (Cost of Debt, Kd):** 採用 GuruFocus 報告的 4.05%。
    *   **稅率 (Tax Rate, t):** 採用 Adobe 2025 財年非 GAAP 稅率 18.5%。
    *   **稅後債務成本 (After-tax Cost of Debt):** `Kd * (1 - t) = 4.05% * (1 - 0.185) = 4.05% * 0.815 = 3.30%`
    *   **資本結構權重 (Weight of Equity, We; Weight of Debt, Wd):**
        *   市值 (Market Cap): 1272.9 億美元 (來自提供數據)。
        *   總債務 (Total Debt): 採用 2025 年 11 月的長期債務 62 億美元。
        *   總資本 (Total Capital) = 股權市值 + 債務 = 1272.9 億 + 62 億 = 1334.9 億美元
        *   `We = 1272.9 / 1334.9 = 0.9535` (約 95.35%)
        *   `Wd = 62 / 1334.9 = 0.0465` (約 4.65%)
    *   **加權平均資本成本 (WACC):** `WACC = (We * Ke) + (Wd * Kd * (1 - t))`
        *   `WACC = (0.9535 * 9.48%) + (0.0465 * 3.30%)`
        *   `WACC = 9.041% + 0.153% = 9.194%` (約 9.20%)
        *   *備註：此 WACC 介於 GuruFocus (12.61%) 和 Alpha Spread (7.97%) 之間，較為合理。*
3.  **增長率 (Growth Rates):**
    *   **短期高增長率 (Explicit Growth Rate):**
        *   FY2026 營收指引為 259 億至 261 億美元，較 FY2025 的 237.7 億美元增長約 9% - 9.8%。
        *   SaaS 市場預計在 2026-2033 年間 CAGR 為 19.3%。
        *   考慮到 Adobe 的規模和市場領導地位，以及 AI 帶來的增長潛力，我們假設：
            *   2026 年: 9.5% (接近公司指引)
            *   2027 年: 9.0%
            *   2028 年: 8.5%
            *   2029 年: 8.0%
            *   2030 年: 7.5%
    *   **長期穩定增長率 (Terminal Growth Rate, g):** 假設為 3.0% (略高於長期通脹和全球 GDP 增長率，反映其持續的創新能力和市場地位)。
4.  **自由現金流 (Free Cash Flow, FCF):**
    *   我們需要從營收和利潤率推導 FCF。
    *   **FY2025 營收:** 237.7 億美元。
    *   **FY2025 營運現金流:** 100.3 億美元。
    *   **資本支出 (Capital Expenditure, CapEx):** 根據提供的數據，沒有直接的 CapEx 數據。我們將使用營運現金流的百分比或營收的百分比進行估計。根據歷史數據，Adobe 的 CapEx 通常佔營收的 2-3%。我們假設為營收的 2.5%。
    *   **FCF = 營運現金流 - 資本支出**
    *   **FY2025 FCF (估計):** 100.3 億美元 (營運現金流) - (237.7 億美元 * 2.5%) = 100.3 億 - 5.9425 億 = 94.3575 億美元。

**DCF 計算步驟：**

1.  **預測未來 5 年的營收 (Revenue):**
    *   FY2025 營收: $23.77 B
    *   FY2026 營收: $23.77 B * (1 + 9.5%) = $26.03 B
    *   FY2027 營收: $26.03 B * (1 + 9.0%) = $28.37 B
    *   FY2028 營收: $28.37 B * (1 + 8.5%) = $30.78 B
    *   FY2029 營收: $30.78 B * (1 + 8.0%) = $33.24 B
    *   FY2030 營收: $33.24 B * (1 + 7.5%) = $35.73 B

2.  **預測未來 5 年的營運現金流 (Operating Cash Flow, OCF):**
    *   Adobe 的營運利潤率 (Oper. Margin) 為 36.63%。 [來自提供數據]
    *   我們假設營運現金流佔營收的比例 (Operating Cash Flow Margin) 保持相對穩定，或略有波動。FY2025 OCF Margin = 100.3 億 / 237.7 億 = 42.2%。我們將使用此比例。
    *   FY2026 OCF: $26.03 B * 42.2% = $10.99 B
    *   FY2027 OCF: $28.37 B * 42.2% = $11.98 B
    *   FY2028 OCF: $30.78 B * 42.2% = $12.99 B
    *   FY2029 OCF: $33.24 B * 42.2% = $14.04 B
    *   FY2030 OCF: $35.73 B * 42.2% = $15.08 B

3.  **預測未來 5 年的資本支出 (Capital Expenditure, CapEx):**
    *   假設 CapEx 佔營收的 2.5%。
    *   FY2026 CapEx: $26.03 B * 2.5% = $0.65 B
    *   FY2027 CapEx: $28.37 B * 2.5% = $0.71 B
    *   FY2028 CapEx: $30.78 B * 2.5% = $0.77 B
    *   FY2029 CapEx: $33.24 B * 2.5% = $0.83 B
    *   FY2030 CapEx: $35.73 B * 2.5% = $0.89 B

4.  **計算未來 5 年的自由現金流 (FCF):** `FCF = OCF - CapEx`
    *   FY2026 FCF: $10.99 B - $0.65 B = $10.34 B
    *   FY2027 FCF: $11.98 B - $0.71 B = $11.27 B
    *   FY2028 FCF: $12.99 B - $0.77 B = $12.22 B
    *   FY2029 FCF: $14.04 B - $0.83 B = $13.21 B
    *   FY2030 FCF: $15.08 B - $0.89 B = $14.19 B

5.  **計算終值 (Terminal Value, TV):**
    *   `TV = FCF_2031 / (WACC - g)`
    *   FCF_2031 = FCF_2030 * (1 + g) = $14.19 B * (1 + 3.0%) = $14.616 B
    *   `TV = $14.616 B / (0.0920 - 0.030) = $14.616 B / 0.062 = $235.74 B`

6.  **計算各期 FCF 和 TV 的現值 (Present Value, PV):**
    *   PV(FCF_2026) = $10.34 B / (1 + 0.0920)^1 = $9.47 B
    *   PV(FCF_2027) = $11.27 B / (1 + 0.0920)^2 = $9.47 B
    *   PV(FCF_2028) = $12.22 B / (1 + 0.0920)^3 = $9.37 B
    *   PV(FCF_2029) = $13.21 B / (1 + 0.0920)^4 = $9.27 B
    *   PV(FCF_2030) = $14.19 B / (1 + 0.0920)^5 = $9.17 B
    *   PV(TV) = $235.74 B / (1 + 0.0920)^5 = $152.36 B

7.  **計算公司總價值 (Enterprise Value, EV):**
    *   `EV = PV(FCF_2026) + ... + PV(FCF_2030) + PV(TV)`
    *   `EV = $9.47 B + $9.47 B + $9.37 B + $9.27 B + $9.17 B + $152.36 B = $199.11 B`

8.  **計算股權價值 (Equity Value):**
    *   `Equity Value = EV + 現金及約當現金 - 總債務`
    *   截至 2025 年 11 月 28 日，現金及短期投資為 66 億美元。
    *   總債務為 62 億美元。
    *   `Equity Value = $199.11 B + $6.6 B - $6.2 B = $199.51 B`

9.  **計算每股價值 (Intrinsic Value Per Share):**
    *   流通股數 (Shares Outstanding): `市值 / 股價 = $127.29 B / $296.12 = 429.85 百萬股`
    *   `Intrinsic Value Per Share = Equity Value / Shares Outstanding`
    *   `Intrinsic Value Per Share = $199.51 B / 429.85 M = $464.14`

**DCF 模型估值結果：**

*   **每股合理價位:** **$464.14**
*   **合理價位區間:** 考慮到假設的敏感性，我們可以設定一個區間。例如，WACC 波動 0.5% (8.7% - 9.7%) 和長期增長率波動 0.5% (2.5% - 3.5%)。
    *   **較低情境 (WACC 9.7%, g 2.5%):** 估計每股約 $400 - $430
    *   **較高情境 (WACC 8.7%, g 3.5%):** 估計每股約 $500 - $530
    *   **合理價位區間:** **$430 - $500**

#### 2.2 相對估值 (Comparable Company Analysis)

*   **Oracle (ORCL):** 在數位行銷、客戶體驗和雲服務方面與 Adobe 競爭。

**估值倍數選擇：**

*   **市盈率 (P/E Ratio):** 衡量公司盈利能力的倍數。
*   **遠期市盈率 (Forward P/E Ratio):** 基於未來預期盈利的倍數，更能反映市場對未來增長的預期。
*   **市銷率 (P/S Ratio):** 衡量公司營收的倍數，適用於高增長公司。
*   **企業價值/EBITDA (EV/EBITDA):** 考慮了債務的倍數，適用於不同資本結構的公司。
*   **企業價值/自由現金流 (EV/FCF):** 衡量公司產生現金流能力的倍數。
