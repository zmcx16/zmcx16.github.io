### **2. 估值方法與計算**

我們將應用折現現金流模型 (DCF) 和相對估值法。由於 Adobe 不支付股息，股息折現模型 (DDM) 不適用。

#### **2.1 加權平均資本成本 (WACC) 計算**

WACC 是 DCF 模型中的折現率。我們將使用 GuruFocus 提供的數據進行計算。

**WACC 假設:**
*   **無風險利率 (Risk-Free Rate):** 4.337% (美國 10 年期國債收益率)。
*   **Beta:** 1.2047。
*   **市場風險溢酬 (Market Risk Premium):** 6%。
*   **股權成本 (Cost of Equity, Ke):**
    Ke = 無風險利率 + Beta * 市場風險溢酬
    Ke = 4.337% + 1.2047 * 6% = 4.337% + 7.2282% = 11.5652% ≈ 11.57%
*   **債務成本 (Cost of Debt, Kd):** 3.992%。
*   **企業稅率 (Corporate Tax Rate):** 21%。
*   **股權市值 (Market Value of Equity, E):** 981.9 億美元 (來自提供數據)。
*   **債務市值 (Market Value of Debt, D):** 66.132 億美元 (來自 GuruFocus 的平均債務賬面價值)。
*   **總資本 (V):** E + D = 981.9 億 + 66.132 億 = 1048.032 億美元
*   **股權權重 (E/V):** 981.9 / 1048.032 = 0.9369 ≈ 93.69%
*   **債務權重 (D/V):** 66.132 / 1048.032 = 0.0631 ≈ 6.31%

**WACC 計算:**
WACC = (E/V) * Ke + (D/V) * Kd * (1 - 稅率)
WACC = 0.9369 * 11.57% + 0.0631 * 3.992% * (1 - 0.21)
WACC = 0.10839 + 0.0631 * 3.992% * 0.79
WACC = 0.10839 + 0.00198
WACC = 0.11037 ≈ **11.04%**

這個計算結果與 GuruFocus 提供的 11.04% WACC 一致。

#### **2.2 折現現金流模型 (Discounted Cash Flow, DCF)**

**DCF 假設:**
*   **預測期間:** 5 年 (2026 財年 - 2030 財年)。
*   **WACC (折現率):** 11.04%。
*   **長期增長率 (Terminal Growth Rate, g):** 2.5% (保守估計，考慮到 Adobe 的規模和市場地位，以及 AI 帶來的潛在增長，但同時也考慮到競爭和增長放緩的擔憂)。
*   **起始營收 (2025 財年):** 237 億美元 (分析師預期)。
*   **營業利潤率 (Operating Margin):** 2026 財年 TTM 為 37.79%。我們假設在預測期內保持在 37% - 38% 之間，然後穩定在 37%。
*   **稅率:** 21%。
*   **資本支出 (CapEx):** 過去十二個月為 1.9 億美元。我們將其預計為營收的 0.8% - 1.0%。
*   **折舊與攤銷 (D&A):** 假設與資本支出大致持平，或略高於資本支出。
*   **營運資本變動 (Change in Working Capital):** 假設為營收的 -0.5% (即營運資本增加，消耗現金)。

**DCF 計算步驟:**

1.  **預測營收 (Revenue Forecast):**
    *   2026 財年: 237 億美元 * (1 + 10.5%) = 261.885 億美元 (基於 2025 財年預期和 2026 財年 ARR 增長預期 10.2% 及 2025 財年營收增長 10.53%)
    *   2027 財年: 261.885 億美元 * (1 + 9.5%) = 286.76 億美元
    *   2028 財年: 286.76 億美元 * (1 + 8.5%) = 311.19 億美元
    *   2029 財年: 311.19 億美元 * (1 + 7.5%) = 334.53 億美元
    *   2030 財年: 334.53 億美元 * (1 + 6.5%) = 356.22 億美元

2.  **預測營業利潤 (Operating Income):**
    *   假設營業利潤率穩定在 37%。
    *   2026 財年: 261.885 億美元 * 37% = 96.89 億美元
    *   2027 財年: 286.76 億美元 * 37% = 106.10 億美元
    *   2028 財年: 311.19 億美元 * 37% = 115.14 億美元
    *   2029 財年: 334.53 億美元 * 37% = 123.78 億美元
    *   2030 財年: 356.22 億美元 * 37% = 131.80 億美元

3.  **計算稅後營業利潤 (NOPAT):**
    NOPAT = 營業利潤 * (1 - 稅率)
    NOPAT = 營業利潤 * (1 - 0.21) = 營業利潤 * 0.79

4.  **預測資本支出 (CapEx) 和折舊與攤銷 (D&A):**
    *   CapEx: 假設為營收的 0.8%。
    *   D&A: 假設為營收的 0.9% (略高於 CapEx)。

5.  **預測營運資本變動 (Change in Working Capital):**
    *   假設為營收的 -0.5%。

6.  **計算自由現金流 (Free Cash Flow, FCF):**
    FCF = NOPAT + D&A - CapEx - 營運資本變動

**預測期內折現 FCF 總和:** 70.36 + 69.36 + 67.80 + 65.64 + 62.95 = **336.11 億美元**

7.  **計算終值 (Terminal Value, TV):**
    我們使用永續增長模型 (Gordon Growth Model)。
    TV = (2030 財年 FCF * (1 + g)) / (WACC - g)
    TV = (106.26 億美元 * (1 + 0.025)) / (0.1104 - 0.025)
    TV = (106.26 億美元 * 1.025) / 0.0854
    TV = 108.9165 億美元 / 0.0854 = **1275.37 億美元**

8.  **計算終值折現值 (Discounted Terminal Value):**
    折現 TV = TV / (1 + WACC)^5
    折現 TV = 1275.37 億美元 / (1 + 0.1104)^5
    折現 TV = 1275.37 億美元 / 1.686 = **756.45 億美元**

9.  **計算公司總價值 (Total Company Value):**
    公司總價值 = 預測期內折現 FCF 總和 + 折現 TV
    公司總價值 = 336.11 億美元 + 756.45 億美元 = **1092.56 億美元**

10. **計算股權價值 (Equity Value):**
    股權價值 = 公司總價值 - 債務市值 + 現金及約當現金
    *   債務市值: 66.132 億美元。
    *   現金及約當現金: 69 億美元 (截至 2026 年 4 月)。
    股權價值 = 1092.56 億美元 - 66.132 億美元 + 69 億美元 = **1095.43 億美元**

11. **計算每股估值 (Intrinsic Value Per Share):**
    流通股數 (Shares Outstanding): 981.9 億美元 (市值) / 244.35499572753906 美元 (收盤價) = 4.018 億股 (約 4.02 億股)。
    每股估值 = 股權價值 / 流通股數
    每股估值 = 1095.43 億美元 / 4.02 億股 = **272.50 美元/股**

**DCF 估值結果與合理價位區間:**
基於上述假設，DCF 模型得出的 Adobe 每股估值約為 **272.50 美元**。
考慮到假設的敏感性，合理價位區間可能在 **250 美元至 300 美元** 之間。
