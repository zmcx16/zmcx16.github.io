    *   **WACC 計算 (簡化):**
        *   **股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM)。
            *   無風險利率 (Risk-Free Rate): 假設為 4.5% (參考美國 10 年期國債收益率)。

        *   **負債成本 (Cost of Debt, Kd):** 根據提供的數據，長期負債權益比為 0.5。假設平均借貸利率為 5.0%。

### 3. 估值模型與結果

#### 3.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型的目標是估計公司未來自由現金流的現值。

**步驟 1: 計算歷史自由現金流 (FCF)**

*   2025 財年營收: 237.7 億美元
*   2025 財年營業利潤率: 36.91% (用戶提供數據)
*   2025 財年營業利潤 (EBIT): 237.7 億 \* 36.91% = 87.73 億美元
*   2025 財年稅率: 20%
*   2025 財年稅後營業利潤 (NOPAT): 87.73 億 \* (1 - 0.20) = 70.18 億美元
*   2025 財年折舊與攤銷 (D&A): 假設為營收的 2.5% = 237.7 億 \* 2.5% = 5.94 億美元
*   2025 財年資本支出 (CapEx): 1.79 億美元
*   2025 財年營運資本變動 (ΔNWC): 假設為營收的 1% = 237.7 億 \* 1% = 2.38 億美元
*   **2025 財年 FCF = NOPAT + D&A - CapEx - ΔNWC**
    *   FCF = 70.18 + 5.94 - 1.79 - 2.38 = 71.95 億美元
    *   *註：此處計算的 FCF 與 Macrotrends 提供的 98.52 億美元 和 MLQ.ai 提供的 98.5 億美元 有差異，主要由於 D&A 和 NWC 的假設。我們將以 Macrotrends 提供的 2025 財年 FCF 98.52 億美元作為基準，並調整未來增長率。*

**步驟 2: 預測未來自由現金流 (2026-2030 財年)**

*   **基準 FCF (2025 財年):** 98.52 億美元
*   **營收增長率:**
    *   2026: 9.0%
    *   2027: 8.5%
    *   2028: 8.0%
    *   2029: 7.5%
    *   2030: 7.0%
*   **FCF 增長率假設:** 由於 Adobe 是一個成熟的軟體公司，其 FCF 增長率可能略低於營收增長率，但由於其高利潤率和訂閱模式，FCF 轉換率較高。我們將假設 FCF 增長率與營收增長率接近。

| 財年 | 營收增長率 | 預計 FCF (億美元) | 折現因子 (WACC=8.5%) | FCF 現值 (億美元) |
| :--- | :--------- | :---------------- | :------------------ | :------------------ |
| 2025 | -          | 98.52             | -                   | -                   |
| 2026 | 9.0%       | 107.39            | 1 / (1+0.085)^1 = 0.9217 | 98.99               |
| 2027 | 8.5%       | 116.53            | 1 / (1+0.085)^2 = 0.8495 | 99.00               |
| 2028 | 8.0%       | 125.85            | 1 / (1+0.085)^3 = 0.7829 | 98.50               |
| 2029 | 7.5%       | 135.27            | 1 / (1+0.085)^4 = 0.7216 | 97.60               |
| 2030 | 7.0%       | 144.74            | 1 / (1+0.085)^5 = 0.6651 | 96.35               |
| **預測期 FCF 現值總和** | | | | **490.44** |

**步驟 3: 計算終值 (Terminal Value, TV)**

*   終值 FCF (2031 年) = 2030 年 FCF \* (1 + 長期增長率) = 144.74 億 \* (1 + 0.025) = 148.36 億美元
*   終值 (TV) = 終值 FCF / (WACC - 長期增長率) = 148.36 億 / (0.085 - 0.025) = 148.36 億 / 0.06 = 2472.67 億美元
*   終值現值 (PV of TV) = 終值 / (1 + WACC)^5 = 2472.67 億 / (1 + 0.085)^5 = 2472.67 億 / 1.5037 = 1644.32 億美元

**步驟 4: 計算公司總價值 (Enterprise Value, EV)**

*   EV = 預測期 FCF 現值總和 + 終值現值 = 490.44 億 + 1644.32 億 = 2134.76 億美元

**步驟 5: 計算股權價值 (Equity Value)**

*   股權價值 = EV + 現金及約當現金 - 總債務
    *   根據 Stock Analysis，Adobe 擁有 69 億美元現金和 66.7 億美元債務，淨現金為 2.29 億美元。
    *   股權價值 = 2134.76 億 + 2.29 億 = 2137.05 億美元

**步驟 6: 計算每股估值 (Intrinsic Value per Share)**

*   流通股數 (Shares Outstanding): 根據市值 952.8 億美元和股價 235.72 美元，流通股數約為 952.8 億 / 235.72 = 4.04 億股。
*   每股估值 = 股權價值 / 流通股數 = 2137.05 億 / 4.04 億 = **529.0 美元/股**

**DCF 估值結果與合理價位區間：**

*   **DCF 估值:** 529.0 美元/股
*   **合理價位區間 (基於 WACC 敏感性分析):**
    *   WACC = 9.5% (較高): 每股估值約 450-500 美元
    *   WACC = 7.5% (較低): 每股估值約 600-650 美元
    *   **綜合區間:** 450 - 650 美元/股
