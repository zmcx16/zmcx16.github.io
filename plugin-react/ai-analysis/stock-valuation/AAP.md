*   **股息增長率 (g):** 考慮到公司盈利不穩定和轉型，初期增長率可能較低，甚至不增長。長期增長率應與DCF模型中的永續增長率一致。
    *   短期 (未來3年): 0% (保守假設，公司可能優先用於再投資或償債)
    *   長期 (之後): 2.0% (與DCF模型一致)
*   **股權成本 (Ke):** 10.37% (來自DCF模型)

**DDM 計算步驟 (兩階段模型):**

1.  **預測未來股息：**
    *   D1 = D0 * (1 + 0%) = 1.00美元
    *   D2 = D1 * (1 + 0%) = 1.00美元
    *   D3 = D2 * (1 + 0%) = 1.00美元
    *   D4 = D3 * (1 + 2.0%) = 1.02美元 (進入永續增長階段的第一年股息)

2.  **計算永續價值 (Terminal Value of Dividends, TVD):**
    *   TVD = D4 / (Ke - g) = 1.02 / (10.37% - 2.0%) = 1.02 / 0.0837 = 12.186 美元

3.  **折現未來股息和 TVD：**

| 年度 | 股息 (美元) | 折現因子 (1/(1+Ke)^n) | 折現股息 (美元) |
| :--- | :---------- | :-------------------- | :-------------- |
| 1    | 1.00        | 0.9060                | 0.906           |
| 2    | 1.00        | 0.8209                | 0.821           |
| 3    | 1.00        | 0.7438                | 0.744           |
| TVD  | 12.186      | 0.7438                | 9.060           |

4.  **計算每股估值：**
    *   每股估值 = 折現股息總和 + 折現 TVD = 0.906 + 0.821 + 0.744 + 9.060 = **11.53 美元/股**

**DDM 模型估值結果：**

*   **每股估值：11.53 美元**
*   **合理價位區間：10.00 - 15.00 美元**

**DDM 模型的局限性：**
DDM 模型對股息增長率和股權成本的假設非常敏感。由於 AAP 的盈利不穩定，且股息政策可能隨公司轉型而調整，DDM 模型在此情況下可能不是最可靠的估值方法，其結果顯著低於當前股價和DCF估值，可能表明市場對其未來增長或股息穩定性有更高的預期，或者該模型不完全適用於當前處於轉型期的公司。
