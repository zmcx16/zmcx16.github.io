**DCF 計算步驟 (簡化)：**

1.  **預測收入 (Revenue)。**
2.  **計算營業利潤 (Operating Income) = 收入 * 營業利潤率。**
3.  **計算稅後營業利潤 (NOPAT) = 營業利潤 * (1 - 稅率)。**
4.  **計算自由現金流 (FCF) = NOPAT + 折舊與攤銷 (D&A) - 資本支出 (CapEx) - 營運資本變動。** (由於缺乏詳細的 D&A 數據，我將使用簡化的 FCF 估計，或基於歷史 FCF 趨勢和管理層指引進行推斷。管理層預計 2026 年 FCF 為 1 億美元。)
5.  **計算終值 (Terminal Value) = (最後一年 FCF * (1 + 長期增長率)) / (WACC - 長期增長率)。**
6.  **將預測期 FCF 和終值折現至現值。**
7.  **計算企業價值 (Enterprise Value) = 折現 FCF 總和 + 折現終值。**
8.  **計算股權價值 (Equity Value) = 企業價值 + 現金 - 債務。**
9.  **計算每股價值 (Intrinsic Value per Share) = 股權價值 / 流通股數。**

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於支付穩定股息且預計未來股息會增長的公司。AAP 目前支付季度股息 $0.25，年化 $1.00。

**估值假設：**

*   **當前股息 (D0)：** $1.00 (年化)。
*   **股息增長率 (g)：** 考慮到公司正在轉型，且過去五年股息增長率為負 (-18.31%)，但管理層預計 2026 年自由現金流轉正，未來可能支持股息增長。我將假設一個保守的長期股息增長率，例如 1.0%。
*   **股權成本 (Ke)：** 根據 Alpha Spread，AAP 的股權成本為 7.21%。我將採用 7.2%。

**DDM 計算 (Gordon Growth Model)：**
*   每股價值 = D1 / (Ke - g)
*   D1 = D0 * (1 + g) = $1.00 * (1 + 0.01) = $1.01
*   每股價值 = $1.01 / (0.072 - 0.01) = $1.01 / 0.062 = $16.29

**DDM 估值結果：**
*   **每股合理價位：$16.29**
*   **合理價位區間：** 考慮到股息增長率的敏感性，如果增長率為 0.5% 到 1.5%，估值可能在 **$15 - $17** 之間波動。

**DDM 的適用性分析：**
AAP 的 DDM 估值結果顯著低於當前股價和其他估值模型。這可能是因為公司目前處於轉型期，自由現金流在 2025 年為負，雖然 2026 年預計轉正，但其股息增長的可持續性和速度存在較大不確定性。此外，其派息率為 44%，但 2025 年的負自由現金流表明股息支付可能需要動用現金儲備或借款。因此，DDM 在 AAP 目前的轉型階段可能不是最合適的估值方法，其結果應謹慎看待。
