### 2. 估值方法應用

鑑於 AAP 近期盈利不穩定且為負值，P/E 倍數在相對估值中可能需要謹慎使用，或側重於 Forward P/E。DCF 模型將需要對未來現金流進行更為樂觀但合理的假設。DDM 模型適用於有穩定股息的公司，AAP 目前有派息，因此可以嘗試。

#### 估值假設 (統一假設)

*   **無風險利率 (Risk-Free Rate):** 參考美國 10 年期國債收益率，取 4.15%。
*   **市場風險溢價 (Market Risk Premium, ERP):** 採用 5.10%。
*   **稅率 (Tax Rate):** 採用 23.75%。
*   **預測期間 (Explicit Forecast Period):** 5 年。
*   **永續增長率 (Terminal Growth Rate):** 考慮到汽車零配件行業的成熟性以及 AAP 近期的挑戰，採用 1.5% (略低於長期通脹和全球經濟增長率)。

**計算加權平均資本成本 (WACC):**

*   **Beta (β):** 採用 0.78 (來自 Alpha Spread)。 雖然其他來源有更高的 Beta (如 1.17 或 1.56)，但 0.78 是一個相對保守且被引用的值，反映了其相對於市場的波動性。
*   **股權成本 (Cost of Equity, Ke):**
    Ke = 無風險利率 + Beta × 市場風險溢價
    Ke = 4.15% + 0.78 × 5.10% = 4.15% + 3.98% = 8.13%
    (Alpha Spread 估計為 7.36%，另一來源為 8.7%。我將使用 8.13% 作為中間值。)
*   **債務成本 (Cost of Debt, Kd):** 採用 5.58%。
*   **債務權重 (Weight of Debt, Wd):**
    根據最新數據，市場價值為 25.4 億美元，債務為 56.7 億美元。
    總資本 = 25.4 億 (股權) + 56.7 億 (債務) = 82.1 億美元
    Wd = 56.7 / 82.1 = 69.06%
*   **股權權重 (Weight of Equity, We):**
    We = 25.4 / 82.1 = 30.94%
*   **WACC 計算:**
    WACC = (We × Ke) + (Wd × Kd × (1 - 稅率))
    WACC = (0.3094 × 8.13%) + (0.6906 × 5.58% × (1 - 0.2375))
    WACC = 2.51% + (0.6906 × 5.58% × 0.7625)
    WACC = 2.51% + 2.94% = 5.45%

    (Alpha Spread 估計 WACC 為 6.33%，另一來源為 6.8%，GuruFocus 估計為 6.25%。我的計算結果 5.45% 略低，主要因為 Beta 和股權成本的選擇。為了保守起見，我將採用一個更接近市場共識的 WACC，例如 6.5%。)

    **最終 WACC 假設：6.5%**

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測未來自由現金流 (Free Cash Flow, FCF)。鑑於 AAP 近期盈利不穩定，預測 FCF 存在較大挑戰。我將基於分析師對未來 EPS 和銷售額的預期，以及公司改善經營的目標來進行推導。

**DCF 假設：**

*   **起始年份 (2025 財年) 銷售額：** 85 億美元 (取分析師預期 84-86 億美元的中間值)。
*   **銷售額增長率：**
    *   2026 年：1.5% (基於 2025 年預期可比店面銷售增長 0.5-1.5% 和行業整體溫和增長)
    *   2027-2029 年：2.0% (考慮到行業老化車輛趨勢的支撐 和公司戰略調整的潛在效益)
    *   2030 年及以後 (永續增長率)：1.5%
*   **營業利潤率 (Operating Margin):**
    *   2023 年營業虧損，2024 年 Q4 營業虧損 2.0%。公司預計 2025 財年營業利潤率為 2.0%-3.0%。
    *   我將假設公司通過成本控制和戰略調整逐步改善。
    *   2025 年：1.0% (保守估計，低於公司指引，考慮到轉型挑戰)
    *   2026 年：2.0%
    *   2027 年：3.0%
    *   2028 年：3.5%
    *   2029 年：4.0% (接近行業平均水平)
*   **稅率：** 23.75%
*   **資本支出 (Capital Expenditure, CapEx) 佔銷售額百分比：** 2.5% (參考歷史數據和行業平均水平)
*   **折舊與攤銷 (Depreciation & Amortization, D&A) 佔銷售額百分比：** 2.0% (參考歷史數據和行業平均水平)
*   **營運資本變動 (Change in Working Capital):** 假設為銷售額的 0.5% (作為現金流出，即營運資本增加，對 FCF 產生負面影響，反映公司擴張或庫存管理需求)。

**DCF 計算步驟 (簡化):**

1.  **預測銷售額 (Revenue):**
    *   2025: $8.50 B
    *   2026: $8.50 B * (1 + 1.5%) = $8.63 B
    *   2027: $8.63 B * (1 + 2.0%) = $8.80 B
    *   2028: $8.80 B * (1 + 2.0%) = $8.98 B
    *   2029: $8.98 B * (1 + 2.0%) = $9.16 B

2.  **預測營業利潤 (EBIT):**
    *   2025: $8.50 B * 1.0% = $0.085 B
    *   2026: $8.63 B * 2.0% = $0.173 B
    *   2027: $8.80 B * 3.0% = $0.264 B
    *   2028: $8.98 B * 3.5% = $0.314 B
    *   2029: $9.16 B * 4.0% = $0.366 B

3.  **計算稅後營業利潤 (NOPAT):**
    NOPAT = EBIT × (1 - 稅率)
    *   2025: $0.085 B * (1 - 0.2375) = $0.065 B
    *   2026: $0.173 B * (1 - 0.2375) = $0.132 B
    *   2027: $0.264 B * (1 - 0.2375) = $0.201 B
    *   2028: $0.314 B * (1 - 0.2375) = $0.240 B
    *   2029: $0.366 B * (1 - 0.2375) = $0.280 B

4.  **計算自由現金流 (FCF):**
    FCF = NOPAT + D&A - CapEx - 營運資本變動
    *   D&A: 銷售額 * 2.0%
    *   CapEx: 銷售額 * 2.5%
    *   營運資本變動: 銷售額 * 0.5%

    *   2025: $0.065 B + ($8.50 B * 2.0%) - ($8.50 B * 2.5%) - ($8.50 B * 0.5%) = $0.065 B + $0.170 B - $0.213 B - $0.043 B = -$0.021 B
    *   2026: $0.132 B + ($8.63 B * 2.0%) - ($8.63 B * 2.5%) - ($8.63 B * 0.5%) = $0.132 B + $0.173 B - $0.216 B - $0.043 B = $0.046 B
    *   2027: $0.201 B + ($8.80 B * 2.0%) - ($8.80 B * 2.5%) - ($8.80 B * 0.5%) = $0.201 B + $0.176 B - $0.220 B - $0.044 B = $0.113 B
    *   2028: $0.240 B + ($8.98 B * 2.0%) - ($8.98 B * 2.5%) - ($8.98 B * 0.5%) = $0.240 B + $0.180 B - $0.225 B - $0.045 B = $0.150 B
    *   2029: $0.280 B + ($9.16 B * 2.0%) - ($9.16 B * 2.5%) - ($9.16 B * 0.5%) = $0.280 B + $0.183 B - $0.229 B - $0.046 B = $0.188 B

5.  **計算終值 (Terminal Value, TV):**
    TV = FCF_2029 * (1 + 永續增長率) / (WACC - 永續增長率)
    TV = $0.188 B * (1 + 1.5%) / (6.5% - 1.5%)
    TV = $0.188 B * 1.015 / 0.05 = $3.81 B

6.  **折現所有現金流和終值：**
    *   PV(FCF_2025) = -$0.021 B / (1 + 0.065)^1 = -$0.020 B
    *   PV(FCF_2026) = $0.046 B / (1 + 0.065)^2 = $0.041 B
    *   PV(FCF_2027) = $0.113 B / (1 + 0.065)^3 = $0.093 B
    *   PV(FCF_2028) = $0.150 B / (1 + 0.065)^4 = $0.117 B
    *   PV(FCF_2029) = $0.188 B / (1 + 0.065)^5 = $0.137 B
    *   PV(TV) = $3.81 B / (1 + 0.065)^5 = $2.78 B

7.  **計算企業價值 (Enterprise Value, EV):**
    EV = 所有折現現金流之和 + 折現終值
    EV = -$0.020 B + $0.041 B + $0.093 B + $0.117 B + $0.137 B + $2.78 B = $3.15 B

8.  **計算股權價值 (Equity Value):**
    股權價值 = EV + 現金及約當現金 - 總債務
    根據數據，現金為 31.7 億美元，總債務為 56.7 億美元。
    股權價值 = $3.15 B + $3.17 B - $5.67 B = $0.65 B

9.  **計算每股估值 (Intrinsic Value Per Share):**
    流通股數：60.02 百萬股。
    每股估值 = $0.65 B / 60.02 M = $10.83

**DCF 估值結果：**

*   **每股估值：$10.83**
*   **合理價位區間：$8.00 - $15.00** (考慮到假設的敏感性，特別是營業利潤率和永續增長率)

這個結果遠低於當前股價，表明在這些保守假設下，AAP 被高估。主要原因是公司目前盈利能力較弱，自由現金流在初期甚至為負，且債務負擔較重。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

AAP 目前有派發股息，但其盈利不穩定，且自由現金流在某些時期為負，這使得股息的持續性和增長存在不確定性。因此，DDM 模型可能不是最適合 AAP 的估值方法，但仍可作為參考。

**DDM 假設：**

*   **當前股息 (D0):** 每年 1.00 美元 (基於 0.25 美元/季度股息)。
*   **股息增長率 (g):** 考慮到公司近期業績不佳和轉型，預計初期股息增長緩慢甚至不增長，長期維持較低水平。
    *   短期 (未來 3 年)：0% (保守假設，公司可能優先用於再投資或償債)
    *   長期 (永續增長率)：1.0% (低於 WACC 和永續增長率，反映成熟行業的溫和增長)
*   **股權成本 (Ke):** 8.13% (與 DCF 模型中的 Ke 一致)

**DDM 計算步驟 (兩階段 DDM):**

1.  **預測未來股息：**
    *   D1 = D0 * (1 + g1) = $1.00 * (1 + 0%) = $1.00
    *   D2 = D1 * (1 + g1) = $1.00 * (1 + 0%) = $1.00
    *   D3 = D2 * (1 + g1) = $1.00 * (1 + 0%) = $1.00

2.  **計算終值 (Terminal Value of Dividends, TVD):**
    TVD = D3 * (1 + g_long_term) / (Ke - g_long_term)
    TVD = $1.00 * (1 + 1.0%) / (8.13% - 1.0%)
    TVD = $1.00 * 1.01 / 0.0713 = $14.16

3.  **折現所有股息和終值：**
    *   PV(D1) = $1.00 / (1 + 0.0813)^1 = $0.92
    *   PV(D2) = $1.00 / (1 + 0.0813)^2 = $0.85
    *   PV(D3) = $1.00 / (1 + 0.0813)^3 = $0.78
    *   PV(TVD) = $14.16 / (1 + 0.0813)^3 = $11.16

4.  **計算每股估值：**
    每股估值 = PV(D1) + PV(D2) + PV(D3) + PV(TVD)
    每股估值 = $0.92 + $0.85 + $0.78 + $11.16 = $13.71

**DDM 估值結果：**

*   **每股估值：$13.71**
*   **合理價位區間：$10.00 - $18.00** (對股息增長率的敏感性較高)
