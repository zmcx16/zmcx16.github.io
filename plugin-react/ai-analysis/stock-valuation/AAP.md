### 2. 估值方法與計算

由於 AAP 的自由現金流在近期為負值，且波動較大，這使得傳統的 DCF 模型在短期預測上可能面臨挑戰。然而，我們仍將嘗試構建一個 DCF 模型，並輔以相對估值和股息折現模型。

#### 2.1. 加權平均資本成本 (WACC) 計算

WACC 是 DCF 模型中的關鍵折現率。
WACC = (E/V) * Re + (D/V) * Rd * (1 - Tc)
其中：
*   E = 股權市值 (Market Value of Equity)
*   D = 債務市值 (Market Value of Debt)
*   V = E + D = 企業總價值 (Total Firm Value)
*   Re = 股權成本 (Cost of Equity)
*   Rd = 債務成本 (Cost of Debt)
*   Tc = 企業稅率 (Corporate Tax Rate)

**a. 股權市值 (E):**
*   市值 (Market Cap) = $3.37 億美元 (用戶提供)

**b. 債務市值 (D):**
*   我們需要估計 AAP 的總債務。根據提供的數據，Debt/Eq = 2.57。
*   D = Debt/Eq * E = 2.57 * $3.37 億美元 = $8.6609 億美元。
*   根據 Stock Analysis，AAP 有 $56.6 億美元的債務。我們將使用這個更直接的數據。
*   D = $5.66 億美元

**c. 企業總價值 (V):**
*   V = E + D = $3.37 億美元 + $5.66 億美元 = $9.03 億美元

**d. 股權權重 (E/V):**
*   E/V = $3.37 億美元 / $9.03 億美元 = 0.3732 (約 37.32%)

**e. 債務權重 (D/V):**
*   D/V = $5.66 億美元 / $9.03 億美元 = 0.6268 (約 62.68%)

**f. 企業稅率 (Tc):**
*   根據 Q4 2024 財報，公司在 2024 財年的有效稅率為 23.6%。我們將使用此數據。
*   Tc = 23.6%

**g. 股權成本 (Re) - 使用資本資產定價模型 (CAPM):**
Re = Rf + Beta * (Rm - Rf)
其中：
*   Rf = 無風險利率 (Risk-free Rate)
*   Beta = 貝塔值 (Beta)
*   Rm - Rf = 市場風險溢酬 (Market Risk Premium)

*   **無風險利率 (Rf):** 通常使用 10 年期美國國債收益率。目前約為 4.5% (假設值，需即時查詢)。
*   **Beta:** 根據 Yahoo Finance 或其他金融數據提供商，AAP 的 Beta 值約為 1.5 (假設值，需即時查詢)。
*   **市場風險溢酬 (Rm - Rf):** 通常在 5% - 6% 之間。我們取 5.5%。

*   Re = 4.5% + 1.5 * 5.5% = 4.5% + 8.25% = 12.75%

**h. 債務成本 (Rd):**
*   我們需要估計 AAP 的借貸成本。考慮到其債務/股權比相對較高 (2.57)，其債務成本可能略高於無風險利率。假設為 6.5%。

**i. WACC 計算:**
*   WACC = (0.3732 * 12.75%) + (0.6268 * 6.5% * (1 - 0.236))
*   WACC = 0.047583 + (0.6268 * 0.065 * 0.764)
*   WACC = 0.047583 + (0.040742 * 0.764)
*   WACC = 0.047583 + 0.031130
*   WACC = 0.078713 或 7.87%

**WACC 假設:**
*   無風險利率 (Rf): 4.5%
*   Beta: 1.5
*   市場風險溢酬 (Rm - Rf): 5.5%
*   債務成本 (Rd): 6.5%
*   企業稅率 (Tc): 23.6%
*   股權市值 (E): $3.37 億美元
*   債務市值 (D): $5.66 億美元

#### 2.2. 折現現金流模型 (DCF)

DCF 模型需要預測未來幾年的自由現金流 (FCF) 和終端價值 (Terminal Value)。由於 AAP 近期 FCF 為負且波動較大，這將增加預測難度。我們將基於其轉型計劃和分析師預期進行保守估計。

**d. 計算企業價值 (Enterprise Value, EV):**
*   EV = PV of FCF + PV of TV = $2.4618 億美元 + $8.6158 億美元 = $11.0776 億美元

**e. 計算股權價值 (Equity Value):**
*   股權價值 = EV - 淨債務 (Net Debt)
*   淨債務 = 總債務 - 現金及現金等價物
*   根據 Stock Analysis，AAP 有 $31.2 億美元的現金和 $56.6 億美元的債務，淨現金頭寸為負 $25.4 億美元。
*   淨債務 = $5.66 億美元 (債務) - $3.12 億美元 (現金) = $2.54 億美元 (此處使用 Stock Analysis 的數據，與之前計算的 D 不同，因為 D 是總債務，而淨債務考慮了現金)。
*   股權價值 = $11.0776 億美元 - $2.54 億美元 = $8.5376 億美元

**f. 計算每股估值:**
*   流通股數 = 市值 / 股價 = $3.37 億美元 / $56.01 = 601.67 萬股 (約 6017 萬股)
*   每股估值 = 股權價值 / 流通股數 = $8.5376 億美元 / 6017 萬股 = $14.19

#### 2.4. 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於支付穩定股息且股息增長可預測的公司。AAP 支付股息，但其盈利和 FCF 波動較大，這使得股息增長預測存在不確定性。

**估值假設:**
*   **當前年度股息 (D0):** $1.00 (基於最新數據)
*   **股息增長率 (g):** 考慮到公司正在轉型且盈利波動，我們假設一個保守的長期股息增長率，例如 2%。
*   **股權成本 (Re):** 12.75% (與 WACC 計算中的 Re 相同)

**a. 計算下一年度股息 (D1):**
*   D1 = D0 * (1 + g) = $1.00 * (1 + 0.02) = $1.02

**b. 計算每股估值:**
*   每股估值 = D1 / (Re - g)
*   每股估值 = $1.02 / (0.1275 - 0.02)
*   每股估值 = $1.02 / 0.1075
*   每股估值 = $9.49

**DDM 模型結果:**
*   **每股估值:** $9.49
*   **合理價位區間:** $8 - $12 (考慮到股息增長的不確定性)
