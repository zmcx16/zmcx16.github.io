### 估值模型與結果

#### 1. 折現現金流模型 (DCF)

**計算自由現金流 (FCFF):**
FCFF = 稅後營業利潤 + 折舊與攤銷 - 資本支出 - 營運資本變動
稅後營業利潤 = 收入 * 營業利潤率 * (1 - 稅率)

*   **折現因子 (Discount Factor):** 1 / (1 + WACC)^n
*   **折現後的 FCFF:**
    *   2026: $186.53 / (1.0488)^1 = $177.85 百萬美元
    *   2027: $257.73 / (1.0488)^2 = $233.25 百萬美元
    *   2028: $297.28 / (1.0488)^3 = $257.00 百萬美元
    *   2029: $332.00 / (1.0488)^4 = $275.80 百萬美元
    *   2030: $360.78 / (1.0488)^5 = $286.00 百萬美元
*   **預測期內 FCFF 現值總和:** $177.85 + $233.25 + $257.00 + $275.80 + $286.00 = $1229.90 百萬美元

**計算終值 (Terminal Value, TV):**
TV = FCFF (2030 年後第一年) / (WACC - g)
FCFF (2031) = FCFF (2030) * (1 + g) = $360.78 * (1 + 0.02) = $367.99 百萬美元
TV = $367.99 / (0.0488 - 0.02) = $367.99 / 0.0288 = $12777.43 百萬美元

**終值現值 (Present Value of Terminal Value):**
PV(TV) = TV / (1 + WACC)^5 = $12777.43 / (1.0488)^5 = $10137.00 百萬美元

**公司總價值 (Enterprise Value):**
EV = 預測期內 FCFF 現值總和 + 終值現值
EV = $1229.90 + $10137.00 = $11366.90 百萬美元

**股權價值 (Equity Value):**
Equity Value = EV + 現金及約當現金 - 總債務
現金及約當現金: $31.2 億美元
總債務: $56.6 億美元
Equity Value = $113.669 億 + $31.2 億 - $56.6 億 = $88.269 億美元

**每股估值:**
每股估值 = 股權價值 / 流通股數
流通股數 = 市值 / 當前股價 = $30.2 億 / $53.09 = 約 56.88 百萬股
每股估值 = $88.269 億 / 56.88 百萬股 = **$155.18 美元**

**DCF 合理價位區間:** 考慮到假設的敏感性，我們設定一個區間。
*   若 WACC 變動 +/- 0.5%，長期增長率變動 +/- 0.5%，每股估值可能在 **$120 - $190 美元** 之間。
