#### **3. 估值方法與計算**

##### **3.1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

1.  **折現率 (WACC - Weighted Average Cost of Capital):**
    *   **無風險利率 (Risk-Free Rate, Rf):** 採用美國 10 年期國債收益率，約 4.2%。
    *   **市場風險溢價 (Market Risk Premium, MRP):** 採用行業標準 5.5%。
    *   **Beta (β):** 1.15。
    *   **股權成本 (Cost of Equity, Ke):** Ke = Rf + β * MRP = 4.2% + 1.15 * 5.5% = 4.2% + 6.325% = 10.525%。
    *   **稅前債務成本 (Cost of Debt, Kd):** 假設 GPN 的平均借貸成本為 6.0%。
    *   **稅率 (Tax Rate):** 21%。
    *   **稅後債務成本 (Kd_after_tax):** Kd * (1 - Tax Rate) = 6.0% * (1 - 0.21) = 4.74%。
    *   **股權市值 (E):** $19.02 Billion。
    *   **債務市值 (D):** $15 Billion (假設債務賬面價值接近市場價值)。
    *   **總資本 (V):** E + D = $19.02B + $15B = $34.02 Billion。
    *   **WACC:** (E/V) * Ke + (D/V) * Kd_after_tax
        = (19.02 / 34.02) * 10.525% + (15 / 34.02) * 4.74%
        = 0.559 * 10.525% + 0.441 * 4.74%
        = 5.88% + 2.09% = 7.97%。
        **為保守起見，我們將 WACC 四捨五入至 8.0%。**

2.  **預測期間:** 5 年的明確預測期，之後進入永續增長階段。

3.  **自由現金流 (FCFF) 預測:**
    *   我們將使用提供的 P/FCF 數據來推導當前 FCFF 作為基數。
    *   當前市值 $19.02B，P/FCF 為 8.8。
    *   因此，當前 FCFF (Year 0) = 市值 / P/FCF = $19.02B / 8.8 = $2.16 Billion。
    *   **增長率假設 (考慮 Netspend 剝離後的業務聚焦和分析師預期):**
        *   Year 1-2: 10% (反映 EPS next Y 增長和業務重組後的效率提升)
        *   Year 3-5: 7% (增長趨於穩定)
        *   永續增長率 (g): 2.5% (略高於長期通脹率，反映成熟行業的溫和增長)。

**DCF 計算步驟：**

*   **預測期自由現金流 (FCFF):**
    *   FCFF_Year 0 = $2.16B
    *   FCFF_Year 1 = $2.16B * (1 + 0.10) = $2.376B
    *   FCFF_Year 2 = $2.376B * (1 + 0.10) = $2.614B
    *   FCFF_Year 3 = $2.614B * (1 + 0.07) = $2.797B
    *   FCFF_Year 4 = $2.797B * (1 + 0.07) = $2.993B
    *   FCFF_Year 5 = $2.993B * (1 + 0.07) = $3.202B

*   **永續價值 (Terminal Value, TV):**
    *   FCFF_Year 6 = FCFF_Year 5 * (1 + g) = $3.202B * (1 + 0.025) = $3.282B
    *   TV = FCFF_Year 6 / (WACC - g) = $3.282B / (0.08 - 0.025) = $3.282B / 0.055 = $59.673 Billion

*   **計算現值 (Present Value):**
    *   PV(FCFF_Y1) = $2.376B / (1.08)^1 = $2.200B
    *   PV(FCFF_Y2) = $2.614B / (1.08)^2 = $2.241B
    *   PV(FCFF_Y3) = $2.797B / (1.08)^3 = $2.220B
    *   PV(FCFF_Y4) = $2.993B / (1.08)^4 = $2.200B
    *   PV(FCFF_Y5) = $3.202B / (1.08)^5 = $2.179B
    *   PV(TV) = $59.673B / (1.08)^5 = $40.630B

*   **企業價值 (Enterprise Value, EV):**
    *   EV = Sum of PV(FCFF) + PV(TV)
    *   EV = ($2.200B + $2.241B + $2.220B + $2.200B + $2.179B) + $40.630B
    *   EV = $11.040B + $40.630B = $51.670 Billion

*   **股權價值 (Equity Value):**
    *   Equity Value = EV + Cash & Equivalents - Total Debt
    *   Equity Value = $51.670B + $2B - $15B = $38.670 Billion

*   **每股內在價值 (Intrinsic Value per Share):**
    *   Intrinsic Value per Share = Equity Value / Shares Outstanding
    *   Intrinsic Value per Share = $38.670B / 236.7M = **$163.37**

**DCF 估值結果與區間：**
*   **DCF 估值：$163.37**
*   **合理價位區間：** 考慮到 WACC 和增長率的敏感性，我們可以設定一個區間。
    *   若 WACC 提高至 8.5%，永續增長率降至 2.0%，則估值會下降。
    *   若 WACC 降低至 7.5%，永續增長率升至 3.0%，則估值會上升。
    *   **DCF 估值區間：$140 - $180**

##### **3.3. 股息折現模型 (Dividend Discount Model, DDM)**

*   **股息率 (Dividend %):** 0.0124
*   **當前股價:** $80.35
*   **每股年度股息 (D0):** $80.35 * 0.0124 = $0.996 (約 $1.00)

GPN 的股息率非常低，表明它並非以股息為主要回報股東的公司，而是將大部分利潤再投資於增長。對於這類成長型公司，DDM 通常不是最合適的估值方法，因為其價值主要來自於未來盈利和現金流的增長，而非穩定的股息。

如果我們強行應用 DDM，假設一個保守的股息增長率 (g_div) 為 5% (高於永續增長率，但仍低於盈利增長)，並使用股權成本 (Ke) 10.525%：
*   D1 = D0 * (1 + g_div) = $1.00 * (1 + 0.05) = $1.05
*   DDM Value = D1 / (Ke - g_div) = $1.05 / (0.10525 - 0.05) = $1.05 / 0.05525 = **$19.00**

**結論：** DDM 估值結果遠低於當前股價和所有其他模型，這再次證明 DDM 不適用於 GPN 這種低股息、高增長潛力的公司。
