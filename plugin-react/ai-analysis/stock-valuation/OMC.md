### 三、 估值模型分析

我們將應用折現現金流模型 (DCF)、相對估值法和股息折現模型 (DDM) 進行估值。

#### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值，以得出其內在價值。

**估值假設：**

*   **預測期間：** 5年 (2024-2028)
*   **長期增長率 (Terminal Growth Rate, g)：** 2.5% (略高於長期通脹和全球GDP增長率，反映成熟服務業的穩定增長)
*   **加權平均資本成本 (Weighted Average Cost of Capital, WACC)：**
    *   **無風險利率 (Risk-Free Rate)：** 4.5% (參考美國10年期國債收益率)
    *   **股權風險溢價 (Equity Risk Premium, ERP)：** 5.5%
    *   **Beta (OMC)：** 1.05 (參考網路搜尋，反映其相對於市場的波動性)
    *   **股權成本 (Cost of Equity, Ke) = 無風險利率 + Beta * ERP**
        *   Ke = 4.5% + 1.05 * 5.5% = 4.5% + 5.775% = 10.275%
    *   **稅前債務成本 (Pre-tax Cost of Debt, Kd)：** 5.0% (參考公司債務利率和市場利率)
    *   **稅率 (Tax Rate)：** 23% (參考美國企業稅率和公司有效稅率)
    *   **稅後債務成本 (After-tax Cost of Debt) = Kd * (1 - Tax Rate)**
        *   After-tax Kd = 5.0% * (1 - 0.23) = 3.85%
    *   **資本結構 (Market Value)：**
        *   市值 (Market Cap, Equity Value) = $25.30 B
        *   總債務 (Total Debt)：根據最新財報（假設約 $10.0 B，需查閱最新10-K/10-Q確認）
        *   企業總價值 (Enterprise Value, V) = $25.30 B (E) + $10.0 B (D) = $35.30 B
        *   股權佔比 (E/V) = $25.30 B / $35.30 B = 0.7167
        *   債務佔比 (D/V) = $10.0 B / $35.30 B = 0.2833
    *   **WACC = (E/V) * Ke + (D/V) * After-tax Kd**
        *   WACC = (0.7167 * 10.275%) + (0.2833 * 3.85%) = 7.36% + 1.09% = **8.45%**

*   **財務預測假設 (基於歷史數據和產業趨勢)：**
    *   **2023年營收 (Revenue)：** 約 $14.7 B (參考最新年報)
    *   **營收增長率 (Revenue Growth Rate)：**
        *   2024年：5.0% (參考 EPS next Y_% 和分析師預期)
        *   2025年：4.5%
        *   2026年：4.0%
        *   2027年：3.5%
        *   2028年：3.0% (逐步趨於長期增長率)
    *   **營運利潤率 (Operating Margin)：** 穩定在 14.0% (參考提供數據)
    *   **資本支出 (CAPEX) / 營收：** 1.8% (服務型公司通常較低)
    *   **折舊與攤銷 (D&A) / 營收：** 1.8% (假設與 CAPEX 相近)
    *   **營運資本變動 (Change in NWC) / 營收變動：** 0.5% (假設輕微正向變動)

**DCF 計算步驟 (簡化展示)：**

1.  **計算預測期內的自由現金流 (FCF)：**
    *   營收 (Revenue)
    *   營運利潤 (EBIT = Revenue * Oper. Margin)
    *   稅後營運利潤 (NOPAT = EBIT * (1 - Tax Rate))
    *   加上折舊與攤銷 (D&A)
    *   減去資本支出 (CAPEX)
    *   減去營運資本變動 (Change in NWC)
    *   得到 FCF

2.  **計算終端價值 (Terminal Value, TV)：**
    *   TV = FCF_year6 * (1 + g) / (WACC - g)
    *   FCF_year6 是預測期結束後第一年的 FCF。

3.  **將所有 FCF 和 TV 折現至現值 (Present Value, PV)。**

4.  **企業價值 (Enterprise Value) = PV(FCF) + PV(TV)**

5.  **股權價值 (Equity Value) = Enterprise Value - 淨債務 (Net Debt)**
    *   淨債務 = 總債務 - 現金及約當現金 (假設 $10.0 B - $1.5 B = $8.5 B)

6.  **每股內在價值 (Intrinsic Value per Share) = Equity Value / 流通股數 (Shares Outstanding)**

#### 3. 股息折現模型 (Dividend Discount Model, DDM)

OMC 具有穩定的股息支付歷史和可觀的股息收益率 (3.62%)，因此 DDM 適用。

**估值假設：**

*   **當前每股股息 (D0)：** $80.03 * 3.62% = $2.90
*   **股權成本 (Ke)：** 10.275% (與 DCF 模型相同)
*   **股息增長率 (g)：**
    *   短期 (3年)：7.0% (考慮到公司穩定的盈利和再投資能力，但低於其高 ROE 推導出的可持續增長率，因為成熟公司通常不會將所有留存收益都轉化為股息增長)
    *   長期 (永續)：3.0% (與長期通脹和經濟增長率掛鉤)

**DDM 計算步驟 (兩階段模型)：**

1.  **計算預測期內的股息：**
    *   D1 = D0 * (1 + g_short)
    *   D2 = D1 * (1 + g_short)
    *   D3 = D2 * (1 + g_short)

2.  **計算預測期結束後的終端股息價值 (Terminal Dividend Value)：**
    *   D4 = D3 * (1 + g_long)
    *   TV_dividend = D4 / (Ke - g_long) (在預測期結束時的價值)

3.  **將所有股息和終端股息價值折現至現值。**

**DDM 估值結果：**

*   D0 = $2.90
*   D1 = $2.90 * 1.07 = $3.103
*   D2 = $3.103 * 1.07 = $3.320
*   D3 = $3.320 * 1.07 = $3.552
*   D4 (用於 TV) = $3.552 * 1.03 = $3.660
*   **PV of D1：** $3.103 / (1.10275)^1 = $2.814
*   **PV of D2：** $3.320 / (1.10275)^2 = $2.720
*   **PV of D3：** $3.552 / (1.10275)^3 = $2.630
*   **終端股息價值 (TV_dividend) (在 Year 3 結束時)：** $3.660 / (10.275% - 3.0%) = $3.660 / 0.07275 = $50.31

*   **每股內在價值 (Intrinsic Value per Share) = PV(D1) + PV(D2) + PV(D3) + PV(TV_dividend)**
    *   = $2.814 + $2.720 + $2.630 + $37.80 = **$45.96**

考慮到股息增長率和股權成本的敏感性：

*   長期股息增長率 2.5% - 3.5%
*   **DDM 合理價位區間：$40 - $55**
