**數據一致性說明：**
提供的基本面數據中，"Close" 價格 $25.82 與 "Market Cap" $154.33B 存在顯著不一致。Sony Group Corporation 的實際流通股數約為 1.25 億股。若以 $25.82 作為每股價格，則市值應為 $25.82 * 1.25B = $32.275B，遠低於提供的 $154.33B。反之，若以 $154.33B 為市值，則每股價格應為 $154.33B / 1.25B = $123.46。

### 2. 估值假設

3.  **營業利益率 (Operating Margin)：**
    *   FY2023 (實際)：9.28% ($7.75B / $83.5B)
    *   FY2024 (基於公司財測)：10.32% ($8.13B / $78.9B)
    *   FY2025-FY2028：逐步提升並穩定在 10.5% - 10.8% (反映公司對獲利能力的重視)。
4.  **稅率 (Tax Rate)：** 25% (基於公司歷史有效稅率及日本企業稅率)。
5.  **折舊與攤銷 (D&A) / 資本支出 (CapEx)：** 假設 D&A 和 CapEx 均為營收的 6% (基於歷史數據和成熟公司的趨勢)。
6.  **營運資本變動 (Change in NWC)：** 假設為營收變動的 5% (營收增長時為現金流出，營收下降時為現金流入)。
7.  **永續增長率 (Terminal Growth Rate, g)：** 1.5% (保守估計，低於WACC，反映成熟企業的長期增長)。
8.  **加權平均資本成本 (WACC) 計算：**
    *   **無風險利率 (Risk-Free Rate, Rf)：** 4.3% (參考美國10年期國債收益率)。
    *   **市場風險溢酬 (Market Risk Premium, MRP)：** 5.5%。
    *   **Beta (β)：** 1.05 (參考Yahoo Finance等數據源)。
    *   **股權成本 (Cost of Equity, Ke) = Rf + β * MRP**
        *   Ke = 4.3% + 1.05 * 5.5% = 4.3% + 5.775% = 10.075%
    *   **債務成本 (Cost of Debt, Kd)：** 6.8% (假設無風險利率 + 2.5%的信用利差)。
    *   **稅後債務成本 = Kd * (1 - Tax Rate)**
        *   稅後 Kd = 6.8% * (1 - 0.25) = 5.1%
    *   **債務權益比 (Debt/Equity Ratio)：** 0.21 (來自提供數據)。
    *   **股權權重 (E/V) = 1 / (1 + D/E) = 1 / (1 + 0.21) = 0.8264**
    *   **債務權重 (D/V) = D/E / (1 + D/E) = 0.21 / (1 + 0.21) = 0.1736**
    *   **WACC = (E/V * Ke) + (D/V * 稅後 Kd)**
        *   WACC = (0.8264 * 10.075%) + (0.1736 * 5.1%) = 8.326% + 0.885% = 9.211%
        *   **WACC 約為 9.2%。**

*   **股息增長率 (g)：** 2%
