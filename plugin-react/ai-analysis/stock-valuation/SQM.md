#### **4.1 折現現金流模型 (DCF)**

1.  計算預測期內的稅後營運利潤 (NOPAT)。
2.  計算預測期內的自由現金流 (FCF = NOPAT + D&A - CapEx - Change in NWC)。
3.  計算預測期末的終端價值 (Terminal Value, TV)。
4.  將所有 FCF 和 TV 折現至現值。
5.  計算企業價值 (Enterprise Value, EV)。
6.  計算股權價值 (Equity Value = EV + Cash - Debt)。
7.  計算每股價值 (Per Share Value = Equity Value / Shares Outstanding)。
