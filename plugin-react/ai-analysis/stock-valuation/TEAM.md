### 三、折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型旨在估計公司未來自由現金流的現值。由於Atlassian的GAAP利潤為負，我們將基於其強勁的自由現金流 (FCF) 表現進行預測。

**1. WACC 計算 (加權平均資本成本):**

*   **無風險利率 (Risk-Free Rate):** 假設為美國10年期國債收益率，約 4.5%。
*   **市場風險溢價 (Market Risk Premium, MRP):** 假設為 5.5%。
*   **Beta 值 (Beta):** 根據市場數據，Atlassian的Beta值約為 1.3。
*   **股權成本 (Cost of Equity, Ke):**
    *   Ke = 無風險利率 + Beta * MRP
    *   Ke = 4.5% + 1.3 * 5.5% = 4.5% + 7.15% = 11.65%
*   **債務成本 (Cost of Debt, Kd):** 假設為 6.0% (考慮其可轉換票據等債務成本)。
*   **稅率 (Tax Rate, t):** 21% (美國企業稅率)。
*   **股權市值 (Market Value of Equity, E):** $43.31 Billion (當前市值)。
*   **債務市值 (Market Value of Debt, D):** 根據最新財報，Atlassian的總債務約為 $1.0 Billion (主要為可轉換票據)。
*   **WACC = (E / (E + D)) * Ke + (D / (E + D)) * Kd * (1 - t)**
    *   E + D = $43.31B + $1.0B = $44.31B
    *   WACC = (43.31 / 44.31) * 11.65% + (1.0 / 44.31) * 6.0% * (1 - 0.21)
    *   WACC = 0.977 * 11.65% + 0.023 * 6.0% * 0.79
    *   WACC = 11.38% + 0.11% = 11.49%
    *   為簡化計算並提供一定安全邊際，我們取 **WACC = 11.5%**。

**2. 自由現金流 (FCF) 預測：**

我們將基於Atlassian過去的FCF表現和預期增長率進行預測。根據最新財報，Atlassian的季度FCF約為$407.7M，年化約為$1.6B。

*   **FY2024 FCF (基準):** $1.60 Billion (基於最新年化數據)
*   **FCF 增長率假設:**
    *   FY2025: 20% (考慮其營收增長和雲端轉型)
    *   FY2026: 15%
    *   FY2027: 10%
    *   FY2028: 5%
    *   FY2029: 3% (趨於穩定)

| 年度 | 預測自由現金流 (FCF, $B) |
| :--- | :----------------------- |
| FY2024 | 1.60                     |
| FY2025 | 1.60 * (1 + 0.20) = 1.92 |
| FY2026 | 1.92 * (1 + 0.15) = 2.21 |
| FY2027 | 2.21 * (1 + 0.10) = 2.43 |
| FY2028 | 2.43 * (1 + 0.05) = 2.55 |
| FY2029 | 2.55 * (1 + 0.03) = 2.63 |

**3. 終值 (Terminal Value, TV) 計算：**

*   永續增長率 (g): 3.0%
*   TV = FCF_FY2030 / (WACC - g)
*   FCF_FY2030 = FCF_FY2029 * (1 + g) = $2.63B * (1 + 0.03) = $2.709B
*   TV = $2.709B / (0.115 - 0.030) = $2.709B / 0.085 = $31.87 Billion

**4. 企業價值 (Enterprise Value, EV) 計算：**

*   **預測期 FCF 現值 (PV of FCFs):**
    *   FY2025: $1.92B / (1.115)^1 = $1.722B
    *   FY2026: $2.21B / (1.115)^2 = $1.775B
    *   FY2027: $2.43B / (1.115)^3 = $1.758B
    *   FY2028: $2.55B / (1.115)^4 = $1.668B
    *   FY2029: $2.63B / (1.115)^5 = $1.541B
    *   總和 = $1.722B + $1.775B + $1.758B + $1.668B + $1.541B = $8.464 Billion
*   **終值現值 (PV of TV):**
    *   PV of TV = $31.87B / (1.115)^5 = $18.67 Billion
*   **企業價值 (EV) = 預測期 FCF 現值 + 終值現值**
    *   EV = $8.464B + $18.67B = $27.134 Billion

**5. 股權價值與每股價值：**

*   **股權價值 (Equity Value) = EV + 現金及等價物 - 總債務**
    *   現金及等價物 (來自最新財報估計): 約 $2.0 Billion
    *   總債務 (來自最新財報估計): 約 $1.0 Billion
    *   Equity Value = $27.134B + $2.0B - $1.0B = $28.134 Billion
*   **流通股數 (Shares Outstanding):** 根據最新數據，約 255 Million 股。
*   **DCF 估值每股價格 = 股權價值 / 流通股數**
    *   DCF 估值每股價格 = $28.134B / 255M = **$110.33**

**DCF 估值結果與合理價位區間：**
基於上述假設，DCF模型估計Atlassian的合理股價約為 **$110.33**。考慮到假設的敏感性，合理價位區間約為 **$100 - $120**。
