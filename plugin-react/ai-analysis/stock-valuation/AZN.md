    *   **Beta (AZN):** 0.8 (製藥公司通常具有較低的 Beta，因其業務相對穩定)
    *   **股權成本 (Cost of Equity, Ke) = Rf + Beta * MRP**
        *   Ke = 4.2% + 0.8 * 5.5% = 4.2% + 4.4% = 8.6%

#### **3. 股息折現模型 (Dividend Discount Model, DDM)**

**估值假設：**

    *   我們假設一個兩階段 DDM：
        *   前 5 年：股息增長率 4% (低於 EPS 增長，反映再投資)
        *   永續增長率：3% (長期可持續增長)

**計算：**
