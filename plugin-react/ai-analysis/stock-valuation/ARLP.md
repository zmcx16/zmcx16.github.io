### **II. 估值假設**

為了進行估值，我們需要設定一些關鍵假設。

**A. 加權平均資本成本 (WACC) 計算**

WACC 是折現現金流的折現率，代表公司為其資產融資的平均成本。

1.  **無風險利率 (Risk-Free Rate, Rf):** 4.30% (參考美國10年期國債收益率)
2.  **股權風險溢價 (Equity Risk Premium, ERP):** 5.50% (市場普遍使用的範圍)
3.  **Beta (β):** 0.90 (參考Yahoo Finance等數據源，ARLP的Beta值通常在0.8-1.0之間，考慮其相對穩定的現金流和高股息，取中等偏低值)
4.  **股權成本 (Cost of Equity, Ke):**
    *   Ke = Rf + β * ERP
    *   Ke = 4.30% + 0.90 * 5.50% = 4.30% + 4.95% = 9.25%

5.  **債務成本 (Cost of Debt, Kd):**
    *   根據ARLP的財務報表，其長期債務約為 $800-$900M，利息費用約 $50-$60M。
    *   Kd ≈ $55M / $900M = 6.11%
    *   我們取 6.10%

6.  **稅率 (Tax Rate, t):** 20% (考慮到ARLP作為有限合夥企業的稅務結構可能與一般公司不同，但為計算NOPAT和WACC，我們使用一個合理的有效稅率，參考其歷史有效稅率和美國企業稅率)

7.  **資本結構：**
    *   市值 (Market Value of Equity, E): $3.04 Billion
    *   債務市值 (Market Value of Debt, D): $0.90 Billion (參考其總債務)
    *   總資本 (E+D): $3.04B + $0.90B = $3.94 Billion
    *   股權佔比 (E/(E+D)): $3.04B / $3.94B = 77.16%
    *   債務佔比 (D/(E+D)): $0.90B / $3.94B = 22.84%

8.  **WACC 計算：**
    *   WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1 - t)
    *   WACC = 0.7716 * 9.25% + 0.2284 * 6.10% * (1 - 0.20)
    *   WACC = 0.07137 + 0.01116 = 0.08253
    *   **WACC ≈ 8.25%**

**B. 增長率假設**

*   **預測期間：** 5年 (2024-2028)
*   **短期增長率 (未來5年)：** 考慮到煤炭產業的長期挑戰和ARLP的多元化努力，我們採取保守的增長假設。

*   **永續增長率 (Terminal Growth Rate, g):** 0.50% (非常保守，反映煤炭產業的長期衰退趨勢，但ARLP的O&G業務和成本效率可能使其能維持微弱的長期增長)。

**2. 計算終端價值 (Terminal Value, TV)**
