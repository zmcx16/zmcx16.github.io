### 2. 估值假設

### 3. 折現現金流模型 (Discounted Cash Flow, DCF)

**計算步驟：**

1.  **預測未來 5 年的自由現金流 (FCF)：**

    *   2024 FCF: $4.54B * (1 + 0.12) = $5.08 Billion

    *   2026 FCF: $5.59B * (1 + 0.08) = $6.04 Billion
    *   2027 FCF: $6.04B * (1 + 0.06) = $6.40 Billion
    *   2028 FCF: $6.40B * (1 + 0.05) = $6.72 Billion

2.  **計算終端價值 (Terminal Value, TV)：**
    *   TV = FCF_2029 * (1 + g) / (WACC - g)
    *   FCF_2029 = FCF_2028 * (1 + g) = $6.72B * (1 + 0.028) = $6.91 Billion
    *   TV = $6.91B / (0.110 - 0.028) = $6.91B / 0.082 = **$84.27 Billion**

3.  **將預測期 FCF 和終端價值折現至現值 (PV)：**
    *   PV(FCF_2024) = $5.08B / (1 + 0.110)^1 = $4.58 Billion
    *   PV(FCF_2025) = $5.59B / (1 + 0.110)^2 = $4.52 Billion
    *   PV(FCF_2026) = $6.04B / (1 + 0.110)^3 = $4.42 Billion
    *   PV(FCF_2027) = $6.40B / (1 + 0.110)^4 = $4.22 Billion
    *   PV(FCF_2028) = $6.72B / (1 + 0.110)^5 = $3.99 Billion
    *   PV(TV) = $84.27B / (1 + 0.110)^5 = $50.00 Billion

4.  **計算企業價值 (Enterprise Value, EV)：**
    *   EV = PV(FCF_2024) + PV(FCF_2025) + PV(FCF_2026) + PV(FCF_2027) + PV(FCF_2028) + PV(TV)
    *   EV = $4.58B + $4.52B + $4.42B + $4.22B + $3.99B + $50.00B = **$71.73 Billion**

5.  **計算股權價值 (Equity Value)：**
    *   Equity Value = EV + 淨現金 (Net Cash)
    *   Equity Value = $71.73B + $8.00B = **$79.73 Billion**

6.  **計算每股內在價值 (Intrinsic Value per Share)：**
