#### 2.1 加權平均資本成本 (WACC) 計算

WACC 是 DCF 模型中的折現率，代表公司為其資產融資的平均成本。

**a. 股權成本 (Cost of Equity, Ke) - 使用資本資產定價模型 (CAPM)**
*   **無風險利率 (Risk-Free Rate, Rf):** 4.2% (參考當前美國10年期國債收益率)
*   **市場風險溢酬 (Market Risk Premium, MRP):** 5.5% (行業常用假設)
*   **Beta (β):** 0.75 (消費必需品公司通常具有較低的 Beta 值，參考金融數據提供商)
*   **Ke = Rf + β * MRP**
    *   Ke = 4.2% + 0.75 * 5.5% = 4.2% + 4.125% = **8.325%**

**b. 債務成本 (Cost of Debt, Kd)**
*   **長期債務權益比 (LT Debt/Eq):** 1.31
*   **總債務權益比 (Debt/Eq):** 1.47
*   假設總債務約為 $12.5 Billion (基於近期財報數據，並結合 Debt/Eq 1.47 * Equity $9.5B = $13.9B，取一個中間值)
*   假設現金及約當現金約為 $0.5 Billion (基於近期財報數據)
*   **利息費用 (Interest Expense):** 假設為 $0.5 Billion (參考近期財報)
*   **稅前債務成本 (Pre-tax Kd) = 利息費用 / 總債務**
    *   Pre-tax Kd = $0.5B / $12.5B = **4.0%**
*   **稅率 (Tax Rate, t):** 22% (參考公司有效稅率)
*   **稅後債務成本 (After-tax Kd) = Pre-tax Kd * (1 - t)**
    *   After-tax Kd = 4.0% * (1 - 0.22) = 4.0% * 0.78 = **3.12%**

**c. 資本結構權重**
*   **股權市值 (E):** $25.11 Billion (Market Cap)
*   **債務市值 (D):** 假設債務的帳面價值接近市場價值 = $12.5 Billion
*   **總資本 (V) = E + D**
    *   V = $25.11B + $12.5B = $37.61 Billion
*   **股權權重 (We) = E / V**
    *   We = $25.11B / $37.61B = **0.6676**
*   **債務權重 (Wd) = D / V**
    *   Wd = $12.5B / $37.61B = **0.3324**

**d. WACC 計算**
*   **WACC = (We * Ke) + (Wd * After-tax Kd)**
    *   WACC = (0.6676 * 8.325%) + (0.3324 * 3.12%)
    *   WACC = 5.556% + 1.038% = **6.594%**
    *   為簡化計算，我們將 WACC 四捨五入為 **6.6%**。
