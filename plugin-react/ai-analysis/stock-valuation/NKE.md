### 二、 估值方法與計算

#### 1. 加權平均資本成本 (Weighted Average Cost of Capital, WACC)

WACC 是用於折現未來現金流的折現率。

**假設:**
*   **無風險利率 (Risk-Free Rate, Rf):** 4.188% (美國10年期國債收益率，截至2025年11月)
*   **Beta (β):** 1.14
*   **市場風險溢酬 (Market Risk Premium, MRP):** 6.0%
*   **股權成本 (Cost of Equity, Ke) = Rf + β \* MRP**
    *   Ke = 4.188% + 1.14 \* 6.0% = 11.028%
*   **債務成本 (Cost of Debt, Kd):** 由於 Nike 2025財年利息支出為負值，我們採用保守估計，假設稅前債務成本為 5.0% (基於其A2信用評級)。
*   **稅率 (Tax Rate):** 17.1% (2025財年有效稅率)
*   **稅後債務成本 (Kd_at) = Kd \* (1 - Tax Rate)**
    *   Kd_at = 5.0% \* (1 - 0.171) = 4.145%
*   **股權權重 (We):** 0.89 (市值 $92.79B / (市值 $92.79B + 債務 $11.4674B))
*   **債務權重 (Wd):** 0.11 (債務 $11.4674B / (市值 $92.79B + 債務 $11.4674B))

**WACC = (We \* Ke) + (Wd \* Kd_at)**
WACC = (0.89 \* 11.028%) + (0.11 \* 4.145%) = 9.815% + 0.456% = 10.271%

**採用 WACC = 10.3%**

#### 2. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型評估公司未來自由現金流的現值。

**估值假設 (樂觀情境):**
*   **預測期間:** 5年 (2026財年 - 2030財年)
*   **永續成長率 (Terminal Growth Rate, g):** 3.0% (保守的長期成長率)
*   **WACC:** 10.3%
*   **營收成長率:**
    *   2026財年: 3.0% (基於分析師預期與TIKR模型)
    *   2027財年: 6.0% (接近5年複合成長率預期)
    *   2028財年: 6.0%
    *   2029財年: 5.0%
    *   2030財年: 4.0%
*   **營運利潤率 (Operating Margin):** 逐步改善，從2025財年的7.99%提升。
    *   2026財年: 9.0%
    *   2027財年: 10.5%
    *   2028財年: 12.0%
    *   2029財年: 13.0%
    *   2030財年: 13.5% (仍略低於Simply Wall St預計的13.65%)
*   **稅率:** 17.1%
*   **資本支出 (CapEx):** 營收的 2.5%
*   **折舊與攤銷 (D&A):** 營收的 2.0%
*   **淨營運資金變動 (Change in NWC):** 營收變動的 10%

**計算過程 (簡化):**

*   **終端價值 (Terminal Value, TV):**
    *   TV = FCF_2030 \* (1 + g) / (WACC - g) = $6,031M \* (1 + 0.03) / (0.103 - 0.03) = $85,093M
*   **終端價值現值 (PV of TV):** $85,093M / (1.103)^5 = $52,191M
*   **企業價值 (Enterprise Value, EV):** 預測期FCF現值總和 + 終端價值現值 = $2,885M + $3,163M + $3,545M + $3,734M + $3,698M + $52,191M = $69,216M
*   **股權價值 (Equity Value):** EV + 現金及約當現金 - 總債務
    *   現金及約當現金 (2025財年): $7,464M
    *   總債務 (2025年11月): $11,467M
    *   股權價值 = $69,216M + $7,464M - $11,467M = $65,213M
*   **每股內在價值 (Intrinsic Value per Share):** 股權價值 / 流通股數
    *   流通股數 (基於當前市值): 1,452 百萬股
    *   每股內在價值 = $65,213M / 1,452M = $44.91

**DCF 模型估值結果:**
*   **每股內在價值:** $44.91
*   **合理價位區間:** **$40 - $50**

#### 4. 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於支付穩定股息且有可預測成長的公司。

**估值假設:**
*   **當前每股股息 (D0):** $1.64
*   **股權成本 (Ke):** 11.028%
*   **股息成長率 (g):** 6.0% (低於短期預期，但更符合長期成熟公司成長)

**DDM 計算 (Gordon Growth Model):**
*   **每股價值 = D1 / (Ke - g)**
*   D1 = $1.64 \* (1 + 0.06) = $1.7384
*   每股價值 = $1.7384 / (0.11028 - 0.06) = $1.7384 / 0.05028 = $34.57

**DDM 模型估值結果:**
*   **每股內在價值:** $34.57
*   **合理價位區間:** **$30 - $40**

### 三、 綜合結論

以下是各估值模型的結果與當前市場價格的比較：

*   **當前市場價格:** $63.90
*   **DCF 模型 (樂觀情境):** $44.91 (區間: $40 - $50)
*   **相對估值 (P/E):** $42.50 - $51.00
*   **相對估值 (P/S):** $47.84 - $63.78
*   **相對估值 (EV/EBITDA):** ~$29.67
*   **DDM 模型:** $34.57 (區間: $30 - $40)
*   **分析師目標價:** $77.24

**分析與比較:**

1.  **DCF 和 DDM 模型:** 這兩個內在價值模型均顯示 Nike 的每股內在價值遠低於當前市場價格，主要落在 $30 - $50 的區間。這表明基於對未來現金流和股息的保守預期，Nike 目前可能被高估。
2.  **相對估值:**
    *   基於 P/E 和 P/S 的估值範圍較廣，P/S 估值區間的上限接近當前股價，但 P/E 估值仍低於當前股價。
    *   EV/EBITDA 估值則顯著低於當前股價，僅約 $29.67。Nike 當前的 EV/EBITDA (24.96) 遠高於同業平均 (12.2)，這強烈暗示市場對 Nike 賦予了顯著的溢價，可能基於其品牌實力或未來成長潛力。
3.  **分析師目標價:** 分析師的目標價 ($74.61 - $77.24) 顯著高於所有內在估值模型，甚至高於當前股價。這可能反映了分析師對 Nike "Win Now" 策略的成功、品牌護城河 以及長期市場領導地位的更樂觀預期。

**綜合結論:**

綜合來看，儘管 Nike 擁有強大的品牌影響力、全球市場地位和 "寬護城河" 的競爭優勢，但根據多數估值模型（DCF、DDM 和基於 EV/EBITDA 的相對估值），其當前股價 $63.90 似乎被高估。

市場可能正在為 Nike 的品牌溢價、未來更強勁的成長復甦（尤其是在其 "Win Now" 策略和成本削減措施下）以及長期盈利能力支付更高的價格。然而，從嚴格的內在價值角度來看，目前的價格並未得到充分支持。

**投資建議:**

對於尋求價值投資的投資者而言，Nike 目前的股價可能不具吸引力。考慮到其內在估值與市場價格之間的差異，投資者可能需要等待股價回調，或者有更明確的證據表明其營收和利潤率能實現顯著且可持續的增長，以證明當前的高估值是合理的。

**合理價位區間 (綜合):** 綜合各模型結果，Nike 的合理估值範圍可能落在 **$40 - $55** 之間。
