##### **2.1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值。

**步驟:**

1.  **計算加權平均資本成本 (WACC)。**
2.  **預測未來 5 年的自由現金流 (FCFF)。**
3.  **計算終端價值 (Terminal Value)。**
4.  **將預測期內的 FCFF 和終端價值折現回現值。**

**2.1.1. 計算 WACC**

    *   Ke = 無風險利率 + Beta \* 市場風險溢價

    *   稅後債務成本 = Kd \* (1 - 稅率)
    *   稅後債務成本 = 5.0% \* (1 - 20%) = 4.0%

*   **WACC 計算:**
