*   股價 = D1 / (r - g)

    *   r = 所需報酬率 (Cost of Equity)
    *   g = 永續股息增長率

**計算：**

*   當前年化股息 (D0) = $1.64
*   預期股息增長率 (g) = 3.0%
*   下一期預期股息 (D1) = $1.64 * (1 + 0.03) = $1.6892

*   **所需報酬率 (r) = 7.5%：**
    *   估值 = $1.6892 / (0.075 - 0.03) = $1.6892 / 0.045 = **$37.54**
*   **所需報酬率 (r) = 8.5%：**
    *   估值 = $1.6892 / (0.085 - 0.03) = $1.6892 / 0.055 = **$30.71**
*   **所需報酬率 (r) = 9.5%：**
    *   估值 = $1.6892 / (0.095 - 0.03) = $1.6892 / 0.065 = **$25.99**

**股息折現模型估值結果：**

*   **合理價位區間：** $25.99 - $37.54
