*   股權成本 (Ke) = 11.04%
*   短期股息增長率 (g1) = 5.0% (未來 5 年)
*   長期股息增長率 (g2) = 2.5% (永續增長)

**步驟 1: 計算預測期內的股息**

| 年度 | D0 * (1+g1)^n | 股息 (D) | 折現因子 (1 / (1+Ke)^n) | 折現股息 |
| :--- | :------------ | :------- | :----------------------- | :------- |
| 2026 | 1.64 * 1.05^1 | 1.72     | 0.9006                   | 1.55     |
| 2027 | 1.64 * 1.05^2 | 1.81     | 0.8111                   | 1.47     |
| 2028 | 1.64 * 1.05^3 | 1.90     | 0.7304                   | 1.39     |
| 2029 | 1.64 * 1.05^4 | 1.99     | 0.6578                   | 1.31     |
| 2030 | 1.64 * 1.05^5 | 2.09     | 0.5924                   | 1.24     |

**步驟 2: 計算永續價值 (Terminal Value of Dividends)**

*   D_2031 = D_2030 * (1 + g2) = 2.09 * (1 + 0.025) = 2.14
*   TV_DDM = D_2031 / (Ke - g2) = 2.14 / (0.1104 - 0.025) = 2.14 / 0.0854 = $25.06
*   折現永續價值 = TV_DDM * 折現因子_2030 = 25.06 * 0.5924 = $14.85

**步驟 3: 計算每股內在價值**

*   每股內在價值 = 折現股息總和 + 折現永續價值
*   每股內在價值 = (1.55 + 1.47 + 1.39 + 1.31 + 1.24) + 14.85 = 6.96 + 14.85 = **$21.81**

**DDM 模型估值結果:**
*   **每股合理價位:** **$21.81**
*   **合理價位區間:** 考慮到股息增長率和股權成本的敏感性，估值可能在 **$18 - $25** 之間。
