#### 3.1 折現現金流模型 (DCF)

**計算 FCFF (自由現金流到公司)**:
FCFF = 稅後營業利潤 (NOPAT) + 折舊與攤銷 - 資本支出 - 營運資本變動
NOPAT = 營業利潤 * (1 - 稅率)

**步驟**:
1.  **計算 NOPAT**: 預計營收 * 營業利潤率 * (1 - 稅率)
2.  **計算 CapEx**: 預計營收 * CapEx 佔營收比例
3.  **計算 D&A**: 預計營收 * D&A 佔營收比例
4.  **計算 Change in NWC**: 預計營收變動 * NWC 佔營收比例
5.  **計算 FCFF**: NOPAT + D&A - CapEx - Change in NWC
6.  **計算終端價值 (Terminal Value, TV)**: TV = (FCFF_last_year * (1 + 永續增長率)) / (WACC - 永續增長率)
7.  **折現 FCFF 和 TV**: 將每年的 FCFF 和 TV 折現回現值。
8.  **計算企業價值 (Enterprise Value)**: 折現後的 FCFF 總和 + 折現後的 TV。
9.  **計算股權價值 (Equity Value)**: 企業價值 + 現金及約當現金 - 總債務。
10. **計算每股價值 (Intrinsic Value per Share)**: 股權價值 / 流通股數。

**假設**:

*   短期股息增長率 (g1): 6.0% (持續 5 年)

2.  **計算第 5 年末的終端股息價值**: D5 = D0 * (1 + g1)^5
3.  **計算終端價值 (Terminal Value of Dividends)**: TV = D5 * (1 + g2) / (Ke - g2)

2.  **相對估值模型**的結果範圍較廣，從 21.17 美元到 45.76 美元不等。其中，基於 P/E 和 EV/EBITDA 的估值與 DCF 結果較為接近，而基於 P/S 的估值略高。P/FCF 由於 Nike 自由現金流的波動性，導致估值異常高，因此在綜合判斷時應謹慎對待或排除。
3.  **DDM 模型**給出的估值為 27.49 美元，介於 DCF 和相對估值的中低端。DDM 適用於穩定派發股息的公司，但其結果也高度依賴於股息增長率的假設。

**綜合結論**:

所有三個估值模型，在我們設定的合理假設下，都顯示 Nike (NKE) 的內在價值遠低於其當前股價 62.24 美元。

*   DCF 估值約為 **21.58 美元**。
*   相對估值 (中位數) 約為 **30-35 美元** (排除 P/FCF 異常值)。
*   DDM 估值約為 **27.49 美元**。

這表明市場可能對 Nike 未來的增長潛力、品牌價值和轉型成功抱有更高的期望，或者當前股價存在顯著的高估。Nike 面臨著宏觀經濟逆風、市場競爭加劇 (來自 On Running 和 Hoka 等新興品牌) 以及自身轉型帶來的短期營收壓力。公司削減成本和加速自動化的舉措 可能有助於長期盈利能力，但短期內仍存在不確定性。

**合理價位區間綜合判斷**: 考慮到各模型的優缺點和結果差異，並傾向於保守估計，我認為 Nike 的合理價位可能落在 **25 美元 - 40 美元/股** 之間。當前股價 62.24 美元顯然高於這個區間。

### 5. 與同產業可比公司的估值比較

從相對估值數據來看：

*   **EV/EBITDA (TTM)**: Nike 的數據未直接提供，但可比公司 Adidas (約 9-13) 和 Skechers (約 7-10) 的 EV/EBITDA 乘數相對較低。如果 Nike 的 EV/EBITDA 與同行接近，那麼其當前股價可能被高估。

總體而言，與主要競爭對手相比，Nike 在多個估值乘數上都顯得較高，尤其是在 P/E 和 Forward P/E 方面。這可能反映了其作為行業領導者的地位和品牌溢價，但也提示投資者應警惕其估值可能偏高。公司在 2024 年面臨行業逆風和自身轉型挑戰，其高估值需要強勁的未來增長來支撐。
