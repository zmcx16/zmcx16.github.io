**DCF 計算步驟 (簡化)：**

1.  **預測營收 (Revenue)**
2.  **預測營業利潤 (Operating Income = Revenue * Operating Margin)**
3.  **計算稅後營業利潤 (NOPAT = Operating Income * (1 - Tax Rate))**
4.  **計算自由現金流 (FCFF = NOPAT + D&A - CapEx - Change in NWC)**
5.  **計算終值 (Terminal Value, TV)**
    *   TV = (FCFF_last_year * (1 + Terminal Growth Rate)) / (WACC - Terminal Growth Rate)
6.  **將預測期內的 FCFF 和 TV 折現至現值**
7.  **加總所有現值得到企業價值 (Enterprise Value)**
8.  **從企業價值中減去淨債務，加上非經營性資產，得到股權價值 (Equity Value)**
9.  **股權價值除以流通股數得到每股估值**

    *   NKE 營收 (TTM) = 46.51 億美元
    *   NKE 市值 = 46.51 億美元 * 1.9325 = 89.97 億美元
    *   每股估值 = 89.97 億美元 / (791.6 億美元 / 53.44 美元) * 1 (此處計算有誤，應為市值除以流通股數)
    *   NKE 流通股數 = 市值 / 股價 = 79.16 億美元 / 53.44 美元 = 1.481 億股
    *   每股估值 = (46.51 億美元 * 1.9325) / 1.481 億股 = 89.97 億美元 / 1.481 億股 = 60.75 美元

**相對估值合理價位區間：44.05 美元 - 60.75 美元**

#### **2.3 股息折現模型 (Dividend Discount Model, DDM)**

DDM 適用於派發穩定股息且預計未來股息會增長的公司。Nike 派發股息，因此適用。

**估值假設：**

*   **當前每股股息 (D0)：** Nike 最近宣布季度股息為 0.41 美元，因此年度股息為 0.41 * 4 = 1.64 美元。
*   **股息增長率 (g)：** 考慮到 Nike 的成熟性和當前挑戰，我們假設短期 (5 年) 股息增長率為 5%，長期增長率為 2.5% (與 DCF 的長期增長率一致)。
*   **折現率 (r)：** 採用 WACC 9.53%。

**兩階段 DDM 模型：**

1.  **預測期股息現值 (Present Value of Dividends in Forecast Period)**
2.  **終值現值 (Present Value of Terminal Value of Dividends)**

**計算步驟：**

*   D0 = 1.64 美元
*   短期增長率 (g1) = 5%
*   長期增長率 (g2) = 2.5%
*   折現率 (r) = 9.53%
*   預測期 = 5 年

| 年度 | 股息 (D_n = D0 * (1+g1)^n) | 折現因子 (1/(1+r)^n) | 股息現值 |
| :--- | :------------------------- | :------------------- | :------- |
| 1    | 1.64 * 1.05 = 1.722        | 0.913                | 1.57     |
| 2    | 1.722 * 1.05 = 1.808       | 0.834                | 1.51     |
| 3    | 1.808 * 1.05 = 1.898       | 0.762                | 1.45     |
| 4    | 1.898 * 1.05 = 1.993       | 0.696                | 1.39     |
| 5    | 1.993 * 1.05 = 2.093       | 0.636                | 1.33     |

**預測期股息現值總和 = 1.57 + 1.51 + 1.45 + 1.39 + 1.33 = 7.25 美元**

**終值計算 (在第 5 年末)：**

*   D6 = D5 * (1 + g2) = 2.093 * (1 + 0.025) = 2.145 美元
*   終值 (P5) = D6 / (r - g2) = 2.145 / (0.0953 - 0.025) = 2.145 / 0.0703 = 30.51 美元
*   終值現值 = P5 / (1 + r)^5 = 30.51 / (1.0953)^5 = 30.51 * 0.636 = 19.40 美元

**DDM 估值 = 預測期股息現值總和 + 終值現值 = 7.25 + 19.40 = 26.65 美元**

**DDM 估值結論：**

DDM 模型給出的估值為 26.65 美元。這個估值遠低於當前股價 53.44 美元。這可能表明市場對 Nike 未來股息增長有更高的預期，或者 DDM 模型對於 Nike 這樣仍有較大增長潛力的公司 (儘管目前面臨挑戰) 可能不夠全面，因為它主要關注股息而非總體盈利和現金流。

### **3. 綜合結論與合理價位區間**

我們應用了三種估值模型，並得到了不同的結果：

*   **折現現金流模型 (DCF)：** 由於簡化估算和數據限制，結果 (約 44.36 億美元企業價值，未換算為每股) 遠低於市值，可靠性較低。
*   **相對估值 (Comparable Company Analysis)：** 估值區間為 **44.05 美元 - 60.75 美元**。
*   **股息折現模型 (DDM)：** 估值為 **26.65 美元**。

**分析師目標價：**

*   最新的分析師中位目標價為 **74.60 美元**。
*   個別分析師目標價範圍從 35.00 美元到 120.00 美元。

**綜合分析：**

1.  **DCF 模型的局限性：** 在沒有詳細財務預測的情況下，DCF 模型的結果波動較大，在此次分析中作為參考價值有限。
2.  **DDM 模型的局限性：** DDM 估值顯著低於當前股價和分析師目標價，這可能反映了 DDM 模型未能充分捕捉 Nike 作為一個全球品牌，其除了股息之外的增長潛力。對於成長型或轉型中的公司，DDM 往往會低估其價值。
3.  **相對估值：** 相對估值提供了一個基於同業比較的合理區間。Nike 的 P/E 倍數高於行業平均，但 P/S 倍數略低。這可能暗示市場對其盈利能力恢復有較高期待，或者其營收規模尚未完全反映在當前股價中。相對估值區間 44.05 美元 - 60.75 美元與當前股價 53.44 美元較為接近。
4.  **市場情緒與分析師預期：** 分析師的共識目標價 74.60 美元遠高於當前股價和我們的相對估值下限，表明市場對 Nike 的長期前景仍持樂觀態度，儘管近期面臨挑戰。近期分析師下調目標價也反映了短期壓力。

**當前市場狀況：**

Nike 目前股價 53.44 美元接近其 52 週低點 52.28 美元。這表明市場對其近期表現持謹慎態度，可能反映了 Q2 財報中毛利率下降、Nike Direct 營收下滑以及北美和中國市場的疲軟。然而，公司正在積極調整策略，並預計在 FY2027 實現營收增長和利潤率擴張。

**綜合結論：**

考慮到 Nike 作為全球領先運動品牌，其強大的品牌價值、創新能力和長期增長潛力，儘管目前面臨庫存、毛利率壓力和部分市場銷售疲軟等短期挑戰，但市場對其長期復甦仍抱有信心。

*   DDM 估值顯著偏低，可能不適用於評估 Nike 的全部價值。
*   簡化 DCF 估值也未能提供可靠結果。
*   **相對估值**提供了一個較為實際的參考區間，即 **44.05 美元 - 60.75 美元**。
*   分析師的共識目標價 **74.60 美元** 則反映了更為樂觀的長期預期。

鑑於當前股價 53.44 美元處於相對估值區間的中間偏下，且接近 52 週低點，如果投資者相信 Nike 能夠成功執行其轉型策略，解決庫存問題並恢復利潤率增長，那麼當前價格可能提供一個有吸引力的切入點。然而，短期內仍需關注其即將發布的 Q3 財報以及宏觀經濟環境的變化。

**合理價位區間：**

綜合考慮相對估值和分析師的共識預期，並考慮到 Nike 的品牌護城河和長期潛力，我認為 Nike 的合理價位區間可能在 **55 美元至 75 美元** 之間。當前股價 53.44 美元略低於此區間，可能反映了市場對其短期挑戰的擔憂。

**與同產業可比公司的估值比較：**

*   Nike 的 P/E 和 Forward P/E 倍數高於 Adidas 和 Lululemon，但低於高增長公司 On Holding。這表明市場對 Nike 的增長預期介於成熟品牌和高增長新興品牌之間。
*   Lululemon 雖然 P/E 較低，但其盈利能力和增長穩定性較強。
*   Adidas 的估值倍數普遍低於 Nike，可能反映了其在市場競爭中的不同地位或增長預期。
*   Puma 和 Under Armour 目前的虧損狀況使得其 P/E 倍數不具可比性，但其較低的 P/S 倍數可能反映了市場對其未來營收增長和盈利能力恢復的較低預期。

總體而言，Nike 的估值倍數反映了其作為行業領導者的地位，但其高於部分成熟競爭對手的 P/E 倍數也要求其在未來能夠證明其盈利增長和利潤率恢復的能力。
