#### 3.3 股息折現模型 (Dividend Discount Model, DDM)

Nike 具有 24 年連續增加股息的記錄，因此 DDM 適用。

**DDM 假設：**

*   **當前每股股息 (D0):** 股息率 2.45% * 當前股價 65.64 美元 = 1.608 美元 (提供數據)

*   **股權成本 (Re):** 10.567% (來自 WACC 計算)

**DDM 計算：**

*   **第一階段股息預測：**
    *   D1 = D0 * (1 + g1) = 1.64 * (1 + 0.06) = 1.7384 美元
