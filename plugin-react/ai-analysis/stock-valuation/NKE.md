作為一位資深股權評價分析師，我將對美股公司 **NKE (Nike)** 進行全面的估值分析，並著重於與同業競爭對手的橫向比較。

### 1. 同業比較分析 (Industry Peer Comparison)

**識別競爭對手**
Nike 在全球運動用品市場中佔據領導地位，其主要競爭對手包括：
*   **Adidas AG (ADDYY)**：德國運動用品巨頭，與 Nike 在全球市場直接競爭。
*   **Lululemon Athletica Inc. (LULU)**：專注於高端運動休閒服飾，近年來增長強勁，對 Nike 構成差異化競爭。
*   **Under Armour, Inc. (UAA)**：美國運動服飾品牌，主要在北美市場與 Nike 競爭。

**估值地位分析**
從上述比較表格可以看出：
*   **Nike (NKE)** 的本益比 (P/E) 和預期本益比 (Forward P/E) 均高於 Adidas，但低於 Lululemon (若有數據)。其 EV/EBITDA 倍數也高於 Adidas 和 Lululemon。這表明市場對 Nike 的估值相對較高，可能反映了其強大的品牌力、全球市場領導地位以及預期的長期穩定性。
*   **Lululemon (LULU)** 在營收增長率、營業利潤率和淨利潤率方面表現出色，其較低的 P/E 和 EV/EBITDA 倍數（相對於其盈利能力）可能暗示其在高端市場的效率和增長潛力。然而，其市值數據未直接取得，但其EV/EBITDA倍數顯著低於Nike，這可能反映了其較小的規模和不同的市場定位。
*   **Adidas (ADDYY)** 在盈利能力指標上與 Nike 接近，但其估值倍數普遍低於 Nike，這可能與其市場份額、品牌溢價或近期業績波動有關。
*   **Under Armour (UAA)** 目前面臨營收下滑和虧損，其負的本益比和營業利潤率顯示其基本面較弱，估值倍數不具可比性。

綜合來看，Nike 相對於 Adidas 處於溢價狀態，這與其更強的品牌影響力、更廣泛的產品線和更穩固的全球市場份額相符。相對於 Lululemon，Nike 的估值倍數較高，但 Lululemon 的盈利能力更強，這可能反映了市場對 Lululemon 高增長和高利潤模式的認可。Nike 的估值倍數是否合理，需進一步結合其成長前景和風險進行深入分析。

### 2. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估算公司未來自由現金流的現值，以得出其內在價值。

**預測假設**
*   **預測期間 (Explicit Forecast Period)**：5 年 (2027-2031 財年)。
*   **營收增長率預期 (Revenue Growth Rate)**：
    *   鑑於 Nike 近期面臨的挑戰（如大中華區銷售下滑、庫存問題），以及分析師對 FY26 營收增長預期僅為 0.1%，我將採取較為保守的增長假設。
    *   FY27: 2.0% (略高於FY26預期，考慮到市場調整和產品創新)
    *   FY28: 3.0%
    *   FY29: 3.5%
    *   FY30: 3.0%
    *   FY31: 2.5% (逐步趨近終值增長率)
*   **營業利潤率 (Operating Margin)**：Nike 目前營業利潤率約為 6.22%。考慮到公司正在進行的成本重組和效率提升，預計未來幾年將逐步改善。
    *   FY27: 6.5%
    *   FY28: 7.0%
    *   FY29: 7.5%
    *   FY30: 7.8%
    *   FY31: 8.0%
*   **稅率 (Tax Rate)**：採用 Nike 最近報告的有效稅率 20%。
*   **資本支出 (Capital Expenditures, CapEx)**：假設為營收的 2.5% (歷史平均水平)。
*   **折舊與攤銷 (Depreciation & Amortization, D&A)**：假設為營收的 2.0% (歷史平均水平)。
*   **營運資本變動 (Change in Net Working Capital, NWC)**：假設為營收變動的 5% (保守估計，反映營運效率提升)。

**折現率與終值**
*   **加權平均資本成本 (Weighted Average Cost of Capital, WACC)**：
    *   無風險利率 (Risk-Free Rate)：採用美國 10 年期國債收益率 4.48% (截至 2026 年 6 月 4 日)。
    *   市場風險溢酬 (Market Risk Premium, ERP)：採用 4.6%。
    *   Beta：採用 1.12。
    *   股權成本 (Cost of Equity, Ke) = 無風險利率 + Beta * ERP = 4.48% + 1.12 * 4.6% = 4.48% + 5.152% = 9.632%。
    *   債務成本 (Cost of Debt, Kd)：根據市場數據，假設為 5.5% (稅前)。
    *   稅後債務成本 (After-tax Cost of Debt) = Kd * (1 - 稅率) = 5.5% * (1 - 20%) = 4.4%。
    *   債務權益比 (Debt/Equity Ratio)：根據提供數據，Debt/Eq 為 0.79。因此，權益佔總資本比重 = 1 / (1 + 0.79) = 0.56；債務佔總資本比重 = 0.79 / (1 + 0.79) = 0.44。
    *   WACC = (Ke * 權益佔比) + (稅後 Kd * 債務佔比) = (9.632% * 0.56) + (4.4% * 0.44) = 5.39% + 1.94% = 7.33%。
    *   *註：此處計算的 WACC 與搜尋結果中的 7.89% 或 8.08% 略有差異，主要因假設細節不同。為保持分析透明度，我將使用我計算的 7.33%。*
*   **終值增長率 (Terminal Growth Rate, g)**：假設為 2.5% (反映長期可持續的經濟增長率)。

**估值結果 (DCF Model)**
(為簡潔起見，此處僅列出關鍵結果，詳細計算過程略)

基於上述假設，通過預測未來 5 年的自由現金流 (FCFF) 並計算終值，然後將所有現金流折現回現值，得出 Nike 的內在價值。

*   **預計 FY26 營收**：$46.52B (基於最近 12 個月營收)
*   **預計 FY26 EBIT**：$3.11B (基於最近 12 個月 EBIT)

*   **終值 (Terminal Value)**：
    *   FCFF (2031) = $3.10 Billion
    *   終值 = FCFF (2031) * (1 + g) / (WACC - g) = 3.10 * (1 + 0.025) / (0.0733 - 0.025) = 3.10 * 1.025 / 0.0483 = 65.83 Billion USD
    *   終值現值 = 65.83 / (1 + 0.0733)^5 = 65.83 / 1.424 = 46.23 Billion USD

*   **企業總價值 (Enterprise Value)** = 預測期內 FCFF 現值總和 + 終值現值 = (2.02 + 2.10 + 2.16 + 2.20 + 2.22) + 46.23 = 10.70 + 46.23 = 56.93 Billion USD

*   **股權價值 (Equity Value)** = 企業總價值 + 現金及約當現金 - 總債務
    *   現金及約當現金：$8.06B
    *   總債務：$11.18B
    *   股權價值 = 56.93 + 8.06 - 11.18 = 53.81 Billion USD

*   **流通在外股數 (Shares Outstanding)**：1.48B (根據 64.87B 市值 / 43.81 股價計算約 1.48B)
*   **每股內在價值 (Intrinsic Value per Share)** = 53.81 Billion / 1.48 Billion = **$36.36**

*   **合理價位區間**：考慮到假設的敏感性，我將給出一個區間，例如在 WACC 和增長率上進行 ±0.5% 的調整。
    *   若 WACC 提高至 7.83%，終值增長率降至 2.0%，則每股價值可能降至 $32-$34。
    *   若 WACC 降低至 6.83%，終值增長率升至 3.0%，則每股價值可能升至 $40-$42。
    *   因此，DCF 模型下的合理價位區間約為 **$34.00 - $40.00**。

### 3. 多維度估值法 (Other Valuation Methods)

**相對估值法 (Relative Valuation)**
我將使用同業的平均估值倍數來推算 Nike 的目標價位。由於 Under Armour 表現不佳，我將主要參考 Adidas 和 Lululemon 的倍數。

*   **P/E Ratio (Forward)**：
    *   Adidas: 16.96
    *   Lululemon: N/A
    *   Nike 目前 Forward P/E: 23.99
    *   若以 Adidas 的 Forward P/E 16.96 作為參考，Nike 的目標價 = 16.96 / 23.99 * $43.81 = $30.95。
    *   考慮到 Nike 的品牌溢價和市場地位，直接採用 Adidas 的倍數可能過於保守。我們可以使用行業平均或給予 Nike 一定溢價。
    *   假設行業平均 Forward P/E (排除 UAA) 約為 20x (介於 Adidas 和 Nike 之間，考慮 LULU 數據缺失)。
    *   Nike 預計 FY27 EPS (基於 FY26 EPS -0.3088 * (1+0.2209) = -0.377，但這與分析師預期 EPS next Y_% 22.09% 矛盾，且 FY26 EPS 為負值，不適合直接計算。根據 Zacks 預計 FY27 EPS 增長 24.6%，FY26 EPS 預計 $1.52，則 FY27 EPS = $1.52 * (1+0.246) = $1.89)。
    *   若採用分析師預期 FY27 EPS $1.89，目標價 = 20 * $1.89 = $37.80。

*   **EV/EBITDA**：
    *   Adidas: 14.15
    *   Lululemon: 5.34
    *   Nike: 19.77
    *   同業平均 EV/EBITDA (排除 UAA) = (14.15 + 5.34) / 2 = 9.745。
    *   若以同業平均 9.745 倍數估值，Nike 的 EV = 9.745 * NKE TTM EBITDA ($3.598B) = $35.06B。
    *   股權價值 = $35.06B + $8.06B (現金) - $11.18B (債務) = $31.94B。
    *   每股價值 = $31.94B / 1.48B = $21.58。
    *   此結果顯著低於當前股價，可能因為 Lululemon 的 EV/EBITDA 倍數較低拉低了平均值，且 Nike 的 EBITDA 增長預期可能高於 Lululemon。

*   **P/S Ratio**：
    *   Nike: 1.39
    *   Lululemon: 1.39
    *   同業平均 P/S (排除 UAA) = (1.39 + 1.39) / 2 = 1.39 (假設 Adidas P/S 與 Nike 接近)。
    *   Nike TTM 營收 $46.52B。
    *   市值 = 1.39 * $46.52B = $64.66B。
    *   每股價值 = $64.66B / 1.48B = $43.69。

綜合相對估值法，考慮到 Nike 的品牌溢價和市場地位，採用 P/S 倍數可能更為穩健，因為營收數據相對穩定。若給予 Nike 略高於同業平均的倍數，目標價可能在 **$40.00 - $45.00** 之間。

**股息折現模型 (Dividend Discount Model, DDM)**
Nike 穩定配息，目前季度股息為 $0.41，年化股息為 $1.64。
*   **當前股息 (D0)**：$1.64
*   **股息增長率 (g)**：考慮到 Nike 的成熟市場地位和近期挑戰，假設長期股息增長率為 3.0%。
*   **股權成本 (Ke)**：9.632% (來自 DCF 模型計算)。

*   **DDM 估值** = D0 * (1 + g) / (Ke - g) = $1.64 * (1 + 0.03) / (0.09632 - 0.03) = $1.64 * 1.03 / 0.06632 = $1.6892 / 0.06632 = **$25.47**。

DDM 模型給出的估值顯著低於當前股價和 DCF 估值。這可能反映了 DDM 對股息增長率和股權成本的敏感性，以及 Nike 未來可能將更多現金流用於再投資而非大幅提高股息。
