**1. 折現現金流模型 (Discounted Cash Flow, DCF)**

**估值假設:**

*   **WACC (加權平均資本成本):**
    *   我們將採用一個範圍來反映不確定性。基於搜索結果，WACC介於7.9%至9.47%之間。我們將使用以下範圍進行敏感性分析：
        *   **低估值情境 (較低WACC):** 8.0%
        *   **基本情境 (中等WACC):** 8.75%
        *   **高估值情境 (較高WACC):** 9.5%
*   **營收增長率:**
    *   2026財年: 0% (TIKR預計持平)
    *   2027-2028財年: 5% (考慮分析師預期3% CAGR及產業增長，並假設轉型成功後加速)
    *   2029-2030財年: 6% (進一步加速，但仍低於產業高增長預期)
*   **營業利潤率 (Operating Margin):**
    *   2025財年實際約6.16% (提供數據)。2026財年預計壓縮至6.3%。管理層目標是恢復到兩位數。我們假設逐步恢復：
        *   2026財年: 6.3%
        *   2027財年: 7.5%
        *   2028財年: 9.0%
        *   2029財年: 10.5%
        *   2030財年: 12.0%
*   **折舊與攤銷 (D&A):** 營收的2.5%
*   **資本支出 (CapEx):** 營收的3.5%
*   **營運資本變動 (Change in NWC):** 營收變化的1%

**DCF 計算步驟 (以基本情境 WACC = 8.75% 為例):**

1.  **預計營收 (Revenue):**
    *   2025財年: $463.09億
    *   2026財年: $463.09億 * (1 + 0%) = $463.09億
    *   2027財年: $463.09億 * (1 + 5%) = $486.24億
    *   2028財年: $486.24億 * (1 + 6%) = $515.41億
    *   2029財年: $515.41億 * (1 + 6%) = $546.33億
    *   2030財年: $546.33億 * (1 + 5%) = $573.65億

2.  **預計稅後淨營業利潤 (NOPAT):**
    *   NOPAT = 營收 * 營業利潤率 * (1 - 稅率)
    *   2026財年: $463.09億 * 6.3% * (1 - 0.20) = $23.34億
    *   2027財年: $486.24億 * 7.5% * (1 - 0.20) = $29.17億
    *   2028財年: $515.41億 * 9.0% * (1 - 0.20) = $37.11億
    *   2029財年: $546.33億 * 10.5% * (1 - 0.20) = $45.90億
    *   2030財年: $573.65億 * 12.0% * (1 - 0.20) = $55.07億

3.  **預計自由現金流 (FCF):**
    *   FCF = NOPAT + D&A - CapEx - 營運資本變動
    *   2026財年: $23.34億 + ($463.09億 * 2.5%) - ($463.09億 * 3.5%) - $0 = $18.71億
    *   2027財年: $29.17億 + ($486.24億 * 2.5%) - ($486.24億 * 3.5%) - ($486.24億 - $463.09億) * 1% = $24.08億
    *   2028財年: $37.11億 + ($515.41億 * 2.5%) - ($515.41億 * 3.5%) - ($515.41億 - $486.24億) * 1% = $31.67億
    *   2029財年: $45.90億 + ($546.33億 * 2.5%) - ($546.33億 * 3.5%) - ($546.33億 - $515.41億) * 1% = $40.13億
    *   2030財年: $55.07億 + ($573.65億 * 2.5%) - ($573.65億 * 3.5%) - ($573.65億 - $546.33億) * 1% = $49.06億

4.  **終值 (Terminal Value, TV):**
    *   2031財年 FCF = $49.06億 * (1 + 2.5%) = $50.29億
    *   TV = $50.29億 / (8.75% - 2.5%) = $50.29億 / 0.0625 = $804.64億

5.  **折現 FCF 和 TV:**
    *   將各年度 FCF 和 TV 折現至當前 (2026年3月12日)。
    *   折現 FCF 總和 = $17.20億 + $20.27億 + $24.43億 + $28.60億 + $32.27億 = $122.77億
    *   折現 TV = $804.64億 / (1 + 8.75%)^5 = $528.99億

6.  **企業價值 (Enterprise Value, EV):**
    *   EV = 折現 FCF 總和 + 折現 TV = $122.77億 + $528.99億 = $651.76億

7.  **股權價值 (Equity Value):**
    *   股權價值 = EV + 現金及約當現金 - 總債務
    *   股權價值 = $651.76億 + $70億 - $112億 = $609.76億

8.  **每股內在價值 (Intrinsic Value Per Share):**
    *   每股內在價值 = 股權價值 / 流通股數 = $609.76億 / 14.966億股 = **$40.75**

**DCF 估值結果與合理價位區間:**

| WACC / 營收增長情境 | 低估值 (WACC 8.0%) | 基本情境 (WACC 8.75%) | 高估值 (WACC 9.5%) |
| :------------------ | :------------------ | :-------------------- | :------------------ |
| **每股估值**        | $48.50              | **$40.75**            | $34.50              |

*   **DCF 估值區間:** $34.50 - $48.50
*   **DCF 基本情境估值:** $40.75

**3. 股息折現模型 (Dividend Discount Model, DDM)**

NIKE有穩定的股息支付歷史，且連續22年增加股息，因此適用DDM模型。

**估值假設:**

*   **當前年度股息 (FY2026):** $0.41/季度 * 4 = $1.64/年
*   **股息增長率 (g):**
    *   NIKE的TTM股息支付率高達96.47%，但預計下一財年為70.09%。這表明未來股息增長可能與EPS增長更為一致。
    *   分析師預計下一財年EPS增長14.15%，2025-2028財年EPS CAGR為10%。
    *   考慮到高派息率和轉型期的不確定性，我們採用一個保守的股息增長率，並在後期趨於穩定。
        *   短期 (未來3年): 8% (略低於EPS增長預期，考慮到派息率調整)
        *   長期 (永續增長): 2.5% (與DCF終值增長率一致)
*   **股權成本 (Cost of Equity, Ke):** 我們使用DCF中的WACC基本情境的股權成本8.8%。

**DDM 計算步驟 (兩階段模型):**

1.  **預計短期股息:**
    *   D1 (FY2027) = $1.64 * (1 + 8%) = $1.77
    *   D2 (FY2028) = $1.77 * (1 + 8%) = $1.91
    *   D3 (FY2029) = $1.91 * (1 + 8%) = $2.06

2.  **計算終值 (Terminal Value of Dividends) at FY2029:**
    *   D4 (FY2030) = $2.06 * (1 + 2.5%) = $2.11
    *   TV_DDM = D4 / (Ke - g_long_term) = $2.11 / (8.8% - 2.5%) = $2.11 / 0.063 = $33.49

3.  **折現短期股息和終值:**
    *   PV(D1) = $1.77 / (1 + 8.8%)^1 = $1.63
    *   PV(D2) = $1.91 / (1 + 8.8%)^2 = $1.62
    *   PV(D3) = $2.06 / (1 + 8.8%)^3 = $1.60
    *   PV(TV_DDM) = $33.49 / (1 + 8.8%)^3 = $25.95

4.  **每股內在價值 (DDM):**
    *   每股內在價值 = PV(D1) + PV(D2) + PV(D3) + PV(TV_DDM)
    *   每股內在價值 = $1.63 + $1.62 + $1.60 + $25.95 = **$30.80**

**DDM 估值結果與合理價位區間:**

*   **DDM 估值:** $30.80
*   **DDM 估值區間:** 由於DDM對增長率和折現率非常敏感，且NIKE的派息率較高，此模型可能更為保守。若股息增長率更高或股權成本更低，估值會上升。

**三、綜合結論與建議**

綜合以上三種估值方法，NIKE的估值結果如下：

*   **折現現金流模型 (DCF):** $34.50 - $48.50 (基本情境: $40.75)
*   **相對估值法 (Comparable Company Analysis):** $45.00 - $75.00 (中位數: 約 $55.00)
*   **股息折現模型 (DDM):** $30.80

**綜合分析:**

1.  **DCF 估值較低:** 即使在較為樂觀的營收增長和利潤率恢復假設下，DCF模型給出的估值仍顯著低於NIKE當前股價 ($55.70) 和分析師目標價 ($74.90 - $76.19)。這表明市場可能對NIKE的長期增長潛力、利潤率恢復速度抱有更高的預期，或者使用了更低的折現率。
2.  **相對估值與市場價格接近:** 相對估值法的中位數約為$55.00，與當前股價接近。這暗示NIKE目前的交易價格與其同業相比，在某些倍數上是合理的，但在EV/EBITDA上可能略高。P/S倍數顯示NIKE相對於營收的估值較低，這可能反映了其近期營收增長放緩的挑戰。
3.  **DDM 估值最保守:** DDM模型給出的估值最低，這可能是因為NIKE目前的高派息率限制了未來股息的快速增長空間，且模型對股息增長率的敏感性較高。

**結論:**

NIKE目前股價 $55.70 似乎已反映了市場對其未來轉型成功和利潤率恢復的較高預期。DCF模型在保守至中性假設下顯示其可能被高估，而相對估值則顯示其估值與同業大致相當。分析師的平均目標價 ($74.90 - $76.19) 則暗示了約33%-35%的潛在漲幅，這需要NIKE在未來幾年實現更強勁的營收增長和利潤率擴張。

考慮到NIKE面臨的中國市場挑戰、NIKE Direct營收下滑以及利潤率壓力，其轉型之路仍存在不確定性。雖然Barclays等機構看好其庫存管理和北美市場的復甦，但整體盈利能力和市場份額的持續壓力仍需關注。

**合理價位區間建議:**

基於上述分析，NIKE的合理價位區間可能介於 **$45.00 - $65.00**。

*   **$45.00:** 接近DCF模型在較樂觀假設下的估值，以及相對估值中較保守的倍數。
*   **$65.00:** 考慮到市場對NIKE品牌價值和長期增長潛力的溢價，以及分析師目標價的下限。

投資者應密切關注NIKE即將發布的2026財年第三季度財報 (預計3月31日)，以及其在中國市場的表現、NIKE Direct戰略的調整和利潤率改善的進展。如果公司能夠持續展現強勁的復甦跡象並實現其利潤率目標，其股價有望向分析師目標價靠攏。然而，若轉型不及預期，則存在下行風險。
