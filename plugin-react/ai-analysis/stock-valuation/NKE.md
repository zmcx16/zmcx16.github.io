### **第一部分：數據採集與預備分析**

#### **1. 公司簡介**

    *   股東權益 (Total Equity): $15.55 B

#### **4. 加權平均資本成本 (WACC) 計算**
WACC 是 DCF 模型中的折現率，代表公司所有資本來源的平均成本。

**a. 無風險利率 (Risk-Free Rate, Rf):**
    *   採用美國10年期國債殖利率。
    *   **假設:** 4.50%

**b. 市場風險溢酬 (Market Risk Premium, MRP):**
    *   代表股票市場預期報酬率與無風險利率之間的差額。
    *   **假設:** 5.50% (此為產業常用範圍，約5%-6%)

**c. Beta (β):**
    *   衡量 NKE 相對於整個市場的股價波動性。
    *   **數據來源 (Yahoo Finance或Bloomberg):** 約 1.05
    *   **假設:** 1.05

**d. 股權成本 (Cost of Equity, Ke):**
    *   採用資本資產定價模型 (CAPM): Ke = Rf + β * MRP
    *   Ke = 4.50% + 1.05 * 5.50% = 4.50% + 5.775% = **10.275%**

**e. 債務成本 (Cost of Debt, Kd):**
    *   從公司利息支出和總債務估算。
    *   FY23 利息支出約 $250 M，總債務約 $8.92 B。
    *   Kd ≈ $250 M / $8.92 B = 2.80% (此為簡化估計，實務上可參考公司債券殖利率或信用評級)。考慮到NKE的信用質量，實際發行新債的成本可能會略高於此，但由於其評級高，仍屬於低利率。
    *   **假設:** 3.80% (考慮市場利率和公司信用)

**f. 有效稅率 (Tax Rate, T):**
    *   根據過去幾年的平均有效稅率。
    *   FY23 (Income Tax Expense / Pretax Income): $1.2 B / $7.1 B ≈ 16.9%
    *   **假設:** 17.0%

**g. 稅後債務成本 (After-Tax Kd):**
    *   After-Tax Kd = Kd * (1 - T)
    *   After-Tax Kd = 3.80% * (1 - 0.17) = 3.80% * 0.83 = **3.154%**

**h. 資本結構 (Capital Structure):**
    *   股權市值 (Equity Market Value): $142.22 B
    *   債務市值 (Debt Market Value): 通常使用帳面價值作為近似，約 $8.92 B
    *   總資本 (Total Capital) = $142.22 B + $8.92 B = $151.14 B
    *   股權權重 (We) = $142.22 B / $151.14 B = **0.941**
    *   債務權重 (Wd) = $8.92 B / $151.14 B = **0.059**

**i. WACC 計算:**
    *   WACC = (We * Ke) + (Wd * After-Tax Kd)
    *   WACC = (0.941 * 10.275%) + (0.059 * 3.154%)
    *   WACC = 9.670% + 0.186% = **9.856%**
    *   **WACC 假設為約 9.90%** (為簡化計算，取近似值)

### **第二部分：估值模型**

#### **模型一：折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型通過預測公司未來自由現金流並折現回現值來估算公司價值。

**1. DCF 模型假設：**
*   **預測期間 (Explicit Forecast Period):** 5年 (FY2025 - FY2029)
*   **長期增長率 (Terminal Growth Rate, g):** 2.50% (此應低於WACC，且通常略高於或等於長期通脹率，反映公司成熟後的穩定增長)
*   **營收增長率 (Revenue Growth Rate):**
    *   NKE 管理層預期 FY24 營收為低個位數增長。FY25 預計回到中個位數。
    *   FY2025: 4.50% (從FY24預計低個位數增長後的回升)
    *   FY2026: 4.00%
    *   FY2027: 3.50%
    *   FY2028: 3.00%
    *   FY2029: 2.80% (逐步過渡到長期增長率)
*   **EBIT 利潤率 (EBIT Margin):**
    *   NKE 歷史 EBIT Margin 約在 12-15% 之間。公司正在轉向 DTC 模式，並投資於供應鏈，這會對利潤率產生影響。
    *   假設：FY2025: 12.50%，之後每年微幅提升至 FY2029: 13.50%
*   **折舊與攤銷 (Depreciation & Amortization, D&A):**
    *   佔營收的百分比，歷史數據約 2.5%-3.0%。
    *   **假設:** 2.80% of Revenue
*   **資本支出 (Capital Expenditures, CapEx):**
    *   佔營收的百分比，歷史數據約 2.0%-2.5%。
    *   **假設:** 2.20% of Revenue
*   **營運資本變動 (Change in Net Working Capital, ΔNWC):**
    *   通常與營收增長相關。假設為每年營收增長的 1.0% (代表營收每增加1元，NWC需增加0.01元)。
*   **稅率 (Tax Rate):** 17.0% (同 WACC 計算中的有效稅率)

**2. 自由現金流到公司 (FCFF) 預測與計算：**

**3. 終端價值 (Terminal Value, TV) 計算：**
*   終端年份 FCFF (FY2029) = $7.17 B
*   長期增長率 (g) = 2.50%
*   WACC = 9.90%
*   TV_FY2029 = FCFF_FY2029 * (1 + g) / (WACC - g)
*   TV_FY2029 = $7.17 B * (1 + 0.025) / (0.0990 - 0.025)
*   TV_FY2029 = $7.17 B * 1.025 / 0.074
*   TV_FY2029 = $7.35 B / 0.074 = **$99.32 B**

**4. 終端價值折現 (Present Value of Terminal Value):**
*   PV(TV) = TV_FY2029 / (1 + WACC)^5
*   PV(TV) = $99.32 B / (1 + 0.0990)^5
*   PV(TV) = $99.32 B / 1.594 = **$62.31 B**

**5. 企業價值 (Enterprise Value, EV) 計算：**
*   EV = 預測期間 FCFF 的現值總和 + 終端價值的現值
*   EV = ($4.97 B + $4.77 B + $4.63 B + $4.44 B + $4.22 B) + $62.31 B
*   EV = $23.03 B + $62.31 B = **$85.34 B**

**6. 股權價值 (Equity Value) 和每股價值 (Per Share Value)：**
*   股權價值 = 企業價值 + 現金及約當現金 - 總債務
*   股權價值 = $85.34 B + $7.55 B - $8.92 B = **$83.97 B**
*   流通股數 = 1.546 B
*   每股價值 = 股權價值 / 流通股數
*   每股價值 = $83.97 B / 1.546 B = **$54.31**

#### **模型三：股息折現模型 (Dividend Discount Model, DDM)**

NKE 是一家成熟的、穩定派發股息的公司，因此適用 DDM。

**1. DDM 模型假設：**
*   **當前年度每股股息 (D0):** $1.40 (最近十二個月 $1.36，假設未來一年 $1.40)
*   **股權成本 (Ke):** 10.275% (同 WACC 計算中的 Ke)
*   **股息增長率 (Dividend Growth Rate, g):**
    *   Nike 過去幾年股息增長率約 10-12%。由於公司規模龐大，未來預計會放緩。
    *   短期 (3年高增長): 8.00%
    *   長期 (永續增長): 3.50% (應低於 Ke，且接近公司長期 FCFF 增長率)

**2. 多階段 DDM 計算：**

| 年度 | Dt (每股股息) | 現值折現因子 (1/(1+Ke)^t) | 現值 (PV)    |
| :--- | :------------ | :------------------------ | :----------- |
| D1   | $1.40 * 1.08 = $1.512 | 1 / (1.10275)^1 = 0.907   | $1.37         |
| D2   | $1.512 * 1.08 = $1.633 | 1 / (1.10275)^2 = 0.822   | $1.34         |
| D3   | $1.633 * 1.08 = $1.764 | 1 / (1.10275)^3 = 0.745   | $1.31         |

**3. 終端股息價值 (Terminal Value of Dividends, TVD) 計算：**
*   TVD_Year3 = D3 * (1 + g_long) / (Ke - g_long)
*   TVD_Year3 = $1.764 * (1 + 0.035) / (0.10275 - 0.035)
*   TVD_Year3 = $1.764 * 1.035 / 0.06775
*   TVD_Year3 = $1.826 / 0.06775 = **$26.95**

**4. 終端股息價值折現 (Present Value of TVD)：**
*   PV(TVD) = TVD_Year3 / (1 + Ke)^3
*   PV(TVD) = $26.95 / (1.10275)^3
*   PV(TVD) = $26.95 / 1.340 = **$20.11**

**5. 每股價值計算：**
*   每股價值 = Sum(PV of D1-D3) + PV(TVD)
*   每股價值 = ($1.37 + $1.34 + $1.31) + $20.11
*   每股價值 = $4.02 + $20.11 = **$24.13**

**DDM 估值結果：** $24.13

*DDM 結果顯著低於當前股價和 DCF 估值。這通常發生在：1) 股息增長率假設過於保守；2) 股息支付率相對較低，公司將大量利潤再投資而非派發股息，這在成長型或穩定型公司中很常見，DDM 未能充分捕捉這些再投資創造的價值；3) Ke 假設過高。對於像 NKE 這樣擁有大量再投資機會的公司，DDM 可能會低估其價值，因為它主要關注直接派發給股東的現金流。*

1.  **DCF 模型的結果 ($50 - $65) 遠低於當前股價。** 這可能表明我的預測假設（尤其是營收增長率和利潤率）相對保守，或者市場對 NKE 未來的增長預期比模型假設的更高。Nike 目前面臨宏觀經濟逆風，中國市場放緩以及庫存調整等挑戰，這確實可能短期內影響其增長。如果對公司營收增長和利潤率有更樂觀的假設 (例如長期增長率達到3%或3.5%，WACC稍低)，DCF 估值會顯著提高。例如，WACC降至9.4%，長期增長率至3%時，估值可達 $76。
2.  **DDM 模型結果 ($20 - $30) 最低。** 這符合預期，因為 Nike 仍在積極再投資以推動其 DTC 和創新戰略，而非將大部分利潤作為股息發放。DDM 往往會低估此類公司的價值。
3.  **相對估值 ($70 - $95) 和歷史 P/E 區間 ($82 - $98) 較為接近當前市場價格 ($92)。** 這兩種方法反映了市場對 NKE 作為行業龍頭的認可，其穩定性、品牌實力以及未來增長潛力。與 Adidas 相比，NKE 的 P/E 和 EV/EBITDA 倍數更高，但這是合理的，因為 NKE 通常擁有更好的盈利能力和市場領導地位。Lululemon 由於其更快的增長和更高的利潤率，享有更高的估值溢價，NKE 估值倍數低於 LULU 也符合市場預期。

**綜合結論：**

綜合來看，DCF 模型給出了一個相對保守的估值，可能未能完全捕捉市場對 NKE 品牌價值和未來戰略轉型的長期信心。DDM 則因其本質上不完全適用於高再投資公司而提供一個過低的估值。
