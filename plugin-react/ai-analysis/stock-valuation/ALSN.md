### **估值方法與計算**

#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2024 - 2028)
*   **加權平均資本成本 (WACC):**
    *   **無風險利率 (Risk-Free Rate, Rf):** 4.5% (參考美國 10 年期國債殖利率)
    *   **Beta (β):** 1.35 (參考 Yahoo Finance 等數據源)
    *   **市場風險溢酬 (Market Risk Premium, MRP):** 5.5% (行業標準假設)
    *   **股權成本 (Cost of Equity, Ke):** `Rf + β * MRP = 4.5% + 1.35 * 5.5% = 11.925%`
    *   **債務成本 (Cost of Debt, Kd):**
        *   根據最新財報，ALSN 的長期債務約為 25 億美元。TTM 利息費用約 1.6 億美元。
        *   `稅前 Kd = 160M / 2500M = 6.4%`
        *   **稅率 (Tax Rate, t):** 22% (參考美國企業有效稅率)
        *   `稅後 Kd = 6.4% * (1 - 0.22) = 4.992%`
    *   **資本結構：**
        *   股權市值 (E): $8.39 Billion
        *   債務市值 (D): $2.5 Billion (假設與賬面價值接近)
        *   總資本 (V): $8.39B + $2.5B = $10.89 Billion
        *   股權權重 (We): $8.39B / $10.89B = 0.7704
        *   債務權重 (Wd): $2.5B / $10.89B = 0.2296
    *   **WACC = We * Ke + Wd * 稅後 Kd = 0.7704 * 11.925% + 0.2296 * 4.992% = 9.187% + 1.146% = 10.333%**
    *   **取 WACC 為 10.3%** (為後續敏感性分析，我們也會考慮 9.5% 的 WACC)

*   **長期增長率 (Terminal Growth Rate, g):** 2.2% (略高於長期通膨和全球 GDP 增長率)

*   **自由現金流 (FCF) 預測：**
    *   **基期 FCF (2023 TTM):** $622 Million (來自 P/FCF 計算)
    *   **2024 年 FCF (來自公司財測中點):** $430 Million (預計大幅下降)
    *   **2025 年 FCF 增長率:** 10% (預期市場復甦和電動化產品貢獻)
    *   **2026 年 FCF 增長率:** 7%
    *   **2027 年 FCF 增長率:** 5%
    *   **2028 年 FCF 增長率:** 3%

**計算過程：**

1.  **預測期自由現金流 (FCF) 及現值 (PV of FCFs):**

    | 年度 | FCF (百萬美元) | 增長率 | 折現因子 (WACC=10.3%) | PV of FCF (百萬美元) |
    | :--- | :------------- | :----- | :-------------------- | :------------------- |
    | 2024 | 430            | -30.9% | 1 / (1+0.103)^1 = 0.9066 | 389.84               |
    | 2025 | 430 * 1.10 = 473 | 10%    | 1 / (1+0.103)^2 = 0.8219 | 388.79               |
    | 2026 | 473 * 1.07 = 506.11 | 7%     | 1 / (1+0.103)^3 = 0.7451 | 376.17               |
    | 2027 | 506.11 * 1.05 = 531.42 | 5%     | 1 / (1+0.103)^4 = 0.6755 | 359.00               |
    | 2028 | 531.42 * 1.03 = 547.36 | 3%     | 1 / (1+0.103)^5 = 0.6124 | 335.19               |
    | **預測期 FCF 現值總和** | | | | **1,848.99** |

2.  **終值 (Terminal Value, TV) 計算：**
    *   `FCF_2029 = FCF_2028 * (1 + g) = 547.36 * (1 + 0.022) = 559.40 百萬美元`
    *   `TV = FCF_2029 / (WACC - g) = 559.40 / (0.103 - 0.022) = 559.40 / 0.081 = 6,906.17 百萬美元`

3.  **終值現值 (PV of TV):**
    *   `PV of TV = TV / (1 + WACC)^5 = 6,906.17 / (1 + 0.103)^5 = 6,906.17 * 0.6124 = 4,229.99 百萬美元`

4.  **企業價值 (Enterprise Value, EV):**
    *   `EV = 預測期 FCF 現值總和 + 終值現值 = 1,848.99 + 4,229.99 = 6,078.98 百萬美元`

5.  **股權價值 (Equity Value):**
    *   需要現金及約當現金 (Cash & Equivalents) 和總債務 (Total Debt)。
    *   根據網路搜尋，ALSN 現金約 10 億美元，總債務約 25 億美元。
    *   `Equity Value = EV + Cash - Total Debt = 6,078.98 + 1,000 - 2,500 = 4,578.98 百萬美元`

6.  **每股內在價值 (Intrinsic Value per Share):**
    *   `每股價值 = Equity Value / Shares Outstanding = 4,578.98M / 83.23M = $55.02`

**DCF 估值結果與合理價位區間：**

*   **基本情境 (WACC=10.3%, g=2.2%):** **$55.02**
*   **敏感性分析 (考慮更樂觀的假設，以接近市場價格):**
    *   若 WACC 降至 9.5%，長期增長率提高至 2.5%，且假設 2024 年 FCF 僅小幅下降至 $550M (而非財測的 $430M)，之後每年增長 8%, 6%, 4%, 3%：
        *   此情境下，每股價值可能達到 **$85 - $95** 之間。
    *   若市場完全忽略 2024 年的 FCF 下降，直接從 TTM FCF ($622M) 開始，並假設 WACC 9.5%，長期增長率 2.5%，預測期增長率為 5%, 4%, 3%, 2.5%：
        *   此情境下，每股價值約為 **$93.01** (計算過程見思考步驟)。

**結論：** 根據公司 2024 年財測的 FCF 預期，並採用較為保守的 WACC 和增長率，DCF 模型顯示的估值遠低於當前股價。這表明市場可能對 ALSN 的未來增長持更樂觀的態度，或者預期其 WACC 更低，或者認為 2024 年的 FCF 下降是暫時且影響有限的。若要支持當前股價，需要更樂觀的 FCF 增長假設和/或更低的折現率。

ALSN 的股息殖利率為 1.07%，相對較低，且其價值主要來自於未來的盈利增長和自由現金流，而非單純的股息分配。公司有較高的 ROE (40.38%)，表明其有能力將利潤再投資以實現增長。因此，DDM 對於 ALSN 而言，可能不是最合適或最能反映其真實價值的估值方法。

**若仍嘗試使用簡單的戈登增長模型 (Gordon Growth Model) 進行估算，並假設：**
*   **當前每股股息 (D0):** $1.0786
*   **股權成本 (Ke):** 11.925% (來自 DCF 模型)
*   **股息增長率 (g):** 假設長期股息增長率為 2.5% (保守估計，低於 EPS 增長預期)
*   `D1 = D0 * (1 + g) = $1.0786 * (1 + 0.025) = $1.1056`
*   `每股價值 = D1 / (Ke - g) = $1.1056 / (0.11925 - 0.025) = $1.1056 / 0.09425 = $11.73`

**DDM 估值結果與合理價位區間：**

*   **$11.73**

**結論：** DDM 模型給出的估值遠低於當前股價，這再次印證了該模型不適用於 ALSN。ALSN 的價值主要來自於其再投資能力和未來的 FCF 增長，而非其當前的股息分配。
