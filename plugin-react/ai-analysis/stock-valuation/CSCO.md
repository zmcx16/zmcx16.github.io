### 3. 多維度估值法

- **目標價** = Forward EPS * 28 = $126.80。

#### 股息折現模型 (DDM)
CSCO 具備穩定配息傳統（Dividend 1.41%）。
- **假設**：D0 = $1.67 (估算)，股息增長率 g = 6% (考慮轉型期現金流充沛)，要求回報率 r = 8.8%。
- **計算**：$1.67 * (1.06) / (0.088 - 0.06) = **$63.22**。
- *註：DDM 顯著低於市價，顯示目前股價更多由「成長預期」而非「股息收益」驅動。*
