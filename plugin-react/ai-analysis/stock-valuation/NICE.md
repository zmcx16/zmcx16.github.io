### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設**:

*   **預測期間 (Explicit Forecast Period)**: 5 年 (2025 年至 2029 年)。
*   **折現率 (Discount Rate / WACC)**:
    *   根據搜尋結果，NICE 的 WACC 範圍約在 8.8% 至 11.5% 之間。我將採用 10% 作為基準 WACC，並進行敏感性分析。
    *   計算 WACC 需要：
        *   **無風險利率 (Risk-Free Rate)**: 參考美國 10 年期國債收益率，約 4.3% (截至 2026 年 1 月 31 日)。
        *   **股權風險溢價 (Market Risk Premium)**: 歷史平均約 5-6%。取 5.5%。
        *   **Beta**: 用戶提供數據為 0.28。
        *   **股權成本 (Cost of Equity, Ke)** = 無風險利率 + Beta * 股權風險溢價 = 4.3% + 0.28 * 5.5% = 4.3% + 1.54% = 5.84%。
        *   **債務成本 (Cost of Debt, Kd)**: 考慮 NICE 的低負債率 (Debt/Eq 0.02)，以及其穩健的財務狀況，假設稅前債務成本為 4.5%。
        *   **稅率 (Tax Rate)**: 參考 2024 年 Q1 淨利潤率 16.1% 和營運利潤率 18.4%，以及美國企業稅率，假設有效稅率為 21%。
        *   **債務市值 (D)**: 4.575 億美元 (截至 2024 年 3 月 31 日)。
        *   **股權市值 (E)**: 65.7 億美元 (用戶提供數據)。
        *   **WACC** = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1 - Tax Rate)
            = (65.7 / (65.7 + 0.4575)) * 5.84% + (0.4575 / (65.7 + 0.4575)) * 4.5% * (1 - 0.21)
            = 0.993 * 5.84% + 0.007 * 4.5% * 0.79
            = 5.79% + 0.025% = 5.815%。
        *   這個 WACC 顯著低於搜尋結果。這可能是因為 Beta 值 0.28 較低，或者市場風險溢價的假設。為了更保守和符合行業慣例，我將採用一個更接近搜尋結果的 WACC 範圍，例如 8.5% - 10.5%。我將以 9.5% 作為基準 WACC。
*   **長期增長率 (Terminal Growth Rate, g)**: 假設 2.5% (略高於長期通脹率，反映其在成長型產業的地位)。
*   **營收增長率 (Revenue Growth Rate)**:
    *   2024 年營收增長 15%。2024 年 Q1 雲端營收增長 27%。
    *   考慮到其在 AI 和雲端 CX 領域的強勁增長，以及分析師對 2025 年 EPS 增長 10% 的預期，我將假設未來 5 年的營收增長率逐步放緩：
        *   2025 年: 12%
        *   2026 年: 10%
        *   2027 年: 8%
        *   2028 年: 7%
        *   2029 年: 6%
*   **營運利潤率 (Operating Margin)**: 2024 年 Q1 GAAP 營運利潤率為 18.4%，用戶提供數據為 21.99%。考慮到其雲端業務的擴張和規模經濟，假設營運利潤率逐步提升至 22%。
*   **稅率 (Tax Rate)**: 21%。
*   **資本支出 (Capital Expenditure, CapEx)**: 參考歷史數據，CapEx 佔營收的比例約為 3-4%。假設未來維持在 3.5%。
*   **折舊與攤銷 (Depreciation & Amortization, D&A)**: 參考歷史數據，D&A 佔營收的比例約為 3%。
*   **營運資本變動 (Change in Net Working Capital)**: 假設為營收的 1%。
*   **流通股數 (Shares Outstanding)**: 市值 65.7 億美元 / 股價 106.41 美元 = 約 6174 萬股。

**DCF 計算步驟**:

1.  **預測營收 (Revenue Forecast)**:
    *   2024 年營收 (基準): 27.35 億美元
    *   2025 年: 27.35 * (1 + 0.12) = 30.63 億美元
    *   2026 年: 30.63 * (1 + 0.10) = 33.69 億美元
    *   2027 年: 33.69 * (1 + 0.08) = 36.39 億美元
    *   2028 年: 36.39 * (1 + 0.07) = 38.94 億美元
    *   2029 年: 38.94 * (1 + 0.06) = 41.27 億美元

2.  **預測息稅前利潤 (EBIT Forecast)**:
    *   EBIT = 營收 * 營運利潤率
    *   假設營運利潤率從 2024 年的 18.4% (Q1 GAAP) 逐步提升至 22%。
    *   2025 年: 30.63 * 0.19 = 5.82 億美元
    *   2026 年: 33.69 * 0.20 = 6.74 億美元
    *   2027 年: 36.39 * 0.21 = 7.64 億美元
    *   2028 年: 38.94 * 0.215 = 8.37 億美元
    *   2029 年: 41.27 * 0.22 = 9.08 億美元

3.  **計算稅後息稅前利潤 (NOPAT)**:
    *   NOPAT = EBIT * (1 - Tax Rate)
    *   稅率 21%
    *   2025 年: 5.82 * (1 - 0.21) = 4.60 億美元
    *   2026 年: 6.74 * (1 - 0.21) = 5.32 億美元
    *   2027 年: 7.64 * (1 - 0.21) = 6.04 億美元
    *   2028 年: 8.37 * (1 - 0.21) = 6.61 億美元
    *   2029 年: 9.08 * (1 - 0.21) = 7.19 億美元

4.  **計算自由現金流 (FCF)**:
    *   FCF = NOPAT + D&A - CapEx - Change in Net Working Capital
    *   D&A = 營收 * 3%
    *   CapEx = 營收 * 3.5%
    *   Change in Net Working Capital = 營收增量 * 1%
    *   2025 年 FCF: 4.60 + (30.63 * 0.03) - (30.63 * 0.035) - (30.63 - 27.35) * 0.01 = 4.60 + 0.92 - 1.07 - 0.03 = 4.42 億美元
    *   2026 年 FCF: 5.32 + (33.69 * 0.03) - (33.69 * 0.035) - (33.69 - 30.63) * 0.01 = 5.32 + 1.01 - 1.18 - 0.03 = 5.12 億美元
    *   2027 年 FCF: 6.04 + (36.39 * 0.03) - (36.39 * 0.035) - (36.39 - 33.69) * 0.01 = 6.04 + 1.09 - 1.27 - 0.03 = 5.83 億美元
    *   2028 年 FCF: 6.61 + (38.94 * 0.03) - (38.94 * 0.035) - (38.94 - 36.39) * 0.01 = 6.61 + 1.17 - 1.36 - 0.03 = 6.39 億美元
    *   2029 年 FCF: 7.19 + (41.27 * 0.03) - (41.27 * 0.035) - (41.27 - 38.94) * 0.01 = 7.19 + 1.24 - 1.44 - 0.02 = 6.97 億美元

5.  **計算終值 (Terminal Value, TV)**:
    *   TV = FCF_2029 * (1 + g) / (WACC - g)
    *   TV = 6.97 * (1 + 0.025) / (0.095 - 0.025) = 6.97 * 1.025 / 0.07 = 102.15 億美元

6.  **計算各期 FCF 和 TV 的現值 (Present Value)**:
    *   PV(FCF_2025) = 4.42 / (1 + 0.095)^1 = 4.04 億美元
    *   PV(FCF_2026) = 5.12 / (1 + 0.095)^2 = 4.28 億美元
    *   PV(FCF_2027) = 5.83 / (1 + 0.095)^3 = 4.39 億美元
    *   PV(FCF_2028) = 6.39 / (1 + 0.095)^4 = 4.44 億美元
    *   PV(FCF_2029) = 6.97 / (1 + 0.095)^5 = 4.43 億美元
    *   PV(TV) = 102.15 / (1 + 0.095)^5 = 64.92 億美元

7.  **計算企業價值 (Enterprise Value, EV)**:
    *   EV = 預測期 FCF 現值總和 + TV 現值
    *   EV = 4.04 + 4.28 + 4.39 + 4.44 + 4.43 + 64.92 = 86.50 億美元

8.  **計算股權價值 (Equity Value)**:
    *   Equity Value = EV - 淨債務 (Net Debt)
    *   淨現金 (Net Cash) = 現金 15.028 億美元 - 債務 4.575 億美元 = 10.453 億美元。
    *   Equity Value = 86.50 億美元 + 10.453 億美元 = 96.953 億美元

9.  **計算每股內在價值 (Intrinsic Value per Share)**:
    *   每股內在價值 = 股權價值 / 流通股數
    *   每股內在價值 = 96.953 億美元 / 6174 萬股 = 157.03 美元
