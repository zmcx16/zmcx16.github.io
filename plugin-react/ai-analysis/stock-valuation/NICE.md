#### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間：** 5年 (2026年 - 2030年)
*   **折現率 (WACC)：**
    *   **無風險利率 (Risk-Free Rate, Rf)：** 採用美國10年期國債收益率。目前約為 4.28%。
    *   **市場風險溢酬 (Market Risk Premium, MRP)：** 通常介於 3-7% 之間。我們採用 5.5%。
    *   **Beta (β)：** 採用 NICE 的最新 Beta 值 0.96。
    *   **股權成本 (Cost of Equity, Ke) = Rf + β * MRP**
        *   Ke = 4.28% + 0.96 * 5.5% = 4.28% + 5.28% = 9.56%
    *   **債務成本 (Cost of Debt, Kd)：** 由於 NICE 目前已無債務，債務成本為 0。
    *   **稅率 (Tax Rate)：** 採用美國企業稅率 21%。

**DCF 計算步驟：**

1.  **計算預測期內的自由現金流 (FCFF)：**
    *   營收 (Revenue)
    *   營運利潤 (Operating Profit) = 營收 * 營運利潤率
    *   稅後營運利潤 (NOPAT) = 營運利潤 * (1 - 稅率)
    *   自由現金流 (FCFF) = NOPAT + D&A - CapEx - 營運資本變動

2.  **計算終值 (Terminal Value, TV)：**
    *   TV = (FCFF_last_year * (1 + Terminal Growth Rate)) / (WACC - Terminal Growth Rate)

3.  **計算企業價值 (Enterprise Value, EV)：**
    *   EV = 預測期內 FCFF 的現值總和 + TV 的現值

4.  **計算股權價值 (Equity Value)：**
    *   Equity Value = EV + 現金及約當現金 - 債務
    *   NICE 截至2025年第四季度末有 $4.174億美元現金，無債務。

5.  **計算每股價值 (Per Share Value)：**
    *   每股價值 = 股權價值 / 流通股數
