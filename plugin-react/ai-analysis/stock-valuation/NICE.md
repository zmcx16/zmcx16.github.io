### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值，加上終值，以得出公司總價值。

#### **1.1 估值假設**

*   **預測期間：** 5年 (2024年 - 2028年)
*   **長期增長率 (永續增長率, g)：** 2.8% (假設公司在預測期後進入穩定增長階段，略低於長期名義GDP增長率和通脹率，反映其成熟期軟體公司的特徵)
*   **稅率 (Effective Tax Rate)：** 22% (基於 NICE 過去幾年的有效稅率以及其全球運營的稅收結構)
*   **加權平均資本成本 (WACC)：** 10.60% (詳情見下方計算)

**WACC 計算：**

*   **無風險利率 (Rf)：** 4.50% (使用美國10年期國債殖利率，截至2024年5月)
*   **市場風險溢價 (MRP)：** 5.50% (行業標準估計)
*   **Beta (β)：** 1.20 (來源：Yahoo Finance, NICE的5年每月Beta值)
*   **股權成本 (Ke) = Rf + β * MRP**
    *   Ke = 4.50% + 1.20 * 5.50% = 4.50% + 6.60% = **11.10%**

*   **債務成本 (Kd)：** 6.50% (估計NICE目前債務的平均利率)
*   **稅後債務成本 (Kd_after_tax) = Kd * (1 - 稅率)**
    *   Kd_after_tax = 6.50% * (1 - 0.22) = 6.50% * 0.78 = **5.07%**

*   **資本結構：**
    *   市值 (Equity Value, E)：$13.01 百億美元 (61.94M 股 * $210/股)
    *   總債務 (Debt Value, D)：約 $1.20 百億美元 (截至2024年Q1末，主要為長期債務)
    *   E/(D+E) = 13.01 / (1.20 + 13.01) = 13.01 / 14.21 = **91.56%**
    *   D/(D+E) = 1.20 / 14.21 = **8.44%**

*   **WACC = (E/(D+E)) * Ke + (D/(D+E)) * Kd_after_tax**
    *   WACC = 0.9156 * 11.10% + 0.0844 * 5.07% = 10.16% + 0.43% = **10.59% ≈ 10.60%**

### **4. 其他合理估值方法：股價對銷售額比率 (P/S)**

儘管 EV/Revenue 更為精確，但 P/S 是一個快速檢視成長型或高利潤率軟體公司的常用指標。

*   **NICE LTM 總收入 (2023)：** $2.38B
*   **NICE 市值：** $13.01B
*   **NICE LTM P/S = 13.01B / 2.38B = 5.47x**

對比可比公司：

| 公司    | LTM P/S (市值/收入) |
| :------ | :------------------ |

| RNG     | 1.76x               |

*   隱含市值 = 2.12x * $2.38B = $5.05B
*   隱含股價 = $5.05B / 61.94M 股 = **$81.53/股**

**結論：** NICE 的當前 P/S 倍數 (5.47x) 顯著高於其可比公司的平均值 (2.12x)。這可能表明：
1.  市場對 NICE 的增長潛力和盈利能力預期更高。
2.  NICE 的盈利能力和自由現金流轉換率優於可比公司。
3.  或者，NICE 相對於其收入可能被高估。

由於 NICE 的盈利能力較強，且正逐步提高，單純的 P/S 倍數可能無法完全捕捉其價值。EV/EBITDA 或 P/E 在這種情況下可能更為合適。
