**模型一：折現現金流模型 (DCF)**

DCF模型需要詳細的財務預測，包括營收、成本、稅費、資本支出、折舊攤銷和營運資本變動，以計算自由現金流 (FCFF)。

**FCFF 計算公式：**
FCFF = 稅後營業利潤 (NOPAT) + 折舊攤銷 - 資本支出 - 營運資本變動
NOPAT = 營業利潤 \* (1 - 稅率)

**簡化計算步驟 (基於假設)：**

1.  **預測營收 (Revenue):**
    *   2025財年 (基期): $106.566B.
    *   2026: $106.566B \* (1 + 1.5%) = $108.16B
    *   2027: $108.16B \* (1 + 2.0%) = $110.32B
    *   2028: $110.32B \* (1 + 2.0%) = $112.53B
    *   2029: $112.53B \* (1 + 2.5%) = $115.34B
    *   2030: $115.34B \* (1 + 2.5%) = $118.22B

2.  **預測營業利潤 (Operating Income) 和 NOPAT:**
    *   假設2025財年營業利潤率為3.8%。
    *   2025營業利潤: $106.566B \* 3.8% = $4.05B
    *   2025 NOPAT: $4.05B \* (1 - 0.2265) = $3.13B
    *   2026營業利潤率: 4.0% -> 營業利潤: $108.16B \* 4.0% = $4.33B -> NOPAT: $4.33B \* (1 - 0.2265) = $3.35B
    *   2027營業利潤率: 4.5% -> 營業利潤: $110.32B \* 4.5% = $4.96B -> NOPAT: $4.96B \* (1 - 0.2265) = $3.84B
    *   2028營業利潤率: 5.0% -> 營業利潤: $112.53B \* 5.0% = $5.63B -> NOPAT: $5.63B \* (1 - 0.2265) = $4.36B
    *   2029營業利潤率: 5.2% -> 營業利潤: $115.34B \* 5.2% = $5.99B -> NOPAT: $5.99B \* (1 - 0.2265) = $4.64B
    *   2030營業利潤率: 5.5% -> 營業利潤: $118.22B \* 5.5% = $6.50B -> NOPAT: $6.50B \* (1 - 0.2265) = $5.03B

3.  **預測資本支出 (CapEx) 和折舊攤銷 (D&A):**
    *   2025 CapEx: $2.891B. 假設D&A約為CapEx的85%。
    *   2025 D&A (估計): $2.891B \* 0.85 = $2.46B
    *   2026 CapEx: $5.0B. D&A: $5.0B \* 0.85 = $4.25B
    *   2027 CapEx: $4.5B. D&A: $4.5B \* 0.85 = $3.83B
    *   2028 CapEx: $4.0B. D&A: $4.0B \* 0.85 = $3.40B
    *   2029 CapEx: $3.8B. D&A: $3.8B \* 0.85 = $3.23B
    *   2030 CapEx: $3.6B. D&A: $3.6B \* 0.85 = $3.06B

4.  **預測營運資本變動 (Change in NWC):**
    *   假設NWC佔營收的比例維持在-4%。
    *   2025 NWC: $106.566B \* (-4%) = -$4.26B
    *   2026 NWC: $108.16B \* (-4%) = -$4.33B -> Change in NWC: -$4.33B - (-$4.26B) = -$0.07B
    *   2027 NWC: $110.32B \* (-4%) = -$4.41B -> Change in NWC: -$4.41B - (-$4.33B) = -$0.08B
    *   2028 NWC: $112.53B \* (-4%) = -$4.50B -> Change in NWC: -$4.50B - (-$4.41B) = -$0.09B
    *   2029 NWC: $115.34B \* (-4%) = -$4.61B -> Change in NWC: -$4.61B - (-$4.50B) = -$0.11B
    *   2030 NWC: $118.22B \* (-4%) = -$4.73B -> Change in NWC: -$4.73B - (-$4.61B) = -$0.12B

5.  **計算自由現金流 (FCFF):**
    *   2026 FCFF: $3.35B + $4.25B - $5.0B - (-$0.07B) = $2.67B
    *   2027 FCFF: $3.84B + $3.83B - $4.5B - (-$0.08B) = $3.25B
    *   2028 FCFF: $4.36B + $3.40B - $4.0B - (-$0.09B) = $3.85B
    *   2029 FCFF: $4.64B + $3.23B - $3.8B - (-$0.11B) = $4.18B
    *   2030 FCFF: $5.03B + $3.06B - $3.6B - (-$0.12B) = $4.61B

6.  **計算終端價值 (Terminal Value, TV):**
    *   TV = FCFF(2030) \* (1 + g) / (WACC - g)
    *   TV = $4.61B \* (1 + 0.02) / (0.0775 - 0.02) = $4.70B / 0.0575 = $81.74B

7.  **計算各期FCFF和TV的現值 (Present Value):**
    *   PV(FCFF 2026) = $2.67B / (1 + 0.0775)^1 = $2.48B
    *   PV(FCFF 2027) = $3.25B / (1 + 0.0775)^2 = $2.82B
    *   PV(FCFF 2028) = $3.85B / (1 + 0.0775)^3 = $3.09B
    *   PV(FCFF 2029) = $4.18B / (1 + 0.0775)^4 = $3.13B
    *   PV(FCFF 2030) = $4.61B / (1 + 0.0775)^5 = $3.18B
    *   PV(TV) = $81.74B / (1 + 0.0775)^5 = $56.44B

8.  **計算企業總價值 (Enterprise Value):**
    *   EV = PV(FCFFs) + PV(TV) = $2.48B + $2.82B + $3.09B + $3.13B + $3.18B + $56.44B = $71.14B

9.  **計算股權價值 (Equity Value) 和每股價值 (Intrinsic Value per Share):**
    *   股權價值 = 企業總價值 + 現金及約當現金 - 總債務

    *   股權價值 = $71.14B + $3.82B - $20.04B = $54.92B
    *   流通股數 (Shares Outstanding): 約451.64M (參考TradingView).
    *   **每股內在價值 = $54.92B / 451.64M = $121.60**

**DCF模型估值結果：**
*   **每股合理價位: $121.60**
*   **合理價位區間:** 考慮到假設的敏感性，我將給出一個區間。如果WACC在7.25%-8.25%之間，永續增長率在1.5%-2.5%之間，每股價值可能在 **$105 - $140** 之間。這個結果與第三方分析的DCF估值 ($131.36 - $140.71) 較為接近.
