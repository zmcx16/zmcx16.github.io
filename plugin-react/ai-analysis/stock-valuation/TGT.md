## TGT (Target Corporation) 股票估值分析

### 公司概述

Target Corporation (TGT) 是一家美國領先的大眾商品零售商，提供從食品、服裝、家居用品到電子產品的廣泛商品。TGT 以其時尚、實惠的產品以及「折扣精品店」的形象而聞名。公司透過實體店面、數位通路（Target.com）和應用程式為客戶提供服務。作為一家成熟的大型零售企業，TGT 營收規模龐大，現金流穩定，並且是長期股息支付者。

### 最新市場數據 (截至分析時點假設)

*   **流通股數 (Shares Outstanding):** 約 **460 百萬股** (來自最新財報)
*   **市值 (Market Cap):** 約 $150.00 * 460M = **$69.00 億**

### 估值假設 (共同部分與個別模型調整)

在進行估值之前，我們需要設定一系列關鍵假設。這些假設將用於所有適用的估值模型。

#### A. 折現率 / WACC (Weighted Average Cost of Capital)

WACC 是衡量公司資本成本的指標，用於折現未來現金流。

1.  **無風險利率 (Risk-Free Rate, Rf):** 採用美國10年期公債殖利率。假設為 **4.20%**。
2.  **市場風險溢酬 (Market Risk Premium, MRP):** 採用歷史平均值，通常在 5.0% - 6.0% 之間。假設為 **5.50%**。
3.  **Beta 值 (β):** 衡量公司股價相對於市場的波動性。TGT 的 Beta 值通常在 0.9 到 1.2 之間。假設為 **1.05** (來源：Yahoo Finance)。

    *   **股權成本 (Cost of Equity, Ke) - CAPM 模型:**
        $Ke = Rf + β * MRP$
        $Ke = 4.20\% + 1.05 * 5.50\% = 4.20\% + 5.775\% = \textbf{9.975\%}$

4.  **債務成本 (Cost of Debt, Kd):**
    *   TGT 的長期債務利息支出 (LTM)：約 $6.5 億 (來自財報)。
    *   TGT 的總債務 (LTM)：約 $150 億 (包含短期和長期)。
    *   估計的債務利率 (稅前) = 利息支出 / 總債務 = $6.5 億 / $150 億 = 4.33%。
    *   也可以參考 TGT 信用評級和發行債券的殖利率。假設稅前債務成本為 **4.50%**。

5.  **有效稅率 (Effective Tax Rate, T):** 採用 TGT 近年財報的平均稅率。假設為 **24.0%**。

    *   **稅後債務成本:** $Kd * (1 - T) = 4.50\% * (1 - 0.24) = 4.50\% * 0.76 = \textbf{3.42\%}$

6.  **資本結構 (Capital Structure):**
    *   市值 (Equity Value, E): $690 億
    *   總債務 (Debt Value, D): $150 億
    *   總企業價值 (Firm Value, V = E + D): $690 億 + $150 億 = $840 億
    *   股權佔比 (E/V): $690 億 / $840 億 = 82.14%
    *   債務佔比 (D/V): $150 億 / $840 億 = 17.86%

7.  **WACC 計算:**
    $WACC = (E/V) * Ke + (D/V) * Kd * (1 - T)$
    $WACC = (0.8214 * 9.975\%) + (0.1786 * 4.50\% * (1 - 0.24))$
    $WACC = 8.194\% + 0.609\% = \textbf{8.803\%}$
    為簡化計算，我們將 WACC 取整為 **8.80%**。

#### B. 增長率與預測期間

*   **明確預測期間 (Explicit Forecast Period):** 對於像 Target 這樣成熟的公司，5 年是標準且合理的預測期。
*   **長期增長率 (Terminal Growth Rate, g):** 該增長率代表公司在明確預測期結束後的永續增長率。它應低於WACC，且通常與長期通膨率或GDP增長率相當。我們假設為 **1.80%**。

### 4. 其他合理估值方法 (簡要考量)

對於 TGT 這樣的公司，除了上述三種方法外，較少使用其他方法作為主要估值依據。

*   **資產基礎估值 (Asset-Based Valuation):** 不適用於零售公司，因為品牌價值、供應鏈、客戶基礎和運營效率等無形資產佔據重要地位，而這些無法簡單地透過資產負債表上的帳面價值來衡量。
*   **清算價值 (Liquidation Value):** 不適用於持續經營的大型上市公司。
*   **期權定價模型 (Options Pricing Models):** 這些模型用於評估期權本身，而非底層股票的內在價值。

因此，我們將主要集中於上述 DCF, 相對估值和 DDM。

1.  **DDM 估值最低:** DDM 模型給出了最低的估值，這可能是因為它僅考慮了股息流。對於像 TGT 這樣不僅派發股息，還進行資本支出以維持和擴大其業務，並可能進行股票回購的公司，DDM 可能無法完全捕捉其內在價值。此外，DDM 對股息增長率和股權成本非常敏感。

2.  **DCF 估值居中偏低:** DCF 估值考慮了公司的全部自由現金流，相對更全面。我們的 DCF 估值結果顯示，在預設的增長和 WACC 條件下，TGT 的公允價值低於當前市場價格。這可能意味著市場對 TGT 未來現金流的預期比我們模型中假設的要樂觀，或者其要求的回報率較低。

3.  **相對估值最高:** 相對估值結果相對較高，且顯著高於當前市場價格。這主要受到可比公司（尤其是像 Costco 這樣的成長型零售商）較高倍數的影響。雖然我們嘗試通過排除 Costco 或使用更保守的倍數來調整，但整體而言，市場可能賦予行業領導者較高的估值溢價。這也反映了投資者傾向於為穩定、高質量現金流的公司支付更高的價格。TGT 的當前市盈率 (16.7x) 遠低於同業中位數 (27.3x)，這可能預示其被低估，也可能說明市場對其增長前景的看法不如WMT或COST樂觀。

**綜合結論:**

鑑於各模型結果的差異，我們需要進行權重分配和判斷。

*   DCF 作為內在價值評估的黃金標準，其結果 $116.39 / 股 提供了堅實的基礎。
*   相對估值 $160 - $210 / 股 則反映了市場對同業的普遍看法。TGT 的當前股價 $150 介於 DCF 基準值和相對估值的低端之間。
*   DDM 由於其敏感性和對零售企業現金流構成的局限性，我們給予較低的權重。

考慮到 TGT 的零售業性質，其現金流受到宏觀經濟、消費者支出的影響較大。其成長性較為溫和，但穩定性高。綜合來看，目前的市場價格 $150.00 / 股 介於我們 DCF 估值和相對估值的低端之間。

*   **若我們假設 TGT 是一個成熟但穩定且具一定增長潛力的公司，其內在價值可能略高於我們的 DCF 基準值，但不太可能達到同業中位數那樣的估值水平 (除非市場普遍情緒大幅提升)。**

### 5. 與同產業可比公司的估值比較 (已融入相對估值部分)

在相對估值部分，我們已經將 TGT 的估值倍數與其同業 Walmart (WMT)、Costco (COST) 和 Kroger (KR) 進行了比較。

**比較總結:**

*   **P/E (TTM):** TGT (16.7x) 顯著低於 WMT (27.3x) 和 COST (34.8x)，但高於 KR (11.7x)。這表明市場對 TGT 的盈利增長預期不如 WMT 和 COST 樂觀，但優於純雜貨零售商 KR。
*   **EV/EBITDA (TTM):** TGT (9.6x) 也低於 WMT (14.2x) 和 COST (35.0x)，接近 KR (8.0x)。這與 P/E 的趨勢一致，反映出 TGT 在企業價值層面上的估值也相對保守。

**潛在原因分析:**

1.  **增長潛力差異:** Costco 以其會員制和快速擴張在某些市場享有較高增長溢價。Walmart 規模龐大，但其電商轉型和國際市場表現也獲得市場認可。TGT 雖然在電商領域有所進展，但其整體增長可能被認為略低於前兩者。
2.  **利潤率和營運效率:** 雖然都是零售業，但各公司的利潤結構和經營模式存在差異。Costco 的利潤很大一部分來自會員費。Walmart 的規模效應極強。TGT 在保持「精品折扣」形象的同時，其利潤率面臨更激烈的競爭壓力。
3.  **市場情緒和投資者偏好:** 在某些時期，市場可能更偏好擁有穩定現金流和防禦性質的雜貨店，或具有強大會員粘性模式的零售商。

**結論:** TGT 相較於 WMT 和 COST 估值較低，部分反映了其不同的增長軌跡和市場定位。但相較於 KR 等純雜貨零售商，TGT 憑藉其更廣泛的商品種類和品牌影響力，享有一定溢價。目前的估值倍數可能暗示 TGT 在市場上並未被顯著高估，甚至可能略有低估，尤其是在其盈利能力穩定或改善時。

### 估值分析的局限性

*   **假設的敏感性:** 估值結果對 WACC、增長率等關鍵假設非常敏感。即使是微小的調整也可能導致最終估值產生較大差異。
*   **數據的時效性:** 分析所用數據基於歷史趨勢和假設，實際數據會隨時間變化。
*   **宏觀經濟影響:** 零售業高度依賴消費者支出和整體經濟環境。通膨、利率、失業率等因素都可能影響 TGT 未來的營收和利潤。
*   **競爭格局:** 零售業競爭激烈，來自線上零售商和實體競爭對手的壓力持續存在。
*   **市場情緒:** 市場價格不僅反映基本面，也受到投資者情緒、行業趨勢和突發事件的影響。
*   **DDM 的適用性:** 對於那些再投資機會較多、股息派發相對保守的公司，DDM 可能會低估其真實價值。
