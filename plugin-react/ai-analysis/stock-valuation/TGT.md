#### **3. 股息折現模型 (Dividend Discount Model, DDM)**

Target 是一家成熟的股息支付公司，股息率為 4.5%。 並且已經連續 54 年增加股息。 因此，DDM 適用。

**估值假設：**

*   **當前每股股息 (D0)**: Target 支付年度股息 $4.56/股。
*   **股息增長率 (g)**:
    *   考慮到 Target 連續 54 年增加股息的歷史，以及其在數字化和門店改造方面的投資，預計未來股息將持續增長。
    *   然而，零售業的增長速度有限，且 Target 的派息率 (55.3%) 已經相對較高。
    *   假設初期股息增長率為 4.0% (未來 3 年)，之後穩定在 2.5% (長期增長率)。
*   **所需報酬率 (Required Rate of Return, Ke)**: 使用之前 DCF 模型中計算的股權成本 8.95%。

**兩階段 DDM 計算：**

*   **階段一 (高增長階段 - 3 年):**
    *   D1 = D0 * (1 + g1) = $4.56 * (1 + 0.04) = $4.7424
    *   D2 = D1 * (1 + g1) = $4.7424 * 1.04 = $4.9321
    *   D3 = D2 * (1 + g1) = $4.9321 * 1.04 = $5.1294

*   **階段二 (穩定增長階段 - 終值):**
    *   D4 = D3 * (1 + g2) = $5.1294 * (1 + 0.025) = $5.2577
    *   終值 (Terminal Value at Year 3) = D4 / (Ke - g2) = $5.2577 / (0.0895 - 0.025) = $5.2577 / 0.0645 = $81.51

*   **計算現值:**
    *   PV(D1) = $4.7424 / (1 + 0.0895)^1 = $4.7424 / 1.0895 = $4.3530
    *   PV(D2) = $4.9321 / (1 + 0.0895)^2 = $4.9321 / 1.1870 = $4.1551
    *   PV(D3) = $5.1294 / (1 + 0.0895)^3 = $5.1294 / 1.2934 = $3.9658
    *   PV(Terminal Value) = $81.51 / (1 + 0.0895)^3 = $81.51 / 1.2934 = $63.0207

*   **每股內在價值 (Intrinsic Value per Share)**:
    *   $4.3530 + $4.1551 + $3.9658 + $63.0207 = $75.49

**DDM 模型估值結果**: $75.49

**合理價位區間**: 考慮到股息增長率和所需報酬率的敏感性，估值可能在 $65 - $85 之間。
