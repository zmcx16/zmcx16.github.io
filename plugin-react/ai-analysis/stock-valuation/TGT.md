#### **2.1 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設:**

**關鍵輸入與預測:**

*   **基準營收 (FY2025):** $1065.7 億美元
*   **營收增長率:**

    *   FY2027: 1.5%
    *   FY2028: 2.0%
    *   FY2029: 2.5%
    *   FY2030: 2.5%
    *   (此後進入永續增長階段，增長率為 2.0%)
*   **營業利潤率 (Operating Margin):** 考慮到競爭和消費者價值導向，我們採用保守的 5.0% (接近原始數據的 4.48%，但低於 Q3 2025 估算的 6.9%，以反映挑戰)。
*   **稅率:** 19.8%

*   **營運資本變動 (Change in Working Capital):** 假設為營收變動的 10%。

**DCF 計算 (簡化):**

#### **2.3 股息折現模型 (Dividend Discount Model, DDM)**

DDM 適用於穩定派發股息且股息增長可預測的公司。

**估值假設:**

*   **股權成本 (Ke):** 7.5%
*   **當前年化股息 (D0):** $4.56/股
*   **股息增長率 (g):** 2.25% (考慮到 Target 的成熟階段和零售業的增長前景，採用一個保守且可持續的長期增長率，介於長期通脹和 GDP 增長之間)

**DDM 計算 (戈登增長模型):**

*   預期下一期股息 (D1) = D0 * (1 + g) = $4.56 * (1 + 0.0225) = $4.6626
*   每股估值 = D1 / (Ke - g) = $4.6626 / (0.075 - 0.0225) = $4.6626 / 0.0525 = **$88.81/股**

**DDM 估值結果與合理價位區間:**
DDM 模型估算出 Target 的每股價值約為 **$88.81**。
考慮到股息增長率和股權成本的敏感性，合理價位區間可能在 **$80 - $100** 之間。

*   **DDM 模型**給出了最低的估值 **$88.81**。DDM 對股息增長率和股權成本非常敏感。由於 Target 是一家成熟公司，其股息增長可能不會像高增長公司那樣快，這限制了 DDM 的估值。此外，DDM 假設所有自由現金流都以股息形式分配，這可能不完全符合公司的資本配置策略。

**最終判斷:**

考慮到當前零售業面臨的挑戰 (消費者支出謹慎、競爭激烈) 以及 Target 近期的營收下滑，DCF 模型可能過於樂觀，尤其是在長期增長率和營業利潤率的假設上。DDM 模型則可能過於保守，未能充分考慮到公司可能將部分現金流用於再投資以促進未來增長。

相對估值提供了一個有用的市場視角。Target 的低倍數相對於 Walmart 和 Costco 確實暗示了潛在的價值。然而，這也可能反映了 Target 在市場份額、規模經濟和品牌溢價方面與頂級競爭對手之間的差距。

綜合來看，Target 的合理估值可能落在 **$100 - $130** 的區間內。當前股價 $110.85 位於這個區間的中間。

*   如果投資者相信 Target 能夠成功應對零售業的挑戰，通過其全通路策略、自有品牌和供應鏈效率提升來恢復增長和盈利能力，那麼當前股價可能是一個合理的買入點，並有潛在的上漲空間接近 DCF 估值的下限或分析師目標價的上限。
