*   **市場情緒：** 遠期本益比 (Forward P/E) 為 15.49，低於歷史平均與同業（如 Constellation Brands），顯示市場仍對其增長潛力持謹慎樂觀態度。

### 2. 估值模型計算

#### A. 折現現金流模型 (DCF Analysis)

這是衡量 BUD 內在價值最核心的方法，因為該公司擁有極強且穩定的自由現金流生成能力。

**假設條件：**
*   **基準 FCF：** 根據數據 P/FCF = 11.67，市值 1,306 億，推算 FCF 約為 **112 億美元**。
*   **預測期增長率 (g1)：** 考慮到 EPS 下年增長預期為 12.46%，我們保守設定前 5 年 FCF 複合年增長率為 **6%**（考慮到全球通膨與高端化策略）。
*   **長期增長率 (g2)：** **2%**（符合成熟消費品產業與通膨預期）。
*   **折現率 (WACC)：** 
    *   無風險利率 (10Y Treasury): 4.2%
    *   Beta: 1.1
    *   股權風險溢價: 5%
    *   計算得出股權成本約 9.7%。考慮到其債務結構，設定 **WACC = 8.0%**。

**計算過程：**
1.  Year 1-5 FCF: 118.7, 125.8, 133.4, 141.4, 149.9 (單位：億美元)
2.  終值 (Terminal Value) = [149.9 * (1+2%)] / (8% - 2%) = 2,548 億美元
3.  折現現值 (PV) = 512 (預測期) + 1,734 (終值折現) = 2,246 億美元
4.  **每股價值 = 2,246 億 / 17.3 億股 ≈ $129.8**

> *註：DCF 結果通常較高，反映了公司長期現金流的潛力，但未完全計入短期債務風險。*
