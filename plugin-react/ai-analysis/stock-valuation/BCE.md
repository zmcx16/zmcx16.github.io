### **第三步：估值模型計算**

#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

*   **預測期：** 5 年
*   **WACC：** 5.5%
*   **長期增長率 (g)：** 0.75%
*   **基礎 FCF (FCF0)：** $32 億

**計算步驟：**

1.  **預測未來 5 年的 FCF：**
    *   FCF1 = $3.20B * (1 + 0.010) = $3.232B
    *   FCF2 = $3.232B * (1 + 0.010) = $3.264B

    *   FCF4 = $3.288B * (1 + 0.0075) = $3.313B
    *   FCF5 = $3.313B * (1 + 0.0075) = $3.338B

2.  **計算終端價值 (Terminal Value, TV)：**
    *   TV = FCF5 * (1 + g) / (WACC - g)
    *   TV = $3.338B * (1 + 0.0075) / (0.055 - 0.0075)
    *   TV = $3.363B / 0.0475 = $70.80B

3.  **計算各期 FCF 和 TV 的現值 (PV)：**
    *   PV(FCF1) = $3.232B / (1.055)^1 = $3.063B
    *   PV(FCF2) = $3.264B / (1.055)^2 = $2.940B
    *   PV(FCF3) = $3.288B / (1.055)^3 = $2.810B

    *   PV(FCF5) = $3.338B / (1.055)^5 = $2.574B
    *   PV(TV) = $70.80B / (1.055)^5 = $54.50B

4.  **計算企業價值 (Enterprise Value, EV)：**

*   **股權成本 (Ke)：** 7.9%

**計算步驟：**

1.  **計算下一期股息 (D1)：**
    *   D1 = D0 * (1 + g) = $1.653 * (1 + 0.0075) = $1.665

2.  **計算每股估值：**

    *   P = $1.665 / (0.079 - 0.0075)
