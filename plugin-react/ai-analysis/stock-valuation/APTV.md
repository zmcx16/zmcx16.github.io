### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型通過預測公司未來的自由現金流 (Free Cash Flow, FCF) 並將其折現回現值來估計公司價值。

**估值假設：**

*   **預測期：** 5 年 (2024 - 2028)
*   **折現率 (WACC)：** 加權平均資本成本 (Weighted Average Cost of Capital)
    *   **無風險利率 (Rf):** 4.5% (參考美國 10 年期國債收益率)
    *   **市場風險溢價 (MRP):** 5.5% (行業標準範圍)
    *   **Beta (β):** 1.6 (參考金融數據網站，反映其行業的波動性)
    *   **股權成本 (Ke) = Rf + β * MRP** = 4.5% + 1.6 * 5.5% = 4.5% + 8.8% = 13.3%
    *   **稅前債務成本 (Kd):** 5.5% (參考公司債券收益率或平均借款利率)
    *   **有效稅率 (t):** 22% (參考最新財報)
    *   **稅後債務成本 = Kd * (1 - t)** = 5.5% * (1 - 0.22) = 5.5% * 0.78 = 4.29%
    *   **債務權益比 (D/E):** 0.88 (來自 JSON)
    *   **股權權重 (We) = E / (D + E)** = 1 / (0.88 + 1) = 1 / 1.88 = 0.532
    *   **債務權重 (Wd) = D / (D + E)** = 0.88 / (0.88 + 1) = 0.88 / 1.88 = 0.468
    *   **WACC = (We * Ke) + (Wd * 稅後債務成本)** = (0.532 * 13.3%) + (0.468 * 4.29%) = 7.07% + 2.01% = **9.08%** (取 9.1%)
*   **自由現金流 (FCF) 預測：**
    *   **基期 FCF (TTM):** 市值 / P/FCF = $16.59B / 9.36 = $1.77B
    *   **營收增長率：**
        *   2024 年：3.8% (基於 2023 年營收 $19.5B 和 2024 年指引中點 $20.25B)
        *   2025 年：6.0% (參考分析師預期，汽車技術市場持續增長)
        *   2026 年：5.0%
        *   2027 年：4.0%
        *   2028 年：3.5% (逐步趨於穩定)
    *   **FCF 利潤率 (FCF/Revenue)：** 假設從 TTM 的 9.1% (1.77B / 19.5B) 逐步提升至 10.5%，反映規模經濟和效率提升。
        *   2024: 9.1%
        *   2025: 9.5%
        *   2026: 10.0%
        *   2027: 10.2%
        *   2028: 10.5%
*   **永續增長率 (g):** 2.5% (略高於長期通脹率，反映公司在成熟市場的長期增長潛力)

**DCF 計算步驟：**

1.  **預測營收：**
    *   2023 營收 (實際): $19.50B
    *   2024 營收: $19.50B * (1 + 0.038) = $20.24B
    *   2025 營收: $20.24B * (1 + 0.060) = $21.45B
    *   2026 營收: $21.45B * (1 + 0.050) = $22.52B
    *   2027 營收: $22.52B * (1 + 0.040) = $23.42B
    *   2028 營收: $23.42B * (1 + 0.035) = $24.24B

2.  **預測自由現金流 (FCF)：**
    *   22024 FCF: $20.24B * 0.091 = $1.84B
    *   2025 FCF: $21.45B * 0.095 = $2.04B
    *   2026 FCF: $22.52B * 0.100 = $2.25B
    *   2027 FCF: $23.42B * 0.102 = $2.39B
    *   2028 FCF: $24.24B * 0.105 = $2.54B

3.  **計算終端價值 (Terminal Value, TV)：**
    *   2029 年 FCF (FCF_year6) = 2028 FCF * (1 + g) = $2.54B * (1 + 0.025) = $2.60B
    *   TV = FCF_year6 / (WACC - g) = $2.60B / (0.091 - 0.025) = $2.60B / 0.066 = $39.39B

4.  **計算各期 FCF 和 TV 的現值 (PV)：**
    *   PV(FCF 2024) = $1.84B / (1 + 0.091)^1 = $1.69B
    *   PV(FCF 2025) = $2.04B / (1 + 0.091)^2 = $1.72B
    *   PV(FCF 2026) = $2.25B / (1 + 0.091)^3 = $1.73B
    *   PV(FCF 2027) = $2.39B / (1 + 0.091)^4 = $1.69B
    *   PV(FCF 2028) = $2.54B / (1 + 0.091)^5 = $1.65B
    *   PV(TV) = $39.39B / (1 + 0.091)^5 = $25.58B

5.  **計算企業價值 (Enterprise Value, EV)：**
    *   EV = PV(FCF 2024) + PV(FCF 2025) + PV(FCF 2026) + PV(FCF 2027) + PV(FCF 2028) + PV(TV)
    *   EV = $1.69B + $1.72B + $1.73B + $1.69B + $1.65B + $25.58B = $34.06B

6.  **計算股權價值 (Equity Value)：**
    *   Equity Value = EV - 淨負債 = $34.06B - $13.099B = $20.96B

7.  **計算每股估值：**
    *   每股估值 = Equity Value / 流通股數 = $20.96B / 2.157 億股 = **$97.17**

**DCF 估值結果：**
*   **合理價位：** $97.17
*   **合理價位區間：** 考慮到假設的敏感性 (WACC ±0.5%, 永續增長率 ±0.5%)，估值區間約為 **$85 - $110**。

### 3. 其他合理估值方法 (PEG Ratio)

JSON 中提供了 APTV 的 PEG 比例為 0.62，這是一個非常吸引人的數字，通常認為 PEG < 1 表示成長被低估。

*   **PEG Ratio = (P/E) / EPS Growth Rate**
*   如果我們使用 APTV 的預期 P/E (12.71x) 和 PEG (0.62)，則隱含的 EPS 增長率為：12.71 / 0.62 ≈ 20.5%。
*   考慮到 2024 年 EPS 從低基數的顯著反彈，以及公司在 ADAS 和電氣化領域的長期增長潛力，20% 左右的未來幾年 EPS 增長率是合理的。
*   如果我們假設一個「合理」的 PEG 比例為 1.0 (即成長股的 P/E 應等於其增長率)，那麼 APTV 的合理預期 P/E 應為 20.5x。
*   **PEG 估值：** 20.5 * 2024 預期 EPS ($6.05) = **$123.03**

**PEG 估值結果：**
*   **合理價位：** $123.03
*   **合理價位區間：** 由於 PEG 對增長率假設非常敏感，此估值應視為潛在上限，區間約為 **$100 - $130**。
