#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於穩定派發股息且股息可預測增長的公司。

**DDM 假設：**
*   **當前年度股息 (D0)：** 0.99 美元/股。
*   **股息增長率 (g)：** 3.13% (過去增長率)。
*   **股權成本 (Ke)：** 6.18% (與 DCF 模型相同)。

**DDM 計算 (戈登增長模型 - Gordon Growth Model)：**
*   由於股息支付率 (82.5% 基於調整後 EPS) 較高，且 Ke (6.18%) 與 g (3.13%) 的差距相對較小，我們需要謹慎。
*   DDM 估值 = D0 * (1 + g) / (Ke - g)
*   DDM 估值 = 0.99 * (1 + 0.0313) / (0.0618 - 0.0313)
*   DDM 估值 = 0.99 * 1.0313 / 0.0305
*   DDM 估值 = 1.021 / 0.0305 = **33.48 美元/股**

**DDM 估值結果：**
*   每股估值 = **33.48 美元/股**

**合理價位區間：** 考慮到股息增長率和股權成本的敏感性，若股息增長率在 2.5% - 3.5% 之間，股權成本在 5.8% - 6.5% 之間，估值可能在 **25.00 - 45.00 美元/股** 之間。

**重要提示：** 儘管 DDM 給出了較高的估值，但 FLO 高達 82.5% 的調整後 EPS 支付率 (甚至有報告稱 GAAP 支付率高達 249.6%) 是一個重大風險。如此高的支付率限制了公司再投資的能力，並可能導致未來股息增長放緩甚至削減。因此，DDM 的結果應謹慎看待。

**綜合結論：**

Flowers Foods (FLO) 的估值分析呈現出複雜的局面。

*   **從 DCF 和 DDM 模型來看，FLO 似乎被嚴重低估。** 這兩種模型都指向遠高於當前股價的內在價值，主要得益於其極低的 Beta 值所帶來的低折現率，以及 DDM 中穩定的股息支付歷史。
