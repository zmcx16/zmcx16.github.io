### 2. 估值模型計算

#### A. 股息折現模型 (Dividend Discount Model, DDM)
由於 FLO 是一家成熟、高分紅且增長穩定的公司，DDM 是最合適的估值方法之一。

*   **假設：**
    *   **預期年度股息 (D1)：** 根據數據，目前股息率約 9%，若以 $10.89 計算，年化股息約為 $0.98。
    *   **股息增長率 (g)：** 過去 5 年平均增長約 3.5% - 4.5%。保守取 **3%**。
    *   **股權成本 (Ke)：** 使用 CAPM 模型。無風險利率 4.2% + Beta (0.45) * 風險溢價 (5.5%) ≈ **6.7%**。
*   **計算：**
    $Value = D1 / (Ke - g) = 0.98 / (0.067 - 0.03) = 0.98 / 0.037 = 26.48$
*   **結果：** **$26.48**

#### B. 折現現金流模型 (DCF)
*   **假設：**
    *   **基準 FCF：** 根據 P/FCF 7.3 與市值 23 億，得出 FCF 約為 3.15 億美元。
    *   **預測期增長率 (5年)：** 考慮到行業成熟度，設為 **2%**。
    *   **終端增長率：** **1.5%**。
    *   **WACC (折現率)：** 考慮到債務比例 (Debt/Eq 1.5)，WACC 估算為 **6.5%**。
*   **計算過程：**
    *   前 5 年現金流折現總和約 14.5 億。
    *   終端價值 (Terminal Value) 折現後約 48 億。
    *   扣除淨債務後除以流通股數。
*   **結果：** **$21.50 - $23.80**

與競爭對手相比，Flowers Foods 的品牌力（尤其是 Dave's Killer Bread）使其擁有較強的定價權。雖然目前面臨消費降級挑戰，但其在有機與健康麵包市場的領先地位提供了長期護城河。
