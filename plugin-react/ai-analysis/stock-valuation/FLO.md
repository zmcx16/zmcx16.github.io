**2. 估值方法應用**

**2.1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2026 年 - 2030 年)。
*   **折現率 (Discount Rate / WACC):** 根據 GuruFocus，Franklin Electric 的 WACC 為 11.01% (截至 2025 年 12 月)。我們將使用此值。
*   **長期增長率 (Terminal Growth Rate):** 考慮到水和燃料輸送系統行業的穩定性以及公司的市場地位，我們假設長期增長率為 2.5%。這通常介於長期通脹率和全球 GDP 增長率之間。
*   **營收增長率 (Revenue Growth Rate):**
    *   2026 年：根據公司指引，銷售額中點增長 3% (22.05 億美元 / 2025 年 21 億美元銷售額)。
    *   2027-2030 年：假設此後增長率逐漸放緩，例如 2.5%、2.0%、1.5%、1.0%。
*   **營業利潤率 (Operating Margin):** 2025 年營業利潤率為 12.6%。假設未來幾年保持穩定或略有提升，我們使用 12.6%。
*   **稅率 (Tax Rate):** 假設為 25%。
*   **資本支出 (Capital Expenditures) 和營運資本變化 (Change in Working Capital):** 假設為營收的穩定百分比，或根據歷史趨勢進行估計。由於缺乏詳細的歷史數據，我們將簡化假設，例如資本支出佔營收的 3%，營運資本變化佔營收變化的 10%。

**DCF 計算步驟 (簡化示例，需要詳細財務報表數據才能精確計算)：**

1.  **預測營收 (Revenue Forecast):**
    *   2025 年銷售額 (估計): 21 億美元 (根據 2026 年指引反推)。
    *   2026 年: 21 億 * (1 + 0.03) = 21.63 億美元
    *   2027 年: 21.63 億 * (1 + 0.025) = 22.17 億美元
    *   2028 年: 22.17 億 * (1 + 0.02) = 22.61 億美元
    *   2029 年: 22.61 億 * (1 + 0.015) = 22.95 億美元
    *   2030 年: 22.95 億 * (1 + 0.01) = 23.18 億美元

2.  **預測營業利潤 (Operating Profit = EBIT):**
    *   EBIT = 營收 * 營業利潤率 (12.6%)
    *   2026 年: 21.63 億 * 0.126 = 2.725 億美元
    *   ... (依此類推)

3.  **計算稅後營業利潤 (NOPAT):**
    *   NOPAT = EBIT * (1 - 稅率)
    *   2026 年: 2.725 億 * (1 - 0.25) = 2.044 億美元
    *   ... (依此類推)

4.  **估計自由現金流 (FCF):**
    *   FCF = NOPAT + 折舊與攤銷 - 資本支出 - 營運資本變化
    *   由於缺乏詳細的折舊與攤銷數據，我們將使用簡化的 FCF 估計，例如基於營業現金流和資本支出。
    *   或者，我們可以從提供的 P/FCF (5.84) 和市值 (44.4 億美元) 反推 2025 年的 FCF: FCF = 市值 / P/FCF = 44.4 億 / 5.84 = 7.60 億美元。這個數字看起來非常高，與營業利潤不符。這可能意味著 P/FCF 數據有誤，或者 FCF 包含了非經常性項目。
    *   鑑於數據限制，我們將基於 NOPAT 進行簡化估計，假設資本支出佔營收的 3%，營運資本變化為營收變化的 10%。
    *   2025 年資本支出 (估計): 21 億 * 0.03 = 0.63 億美元
    *   2026 年資本支出: 21.63 億 * 0.03 = 0.649 億美元
    *   2026 年營運資本變化: (21.63 億 - 21 億) * 0.10 = 0.063 億美元
    *   假設折舊與攤銷約等於資本支出 (簡化)。
    *   FCF = NOPAT - 資本支出 - 營運資本變化 (簡化，忽略折舊與攤銷的非現金性質)
    *   2026 年 FCF: 2.044 億 - 0.0649 億 - 0.0063 億 = 1.332 億美元 (此為非常簡化的估計，實際需要更詳細的財務數據)

5.  **計算終端價值 (Terminal Value, TV):**
    *   TV = (最後一年 FCF * (1 + 長期增長率)) / (WACC - 長期增長率)
    *   假設 2030 年 FCF 為 1.332 億美元 (為簡化，假設 FCF 保持不變，實際應根據增長率計算)
    *   TV = (1.332 億 * (1 + 0.025)) / (0.1101 - 0.025) = 1.3653 億 / 0.0851 = 16.04 億美元

6.  **折現 FCF 和 TV：**
    *   將每年 FCF 和 TV 折現回現值。
    *   由於上述 FCF 估計的簡化性，這裡的 DCF 結果將僅作為一個概念性展示，而非精確估值。

**DCF 估值結果 (概念性):**
由於缺乏詳細的財務報表數據（如折舊攤銷、實際資本支出、營運資本變化等），無法進行精確的 DCF 計算。然而，基於上述假設，如果能獲得完整的財務數據，DCF 模型將提供一個內在價值區間。

**合理價位區間 (DCF):**
基於上述簡化假設，DCF 估值需要更詳細的財務預測。通常，DCF 估值會給出一個範圍，例如在 WACC 和長期增長率的不同假設下。

**2.3. 股息折現模型 (Dividend Discount Model, DDM)**

Franklin Electric 已連續 34 年增加股息，顯示其股息政策穩定且具有增長性，因此 DDM 適用。

**估值假設：**

*   **當前股息 (D0):** 1.12 美元 (年化股息)。
*   **股息增長率 (g):**
    *   短期 (5 年): 考慮到公司預計 2026 年 EPS 增長 9%，以及過去 3 年股息複合年增長率為 8%，我們假設短期股息增長率為 7%。
    *   長期 (永續): 假設為 2.5% (與 DCF 的長期增長率一致)。
*   **股權成本 (Ke):** 我們將使用 WACC 作為股權成本的近似值，即 11.01%。

**DDM 計算步驟 (兩階段 DDM):**

1.  **預測未來 5 年股息 (D1-D5):**
    *   D1 (2026): 1.12 * (1 + 0.07) = 1.1984 美元
    *   D2 (2027): 1.1984 * (1 + 0.07) = 1.2823 美元
    *   D3 (2028): 1.2823 * (1 + 0.07) = 1.3720 美元
    *   D4 (2029): 1.3720 * (1 + 0.07) = 1.4680 美元
    *   D5 (2030): 1.4680 * (1 + 0.07) = 1.5708 美元

2.  **計算終端股息 (D6) 和終端價值 (Terminal Value of Dividends):**
    *   D6 = D5 * (1 + 長期增長率) = 1.5708 * (1 + 0.025) = 1.6101 美元
    *   終端價值 = D6 / (Ke - 長期增長率) = 1.6101 / (0.1101 - 0.025) = 1.6101 / 0.0851 = 18.92 美元

3.  **折現未來股息和終端價值：**

    *   PV(終端價值) = 18.92 / (1 + 0.1101)^5 = 11.22 美元

4.  **DDM 估值 = 所有折現股息之和 + 折現終端價值**
