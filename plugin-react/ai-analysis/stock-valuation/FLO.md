## 美股公司 FLO 股權評價分析報告

### 執行摘要

本報告針對美股公司 Flowers Foods, Inc. (NYSE: FLO) 進行全面的股權評價分析。FLO 是一家在美國生產和銷售包裝烘焙食品的公司，旗下擁有多個知名品牌，如 Nature's Own、Dave's Killer Bread 和 Tastykake 等。

近期，FLO 面臨傳統麵包類別表現不佳、大宗商品成本上升以及促銷環境激烈等挑戰。 儘管如此，公司在 2026 年第一季度實現了每股盈餘 (EPS) 超出預期，並透過定價策略和收購 Simple Mills 帶動淨銷售額年增 1.1%。 管理層已重申全年展望，並透過 Nature's Own 品牌重塑和股息調整來優化資本配置，以期在 2027 財年末將槓桿率降至 3 倍以下。

本報告將透過同業比較分析、折現現金流 (DCF) 模型以及相對估值法，推算 FLO 的內在價值與合理價位區間。

### 1. 同業比較分析 (Industry Peer Comparison)

#### 識別競爭對手
根據 FLO 的業務性質（包裝烘焙食品），我們識別出以下三家主要產業競爭對手：
*   **Conagra Brands Inc. (CAG)**：一家大型包裝食品公司，擁有廣泛的品牌組合。
*   **General Mills Inc. (GIS)**：另一家大型包裝食品公司，在穀物、烘焙產品和零食領域佔有重要地位。
*   **The Kraft Heinz Company (KHC)**：一家大型食品飲料公司，在包裝食品市場具有顯著影響力。

#### 估值地位分析
從同業比較來看，FLO 的估值倍數呈現以下特點：
*   **市值：** FLO 的市值 ($1.59B) 遠小於其主要競爭對手 (CAG $6.18B, GIS $17.17B, KHC $26.64B)，顯示其為一家規模較小的公司。
*   **本益比 (P/E)：** FLO 的追蹤本益比 (21.88) 高於 GIS (7.87)，但由於 CAG 和 KHC 的 EPS 為負，無法直接比較。然而，FLO 的預期本益比 (9.03) 與 CAG (7.59) 和 GIS (9.42) 相比，處於合理區間，甚至略高於 CAG。這可能反映了市場對 FLO 未來盈利改善的預期，儘管其近期面臨挑戰。
*   **股價營收比 (P/S)：** FLO 的 P/S (0.3) 顯著低於所有同業 (CAG 0.55, GIS 0.93, KHC 1.07)，這表明相對於其營收，FLO 的股價處於折價狀態。這可能與其較低的淨利潤率和營收增長率有關。
*   **企業價值倍數 (EV/EBITDA)：** FLO 的 EV/EBITDA (7.0x) 與同業 (CAG 8.1x, GIS 7.56x, KHC 7.9x) 相比，處於較低水平，顯示其企業價值相對於其未計利息、稅項、折舊及攤銷前的盈利能力而言，具有一定的折價。
*   **營收增長率：** FLO 在 Q1 2026 的營收增長率為 1.1%，介於 CAG (2.4%) 和 GIS (-3%) 之間。KHC 的營收增長趨勢較為複雜，Q1 增長但 Q2 指引為負。FLO 的增長表現並不出色，但也沒有明顯落後於整個行業。
*   **利潤率：** FLO 的營業利潤率 (6.7%) 和淨利潤率 (1.38%) 顯著低於 GIS (營業利潤率 14.54%, 淨利潤率 12.12%)。與 CAG 相比，FLO 的營業利潤率較低，但淨利潤率為正，而 CAG 為負。KHC 的利潤率則為負值，顯示其面臨嚴峻的盈利挑戰。FLO 較低的利潤率是其估值折價的一個重要原因。

**總結：** FLO 相對於同業而言，在市值上是較小的參與者。其 P/S 和 EV/EBITDA 倍數顯示其估值處於折價狀態，這主要是由於其較低的利潤率和相對平庸的營收增長。然而，其預期 P/E 倍數與同業相近，可能反映了市場對其未來改善的期望。FLO 的估值折價在一定程度上是合理的，因為其盈利能力和規模不及主要競爭對手。若要實現估值溢價，FLO 需要展現出更強勁的營收增長或顯著的利潤率改善。

### 2. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估算公司未來自由現金流的現值，以得出其內在價值。

#### 預測假設
*   **預測期間：** 5 年（2026-2030）。
*   **營收增長率預期：**
    *   基於 FLO Q1 2026 營收年增 1.1%，以及管理層預期 2026 年營收「基本持平」的指引，我們預計 2026 年營收增長率為 0.5%。
    *   考慮到傳統麵包市場的挑戰和公司轉型努力，我們預計未來幾年營收將溫和增長。預計 2027-2028 年為 1.5%，2029-2030 年降至 1.0%。
*   **營業利潤率 (Operating Margin)：** FLO 目前的營業利潤率為 6.7%。考慮到大宗商品成本上升和行銷投資增加帶來的近期壓力，我們預計短期內利潤率將保持穩定或略有下降，然後隨著效率提升和品牌重塑效果顯現而逐步改善。
    *   預計 2026-2027 年為 6.5%，2028-2030 年逐步提升至 7.0%。
*   **稅率：** 假設有效稅率為 25%。
*   **資本支出 (Capital Expenditure, CapEx)：** 根據 Q1 2026 財報電話會議，資本支出指引為 $115M-$125M，其中約 $2M/麵包店用於維護，其餘用於增長和生產力措施。 我們假設 CapEx 佔營收的比例約為 2.5%-2.0%，並隨營收增長而調整。
*   **營運資本變動 (Change in Working Capital)：** 假設營運資本變動佔營收的比例為 0.5%。

#### 折現率與終值
*   **加權平均資本成本 (WACC) 計算邏輯：**
    *   **股權成本 (Cost of Equity, Ke)：** 使用資本資產定價模型 (CAPM)。
        *   **無風險利率 (Risk-Free Rate)：** 採用美國 10 年期國債收益率，假設為 4.5%。
        *   **市場風險溢酬 (Market Risk Premium, MRP)：** 假設為 5.5%。
        *   **Beta：** FLO 的 Beta 為 0.40。
        *   Ke = 無風險利率 + Beta * MRP = 4.5% + 0.40 * 5.5% = 4.5% + 2.2% = 6.7%。
    *   **債務成本 (Cost of Debt, Kd)：**
        *   FLO 的長期債務/股權比率為 1.21。 考慮到當前利率環境和公司債務水平，假設稅前債務成本為 6.0%。
        *   **稅率：** 25%。
        *   稅後債務成本 = Kd * (1 - 稅率) = 6.0% * (1 - 0.25) = 4.5%。
    *   **資本結構：**
        *   股權市值 (E) = $1.59B。
        *   債務市值 (D) = 股權市值 * 債務/股權比率 = $1.59B * 1.57 = $2.49B (使用提供的 Debt/Eq 1.57)。
        *   總資本 (V) = E + D = $1.59B + $2.49B = $4.08B。
        *   股權權重 (E/V) = $1.59B / $4.08B = 39%。
        *   債務權重 (D/V) = $2.49B / $4.08B = 61%。
    *   **WACC = (E/V) * Ke + (D/V) * Kd * (1 - 稅率)**
        *   WACC = 39% * 6.7% + 61% * 4.5% = 2.61% + 2.75% = 5.36%。
*   **終值增長率 (Terminal Growth Rate)：** 考慮到烘焙食品行業的成熟性質和 FLO 面臨的挑戰，我們採用保守的終值增長率，假設為 1.0%。

### 3. 多維度估值法 (Other Valuation Methods)

#### 相對估值法 (Relative Valuation)
我們將使用同業的平均估值倍數來推算 FLO 的目標價位。

*   **P/S 倍數法：**
    *   同業平均 P/S (CAG 0.55, GIS 0.93, KHC 1.07) = (0.55 + 0.93 + 1.07) / 3 = 0.85。
    *   FLO 營收 (假設為 $5.256B，來自年銷售額數據)。
    *   FLO 目標市值 = 營收 * 同業平均 P/S = $5.256B * 0.85 = $4.468B。
    *   FLO 目標股價 = 目標市值 / 流通股數 (約 211.99M 股) = $4.468B / 211.99M = **$21.08**。

*   **EV/EBITDA 倍數法：**
    *   同業平均 EV/EBITDA (CAG 8.1x, GIS 7.56x, KHC 7.9x) = (8.1 + 7.56 + 7.9) / 3 = 7.85x。
    *   FLO EBITDA (假設為 $642M，來自數據)。
    *   FLO 目標企業價值 (EV) = EBITDA * 同業平均 EV/EBITDA = $642M * 7.85 = $5.03B。
    *   FLO 股權價值 = 目標 EV - 淨債務。
        *   FLO 總債務 (Debt/Eq 1.57, Equity $1.59B) = $1.59B * 1.57 = $2.49B。
        *   假設現金為 $100M (未提供具體數據，保守估計)。
        *   淨債務 = $2.49B - $0.1B = $2.39B。
    *   FLO 目標市值 = $5.03B - $2.39B = $2.64B。
    *   FLO 目標股價 = 目標市值 / 流通股數 = $2.64B / 211.99M = **$12.45**。

*   **P/E 倍數法 (Forward)：**
    *   同業平均預期 P/E (CAG 7.59, GIS 9.42) = (7.59 + 9.42) / 2 = 8.51。
    *   FLO 預期 EPS (Forward P/E 9.03, Current Price 7.48) = 7.48 / 9.03 = $0.828。
    *   FLO 目標股價 = 預期 EPS * 同業平均預期 P/E = $0.828 * 8.51 = **$7.04**。

相對估值法結果差異較大，P/S 倍數法給出較高估值，EV/EBITDA 倍數法居中，P/E 倍數法給出較低估值。這反映了不同指標對公司不同層面的重視。考慮到 FLO 較低的利潤率，P/S 和 EV/EBITDA 可能更能反映其資產和營運規模的價值。

#### 股息折現模型 (DDM)
FLO 過去 12 年連續提高股息，但 2026 年 EPS 低於股息承諾，股息策略正在審查中。這使得 DDM 模型在此時的可靠性降低。然而，如果假設公司能夠維持其股息支付能力並實現溫和增長，我們可以進行一個簡化的 DDM 分析。

*   **當前股息：** 根據提供的數據，Dividend % 為 0.1326，但這通常是股息收益率。從 Google Finance 來看，季度股息為 $0.13，年化約為 $0.52。
*   **股息增長率 (g)：** 考慮到公司面臨的挑戰和股息審查，我們假設一個非常低的長期股息增長率，例如 0.5%。
*   **股權成本 (Ke)：** 沿用 DCF 模型中的 6.7%。

*   **DDM 估值 = D1 / (Ke - g)**
    *   D1 = 當前股息 * (1 + g) = $0.52 * (1 + 0.005) = $0.5226。
    *   DDM 估值 = $0.5226 / (0.067 - 0.005) = $0.5226 / 0.062 = **$8.43**。

DDM 模型給出的估值與當前股價接近，但由於股息政策的不確定性，此結果應謹慎參考。
