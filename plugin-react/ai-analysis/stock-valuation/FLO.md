### 2. 估值假設

在進行估值之前，我們需要設定一些關鍵假設：

*   **無風險利率 (Risk-Free Rate)：** 參考美國10年期國債收益率，假設為 4.20%。
*   **市場風險溢價 (Market Risk Premium, MRP)：** 採用歷史平均值，假設為 5.5%。
*   **Beta 值：** 根據網路搜尋，FLO 的 Beta 值約為 1.35。
*   **股權成本 (Cost of Equity, Ke)：** 根據資本資產定價模型 (CAPM) 計算：
    Ke = 無風險利率 + Beta * 市場風險溢價
    Ke = 4.20% + 1.35 * 5.5% = 4.20% + 7.425% = 11.625%
*   **債務成本 (Cost of Debt, Kd)：** 根據公司最近的財報和利率環境，假設稅前債務成本為 6.0%。
*   **企業稅率 (Tax Rate)：** 採用美國聯邦企業稅率，假設為 25%。
*   **負債權益比 (Debt/Equity Ratio)：** 根據提供數據，Debt/Eq = 1.5。
*   **加權平均資本成本 (WACC)：**
    WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1 - Tax Rate)
    其中 E/(E+D) = 1 / (1 + D/E) = 1 / (1 + 1.5) = 1 / 2.5 = 0.4
    D/(E+D) = D/E / (1 + D/E) = 1.5 / 2.5 = 0.6
    WACC = 0.4 * 11.625% + 0.6 * 6.0% * (1 - 0.25)
    WACC = 4.65% + 0.6 * 6.0% * 0.75
    WACC = 4.65% + 2.7% = 7.35%
*   **長期增長率 (Terminal Growth Rate, g)：** 假設為 2.5% (略低於長期通脹和全球經濟增長率)。
*   **預測期間：** 5 年。

### 3. 估值模型與結果

#### 3.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測未來自由現金流 (Free Cash Flow, FCF)。由於缺乏詳細的歷史FCF數據，我們將基於營收增長預期、利潤率和資本支出進行估算。

**DCF 估值步驟：**
1.  **預測營收：** 根據2024年銷售額46.5億美元和2025年1%-3%的增長預期，我們假設未來5年營收增長率如下：
    *   2025年：2.0% (取預期中值)
    *   2026年：2.5%
    *   2027年：2.5%
    *   2028年：2.0%
    *   2029年：2.0%
2.  **預測息稅前利潤 (EBIT)：** 根據提供的 Oper. Margin (營業利潤率) 0.0691，我們假設未來幾年營業利潤率穩定在 6.9%。
3.  **計算稅後息稅前利潤 (NOPAT)：** NOPAT = EBIT * (1 - Tax Rate)。
4.  **預測資本支出 (CAPEX) 和折舊攤銷 (D&A)：** 假設 CAPEX 略高於 D&A，以支持增長。
5.  **預測營運資本變動 (Change in Working Capital)：** 假設與營收增長成比例。
6.  **計算自由現金流 (FCF)：** FCF = NOPAT + D&A - CAPEX - Change in Working Capital。
7.  **計算終端價值 (Terminal Value, TV)：** TV = FCF_last_year * (1 + g) / (WACC - g)。
8.  **折現 FCF 和 TV：** 將所有 FCF 和 TV 折現到現在。

#### 3.3 股息折現模型 (Dividend Discount Model, DDM)

我們將使用戈登增長模型 (Gordon Growth Model)，假設股息以一個恆定速度增長。
DDM = DPS_next_year / (Ke - g_dividend)

*   **股息增長率 (g_dividend)：** 考慮到公司營收增長預期和盈利能力，我們假設股息長期增長率為 2.0%。
*   **股權成本 (Ke)：** 11.625%

*   **計算：**
    DPS_next_year = 1.00 * (1 + 0.02) = 1.02 美元
    DDM = 1.02 / (0.11625 - 0.02) = 1.02 / 0.09625 = 10.60 美元

3.  **相對估值 (16.00 - 20.00美元)** 顯示出較高的估值，特別是基於 P/E 倍數。這可能暗示 FLO 相對於行業平均水平，其盈利能力或增長潛力被市場低估，或者其當前股價相對於同業而言具有吸引力。然而，P/S 和 P/B 估值則相對保守。FLO 的 P/S (0.45) 遠低於行業平均 (1.5)，這可能表明其營收規模大但利潤率相對較低，或者市場對其營收增長前景持謹慎態度。

**綜合結論：**

綜合來看，FLO 的估值呈現出一定的分歧。

*   **DCF 模型** 由於其對未來現金流預測的敏感性以及我們在數據限制下的簡化假設，給出了較低的估值。如果公司的自由現金流在未來能夠實現更強勁的增長，DCF 估值將會上升。
*   **DDM 模型** 與當前股價接近，表明市場可能認可其穩定的股息支付能力。
*   **相對估值模型** 則顯示 FLO 相對於同業可能存在被低估的潛力，尤其是在 P/E 倍數方面。然而，其較低的 P/S 和 P/B 倍數也可能反映了市場對其盈利能力和資產效率的擔憂。

考慮到 FLO 在家居和安全產品領域的市場地位，以及其穩定的股息支付，當前股價 11.11 美元可能處於一個相對合理的區間。如果公司能夠實現其2025年的盈利預期並展現出更強勁的營收增長，那麼其股價有望向相對估值模型所暗示的更高區間靠攏。
