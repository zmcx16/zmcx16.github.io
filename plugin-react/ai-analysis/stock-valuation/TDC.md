### **估值方法與計算**

#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型旨在估計公司未來自由現金流的現值，以得出其內在價值。

**估值假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2024 - 2028)
*   **長期增長率 (Terminal Growth Rate, g):** 2.0% (略低於長期 GDP 增長率，反映成熟企業的穩健增長)
*   **加權平均資本成本 (Weighted Average Cost of Capital, WACC):**

    *   **無風險利率 (Risk-Free Rate, Rf):** 4.3% (參考美國 10 年期國債收益率)
    *   **市場風險溢酬 (Market Risk Premium, MRP):** 6.0% (行業標準範圍)
    *   **Beta (TDC):** 1.3 (透過網路查詢，反映其相對於市場的波動性)
    *   **股權成本 (Cost of Equity, Ke) = Rf + Beta * MRP**
        *   Ke = 4.3% + 1.3 * 6.0% = 4.3% + 7.8% = 12.1%
    *   **債務成本 (Cost of Debt, Kd):** 5.5% (根據公司利息支出與總債務估算)
    *   **稅率 (Tax Rate):** 22% (參考公司歷史有效稅率及未來預期)
    *   **稅後債務成本 (After-tax Kd) = Kd * (1 - Tax Rate)**
        *   After-tax Kd = 5.5% * (1 - 0.22) = 5.5% * 0.78 = 4.29%
    *   **股權市值 (Market Value of Equity, E):** $2.86 Billion (市值)
    *   **債務市值 (Market Value of Debt, D):** $1.2 Billion (總債務)
    *   **總資本 (Total Capital, V) = E + D**
        *   V = $2.86B + $1.2B = $4.06 Billion
    *   **WACC = (E/V) * Ke + (D/V) * After-tax Kd**
        *   WACC = ($2.86B / $4.06B) * 12.1% + ($1.2B / $4.06B) * 4.29%
        *   WACC = 0.704 * 12.1% + 0.296 * 4.29% = 8.52% + 1.27% = 9.79%
        *   **WACC ≈ 9.8%**

*   **自由現金流 (Free Cash Flow, FCF) 預測：**
    *   我們將以 2024 財年指引的 FCF 中位數 $340 百萬作為起始點。
    *   考慮到公司雲端轉型帶來的效率提升和潛在的營收增長，我們預計 FCF 增長率將在預測期內逐步放緩。
        *   2024 (FY24 Guidance): $340 百萬
        *   2025: 5% 增長 ($340M * 1.05 = $357.0M)
        *   2026: 4% 增長 ($357.0M * 1.04 = $371.3M)
        *   2027: 3% 增長 ($371.3M * 1.03 = $382.4M)
        *   2028: 2.5% 增長 ($382.4M * 1.025 = $392.0M)

**DCF 計算步驟：**

1.  **計算預測期內的自由現金流現值 (PV of FCFs):**
    *   PV(FCF 2024) = $340M / (1 + 0.098)^1 = $309.65M
    *   PV(FCF 2025) = $357M / (1 + 0.098)^2 = $296.90M
    *   PV(FCF 2026) = $371.3M / (1 + 0.098)^3 = $279.00M
    *   PV(FCF 2027) = $382.4M / (1 + 0.098)^4 = $259.90M
    *   PV(FCF 2028) = $392.0M / (1 + 0.098)^5 = $242.90M
    *   **總計 PV(FCFs) = $1,388.35M**

2.  **計算終值 (Terminal Value, TV):**
    *   TV = FCF_2028 * (1 + g) / (WACC - g)
    *   TV = $392.0M * (1 + 0.02) / (0.098 - 0.02)
    *   TV = $399.84M / 0.078 = $5,126.15M

3.  **計算終值的現值 (PV of TV):**
    *   PV(TV) = $5,126.15M / (1 + 0.098)^5 = $3,177.00M

4.  **計算企業價值 (Enterprise Value, EV):**
    *   EV = 總計 PV(FCFs) + PV(TV)
    *   EV = $1,388.35M + $3,177.00M = $4,565.35M

5.  **計算股權價值 (Equity Value):**
    *   Equity Value = EV + 現金及約當現金 - 總債務
    *   Equity Value = $4,565.35M + $500M - $1,200M = $3,865.35M

6.  **計算每股內在價值 (Intrinsic Value per Share):**
    *   Intrinsic Value per Share = Equity Value / 流通股數
    *   Intrinsic Value per Share = $3,865.35M / 93.03M = **$41.55**

**DCF 估值結果與合理價位區間：**
*   **DCF 估值：$41.55**
*   **合理價位區間 (敏感性分析):**
