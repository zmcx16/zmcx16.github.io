### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來產生的自由現金流的現值。

**估值假設：**

*   **預測期：** 5 年 (2026 年至 2030 年)。
*   **折現率 (WACC - 加權平均資本成本)：**
    *   由於沒有直接提供 WACC，我們需要估計。
    *   **無風險利率 (Rf)：** 參考美國 10 年期國債收益率，假設為 4.5%。
    *   **市場風險溢價 (MRP)：** 假設為 5.5%。
    *   **Beta (β)：** 雖然未直接提供，但考慮到其科技服務性質，且與 Accenture 等公司競爭，其 Beta 可能略高於 1。我們假設 Beta 為 1.2。
    *   **股本成本 (Ke) = Rf + β * MRP** = 4.5% + 1.2 * 5.5% = 4.5% + 6.6% = 11.1%。
    *   **債務成本 (Kd)：** 假設為 6% (考慮到其低負債率和當前利率環境)。
    *   **稅率 (t)：** 假設為 25% (綜合考慮美國聯邦和州稅率，以及其國際業務)。
    *   **債務/股本比 (D/E)：** 根據提供的數據，Debt/Eq 為 0.23。
    *   **WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1-t)**
        *   E/(E+D) = 1 / (1 + D/E) = 1 / (1 + 0.23) = 1 / 1.23 ≈ 0.813
        *   D/(E+D) = D/E / (1 + D/E) = 0.23 / 1.23 ≈ 0.187
        *   WACC = 0.813 * 11.1% + 0.187 * 6% * (1 - 0.25)
        *   WACC = 0.0902 + 0.187 * 0.06 * 0.75 = 0.0902 + 0.008415 ≈ 0.0986 = 9.86%。
        *   **我們將使用約 10% 作為 WACC。**
*   **收入增長率：**
    *   2026 年收入指導為 0.2% 至 2.2%。考慮到其 AI 策略和強勁的預訂量，我們假設在預測期內收入增長將逐步恢復。
    *   2026 年：2% (取指導區間中值)
    *   2027 年：5%
    *   2028 年：7%
    *   2029 年：6%
    *   2030 年：5%
*   **營運利潤率：** 2026 年指導為 14% 至 15%。我們假設在預測期內維持在 14.5% 左右。
*   **稅率：** 25%。
*   **資本支出 (CAPEX)：** 假設為收入的 3%。
*   **折舊與攤銷 (D&A)：** 假設為收入的 2%。
*   **營運資本變動 (Change in NWC)：** 假設為收入增長的 5%。
*   **永續增長率 (Terminal Growth Rate)：** 假設為 3% (考慮到其所處的科技服務行業的長期增長潛力)。

**DCF 計算步驟：**

1.  **預測收入 (Revenue)。**
2.  **計算營運利潤 (Operating Profit = Revenue * Operating Margin)。**
3.  **計算稅後營運利潤 (NOPAT = Operating Profit * (1 - Tax Rate))。**
4.  **計算自由現金流 (FCFF = NOPAT + D&A - CAPEX - Change in NWC)。**
5.  **計算終端價值 (Terminal Value, TV)。**
6.  **將所有 FCFF 和 TV 折現回現值。**
