#### **1. 折現現金流模型 (Discounted Cash Flow, DCF)**

**步驟 1：計算預測期內的自由現金流 (Free Cash Flow to Firm, FCFE)**

**步驟 2：計算終值 (Terminal Value, TV)**

終值代表預測期之後的所有自由現金流的現值。
終值 (TV) = (2030 年自由現金流 * (1 + 長期增長率)) / (WACC - 長期增長率)
TV = (390.50 * (1 + 0.025)) / (0.08 - 0.025)
TV = (390.50 * 1.025) / 0.055
TV = 400.26 / 0.055
TV = 7,277.45 百萬美元

**步驟 3：計算終值的現值**

終值的現值 = 終值 / (1 + WACC)^預測期年數
終值的現值 = 7,277.45 / (1 + 0.08)^5
終值的現值 = 7,277.45 / 1.4693
終值的現值 = 4,953.07 百萬美元

**步驟 4：計算公司總價值 (Total Firm Value)**

公司總價值 = 預測期內自由現金流的現值總和 + 終值的現值

公司總價值 = 6,239.90 百萬美元

**步驟 5：計算股權價值 (Equity Value)**

股權價值 = 公司總價值 - 淨債務

股權價值 = 6,239.90 - 116.4 = 6,123.50 百萬美元

每股估值 = 6,123.50 百萬美元 / 42.94 百萬股
