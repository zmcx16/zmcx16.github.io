### III. 估值模型

我們將應用折現現金流模型 (DCF)、相對估值和股息折現模型 (DDM) 進行分析。

#### A. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**1. 估值假設：**

*   **預測期間：** 5 年明確預測期，之後進入永續成長期。
*   **起始營收：** 根據 2024 財年 Q2 財報，FY23 營收約 291 億美元。FY24 營收指引為 0-2% 成長。我們取中位數 1% 成長，FY24 營收約 293.9 億美元。
*   **營收成長率：**
    *   考慮到 AI 伺服器和 GreenLake 的成長，以及 Juniper 收購的潛在影響，但同時考慮到傳統業務的壓力，我們假設：
        *   FY25-FY26: 3.5% (考慮 Juniper 整合和 AI 伺服器持續成長)
        *   FY27-FY28: 2.5%
        *   永續成長率 (g)：1.5% (保守估計，略低於長期 GDP 成長率)
*   **營業利潤率 (Operating Margin)：** 目前為 4.89%。考慮到公司轉型和效率提升，以及 AI 伺服器帶來的潛在利潤率改善，但同時考慮競爭壓力，我們假設逐步提升：
    *   FY24: 5.0%
    *   FY25: 5.5%
    *   FY26: 6.0%
    *   FY27: 6.5%
    *   FY28 及以後: 7.0%
*   **稅率：** 根據近期財報和行業平均，假設有效稅率為 22%。
*   **資本支出 (CapEx) 與折舊攤銷 (D&A)：**
    *   CapEx 佔營收比：3.0% (作為成熟科技公司，維持在一定水平)
    *   D&A 佔營收比：2.5% (通常略低於 CapEx)
*   **營運資本變動 (Change in NWC)：** 假設為營收變動的 1% (成熟公司通常變動不大)。
*   **加權平均資本成本 (WACC) 計算：**
    *   **無風險利率 (Risk-Free Rate)：** 採用美國 10 年期國債收益率，約 4.2%。
    *   **市場風險溢酬 (Market Risk Premium, MRP)：** 5.5% (行業常用範圍)。
    *   **Beta：** 1.1 (根據網路搜尋，HPE 的 Beta 值約在 1.0-1.2 之間)。
    *   **股權成本 (Cost of Equity, Ke)：** Ke = 無風險利率 + Beta * MRP = 4.2% + 1.1 * 5.5% = 4.2% + 6.05% = 10.25%。
    *   **債務成本 (Cost of Debt, Kd)：** 考慮當前利率環境和公司債務水平，假設為 5.5%。
    *   **稅率：** 22%。
    *   **債務權益比 (Debt/Equity)：** 0.98 (來自提供數據)。
    *   **權重：**
        *   股權市值 (E) = 326.8 億美元。
        *   債務市值 (D) = 股權市值 * 債務權益比 = 326.8 億 * 0.98 = 320.26 億美元。
        *   總資本 (V) = E + D = 326.8 + 320.26 = 647.06 億美元。
        *   股權權重 (We) = E / V = 326.8 / 647.06 = 0.505。
        *   債務權重 (Wd) = D / V = 320.26 / 647.06 = 0.495。
    *   **WACC = (We * Ke) + (Wd * Kd * (1 - Tax Rate))**
        *   WACC = (0.505 * 10.25%) + (0.495 * 5.5% * (1 - 0.22))
        *   WACC = 5.176% + (0.495 * 5.5% * 0.78)
        *   WACC = 5.176% + 2.123% = 7.299%
        *   **取整為 7.3%。**

*   **永續價值 (Terminal Value, TV) 計算：**
    *   TV = (FCF_FY28 * (1 + g)) / (WACC - g)
    *   TV = (15.68 億 * (1 + 0.015)) / (0.073 - 0.015)
    *   TV = (15.68 * 1.015) / 0.058
    *   TV = 15.914 / 0.058 = 274.38 億美元
*   **永續價值現值：**
    *   PV(TV) = TV / (1 + WACC)^5 = 274.38 億 / (1.073)^5 = 274.38 億 / 1.422 = 192.95 億美元

*   **企業價值 (Enterprise Value, EV)：**
    *   EV = 預測期 FCF 現值總和 + 永續價值現值
    *   EV = 49.52 億 + 192.95 億 = 242.47 億美元

*   **股權價值 (Equity Value)：**
    *   股權價值 = EV + 現金及約當現金 - 總債務
    *   根據最新財報，HPE 現金約 30 億美元，總債務約 320 億美元 (基於 D/E 0.98 和市值計算)。
    *   股權價值 = 242.47 億 + 30 億 - 320 億 = -47.53 億美元。

**DCF 估值結果分析：**
這個結果顯示股權價值為負，這是一個非常不尋常且需要警惕的結果。這通常意味著：
1.  **假設過於保守：** 營收成長、利潤率或永續成長率可能被低估。
2.  **WACC 過高：** 折現率可能過高，導致未來現金流現值被嚴重低估。
3.  **公司財務結構問題：** 債務過高，導致即使企業價值為正，股權價值也可能為負。
4.  **模型不適用：** 對於盈利能力極低或負值，且債務較高的公司，DCF 模型可能需要更精細的調整或不完全適用。

    *   TV = (22.72 億 * (1 + 0.015)) / (0.073 - 0.015) = 23.06 億 / 0.058 = 397.59 億美元

1.  **市場對 HPE 的估值，可能並非完全基於其當前的自由現金流和傳統 DCF 假設。**
2.  **市場可能對 Juniper 收購案帶來的長期協同效應和成長潛力有更高的預期，而這些預期在保守的 DCF 模型中難以完全捕捉。**
3.  **HPE 的高債務水平對股權價值產生了顯著的稀釋作用。**
4.  **DCF 模型對於處於轉型期、盈利能力波動較大或有重大收購案的公司，其假設的敏感性極高，需要非常謹慎。**

**合理價位區間 (DCF)：** 考慮到上述挑戰，DCF 模型在此可能不是最直接的估值工具。但如果堅持其結果，則暗示股價被高估。然而，我們必須承認，市場顯然有不同的看法。

*   **EV/EBITDA：** 如果我們假設 HPE 的 EV/EBITDA 應該與行業平均 (約 10-13) 接近，但考慮其轉型和盈利挑戰，給予略低的倍數，例如 9-11 倍。
    *   假設 FY24 EBITDA 為 30 億美元 (基於分析師預期和公司指引推算)。
    *   EV = 9 * 30 億 = 270 億美元。
    *   EV = 11 * 30 億 = 330 億美元。
    *   股權價值 = EV + 現金 - 債務 = 270 億 + 30 億 - 320 億 = -20 億美元。
    *   股權價值 = 330 億 + 30 億 - 320 億 = 40 億美元。
    *   每股價值 = 40 億 / 13.34 億股 = **$3.00**。
    *   這個結果再次顯示，在考慮其債務後，基於 EBITDA 的估值也偏低。

**相對估值結論：**
HPE 在 P/S、P/B 和 Forward P/E 倍數上均顯著低於其主要競爭對手。這可能反映了市場對其歷史盈利能力、成長速度和轉型風險的擔憂。然而，極低的 Forward P/E (8.91) 是一個強烈的信號，表明如果公司能夠實現其預期的未來盈利，其股價可能存在上漲空間。市場可能正在等待 Juniper 收購案的完成和整合效果，以及 AI 伺服器業務能否持續高速成長並提升整體利潤率。

**合理價位區間 (相對估值)：**
*   如果 HPE 的 Forward P/E 應達到行業平均的低端 (例如 14 倍)，則其合理股價應為 $24.49 * (14 / 8.91) = $38.47。
*   如果 HPE 的 P/S 應達到行業平均的低端 (例如 1.5 倍)，則其合理股價應為 $24.49 * (1.5 / 0.95) = $38.67。
*   但考慮到 HPE 較低的盈利能力和較高的債務，給予行業平均的估值可能不合理。如果給予其 Forward P/E 10-12 倍的區間：
    *   10 倍 Forward P/E: $24.49 * (10 / 8.91) = $27.48
    *   12 倍 Forward P/E: $24.49 * (12 / 8.91) = $32.90
*   **相對估值合理區間：$27.50 - $33.00**

#### C. 股息折現模型 (Dividend Discount Model, DDM)

HPE 支付穩定的股息，因此 DDM 適用。然而，由於其盈利能力較低且 FCF 遠高於股息，表明公司有大量資金用於再投資或償債，而非僅僅分配股息，因此 DDM 可能無法完全捕捉其價值。

**1. 估值假設：**

*   **當前股息 (D0)：** 股價 $24.49 * 股息收益率 2.17% = $0.532 每股。
*   **股權成本 (Ke)：** 10.25% (與 DCF 模型相同)。
*   **股息成長率 (g)：** 考慮到 HPE 較低的 ROE 和盈利能力，以及其轉型期的再投資需求，股息成長率應保守。
    *   我們假設一個兩階段成長模型：
        *   第一階段 (3 年)：3% (略高於長期通脹，但低於 EPS 成長預期，反映再投資需求)
        *   第二階段 (永續成長)：1.5% (與 DCF 永續成長率一致)

**2. DDM 計算過程：**

*   D1 = $0.532 * (1 + 0.03) = $0.548
*   D2 = $0.548 * (1 + 0.03) = $0.564
*   D3 = $0.564 * (1 + 0.03) = $0.581
*   D4 (用於計算永續價值) = $0.581 * (1 + 0.015) = $0.590

*   **永續價值 (Terminal Value, TV) 計算：**
    *   TV = D4 / (Ke - g_terminal) = $0.590 / (0.1025 - 0.015) = $0.590 / 0.0875 = $6.74

*   **現值計算：**
    *   PV(D1) = $0.548 / (1.1025)^1 = $0.497
    *   PV(D2) = $0.564 / (1.1025)^2 = $0.464
    *   PV(D3) = $0.581 / (1.1025)^3 = $0.436
    *   PV(TV) = $6.74 / (1.1025)^3 = $5.06

*   **每股價值 = PV(D1) + PV(D2) + PV(D3) + PV(TV)**
    *   每股價值 = $0.497 + $0.464 + $0.436 + $5.06 = **$6.46**

**DDM 結論：**
DDM 模型得出的每股價值 $6.46 遠低於當前股價。這再次印證了 DDM 對於 HPE 這種公司可能不是最合適的估值方法。HPE 的低 ROE 和高 FCF/股息比率表明，公司將大部分現金流用於再投資或償債，而非僅僅通過股息回報股東。因此，單純基於股息成長來估值會嚴重低估其內在價值。

**合理價位區間 (DDM)：** 由於 DDM 在此情況下可能不適用，其結果僅供參考，不作為主要估值依據。

### IV. 綜合結論與合理價位區間

我們應用了三種估值模型，並得到了截然不同的結果：

*   **DCF 模型：** 即使調整了 FCF 假設，得出的每股價值約為 **$5.40**，遠低於現價。這主要受高債務和相對保守的成長/利潤率假設影響。
*   **相對估值模型：** 基於 Forward P/E 和 P/S 倍數，並考慮到 HPE 相對較低的盈利能力，合理價位區間約為 **$27.50 - $33.00**。
*   **DDM 模型：** 得出的每股價值約為 **$6.46**，遠低於現價，表明該模型可能不適用於 HPE。

**模型間的估值結果比較與分析：**

DCF 和 DDM 模型都給出了遠低於市場價格的估值，這通常意味著：
1.  **市場對 HPE 的未來成長和盈利能力有著比模型假設更樂觀的預期。** 特別是 AI 伺服器和 GreenLake 的成長，以及 Juniper 收購案的潛在協同效應，可能被市場賦予了更高的價值。
2.  **HPE 的高債務水平對股權價值產生了顯著的稀釋作用，這在 DCF 模型中尤為明顯。**
3.  **HPE 處於轉型期，其盈利模式和現金流結構正在變化，傳統的靜態模型可能難以完全捕捉其動態價值。** 尤其是其 P/E 為負，但 Forward P/E 卻很低，這表明市場預期其盈利將大幅改善。

相對估值模型則顯示，HPE 在多個倍數上都顯著低於同行，尤其是 Forward P/E。這暗示如果公司能夠實現其預期的盈利改善，其股價可能存在被低估的空間。分析師的平均目標價 $26.47 也支持了股價有小幅上漲空間的觀點。

**綜合結論：**

HPE 是一家正在經歷重大轉型的公司，其傳統業務面臨挑戰，但 AI 伺服器和即服務模式 (GreenLake) 帶來了新的成長動力。Juniper Networks 的收購案也將重塑其競爭格局。

儘管 DCF 和 DDM 模型給出了較低的估值，但這可能更多地反映了這些模型在處理高債務、低盈利能力和重大轉型期公司時的局限性，以及市場對未來潛力的更高預期。

相對估值模型則指出，HPE 目前的估值倍數 (尤其是 Forward P/E) 相對於同行顯得非常便宜。如果公司能夠成功整合 Juniper，並持續推動 AI 伺服器和 GreenLake 業務的成長，同時改善其盈利能力，那麼其股價有潛力向行業平均水平靠攏。

**合理價位區間：**

考慮到市場對 HPE 未來成長的樂觀預期，以及其相對估值的吸引力，我們將更側重於相對估值的結果，並結合分析師目標價。

*   **保守估計：** 考慮到其盈利能力和債務挑戰，給予其 Forward P/E 10-11 倍的估值，對應股價約為 **$27.50 - $30.00**。
*   **樂觀估計：** 如果轉型成功且盈利能力顯著改善，達到行業平均的低端，則股價可能達到 **$30.00 - $33.00**。

**最終綜合合理價位區間：$27.50 - $33.00**

目前股價 $24.49 略低於此區間的下限，表明存在一定的上漲空間，但幅度不大。投資者應密切關注 Juniper 收購案的進展、AI 伺服器業務的持續成長以及公司整體盈利能力的改善情況。
