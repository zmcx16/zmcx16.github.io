這是一份針對甲骨文公司（Oracle Corporation, 股票代碼：**ORCL**）的深度估值分析報告。

### 1. 市場背景與最新動態（網路搜尋補充）
在執行估值前，必須考慮 Oracle 最近的重大基本面轉變：
*   **AI 與雲端轉型成功：** Oracle 已從傳統資料庫軟體商轉型為雲端基礎設施（OCI）的主要參與者。2025 財年第一季財報顯示，其剩餘履約義務（RPO）大幅增長 53% 至 990 億美元，顯示 AI 需求極其強勁。
*   **戰略聯盟：** Oracle 近期與 AWS、Google Cloud 和 Microsoft Azure 達成了「多雲（Multi-cloud）」合作協議，這打破了以往的競爭壁壘，使其資料庫服務能直接在競爭對手的雲端運行，極大擴張了潛在市場。
*   **財務指引：** 管理層預計 2025 財年營收將實現兩位數增長，並上調了 2029 財年的營收目標至 1,040 億美元。

### 2. 估值模型計算

#### A. 折現現金流模型 (DCF)
這是衡量 Oracle 長期價值的核心模型。

**估值假設：**
*   **基準自由現金流 (FCF)：** 參考 TTM FCF 約為 113 億美元。
*   **預測期增長率 (Next 5Y)：** 考慮到 RPO 的爆發式增長與 AI 基礎設施需求，設定為 **15%**（高於歷史，符合目前雲端擴張期）。
*   **終端增長率 (Terminal Growth Rate)：** **3%**（符合全球經濟長期增長）。
*   **折現率 (WACC)：** 
    *   無風險利率 (10Y Treasury): 4.0%
    *   Beta: 1.0
    *   股權風險溢價: 5.0%
    *   **WACC 計算結果約為 8.5%**（考慮到 Oracle 較高的債務比例 Debt/Eq 4.21，加權平均資金成本會受債務成本影響）。

**計算過程簡述：**
1.  預測未來 5 年 FCF：$13.0B, $14.9B, $17.2B, $19.8B, $22.7B。
2.  計算終端價值 (TV)：$22.7B * (1.03) / (0.085 - 0.03) = $425.1B。
3.  折現至現值：PV of FCF + PV of TV ≈ $345B。
4.  調整淨債務後，得出股權價值。

*   **DCF 估值結果：約 $155 - $170 美元。**

#### B. 相對估值法 (Comparable Company Analysis)
參考同行業（Microsoft, SAP, Salesforce）的倍數。

*   **Forward P/E (預期本益比)：** 
    *   Oracle 目前 Forward P/E 為 **17.32x**。
    *   行業平均（雲端與軟體）約為 **25x - 30x**。
    *   若給予 Oracle 22x 的保守預期本益比（考慮其增長加速）：$6.20 (預估 EPS) * 22 = **$136.4**。
    *   若給予 28x（AI 溢價）：$6.20 * 28 = **$173.6**。
*   **PEG 比率：** 
    *   數據顯示 PEG 為 **0.81**。通常 PEG < 1 被視為嚴重低估。若 PEG 回升至 1.2 的合理水平，股價具備大幅上行空間。

*   **相對估值區間：$136 - $175 美元。**

---

#### C. 股息折現模型 (DDM)
Oracle 穩定配息，適用於評估其價值底線。

*   **當前年化股息：** $1.60
*   **股息增長率 (g)：** 過去 5 年平均約 10%，預計未來維持 **8%**。
*   **股權成本 (Ke)：** 9.0% (CAPM 模型)。
*   **計算：** $1.60 * (1.08) / (0.09 - 0.08) = $172.8。
*   *註：DDM 對增長率極其敏感，此結果顯示 Oracle 的股息政策支撐了其目前的股價水平。*

*   **DDM 估值結果：約 $160 - $175 美元。**

**總結：** Oracle 目前是一隻「具備高增長潛力的價值股」。建議在 $140 以下分批佈局，長期目標價看向 $180 以上。
