### **三、估值方法與計算**

#### **1. 加權平均資本成本 (WACC) 計算**

WACC 是 DCF 模型的核心折現率。

*   **A. 股權成本 (Cost of Equity, Ke) - 使用資本資產定價模型 (CAPM)**
    *   無風險利率 (Rf): 4.25% (美國 10 年期國債收益率)
    *   Beta (β): 1.05
    *   市場風險溢價 (MRP): 5.5%
    *   Ke = Rf + β * MRP
    *   Ke = 4.25% + 1.05 * 5.5% = 4.25% + 5.775% = **10.025%**

*   **B. 債務成本 (Cost of Debt, Kd)**
    *   Oracle 擁有投資級信用評級。假設其稅前債務成本約為 5.5%。
    *   有效稅率 (t): 18%
    *   稅後債務成本 = Kd * (1 - t) = 5.5% * (1 - 0.18) = 5.5% * 0.82 = **4.51%**

*   **C. 資本結構權重**
    *   股權市值 (E): $3850 億
    *   淨債務 (D): 總債務 - 現金 = $900 億 - $100 億 = $800 億
    *   總資本 (V): E + D = $3850 億 + $800 億 = $4650 億
    *   股權權重 (We): E / V = $3850 億 / $4650 億 = 0.828
    *   債務權重 (Wd): D / V = $800 億 / $4650 億 = 0.172

*   **D. WACC 計算**
    *   WACC = (We * Ke) + (Wd * 稅後債務成本)
    *   WACC = (0.828 * 10.025%) + (0.172 * 4.51%)
    *   WACC = 8.30% + 0.78% = **9.08%** (約 9.1%)

#### **2. 折現現金流模型 (Discounted Cash Flow, DCF)**

DCF 模型通過預測公司未來的自由現金流 (FCF) 並將其折現至現值來估計公司價值。

*   **估值假設：**

    *   **起始 FCF (TTM FY24):** $180 億

    *   **折現率 (WACC):** 9.1%

*   **計算過程：**

| 年度 | FCF 增長率 | 預測 FCF (億美元) | 折現因子 (1/(1+WACC)^n) | FCF 現值 (億美元) |
| :--- | :--------- | :---------------- | :----------------------- | :------------------ |
| TTM  | -          | 180.00            | -                        | -                   |
| Y1   | 10%        | 198.00            | 1 / (1.091)^1 = 0.9166   | 181.50              |
| Y2   | 10%        | 217.80            | 1 / (1.091)^2 = 0.8401   | 183.00              |
| Y3   | 10%        | 239.58            | 1 / (1.091)^3 = 0.7700   | 184.48              |
| Y4   | 8%         | 258.75            | 1 / (1.091)^4 = 0.7058   | 182.58              |
| Y5   | 8%         | 279.45            | 1 / (1.091)^5 = 0.6470   | 180.80              |

    *   **永續價值 (Terminal Value, TV) 計算 (在 Y5 結束時)：**
        *   FCF_Y6 = FCF_Y5 * (1 + g) = $279.45 億 * (1 + 0.03) = $287.83 億
        *   TV = FCF_Y6 / (WACC - g) = $287.83 億 / (0.091 - 0.03) = $287.83 億 / 0.061 = $4718.52 億
        *   TV 現值 = TV / (1 + WACC)^5 = $4718.52 億 / (1.091)^5 = $4718.52 億 * 0.6470 = $3052.00 億

        *   每股估值 = $3164.36 億 / 27.5 億股 = **$115.07**
