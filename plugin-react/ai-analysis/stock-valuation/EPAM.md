#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測公司的自由現金流 (Free Cash Flow, FCF) 並將其折現。

**DCF 假設：**
*   **預測期間 (Explicit Forecast Period):** 5 年 (2026 - 2030)。
*   **長期增長率 (Terminal Growth Rate, g):** 考慮到 IT 諮詢服務行業的長期增長潛力（全球 IT 專業服務市場預計 CAGR 為 9.5% 至 2030 年），以及 EPAM 作為成熟公司的規模，我們假設長期增長率為 2.5%。這略高於長期通脹率，反映了行業的持續創新。
*   **加權平均資本成本 (Weighted Average Cost of Capital, WACC):**
    *   **股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM)。
        *   `Ke = 無風險利率 + Beta * 市場風險溢酬`
        *   EPAM 的 Beta 值：由於未提供，我們將使用行業平均 Beta 或通過搜尋獲取。
        *   **搜尋 EPAM Beta 值：**
根據搜尋結果，EPAM 的 5 年 Beta 值為 1.77 或 1.72，也有 1 年 Beta 值為 1.29。考慮到 EPAM 股價波動性較高，我們採用 5 年 Beta 值 1.77。

        *   `Ke = 4.28% + 1.77 * 5.5% = 4.28% + 9.735% = 14.015%`
    *   **債務成本 (Cost of Debt, Kd):** 由於 EPAM 的負債權益比 (Debt/Eq) 僅為 0.04，長期負債權益比 (LT Debt/Eq) 為 0.03，債務佔比很小。我們可以使用公司債券收益率或估計。由於沒有直接提供債務利率，我們可以使用無風險利率加上一個信用利差。考慮到 EPAM 的財務狀況良好 (Quick Ratio 2.59, Current Ratio 2.59)，信用利差可以相對較低。我們假設債務成本為 5.5%。
    *   **債務市場價值 (Market Value of Debt, D):** 由於負債權益比很低，我們可以使用資產負債表上的長期債務。EPAM 的長期債務為 2580 萬美元 (截至 2024 年 3 月 31 日)。
    *   **股權市場價值 (Market Value of Equity, E):** 市值為 73.5 億美元。
    *   **WACC 計算：**
        *   `WACC = (E / (E + D)) * Ke + (D / (E + D)) * Kd * (1 - 稅率)`
        *   `E = 7,350,000,000`
        *   `D = 25,800,000` (假設長期債務為主要債務)

        *   `WACC = (7,350,000,000 / 7,375,800,000) * 14.015% + (25,800,000 / 7,375,800,000) * 5.5% * (1 - 0.24)`

        *   `WACC = 13.966% + 0.0035 * 4.18%`
        *   `WACC = 13.966% + 0.0146% = 13.98%` (約 14.0%)

        *   2024: -1.4% (公司指引)
        *   2025: 8.5% (分析師預期)
        *   2026: 7.0% (略低於分析師預期 7.35%，考慮到近期下調指引)
        *   2027: 6.0%
        *   2028: 4.0%
        *   2029: 3.0%
        *   2030: 2.5% (長期增長率)
    *   **營業利潤率 (Operating Margin):** EPAM 2024 年第一季度非 GAAP 營業利潤率為 14.9%。公司預計 2024 年非 GAAP 營業利潤率為 15-15.5%。我們假設營業利潤率在預測期內穩定在 15%。

    *   **資本支出 (Capital Expenditure, CapEx):** 歷史上，EPAM 的資本支出佔營收的比例相對較低。我們假設 CapEx 佔營收的 2%。

    *   **營運資本變動 (Change in Working Capital):** 假設營運資本變動佔營收的 0.5%。

**DCF 計算步驟：**

1.  **計算預測期內的營收 (Revenue)。**
2.  **計算營業利潤 (EBIT) = Revenue * Operating Margin。**
3.  **計算稅後營業利潤 (NOPAT) = EBIT * (1 - Tax Rate)。**
4.  **計算自由現金流 (FCF) = NOPAT + D&A - CapEx - Change in Working Capital。**
5.  **計算終值 (Terminal Value, TV) = FCF (最後一年) * (1 + g) / (WACC - g)。**
6.  **將預測期內的 FCF 和 TV 折現至現值 (Present Value)。**
7.  **計算股權價值 (Equity Value) = 所有 FCF 和 TV 的現值 - 淨債務。**
8.  **計算每股價值 (Intrinsic Value per Share) = 股權價值 / 流通股數。**

### 3. 綜合結論

儘管 DCF 估值較低，但相對估值和分析師的共識目標價均顯示 EPAM 存在一定的上漲潛力。當前股價 135.67 美元位於分析師目標價區間的下限附近。
