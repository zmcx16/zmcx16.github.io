#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型旨在估計公司未來自由現金流的現值。

**估值假設：**

        *   無風險利率 (Risk-Free Rate): 參考美國10年期國債收益率，假設為4.5%。
        *   市場風險溢價 (Market Risk Premium): 假設為5.5%。

    *   **稅後債務成本**: Kd * (1 - Tax Rate) = 6.0% * (1 - 0.21) = 6.0% * 0.79 = 4.74%。
    *   **資本結構**: 股權佔比 = 1 / (1 + Debt/Eq) = 1 / (1 + 0.04) = 1 / 1.04 ≈ 0.9615 (96.15%)。債務佔比 = 0.04 / (1 + 0.04) ≈ 0.0385 (3.85%)。
