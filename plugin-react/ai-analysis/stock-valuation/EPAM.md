### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間:** 5年 (2026-2030)
*   **折現率 (WACC):** 加權平均資本成本 (WACC)
*   **長期增長率 (Terminal Growth Rate):** 2.5% (略高於長期通脹和全球經濟增長率，考慮到IT服務行業的持續需求)

**WACC 計算：**

WACC = (E/V) \* Re + (D/V) \* Rd \* (1 - Tc)
其中：
*   Re = 股權成本 (Cost of Equity)
*   Rd = 債務成本 (Cost of Debt)
*   E = 股權市值 (Market Value of Equity)
*   D = 債務市值 (Market Value of Debt)
*   V = E + D (公司總價值)
*   Tc = 企業稅率 (Corporate Tax Rate)

**a. 股權成本 (Re) - 使用資本資產定價模型 (CAPM)：**

Re = Rf + β \* (Rm - Rf)
其中：
*   Rf = 無風險利率 (Risk-free Rate)
*   β = Beta 值
*   (Rm - Rf) = 市場風險溢價 (Market Risk Premium)

1.  **無風險利率 (Rf):** 參考美國10年期國債收益率。假設為 4.2%。
2.  **Beta (β):** 提供的數據中沒有Beta值。我將查詢 EPAM 的Beta值。

*   **Beta (β):** 根據 GuruFocus，EPAM 的 Beta 值為 1.46。Stock Analysis 報導為 1.72。Alpha Spread 則指出其WACC為8.18%，其中股權成本為8.18%，債務成本為10.43%，債務權重為0.22%。我將採用 GuruFocus 的 Beta 值 1.46，因為它在近期報導中出現。
*   **市場風險溢價 (Rm - Rf):** 歷史平均約為 5-6%。我將採用 5.5%。

Re = 4.2% + 1.46 \* 5.5% = 4.2% + 8.03% = 12.23%

**b. 債務成本 (Rd)：**

1.  **總債務 (Total Debt):** 根據 Trading Economics，截至2025年9月，EPAM 的債務為25.04百萬美元。Companies Market Cap 報導為0.15億美元 (1.5億美元)。Simply Wall St 報導為25.0百萬美元。Investing.com 報導總債務為1.5522億美元。我將採用 Investing.com 的最新數據：1.5522億美元。
2.  **利息費用 (Interest Expense):** 根據 Trading Economics，截至2025年9月，利息費用為13.96百萬美元。Investing.com 顯示利息費用為 "-"，但有淨利息費用。我將使用 Trading Economics 的數據。
3.  **企業稅率 (Tc):** 美國聯邦企業所得稅率為21%。

Rd = 利息費用 / 總債務 = 13.96百萬美元 / 155.22百萬美元 = 0.0899 = 8.99%

**c. 資本結構權重：**

*   **股權市值 (E):** 98億美元 (來自提供數據)。根據2026年2月的報導，市值約為98.8億美元。我將使用98億美元。
*   **債務市值 (D):** 1.5522億美元。

V = E + D = 98億美元 + 1.5522億美元 = 99.5522億美元
E/V = 98億美元 / 99.5522億美元 = 0.9844
D/V = 1.5522億美元 / 99.5522億美元 = 0.0156

**d. WACC 計算：**

WACC = (0.9844 \* 12.23%) + (0.0156 \* 8.99% \* (1 - 21%))
WACC = 12.04% + (0.0156 \* 8.99% \* 0.79)
WACC = 12.04% + 0.11% = 12.15%

Alpha Spread 報導 EPAM 的 WACC 為 8.18%。Stock Analysis 報導為 13.49%。我的計算結果 12.15% 介於兩者之間。我將使用我的計算結果 12.15%。

**自由現金流 (FCF) 預測：**

我將從2025年的自由現金流開始預測。
*   2024年年度自由現金流為5.27億美元。
*   截至2025年9月30日的過去12個月，自由現金流為4.59114億美元。
*   EPAM 的自由現金流增長率在過去幾年波動較大。過去12個月下降19.99%。過去3年複合年增長率為0.98%，過去5年為2.17%，過去10年為17.05%。

考慮到IT諮詢行業的增長趨勢 (2026年預計增長7%) 以及 EPAM 在AI領域的投入，我將假設一個保守的短期增長率，然後逐漸趨於長期增長率。

*   **2025 FCF (基期):** 採用截至2025年9月30日的TTM FCF: 4.59億美元。
*   **預測期增長率:**
    *   2026年: 5% (考慮到行業復甦和AI驅動的需求)
    *   2027年: 4%
    *   2028年: 3%
    *   2029年: 2.5%
    *   2030年: 2.5% (進入永續增長階段)

| 年度 | 2025 (基期) | 2026 | 2027 | 2028 | 2029 | 2030 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| FCF (百萬美元) | 459 | 482 | 501 | 516 | 529 | 542 |
| 增長率 | - | 5% | 4% | 3% | 2.5% | 2.5% |

**終值 (Terminal Value) 計算：**

終值 (TV) = (最後一年FCF \* (1 + 長期增長率)) / (WACC - 長期增長率)
TV = (542百萬美元 \* (1 + 2.5%)) / (12.15% - 2.5%)
TV = (542百萬美元 \* 1.025) / 0.0965
TV = 555.55百萬美元 / 0.0965 = 5757百萬美元

**現值 (Present Value) 計算：**

| 年度 | FCF (百萬美元) | 折現因子 (1 / (1 + WACC)^n) | FCF 現值 (百萬美元) |
| :--- | :--- | :--- | :--- |
| 2026 | 482 | 1 / (1 + 0.1215)^1 = 0.8917 | 430 |
| 2027 | 501 | 1 / (1 + 0.1215)^2 = 0.7951 | 398 |
| 2028 | 516 | 1 / (1 + 0.1215)^3 = 0.7090 | 366 |
| 2029 | 529 | 1 / (1 + 0.1215)^4 = 0.6322 | 334 |
| 2030 | 542 | 1 / (1 + 0.1215)^5 = 0.5637 | 305 |
| 終值 | 5757 | 1 / (1 + 0.1215)^5 = 0.5637 | 3246 |

**總企業價值 (Total Enterprise Value) = 預測期FCF現值總和 + 終值現值**
總企業價值 = (430 + 398 + 366 + 334 + 305) + 3246 = 1833 + 3246 = 5079百萬美元 = 50.79億美元

**股權價值 (Equity Value) = 總企業價值 + 現金及約當現金 - 總債務**

*   **現金及約當現金:** 根據 Trading Economics，截至2025年9月，為1.24億美元。
*   **總債務:** 1.5522億美元。

股權價值 = 50.79億美元 + 1.24億美元 - 1.5522億美元 = 50.4778億美元

**每股估值 (Intrinsic Value Per Share) = 股權價值 / 流通股數**

*   **流通股數:** 根據 Stock Analysis，EPAM Systems 有55.24百萬股流通股。

每股估值 = 50.4778億美元 / 55.24百萬股 = 91.38美元

**DCF 模型估值結果：**

*   **每股估值:** 91.38美元
*   **合理價位區間:** 80 - 100美元 (考慮到假設的敏感性)

這個結果遠低於當前股價177.47美元，表明根據DCF模型，EPAM可能被高估。
