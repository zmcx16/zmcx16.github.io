### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設:**

*   **預測期間:** 5 年 (2026 年 - 2030 年)
*   **折現率 (WACC):**
    *   **無風險利率 (Risk-Free Rate):** 採用美國 10 年期國債收益率。假設為 4.2%。
    *   **市場風險溢價 (Market Risk Premium, MRP):** 歷史平均約 5.5%。
    *   **Beta:** OKE 的 Beta 值為 0.81。
    *   **股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM): Ke = 無風險利率 + Beta \* MRP
        Ke = 4.2% + 0.81 \* 5.5% = 4.2% + 4.455% = 8.655%
    *   **債務成本 (Cost of Debt, Kd):** 考慮 OKE 的 BBB 信用評級，假設稅前債務成本為 5.5%。
    *   **稅率 (Tax Rate):** 採用美國企業稅率 21%。
    *   **債務權重 (Weight of Debt, Wd) 與股權權重 (Weight of Equity, We):**
        *   債務權益比 (Debt/Eq) = 1.53
        *   股權權重 We = 1 / (1 + 1.53) = 1 / 2.53 = 0.395
        *   債務權重 Wd = 1.53 / (1 + 1.53) = 1.53 / 2.53 = 0.605
    *   **WACC = (We \* Ke) + (Wd \* Kd \* (1 - Tax Rate))**
        WACC = (0.395 \* 8.655%) + (0.605 \* 5.5% \* (1 - 0.21))
        WACC = 3.429% + (0.605 \* 5.5% \* 0.79)
        WACC = 3.429% + 2.63% = 6.059% (約 6.1%)
*   **收入增長率 (Revenue Growth Rate):**
    *   2025 年 Q3 營收同比增長 71.9%，但這可能包含收購影響。
    *   2026 年預計營收增長 35.6% (Q4 2026 預期)。
    *   考慮到中游產業的穩定性以及 ONEOK 的收購和費用型業務，我們假設：
        *   2026 年: 15% (考慮到收購和初期整合效益)
        *   2027 年: 8%
        *   2028-2030 年: 5%
*   **永續增長率 (Terminal Growth Rate, g):** 2.5% (略高於長期通脹率，反映穩定增長)
*   **營業利潤率 (Operating Margin):** 採用最新數據 18.68%。
*   **稅率:** 21%。
*   **資本支出 (Capital Expenditure, CapEx):** 假設為收入的 5%。
*   **折舊與攤銷 (Depreciation & Amortization, D&A):** 假設為收入的 3%。
*   **營運資本變動 (Change in Working Capital):** 假設為收入的 1%。

**DCF 計算步驟 (簡化):**

1.  **預測收入 (Revenue):**
    *   2025 年全年營收預計 $34.07 億美元 (Zacks Consensus Estimate)。
    *   2026: $34.07B \* (1 + 0.15) = $39.18B
    *   2027: $39.18B \* (1 + 0.08) = $42.31B
    *   2028: $42.31B \* (1 + 0.05) = $44.43B
    *   2029: $44.43B \* (1 + 0.05) = $46.65B
    *   2030: $46.65B \* (1 + 0.05) = $48.98B

2.  **預測營業利潤 (EBIT):** Revenue \* Operating Margin
    *   2026: $39.18B \* 0.1868 = $7.32B
    *   2027: $42.31B \* 0.1868 = $7.90B
    *   2028: $44.43B \* 0.1868 = $8.30B
    *   2029: $46.65B \* 0.1868 = $8.72B
    *   2030: $48.98B \* 0.1868 = $9.15B

3.  **預測稅後營業利潤 (NOPAT):** EBIT \* (1 - Tax Rate)
    *   2026: $7.32B \* (1 - 0.21) = $5.78B
    *   2027: $7.90B \* (1 - 0.21) = $6.24B
    *   2028: $8.30B \* (1 - 0.21) = $6.56B
    *   2029: $8.72B \* (1 - 0.21) = $6.89B
    *   2030: $9.15B \* (1 - 0.21) = $7.23B

4.  **預測自由現金流 (Free Cash Flow, FCF):** NOPAT + D&A - CapEx - Change in Working Capital
    *   D&A: Revenue \* 0.03
    *   CapEx: Revenue \* 0.05
    *   Change in Working Capital: Revenue \* 0.01
    *   FCF = NOPAT + (Revenue \* 0.03) - (Revenue \* 0.05) - (Revenue \* 0.01)
    *   FCF = NOPAT - (Revenue \* 0.03)
    *   2026: $5.78B - ($39.18B \* 0.03) = $5.78B - $1.175B = $4.605B
    *   2027: $6.24B - ($42.31B \* 0.03) = $6.24B - $1.269B = $4.971B
    *   2028: $6.56B - ($44.43B \* 0.03) = $6.56B - $1.333B = $5.227B
    *   2029: $6.89B - ($46.65B \* 0.03) = $6.89B - $1.399B = $5.491B
    *   2030: $7.23B - ($48.98B \* 0.03) = $7.23B - $1.470B = $5.760B

5.  **計算終值 (Terminal Value, TV):** TV = FCF(2030) \* (1 + g) / (WACC - g)
    TV = $5.760B \* (1 + 0.025) / (0.061 - 0.025)
    TV = $5.760B \* 1.025 / 0.036
    TV = $5.904B / 0.036 = $164.0B

6.  **計算各期 FCF 和 TV 的現值 (Present Value):**
    *   PV(FCF 2026) = $4.605B / (1 + 0.061)^1 = $4.340B
    *   PV(FCF 2027) = $4.971B / (1 + 0.061)^2 = $4.417B
    *   PV(FCF 2028) = $5.227B / (1 + 0.061)^3 = $4.369B
    *   PV(FCF 2029) = $5.491B / (1 + 0.061)^4 = $4.321B
    *   PV(FCF 2030) = $5.760B / (1 + 0.061)^5 = $4.273B
    *   PV(TV) = $164.0B / (1 + 0.061)^5 = $121.7B

7.  **企業總價值 (Enterprise Value, EV):** 所有現值之和
    EV = $4.340B + $4.417B + $4.369B + $4.321B + $4.273B + $121.7B = $143.42B

8.  **股權價值 (Equity Value):** EV + 現金及約當現金 - 總債務
    *   現金及約當現金: 2023 年底有超過 $3.35 億美元現金。假設為 $0.5B。
    *   總債務: 2025 年 9 月 30 日，長期債務（包括流動負債）的賬面價值為 $33.7 億美元。
    *   Equity Value = $143.42B + $0.5B - $33.7B = $110.22B

9.  **每股內在價值 (Intrinsic Value per Share):** 股權價值 / 流通股數
    *   流通股數: 截至 2025 年 10 月 20 日為 629,231,557 股 (約 0.629B 股)。
    *   Intrinsic Value per Share = $110.22B / 0.629B = $175.23

**DCF 估值結果:**

*   **每股內在價值:** $175.23
*   **合理價位區間:** 考慮到假設的敏感性，若 WACC 變動 ±0.5%，永續增長率變動 ±0.5%，則合理價位區間約為 **$140 - $210**。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於股息穩定增長的公司。ONEOK 是一家支付穩定股息並持續提高股息的公司，因此適用此模型。

**估值假設:**

*   **最新年化股息 (D0):** 2025 年 1 月提高至每股 $3.96，2026 年 2 月 4 日新聞顯示再次提高 4%。
    *   D0 = $3.96 \* (1 + 0.04) = $4.1184
*   **股息增長率 (g):**
    *   中游產業歷史平均現金派發增長率為 5%。
    *   ONEOK 預計 2025 年 OCF per share 增長率為高個位數到低兩位數。
    *   假設短期 (未來 5 年) 股息增長率為 4.5% (略低於 OCF 增長預期，保持保守)。
    *   長期 (永續) 股息增長率為 2.5%。
*   **股權成本 (Ke):** 8.655% (與 DCF 模型相同)

**DDM 計算步驟 (兩階段增長模型):**

1.  **預測未來 5 年股息 (D1 - D5):**
    *   D1 (2026) = $4.1184 \* (1 + 0.045) = $4.3037
    *   D2 (2027) = $4.3037 \* (1 + 0.045) = $4.4973
    *   D3 (2028) = $4.4973 \* (1 + 0.045) = $4.7002
    *   D4 (2029) = $4.7002 \* (1 + 0.045) = $4.9117
    *   D5 (2030) = $4.9117 \* (1 + 0.045) = $5.1328

2.  **計算第 5 年末的終值 (Terminal Value of Dividends, TVD):**
    TVD = D5 \* (1 + g_long-term) / (Ke - g_long-term)
    TVD = $5.1328 \* (1 + 0.025) / (0.08655 - 0.025)
    TVD = $5.1328 \* 1.025 / 0.06155
    TVD = $5.2611 / 0.06155 = $85.48

3.  **計算各期股息和終值股息的現值:**
    *   PV(D1) = $4.3037 / (1 + 0.08655)^1 = $3.9610
    *   PV(D2) = $4.4973 / (1 + 0.08655)^2 = $3.8060
    *   PV(D3) = $4.7002 / (1 + 0.08655)^3 = $3.6560
    *   PV(D4) = $4.9117 / (1 + 0.08655)^4 = $3.5110
    *   PV(D5) = $5.1328 / (1 + 0.08655)^5 = $3.3700
    *   PV(TVD) = $85.48 / (1 + 0.08655)^5 = $56.30

4.  **每股內在價值 (Intrinsic Value per Share):** 所有現值之和
    Intrinsic Value per Share = $3.9610 + $3.8060 + $3.6560 + $3.5110 + $3.3700 + $56.30 = $74.60

**DDM 估值結果:**

*   **每股內在價值:** $74.60
*   **合理價位區間:** 考慮到股息增長率和股權成本的敏感性，合理價位區間約為 **$65 - $85**。

### 3. 綜合結論與比較

| 估值模型       | 每股估值結果 | 合理價位區間 |
| :------------- | :----------- | :----------- |
| DCF 模型       | $175.23      | $140 - $210  |
| 相對估值法 (CCA) | $88.97       | $80 - $100   |
| DDM 模型       | $74.60       | $65 - $85    |
| 分析師共識目標價 | $91.24       | $75 - $110   |

**比較與分析:**

*   **DCF 模型** 給出了最高的估值，這通常是因為 DCF 對增長率和永續增長率的假設較為敏感。ONEOK 近期有較大的收購活動，這可能導致短期內自由現金流的波動，使得 DCF 模型的預測更具挑戰性。此外，中游產業的資本密集型特性也可能影響 FCF 的穩定性。
*   **相對估值法 (CCA)** 的結果與分析師共識目標價較為接近，這表明市場對 OKE 的估值可能更傾向於與同業進行比較。$88.97 的平均值落在分析師目標價區間內。
*   **DDM 模型** 給出了最低的估值。雖然 OKE 是一家穩定的股息支付公司，但 DDM 模型對股息增長率的假設非常敏感。如果未來股息增長放緩，估值將會下降。

**綜合結論:**

DCF 模型由於其對未來現金流的長期預測，往往會產生較高的估值，尤其是在公司有較強增長預期或近期有重大收購的情況下。然而，其結果的可靠性高度依賴於輸入假設的準確性。

相對估值法和 DDM 模型則提供了更為保守且與市場預期更為一致的估值。分析師的共識目標價 $91.24 也支持了相對估值法的結果。

考慮到 OKE 穩定的中游業務模式、費用型收入佔比高、以及持續的股息增長，但同時也面臨產業挑戰和資本支出需求，我認為相對估值法和分析師共識目標價所反映的價值區間更具參考性。

**最終合理價位區間:**

綜合考量，我認為 OKE 的合理價位區間應落在 **$85 - $100** 之間。目前的股價 $80.34 位於此區間的下緣，顯示其可能略被低估或處於合理範圍內。
