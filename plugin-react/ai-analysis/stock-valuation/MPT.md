*   折現終值 = P2030 * 折現因子 (2030年) = 3.094 * 0.5487 = 1.698 美元

**DDM 估值結果：**

*   每股價值 = 折現股息總和 + 折現終值 = 1.273 + 1.698 = **2.971 美元**

**合理價位區間：** 根據 DDM 模型，MPT 的合理價位約為 **2.97 美元**。這個結果遠低於當前股價，反映了在低股息增長假設下，高股權成本對估值的影響。

#### **4. 其他合理估值方法 (P/B Ratio)**

由於 MPT 是一家 REIT，其資產主要是房地產，因此 P/B 比率是一個重要的估值指標。

**P/B 估值結果：**

如果 MPT 的 P/B 能夠回歸到行業中位數 1.5，則其股價潛在價值為：
*   估計股價 = 當前股價 * (目標 P/B / 當前 P/B) = 4.94 * (1.5 / 0.64) = **11.58 美元**
