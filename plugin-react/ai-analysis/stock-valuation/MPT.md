### 2. 估值方法與計算

由於 MPT 是一家 REIT，其盈利能力通常以 FFO (Funds From Operations) 或 NFFO (Normalized FFO) 來衡量，而非傳統的 GAAP EPS。因此，在估值中我將側重於這些指標。

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測未來的自由現金流 (Free Cash Flow, FCF) 並折現回現值。對於 REIT，通常會使用調整後的營運現金流或 AFFO 作為現金流的代理。然而，由於 MPT 的 FCF 數據較為波動且近期盈利為負，我將嘗試基於 NFFO 和預期增長來推導現金流。

**估值假設:**

*   **預測期間**: 5年 (2026-2030)
*   **長期增長率 (Terminal Growth Rate)**: 2.0% (考慮到醫療保健 REIT 行業的長期穩定增長趨勢和人口老齡化，但 MPT 自身面臨挑戰，因此採用保守增長率)
*   **折現率 (WACC - Weighted Average Cost of Capital)**:
    *   **股權成本 (Cost of Equity, Ke)**: 使用資本資產定價模型 (CAPM)
        *   無風險利率 (Risk-Free Rate): 4.5% (參考美國10年期國債收益率，假設為當前市場環境下的合理長期無風險利率)
        *   市場風險溢價 (Market Risk Premium): 5.5% (行業標準範圍)
        *   Beta: 1.2 (REITs 通常具有較高的 Beta，反映其對利率敏感性及特定風險)
        *   Ke = 無風險利率 + Beta * 市場風險溢價 = 4.5% + 1.2 * 5.5% = 4.5% + 6.6% = 11.1%
    *   **債務成本 (Cost of Debt, Kd)**: MPT 的利息覆蓋率較低 (1.2x)，債務股本比高 (2.13)，表明其債務成本可能較高。假設稅前債務成本為 7.0%。
    *   **稅率 (Tax Rate)**: REITs 通常享有特殊的稅務待遇，但為計算 WACC，我們需要一個有效的公司稅率。由於 MPT 近期有虧損，且 REITs 將大部分收入分配給股東以避免公司稅，我們將使用一個較低的有效稅率，例如 5% (僅用於 WACC 計算，不代表實際稅負)。
    *   **市值權重 (Market Value of Equity, E)**: $2.75 billion (用戶提供)
    *   **債務市值權重 (Market Value of Debt, D)**: $9.83 billion (參考總債務數據)
    *   **總資本 (V)** = E + D = $2.75B + $9.83B = $12.58B
    *   **WACC** = (E/V) * Ke + (D/V) * Kd * (1 - Tax Rate)
        *   WACC = (2.75 / 12.58) * 11.1% + (9.83 / 12.58) * 7.0% * (1 - 0.05)
        *   WACC = 0.2186 * 11.1% + 0.7814 * 7.0% * 0.95
        *   WACC = 2.426% + 5.209% = 7.635% (約 7.6%)

**現金流預測 (基於 NFFO 和 FCF):**

*   2025年 NFFO/股: $0.58
*   2025年 FCF/股: $0.33
*   考慮到 MPT 正在轉型，預計未來幾年 NFFO 和 FCF 將逐步改善。分析師預計 FFO/股到2030年將增長至0.83美元，5年複合年增長率為7%。

1.  **預測未來5年 FCF/股:**

    *   2026 FCF/股: $0.33 * (1 + 0.15) = $0.3795

    *   2028 FCF/股: $0.41745 * (1 + 0.07) = $0.44667

2.  **計算終值 (Terminal Value, TV):**

3.  **折現未來現金流和終值:**

    *   PV (2030 FCF) = $0.48307 / (1 + 0.076)^5 = $0.3400
    *   PV (TV) = $8.7988 / (1 + 0.076)^5 = $8.7988 / 1.4402 = $6.1094

4.  **DCF 估值結果:**
    *   總股權價值 (Total Equity Value) = 所有折現現金流之和 + 折現終值
    *   DCF 估值/股 = $0.3527 + $0.3610 + $0.3598 + $0.3547 + $0.3400 + $6.1094 = $7.8776
