Medical Properties Trust (MPT) 是一家專注於醫院設施的醫療房地產投資信託基金 (REIT)。公司於2026年2月19日公佈了2025年第四季度及全年財報。2025年第四季度，MPT 的淨收入為每股0.03美元，調整後營運資金 (NFFO) 為每股0.18美元。然而，2025年全年公司錄得每股淨虧損0.46美元，NFFO為每股0.58美元。

**加權平均資本成本 (WACC) 計算：**

    *   總資本 (V): E + D = 27.3億美元 + 97億美元 = 124.3億美元

5.  **WACC = (E/V) \* Ke + (D/V) \* Kd \* (1 - t)**
