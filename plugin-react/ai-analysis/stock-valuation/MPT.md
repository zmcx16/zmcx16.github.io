#### 2. 股息折現模型（Dividend Discount Model, DDM）

MPT在2026年2月宣布將季度股息提高至0.09美元，年化股息為0.36美元。Seeking Alpha文章指出，基於2026年預期FFO $0.63/股，前瞻FFO派息率約為57%，顯示股息可持續性有所改善。

**估值假設：**
*   **當前股息 (D0):** $0.36/股
*   **折現率 (Required Rate of Return, r):** 對於REITs，通常使用WACC。考慮到MPT的高槓桿和近期挑戰，其風險較高。我們將使用較高的折現率。
    *   無風險利率 (Risk-Free Rate): 參考美國10年期國債收益率，約4.2% (假設)
    *   股權風險溢價 (Equity Risk Premium): 5.5% (行業通用)
    *   Beta: 根據用戶提供數據，MPT的Beta未提供。REITs的Beta通常接近1。考慮其波動性，我們假設Beta為1.2。
    *   股權成本 (Cost of Equity, Ke) = 無風險利率 + Beta * 股權風險溢價 = 4.2% + 1.2 * 5.5% = 4.2% + 6.6% = 10.8%
    *   債務成本 (Cost of Debt, Kd): 考慮到其高槓桿，假設稅前債務成本為6.5%。
    *   稅率 (Tax Rate): REITs通常有特殊的稅務處理，但為簡化計算，我們假設有效稅率為0% (因為REITs通常將大部分利潤作為股息分配，以避免公司層面徵稅)。
    *   WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1-T)
        *   股東權益 (E) = 4.6B
        *   總債務 (D) = 9.7B
        *   E/(E+D) = 4.6 / (4.6 + 9.7) = 0.32
        *   D/(E+D) = 9.7 / (4.6 + 9.7) = 0.68
        *   WACC = 0.32 * 10.8% + 0.68 * 6.5% * (1-0%) = 3.456% + 4.42% = 7.876% (約7.9%)
*   **股息增長率 (g):** 考慮到MPT的轉型和行業前景，我們假設短期內股息增長較慢，長期穩定增長。
    *   **情境一 (零增長):** 假設股息不再增長，g = 0%。
        *   股價 = D0 / r = $0.36 / 0.079 = $4.56
    *   **情境二 (穩定增長):** 假設股息能以每年2%的穩定速度增長 (低於醫療保健行業的長期增長率，反映其高槓桿和轉型挑戰)。
        *   股價 = D1 / (r - g) = ($0.36 * (1 + 0.02)) / (0.079 - 0.02) = $0.3672 / 0.059 = $6.22

**股息折現模型結果：**
*   **零增長情境:** $4.56
*   **穩定增長情境 (2%):** $6.22

**合理價位區間 (DDM):** $4.50 - $6.50

#### 3. 折現現金流模型（Discounted Cash Flow, DCF）

DCF模型對於REITs通常使用FFO或AFFO作為現金流代理，而不是傳統的自由現金流 (FCF)，因為REITs的折舊攤銷很高，不完全是現金費用。然而，由於缺乏詳細的FFO/AFFO預測數據，且公司目前仍有淨虧損，直接預測未來幾年的FFO/AFFO並進行DCF估值會非常複雜且不確定性高。

鑑於MPT的負盈利和高槓桿，以及REITs的特殊性，我們將簡化DCF模型，專注於其NFFO的恢復和增長。

**估值假設：**
*   **預測期間:** 5年 (2026-2030)
*   **折現率 (WACC):** 7.9% (同DDM模型)
*   **NFFO增長率：**
    *   2026年: 預計NFFO為$0.63/股，相較於2025年的$0.58/股，增長率約為 (0.63-0.58)/0.58 = 8.6%。
    *   2027-2028年: 假設隨著租戶問題解決和資產組合優化，NFFO能以5%的速度增長。
    *   2029-2030年: 增長率放緩至3%。
*   **永續增長率 (Terminal Growth Rate, g):** 2% (與長期通脹和醫療保健REITs的長期增長潛力相符)
*   **流通股數:** 假設約6億股 (根據2025年財報中的加權平均股數)

**簡化DCF計算 (基於每股NFFO)：**

| 年份 | 預計每股NFFO | 折現因子 (7.9%) | 折現NFFO |
| :--- | :----------- | :--------------- | :------- |
| 2026 | $0.63         | 1 / (1.079)^1 = 0.9268 | $0.584   |
| 2027 | $0.63 * 1.05 = $0.6615 | 1 / (1.079)^2 = 0.8589 | $0.568   |
| 2028 | $0.6615 * 1.05 = $0.6946 | 1 / (1.079)^3 = 0.7960 | $0.553   |
| 2029 | $0.6946 * 1.03 = $0.7154 | 1 / (1.079)^4 = 0.7377 | $0.528   |
| 2030 | $0.7154 * 1.03 = $0.7370 | 1 / (1.079)^5 = 0.6837 | $0.504   |

**預測期內折現NFFO總和:** $0.584 + $0.568 + $0.553 + $0.528 + $0.504 = $2.737

**終值 (Terminal Value) 計算 (基於2030年NFFO):**
*   終值 NFFO = 2030年NFFO * (1 + g) / (r - g) = $0.7370 * (1 + 0.02) / (0.079 - 0.02) = $0.75174 / 0.059 = $12.74
*   終值折現到現在 = $12.74 / (1.079)^5 = $12.74 * 0.6837 = $8.70

**DCF估值 (每股):**
*   總估值 = 預測期內折現NFFO總和 + 終值折現到現在 = $2.737 + $8.70 = $11.437

**DCF估值結果：** $11.44

**合理價位區間 (DCF):** $10.00 - $13.00 (考慮到NFFO增長假設的敏感性)

**重要提示:** DCF模型對增長率和折現率的假設非常敏感。MPT目前處於轉型期，其未來NFFO的增長存在較高不確定性。此處的DCF估值結果應被視為一個較為樂觀的長期潛力評估。
