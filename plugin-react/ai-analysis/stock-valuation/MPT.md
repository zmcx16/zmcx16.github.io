*   D1 = $0.36

2.  **相對估值 (CCA)：** 相對估值結果顯示 MPT 在 P/B、P/S 和 P/FCF 等倍數上均顯著低於行業平均水平。這可能暗示市場認為 MPT 的資產質量、盈利能力或增長前景不如同行，因此給予較低的估值。然而，如果 MPT 能夠成功解決其租戶問題並改善營運，這些較低的倍數可能預示著潛在的上升空間。但 EV/EBITDA 倍數再次印證了其高額債務對股權價值的侵蝕。

3.  **股息折現模型 (DDM)：** DDM 模型給出的估值遠低於當前股價，約在 2.82 美元至 3.09 美元之間。這主要是因為 MPT 近期大幅削減股息，且其盈利能力和股息可持續性存在高度不確定性。對於依賴股息收入的 REIT 投資者而言，這是一個重要的警示信號。

**綜合結論：**

MPT 目前面臨嚴峻的財務挑戰，主要源於其主要租戶 Steward 的破產問題以及自身的高負債水平。

*   **風險與不確定性：** Steward 的破產對 MPT 的租金收入和資產價值產生了重大負面影響，導致巨額減值和淨虧損。儘管 MPT 正在積極出售資產以增加流動性並減少債務，但這些交易的執行情況和對未來盈利能力的影響仍需密切關注。高負債和較低的流動性比率也增加了公司的財務風險。
*   **估值差異：** DCF 和 DDM 模型均給出了較低的估值，甚至負值，這與 MPT 當前的股價和相對估值形成鮮明對比。這種差異主要源於市場對 MPT 未來現金流和股息可持續性的悲觀預期。相對估值雖然顯示 MPT 相對於同行可能被低估，但這是在不完全考慮其獨特風險的情況下。
*   **投資建議：** 鑑於 MPT 面臨的重大不確定性和財務挑戰，其股票具有較高的風險。儘管其股價可能因資產出售和潛在的營運改善而出現短期波動，但長期投資者應對其基本面轉型和債務解決方案保持高度警惕。在公司能夠證明其盈利能力和現金流可持續性之前，投資 MPT 存在較大風險。
