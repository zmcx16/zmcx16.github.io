#### **2.1. 折現現金流模型 (Discounted Cash Flow, DCF)**

對於 REIT，通常會使用 FFO 或 AFFO 作為現金流的基礎進行預測。由於 MPT 的盈利不穩定且為負值，直接預測自由現金流 (FCFF/FCFE) 較為困難。我將嘗試基於 NFFO 進行簡化估值，並考慮其資產出售和債務削減策略。

**估值假設：**

*   **預測期間：** 5年 (2026-2030)
*   **折現率 (WACC)：**
    *   **無風險利率 (Rf)：** 參考美國10年期國債收益率，假設為 4.5%。
    *   **市場風險溢價 (MRP)：** 假設為 5.5%。
    *   **Beta：** 根據您提供的數據，MPT 的 Beta 為 1.19 (來自 Simply Wall St 的類似數據)。
    *   **股權成本 (Ke) = Rf + Beta * MRP** = 4.5% + 1.19 * 5.5% = 4.5% + 6.545% = 11.045%
    *   **債務成本 (Kd)：** 考慮到 MPT 的高槓桿和近期債務到期，以及當前利率環境，假設債務成本為 6.5%。
    *   **稅率 (t)：** REIT 通常有特殊的稅務處理，但為了 WACC 計算，我們可以使用一個名義稅率，或者考慮其免稅性質。由於 REIT 將大部分收入分配給股東以避免公司層面徵稅，有效稅率可能較低。然而，在計算 WACC 時，我們需要考慮債務利息的稅盾效應。假設一個名義公司稅率為 21%。
    *   **債務權益比 (D/E)：** 2.13 (來自提供數據)。
    *   **WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1-t)**
        *   E/(E+D) = 1 / (1 + D/E) = 1 / (1 + 2.13) = 1 / 3.13 = 0.3195
        *   D/(E+D) = D/E / (1 + D/E) = 2.13 / 3.13 = 0.6805
        *   WACC = 0.3195 * 11.045% + 0.6805 * 6.5% * (1 - 0.21)
        *   WACC = 3.529% + 0.6805 * 6.5% * 0.79
        *   WACC = 3.529% + 3.496% = 7.025%
        *   **取 WACC 約為 7.0%**

*   **長期來看，如果 MPT 能夠成功解決其債務問題，多元化租戶，並恢復穩定的 FFO 增長，其資產的潛在價值 (如 P/B 估值所示) 可能會釋放。**

考慮到所有模型的結果以及 MPT 當前的風險與機遇，我認為 MPT 的合理估值區間可能在 **$4.50 - $7.00/股** 之間。

*   **$4.50 - $5.50：** 反映了當前市場對其營運現金流和短期風險的看法。
*   **$5.50 - $7.00：** 如果公司在去槓桿、租戶多元化和 FFO 增長方面取得顯著進展，則可能達到此區間。
