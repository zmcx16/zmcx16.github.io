### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

對於 REITs，通常使用調整後營運資金 (AFFO) 或 FFO 作為現金流的代理，並將其折現。由於我們有 NFFO 數據，我們將使用 NFFO 作為基礎。

**估值假設：**

*   **預測期間：** 5年 (2026-2030)
*   **折現率 (WACC)：** 我們將採用一個範圍。Alpha Spread 估計 WACC 為 6.95%，另一來源為 5.5%。考慮到 MPT 面臨的挑戰和較高的債務水平，我們將採用 7.5% 作為基準 WACC，並在敏感性分析中考慮 7.0% 和 8.0%。
*   **NFFO 增長率：**
    *   **短期 (2026-2028):** 考慮到 MPT 正在進行資產出售、租戶過渡以及Steward破產的影響，短期內 NFFO 增長可能波動。Q1 2024 NFFO 為 $0.24/股，而 Q4 2025 NFFO 為 $0.18/股，全年 NFFO 2025 為 $0.58/股。 2026年預計年化現金租金目標為10億美元，這可能帶來增長。 我們假設2026年 NFFO 略有恢復，然後逐步增長。
        *   2026年：預計NFFO為 $0.65/股 (考慮到Q4 2025的$0.18和全年$0.58，以及管理層對現金租金的預期)。
        *   2027-2028年：考慮到租戶穩定和資產出售後的再投資，我們假設 NFFO 每年增長 3%。
    *   **長期 (2029年及以後的終值增長率)：** 考慮到醫療保健 REIT 行業的長期穩定性，但 MPT 過去的挑戰，我們假設長期增長率為 1.5%。
*   **起始 NFFO：** 我們將使用2025年全年 NFFO $0.58/股作為基礎，並調整2026年的預期。

**計算步驟：**

1.  **預測 NFFO：**
    *   2025年 (實際): $0.58
    *   2026年 (預估): $0.65
    *   2027年 (預估): $0.65 * (1 + 0.03) = $0.67
    *   2028年 (預估): $0.67 * (1 + 0.03) = $0.69
    *   2029年 (預估): $0.69 * (1 + 0.03) = $0.71
    *   2030年 (預估): $0.71 * (1 + 0.03) = $0.73

2.  **計算終值 (Terminal Value, TV)：**
    *   TV = (2030年 NFFO * (1 + 長期增長率)) / (WACC - 長期增長率)
    *   TV = ($0.73 * (1 + 0.015)) / (0.075 - 0.015) = ($0.73 * 1.015) / 0.06 = $0.74095 / 0.06 = $12.35

3.  **折現 NFFO 和終值：**
    *   2026年 NFFO 折現值 = $0.65 / (1 + 0.075)^1 = $0.60
    *   2027年 NFFO 折現值 = $0.67 / (1 + 0.075)^2 = $0.58
    *   2028年 NFFO 折現值 = $0.69 / (1 + 0.075)^3 = $0.55
    *   2029年 NFFO 折現值 = $0.71 / (1 + 0.075)^4 = $0.53
    *   2030年 NFFO 折現值 = $0.73 / (1 + 0.075)^5 = $0.51
    *   終值折現值 = $12.35 / (1 + 0.075)^5 = $8.62

4.  **加總得到每股估值：**
    *   DCF 估值 = $0.60 + $0.58 + $0.55 + $0.53 + $0.51 + $8.62 = **$11.39**

**DCF 估值結果與合理價位區間：**

*   **DCF 估值 (基準 WACC 7.5%)：** $11.39
*   **合理價位區間 (敏感性分析)：**
    *   **WACC 7.0%, 長期增長率 1.5%:** TV = ($0.73 * 1.015) / (0.07 - 0.015) = $0.74095 / 0.055 = $13.47. DCF = $0.61 + $0.59 + $0.57 + $0.55 + $0.53 + $9.59 = **$12.84**
    *   **WACC 8.0%, 長期增長率 1.5%:** TV = ($0.73 * 1.015) / (0.08 - 0.015) = $0.74095 / 0.065 = $11.40. DCF = $0.60 + $0.57 + $0.54 + $0.51 + $0.48 + $7.76 = **$10.46**

因此，DCF 模型估值區間約為 **$10.46 - $12.84**。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

MPT 支付高額股息，因此 DDM 適用。然而，其股息增長率在過去三年為負 (-30.00%)，這使得單一增長率模型難以應用。我們將使用兩階段 DDM，假設短期內股息穩定或略有下降，長期恢復溫和增長。

**估值假設：**

*   **當前年化股息 (D0)：** $0.36/股 (基於最新季度股息 $0.09 * 4)
*   **要求回報率 (r)：** 我們將使用 WACC 的股權成本作為要求回報率。Alpha Spread 估計為 7.34%，另一來源為 6.65%。我們取 7.5% 作為基準。
*   **第一階段增長率 (g1)：** 考慮到 MPT 過去的股息削減和當前的挑戰，我們假設未來2年股息持平 (0%)。
*   **第二階段增長率 (g2，長期增長率)：** 假設 MPT 穩定後，股息能以 1.0% 的速度長期增長。

**計算步驟：**

1.  **預測未來股息：**
    *   D1 (2026年) = $0.36 * (1 + 0%) = $0.36
    *   D2 (2027年) = $0.36 * (1 + 0%) = $0.36

2.  **計算第二階段的終值 (Terminal Value)：**
    *   P2 (2027年末) = D3 / (r - g2)
    *   D3 (2028年) = D2 * (1 + g2) = $0.36 * (1 + 0.01) = $0.3636
    *   P2 = $0.3636 / (0.075 - 0.01) = $0.3636 / 0.065 = $5.59

3.  **折現未來股息和終值：**
    *   D1 折現值 = $0.36 / (1 + 0.075)^1 = $0.33
    *   D2 折現值 = $0.36 / (1 + 0.075)^2 = $0.31
    *   P2 折現值 = $5.59 / (1 + 0.075)^2 = $4.85

4.  **加總得到每股估值：**
    *   DDM 估值 = $0.33 + $0.31 + $4.85 = **$5.49**

**DDM 估值結果與合理價位區間：**

*   **DDM 估值 (基準要求回報率 7.5%)：** $5.49
*   **合理價位區間 (敏感性分析)：**
    *   **要求回報率 7.0%, 長期增長率 1.0%:** P2 = $0.3636 / (0.07 - 0.01) = $0.3636 / 0.06 = $6.06. DDM = $0.34 + $0.31 + $5.29 = **$5.94**
    *   **要求回報率 8.0%, 長期增長率 1.0%:** P2 = $0.3636 / (0.08 - 0.01) = $0.3636 / 0.07 = $5.19. DDM = $0.33 + $0.30 + $4.45 = **$5.08**

因此，DDM 模型估值區間約為 **$5.08 - $5.94**。

### 3. 綜合結論與比較

| 估值模型         | 估值結果 (基準假設) | 合理價位區間      |

**分析與比較：**

1.  **DCF 模型結果顯著高於其他模型和當前股價。** 這可能反映了 DCF 模型對未來 NFFO 增長和終值假設的敏感性。如果 MPT 能夠成功解決租戶問題，實現管理層預期的現金租金目標，並恢復穩定的 NFFO 增長，那麼其內在價值可能遠高於當前股價。然而，考慮到 MPT 過去的挑戰和不確定性，DCF 模型中的增長率和 WACC 假設可能需要更為保守。
2.  **相對估值和 DDM 模型結果較為接近，且與當前股價 ($4.53) 接近或略高。** 這兩個模型反映了市場對 MPT 當前盈利能力、股息可持續性以及行業平均水平的看法。MPT 的 P/FFO 和 P/B 倍數顯著低於同行，這可能表明市場已經將其面臨的風險計入股價。高股息收益率也可能反映了市場對其未來增長潛力的擔憂。
3.  **分析師目標價：** 分析師的共識目標價約為 $5.57 - $5.79，這與相對估值和 DDM 模型結果更為一致。GuruFocus 估計的公允價值為 $5.32，也落在這個範圍內。

**綜合結論：**

Medical Properties Trust (MPT) 目前正處於一個轉型期，面臨著租戶問題（尤其是 Steward 的破產）和高利率環境帶來的挑戰。這些因素導致其 GAAP 淨利潤為負，股息增長率下降，並在市場上表現不佳。

儘管 DCF 模型在樂觀假設下顯示出較高的潛在價值，但考慮到 MPT 營運的不確定性和風險，相對估值和股息折現模型可能更能反映市場對其當前狀況的看法。這些模型表明，MPT 的合理價位可能在 **$5.00 - $8.00** 之間。

當前股價 $4.53 略低於大多數分析師目標價和相對估值/DDM 模型的低端。這可能意味著市場已經充分反映了 MPT 的負面消息和風險。如果 MPT 能夠成功執行其資產出售策略，穩定租戶組合，並實現其年化現金租金目標，那麼其股價存在一定的上漲空間。

**投資建議：**

MPT 是一項高風險高回報的投資。對於尋求高股息收益且能承受較高風險的投資者，MPT 可能具有吸引力，但需要密切關注其租戶狀況、資產出售進度以及債務管理。在投資之前，應仔細評估其風險，並考慮其股息的可持續性。如果公司能夠成功度過當前難關，其估值有望向行業平均水平靠攏。
