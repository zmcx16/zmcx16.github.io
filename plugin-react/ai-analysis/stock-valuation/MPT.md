#### 2.1. WACC 計算

首先，我們需要獲取MPT的Beta值。

**WACC 計算所需數據：**

*   **無風險利率 (Rf)：** 4.206% (美國10年期國債收益率，截至2026年2月6日)
*   **Beta：** 1.44 (截至2026年2月6日)
*   **股權風險溢價 (ERP)：** 假設為5.5% (行業常用範圍)
*   **股本成本 (Ke)：**
    Ke = Rf + Beta * ERP
    Ke = 4.206% + 1.44 * 5.5% = 4.206% + 7.92% = 12.126%

*   **債務成本 (Kd)：**
    MPT的債務情況較為複雜。截至2025年9月，長期債務為90.485億美元。 2024年9月，MPT的5%無擔保票據 (2027年到期) 收益率為10.3%，4.625%無擔保票據 (2029年到期) 收益率為9.9%，3.5%無擔保票據 (2031年到期) 收益率約為70 (價格) / 9.9% (收益率)。 考慮到這些數據，我們將假設一個平均債務成本。
    我們將使用Simply Wall St提供的資訊，MPT的利息覆蓋率為1.2倍，這表明其利息支付覆蓋率較低。 考慮到當前利率環境和MPT的債務狀況，我們假設其稅前債務成本為8.0%。

*   **稅率 (Tax Rate)：** REITs通常將大部分應稅收入分配給股東，以避免公司層面的所得稅。因此，其有效稅率通常較低。GuruFocus計算的MPT TTM稅率為-5.96%，因此設為0%。 我們將使用0%的有效稅率進行WACC計算，因為REITs的特殊稅務結構。

*   **資本結構：**
    *   **股權市值 (E)：** 32.8億美元 (截至2026年2月11日)
    *   **債務市值 (D)：** 我們使用截至2025年9月的長期債務90.485億美元作為債務的近似值。 GuruFocus計算的2025年9月平均債務賬面價值為95.015億美元。 我們將使用95.015億美元作為債務價值。
    *   總資本 = E + D = 32.8億美元 + 95.015億美元 = 127.815億美元
    *   股權權重 (We) = E / (E + D) = 32.8 / 127.815 = 0.2566 (約25.66%)
    *   債務權重 (Wd) = D / (E + D) = 95.015 / 127.815 = 0.7434 (約74.34%)

*   **WACC 計算：**
    WACC = (We * Ke) + (Wd * Kd * (1 - Tax Rate))
    WACC = (0.2566 * 12.126%) + (0.7434 * 8.0% * (1 - 0%))
    WACC = 3.11% + 5.95% = 9.06%

    GuruFocus計算的MPT WACC為7%。 考慮到其債務成本和資本結構，我們將採用一個更保守的WACC，例如9.0% - 10.0%之間。為了估值，我們將使用**9.5%**作為WACC。

### 3. 估值模型

#### 3.1. 折現現金流模型 (Discounted Cash Flow, DCF)

對於REITs，自由現金流 (FCF) 的計算可能需要調整，因為折舊和攤銷通常是非現金費用，而資本支出 (CapEx) 對於維持和擴大房地產投資組合至關重要。我們將使用營運現金流 (Operating Cash Flow) 減去資本支出來估算FCF。

**DCF 假設：**

*   **預測期：** 5年 (2026-2030)
*   **WACC：** 9.5%
*   **長期增長率：** 2.0% (考慮到醫療保健行業的穩定增長和通脹)

**歷史數據 (百萬美元)：**

*   2024年營運現金流 (OCF)：2.4548億美元
*   2024年資本支出 (Acquisition of Real Estate Assets)：3.4148億美元 (注意：REIT的資本支出主要為房地產收購)
*   2025年Q3營運現金流 (TTM)：1.4739億美元
*   2025年Q3資本支出 (TTM Acquisition of Real Estate Assets)：1.775億美元

由於2024年營運現金流為正，但收購房地產資產的支出更高，導致自由現金流為負。MPT在2024財年淨虧損24億美元，2025年Q3淨虧損7773萬美元。 這種情況下，傳統的FCF估值可能不適用或需要大量調整。對於REITs，FFO和AFFO是更常用的現金流衡量指標。

然而，我們仍嘗試構建一個簡化的FCF模型。

**FCF 預測 (百萬美元)：**

*   **基期 (2025年TTM) 營運現金流：** 147.39百萬美元
*   **基期 (2025年TTM) 房地產收購：** 177.5百萬美元
*   **基期 FCF =** 147.39 - 177.5 = -30.11百萬美元

由於基期FCF為負，我們需要對未來的營運現金流和資本支出進行更樂觀的假設，或者承認DCF模型可能不適用於當前MPT的狀況。

MPT的目標是到2026年底實現超過10億美元的年化現金租金。 這表明其收入和現金流有望改善。

**簡化 FCF 預測假設：**

*   假設營運現金流在2026年開始恢復正增長，並在未來幾年逐步改善。
*   假設資本支出與營運現金流的比例會逐漸下降，以產生正的FCF。

這是一個高度假設性的預測，因為MPT近期面臨虧損和較高的資本支出。

**替代方法：基於FFO的DCF**

對於REITs，通常會使用FFO或AFFO來進行DCF估值。

**FFO 數據：**

*   2025年Q3 NFFO：0.13美元/股
*   2025年全年FFO預期：0.55美元/股
*   流通股數：6.015億股 (截至2025年11月4日)

**預測 FFO (百萬美元)：**

*   2025年FFO = 0.55美元/股 * 601.5百萬股 = 330.825百萬美元

**FFO 增長率假設：**

*   考慮到公司目標2026年現金租金超過10億美元，以及行業的穩定性，我們假設FFO在預測期內有適度增長。
*   2026年：5%
*   2027年：4%
*   2028年：3%
*   2029年：2%
*   2030年：2%

**FFO 預測：**

*   2025 (基期)：330.825
*   2026：330.825 * 1.05 = 347.366
*   2027：347.366 * 1.04 = 361.261
*   2028：361.261 * 1.03 = 372.099
*   2029：372.099 * 1.02 = 379.541
*   2030：379.541 * 1.02 = 387.132

**終值 (Terminal Value) 計算：**

終值 (TV) = (最後一年FFO * (1 + 長期增長率)) / (WACC - 長期增長率)
TV = (387.132 * (1 + 0.02)) / (0.095 - 0.02)
TV = (387.132 * 1.02) / 0.075
TV = 394.875 / 0.075 = 5265百萬美元 (在2030年末)

**股權價值 (Equity Value) =** 企業價值 + 現金及約當現金 - 總債務
*   現金及約當現金 (2024財年)：3.323億美元
*   總債務 (2024財年)：88億美元 (使用長期債務作為近似值)

這個結果顯示股權價值為負，這表明基於FFO的DCF模型在當前假設下，MPT的內在價值可能低於其債務。這可能是由於其高負債和近期盈利能力不佳所致。

**DCF模型結果：** 由於FFO-DCF模型產生負的股權價值，這表明在當前假設下，該模型認為MPT的內在價值無法覆蓋其債務。這可能反映了市場對其高負債和盈利壓力的擔憂。

#### 3.3. 股息折現模型 (Dividend Discount Model, DDM)

MPT支付股息，但其股息在2024財年大幅下降，且近期有所波動。 2024財年股息為0.46美元/股，較前一年下降47.7%。 最新季度股息為0.09美元/股，年化為0.36美元/股。 股息增長率為-12.82%。

由於股息不穩定且近期呈下降趨勢，戈登增長模型 (Gordon Growth Model) 可能不適用，因為它假設股息穩定增長。我們將使用一個兩階段DDM模型，假設短期內股息穩定或略有下降，然後進入長期穩定增長。

**DDM 假設：**

*   **股本成本 (Ke)：** 12.126%
*   **當前年化股息 (D0)：** 0.36美元/股 (基於最新季度股息0.09美元/股)
*   **短期股息增長率 (g1)：** 假設未來2年股息持平 (0%)，反映其穩定性努力。
*   **長期股息增長率 (g2)：** 2.0% (與DCF長期增長率一致)
*   **預測期：** 2年穩定，之後進入永續增長。

**股息預測：**

*   D1 (2026) = 0.36美元
*   D2 (2027) = 0.36美元

**終值 (Terminal Value) 計算 (在2027年末)：**

TV = D2 * (1 + g2) / (Ke - g2)
TV = 0.36 * (1 + 0.02) / (0.12126 - 0.02)
TV = 0.36 * 1.02 / 0.10126
TV = 0.3672 / 0.10126 = 3.626美元

**折現股息和終值：**

| 年份 | 股息 (美元/股) | 折現因子 (1/(1+Ke)^n) | 折現股息 (美元/股) |
| :--- | :------------- | :--------------------- | :------------------- |
| 2026 | 0.36           | 1/(1.12126)^1 = 0.8918 | 0.321                |

### 4. 綜合結論與合理價位區間

我們應用了三種估值方法，並得到了不同的結果：

*   **DCF模型 (基於FFO)：** 產生負的股權價值，表明在當前假設下，公司內在價值無法覆蓋其債務。這是一個強烈的警示信號，反映了MPT高負債和近期盈利能力不佳的風險。
*   **相對估值 (P/FFO)：** 根據行業平均P/FFO，MPT的估值可能在8.25美元/股至11.00美元/股之間，遠高於當前股價。然而，MPT的自身倍數遠低於行業平均，這表明市場對其風險有較高的折價。
*   **DDM模型：** 3.49美元/股，低於當前股價，暗示可能被高估。

**綜合分析：**

DCF模型和DDM模型都指向MPT的估值可能存在挑戰，甚至可能被高估（DDM）或其內在價值無法覆蓋債務（DCF）。相對估值雖然給出了較高的潛在價格，但這是基於與財務狀況更健康的同行進行比較，而MPT目前面臨獨特的挑戰。

MPT近期面臨的主要問題包括：

*   **高負債：** 債務股權比率高達206.3%。
*   **盈利能力下降：** 2024財年淨虧損24億美元，2025年Q3繼續虧損。
*   **股息削減：** 2024財年股息大幅下降，儘管近期有所回升，但仍低於歷史水平。
*   **租戶風險：** 與主要租戶Steward Health的破產問題對公司造成了重大影響，儘管已達成和解，但仍需時間恢復。

儘管公司預計2026年底現金租金將超過10億美元，並採取了股票回購計劃，但市場顯然對其轉型和去槓桿化的能力持謹慎態度。

**合理價位區間：**

考慮到DCF和DDM模型的保守結果，以及相對估值中MPT自身較低的倍數，我們認為MPT的合理價位區間應更為保守。

*   **下限：** 考慮到DDM模型結果和當前市場對其風險的定價，約為 **3.50美元/股**。
*   **上限：** 考慮到其未來現金租金增長潛力以及如果成功去槓桿化和改善盈利，其倍數可能向行業低端靠攏，約為 **7.00美元/股** (接近分析師平均目標價的低端)。

**綜合結論：**

Medical Properties Trust (MPT) 目前正處於一個轉型期，面臨高負債和盈利壓力。儘管其在醫療保健REITs行業擁有長期增長潛力，但短期內其財務表現和市場情緒受到挑戰。

基於本次分析，MPT的內在價值存在較大不確定性。DCF模型顯示其股權價值可能為負，而DDM模型給出的估值低於當前股價。相對估值雖然在理論上給出了較高潛在價格，但這需要MPT的財務狀況顯著改善才能實現。

因此，對於MPT的投資者而言，風險較高。其當前股價5.46美元處於我們估計的合理區間內，但更傾向於區間的上限。投資者應密切關注公司去槓桿化進程、租戶表現以及現金流改善情況。如果公司能夠成功執行其戰略並恢復穩定的盈利和現金流，其估值有望提升。
