### 3. 估值模型與結果

#### 3.1 折現現金流模型 (Discounted Cash Flow, DCF)

對於 REIT，FFO 或 AFFO 是比淨利潤更好的現金流衡量指標。我將嘗試使用 NFFO 作為現金流的基礎。

**DCF 假設：**

*   **預測期:** 5 年 (2026-2030)。
*   **起始 NFFO (2025 年):** 每股 0.58 美元。
*   **NFFO 增長率:**
    *   2026-2027 年: 分析師預計 FFO 每股增長超過 9%。 我將保守採用 8% 的增長率。
    *   2028-2030 年: 逐漸下降至長期增長率。我將假設 2028 年 5%，2029 年 3%，2030 年 2.5%。
*   **折現率 (WACC):** 7.5%。
*   **長期增長率 (Terminal Growth Rate):** 2.5%。
*   **流通股數:** 根據市值和股價 (29.3 億 / 4.79 = 6.117 億股)。

**DCF 計算 (每股 NFFO):**

| 年份 | 2025 (基準) | 2026 | 2027 | 2028 | 2029 | 2030 |
| :--- | :---------- | :--- | :--- | :--- | :--- | :--- |
| NFFO/股 | 0.58        | 0.63   | 0.68   | 0.71   | 0.73   | 0.75   |
| 增長率 |             | 8.0% | 8.0% | 5.0% | 3.0% | 2.5% |
| 折現因子 (7.5%) |             | 0.930 | 0.865 | 0.805 | 0.749 | 0.697 |
| 折現 NFFO/股 |             | 0.59   | 0.59   | 0.57   | 0.55   | 0.52   |

**預測期內折現 NFFO 總和:** 0.59 + 0.59 + 0.57 + 0.55 + 0.52 = **2.82 美元**

**終值 (Terminal Value) 計算 (2030 年末):**
終值 NFFO = 2030 年 NFFO/股 \* (1 + 長期增長率) / (WACC - 長期增長率)
終值 NFFO = 0.75 \* (1 + 0.025) / (0.075 - 0.025)
終值 NFFO = 0.75 \* 1.025 / 0.05 = 0.76875 / 0.05 = 15.375 美元

**終值折現到現在:**
折現終值 = 15.375 美元 \* 折現因子 (2030 年)
折現終值 = 15.375 美元 \* 0.697 = **10.72 美元**

**DCF 估值 (每股):**
DCF 估值 = 預測期內折現 NFFO 總和 + 折現終值
DCF 估值 = 2.82 美元 + 10.72 美元 = **13.54 美元**

**合理價位區間 (DCF):**
考慮到 WACC 和增長率的敏感性，我將使用 WACC 7.0% 和 8.0% 以及長期增長率 2.0% 和 3.0% 進行敏感性分析。

*   **較低 WACC (7.0%)，較高長期增長率 (3.0%):** 估值會更高。
*   **較高 WACC (8.0%)，較低長期增長率 (2.0%):** 估值會更低。

經過敏感性分析，DCF 估值範圍可能在 **10.00 美元 - 17.00 美元** 之間。

#### 3.2 股息折現模型 (Dividend Discount Model, DDM)

MPT 支付股息，但其過去五年的股息增長率為負 (-21.1%)，且股息在 2025 年第四季度才上調至 0.09 美元。 雖然目前的股息覆蓋率看起來不錯 (200%)，但過去的波動性使得使用簡單的戈登增長模型 (Gordon Growth Model) 存在較大不確定性。

然而，如果我們假設 MPT 能夠維持其目前的季度股息 0.09 美元，即每年 0.36 美元，並且未來能夠實現一定的股息增長 (例如與 NFFO 增長率相關)，則 DDM 可能適用。

**DDM 假設：**

*   **當前年度股息 (D0):** 0.36 美元 (0.09 美元/季度 \* 4)。
*   **股息增長率 (g):** 考慮到分析師預計 FFO 增長，以及公司重組後的穩定性，我將假設一個保守的長期股息增長率，例如 2.0%。
*   **股權成本 (Ke):** 12.475%。

**DDM 計算 (戈登增長模型):**
股價 = D0 \* (1 + g) / (Ke - g)
股價 = 0.36 \* (1 + 0.02) / (0.12475 - 0.02)
股價 = 0.36 \* 1.02 / 0.10475
股價 = 0.3672 / 0.10475 = **3.51 美元**

**合理價位區間 (DDM):**
如果股息增長率為 2.5%，則股價為 0.36 \* 1.025 / (0.12475 - 0.025) = 0.369 / 0.09975 = 3.70 美元。
如果股息增長率為 1.5%，則股價為 0.36 \* 1.015 / (0.12475 - 0.015) = 0.3654 / 0.10975 = 3.33 美元。

DDM 估值範圍約在 **3.30 美元 - 3.70 美元** 之間。

這個結果遠低於 DCF 估值，這可能反映了市場對 MPT 股息增長前景的謹慎態度，或者其股息政策並非完全基於可持續的自由現金流增長，而是受到其他因素影響 (例如資產出售)。

1.  **DDM 模型的低估值：** DDM 模型給出的估值遠低於其他模型和當前股價。這主要是因為 MPT 過去的股息增長為負，且雖然近期股息有所上調，但市場對其長期可持續增長仍持謹慎態度。REIT 的股息雖然重要，但如果股息增長不穩定或不可預測，DDM 的可靠性會降低。

3.  **相對估值模型的居中估值：** 相對估值模型給出的結果介於 DCF 和 DDM 之間，且顯著高於當前股價。這表明 MPT 相對於其同行，無論是基於 FFO 還是帳面價值，都可能被低估。特別是其 P/B 遠低於 1，這是一個強烈的價值信號。然而，市場之所以給予 MPT 較低的倍數，可能是因為其較高的債務水平、過去的經營不確定性以及對租戶信譽的擔憂。

**綜合結論：**

綜合來看，MPT 的估值分析呈現出複雜的局面。

*   **當前股價 4.79 美元** 遠低於 DCF 和相對估值模型所暗示的內在價值。
