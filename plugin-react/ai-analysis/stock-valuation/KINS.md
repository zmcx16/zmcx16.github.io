### 2. 估值模型計算

#### A. 折現現金流模型 (DCF)

對於保險公司，通常使用「股權自由現金流 (FCFE)」或基於盈餘的折現。這裡我們使用 P/FCF 數據推算的現金流進行保守估計。

*   **假設條件**：
    *   **基期 FCF**：根據市值 215.85M / P/FCF 2.95 ≈ **73.17M** (註：保險業現金流波動大，此處採保守調整後的 50M 作為預測起點)。
    *   **增長率 (Next 5Y)**：預期 EPS 增長 26%，但考慮到保險週期的穩定性，前 5 年設為 **10%**，之後進入永續增長。
    *   **折現率 (WACC/COE)**：考慮到小市值溢價與行業風險，設定為 **11.5%**。
    *   **終端增長率 (g)**：**2.5%** (符合長期通膨與經濟增長)。

*   **計算過程**：
    1.  Year 1-5 FCF: 55M, 60.5M, 66.55M, 73.2M, 80.5M
    2.  終端價值 (TV) = 80.5 * (1.025) / (0.115 - 0.025) ≈ 914.5M
    3.  折現現值 (PV) ≈ 43.5 + 43.2 + 42.8 + 42.4 + 42.0 + 478.5 (TV折現) = **692.4M**
*   **DCF 估值結果**：約 **$48.00** (顯著高於現價，反映若能維持目前現金流，價值極度低估)。
    *   *注意：保險業 DCF 敏感度極高，通常作為參考上限。*

#### B. 相對估值法 (Comparable Company Analysis)

選取同類型小市值 P&C 保險公司（如 UVE, HRTG, HCI）的平均倍數。

*   **P/E 估值**：
    *   行業平均 P/E：約 10x - 12x。
    *   KINS Forward P/E：5.14x。
    *   合理價 = Forward EPS (約 2.93) * 10x = **$29.30**。
*   **P/B 估值**：
    *   KINS P/B 為 1.75x，高於行業平均（約 1.2x-1.4x），這歸功於其極高的 ROE (43%)。
    *   根據 Gordon Growth Model (P/B = (ROE - g) / (COE - g))：
        合理 P/B = (0.43 - 0.025) / (0.115 - 0.025) ≈ 4.5x。
    *   目前 1.75x 顯然未完全反映其盈利能力。

#### C. 股息折現模型 (DDM)

*   **數據**：現價 15.08，殖利率 1.01%，年度股息約 $0.15。
*   **假設**：股息增長率 5%，折現率 11.5%。
*   **計算**：0.15 * (1.05) / (0.115 - 0.05) = **$2.42**。
*   **結論**：DDM 不適用於 KINS，因為該公司目前優先將盈餘用於增長與強化資本結構，而非發放高額股息。
