*   **基於行業平均 P/S (1.76):**

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

Halliburton 支付股息，因此 DDM 適用。

**估值假設:**

*   **當前股息 (Current Dividend):** 2024 年第一季度股息為每股 0.17 美元。因此，年化股息為 0.17 * 4 = 0.68 美元。
*   **股息增長率 (Dividend Growth Rate, g):**
    *   公司將 2024 年第一季度股息提高了 0.01 美元。
    *   考慮到公司強調自由現金流和回報股東，但行業增長相對溫和，我們假設一個保守的長期股息增長率。
    *   我們將使用兩階段 DDM。
    *   **第一階段 (5 年):** 假設股息每年增長 3.0% (略低於行業增長，但考慮公司穩定性)。
    *   **第二階段 (永續增長):** 假設永續股息增長率為 2.0%。
*   **所需報酬率 (Required Rate of Return, r):** 我們將使用之前計算的 WACC 作為股權的所需報酬率，即 8.45%。

**DDM 計算步驟:**

1.  **預測未來 5 年股息:**
    *   D0 = 0.68 美元
    *   D1 (2024) = 0.68 * (1 + 0.03) = 0.7004 美元
    *   D2 (2025) = 0.7004 * (1 + 0.03) = 0.7214 美元
    *   D3 (2026) = 0.7214 * (1 + 0.03) = 0.7430 美元
    *   D4 (2027) = 0.7430 * (1 + 0.03) = 0.7653 美元
    *   D5 (2028) = 0.7653 * (1 + 0.03) = 0.7882 美元

2.  **計算第 5 年末的終端股息 (D6):**
    *   D6 = D5 * (1 + 永續增長率) = 0.7882 * (1 + 0.02) = 0.8039 美元

3.  **計算第 5 年末的終端價值 (Terminal Value, TV):**
    *   TV = D6 / (r - g_永續) = 0.8039 / (0.0845 - 0.02) = 0.8039 / 0.0645 = 12.46 美元

4.  **計算各期股息和終端價值的現值:**
    *   PV (D1) = 0.7004 / (1 + 0.0845)^1 = 0.6458 美元
    *   PV (D2) = 0.7214 / (1 + 0.0845)^2 = 0.6130 美元
    *   PV (D3) = 0.7430 / (1 + 0.0845)^3 = 0.5819 美元
    *   PV (D4) = 0.7653 / (1 + 0.0845)^4 = 0.5525 美元
    *   PV (D5) = 0.7882 / (1 + 0.0845)^5 = 0.5247 美元
    *   PV (TV) = 12.46 / (1 + 0.0845)^5 = 8.38 美元

5.  **計算每股估值:**
    *   每股估值 = 所有股息現值之和 + 終端價值現值
    *   每股估值 = 0.6458 + 0.6130 + 0.5819 + 0.5525 + 0.5247 + 8.38 = 11.30 美元

**DDM 估值結果:**
*   **每股估值:** 11.30 美元
*   **合理價位區間:** 10.00 - 13.00 美元 (DDM 對增長率和所需報酬率非常敏感，且 Halliburton 的股息增長相對溫和，導致估值較低)
