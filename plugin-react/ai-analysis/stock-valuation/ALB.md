### 一、公司簡介與行業概覽

Albemarle Corporation (ALB) 是一家全球領先的特種化學品公司，主要業務分為三個部門：
1.  **鋰 (Lithium)：** 主要為電動車電池、便攜式電子產品和其他應用提供鋰化合物。這是公司主要的增長驅動力。
2.  **溴 (Bromine)：** 主要用於阻燃劑、石油鑽井液和農業化學品。
3.  **催化劑 (Catalysts)：** 為煉油和聚合物生產提供催化劑解決方案。

ALB 在全球鋰市場中佔據重要地位，其估值高度依賴於鋰的未來供需、價格走勢以及公司擴張項目的執行情況。近期鋰價的波動對公司盈利和股價造成了顯著影響。

### 二、估值假設

在進行估值之前，我們需要設定一系列關鍵假設。這些假設將直接影響估值結果。

#### 1. 折現率 (WACC - Weighted Average Cost of Capital) 計算

WACC 是 DCF 模型中的關鍵折現率。

**A. 無風險利率 (Risk-Free Rate, Rf)：**
*   參考美國10年期國債收益率：**4.50%** (截至2024年5月)。

**B. 市場風險溢價 (Market Risk Premium, MRP)：**
*   採用行業常用估計值：**5.50%**。

**C. Beta (β)：**
*   根據Yahoo Finance或其他金融數據提供商，ALB的Beta值約為 **1.45**。

**D. 股權成本 (Cost of Equity, Ke)：**
*   使用資本資產定價模型 (CAPM)：
    $K_e = R_f + \beta \times MRP$
    $K_e = 4.50\% + 1.45 \times 5.50\% = 4.50\% + 7.98\% = 12.48\%$

**E. 債務成本 (Cost of Debt, Kd)：**
*   參考ALB近期發行債券的收益率或其利息費用與總債務的比率。假設稅前債務成本為 **6.50%**。

**F. 有效稅率 (Effective Tax Rate, T)：**
*   根據ALB近四季 (TTM) 財報：約 **18.00%**。
*   稅後債務成本 $K_d(after-tax) = K_d(pre-tax) \times (1 - T) = 6.50\% \times (1 - 0.18) = 6.50\% \times 0.82 = 5.33\%$

**G. 資本結構 (Capital Structure)：**
*   **市值 (Market Value of Equity, E)：** $13.77 億美元 (117.2M 股 * $117.50/股)
*   **總債務 (Total Debt, D)：** 約 $38.0 億美元 (參考最新財報)
*   **總資本 (D+E)：** $38.0 + 137.7 = $175.7 億美元
*   **股權權重 (E / (D+E))：** $137.7 / 175.7 = 78.37\%$
*   **債務權重 (D / (D+E))：** $38.0 / 175.7 = 21.63\%$

**H. 加權平均資本成本 (WACC)：**
*   $WACC = (E / (D+E)) \times K_e + (D / (D+E)) \times K_d(after-tax)$
*   $WACC = (0.7837 \times 12.48\%) + (0.2163 \times 5.33\%)$
*   $WACC = 9.78\% + 1.15\% = 10.93\%$
*   **最終 WACC 採用：11.00%** (取整方便計算及考慮一些緩衝)

#### 3. 股息折現模型 (DDM)

ALB 支付股息，但考慮到其作為成長型公司，自由現金流更多用於資本支出擴張而非高額股息派發，DDM 可能不是最能反映其內在價值的模型。然而，我們仍將按要求進行計算。

*   **當前年度股息 (D0)：** $1.60/股
*   **股權成本 (Ke)：** 12.48%

**採用兩階段 DDM：**
*   第一階段 (5年)：股息增長率 $g_1 = 3.0\%$
*   第二階段 (永續)：股息增長率 $g_2 = 2.5\%$

1.  **預測期股息現值：**
    *   $D_1 = 1.60 \times (1+0.03) = 1.648$
    *   $PV(D_1) = 1.648 / (1.1248)^1 = 1.465$
    *   $D_2 = 1.648 \times (1+0.03) = 1.697$
    *   $PV(D_2) = 1.697 / (1.1248)^2 = 1.341$
    *   $D_3 = 1.697 \times (1+0.03) = 1.748$
    *   $PV(D_3) = 1.748 / (1.1248)^3 = 1.230$
    *   $D_4 = 1.748 \times (1+0.03) = 1.800$
    *   $PV(D_4) = 1.800 / (1.1248)^4 = 1.130$
    *   $D_5 = 1.800 \times (1+0.03) = 1.854$
    *   $PV(D_5) = 1.854 / (1.1248)^5 = 1.040$
    *   **預測期股息現值總和：$1.465 + 1.341 + 1.230 + 1.130 + 1.040 = **$6.21/股**

2.  **終端股息價值 (Terminal Dividend Value) 在 DDM 中的計算：**
    *   $D_6 = D_5 \times (1 + g_2) = 1.854 \times (1 + 0.025) = 1.900$
    *   $TV_{DDM} = D_6 / (K_e - g_2) = 1.900 / (0.1248 - 0.025) = 1.900 / 0.0998 = $19.04/股$
    *   **終端股息價值現值 (PV of TV_DDM)：**
        *   $PV(TV_{DDM}) = TV_{DDM} / (1 + K_e)^5 = 19.04 / (1.1248)^5 = 19.04 / 1.799 = **$10.58/股**

3.  **DDM 每股估值：**
    *   $6.21 + $10.58 = **$16.79/股**

**DDM 估值結果：** $16.79/股
**DDM 合理價位區間：** $15.00 - $20.00/股

**DDM 適用性說明：** 由於 ALB 是一家資本密集型、高增長導向的公司，其大部分盈利被再投資於擴張和新項目，而非以股息形式分發。因此，DDM 模型的估值結果顯著低於其他模型，可能無法完全捕捉公司的內在價值。

**綜合分析：**

1.  **DDM 模型的結果顯著偏低。** 這主要是因為 ALB 的股息支付比例較低，且作為成長型公司，其價值主要來源於未來現金流的增長而非當前股息。因此，DDM 對於 ALB 的估值參考意義有限。
2.  **DCF 模型結果：$95.60。** 此模型考慮了公司未來的增長潛力、資本支出和現金流，對於成長型公司通常被認為是最全面的方法。鋰市場的波動性使得DCF的假設更具挑戰性，但通過敏感性分析，我們得到了一個相對穩健的區間。
3.  **相對估值結果分化嚴重。**
    *   基於 **P/E 倍數** 的估值結果 (約 $152-$174) 遠高於 DCF 和 EV/EBITDA。這可能由於 ALB 在過去一年經歷了高盈利時期，使得其 TTM EPS 較高，但市場已預期未來盈利壓力。如果使用未來預期盈利 (Forward EPS)，P/E 估值可能會更低。ALB 目前的 P/E (9.3x) 確實低於同業平均，這可能暗示其被低估或市場預期其EPS將大幅下降。
    *   基於 **EV/EBITDA 倍數** 的估值結果 (約 $75-$83) 與 DCF 估值較為接近，甚至略低。EV/EBITDA 相比 P/E 較少受會計政策影響，且能更好地反映高資本支出公司的運營價值。ALB 目前的 EV/EBITDA (8.6x) 略高於同業平均，可能暗示市場對其高資本支出後的EBITDA增長有較高預期。

**綜合結論與建議：**

考慮到 ALB 作為一家成長型且資本密集型的特種化學品公司，其價值核心在於未來的自由現金流創造能力。

*   **DCF 模型** 提供了一個相對穩健的內部價值視角，其估值 $95.60 處於當前股價 (約 $117.50) 之下。
*   **EV/EBITDA 倍數** 估值結果與 DCF 相近，進一步印證了其在 $75 - $95 範圍內的估值。這表明，若按當前同業平均 EBITDA 倍數衡量，ALB 可能被高估。
*   **P/E 倍數** 估值結果較高，可能受到鋰價高峰期盈利的殘留影響，或市場對其未來盈利恢復抱有更高期望。

**基於以上分析，並權衡各模型的可靠性，尤其考慮到鋰市場的波動性和ALB的資本支出計劃，我認為 DCF 和 EV/EBITDA 提供的估值結果更具參考價值。**

**綜合判斷的合理價位區間應落在 $85.00 - $110.00/股。** 當前股價 $117.50 略高於這個範圍的上限。這可能意味著市場對鋰價的長期回升和ALB擴張計劃的成功給予了更高的溢價。

**投資者需要密切關注：**
1.  **鋰價走勢：** 這是影響 ALB 盈利和現金流的關鍵因素。
2.  **新項目投產進度及成本：** 決定了公司能否按預期實現產量和成本效益。

4.  **競爭格局：** 新進入者和既有生產商的產能擴張。

### 五、與同產業可比公司的估值比較 (簡要)

當前 ALB 的 P/E 倍數 (約 9.3x TTM) 低於同業平均 (約 12.0x)，而 EV/EBITDA 倍數 (約 8.6x TTM) 則略高於同業平均 (約 6.5x)。
