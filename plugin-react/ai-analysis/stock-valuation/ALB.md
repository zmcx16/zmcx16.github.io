**階段一：高增長階段 (5年)**

| 年度 | 股息 (D0 * (1+g)^n) | 折現因子 (1 / (1+Ke)^n) | 折現股息 |
| :--- | :--- | :--- | :--- |
| D0 (2025) | 1.62美元 | - | - |
| D1 (2026) | 1.62 * (1.05)^1 = 1.70美元 | 1 / (1.1225)^1 = 0.8909 | 1.51美元 |
| D2 (2027) | 1.62 * (1.05)^2 = 1.79美元 | 1 / (1.1225)^2 = 0.7937 | 1.42美元 |
| D3 (2028) | 1.62 * (1.05)^3 = 1.88美元 | 1 / (1.1225)^3 = 0.7071 | 1.33美元 |
| D4 (2029) | 1.62 * (1.05)^4 = 1.97美元 | 1 / (1.1225)^4 = 0.6299 | 1.24美元 |
| D5 (2030) | 1.62 * (1.05)^5 = 2.07美元 | 1 / (1.1225)^5 = 0.5612 | 1.16美元 |

**階段一折現股息總和 = 1.51 + 1.42 + 1.33 + 1.24 + 1.16 = 6.66美元**

**階段二：永續增長階段 (從第6年開始)**

*   **D6 (2031) = D5 * (1 + 長期增長率) = 2.07美元 * (1 + 0.025) = 2.12美元**
*   **終值 (Terminal Value at end of Year 5) = D6 / (Ke - 長期增長率)**
    *   TV = 2.12美元 / (0.1225 - 0.025) = 2.12美元 / 0.0975 = 21.74美元
*   **終值折現值 = TV / (1 + Ke)^5 = 21.74美元 / (1.1225)^5 = 21.74美元 * 0.5612 = 12.20美元**

**DDM 估值結果：**

*   **每股估值 = 階段一折現股息總和 + 終值折現值 = 6.66美元 + 12.20美元 = 18.86美元**

**DDM 模型估值結果：**

*   **估值結果：** 18.86美元
*   **合理價位區間：** 考慮到股息增長率的敏感性，合理價位區間可能在 **15美元至25美元** 之間。

DDM 估值結果遠低於當前股價，這表明市場對 ALB 的估值並非主要基於其當前股息支付能力，而是基於其未來增長潛力（尤其是鋰業務）。
