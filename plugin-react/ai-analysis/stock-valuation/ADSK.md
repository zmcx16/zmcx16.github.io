## Autodesk (ADSK) 股票估值分析

本報告將對美股公司 Autodesk (ADSK) 進行全面的股票估值分析，結合其基本面數據、最新市場動態和產業趨勢，並應用折現現金流模型 (DCF) 和相對估值法。

### 公司概況與最新動態

Autodesk 是一家全球領先的 3D 設計、工程和娛樂技術解決方案公司，其產品廣泛應用於建築、工程、施工 (AEC)、產品設計、製造 (MFG) 以及媒體和娛樂 (M&E) 等領域。公司已完成向訂閱制和雲端平台的轉型，並積極佈局 AI 領域，預計在 2027 財年推出新的 AI 功能。

**最新財報與營運亮點 (截至 2026 財年)**：
*   2026 財年營收達到 72.1 億美元，年增 17.53%。
*   2026 財年自由現金流為 24.09 億美元，年增 54%。
*   2026 財年非 GAAP 營業利潤率為 38%。
*   公司在 2026 財年第四季度營收為 19.6 億美元，年增 19%。
*   Autodesk 正在將內部資源重新分配到雲端、平台和 AI 等關鍵領域，並優化其市場推廣職能，以更好地滿足客戶和渠道合作夥伴不斷變化的需求。
*   公司預計在 2027 財年實現持續增長，並預計營收在 81 億至 81.7 億美元之間。
*   Autodesk 於 2026 年 5 月與 Permiso Security 合作，部署新的 AI 安全功能，以應對日益增長的 AI 風險和企業環境中的安全問題。

### 估值方法與假設

#### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值，以得出其內在價值。

**估值假設：**

*   **預測期間：** 5 年 (2027 財年 - 2031 財年)，之後進入永續增長階段。
*   **起始營收 (2026 財年)：** 72.1 億美元。
*   **營收增長率：**
    *   2027 財年：根據公司指引，預計營收增長約 12.4% (從 7.21B 到 8.1B-8.17B)。
    *   2028-2031 財年：預計逐步放緩至 8%-6% (考慮到公司規模擴大和市場成熟)。
    *   永續增長率 (g)：3% (略高於長期通脹率，反映軟體行業的持續創新和市場需求)。
*   **營業利潤率 (Operating Margin)：**
    *   2026 財年非 GAAP 營業利潤率為 38%。公司目標是到 2029 財年達到 41% 的非 GAAP 營業利潤率。我們將假設營業利潤率從 2027 財年的 38% 逐步提升至 2031 財年的 41%。
*   **稅率 (Tax Rate)：** 採用非 GAAP 稅率 19%。
*   **資本支出 (Capital Expenditures, CapEx)：** 2026 財年為 7600 萬美元。預計未來幾年保持相對穩定，約為營收的 0.7% - 0.8%。
*   **折舊與攤銷 (Depreciation & Amortization, D&A)：** 預計約為營收的 1.5% - 2%。
*   **營運資本變動 (Change in Net Working Capital)：** 預計為營收增長的 5% (作為營收增長的百分比，反映營運資金需求)。
*   **加權平均資本成本 (Weighted Average Cost of Capital, WACC)：**
    *   根據搜索結果，ADSK 的 WACC 介於 8.26% 至 10.93% 之間。我們將採用 9.0% 作為基準 WACC。
    *   **計算 WACC 的組成部分 (參考 Alpha Spread 和 GuruFocus 數據):**
        *   無風險利率 (Risk-Free Rate)：約 4.3% - 4.42% (參考 10 年期美國國債收益率)。
        *   股票 Beta：0.96 - 1.32。我們採用 1.1。
        *   股權風險溢價 (Equity Risk Premium, ERP)：約 4.18% - 5.10%。我們採用 4.5%。
        *   股權成本 (Cost of Equity) = 無風險利率 + Beta \* ERP = 4.3% + 1.1 \* 4.5% = 9.25%。
        *   債務成本 (Cost of Debt)：約 4.88% - 5%。我們採用 5%。
        *   稅率：19%。
        *   債務權益比 (Debt/Equity)：根據提供的數據，Debt/Eq 為 0.9，但這通常是賬面價值。根據 Alpha Spread，債務權重約為 4.71%。我們將使用市場價值權重，假設股權權重為 90%，債務權重為 10% (考慮到其低債務水平)。
        *   WACC = (股權成本 \* 股權權重) + (債務成本 \* (1 - 稅率) \* 債務權重)
        *   WACC = (9.25% \* 0.90) + (5% \* (1 - 0.19) \* 0.10) = 8.325% + 0.405% = 8.73%。
        *   為保守起見，我們將使用 9.0% 作為 WACC。

**DCF 計算步驟：**

1.  **預測營收：**
    *   2026 財年營收：7.21B 美元
    *   2027 財年：7.21B \* (1 + 0.124) = 8.10B 美元
    *   2028 財年：8.10B \* (1 + 0.08) = 8.75B 美元
    *   2029 財年：8.75B \* (1 + 0.07) = 9.36B 美元
    *   2030 財年：9.36B \* (1 + 0.06) = 9.92B 美元
    *   2031 財年：9.92B \* (1 + 0.06) = 10.51B 美元

2.  **計算 EBIT (營業利潤)：** 營收 \* 營業利潤率
    *   2027 財年：8.10B \* 0.38 = 3.08B 美元
    *   2028 財年：8.75B \* 0.39 = 3.41B 美元
    *   2029 財年：9.36B \* 0.40 = 3.74B 美元
    *   2030 財年：9.92B \* 0.405 = 4.02B 美元
    *   2031 財年：10.51B \* 0.41 = 4.31B 美元

3.  **計算 NOPAT (稅後淨營業利潤)：** EBIT \* (1 - 稅率)
    *   稅率：19%
    *   2027 財年：3.08B \* (1 - 0.19) = 2.50B 美元
    *   2028 財年：3.41B \* (1 - 0.19) = 2.76B 美元
    *   2029 財年：3.74B \* (1 - 0.19) = 3.03B 美元
    *   2030 財年：4.02B \* (1 - 0.19) = 3.26B 美元
    *   2031 財年：4.31B \* (1 - 0.19) = 3.49B 美元

4.  **計算自由現金流 (FCF)：** NOPAT + D&A - CapEx - 營運資本變動
    *   D&A (預計營收的 1.5%)：
        *   2027 財年：8.10B \* 0.015 = 0.12B 美元
        *   2028 財年：8.75B \* 0.015 = 0.13B 美元
        *   2029 財年：9.36B \* 0.015 = 0.14B 美元
        *   2030 財年：9.92B \* 0.015 = 0.15B 美元
        *   2031 財年：10.51B \* 0.015 = 0.16B 美元
    *   CapEx (預計營收的 0.7%)：
        *   2027 財年：8.10B \* 0.007 = 0.06B 美元
        *   2028 財年：8.75B \* 0.007 = 0.06B 美元
        *   2029 財年：9.36B \* 0.007 = 0.07B 美元
        *   2030 財年：9.92B \* 0.007 = 0.07B 美元
        *   2031 財年：10.51B \* 0.007 = 0.07B 美元
    *   營運資本變動 (預計營收增長的 5%)：
        *   2027 財年：(8.10B - 7.21B) \* 0.05 = 0.04B 美元
        *   2028 財年：(8.75B - 8.10B) \* 0.05 = 0.03B 美元
        *   2029 財年：(9.36B - 8.75B) \* 0.05 = 0.03B 美元
        *   2030 財年：(9.92B - 9.36B) \* 0.05 = 0.03B 美元
        *   2031 財年：(10.51B - 9.92B) \* 0.05 = 0.03B 美元

5.  **計算終端價值 (Terminal Value, TV)：**
    *   TV = FCF_2031 \* (1 + g) / (WACC - g)
    *   TV = 3.55B \* (1 + 0.03) / (0.09 - 0.03) = 3.66B / 0.06 = 61.00B 美元

6.  **折現所有現金流和終端價值：**
    *   現值 (PV) = FCF_t / (1 + WACC)^t
    *   PV(FCF_2027) = 2.52B / (1 + 0.09)^1 = 2.31B 美元
    *   PV(FCF_2028) = 2.80B / (1 + 0.09)^2 = 2.35B 美元
    *   PV(FCF_2029) = 3.07B / (1 + 0.09)^3 = 2.37B 美元
    *   PV(FCF_2030) = 3.31B / (1 + 0.09)^4 = 2.35B 美元
    *   PV(FCF_2031) = 3.55B / (1 + 0.09)^5 = 2.31B 美元
    *   PV(TV) = 61.00B / (1 + 0.09)^5 = 39.65B 美元

7.  **計算公司總價值 (Enterprise Value, EV)：**
    *   EV = 2.31 + 2.35 + 2.37 + 2.35 + 2.31 + 39.65 = 51.34B 美元

8.  **計算股權價值 (Equity Value)：** EV + 現金及約當現金 - 總債務
    *   現金及約當現金：26 億美元
    *   總債務：27.3 億美元
    *   股權價值 = 51.34B + 2.60B - 2.73B = 51.21B 美元

9.  **計算每股內在價值：** 股權價值 / 流通股數
    *   流通股數：根據市值 499.6 億美元和股價 236.62 美元，流通股數約為 499.6B / 236.62 = 211.14 百萬股。
    *   每股內在價值 = 51.21B / 211.14M = **242.54 美元/股**

**DCF 估值結果與合理價位區間：**
*   DCF 模型得出的每股內在價值為 **242.54 美元**。
*   考慮到假設的敏感性，合理價位區間可能在 **220 - 260 美元** 之間。
