### 3.3 股息折現模型 (Dividend Discount Model, DDM)

CVS 是一家成熟且支付穩定股息的公司，DDM 適用。我們將使用兩階段 DDM。

**估值假設：**

1.  **當前股息 (D0)：** $79.61 * 0.0336 = $2.67/股。
2.  **股權成本 (Required Rate of Return, Ke)：** 9.15% (與 DCF 模型相同)。
3.  **股息增長率：**
    *   **第一階段 (高增長，未來 3 年)：** 5.0% (考慮公司轉型和盈利恢復，但仍需大量再投資)。
    *   **第二階段 (永續增長，之後)：** 2.5% (與 DCF 的長期增長率一致)。

**DDM 計算過程：**

*   **未來 3 年股息：**
    *   D1 = D0 * (1 + 0.05) = $2.67 * 1.05 = $2.80
    *   D2 = D1 * (1 + 0.05) = $2.80 * 1.05 = $2.94
    *   D3 = D2 * (1 + 0.05) = $2.94 * 1.05 = $3.09

*   **各期股息現值：**
    *   PV(D1) = $2.80 / (1 + 0.0915)^1 = $2.80 / 1.0915 = $2.56
    *   PV(D2) = $2.94 / (1 + 0.0915)^2 = $2.94 / 1.1914 = $2.47
