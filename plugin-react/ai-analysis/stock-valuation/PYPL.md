*   股權成本 (Ke) = 11.915%

*   每股價值 = 0.56 * 1.025 / 0.09415 = 0.574 / 0.09415 = 6.09 美元

**DDM 估值結果與合理價位區間:**

DDM 估值結果非常低，僅為 **6.09 美元**。這再次印證了 PayPal 並非典型的股息成長型股票，其價值主要來自於未來的盈利增長和自由現金流。因此，DDM 在此情況下參考價值有限。

#### 3.4. 其他合理估值方法 (PEG Ratio)

*   PEG = P/E / EPS Growth Rate
*   PYPL PEG = 1.09 (用戶提供數據)
*   PYPL P/E (TTM) = 7.45
*   PYPL EPS next Y_% = 0.09 (9%) (用戶提供數據) 或 11.93%

如果使用 11.93% 的 EPS 增長率：
*   PEG = 7.45 / 11.93 = 0.62

通常認為 PEG 接近 1 表示合理估值，小於 1 可能表示低估。0.62 的 PEG 比率強烈暗示 PayPal 目前被低估。

PayPal 面臨著激烈的市場競爭和宏觀經濟逆風，Q4 2025 財報顯示營收和 EPS 均未達預期，這可能是導致其股價低迷的原因。然而，公司預計 2026 年調整後自由現金流至少為 60 億美元，這是一個積極信號。如果 PayPal 能夠有效執行其轉型策略，改善盈利能力和增長前景，並在競爭激烈的數位支付市場中保持其領先地位，那麼其股價有望在未來幾年內實現顯著回升。

**合理價位區間綜合判斷：**

考慮到 DCF 估值和相對估值所暗示的潛力，以及市場對其增長挑戰的擔憂，我認為 PayPal 的合理價位區間應在 **75 - 100 美元**之間。這代表著從當前股價有 86% 到 148% 的上漲空間。投資者應密切關注公司未來的財報表現、新產品發布以及市場份額的變化。

### 5. 與同產業可比公司的估值比較

如相對估值部分所示，PayPal 的 P/E (7.45)、Forward P/E (7.54)、P/S (1.08)、P/B (1.83)、EV/EBITDA (5.86) 和 P/FCF (7.0) 等估值乘數均顯著低於其主要競爭對手如 Visa、Mastercard、Block 和 Adyen。

*   **Visa 和 Mastercard** 作為成熟的支付網絡巨頭，擁有更高的利潤率和更穩定的增長，因此享有更高的估值乘數 (P/E 通常在 25-35 倍，P/S 在 15-20 倍)。
*   **Block (Square)** 和 **Adyen** 則代表了金融科技領域的創新者，儘管盈利能力可能不如 Visa/Mastercard，但其高增長潛力也使其估值乘數較高 (P/E 甚至可達 40-90 倍，P/S 在 2-20 倍)。

PayPal 目前的估值更接近於一家增長停滯或面臨嚴重挑戰的「價值股」，而非其歷史上曾被視為的「成長股」。這種低估值反映了市場對其未來增長路徑和競爭力的擔憂。然而，這也為尋求價值投資的投資者提供了潛在的機會，前提是公司能夠成功扭轉局面並恢復增長。
