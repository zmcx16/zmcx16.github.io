### 2. 估值方法應用

鑑於 PayPal 的最新財報和指引顯示增長放緩，且股息率相對較低，我們將重點關注 DCF 和相對估值法。

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2026 - 2030)
*   **折現率 (Discount Rate / WACC):** 根據 GuruFocus，PayPal 的 WACC 為 11.07%。我們將使用此數據。
*   **長期增長率 (Terminal Growth Rate):** 考慮到金融科技產業的成熟和競爭加劇，以及 PayPal 自身增長放緩的指引，我們將採用保守的長期增長率。全球 GDP 增長率約為 2-3%，因此我們將使用 2.5% 作為長期增長率。
*   **營收增長率 (Revenue Growth Rate):**
    *   2026 年：根據公司指引，預計為低個位數增長。我們假設 3%。
    *   2027-2030 年：考慮到競爭和市場飽和，我們假設增長率逐步放緩，例如 2027 年 4%，2028 年 3.5%，2029 年 3%，2030 年 2.5%。
*   **營業利潤率 (Operating Margin):** 根據提供數據，目前營業利潤率為 19.69%。我們假設在預測期內，公司通過成本控制和效率提升，營業利潤率能維持在 19-20% 之間。
*   **稅率 (Tax Rate):** 根據提供數據，利潤率為 15.7%。我們將使用 20% 作為有效稅率。
*   **資本支出 (Capital Expenditure, CapEx) 和折舊攤銷 (D&A):** 根據歷史數據和公司規模，我們假設 CapEx 佔營收的 3-4%，D&A 佔營收的 4-5%。
*   **營運資本變動 (Change in Working Capital):** 假設為營收的 0.5% - 1% 之間，或保持相對穩定。

**DCF 計算步驟 (簡化):**

1.  **預測營收 (Revenue Forecast):**
    *   2025 年營收 (基期): 86.8 億美元 (Q4 2025) * 4 = 347.2 億美元 (年化) (此處為簡化處理，實際應使用全年營收)
    *   根據 Q4 2023 財報，2023 年全年營收為 298 億美元。2024 年 Q1 營收未提供，但 2024 年全年預計自由現金流約 50 億美元。
    *   我們使用 2023 年全年營收 298 億美元作為基期，並根據 2026 年低個位數增長指引進行調整。
    *   **更正基期：** 根據 Q4 2023 財報，2023 年全年淨營收為 298 億美元。根據 Q4 2025 財報，2025 年全年營收為 332 億美元。我們將使用 2025 年營收 332 億美元作為基期。

    *   **營收預測 (基於 2025 年營收 332 億美元):**
        *   2026E: 332 * (1 + 3%) = 341.96 億美元
        *   2027E: 341.96 * (1 + 4%) = 355.64 億美元
        *   2028E: 355.64 * (1 + 3.5%) = 368.06 億美元
        *   2029E: 368.06 * (1 + 3%) = 379.10 億美元
        *   2030E: 379.10 * (1 + 2.5%) = 388.58 億美元

2.  **計算稅後淨營業利潤 (NOPAT):**
    *   NOPAT = 營收 * 營業利潤率 * (1 - 稅率)
    *   假設營業利潤率為 19.69% (提供數據)，稅率 20%。
    *   2026E NOPAT = 341.96 * 0.1969 * (1 - 0.20) = 53.83 億美元
    *   2027E NOPAT = 355.64 * 0.1969 * (1 - 0.20) = 56.00 億美元
    *   2028E NOPAT = 368.06 * 0.1969 * (1 - 0.20) = 57.96 億美元
    *   2029E NOPAT = 379.10 * 0.1969 * (1 - 0.20) = 59.70 億美元
    *   2030E NOPAT = 388.58 * 0.1969 * (1 - 0.20) = 61.20 億美元

3.  **計算自由現金流 (Free Cash Flow, FCF):**
    *   FCF = NOPAT + 折舊攤銷 (D&A) - 資本支出 (CapEx) - 營運資本變動 (Change in Working Capital)
    *   假設 D&A 佔營收 4.5%，CapEx 佔營收 3.5%，營運資本變動為營收的 0.5%。
    *   2026E FCF = 53.83 + (341.96 * 0.045) - (341.96 * 0.035) - (341.96 * 0.005) = 53.83 + 15.39 - 11.97 - 1.71 = 55.54 億美元
    *   2027E FCF = 56.00 + (355.64 * 0.045) - (355.64 * 0.035) - (355.64 * 0.005) = 56.00 + 16.00 - 12.45 - 1.78 = 57.77 億美元
    *   2028E FCF = 57.96 + (368.06 * 0.045) - (368.06 * 0.035) - (368.06 * 0.005) = 57.96 + 16.56 - 12.88 - 1.84 = 59.80 億美元
    *   2029E FCF = 59.70 + (379.10 * 0.045) - (379.10 * 0.035) - (379.10 * 0.005) = 59.70 + 17.06 - 13.27 - 1.90 = 61.59 億美元
    *   2030E FCF = 61.20 + (388.58 * 0.045) - (388.58 * 0.035) - (388.58 * 0.005) = 61.20 + 17.49 - 13.59 - 1.94 = 63.16 億美元

4.  **計算終值 (Terminal Value, TV):**
    *   TV = (2030E FCF * (1 + 長期增長率)) / (WACC - 長期增長率)
    *   TV = (63.16 * (1 + 0.025)) / (0.1107 - 0.025) = (63.16 * 1.025) / 0.0857 = 64.74 / 0.0857 = 755.43 億美元

5.  **計算各期 FCF 和 TV 的現值 (Present Value):**
    *   PV(FCF 2026) = 55.54 / (1 + 0.1107)^1 = 49.99 億美元
    *   PV(FCF 2027) = 57.77 / (1 + 0.1107)^2 = 46.74 億美元
    *   PV(FCF 2028) = 59.80 / (1 + 0.1107)^3 = 44.00 億美元
    *   PV(FCF 2029) = 61.59 / (1 + 0.1107)^4 = 41.59 億美元
    *   PV(FCF 2030) = 63.16 / (1 + 0.1107)^5 = 39.45 億美元
    *   PV(TV) = 755.43 / (1 + 0.1107)^5 = 471.77 億美元

6.  **計算企業價值 (Enterprise Value, EV):**
    *   EV = PV(FCF 2026-2030) + PV(TV)
    *   EV = 49.99 + 46.74 + 44.00 + 41.59 + 39.45 + 471.77 = 693.54 億美元

7.  **計算股權價值 (Equity Value):**
    *   Equity Value = EV + 現金及約當現金 - 總債務
    *   根據 Q4 2023 財報，現金、現金等價物和投資為 173 億美元，債務為 113 億美元。
    *   Equity Value = 693.54 + 173 - 113 = 753.54 億美元

8.  **計算每股價值 (Intrinsic Value Per Share):**
    *   根據 Seeking Alpha，流通股數約為 10.2 億股。
    *   Intrinsic Value Per Share = Equity Value / 流通股數
    *   Intrinsic Value Per Share = 753.54 億美元 / 10.2 億股 = 73.88 美元/股

**DCF 估值結果：**

*   **每股合理價位：** 73.88 美元
*   **合理價位區間：** 考慮到假設的敏感性，我們可以進行敏感性分析。
    *   若 WACC 變動 +/- 0.5%，長期增長率變動 +/- 0.5%，則合理價位區間可能在 **60 - 90 美元** 之間。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

根據提供的數據，PayPal 的股息率為 0.0062%，非常低。雖然最新資訊顯示 PayPal 已於 Q4 2025 首次宣布季度股息 0.14 美元/股，但這是一個新的發展，且股息金額相對較小。對於像 PayPal 這樣仍處於增長階段 (儘管近期放緩) 的科技公司，其主要價值驅動因素是未來的盈利增長和自由現金流，而非當前股息。因此，DDM 在此情況下可能不是最合適或最能反映公司真實價值的估值方法。

如果我們嘗試應用 DDM，由於股息歷史短且金額小，預測未來股息增長將非常困難且不確定。因此，我們將不將 DDM 作為主要估值方法，並認為其不適用於當前對 PayPal 的全面估值。

*   **折現現金流模型 (DCF):** 每股合理價位約 **73.88 美元** (區間：60 - 90 美元)
*   **相對估值法 (Comparable Company Analysis):** 每股合理價位約 **100 - 165 美元**

**分析與綜合結論：**

兩種估值方法給出了顯著不同的結果。DCF 模型給出的估值較為保守，而相對估值法則顯示 PayPal 被嚴重低估。

造成這種差異的原因可能包括：

1.  **DCF 假設的保守性：** 我們的 DCF 假設了相對保守的營收增長率和穩定的利潤率，這可能未能充分反映公司在未來轉型成功後的潛在增長。
2.  **市場情緒與相對估值：** 相對估值法反映了市場對可比公司的估值水平。PayPal 目前的股價和倍數遠低於同行，這可能反映了市場對其近期業績不佳、CEO 變動以及競爭加劇的悲觀情緒。然而，如果 PayPal 能夠成功執行其轉型策略，其估值倍數有望向行業平均水平靠攏。
3.  **成長前景的差異：** 雖然 PayPal 面臨挑戰，但其在數位支付領域的市場地位和品牌認知度依然強勁。可比公司如 Adyen 和 Block 可能在某些細分市場表現出更強勁的增長勢頭，因此獲得更高的估值倍數。

**綜合考量：**

儘管 PayPal 面臨挑戰，但其強大的自由現金流生成能力 (2024 年預計 50 億美元) 和相對較低的估值倍數 (P/FCF 僅 7.48) 表明其可能被市場低估。DCF 模型提供了一個基於內在價值的保守估計，而相對估值則揭示了與同行相比的巨大估值差距。

考慮到分析師的平均目標價在 50.71 美元至 59.03 美元之間，且 WealthAP 認為其公允價值為 82.00 美元，這與我們的 DCF 估值結果更為接近。相對估值法雖然顯示出巨大的潛在上漲空間，但這需要 PayPal 在未來幾年內成功扭轉其增長困境，並重新獲得市場的信心。

**最終合理價位區間：**

綜合 DCF 模型的保守估計和相對估值法所揭示的潛在價值，並考慮到當前市場對 PayPal 的謹慎態度，我認為 PayPal 的合理價位區間應在 **55 美元至 85 美元** 之間。

*   **下限 (55 美元):** 略高於當前股價，反映了其穩定的現金流和被低估的現狀，但考慮到近期挑戰和不確定性。
*   **上限 (85 美元):** 接近分析師較樂觀的目標價和 DCF 估值的上限，假設公司能夠在未來幾年內穩定業務並實現溫和增長。

**與同產業可比公司的估值比較：**

如前所述，PayPal 目前的估值倍數 (P/E, P/S, P/FCF, EV/EBITDA) 遠低於其主要競爭對手 Block 和 Adyen，甚至低於更為成熟的支付網絡公司 Visa 和 Mastercard。這表明市場對 PayPal 的增長前景和競爭地位存在顯著的擔憂。如果 PayPal 能夠成功執行其戰略轉型，提升其品牌結帳業務的增長，並有效應對競爭，其估值倍數有望向行業平均水平回歸，從而帶來顯著的股價上漲空間。然而，這也伴隨著執行風險。
