#### 加權平均資本成本 (WACC) 計算

WACC = (E/V) \* Re + (D/V) \* Rd \* (1 - T)

其中：
*   Re = 股權成本 (Cost of Equity)
*   Rd = 債務成本 (Cost of Debt)
*   E = 股權市值 (Market Value of Equity)
*   D = 債務市值 (Market Value of Debt)
*   V = 總資本市值 (E + D)
*   T = 稅率 (Corporate Tax Rate)

**a. 股權成本 (Re) - 使用資本資產定價模型 (CAPM):**
Re = 無風險利率 + Beta \* 市場風險溢酬
Re = 4.12% + 1.43 \* 5.5% = 4.12% + 7.865% = 11.985%

**b. 債務成本 (Rd):**
根據 PayPal 的最新財報，其利息支出為 1.11 億美元，總債務約為 99.8 億美元。
Rd = 利息支出 / 總債務 = 1.11 億美元 / 99.8 億美元 ≈ 1.11%
這個債務成本看起來非常低，可能反映了其強大的信用評級。

**c. 股權市值 (E) 與債務市值 (D):**
*   股權市值 (E) = 市值 = 430.4 億美元
*   債務市值 (D) = 總債務 = 99.8 億美元
*   總資本市值 (V) = E + D = 430.4 億美元 + 99.8 億美元 = 530.2 億美元

**d. WACC 計算:**
WACC = (430.4 / 530.2) \* 11.985% + (99.8 / 530.2) \* 1.11% \* (1 - 20%)
WACC = 0.8117 \* 11.985% + 0.1883 \* 1.11% \* 0.8
WACC = 9.728% + 0.167% = 9.895% ≈ 9.90%

#### 3.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測未來自由現金流 (Free Cash Flow, FCF)。由於缺乏詳細的營運資本、資本支出等預測數據，我們將基於營收增長和利潤率假設來估計 FCF。

**假設:**
*   **營收增長率:**
    *   2026 年: 考慮到 2026 年展望的低個位數增長，我們假設 3% (基於 2025 年 4% 增長和 2026 年預期下降)。

    *   2029-2030 年: 進一步放緩至 4%。
*   **營業利潤率 (Operating Margin):** 根據用戶提供數據為 0.1969 (19.69%)。我們假設未來 5 年維持在 19.5% 左右。
*   **稅率:** 20%。
*   **資本支出 (CapEx) / 營收:** 假設為 3% (基於歷史數據和行業平均水平)。
*   **折舊與攤銷 (D&A) / 營收:** 假設為 2% (通常低於 CapEx)。
*   **營運資本變動 (Change in NWC) / 營收:** 假設為 1% (考慮到支付行業的營運資本需求)。

**計算步驟:**

1.  **預測營收 (Revenue):**
    *   2025 年營收: 332 億美元
    *   2026 年: 332 億美元 \* (1 + 3%) = 341.96 億美元
    *   2027 年: 341.96 億美元 \* (1 + 5%) = 359.06 億美元
    *   2028 年: 359.06 億美元 \* (1 + 5%) = 377.01 億美元
    *   2029 年: 377.01 億美元 \* (1 + 4%) = 392.09 億美元

2.  **預測稅後營業利潤 (NOPAT):**
    *   NOPAT = 營收 \* 營業利潤率 \* (1 - 稅率)

    *   2030 年: 407.77 億美元 \* 19.5% \* (1 - 20%) = 63.63 億美元

3.  **預測自由現金流 (FCF):**
    *   FCF = NOPAT + D&A - CapEx - 營運資本變動
    *   D&A = 營收 \* 2%
    *   CapEx = 營收 \* 3%

4.  **計算終值 (Terminal Value, TV):**

#### 3.3 股息折現模型 (Dividend Discount Model, DDM)

PayPal 於 2026 年 3 月 25 日支付每股 0.14 美元的現金股息，年化股息為 0.56 美元，股息收益率為 1.2%。由於其股息收益率較低，且公司目前處於轉型和投資階段，股息增長可能不穩定，因此 DDM 可能不是最適合 PayPal 的主要估值方法。

然而，我們可以嘗試使用戈登增長模型 (Gordon Growth Model) 進行簡單計算，假設股息以一個恆定速度增長。

**假設:**
*   當前股息 (D0): 0.56 美元 (年化)
*   股息增長率 (g): 假設為 2% (低於長期增長率，反映保守預期)
*   股權成本 (Re): 11.985% (來自 WACC 計算)

**計算:**
股價 = D0 \* (1 + g) / (Re - g)
股價 = 0.56 美元 \* (1 + 2%) / (11.985% - 2%)
股價 = 0.56 美元 \* 1.02 / 0.09985
股價 = 0.5712 美元 / 0.09985 = 5.72 美元

這個結果遠低於當前股價，再次印證了 DDM 對於低股息、高增長潛力（或轉型期）的公司不適用。PayPal 的價值主要來自其未來的自由現金流增長，而非當前股息。因此，我們認為 DDM 不適用於 PayPal 的主要估值。
