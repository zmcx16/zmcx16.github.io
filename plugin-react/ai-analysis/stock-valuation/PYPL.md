### 2. 估值方法應用

#### 2.1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設:**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2026 年 - 2030 年)。
*   **折現率 (Discount Rate - WACC):** 我將使用 9.97% 作為基準 WACC。
    *   **敏感性分析範圍:** 9.0% - 11.0%。
*   **長期增長率 (Terminal Growth Rate):** 2.5% (略高於長期通脹率，反映成熟支付公司的溫和增長)。
    *   **敏感性分析範圍:** 2.0% - 3.0%。
*   **起始自由現金流 (Base Year Free Cash Flow):** 根據 2025 財年自由現金流 $5.6 億 和 2026 年預計超過 $60 億，我將使用 $60 億作為 2026 年的起始自由現金流。
*   **自由現金流增長率 (FCF Growth Rate):**
    *   2026 年: 預計約 7% (基於 2025 年 $5.6B 到 2026 年預計超過 $6B 的增長，以及分析師對 EPS 增長的預期)。
    *   2027-2028 年: 5% (考慮到競爭加劇和品牌結帳業務的挑戰，但有新產品和成本優化支持)。
    *   2029-2030 年: 4% (逐漸趨於穩定)。

**計算步驟:**

1.  **預測自由現金流 (Projected Free Cash Flow, FCF):**
    *   2026 年: $6.00 億
    *   2027 年: $6.00 * (1 + 0.07) = $6.42 億
    *   2028 年: $6.42 * (1 + 0.05) = $6.74 億
    *   2029 年: $6.74 * (1 + 0.04) = $7.01 億
    *   2030 年: $7.01 * (1 + 0.04) = $7.29 億

2.  **計算終值 (Terminal Value, TV):**
    *   TV = (2030 年 FCF * (1 + 長期增長率)) / (WACC - 長期增長率)
    *   TV = ($7.29 * (1 + 0.025)) / (0.0997 - 0.025)
    *   TV = ($7.29 * 1.025) / 0.0747
    *   TV = $7.47225 / 0.0747 = $100.03 億

3.  **計算預測期內 FCF 的現值 (Present Value of FCF):**
    *   2026 年 PV: $6.00 / (1 + 0.0997)^1 = $5.46 億
    *   2027 年 PV: $6.42 / (1 + 0.0997)^2 = $5.31 億
    *   2028 年 PV: $6.74 / (1 + 0.0997)^3 = $5.07 億
    *   2029 年 PV: $7.01 / (1 + 0.0997)^4 = $4.79 億
    *   2030 年 PV: $7.29 / (1 + 0.0997)^5 = $4.52 億
    *   預測期 FCF 現值總和 = $5.46 + $5.31 + $5.07 + $4.79 + $4.52 = $25.15 億

4.  **計算終值的現值 (Present Value of Terminal Value, PV of TV):**
    *   PV of TV = $100.03 / (1 + 0.0997)^5 = $100.03 / 1.608 = $62.21 億

5.  **計算公司總價值 (Enterprise Value, EV):**
    *   EV = 預測期 FCF 現值總和 + 終值的現值
    *   EV = $25.15 億 + $62.21 億 = $87.36 億

6.  **計算股權價值 (Equity Value):**
    *   股權價值 = EV + 現金及約當現金 - 總債務
    *   根據 2025 年財報，現金、現金等價物和投資總計 $14.8 億，債務總計 $11.6 億。
    *   股權價值 = $87.36 億 + $14.8 億 - $11.6 億 = $90.56 億

7.  **計算每股估值 (Intrinsic Value Per Share):**
    *   流通股數 (Shares Outstanding): 市值 $41.74B / 股價 $45.34 = 約 920.66 百萬股
    *   每股估值 = 股權價值 / 流通股數
    *   每股估值 = $90.56 億 / 9.2066 億股 = $98.36

**DCF 估值結果:**

*   **每股估值:** $98.36
*   **合理價位區間 (基於敏感性分析):**

| WACC / 長期增長率 | 2.0%      | 2.5%      | 3.0%      |
| :---------------- | :-------- | :-------- | :-------- |

**DCF 模型結論:**
基於基準假設，DCF 模型顯示 PayPal 的每股估值約為 **$98.36**。這遠高於當前股價 $45.34，表明公司可能被嚴重低估。然而，DCF 模型對增長率和折現率的假設非常敏感。
