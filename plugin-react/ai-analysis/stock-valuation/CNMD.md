### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期：** 5 年 (2026-2030)
*   **WACC (加權平均資本成本)：** 6.91%
    *   **股權成本 (Cost of Equity, Ke)：** 採用資本資產定價模型 (CAPM) 計算。
        *   無風險利率 (Risk-Free Rate)：假設為 4.0% (長期美國國債收益率)。
        *   市場風險溢酬 (Market Risk Premium)：假設為 5.5%。
        *   Beta：0.97 (取 0.96 和 0.98 的平均值)。
        *   Ke = 無風險利率 + Beta \* 市場風險溢酬 = 4.0% + 0.97 \* 5.5% = 4.0% + 5.335% = 9.335%。

        *   E/(E+D) = 1.176 / (1.176 + 1) = 0.540
        *   D/(E+D) = 1 / (1.176 + 1) = 0.460
        *   WACC = (0.540 \* 9.335%) + (0.460 \* 4.8% \* (1 - 0.25)) = 5.04% + 1.656% = 6.696%。

### 3. 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於支付穩定股息且股息預期增長的公司。

**估值假設：**

*   **當前年度股息 (D0)：** 0.80 美元 (每股)
*   **股權成本 (r)：** 9.335% (來自 DCF 模型中的 CAPM 計算)
*   **股息增長率 (g)：**
    *   CNMD 的股息歷史上相對穩定，每季 0.20 美元。
    *   儘管公司有盈利增長，但股息增長可能較慢。
    *   我們將採用一個保守的、穩定的股息增長率，例如 1.5%。

**DDM 計算 (戈登增長模型)：**

*   **每股估值 = D0 \* (1 + g) / (r - g)**
*   每股估值 = 0.80 美元 \* (1 + 0.015) / (0.09335 - 0.015)
*   每股估值 = 0.80 美元 \* 1.015 / 0.07835
*   每股估值 = 0.812 美元 / 0.07835 ≈ **10.36 美元**

**DDM 模型估值結果：**

*   **合理價位：** 10.36 美元
*   **合理價位區間：** 考慮到股息增長率的敏感性，合理區間可能在 **9 美元 - 12 美元**。
