#### 2.3. 股息折現模型 (Dividend Discount Model, DDM)

EL有穩定的股息支付歷史，但近期股息增長率為負 (-39.91% 近12個月，-17.13% 近36個月)。 儘管如此，公司預計明年的股息支付率為45.75%，表明股息可持續。 由於近期股息增長不穩定，且公司處於復甦期，單階段DDM可能不適用。我們嘗試使用兩階段DDM，假設短期內股息增長較慢或持平，長期恢復穩定增長。

**估值假設：**

*   **當前股息 (D0)：** 1.40 美元/股 (年化)
*   **短期增長率 (g1)：** 考慮到公司處於復甦期，短期內股息增長可能有限。我們假設未來3年股息增長率為0%。
*   **長期增長率 (g2)：** 假設長期股息增長率為2.5% (與DCF永續增長率一致)。
*   **折現率 (r)：** 使用股權成本 (Cost of Equity) 7.56%。

**DDM 計算步驟：**

1.  **計算預測期內的股息。**
    *   D1 = D0 * (1 + g1)
    *   D2 = D1 * (1 + g1)
    *   D3 = D2 * (1 + g1)

2.  **計算終端價值 (Terminal Value) (基於D3之後的永續增長)。**
    *   P3 = D3 * (1 + g2) / (r - g2)

3.  **將預測期內的股息和終端價值折現至現值。**
    *   每股價值 = D1/(1+r)^1 + D2/(1+r)^2 + (D3 + P3)/(1+r)^3

**DDM 計算：**

*   D0 = 1.40 美元
*   g1 = 0%
*   g2 = 2.5%
*   r = 7.56%

*   D1 = 1.40 * (1 + 0%) = 1.40
*   D2 = 1.40 * (1 + 0%) = 1.40
*   D3 = 1.40 * (1 + 0%) = 1.40

*   P3 = 1.40 * (1 + 0.025) / (0.0756 - 0.025) = 1.435 / 0.0506 = 28.36 美元

*   每股價值 = 1.40/(1.0756)^1 + 1.40/(1.0756)^2 + (1.40 + 28.36)/(1.0756)^3
    *   = 1.30 + 1.21 + 29.76 / 1.24 = 1.30 + 1.21 + 24.00 = **26.51 美元**

**DDM 估值結果：**
基於上述假設，DDM模型得出的每股估值約為 **26.51美元**。

**合理價位區間 (DDM)：**
DDM對增長率和折現率非常敏感。如果長期增長率略高，例如3%，則估值會顯著上升。
*   如果 g2 = 3.0%：P3 = 1.40 * (1 + 0.03) / (0.0756 - 0.03) = 1.442 / 0.0456 = 31.62 美元
    *   每股價值 = 1.30 + 1.21 + (1.40 + 31.62)/(1.0756)^3 = 1.30 + 1.21 + 33.02 / 1.24 = 1.30 + 1.21 + 26.63 = 29.14 美元
DDM模型得出的合理價位區間約為 **25 - 30 美元**。

    *   EV/EBITDA的估值介於中間，反映了公司在扣除折舊攤銷前的營運表現，考慮了債務因素，相對更為穩健。

    *   P/S和P/FCF的估值與分析師目標價更為接近，甚至更高，這支持了市場對EL品牌價值和現金流能力的認可。

**綜合結論：**

Estée Lauder (EL) 目前正處於一個轉型和復甦的階段。公司在2024財年第三季度表現出積極的復甦跡象，並預計下半年將是業績的轉折點。其強大的品牌組合和在高端美妝市場的領導地位是其核心競爭優勢。

然而，估值模型顯示出複雜的結果：
*   **基於盈利的估值 (DCF, DDM, Forward P/E) 普遍較低**，這反映了公司近期盈利能力受損以及未來盈利增長的不確定性。儘管預計未來EPS將大幅增長，但從負轉正的過程以及達到穩定高盈利水平仍需時間。
*   **基於收入和現金流的估值 (P/S, P/FCF) 則相對較高**，這表明市場可能更看重EL作為高端美妝品牌的長期潛力、品牌價值和穩定的現金流生成能力，而非短期盈利波動。

**綜合來看，EL的估值存在較大分歧。**
*   如果投資者主要關注短期盈利能力和傳統的盈利倍數，那麼EL目前的股價可能被高估。
*   如果投資者更看重公司的品牌資產、市場地位、長期增長潛力以及自由現金流能力，並相信其「利潤復甦和增長計劃」能夠成功實施，那麼目前的股價可能被認為是合理的，甚至有上漲空間。分析師的普遍「買入/持有」評級和較高的目標價也支持了後一種觀點。

**合理價位區間建議：**
考慮到EL的品牌實力、市場復甦潛力以及分析師的普遍預期，但同時也考慮到其近期盈利挑戰和DCF/DDM的保守估值，我建議將EL的合理價位區間設定在 **65 - 95 美元**。
