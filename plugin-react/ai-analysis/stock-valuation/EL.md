    *   EL 的市值 = 2.84 * $150 億 = $426 億。
    *   每股價值 = $426 億 / 3.633 億股 = **$117.26**

*   **P/B 估值**:
    *   EL 的 P/B = 10.57。
    *   可比公司平均 P/B = 3.76。
    *   這表明 EL 的 P/B 遠高於行業平均水平，可能被高估。
    *   如果使用行業平均 P/B 3.76，且 EL 的每股帳面價值 (Book Value per Share) = 股價 / P/B = $114.46 / 10.57 = $10.83。
    *   則合理股價 = 3.76 * $10.83 = **$40.73**

**相對估值結果**:

*   基於 Forward EV/EBITDA: 每股 **$40.54**
*   基於 P/S: 每股 **$117.26**
*   基於 P/B: 每股 **$40.73**

**合理價位區間 (相對估值)**: 由於 P/S 估值受營收影響較大，而 EL 目前盈利能力較弱，P/S 可能會高估。EV/EBITDA 和 P/B 估值結果較為接近。綜合來看，相對估值可能在 **$40 - $120** 之間，但考慮到 EL 目前的盈利狀況，偏向低端。

#### 3.3 股息折現模型 (Dividend Discount Model, DDM)

EL 支付股息 (Dividend %: 0.0123 或 1.22%)。由於公司目前盈利為負，但預計未來會恢復增長，因此 DDM 適用。

**DDM 假設**:

*   **當前股息 (D0)**: 股價 $114.46 * 0.0123 = $1.41 (每年)。
*   **股息增長率 (g)**:
    *   短期 (5 年): 考慮到公司盈利恢復和增長計劃，假設股息增長率為 5%。
    *   長期 (永續): 2.5% (與 DCF 的長期增長率一致)。
*   **股權成本 (Re)**: 10.525% (來自 WACC 計算)。

**DDM 計算步驟 (兩階段模型)**:

1.  **預測未來 5 年股息 (D1-D5)**
2.  **計算第 5 年末的終值 (Terminal Value of Dividends, D5 之後)**
3.  **將所有股息和終值折現回現值**

**簡化計算**:

*   D0 = $1.41
*   D1 = $1.41 * (1 + 0.05) = $1.48
*   D2 = $1.48 * (1 + 0.05) = $1.55
*   D3 = $1.55 * (1 + 0.05) = $1.63
*   D4 = $1.63 * (1 + 0.05) = $1.71
*   D5 = $1.71 * (1 + 0.05) = $1.80

*   **D1-D5 的現值**:
    *   PV(D1) = $1.48 / (1.10525)^1 = $1.34
    *   PV(D2) = $1.55 / (1.10525)^2 = $1.27
    *   PV(D3) = $1.63 / (1.10525)^3 = $1.21
    *   PV(D4) = $1.71 / (1.10525)^4 = $1.16
    *   PV(D5) = $1.80 / (1.10525)^5 = $1.10
    *   **預測期股息現值總和**: $1.34 + $1.27 + $1.21 + $1.16 + $1.10 = **$6.08**

*   **終值 (Terminal Value, TV)**:
    *   TV_5 = D6 / (Re - g_long_term)
    *   D6 = D5 * (1 + g_long_term) = $1.80 * (1 + 0.025) = $1.845
    *   TV_5 = $1.845 / (0.10525 - 0.025) = $1.845 / 0.08025 = $22.99
    *   **終值現值 (PV of TV_5)** = $22.99 / (1.10525)^5 = $22.99 / 1.646 = **$13.97**

*   **每股價值 (Intrinsic Value per Share)** = 預測期股息現值總和 + 終值現值 = $6.08 + $13.97 = **$20.05**

**DDM 估值結果**: 每股 **$20.05**

**合理價位區間 (DDM)**: 考慮到股息增長率和股權成本的敏感性，DDM 估值可能在 **$15 - $25** 之間。

    *   **Forward EV/EBITDA ($40.54) 和 P/B ($40.73)** 估值顯著低於當前股價。這反映了 EL 目前較高的債務水平、較低的盈利能力 (甚至虧損) 和相對較高的帳面價值倍數，與行業平均水平相比可能被高估。
3.  **DDM 估值 ($20.05)**: 該模型給出的估值最低。這可能是因為 EL 的股息支付相對保守，且股息增長率假設也較為溫和，未能充分反映公司未來的盈利恢復和增長潛力。此外，對於處於轉型期的公司，股息可能不是其價值創造的主要驅動力。

**綜合結論**:

Estée Lauder (EL) 目前正處於轉型期，其盈利能力在過去一年受到挑戰 (TTM EPS 為負)。儘管公司正在積極實施「Beauty Reimagined」戰略，並在 Q1 FY2026 顯示出初步的復甦跡象 (有機銷售增長，利潤率改善)，但其當前股價 ($114.46) 相對於多數估值模型而言，可能存在高估。

*   **P/S 估值** 接近當前股價，表明市場對其營收和品牌價值的認可。
*   **DCF 估值** 顯示約 $72.83，暗示當前股價存在約 36% 的下行空間。
*   **EV/EBITDA 和 P/B 估值** 則給出更低的價格，約 $40 左右，這強烈暗示當前股價被高估。
*   **DDM 估值** 最低，約 $20，這可能反映了公司在盈利不確定時期股息增長潛力的限制。

考慮到 EL 負的 TTM EPS 和較高的債務水平，以及分析師對未來盈利的樂觀預期，市場可能正在為其未來的復甦和增長支付溢價。然而，從保守的估值角度來看，特別是基於盈利能力和資產的倍數，以及 DCF 模型，當前股價似乎偏高。

**綜合判斷，EL 的合理價位可能落在 DCF 估值和相對估值 (EV/EBITDA, P/B) 的中位數附近，即約 $60 - $80 之間。** 當前股價 $114.46 存在一定的下行風險，除非公司能夠在未來幾個季度展現出超預期的盈利恢復和增長，並有效執行其轉型策略。
