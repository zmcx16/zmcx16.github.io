### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間：** 5 年 (2026 - 2030)
*   **折現率 (WACC)：** 6.95%
    *   Cost of Equity (Ke) = Risk-Free Rate + Beta \* ERP = 3.95% + 0.77 \* 4.18% = 3.95% + 3.22% = 7.17%
    *   Cost of Debt (Kd) = 4.74%
    *   稅率：由於 TTM 稅率異常高，我們採用更合理的長期預期稅率，例如 25%。
    *   債務權益比 (D/E) = 2.33 (來自您提供的數據和搜尋結果)
    *   權益比重 = 1 / (1 + D/E) = 1 / (1 + 2.33) = 0.30
    *   債務比重 = D/E / (1 + D/E) = 2.33 / (1 + 2.33) = 0.70
    *   WACC = (權益比重 \* Ke) + (債務比重 \* Kd \* (1 - 稅率))
        *   WACC = (0.30 \* 7.17%) + (0.70 \* 4.74% \* (1 - 0.25)) = 2.15% + (0.70 \* 4.74% \* 0.75) = 2.15% + 2.49% = 4.64%
        *   **注意：** Alpha Spread 提供的 WACC 為 6.95%，其權重可能與我基於 D/E 計算的不同，或者使用了不同的市場價值權重。為了保持一致性，我將直接使用 Alpha Spread 提供的 6.95% 作為 WACC。
*   **長期增長率 (Terminal Growth Rate, g)：** 2.5% (略高於長期通膨率，反映成熟公司的可持續增長)
*   **營收增長率：**
    *   2026財年預計銷售額增長 3-5%，有機銷售額增長 1-3%。
    *   考慮到公司正在執行「Beauty Reimagined」策略，並預期未來幾年盈利能力改善，我們假設：
        *   2026年：4% (基於財測中位數)
        *   2027年：5%
        *   2028年：4%
        *   2029年：3.5%
        *   2030年：3% (逐步趨近長期增長率)
*   **營業利益率 (Operating Margin)：**
    *   最新調整後營業利益率為 14.4%。
    *   考慮到公司正在執行「Profit Recovery Plan」以恢復利潤，我們假設營業利益率逐步改善並穩定：
        *   2026年：10% (保守估計，考慮到全年財測和投資)
        *   2027年：12%
        *   2028年：13%
        *   2029年：14%
        *   2030年：14.5%
*   **稅率：** 25% (長期有效稅率)
*   **資本支出 (CapEx)：** 佔營收的 3.5% (基於歷史數據和產業平均)
*   **折舊與攤銷 (D&A)：** 佔營收的 2.5% (基於歷史數據和產業平均)
*   **營運資本變動 (Change in NWC)：** 佔營收的 1% (假設營運效率穩定)

**DCF 計算步驟：**

1.  **計算預測期內的營收 (Revenue)。**
    *   TTM Revenue (2025財年假設) = $14.67 億美元
2.  **計算預測期內的營業利益 (EBIT)。**
    *   EBIT = Revenue \* Operating Margin
3.  **計算稅後營業利益 (NOPAT)。**
    *   NOPAT = EBIT \* (1 - Tax Rate)
4.  **計算自由現金流 (FCFF)。**
    *   FCFF = NOPAT + D&A - CapEx - Change in NWC
5.  **計算終端價值 (Terminal Value, TV)。**
    *   TV = FCFF_2030 \* (1 + g) / (WACC - g)
6.  **將預測期內的 FCFF 和 TV 折現至現值。**
7.  **加總所有現值，得到企業價值 (Enterprise Value, EV)。**
8.  **從 EV 減去淨債務 (Net Debt)，得到股權價值 (Equity Value)。**
    *   淨債務 = 總債務 - 現金。EL 有 $30.8 億美元現金和 $93.9 億美元債務，淨債務為 $63.1 億美元。
9.  **將股權價值除以流通股數，得到每股估值。**
    *   流通股數 = 3.6173 億股 (基於市值 $30.97 億 / 股價 $88.76)
