### **2. 加權平均資本成本 (WACC) 計算**

**1. 股權成本 (Cost of Equity, Ke) - 資本資產定價模型 (CAPM)：**
*   $Ke = Rf + \beta \times MRP$
*   $Ke = 4.40\% + 1.15 \times 5.50\% = 4.40\% + 6.325\% = 10.725\%$

**2. 債務成本 (Cost of Debt, Kd) - 稅後：**
*   $Kd_{\text{after-tax}} = \text{Pre-tax Cost of Debt} \times (1 - \text{Tax Rate})$
*   $Kd_{\text{after-tax}} = 6.00\% \times (1 - 0.23) = 6.00\% \times 0.77 = 4.62\%$

**3. 資本結構權重：**
*   市場股權價值 (E): $42.60 B
*   市場債務價值 (D): $6.80 B
*   總資本價值 (V = E + D): $42.60 B + $6.80 B = $49.40 B
*   股權權重 (We): $42.60 B / $49.40 B = 0.862$
*   債務權重 (Wd): $6.80 B / $49.40 B = 0.138$

**4. WACC 計算：**
*   $WACC = (We \times Ke) + (Wd \times Kd_{\text{after-tax}})$
*   $WACC = (0.862 \times 10.725\%) + (0.138 \times 4.62\%)$
*   $WACC = 9.244\% + 0.637\% = 9.881\%$
*   **取整 WACC ≈ 9.90%**

### **3. 估值模型分析**

#### **3.1 折現現金流模型 (Discounted Cash Flow, DCF)**

**基本假設：**
*   **預測期間:** 5 年 (FY2025 - FY2029)。
*   **基期營收 (FY2024估計):** $15.70 B (參考分析師共識預期，比TTM略高，預期緩慢復甦)。
*   **營收增長率 (Revenue Growth Rate):** 雅詩蘭黛近期受中國市場和旅遊零售影響較大，預期未來會逐步復甦，但不會回到高速增長。
    *   FY2025: 8.0% (強勁復甦)
    *   FY2026: 7.0%
    *   FY2027: 6.0%
    *   FY2028: 5.0%
    *   FY2029: 4.0%
*   **EBIT 利潤率 (EBIT Margin):** 近期EBIT利潤率較低 (~10.3%)，歷史上可達18-20%。預計未來幾年逐步恢復，但由於競爭加劇，可能難以回到歷史峰值。
    *   FY2025: 12.0%
    *   FY2026: 13.5%
    *   FY2027: 15.0%
    *   FY2028: 15.5%
    *   FY2029: 16.0%
*   **折舊與攤銷 (D&A):** 預計為營收的 4.50%。
*   **資本支出 (CAPEX):** 預計為營收的 4.50% (假設與 D&A 持平，反映成熟企業的再投資)。
*   **淨營運資金變化 (Change in NWC):** 假設淨營運資金佔營收的 10.0%。每年NWC的變化是當期NWC減去前一期NWC。

**自由現金流計算 (FCFF)：**

**終值 (Terminal Value) 計算：**
*   FCFF (FY29) = $2.51 B
*   長期增長率 (g) = 2.50%
*   $FCFF_{\text{FY30}} = FCFF_{\text{FY29}} \times (1 + g) = \$2.51 B \times (1 + 0.025) = \$2.57 B$
*   $TV_{\text{at end of FY29}} = FCFF_{\text{FY30}} / (WACC - g) = \$2.57 B / (0.099 - 0.025) = \$2.57 B / 0.074 = \$34.73 B$
*   **終值現值 (PV of TV):** $34.73 B / (1.099)^5 = \$21.75 B$

**總企業價值 (Enterprise Value) 計算：**
*   $EV = \text{PV of Explicit FCFFs} + \text{PV of Terminal Value}$
*   $EV = (\$1.31B + \$1.47B + \$1.59B + \$1.58B + \$1.56B) + \$21.75B$
*   $EV = \$7.51B + \$21.75B = \$29.26B$

**股權價值 (Equity Value) 和每股價值 (Per Share Value)：**
*   $Equity Value = EV - \text{Net Debt}$
*   $Equity Value = \$29.26B - \$2.80B = \$26.46B$
*   $Per Share Value = Equity Value / \text{Shares Outstanding}$
*   $Per Share Value = \$26.46B / 355M = \$74.54$

**DCF 估值結果與區間：**
在這些相對樂觀的復甦假設下，DCF 模型的估值約為 **$74.54 /股**。
**合理價位區間：$70.00 - $90.00** (考慮到營收增長率、EBIT 利潤率和 WACC 的敏感性分析)。

#### **3.3 股息折現模型 (Dividend Discount Model, DDM)**

EL 是一家成熟的、派發股息的公司，但其價值增長主要來自盈利增長而非單純的股息。因此，DDM 可能不是最佳的獨立估值方法，但可用作參考。

**基本假設：**
*   **最新年化股息 (D0):** $2.60 /股
*   **股權成本 (Ke):** 10.725% (來自 WACC 計算)
*   **股息增長率 (g_dividend):**
    *   鑑於EL的成長潛力，預計未來3年股息增長會較穩定，然後進入長期穩定增長。
    *   前三年 (g1): 5.00%
    *   長期穩定增長 (g_terminal): 2.50% (與DCF的長期增長率一致)

**多階段 DDM 計算：**
*   D1 = $2.60 * (1 + 0.05) = $2.73
*   D2 = $2.73 * (1 + 0.05) = $2.87
*   D3 = $2.87 * (1 + 0.05) = $3.01

*   **終值計算 (在第 3 年末):**
    *   D4 = D3 * (1 + g_terminal) = $3.01 * (1 + 0.025) = $3.085
    *   $Terminal Value_{\text{end of Year 3}} = D4 / (Ke - g_{\text{terminal}})$
    *   $Terminal Value_{\text{end of Year 3}} = \$3.085 / (0.10725 - 0.025) = \$3.085 / 0.08225 = \$37.51$

*   **計算所有現金流的現值：**
    *   PV(D1) = $2.73 / (1.10725)^1 = $2.47
    *   PV(D2) = $2.87 / (1.10725)^2 = $2.34
    *   PV(D3) = $3.01 / (1.10725)^3 = $2.20
    *   PV(Terminal Value) = $37.51 / (1.10725)^3 = $27.42

*   **DDM 每股價值：**
    *   $Total Value = PV(D1) + PV(D2) + PV(D3) + PV(Terminal Value)$
    *   $Total Value = \$2.47 + \$2.34 + \$2.20 + \$27.42 = \$34.43$

**DDM 估值結果與區間：**
DDM 模型的估值約為 **$34.43 /股**。
**合理價位區間：$30.00 - $45.00** (此模型對股息增長率和股權成本非常敏感)。

**比較不同模型間的估值結果：**

*   **DCF 估值**顯著低於當前市場價格 ($120) 和相對估值。這表明：
    *   市場對 EL 未來的營收增長和利潤率復甦抱有極高的期望，可能比本分析中的「樂觀但合理」的假設更為激進。
    *   或市場認為 EL 的 WACC 更低，或終端增長率更高。
    *   DCF 模型的結果可能是一個更為保守的「基本面」或「下限」估值，尤其是在公司經歷短期業績挑戰後，分析師和市場對其復甦路徑的看法存在較大差異時。
*   **相對估值 (Multiples)** 和 **P/Sales 倍數** 的結果與當前市場價格 ($120) 更為接近，尤其是使用前瞻性指標（如 FY25 Fwd P/E）時。這表明市場目前主要是根據 EL 相對於其同業的表現和預期復甦來定價的。由於 EL 擁有一系列高端品牌，其在正常營運下通常能享受品牌溢價，這反映在它相對較高的估值倍數上。
*   **DDM 估值**最低，這符合預期。EL 並非主要依賴股息來創造股東價值的公司，其大部分價值來自於再投資帶來的盈利增長和資本增值，因此 DDM 並不適合作為其主要估值方法。

**綜合結論：**

考量到雅詩蘭黛作為全球頂級美妝企業的品牌價值、市場地位以及其從近期挑戰中復甦的潛力，相對估值（尤其是基於前瞻性盈利和營收的倍數）提供了更貼近市場共識的觀點。儘管 DCF 顯示出一個更低的價值，但這可能是由於其嚴格的假設無法完全捕捉市場對 EL 強勁復甦的信心。

綜合各模型結果，並權衡 EL 的品牌力、市場地位及其復甦預期，我們認為其合理價位應更接近相對估值的範圍。

**綜合合理價位區間：$105.00 - $130.00**

EL 目前股價約 $120，位於此合理區間的中部。這表明市場已經在很大程度上消化了 EL 的復甦預期。如果公司能夠超出預期實現更快的營收增長或更高的利潤率，股價可能會有進一步的上漲空間；反之，若復甦不及預期，則存在下行風險。
