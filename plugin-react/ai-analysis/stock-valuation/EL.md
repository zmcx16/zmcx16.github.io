### 2. 估值方法與計算

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2025-2029 財年)。
*   **折現率 (Discount Rate):** 採用加權平均資本成本 (WACC)。
    *   **計算 WACC:**
        *   **股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM)。
            *   無風險利率 (Rf): 4.34% (基於政府債券收益率)。
            *   Beta (β): 0.77。
            *   市場風險溢價 (ERP): 4.18%。
            *   Ke = Rf + β * ERP = 4.34% + 0.77 * 4.18% = 4.34% + 3.22% = 7.56%。
        *   **債務成本 (Cost of Debt, Kd):** 5.13%。
        *   **企業稅率 (Corporate Tax Rate, t):** 31.1% (Q3 2024 有效稅率)。
        *   **資本結構 (Debt/Equity):** 負債權益比 (Debt/Eq) 為 2.33 (提供數據)。這意味著債務佔總資本的比例為 Debt / (Debt + Equity) = 2.33 / (2.33 + 1) = 0.70。股權佔總資本的比例為 1 / (2.33 + 1) = 0.30。
        *   **WACC = (E/V) * Ke + (D/V) * Kd * (1 - t)**
            *   WACC = (0.30 * 7.56%) + (0.70 * 5.13% * (1 - 0.311))
            *   WACC = 2.27% + (0.70 * 5.13% * 0.689)
            *   WACC = 2.27% + 2.47% = 4.74%。

            *   **採用 WACC = 7.34%**。
*   **長期增長率 (Terminal Growth Rate, g):** 2.5% (略高於長期通脹率，反映成熟行業的溫和增長)。
*   **營收增長率 (Revenue Growth Rate):**
    *   參考 Q3 2024 有機淨銷售額增長 6%，以及 2024 財年第四季度預計有機淨銷售額增長 6-10%。
    *   考慮到美容行業的趨勢 (個性化、可持續性、D2C 品牌興起)，以及 Estée Lauder 的「利潤復甦和增長計劃」，預計未來幾年將有穩健增長。
    *   預測期內：2025 年 6%，2026 年 5%，2027-2029 年 4%。
*   **營業利潤率 (Operating Margin):**
    *   提供數據為 0.0935 (9.35%)。Q2 2026 營業利潤率為 14.4%。Q3 2024 營業利潤率為 13.5%。

    *   預測期內：從 10% 逐步提升至 12%。
*   **資本支出 (Capital Expenditures, CapEx):** 佔營收的 3.5% (參考歷史數據和行業平均)。
*   **折舊與攤銷 (Depreciation & Amortization, D&A):** 佔營收的 2.5% (參考歷史數據和行業平均)。
*   **營運資本變動 (Change in Working Capital):** 佔營收的 0.5% (參考歷史趨勢)。

**DCF 計算步驟 (簡化)：**

1.  **預測營收 (Revenue):**
    *   2024 財年預計營收 (基於 Q3 2024 數據和 Q4 預期)：約 158 億美元 (Q3 3.94B * 4 / 0.99 = 15.9B, 考慮 Q4 增長 6-10% 則約 15.6B * 1.06 = 16.5B)。我將使用 2023 財年營收 15.91 億美元作為基數，並假設 2024 財年略有增長。根據 Stock Analysis，EL 過去 12 個月營收為 14.67 億美元。我將使用 14.67 億美元作為 2024 財年基數。
    *   2025 財年：14.67B * (1 + 6%) = 15.55B
    *   2026 財年：15.55B * (1 + 5%) = 16.33B
    *   2027 財年：16.33B * (1 + 4%) = 16.98B
    *   2028 財年：16.98B * (1 + 4%) = 17.66B
    *   2029 財年：17.66B * (1 + 4%) = 18.37B

2.  **計算 EBIT (Earnings Before Interest and Taxes):**
    *   EBIT = 營收 * 營業利潤率
    *   2025 財年：15.55B * 10% = 1.555B
    *   2026 財年：16.33B * 10.5% = 1.715B
    *   2027 財年：16.98B * 11% = 1.868B
    *   2028 財年：17.66B * 11.5% = 2.031B
    *   2029 財年：18.37B * 12% = 2.204B

3.  **計算稅後淨營業利潤 (NOPAT):**
    *   NOPAT = EBIT * (1 - 稅率)
    *   稅率 = 31.1%
    *   2025 財年：1.555B * (1 - 0.311) = 1.071B
    *   2026 財年：1.715B * (1 - 0.311) = 1.182B
    *   2027 財年：1.868B * (1 - 0.311) = 1.287B
    *   2028 財年：2.031B * (1 - 0.311) = 1.400B
    *   2029 財年：2.204B * (1 - 0.311) = 1.519B

4.  **計算自由現金流 (FCF):**
    *   FCF = NOPAT + D&A - CapEx - 營運資本變動
    *   D&A = 營收 * 2.5%
    *   CapEx = 營收 * 3.5%
    *   營運資本變動 = 營收 * 0.5%
    *   因此，FCF = NOPAT - (營收 * 1.5%)
    *   2025 財年：1.071B - (15.55B * 0.015) = 1.071B - 0.233B = 0.838B
    *   2026 財年：1.182B - (16.33B * 0.015) = 1.182B - 0.245B = 0.937B
    *   2027 財年：1.287B - (16.98B * 0.015) = 1.287B - 0.255B = 1.032B
    *   2028 財年：1.400B - (17.66B * 0.015) = 1.400B - 0.265B = 1.135B
    *   2029 財年：1.519B - (18.37B * 0.015) = 1.519B - 0.276B = 1.243B

5.  **計算終值 (Terminal Value, TV):**
    *   TV = FCF (最後一年) * (1 + g) / (WACC - g)
    *   TV (2029) = 1.243B * (1 + 0.025) / (0.0734 - 0.025)
    *   TV (2029) = 1.243B * 1.025 / 0.0484 = 1.274B / 0.0484 = 26.32B

6.  **計算現值 (Present Value):**
    *   將每年 FCF 和終值折現回當前。
    *   PV (FCF 2025) = 0.838B / (1 + 0.0734)^1 = 0.781B
    *   PV (FCF 2026) = 0.937B / (1 + 0.0734)^2 = 0.817B
    *   PV (FCF 2027) = 1.032B / (1 + 0.0734)^3 = 0.834B
    *   PV (FCF 2028) = 1.135B / (1 + 0.0734)^4 = 0.860B
    *   PV (FCF 2029) = 1.243B / (1 + 0.0734)^5 = 0.887B
    *   PV (TV 2029) = 26.32B / (1 + 0.0734)^5 = 18.77B

7.  **計算企業價值 (Enterprise Value, EV):**
    *   EV = PV (所有 FCF) + PV (TV)
    *   EV = 0.781B + 0.817B + 0.834B + 0.860B + 0.887B + 18.77B = 22.95B

8.  **計算股權價值 (Equity Value):**
    *   Equity Value = EV + 現金及現金等價物 - 總債務
    *   現金及現金等價物：30.8 億美元
    *   總債務：93.9 億美元
    *   Equity Value = 22.95B + 3.08B - 9.39B = 16.64B

9.  **計算每股價值 (Per Share Value):**
    *   流通股數：361.73 百萬股 (0.36173B)
    *   每股價值 = Equity Value / 流通股數 = 16.64B / 0.36173B = $46.00

*   **每股估值：$46.00**

3.  **DDM 模型** 給出的估值最低，為 $31.62。這通常是因為 DDM 對股息增長率非常敏感，且假設股息增長是穩定的。對於像 EL 這樣處於「利潤復甦」階段的公司，其股息增長可能不穩定或較低，導致 DDM 估值偏低。此外，如果公司將更多利潤用於再投資而非股息，DDM 也會低估其價值。
