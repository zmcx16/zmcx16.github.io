#### 2.3. 股息折現模型 (Dividend Discount Model, DDM)

    *   PV(P5) = P5 / (1 + Ke)^5 = 23.60 / (1.099)^5 = 23.60 / 1.5938 = $14.81

    *   每股價值 = Sum(PV(D1-D5)) + PV(P5) = 6.02 + 14.81 = $20.83
