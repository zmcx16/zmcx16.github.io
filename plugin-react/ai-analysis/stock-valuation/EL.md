    *   EL 估值 (基於 EV/EBITDA) = (22.03 * EBITDA 14.76 億美元 + 淨債務 150 億美元) / 3.60 億股 = (325.1 億美元 + 150 億美元) / 3.60 億股 = 475.1 / 3.60 = **131.97 美元** (此處計算有誤，EV/EBITDA 估算的是企業價值，需要減去淨債務才能得到股權價值，再除以股數。如果使用平均 EV/EBITDA 22.03，則企業價值 = 22.03 * 14.76 億美元 = 325.1 億美元。股權價值 = 325.1 億美元 - 淨債務 150 億美元 = 175.1 億美元。每股估值 = 175.1 億美元 / 3.60 億股 = **48.64 美元**)。

**相對估值合理價位區間：** **45 - 70 美元**。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

Estée Lauder 支付股息 (Dividend % 為 0.0123)。我將使用戈登增長模型 (Gordon Growth Model) 進行估值，假設股息穩定增長。

**估值假設：**

*   **當前股價：** 113.73 美元
*   **股息率：** 0.0123 (1.23%)
*   **每股股息 (D0)：** 113.73 美元 * 0.0123 = 1.40 美元 (這是基於當前股價和股息率的估算，實際應查詢最新每股股息)

**獲取最新每股股息：**

*   **最新每股股息 (D0)：** 0.35 美元 (季度股息)，因此年化股息為 0.35 * 4 = 1.40 美元。
*   **股息增長率 (g)：** 過去 1 年股息增長率為 -46.97%，3 年為 -16.45%，5 年為 -0.56%，10 年為 3.85%。 由於近期股息增長為負，且公司正在重組，未來增長存在不確定性。我將採用一個保守的長期股息增長率，例如 **1.0%**。
*   **股權成本 (Ke)：** 10.325% (與 DCF 模型中的股權成本一致)

**DDM 估值計算 (Gordon Growth Model)：**
P = D1 / (Ke - g)
其中 D1 = D0 * (1 + g)
D1 = 1.40 美元 * (1 + 0.01) = 1.414 美元
P = 1.414 美元 / (0.10325 - 0.01)
P = 1.414 美元 / 0.09325
P = **15.16 美元**

**DDM 估值結果與合理價位區間：**
基於上述假設，DDM 模型得出的每股估值約為 **15.16 美元**。
**合理價位區間：** 考慮到股息增長率的敏感性，合理價位區間可能在 **10 - 20 美元** 之間。

**DDM 模型的局限性：**
*   高度依賴股息增長率的假設，而 Estée Lauder 近期股息增長不穩定。
*   對於處於重組期或盈利不穩定的公司，DDM 可能不適用或結果不準確。

3.  **當前市場價格 (113.73 美元) 遠高於所有估值模型的結果。** 這是一個重要的觀察結果。分析師的平均目標價約為 98.19 美元至 106.76 美元，也低於當前股價。這可能意味著市場對 Estée Lauder 的估值存在一定的泡沫，或者市場預期公司在「Beauty Reimagined」戰略和「利潤復甦和增長計劃」下能實現更強勁的復甦和增長，從而證明其高估值是合理的。

**綜合結論：**

基於本次分析，Estée Lauder (EL) 的當前股價 113.73 美元，從內在價值 (DCF 和 DDM) 和相對估值來看，似乎存在被高估的風險。儘管公司擁有強大的品牌組合和市場地位，且正在積極實施轉型戰略，但其估值倍數顯著高於行業平均水平，且內在價值模型得出的結果遠低於當前股價。
