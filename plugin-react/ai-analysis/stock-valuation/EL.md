    *   債務 = 股本價值 \* 債務股本比 (此處的債務股本比是基於賬面價值，而非市值。為簡化，我們假設債務約為股本價值的2.33倍，這是一個粗略估計，實際應從財報獲取總債務。)

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

EL支付股息 (Dividend %: 0.0124)。如果股息增長穩定且可預測，DDM是適用的。

**DDM 假設:**

*   **當前股息 (D0):** 股價 \* 股息率 = 113.02美元 \* 0.0124 = 1.40美元
*   **股息增長率 (g):** 考慮到公司近期盈利挑戰，但長期美妝市場穩定，我們假設：
    *   短期 (5年) 股息增長率: 3% (保守估計，低於EPS增長預期，反映公司可能優先用於再投資或償債)
    *   長期 (永續) 股息增長率: 2.0% (低於長期增長率，反映股息增長通常較為保守)
*   **股權成本 (Ke):** 9.7% (與DCF模型中的股權成本一致)

**DDM 計算 (兩階段模型):**

1.  **預測未來5年股息:**
    *   D1 = 1.40 \* (1 + 0.03) = 1.44美元
    *   D2 = 1.44 \* (1 + 0.03) = 1.48美元
    *   D3 = 1.48 \* (1 + 0.03) = 1.52美元
    *   D4 = 1.52 \* (1 + 0.03) = 1.57美元
    *   D5 = 1.57 \* (1 + 0.03) = 1.62美元

2.  **計算第5年末的終值 (Terminal Value of Dividends):**
    *   P5 = D6 / (Ke - g_long_term) = (D5 \* (1 + g_long_term)) / (Ke - g_long_term)
    *   P5 = (1.62 \* (1 + 0.02)) / (0.097 - 0.02)
    *   P5 = (1.62 \* 1.02) / 0.077
    *   P5 = 1.6524 / 0.077 = 21.46美元

3.  **折現所有股息和終值到現在:**
    *   PV(D1) = 1.44 / (1 + 0.097)^1 = 1.31美元
    *   PV(D2) = 1.48 / (1 + 0.097)^2 = 1.23美元
    *   PV(D3) = 1.52 / (1 + 0.097)^3 = 1.16美元
    *   PV(D4) = 1.57 / (1 + 0.097)^4 = 1.10美元
    *   PV(D5) = 1.62 / (1 + 0.097)^5 = 1.03美元
    *   PV(P5) = 21.46 / (1 + 0.097)^5 = 13.59美元

4.  **計算股價 (P0):**
    *   P0 = PV(D1) + PV(D2) + PV(D3) + PV(D4) + PV(D5) + PV(P5)
    *   P0 = 1.31 + 1.23 + 1.16 + 1.10 + 1.03 + 13.59 = 19.42美元

**DDM 估值結果:**
