#### **C. 股息折現模型 (Dividend Discount Model, DDM)**

SM Energy 支付固定股息，因此 DDM 適用。

**估值假設：**

1.  **當前股息 (D0)：** 2024 年第四季度起，固定股息為每年每股 0.80 美元。
2.  **股息增長率 (g)：**
    *   SM Energy 預計 2025 年自由現金流增長 40%，並將支持股息和債務削減。
    *   公司在 2024 年將股息提高了 20%。
    *   考慮到公司承諾回報股東，但能源行業的波動性，我將假設一個溫和的長期股息增長率。
    *   假設股息在未來 3 年 (2026-2028) 增長 **5%**，之後永續增長 **2%**。
3.  **股權成本 (Ke)：** 10.165% (來自 DCF 模型)。

**DDM 計算步驟：**

*   **預測期股息：**
    *   D1 (2026) = D0 \* (1 + 5%) = 0.80 \* 1.05 = 0.84 美元
    *   D2 (2027) = D1 \* (1 + 5%) = 0.84 \* 1.05 = 0.882 美元
    *   D3 (2028) = D2 \* (1 + 5%) = 0.882 \* 1.05 = 0.9261 美元
*   **終端股息 (D4)：**
    *   D4 (2029) = D3 \* (1 + 永續增長率) = 0.9261 \* 1.02 = 0.9446 美元
*   **終值 (Terminal Value, TV) (在預測期結束時的股價)：**
    *   TV = D4 / (Ke - 永續增長率) = 0.9446 / (0.10165 - 0.02) = 0.9446 / 0.08165 = 11.57 美元
*   **折現股息和終值：**

| 年度 | 股息 (美元) | 折現因子 (1/(1+Ke)^n) | 現值 (美元) |
| :--- | :---------- | :-------------------- | :---------- |
| 2026 | 0.84        | 1/(1+0.10165)^1 = 0.9077 | 0.762       |
| 2027 | 0.882       | 1/(1+0.10165)^2 = 0.8240 | 0.727       |
| 2028 | 0.9261      | 1/(1+0.10165)^3 = 0.7480 | 0.693       |
| **終值** | 11.57       | 1/(1+0.10165)^3 = 0.7480 | 8.655       |

*   **每股估值 = 預測期股息現值之和 + 終值現值**
    *   每股估值 = 0.762 + 0.727 + 0.693 + 8.655 = **10.837 美元**。

DDM 模型給出的每股估值為 **10.84 美元**。這個結果也遠低於當前股價，表明 DDM 模型可能不完全適用於 SM Energy，或者其股息增長率假設過於保守。對於成長型或資本密集型公司，股息可能無法完全反映其內在價值。
