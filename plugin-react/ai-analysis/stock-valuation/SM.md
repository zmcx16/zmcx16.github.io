**1. 加權平均資本成本 (Weighted Average Cost of Capital, WACC) 計算**

*   **股權成本 (Cost of Equity, Ke)：** 採用資本資產定價模型 (CAPM)。
    *   **Beta (SM Energy)：** 根據Morningstar數據，Beta為1.25。
    *   Ke = 無風險利率 + Beta * 市場風險溢價
    *   Ke = 4.25% + 1.25 * 5.5% = 4.25% + 6.875% = 11.125%

*   **債務成本 (Cost of Debt, Kd)：** 根據SM Energy公司債券的收益率，其債券收益率約在6.22%至6.56%之間。 我們取平均值6.4%。
    *   稅後債務成本 = Kd * (1 - 稅率) = 6.4% * (1 - 28.5%) = 6.4% * 0.715 = 4.576%

*   **資本結構：**
    *   市值 (E)：51.1億美元。
    *   債務 (D)：合併後SM Energy承擔了約50億美元的Civitas淨債務。 加上SM Energy自身的長期債務，我們需要一個更精確的總債務數字。根據提供的數據，Debt/Eq為0.58，LT Debt/Eq為0.49。
        *   如果股權市值為51.1億美元，則總債務 D = 51.1億美元 * 0.58 = 29.638億美元。
        *   然而，合併後債務顯著增加。考慮到合併帶來約50億美元的淨債務，我們將使用一個調整後的債務估計。假設合併後的總債務約為50億美元 (考慮到剝離資產的計劃，這是一個動態數字)。
        *   總資本 = E + D = 51.1億美元 + 50億美元 = 101.1億美元。
        *   股權權重 = E / (E + D) = 51.1 / 101.1 = 0.5054
        *   債務權重 = D / (E + D) = 50 / 101.1 = 0.4946

*   **WACC 計算：**

**4. 股息折現模型 (Dividend Discount Model, DDM)**

*   **股權成本 (Ke)：** 11.125% (來自WACC計算)

*   **DDM 公式 (戈登增長模型)：** 股價 = D0 * (1 + g) / (Ke - g)

    *   股價 = 0.80 * 1.03 / 0.08125
    *   股價 = 0.824 / 0.08125 = **10.14美元**
