#### 3. 股息折現模型 (Dividend Discount Model, DDM)

SM Energy 支付股息，因此 DDM 適用。公司已將年度固定股息政策提高了 10% 至每股 0.88 美元，按季度支付。

    *   長期穩定增長率 (永續增長率): 2.0% (與 DCF 的長期增長率一致)
*   股權成本 (Ke): 10.55% (與 DCF 模型一致)

**DDM 計算 (兩階段模型):**

**階段一：預測期股息 (3 年)**

*   D1 = D0 \* (1 + g1) = 0.88 \* (1 + 0.05) = 0.924 美元
*   D2 = D1 \* (1 + g1) = 0.924 \* (1 + 0.05) = 0.9702 美元
*   D3 = D2 \* (1 + g1) = 0.9702 \* (1 + 0.05) = 1.01871 美元
