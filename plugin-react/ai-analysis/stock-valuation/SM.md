2.  **預測 EBIT (息稅前利潤)：** 營收 \* 營業利潤率。
3.  **計算稅後 EBIT：** EBIT \* (1 - 稅率)。
4.  **計算自由現金流 (FCF)：** 稅後 EBIT + D&A - CapEx - 營運資本變動。
5.  **計算終值 (Terminal Value, TV)：** TV = (最後一年 FCF \* (1 + 永續增長率)) / (WACC - 永續增長率)。
6.  **折現 FCF 和 TV：** 使用 WACC 將所有現金流折現至當前。
7.  **計算企業價值 (Enterprise Value, EV)：** 折現 FCF 總和 + 折現 TV。
8.  **計算股權價值 (Equity Value)：** EV - 淨債務。
9.  **計算每股價值 (Per Share Value)：** 股權價值 / 總股數。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

SM Energy 支付季度股息 0.20 美元，即每年 0.80 美元。 股息收益率約為 5.32% (根據提供數據) 或 4.39% (根據最新資訊)。

**DDM 假設：**

*   **當前股息 (D0):** 0.80 美元 (每年)
*   **股息增長率 (g):** 考慮到公司預期 EPS 下降，以及能源行業的週期性，我將假設一個較低的長期股息增長率。
    *   短期 (未來 2 年): 0% (由於 EPS 預期下降，可能維持股息不變或小幅調整)
    *   長期 (永續): 1.0% (保守估計)
*   **股權成本 (Ke):** 12.45% (與 DCF 模型相同)

**DDM 計算 (戈登增長模型):**

*   **股價 (P0) = D1 / (Ke - g)**
    *   D1 = D0 \* (1 + g)
    *   由於短期增長率為 0%，D1 = 0.80 美元。
    *   P0 = 0.80 / (12.45% - 1.0%) = 0.80 / 0.1145 = **6.99 美元**

**DDM 估值結果：** 6.99 美元

**合理價位區間：** 考慮到股息增長率的敏感性，若長期增長率在 0.5% - 1.5% 之間，估值可能在 **6.60 美元 - 7.40 美元** 之間。

這個結果遠低於當前股價和分析師目標價，這表明 DDM 可能不完全適用於 SM Energy，或者市場對其未來股息增長有更高的預期，或者其價值主要來自於資本增值而非股息。能源公司的股息往往與商品價格高度相關，波動性較大，因此簡單的 DDM 可能無法捕捉其全部價值。
