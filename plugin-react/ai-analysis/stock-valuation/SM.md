**DCF 計算步驟：**

1.  **預測收入 (Revenue)**
2.  **預測營業利潤 (EBIT)** = 收入 * 營業利潤率
3.  **計算稅後營業利潤 (NOPAT)** = EBIT * (1 - 稅率)
4.  **計算自由現金流 (FCF)** = NOPAT + D&A - CapEx - 營運資本變動
5.  **計算終值 (Terminal Value, TV)** = (FCF_last_year * (1 + 長期增長率)) / (WACC - 長期增長率)
6.  **將 FCF 和 TV 折現回現值**
7.  **計算企業價值 (Enterprise Value, EV)** = 折現 FCF 總和 + 折現 TV
8.  **計算股權價值 (Equity Value)** = EV - 淨債務 (Net Debt)
9.  **計算每股價值 (Intrinsic Value per Share)** = 股權價值 / 流通股數

#### 2.3 股息折現模型（Dividend Discount Model, DDM）

**估值假設：**

*   **當前股息 (D0)：** 0.80 美元 (年化)
*   **股息增長率 (g)：** 考慮到公司預計 EPS 下降，以及油氣行業的波動性，我將採用一個保守的長期股息增長率。雖然公司有回購股票和降低債務的計劃，但 EPS 預計下降會影響未來股息增長。我將假設一個初始的 2% 增長率，並在長期趨於 1.5%。
*   **股權成本 (Ke)：** 採用 WACC 中的股權成本 11.9%。

**DDM 計算 (戈登增長模型)：**
*   P = D1 / (Ke - g)
*   D1 = D0 * (1 + g)

**計算步驟：**

1.  **短期股息增長：** 考慮到 EPS 預計下降，短期內股息增長可能有限。但公司有穩定的股息支付歷史。我將假設未來 5 年股息保持穩定或略有增長，然後進入永續增長階段。

    *   或者，更直接地使用戈登增長模型，但採用一個非常保守的增長率。

**簡化 DDM (戈登增長模型)：**
*   D0 = 0.80 美元
*   Ke = 11.9%
*   g = 1.5% (長期增長率)
