#### 2.1 折現現金流模型 (DCF)

DCF 模型需要預測自由現金流 (FCF)。

**計算步驟:**

1.  **預測營收 (Revenue)**
2.  **預測營業利潤 (EBIT)** = 營收 * 營業利潤率
3.  **計算稅後營業利潤 (NOPAT)** = EBIT * (1 - 稅率)
4.  **加上折舊與攤銷 (D&A)**
5.  **減去資本支出 (CapEx)**
6.  **減去營運資本變動 (Change in Working Capital)**
7.  **得出自由現金流 (FCF)** = NOPAT + D&A - CapEx - Change in Working Capital
8.  **計算終值 (Terminal Value, TV)** = (FCF_last_forecast_year * (1 + 長期增長率)) / (WACC - 長期增長率)
9.  **將預測期 FCF 和 TV 折現至現值**
10. **加總得出企業價值 (Enterprise Value)**
11. **減去淨債務 (Net Debt)**，加上非經營性資產，得出股權價值 (Equity Value)
12. **除以流通股數**，得出每股估值

#### 2.3 股息折現模型 (DDM)

STMicroelectronics 支付股息，因此 DDM 適用。

**DDM 假設:**

*   **當前年度股息 (D0):** 0.32 美元/股 (年股息)。
*   **股息增長率 (g):**
    *   過去三年平均股息增長率為 19.35%。 然而，考慮到公司近期盈利波動和半導體行業的週期性，如此高的增長率可能難以持續。
    *   我將採用兩階段 DDM：
        *   **第一階段 (高增長期，3 年):** 假設股息增長率為 8% (低於歷史，但考慮到未來盈利預期增長)。
        *   **第二階段 (永續增長期):** 採用 2.5% 的長期增長率 (與 DCF 終值增長率一致)。
*   **所需報酬率 (Required Rate of Return, Ke):** 我將使用 WACC 的成本權益部分，或直接使用 WACC 作為近似值。根據搜尋結果，Cost of Equity 為 9.25% 或 9.75%。 我將採用 9.5% 作為 Ke。

**DDM 估值計算:**

*   **第一階段股息預測:**
    *   D0 = 0.32 美元
    *   D1 (2026) = 0.32 * (1 + 0.08) = 0.3456 美元
    *   D2 (2027) = 0.3456 * (1 + 0.08) = 0.3732 美元
    *   D3 (2028) = 0.3732 * (1 + 0.08) = 0.4031 美元

*   **終值 (Terminal Value) 計算 (在 DDM 中稱為終端股價 P3):**
    *   P3 = D4 / (Ke - g_long_term)
    *   D4 = D3 * (1 + g_long_term) = 0.4031 * (1 + 0.025) = 0.4132 美元
    *   P3 = 0.4132 / (0.095 - 0.025) = 0.4132 / 0.07 = 5.90 美元

*   **折現至現值:**
    *   PV(D1) = 0.3456 / (1 + 0.095)^1 = 0.3456 / 1.095 = 0.3156 美元
    *   PV(D2) = 0.3732 / (1 + 0.095)^2 = 0.3732 / 1.199 = 0.3113 美元
    *   PV(D3) = 0.4031 / (1 + 0.095)^3 = 0.4031 / 1.313 = 0.3070 美元
    *   PV(P3) = 5.90 / (1 + 0.095)^3 = 5.90 / 1.313 = 4.4935 美元

*   **每股估值 (DDM)** = PV(D1) + PV(D2) + PV(D3) + PV(P3)
    *   每股估值 = 0.3156 + 0.3113 + 0.3070 + 4.4935 = **5.43 美元/股**

**DDM 估值結果分析:**

2.  **相對估值結果較為分散。** 基於 P/E 和 Forward P/E 的估值 (15.85 - 22.50 美元) 仍低於當前股價，但與 DCF 和 DDM 相比，差距較小。而基於 P/B 和 P/S 的估值則遠高於當前股價，這可能反映了 STM 相對於其資產和銷售額的低估，但也可能因為其盈利能力較弱，導致這些倍數的適用性降低。
3.  **市場預期與基本面存在差異。** 當前股價 26.06 美元和分析師目標價 29.96 美元，遠高於內在價值模型得出的結果。這可能反映了市場對 STM 未來增長潛力 (尤其是在電動車、SiC 和 IoT 領域) 的強烈信心，以及對其盈利能力和自由現金流在未來幾年內大幅改善的預期。其較低的 Forward P/E (23.17) 相對於當前 P/E (46.01) 也支持了這種預期。
4.  **行業週期性影響。** 半導體行業具有強烈的週期性。STM 在 2025 年經歷了下行週期，Q1 表現疲軟，但 Q2 和 Q3 顯示出復甦跡象，並預計 2026 年將有顯著改善。 市場可能正在提前定價這種復甦。

**與同產業可比公司估值比較 (簡要):**

*   STMicroelectronics 的 P/E (46.01) 高於許多傳統半導體公司 (如 Analog Devices, Microchip Technology, NXP Semiconductors, ON Semiconductor, Texas Instruments 的 P/E 估計在 20-40 之間)。這表明市場對其當前盈利的估值較高，或者其當前盈利較低。
*   其 P/B (1.3) 和 P/S (1.98) 相對較低，這可能暗示其資產和銷售額的估值較低，或者其資產和銷售額的盈利效率不高 (與其低 ROE 相符)。
*   與競爭對手 Analog Devices 相比，STM 的淨利潤率 (4.55%) 低於 ADI (20.58%)，ROE 也低於 ADI (3.0% vs 11.23%)。 這解釋了為什麼在 P/E 估值上，STM 的內在價值會顯得較低。

**最終結論:**

基於內在價值模型 (DCF 和 DDM)，STMicroelectronics (STM) 的當前股價 26.06 美元似乎被高估。這兩個模型得出的合理價位區間在 5.00 美元至 15.00 美元之間。相對估值法 (P/E 和 Forward P/E) 也傾向於略低於當前股價的估值，約在 20.00 美元至 35.00 美元之間。

然而，市場對 STM 的估值顯然包含了對其未來增長潛力 (特別是在電動車、SiC 和 IoT 等高增長領域) 以及半導體行業復甦的強烈預期。公司在 Q3 2025 財報中也顯示出復甦跡象，並預計 Q4 和 2026 年將持續改善。

**綜合來看，STM 的估值存在較大的不確定性。** 儘管內在價值模型顯示其被高估，但如果公司能夠成功執行其戰略，實現預期的盈利能力和自由現金流增長，特別是在高利潤的汽車和工業市場，那麼其當前估值可能在長期內得到支撐。
