### 二、估值方法與計算

#### 1. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **預測期 (Explicit Forecast Period):** 5 年
*   **折現率 (WACC - Weighted Average Cost of Capital):**
    *   **無風險利率 (Risk-Free Rate, Rf):** 採用美國 10 年期國債收益率，約 4.2% (截至近期查詢)。
    *   **股權風險溢價 (Equity Risk Premium, ERP):** 採用行業標準，5.5%。
    *   **Beta (β):** 根據網路查詢，BLD 的 Beta 值約為 1.4 (考慮其行業週期性但又具備一定規模和穩定性)。
    *   **股權成本 (Cost of Equity, Ke) = Rf + β * ERP**
        *   Ke = 4.2% + 1.4 * 5.5% = 4.2% + 7.7% = 11.9%
    *   **稅率 (Tax Rate, t):** 根據公司財報和美國企業稅率，假設為 24%。
    *   **債務成本 (Cost of Debt, Kd):** 考慮公司負債水平和市場利率，假設為 6.5%。
    *   **市值 (E):** $12.02 Billion
    *   **總債務 (D):** 根據 Debt/Eq 1.39 和市值，以及網路查詢，假設總債務約為 $4.0 Billion (市場價值通常近似於賬面價值)。
    *   **WACC = (E / (E+D)) * Ke + (D / (E+D)) * Kd * (1 - t)**
        *   WACC = ($12.02B / ($12.02B + $4.0B)) * 11.9% + ($4.0B / ($12.02B + $4.0B)) * 6.5% * (1 - 0.24)
        *   WACC = (0.750) * 11.9% + (0.250) * 6.5% * 0.76
        *   WACC = 8.925% + 1.235% = 10.16% ≈ **10.2%**
*   **自由現金流 (Free Cash Flow, FCF) 預測：**
    *   **當前 FCF (TTM):** 根據 P/FCF = 15.19 和 Market Cap = $12.02B，估計 FCF (TTM) = $12.02B / 15.19 = $0.791 Billion (約 $791 Million)。
    *   **增長率：** 考慮到今年 EPS 負增長但明年預期轉正，以及行業的週期性，我們採取一個相對溫和但逐步恢復的增長路徑。
        *   Year 1: 10% (從低基數恢復)
        *   Year 2: 8%
        *   Year 3: 6%
        *   Year 4: 4%
        *   Year 5: 3%
*   **永續增長率 (Terminal Growth Rate, g):** 2.5% (略高於長期通膨和美國 GDP 增長率，反映公司作為行業領導者的潛力)。

**DCF 計算步驟：**

1.  **預測未來 5 年的 FCF：**
    *   FCF0 (TTM) = $791M
    *   FCF1 = $791M * (1 + 0.10) = $870.1M
    *   FCF2 = $870.1M * (1 + 0.08) = $939.7M
    *   FCF3 = $939.7M * (1 + 0.06) = $996.1M
    *   FCF4 = $996.1M * (1 + 0.04) = $1,035.9M
    *   FCF5 = $1,035.9M * (1 + 0.03) = $1,067.0M

2.  **計算終端價值 (Terminal Value, TV)：**
    *   TV = FCF5 * (1 + g) / (WACC - g)
    *   TV = $1,067.0M * (1 + 0.025) / (0.102 - 0.025)
    *   TV = $1,067.0M * 1.025 / 0.077 = $1,093.7M / 0.077 = $14,190.9M (約 $14.19 Billion)

3.  **將 FCF 和 TV 折現至現值：**
    *   PV(FCF1) = $870.1M / (1.102)^1 = $789.56M
    *   PV(FCF2) = $939.7M / (1.102)^2 = $770.80M
    *   PV(FCF3) = $996.1M / (1.102)^3 = $740.80M
    *   PV(FCF4) = $1,035.9M / (1.102)^4 = $699.00M
    *   PV(FCF5) = $1,067.0M / (1.102)^5 = $656.00M
    *   PV(TV) = $14,190.9M / (1.102)^5 = $8,728.00M

4.  **計算企業價值 (Enterprise Value, EV)：**
    *   EV = PV(FCF1) + PV(FCF2) + PV(FCF3) + PV(FCF4) + PV(FCF5) + PV(TV)
    *   EV = $789.56M + $770.80M + $740.80M + $699.00M + $656.00M + $8,728.00M = $12,384.16M (約 $12.38 Billion)

5.  **計算股權價值 (Equity Value)：**
    *   股權價值 = EV + 現金及約當現金 - 總債務
    *   根據網路查詢，假設 BLD 現金約為 $150M。
    *   股權價值 = $12.38B + $0.15B - $4.0B = $8.53 Billion

6.  **計算每股內在價值 (Intrinsic Value per Share)：**
    *   流通股數 (Shares Outstanding) = 市值 / 股價 = $12.02B / $431.92 ≈ 27.83 Million
