#### **3.1 折現現金流模型 (Discounted Cash Flow, DCF)**

**估值假設：**

*   **預測期間：** 5年 (2026-2030)
*   **折現率 (WACC)：**

    *   **市場風險溢價 (MRP):** 假設為 5.5%。
    *   **Beta (β):** 0.67x。
    *   **股權成本 (Ke) = Rf + β * MRP** = 4.2% + 0.67 * 5.5% = 4.2% + 3.685% = 7.885%

    *   **負債權益比 (D/E):** 0.38x。
    *   **WACC = (E/(D+E)) * Ke + (D/(D+E)) * Kd * (1 - Tax Rate)**
        *   E/(D+E) = 1 / (1 + D/E) = 1 / (1 + 0.38) = 1 / 1.38 ≈ 0.7246
        *   D/(D+E) = D/E / (1 + D/E) = 0.38 / 1.38 ≈ 0.2754
        *   WACC = 0.7246 * 7.885% + 0.2754 * 6% * (1 - 0.25)
        *   WACC = 0.0571 + 0.2754 * 0.06 * 0.75 = 0.0571 + 0.0124 = 0.0695 = **6.95%** (約 7.0%)
*   **長期增長率 (Terminal Growth Rate, g):** 假設為 2.0% (略低於長期通脹和全球經濟增長率)。
