*   **股權成本 (Ke):** 12.25% (同 DCF 模型)。

**B. 估值計算 (Gordon Growth Model):**

*   P0 = D0 * (1 + g) / (Ke - g)

*   P0 = 0.20 美元 * 1.03 / 0.0925
*   P0 = 0.206 美元 / 0.0925 ≈ 2.23 美元/股

**DDM 估值結果：** DDM 模型給出的估值非常低，約 **2.23 美元/股**。這表明 U.S. Steel 的股息支付相對於其盈利能力和股權成本而言，並不是其價值的主要驅動力，或者其股息增長率遠低於市場預期。對於成長性或週期性較強的公司，DDM 往往不是最合適的估值方法。
