**DCF 計算步驟:**

1.  **預測營收 (Revenue)**
2.  **預測 EBIT (Earnings Before Interest and Taxes) = Revenue \* Operating Margin**
3.  **計算稅後 EBIT (NOPAT) = EBIT \* (1 - Tax Rate)**
4.  **計算自由現金流 (FCF) = NOPAT + D&A - CapEx - Change in NWC**
5.  **計算終值 (Terminal Value, TV) = FCF(最後一年) \* (1 + 長期增長率) / (WACC - 長期增長率)**
6.  **將所有 FCF 和 TV 折現回現值。**
7.  **計算企業價值 (Enterprise Value, EV) = 所有 FCF 現值 + TV 現值**
8.  **計算股權價值 (Equity Value) = EV + 現金 - 債務**
9.  **計算每股價值 (Intrinsic Value per Share) = Equity Value / 流通股數**
