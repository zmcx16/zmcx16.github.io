### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來現金流的現值。

**估值假設:**

*   **預測期間 (Explicit Forecast Period):** 5 年 (2026-2030)
*   **長期增長率 (Terminal Growth Rate):** 3% (略高於長期通膨和全球 GDP 增長，考慮到糖尿病管理市場的持續增長和 Insulet 的創新能力，但保守估計)
*   **折現率 (Discount Rate / WACC):**
    *   **無風險利率 (Risk-Free Rate):** 4.5% (參考美國 10 年期國債收益率，假設當前市場利率)
    *   **市場風險溢價 (Market Risk Premium):** 5.5% (行業常用範圍)
    *   **Beta (β):** 1.2 (醫療設備行業通常較高，反映其成長性和一定波動性)
    *   **股權成本 (Cost of Equity, Ke) = 無風險利率 + Beta * 市場風險溢價**
        *   Ke = 4.5% + 1.2 * 5.5% = 4.5% + 6.6% = 11.1%
    *   **債務成本 (Cost of Debt, Kd):** 6.0% (假設公司債務利率)
    *   **稅率 (Tax Rate):** 21% (美國企業稅率)
    *   **債務稅後成本 = Kd * (1 - Tax Rate)** = 6.0% * (1 - 0.21) = 4.74%
    *   **股權市值 (Market Value of Equity, E):** 10.21 億美元 (Market Cap)
    *   **債務市值 (Market Value of Debt, D):** 根據 Debt/Eq = 0.73，D = E * 0.73 = 10.21 億 * 0.73 = 7.4533 億美元
    *   **WACC = (E / (E + D)) * Ke + (D / (E + D)) * Kd * (1 - Tax Rate)**
        *   WACC = (10.21 / (10.21 + 7.4533)) * 11.1% + (7.4533 / (10.21 + 7.4533)) * 4.74%
        *   WACC = (10.21 / 17.6633) * 11.1% + (7.4533 / 17.6633) * 4.74%
        *   WACC = 0.577 * 11.1% + 0.423 * 4.74% = 6.40% + 2.00% = 8.40%

    *   2030 年: 10%

*   **折舊與攤銷 (Depreciation & Amortization, D&A) / 營收:** 3% (假設)

#### 2.2 相對估值 (Comparable Company Analysis, CCA)

相對估值法通過比較 PODD 與其同業公司的估值倍數來評估其價值。

**可比公司 (Competitors):** DexCom (DXCM), Tandem Diabetes Care (TNDM), Medtronic (MDT) (雖然 Medtronic 業務更廣泛，但其糖尿病部門是直接競爭對手，且有剝離計劃)

**選取估值倍數:**
*   **P/E (Price/Earnings):** 衡量每股收益的價格。
*   **Forward P/E (Forward Price/Earnings):** 衡量未來一年預期每股收益的價格。
*   **P/S (Price/Sales):** 衡量每股銷售額的價格，適用於成長型公司或盈利不穩定的公司。
*   **P/FCF (Price/Free Cash Flow):** 衡量每股自由現金流的價格。
*   **EV/EBITDA (Enterprise Value/Earnings Before Interest, Taxes, Depreciation, and Amortization):** 衡量企業總價值與息稅折舊攤銷前利潤的比率，適用於不同資本結構的公司。
*   **PEG (Price/Earnings to Growth):** 衡量 P/E 與預期收益增長率的比率，用於評估成長型股票的估值。

#### 2.4 其他合理估值方法 (PEG Ratio 分析)

PEG Ratio (Price/Earnings to Growth Ratio) 是一個常用的指標，用於評估成長型股票的估值。PEG < 1 通常被認為是相對低估，PEG > 1 則可能被認為是相對高估。

*   **PODD 的 PEG:** 1.21
*   **DXCM 的 PEG:** 1.06
*   **MDT 的 PEG:** 1.83

PODD 的 PEG (1.21) 介於 DXCM (1.06) 和 MDT (1.83) 之間，且略高於 1。這表明市場對其增長預期已經有所反映。與 DXCM 相比，PODD 的 PEG 略高，但考慮到其 Q1 2026 營收增長 33.9% 和 EPS 增長 24.22%，其成長性依然強勁。如果其增長能夠持續，目前的 PEG 可能是合理的。
