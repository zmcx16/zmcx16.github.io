### **2. 估值假設**

在進行估值之前，我們需要設定一些關鍵假設。

**A. 加權平均資本成本 (WACC) 計算**

WACC 是 DCF 模型中的折現率，代表公司為其資產融資的平均成本。

*   **1. 無風險利率 (Risk-Free Rate):** 參考美國10年期國債收益率。假設為 **4.5%**。
*   **2. 市場風險溢價 (Market Risk Premium, MRP):** 歷史平均值，通常在 5% - 6% 之間。假設為 **5.5%**。
*   **3. Beta 值 (Beta):** 反映股票相對於市場的波動性。對於政府承包商，Beta 通常接近1。透過網路搜尋，SAIC 的 Beta 值約為 **1.0**。
*   **4. 股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM)。
    *   Ke = 無風險利率 + Beta * 市場風險溢價
    *   Ke = 4.5% + 1.0 * 5.5% = **10.0%**
*   **5. 債務成本 (Cost of Debt, Kd):** 參考公司債券收益率或類似信用評級公司的平均借款利率。假設稅前債務成本為 **6.0%**。
*   **6. 稅率 (Tax Rate):** 參考公司有效稅率。假設為 **23%** (基於美國企業稅率和公司歷史數據)。
    *   稅後債務成本 = 稅前債務成本 * (1 - 稅率)
    *   稅後 Kd = 6.0% * (1 - 0.23) = **4.62%**
*   **7. 資本結構 (Capital Structure):**
    *   市值 (Equity Value) = $4.64 Billion
    *   總債務 (Total Debt): 根據 Debt/Eq = 1.75，總債務 = 1.75 * $4.64B = $8.12 Billion
    *   總資本 (Total Capital) = 股權 + 債務 = $4.64B + $8.12B = $12.76 Billion
    *   股權權重 (We) = $4.64B / $12.76B = **0.3636**
    *   債務權重 (Wd) = $8.12B / $12.76B = **0.6364**
*   **8. WACC 計算:**
    *   WACC = (We * Ke) + (Wd * 稅後 Kd)
    *   WACC = (0.3636 * 10.0%) + (0.6364 * 4.62%)
    *   WACC = 3.636% + 2.940% = **6.576%**
    *   為簡化計算，我們將 WACC 四捨五入為 **6.6%**。

**B. 自由現金流 (FCF) 增長率假設**

*   **預測期間:** 5 年 (2024-2028)。
*   **短期增長率 (未來5年):** 考慮到 SAIC 提供的 "EPS next Y_%" (-4.69%) 和 "Sales Q/Q" (-5.57%) 數據，短期內可能面臨壓力。但作為政府承包商，業務相對穩定，預計會逐步恢復。
    *   Year 1 (2024): **0%** (預計穩定，克服短期逆風)
    *   Year 2 (2025): **2%** (溫和恢復)
    *   Year 3-5 (2026-2028): **3%** (穩定增長，符合行業平均水平)
*   **永續增長率 (Terminal Growth Rate, g):** 預測期後，公司將以穩定速度增長。對於成熟的政府承包商，通常設定在 1.5% - 2.5% 之間，且必須低於 WACC。假設為 **2.5%**。

### **3. 估值模型與結果**

#### **模型一：折現現金流模型 (Discounted Cash Flow, DCF)**

**步驟 1: 計算當前自由現金流 (FCF0)**
*   我們從提供的數據中計算出 TTM FCF = $427.65 Million。

**步驟 2: 預測未來 5 年的自由現金流 (FCF)**
*   FCF0 = $427.65M
*   FCF1 (2024) = $427.65M * (1 + 0%) = $427.65M
*   FCF2 (2025) = $427.65M * (1 + 2%) = $436.20M
*   FCF3 (2026) = $436.20M * (1 + 3%) = $449.29M
*   FCF4 (2027) = $449.29M * (1 + 3%) = $462.77M
*   FCF5 (2028) = $462.77M * (1 + 3%) = $476.65M

**步驟 3: 計算預測期內 FCF 的現值**
*   PV(FCF1) = $427.65M / (1 + 0.066)^1 = $401.17M
*   PV(FCF2) = $436.20M / (1 + 0.066)^2 = $384.00M
*   PV(FCF3) = $449.29M / (1 + 0.066)^3 = $372.00M
*   PV(FCF4) = $462.77M / (1 + 0.066)^4 = $360.00M
*   PV(FCF5) = $476.65M / (1 + 0.066)^5 = $348.00M
*   **預測期 FCF 現值總和 = $1,865.17M**

**步驟 4: 計算終值 (Terminal Value, TV)**
*   TV = FCF5 * (1 + g) / (WACC - g)
*   TV = $476.65M * (1 + 0.025) / (0.066 - 0.025)
*   TV = $476.65M * 1.025 / 0.041
*   TV = $488.566M / 0.041 = $11,916.24M

**步驟 5: 計算終值的現值 (PV of TV)**
*   PV(TV) = TV / (1 + WACC)^5
*   PV(TV) = $11,916.24M / (1 + 0.066)^5 = $11,916.24M / 1.3765 = $8,656.90M

**步驟 6: 計算企業價值 (Enterprise Value, EV)**
*   EV = 預測期 FCF 現值總和 + 終值的現值
*   EV = $1,865.17M + $8,656.90M = $10,522.07M

**步驟 7: 計算股權價值 (Equity Value) 和每股價值**
*   股權價值 = EV + 現金及約當現金 - 總債務
    *   假設現金及約當現金為 $100M (需從財報中獲取精確數字，此為估計值)
    *   總債務 = $8.12 Billion (從 WACC 計算中得出)
*   股權價值 = $10,522.07M + $100M - $8,120M = $2,502.07M
*   每股價值 = 股權價值 / 流通股數
*   每股價值 = $2,502.07M / 45.17M = **$55.39**

**DCF 估值結果與合理價位區間：**
*   **DCF 估值：$55.39**
*   **合理價位區間：$50 - $65** (考慮到增長率和 WACC 假設的敏感性)
*   **分析：** 根據這些保守的增長假設和高負債，DCF 模型得出的估值遠低於當前股價。這表明市場可能對 SAIC 的未來增長和現金流產生更樂觀的預期，或者我的增長假設過於保守。高負債是壓低股權價值的一個重要因素。

#### **模型二：股息折現模型 (Dividend Discount Model, DDM)**

由於 SAIC 支付股息，DDM 適用。我們將使用戈登增長模型 (Gordon Growth Model)，假設股息以恆定速度增長。

*   **當前每股股息 (DPS0):** $1.48 (每年)
*   **股權成本 (Ke):** 10.0% (從 WACC 計算中得出)
*   **股息增長率 (g):** 考慮到 EPS 預期負增長，股息增長可能非常緩慢或持平。我們假設一個非常保守的長期股息增長率，例如 **1.5%** (必須低於 Ke)。

**計算：**
*   DPS1 = DPS0 * (1 + g) = $1.48 * (1 + 0.015) = $1.5022
*   每股價值 = DPS1 / (Ke - g)
*   每股價值 = $1.5022 / (0.100 - 0.015)
*   每股價值 = $1.5022 / 0.085 = **$17.67**

**DDM 估值結果與合理價位區間：**
*   **DDM 估值：$17.67**
*   **合理價位區間：$15 - $20** (對增長率和 Ke 極度敏感)
*   **分析：** DDM 模型得出的估值非常低，遠低於當前股價。這表明 SAIC 並非主要以股息增長為導向的股票，市場對其的估值更多基於其盈利能力和自由現金流潛力，而非當前的股息流。此模型在此情況下可能不具代表性。

**最終合理價位區間：**

*   **上限：** 參考分析師目標價，並考慮到如果公司能改善增長並縮小與同業的估值差距，股價有機會達到更高的水平。
