        *   股權市值 (E) = 市值 = $72.8 億美元

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

PAYC 於 2024 年底開始支付季度股息 $0.375/股。年化股息為 $0.375 * 4 = $1.50/股。

**估值假設:**

*   **當前股息 (D0):** $1.50/股
*   **股息增長率 (g):** 考慮到公司剛開始支付股息，且營收增長放緩，但現金流轉換良好，我們假設初始增長率較高，然後逐漸趨於穩定。
    *   短期 (未來 3 年) 股息增長率: 8% (略低於 EPS 增長率，考慮到公司可能保留部分利潤用於再投資)
    *   長期 (永續) 股息增長率: 3% (略高於長期經濟增長率)
*   **所需報酬率 (Required Rate of Return, Ke):** 9.7% (與 DCF 模型中的股權成本一致)

**兩階段 DDM 模型:**

**第一階段 (高速增長階段 - 3 年):**

*   D1 = D0 * (1 + g1) = $1.50 * (1 + 0.08) = $1.62
*   D2 = D1 * (1 + g1) = $1.62 * (1 + 0.08) = $1.75
*   D3 = D2 * (1 + g1) = $1.75 * (1 + 0.08) = $1.89

**第二階段 (永續增長階段):**

*   P3 (第三年末的股票價值) = D4 / (Ke - g2)
*   D4 = D3 * (1 + g2) = $1.89 * (1 + 0.03) = $1.95
*   P3 = $1.95 / (0.097 - 0.03) = $1.95 / 0.067 = $29.10

**現值計算:**

*   P0 = D1 / (1 + Ke)^1 + D2 / (1 + Ke)^2 + D3 / (1 + Ke)^3 + P3 / (1 + Ke)^3
*   P0 = $1.62 / (1.097)^1 + $1.75 / (1.097)^2 + $1.89 / (1.097)^3 + $29.10 / (1.097)^3
*   P0 = $1.62 / 1.097 + $1.75 / 1.2034 + $1.89 / 1.3199 + $29.10 / 1.3199
*   P0 = $1.476 + $1.454 + $1.432 + $22.047
*   P0 = **$26.41**

**DDM 模型估值結果與合理價位區間:**

*   **每股估值:** $26.41
*   **合理價位區間:** 考慮到 DDM 對增長率和所需報酬率的敏感性，以及 PAYC 剛開始支付股息，此模型可能不完全適用於快速增長的科技公司。此估值結果顯著低於當前股價和 DCF/相對估值結果，表明 PAYC 的價值主要來自於未來的增長潛力而非當前股息。因此，DDM 在此情況下可能不是最合適的估值方法，或者需要更激進的股息增長假設。
