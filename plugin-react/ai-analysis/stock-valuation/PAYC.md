## Paycom Software Inc. (PAYC) 股票估值分析報告

作為一名資深股權評價分析師，我將對美股公司 Paycom Software Inc. (PAYC) 進行全面的股票估值分析，並著重與其主要同業競爭對手進行橫向比較。

### 1. 同業比較分析 (Industry Peer Comparison)

**識別競爭對手**
根據市場搜尋結果，PAYC 的主要產業競爭對手包括：
*   **Automatic Data Processing (ADP)**：人力資本管理 (HCM) 領域的巨頭，提供全面的薪資和 HR 解決方案。
*   **Workday (WDAY)**：專注於雲端 HCM 和財務管理軟體，服務於大型企業。
*   **Paylocity (PCTY)**：提供中小型企業雲端薪資和 HCM 解決方案。
*   **Dayforce (CDAY)** (原 Ceridian HCM)：提供雲端 HCM 平台，涵蓋薪資、福利、勞動力管理等。

**估值地位分析**
PAYC 相較於同業，在估值上呈現以下特點：
*   **本益比 (P/E Ratio)**：PAYC 的 TTM P/E (15.94) 顯著低於 ADP (20.86) 和 Workday (48.89)，但 Dayforce 的 P/E 由於盈利不穩定而呈現極高或負值。這表明市場對 PAYC 的當前盈利給予了相對保守的估值，可能反映了其較低的預期增長率或市場對其未來盈利穩定性的擔憂。然而，其 Forward P/E (11.73) 則顯示出較強的盈利增長預期，使其相對於同業更具吸引力。
*   **股價營收比 (P/S Ratio)**：PAYC 的 P/S (3.18) 與 Workday (3.21) 相近，略低於 ADP (3.94)，但遠低於 Dayforce (6.32)。這表明 PAYC 在營收基礎上的估值相對合理，並未出現顯著溢價。
*   **企業價值倍數 (EV/EBITDA)**：PAYC 的 EV/EBITDA (約 7.39) 遠低於 Workday (19.14) 和 ADP (估計 14.50)，也低於 Dayforce (估計 12.00)。這強烈暗示 PAYC 相對於其營運現金流而言，目前處於折價狀態。這可能與其較低的營收增長預期有關，但也可能被市場低估了其盈利能力和效率。
*   **盈利能力**：PAYC 在營業利潤率 (36.75%) 和淨利潤率 (27.2%) 方面表現出色，顯著優於 Workday 和 Dayforce，甚至可能優於 ADP (需進一步確認 ADP 的精確利潤率)。這表明 PAYC 擁有高效的營運模式和強大的盈利能力。
*   **成長性**：PAYC 在 Q1 2026 的營收增長率為 7.8%，低於 Workday (18.1%) 和 Dayforce (16.25%)，但與 ADP (6.2%) 相近。這解釋了其較低的估值倍數，市場通常會給予高成長公司更高的估值溢價。

**總結**：PAYC 相對於其同業而言，在盈利能力方面表現優異，但成長性相對較低。其估值倍數（尤其是 P/E 和 EV/EBITDA）普遍低於高成長的 Workday 和 Dayforce，甚至低於更為成熟的 ADP。這表明 PAYC 目前可能被市場低估，尤其是在考慮到其強勁的利潤率和穩健的現金流生成能力時。市場可能尚未充分反映其在 HCM 解決方案自動化和效率提升方面的競爭優勢。

### 2. 折現現金流模型 (Discounted Cash Flow, DCF)

**預測假設**
*   **預測期間**：5 年 (2026-2030)。
*   **起始營收 (2025)**：$2.05B。
*   **營收增長率預期**：
    *   2026: 6.5% (基於公司指引中點 $2.175B - $2.195B)
    *   2027: 7.0% (預期略高於 2026，因市場對其 AI 解決方案和自動化策略的採用)
    *   2028: 6.0% (逐步趨緩)
    *   2029: 5.0%
    *   2030: 4.0%
*   **營業利潤率 (Operating Margin)**：
    *   2026: 37.0% (基於 Q1 2026 的 36.75% 並考慮全年指引)
    *   2027-2030: 逐步提升至 38.0% (考慮其自動化和效率提升的潛力)。
*   **稅率 (Tax Rate)**：24% (基於美國企業稅率及公司實際稅負考量)。
*   **資本支出 (Capital Expenditure, CapEx)**：預計為營收的 3.0% (基於歷史趨勢和維持增長所需投資)。
*   **折舊與攤銷 (Depreciation & Amortization, D&A)**：預計為營收的 5.0% (基於歷史趨勢)。
*   **營運資本變動 (Change in Working Capital)**：預計為營收的 0.5% (保守估計，假設營運效率穩定)。

**折現率與終值**
*   **無風險利率 (Risk-Free Rate)**：美國 10 年期國債收益率 4.48% (截至 2026 年 6 月 4 日)。
*   **市場風險溢酬 (Market Risk Premium)**：5.5%。
*   **Beta 值 (Beta)**：0.79。
*   **股權成本 (Cost of Equity, Ke)**：Ke = 無風險利率 + Beta * 市場風險溢酬 = 4.48% + 0.79 * 5.5% = 4.48% + 4.345% = 8.825%。
*   **債務成本 (Cost of Debt, Kd)**：PAYC 於 Q1 2026 提取了 $675M 的信貸額度。假設債務利率為 6.5%。稅後債務成本 = Kd * (1 - 稅率) = 6.5% * (1 - 24%) = 4.94%。
*   **債務權益比 (Debt-to-Equity Ratio)**：
    *   市值 (Equity Value) = $6.42B
    *   債務 (Debt) = $675M
    *   D/E = 0.675B / 6.42B = 0.105。
*   **加權平均資本成本 (Weighted Average Cost of Capital, WACC)**：
    *   WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1 - 稅率)
    *   WACC = (6.42 / (6.42 + 0.675)) * 8.825% + (0.675 / (6.42 + 0.675)) * 4.94%
    *   WACC = 0.905 * 8.825% + 0.095 * 4.94% = 7.99% + 0.47% = 8.46%。
*   **終值增長率 (Terminal Growth Rate, g)**：2.5% (保守估計，反映長期可持續增長)。

### 3. 多維度估值法 (Other Valuation Methods)

**相對估值法 (Relative Valuation)**
我們將使用同業的平均估值倍數來推算 PAYC 的目標價位。

*   **P/E Ratio (Forward)**：
    *   同業平均 Forward P/E (ADP, WDAY, CDAY 估計值)：(20.00 + 35.00 + 40.00) / 3 = 31.67
    *   PAYC 預期 EPS (基於 2026 年營收指引和利潤率估計，約 $9.50 - $10.00)
    *   若以 2026 年預期 EPS $9.80 計算，目標價 = 9.80 * 31.67 = $310.37
    *   考慮到 PAYC 成長性較低，給予適當折價，採用同業中位數 P/E 20.00 (ADP) 作為參考。
    *   目標價 = 9.80 * 20.00 = **$196.00**

*   **P/S Ratio (TTM)**：
    *   同業平均 P/S (ADP, WDAY, CDAY)：(3.94 + 3.21 + 6.32) / 3 = 4.49
    *   PAYC TTM 營收 (2025 年 $2.05B)
    *   PAYC 稀釋後流通股數 51M
    *   PAYC TTM 每股營收 = 2050 / 51 = $40.20
    *   目標價 = 40.20 * 4.49 = **$180.49**

*   **EV/EBITDA (TTM)**：
    *   同業平均 EV/EBITDA (WDAY, ADP, CDAY 估計值)：(19.14 + 14.50 + 12.00) / 3 = 15.21
    *   PAYC 2025 年 EBITDA = $882M
    *   PAYC 企業價值 = 882 * 15.21 = 13,415 百萬美元
    *   股權價值 = 13,415 + 154 - 675 = 12,894 百萬美元
    *   目標價 = 12,894 / 51 = **$252.82**

綜合相對估值法，考慮到 PAYC 的盈利能力和相對較低的成長性，其合理價位區間約為 **$180 - $250**。

**股息折現模型 (Dividend Discount Model, DDM)**
PAYC 的股息率為 1.09%，季度股息為 $0.38。雖然公司有派發股息，但其股息收益率相對較低，且公司在 Q1 2026 進行了大規模的股票回購 ($1.06B)，顯示其資本配置策略更傾向於回購而非高額股息。因此，DDM 並非評估 PAYC 內在價值的最合適模型。
