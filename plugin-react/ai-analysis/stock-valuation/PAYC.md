NOPAT = 營業收入 \* (1 - 稅率)

對於成長型公司，股息折現模型通常不是最主要的估值方法，因為它們傾向於將大部分收益再投資於業務以實現增長，而非支付高額股息。然而，我們可以嘗試使用兩階段 DDM。

**假設：**
*   當前年度股息 (D0): 1.50 美元 (0.375 \* 4)
*   股息增長率 (g1) (前 5 年): 考慮到公司盈利增長和再投資需求，我們假設股息在前 5 年以 8% 的速度增長。
*   永續股息增長率 (g2) (永續期): 2.5% (與 DCF 的長期增長率一致)
*   股權成本 (Ke): 8.545% (與 DCF 的 WACC 相似)

**計算：**

| 年度 | Dt (股息) | 折現因子 (Ke=8.545%) | 折現股息 |
| :--- | :--- | :--- | :--- |
| 1 (2024) | 1.50 \* (1+0.08) = 1.62 | 0.9213 | 1.49 |
| 2 (2025) | 1.62 \* (1+0.08) = 1.75 | 0.8488 | 1.48 |
| 3 (2026) | 1.75 \* (1+0.08) = 1.89 | 0.7820 | 1.48 |
| 4 (2027) | 1.89 \* (1+0.08) = 2.04 | 0.7199 | 1.47 |
| 5 (2028) | 2.04 \* (1+0.08) = 2.20 | 0.6632 | 1.46 |

**終值 (Terminal Value) 股息：**
D6 = D5 \* (1 + g2) = 2.20 \* (1 + 0.025) = 2.255 美元
TV (DDM) = D6 / (Ke - g2) = 2.255 / (0.08545 - 0.025) = 2.255 / 0.06045 = 37.30 美元

**折現終值：**
折現 TV = TV (DDM) \* 折現因子 (第 5 年) = 37.30 \* 0.6632 = 24.75 美元

**每股估值 = 折現股息之和 + 折現終值**
每股估值 = 1.49 + 1.48 + 1.48 + 1.47 + 1.46 + 24.75 = **32.13 美元**

**DDM 模型合理價位區間：30 - 35 美元**

這個結果顯著低於當前股價和 DCF/CCA 估值。這再次印證了 DDM 對於成長型公司可能不是最合適的估值方法，因為其股息政策可能無法完全反映其內在價值和盈利能力。公司將大部分利潤再投資於業務以實現高增長，這在 DDM 中沒有得到充分體現。

### **4. 綜合結論與合理價位區間**
