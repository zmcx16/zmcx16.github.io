### 估值方法一：折現現金流模型 (Discounted Cash Flow, DCF)

**DCF 模型的挑戰與調整 (Challenges & Adjustments for REITs):**
對於 REITs 而言，標準的自由現金流 (FCFF 或 FCFE) 模型較難直接應用，因為其折舊攤銷很高，資本支出主要用於物業收購而非傳統的營運開支，且通常透過債務和股權融資進行成長。此外，MPW 目前處於出售資產、應對租戶困境的特殊時期，導致未來現金流高度不確定。儘管如此，為滿足要求，我們將嘗試建構一個簡化的、基於調整後營運現金流的 FCFF 模型，並明確指出其限制。

**估值假設 (Valuation Assumptions):**

1.  **預測期 (Explicit Forecast Period):** 5 年 (2024 - 2028)
2.  **營收增長率 (Revenue Growth Rate - Rental Income):**
    *   2024: -5% (預計受資產出售和租金減免影響)
    *   2025: 0% (穩定化階段)
    *   2026-2028: 1.0% (非常保守的低成長，假設業務初步穩定)
3.  **EBITDA 利潤率 (EBITDA Margin):** 70% (REITs 租金收入的 EBITDA 利潤率通常較高)
4.  **折舊與攤銷 (Depreciation & Amortization, D&A):** 假設為營收的 25% (基於歷史數據及物業性質)
5.  **資本支出 (Capital Expenditures, CAPEX):**
    *   維持性資本支出 (Maintenance Capex): 營收的 2% (用於物業維護)
    *   成長性資本支出 (Growth Capex - Property Acquisitions): 鑑於 MPW 的去槓桿化策略和資產出售，預計未來幾年淨成長性資本支出為負（即淨現金流入來自資產出售），或為零。
        *   2024-2025: 淨資產出售產生 $200M/年 現金流入 (反映去槓桿和資產重組)
        *   2026-2028: 淨成長性資本支出為零 (假設不再積極擴張或出售)
6.  **營運資金變動 (Change in Net Working Capital, ΔNWC):** 0% (對 REITs 影響通常很小)
7.  **稅率 (Tax Rate):** REITs 若分配大部分收益則免徵企業所得稅。此處假設有效現金稅率為 5% (用於計算稅盾和少數未分配收益)。
8.  **永續增長率 (Terminal Growth Rate, g):** 0.5% (對於成熟且目前面臨挑戰的 REIT，非常保守)
9.  **加權平均資本成本 (Weighted Average Cost of Capital, WACC):**
    *   **無風險利率 (Risk-Free Rate):** 4.5% (使用美國 10 年期國債殖利率)
    *   **股權風險溢價 (Equity Risk Premium, ERP):** 5.5%
    *   **Beta (β):** 1.8 (MPW 股價波動性高，尤其是在困境中)
    *   **股權成本 (Cost of Equity, Ke):** Ke = 無風險利率 + β * ERP = 4.5% + 1.8 * 5.5% = 14.4%
    *   **稅前債務成本 (Pre-Tax Cost of Debt, Kd):** 7.5% (考慮到其高槓桿和再融資風險)
    *   **市場價值資本結構 (Market Value Capital Structure):**
        *   市值 (Equity Value): $2.82B
        *   淨債務 (Net Debt): $10.5B - $0.3B = $10.2B
        *   總資本 (Total Capital): $2.82B + $10.2B = $13.02B
        *   權重 (Equity Weight, E/V): $2.82B / $13.02B = 21.66%
        *   債務權重 (Debt Weight, D/V): $10.2B / $13.02B = 78.34%
    *   **WACC:** E/V * Ke + D/V * Kd * (1 - 有效稅率)
        *   WACC = 21.66% * 14.4% + 78.34% * 7.5% * (1 - 0.05)
        *   WACC = 3.12% + 5.58% = **8.70%** (取整為 8.7%)

### 估值方法三：股息折現模型 (Dividend Discount Model, DDM)

**適用性判斷 (Applicability Judgment):**
MPW 在 2024 年初大幅削減了股息，從每股 $0.15/季度 (即 $0.60/年) 降至每股 $0.03/季度 (即 $0.12/年)。這使得 DDM 模型在此時幾乎不適用。
1.  **股息不可預測性高:** 鑑於其租戶問題和去槓桿化策略，MPW 未來的股息政策高度不確定。市場預期其股息仍有可能進一步調整，或需要很長時間才能恢復。
2.  **股息不代表盈利能力:** 當前極低的股息不反映其潛在的盈利能力 (FFO/AFFO)，而是反映了公司在現金流和債務償還方面的壓力。
3.  **不符合穩定增長假設:** DDM 假設股息穩定增長，這與 MPW 的現狀相悖。

**結論:** 鑑於 MPW 當前的財務困境和不穩定的股息政策，**股息折現模型不適用於對 MPW 的估值。**

### 估值方法四：淨資產價值法 (Net Asset Value, NAV)

**NAV 方法的合理性 (Rationale for NAV):**
對於 REITs 而言，淨資產價值 (NAV) 是非常重要的估值方法，尤其是在公司面臨困境時，NAV 可以提供一個對其基礎資產價值的保守估計。它通過估計所有物業的市場價值，減去總債務和其他負債，並加上其他資產來計算。

**估值假設 (Valuation Assumptions):**

1.  **淨營運收入 (Net Operating Income, NOI):** 我們將使用 MPW 最新的 LTM NOI 作為基礎。根據其財務數據，LTM EBITDA 約為 $1.1B，且由於 REIT 的運營模式，NOI 通常與 EBITDA 相近。我們假設 LTM NOI 為 **$1.1B**。
2.  **資本化率 (Capitalization Rate, Cap Rate):** Cap rate 是評估 REIT 物業價值的關鍵。對於醫院物業，健康時期通常在 6.5%-8.0% 之間。但考慮到 MPW 租戶的信用風險、高槓桿和整體市場對醫療地產的謹慎態度，我們將使用較高的 Cap Rate 來反映風險。
    *   **基準 Cap Rate:** 8.5%
    *   **範圍:** 8.0% (較樂觀) - 9.0% (較悲觀)
3.  **其他資產與負債 (Other Assets & Liabilities):**
    *   現金及約當現金: $0.3B
    *   總債務: $10.5B
    *   其他淨資產/負債: 假設為零或不顯著，以簡化模型。

**計算過程 (NAV Calculation):**

*   **估計物業市場價值 (Estimated Property Market Value):**
    *   基準: NOI / Cap Rate = $1.1B / 0.085 = **$12.94B**
*   **計算淨資產價值 (NAV):**
    *   物業市場價值 + 現金及約當現金 - 總債務
    *   $12.94B + $0.3B - $10.5B = **$2.74B**
*   **每股淨資產價值 (NAV per Share):**
    *   $2.74B / 587 百萬股 = **$4.67 / 股**

**NAV 估值結果與合理價位區間 (NAV Valuation Result & Range):**
*   **估值結果:** **$4.67 / 股**
*   **合理價位區間 (Sensitivity Analysis to Cap Rate):**
    *   Cap Rate 8.0%: 物業價值 $1.1B / 0.08 = $13.75B => NAV = $13.75B + $0.3B - $10.5B = $3.55B => NAV/股 = **$6.05 / 股**
    *   Cap Rate 9.0%: 物業價值 $1.1B / 0.09 = $12.22B => NAV = $12.22B + $0.3B - $10.5B = $2.02B => NAV/股 = **$3.44 / 股**
*   **合理價位區間:** **$3.44 - $6.05 / 股**
