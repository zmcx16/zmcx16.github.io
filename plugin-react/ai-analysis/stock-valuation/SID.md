### **1. 估值假設 (共同假設)**

在進行各項估值之前，我們需要設定一些共同的關鍵假設：

**1.1. 折現率 (WACC - 加權平均資本成本)**

*   **風險無風險利率 (Rf)：** 採用美國10年期公債殖利率，假設為 **4.20%**。
*   **股票風險溢價 (ERP)：** 採用廣泛使用的市場風險溢價，假設為 **5.50%**。
*   **Beta (β)：** 根據歷史數據和分析師報告，考慮 SID 的業務性質（週期性強）和新興市場風險，選取貝他值 **1.25**。
*   **股權成本 (Ke)：** 根據資本資產定價模型 (CAPM)：
    *   $Ke = Rf + \beta \times ERP$
    *   $Ke = 4.20\% + 1.25 \times 5.50\% = 4.20\% + 6.88\% = \textbf{11.08\%}$

*   **債務成本 (Kd)：** 考慮巴西的利率環境和公司債務結構，假設稅前債務成本為 **9.00%**。
*   **有效稅率 (t)：** 巴西的企業所得稅率較高，加上社會貢獻金，有效稅率假設為 **34%**。
    *   稅後債務成本 = $Kd \times (1 - t) = 9.00\% \times (1 - 0.34) = 9.00\% \times 0.66 = \textbf{5.94\%}$

*   **資本結構：** (截至2023年末數據，粗略換算為美元，實際以 BRL 為準)
    *   市場股權價值 (E)：約 $3.20 \times 1,328.7M = \textbf{$4.25B}$
    *   總債務 (D)：約 **$6.00B** (根據CSN 2023年末財務報告的淨負債約BRL 29.8B，總負債約BRL 41.5B，折合約$8.0B)。為簡化，此處使用一個接近市場價值的估計值。
    *   總資本 (V = E + D) = $4.25B + $6.00B = $10.25B$
    *   股權權重 (We) = $4.25B / $10.25B = **41.46%**
    *   債務權重 (Wd) = $6.00B / $10.25B = **58.54%**

*   **加權平均資本成本 (WACC)：**
    *   $WACC = (We \times Ke) + (Wd \times Kd \times (1 - t))$
    *   $WACC = (0.4146 \times 11.08\%) + (0.5854 \times 5.94\%)$
    *   $WACC = 4.594\% + 3.478\% = \textbf{8.07\%}$
    *   **WACC 範圍：** 考量到參數波動性，我們將使用 **7.5% - 8.5%** 的 WACC 範圍進行敏感性分析。

**1.2. 匯率：**

### **4. 股息折現模型 (Dividend Discount Model, DDM)**

SID 作為一家新興市場的鋼鐵與礦業公司，其股息支付具有較大的波動性和週期性，不一定呈現穩定的增長趨勢。因此，DDM 可能不是最適合 SID 的估值方法。然而，我們仍可以嘗試應用一個簡化的模型以供參考。

**4.1. 假設：**
*   **最新年化股息 (D0)：** 考慮 SID 近年的股息支付，假設其穩定化的年度股息為每股約 **$0.10 USD/ADR** (這是基於歷史派發的簡化假設，實際可能不穩定)。
*   **股息增長率 (g)：** 由於業務週期性強且增長不穩定，我們假設長期股息增長率僅為 **1.0%**，非常保守。
*   **股權成本 (Ke)：** 沿用 WACC 計算中的股權成本 **11.08%**。

**4.2. 計算 (Gordon Growth Model)：**
*   $P = D1 / (Ke - g) = D0 \times (1 + g) / (Ke - g)$
*   $P = $0.10 \times (1 + 0.01) / (0.1108 - 0.01)$
*   $P = $0.101 / 0.1008 = \textbf{$1.00 USD/ADR}$

**4.3. DDM 估值結果及合理價位區間：**

根據 DDM 模型，SID 的估值約為 **$1.00 USD/ADR**。

**DDM 適用性分析：**
這個結果顯著低於其他模型，主要原因在於：
1.  **股息不穩定性：** SID 的股息支付極度依賴於商品價格和經濟週期，其增長率難以預測。
2.  **低增長率假設：** 為了保守和反映歷史不穩定性，假設的長期股息增長率非常低，導致估值較低。
3.  **非核心業務：** 對於像 SID 這樣週期性強、資本密集且具有成長潛力的公司，其價值可能更多體現在未來現金流和資產價值而非穩定股息。

因此，DDM 對於 SID 的參考價值較低，應謹慎使用。

### **5. 其他合理估值方法 (簡要說明)**

*   **資產基礎估值 (Asset-Based Valuation)：** 對於擁有大量實體資產（鋼鐵廠、礦山、土地）的公司，資產基礎估值（例如，重置成本法或清算價值法）可能適用。然而，這需要對公司各項資產進行詳細的盤點和市場價值評估，超出本次分析的範圍。考慮到其龐大的礦產儲量和基礎設施，這可能是一個潛在的額外價值來源。
*   **分部加總估值 (Sum-of-the-Parts Valuation)：** 由於 CSN 業務多元化（鋼鐵、礦業、物流、能源、水泥），可以將每個業務部門單獨估值，然後加總得出公司總價值。這需要更詳細的分部財務數據和可比公司。
