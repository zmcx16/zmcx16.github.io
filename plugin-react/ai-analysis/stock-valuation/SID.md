#### 3. 股息折現模型（Dividend Discount Model, DDM）

如果我們假設股息是可持續的，並且以一個穩定的速度增長，我們可以嘗試一個簡化的 DDM。

**假設：**

*   **當前每股股息 (D0):** 1.44 美元 (股價) \* 5.69% (股息收益率) = 0.0819 美元

*   **股本成本 (Re):** 13.63% (來自 DCF 模型)

**DDM 估值 = D0 \* (1 + g) / (Re - g):**
= 0.0819 \* (1 + 0.005) / (0.1363 - 0.005)
= 0.0819 \* 1.005 / 0.1313
= 0.0823 / 0.1313 = **0.63 美元**
