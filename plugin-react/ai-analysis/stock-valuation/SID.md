    *   假設營運利潤率從 10% 逐步提升至 13%。
    *   NOPAT = EBIT * (1 - 稅率)
3.  **預測資本支出 (CAPEX) 和折舊攤銷 (D&A)：**

    *   FCFF = NOPAT + D&A - CAPEX - Change in NWC
6.  **計算終值 (Terminal Value, TV)：**

7.  **計算企業價值 (Enterprise Value, EV)：** 將預測期內的 FCFF 和 TV 折現到現在。
8.  **計算股權價值 (Equity Value)：** EV - 淨債務 + 非經營性資產。
9.  **計算每股價值 (Intrinsic Value per Share)：** 股權價值 / 總股數。
