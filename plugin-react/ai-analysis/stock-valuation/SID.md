Companhia Siderúrgica Nacional (NYSE: SID) 是一家在巴西和拉丁美洲營運的綜合鋼鐵生產商，業務涵蓋鋼鐵、採礦、物流、能源和水泥五大板塊。鋼鐵業務是其主要收入來源，而採礦業務則被視為重要的增長動力。 該公司總部位於巴西聖保羅。

截至2026年3月，SID的市值約為15.1億美元，股價約在1.14美元至1.20美元之間。

**最新財務表現與市場動態：**

SID在2025年第三季度報告的每股收益（EPS）為0.01美元，低於分析師預期的0.10美元。 儘管2025年第四季度報告的每股收益為-0.0102美元，超出分析師預期，營收達22.1億美元，也高於預期，但由於槓桿率上升（3.47倍）、即將到期的10億美元債券以及鋼鐵行業的整體拋售，股價仍大幅下跌。

2025年，該公司的EBITDA增長了15%，主要得益於採礦和物流業務的創紀錄銷量以及鋼鐵成本的降低，EBITDA利潤率達到28%。 然而，調整後的現金流在2025年為負2.61億雷亞爾。

SID的盈利能力面臨挑戰，過去12個月（TTM）的淨收入為-3.6553億美元，稀釋後每股收益為-0.28美元，淨利潤率為-4.47%。 股東權益報酬率（ROE）、資產報酬率（ROA）和投資資本報酬率（ROI）均為負值。

該公司債務水平高企，最近報告的總債務為98.6億美元，股債比為343.27%。 為降低債務和槓桿，SID計劃通過資產剝離籌集150億至180億雷亞爾。

自由現金流方面，過去12個月的槓桿自由現金流為-9486萬美元，無槓桿自由現金流為4.0625億美元，每股自由現金流為-0.95美元。 負的市現率（P/FCF）為-54.56，被視為潛在的警示信號。

分析師普遍對SID持「強烈賣出」或「中度賣出」的共識評級，平均目標價為1.40美元。

**股息政策：**

SID在過去一年中沒有支付任何股息，且下一個除息日未知。 因此，股息折現模型（DDM）不適用於本次估值分析。

### 估值分析

由於SID的盈利能力不穩定且自由現金流為負，傳統的折現現金流模型（DCF）和基於盈利的相對估值方法（如P/E）將面臨挑戰。我們將主要採用無槓桿自由現金流（UFCF）的DCF模型，並輔以基於資產和銷售的相對估值方法。

#### 1. 折現現金流模型（Discounted Cash Flow, DCF）

DCF模型旨在估計公司未來現金流的現值。由於SID的槓桿自由現金流為負，我們將專注於無槓桿自由現金流（UFCF）的估計。

**估值假設：**

*   **預測期間：** 5年（2026-2030年）的明確預測期，之後為永續增長階段。
*   **折現率（WACC）：** 加權平均資本成本（WACC）將作為折現率。

    *   **股權成本（Cost of Equity, Re）：** 使用資本資產定價模型（CAPM）。
        *   無風險利率（Risk-Free Rate）：採用美國10年期國債收益率，約為4.30%。
        *   市場風險溢價（Market Risk Premium）：採用4.23%。
        *   Beta：採用1.48。
        *   Re = 無風險利率 + Beta × 市場風險溢價
        *   Re = 4.30% + 1.48 × 4.23% = 4.30% + 6.26% = 10.56%

    *   **債務成本（Cost of Debt, Rd）：** 考慮到SID的高槓桿率和巴西市場的風險，我們將假設一個較高的稅前債務成本。由於缺乏具體的債務利率數據，我們將估計稅前債務成本為8.0%。
        *   稅率（Tax Rate）：巴西的企業所得稅率約為34%（包括聯邦稅和社會貢獻）。
        *   稅後債務成本 = 稅前債務成本 × (1 - 稅率)
        *   稅後債務成本 = 8.0% × (1 - 0.34) = 8.0% × 0.66 = 5.28%

    *   **資本結構：**
        *   市值（Market Cap, E）= 15.1億美元
        *   總債務（Total Debt, D）= 98.6億美元
        *   總資本（V）= E + D = 15.1億美元 + 98.6億美元 = 113.7億美元
        *   股權權重（E/V）= 15.1億美元 / 113.7億美元 = 13.28%
        *   債務權重（D/V）= 98.6億美元 / 113.7億美元 = 86.72%

    *   **WACC 計算：**
        *   WACC = (E/V × Re) + (D/V × 稅後債務成本)
        *   WACC = (0.1328 × 10.56%) + (0.8672 × 5.28%)
        *   WACC = 1.40% + 4.58% = 5.98%
        *   我們將採用約 **6.0%** 作為WACC。

*   **收入增長率：** 考慮到鋼鐵行業的挑戰和中國鋼鐵供應的影響，以及公司多元化業務的潛力，我們假設：
    *   2026年：2.0%
    *   2027年：3.0%
    *   2028年：3.5%
    *   2029年：3.0%
    *   2030年：2.5%
*   **營業利潤率（Operating Margin）：** 初始數據為10.59%。考慮到行業壓力，我們假設在預測期內維持在10%左右，並在後期略有改善。
*   **資本支出（CapEx）：** 根據2025年CapEx增長42.4%至59億雷亞爾，我們將其作為收入的百分比進行預測，並假設其在預測期內逐漸穩定。
*   **營運資本變動（Change in NWC）：** 假設為收入的穩定百分比（例如，1%）。
*   **永續增長率（Terminal Growth Rate）：** 考慮到巴西經濟和鋼鐵行業的長期增長潛力，我們採用保守的1.5%。

**DCF 計算步驟（簡化）：**

1.  **預測收入：** 基於TTM收入81.8億美元 和上述增長率。
2.  **預測營業利潤（EBIT）：** 收入 × 營業利潤率。
3.  **計算稅後營業利潤（NOPAT）：** EBIT × (1 - 稅率)。
4.  **計算無槓桿自由現金流（UFCF）：** NOPAT + 折舊與攤銷 - 資本支出 - 營運資本變動。
    *   由於缺乏詳細的折舊與攤銷數據，我們將假設其與資本支出大致相抵或作為收入的固定百分比。
    *   根據Seeking Alpha，SID的TTM無槓桿自由現金流為4.0625億美元。 我們將以此為基礎進行預測。

| 年度 | 收入 (百萬美元) | 增長率 | 營業利潤率 | NOPAT (百萬美元) | 資本支出 (百萬美元) | 營運資本變動 (百萬美元) | UFCF (百萬美元) |
| :--- | :-------------- | :----- | :--------- | :--------------- | :---------------- | :-------------------- | :-------------- |
| 2025 (TTM) | 8,180 | -      | 10.59%     | 570              | 1,180 (估計)      | 80 (估計)             | 406   |
| 2026 | 8,344           | 2.0%   | 10.50%     | 578              | 1,200             | 83                    | 400             |
| 2027 | 8,594           | 3.0%   | 10.60%     | 600              | 1,250             | 86                    | 410             |
| 2028 | 8,895           | 3.5%   | 10.70%     | 630              | 1,300             | 89                    | 430             |
| 2029 | 9,162           | 3.0%   | 10.80%     | 660              | 1,350             | 92                    | 450             |
| 2030 | 9,391           | 2.5%   | 10.90%     | 688              | 1,400             | 94                    | 460             |

*   **計算永續價值（Terminal Value, TV）：**
    *   TV = UFCF (2030) × (1 + 永續增長率) / (WACC - 永續增長率)
    *   TV = 460 × (1 + 0.015) / (0.060 - 0.015)
    *   TV = 460 × 1.015 / 0.045 = 466.9 / 0.045 = 10,375.56 百萬美元

*   **計算各年UFCF的現值和TV的現值：**
    *   PV (UFCF 2026) = 400 / (1 + 0.06)^1 = 377.36
    *   PV (UFCF 2027) = 410 / (1 + 0.06)^2 = 364.59
    *   PV (UFCF 2028) = 430 / (1 + 0.06)^3 = 361.04
    *   PV (UFCF 2029) = 450 / (1 + 0.06)^4 = 356.40
    *   PV (UFCF 2030) = 460 / (1 + 0.06)^5 = 343.09
    *   PV (TV) = 10,375.56 / (1 + 0.06)^5 = 10,375.56 / 1.3382 = 7,753.30

*   **企業總價值（Enterprise Value, EV）：**
    *   EV = 預測期UFCF現值之和 + TV現值
    *   EV = (377.36 + 364.59 + 361.04 + 356.40 + 343.09) + 7,753.30
    *   EV = 1,802.48 + 7,753.30 = 9,555.78 百萬美元

*   **股權價值（Equity Value）：**
    *   股權價值 = EV + 現金及約當現金 - 總債務
    *   現金及約當現金（MRQ）：27億美元
    *   總債務（MRQ）：98.6億美元
    *   股權價值 = 9,555.78 + 2,700 - 9,860 = 2,395.78 百萬美元

*   **每股估值：**
    *   流通股數：約13.3億股 (根據市值1.51B / 股價1.14 = 1.32B，或Seeking Alpha的1.3B)
    *   每股估值 = 股權價值 / 流通股數
    *   每股估值 = 2,395.78 百萬美元 / 1,330 百萬股 = **1.80 美元/股**

**DCF模型估值結果：**

*   **合理價位區間：** 考慮到假設的敏感性，合理價位區間可能在 **1.50美元至2.10美元** 之間。

    *   每股估值 = 15.33億美元 / 13.3億股 = **1.15 美元/股**

    *   估值 = 平均P/S × SID的銷售額 = 0.36 × 81.8億美元 = 29.45億美元

    *   估值（EV）= 平均EV/EBITDA × SID的EBITDA = 4.88 × 14.6億美元 = 71.25億美元

*   基於P/S：2.21 美元/股
*   基於EV/EBITDA：無法得出正向股權價值

**合理價位區間：** 考慮到P/B和P/S的結果，以及EV/EBITDA的負值，相對估值結果差異較大。P/B估值接近當前股價，而P/S估值則顯著更高。這可能反映了市場對SID盈利能力和高債務的擔憂，導致其市銷率較低。

#### 3. 綜合結論

*   **DCF模型估值：** 1.80 美元/股，合理價位區間為1.50美元至2.10美元。
*   **相對估值（P/B）：** 1.15 美元/股。
*   **相對估值（P/S）：** 2.21 美元/股。
*   **分析師目標價：** 平均1.40美元，共識為「強烈賣出」。

綜合來看，DCF模型給出的估值（1.80美元）高於當前股價（1.14美元）和分析師的平均目標價（1.40美元）。這表明如果公司能夠實現預測的現金流增長和利潤率改善，其股價存在上漲空間。

然而，相對估值結果差異較大，P/B估值與當前股價接近，而P/S估值則較高。EV/EBITDA模型因高債務導致股權價值為負，這強烈凸顯了SID的財務槓桿風險。

**主要風險與考量：**

1.  **高債務水平：** SID的債務水平極高，這增加了公司的財務風險，並可能限制其未來投資和增長。 公司計劃通過資產剝離來降低槓桿，這是一個積極的信號，但執行情況仍需觀察。
2.  **盈利能力不穩定：** 公司目前處於虧損狀態，且自由現金流為負，這使得未來現金流的預測存在較大不確定性。
3.  **行業逆風：** 巴西鋼鐵行業面臨中國鋼鐵的激烈競爭、國內需求疲軟以及難以轉嫁成本的壓力，導致利潤率較低。
4.  **分析師情緒：** 大多數分析師對SID持負面看法，這可能對股價構成壓力。

**綜合判斷：**

儘管DCF模型在假設未來現金流改善的情況下給出了較高的估值，但SID目前面臨的巨大債務壓力、不穩定的盈利能力以及行業逆風是不可忽視的重大風險。相對估值中的P/B倍數與當前股價接近，而EV/EBITDA的負股權價值則敲響了警鐘。分析師的「強烈賣出」共識也反映了市場對其前景的擔憂。

考慮到上述因素，SID的投資風險較高。雖然其多元化業務（採礦、物流、能源、水泥）可能提供一定的緩衝和增長潛力，但高槓桿和鋼鐵主業的挑戰仍是主要障礙。

**合理價位區間（綜合）：**

基於DCF模型和相對估值結果，並考慮到當前市場情緒和公司面臨的風險，SID的合理價位區間可能在 **1.10美元至1.60美元** 之間。當前股價1.14美元處於此區間的下限，反映了市場對其風險的定價。任何投資決策都應仔細評估其高風險特性。
