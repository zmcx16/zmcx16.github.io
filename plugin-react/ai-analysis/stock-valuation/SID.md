#### 3.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 估值需要對公司未來的自由現金流進行預測，並使用適當的折現率將其折現至現值。

**估值假設：**

    *   Q4 2025 收入同比增長2.54%。
    *   我將假設一個保守的收入增長率：
        *   2026年：3.0% (考慮到南美洲市場增長和公司多元化業務)
        *   2027年：2.5% (全球鋼鐵需求加速，但公司去槓桿化可能限制擴張)

*   **營業利潤率 (Operating Margin)：** 原始數據為10.59%。鑑於公司目前虧損，且鋼鐵行業利潤率波動較大，我將假設公司在預測期內逐步恢復並穩定營業利潤率。
    *   2026年：5.0% (從虧損中恢復，1Q26鋼鐵業務EBITDA利潤率為7.0%，但整體仍需改善)
    *   2027年：7.0%
    *   2028年：8.0%
    *   2029-2030年：9.0% (逐步接近歷史水平但仍保守)
*   **稅率：** 巴西企業所得稅率約為34%。

*   **營運資本變動 (Change in Working Capital)：** 假設為收入的1%。
*   **折現率 (WACC - Weighted Average Cost of Capital)：**
    *   **股權成本 (Cost of Equity, Ke)：** 使用資本資產定價模型 (CAPM)。
        *   無風險利率 (Risk-Free Rate)：參考美國10年期國債收益率，約4.5%。

        *   Ke = 無風險利率 + Beta * MRP = 4.5% + 1.5 * 5.5% = 4.5% + 8.25% = 12.75%。
    *   **債務成本 (Cost of Debt, Kd)：** 考慮到公司高負債和巴西市場風險，假設稅前債務成本為8.0%。
        *   稅後債務成本 = Kd * (1 - 稅率) = 8.0% * (1 - 0.34) = 8.0% * 0.66 = 5.28%。
    *   **資本結構：**
        *   債務/股權 (Debt/Eq) = 3.363
        *   股權比重 (E) = 1 / (1 + 3.363) = 1 / 4.363 ≈ 0.229
        *   債務比重 (D) = 3.363 / (1 + 3.363) = 3.363 / 4.363 ≈ 0.771
    *   **WACC = (E / (E+D)) * Ke + (D / (E+D)) * Kd * (1 - 稅率)**

**DCF 計算步驟 (簡化)：**

1.  **計算預測期內的自由現金流 (FCFF)：**
    *   收入 (Revenue)
    *   營業利潤 (Operating Income = Revenue * Operating Margin)
    *   稅後營業利潤 (NOPAT = Operating Income * (1 - 稅率))
    *   加回折舊攤銷 (Depreciation & Amortization, D&A)：假設為收入的3%。
    *   減去資本支出 (CapEx)
    *   減去營運資本變動 (Change in Working Capital)
    *   FCFF = NOPAT + D&A - CapEx - Change in Working Capital

2.  **計算終值 (Terminal Value, TV)：**
    *   TV = (FCFF_last_year * (1 + g)) / (WACC - g)

3.  **將預測期內的 FCFF 和終值折現至現值：**
    *   PV(FCFF) = Σ (FCFF_t / (1 + WACC)^t)
    *   PV(TV) = TV / (1 + WACC)^last_year

4.  **企業價值 (Enterprise Value, EV) = PV(FCFF) + PV(TV)**

5.  **股權價值 (Equity Value) = EV - 淨債務 (Net Debt)**

    *   債務 = 股權價值 * 債務/股權比率 = 1.84億 * 3.363 = 6.188億美元
    *   假設現金為1億美元 (保守估計，實際需查閱財報)
    *   淨債務 = 6.188億 - 1億 = 5.188億美元

6.  **每股估值 = 股權價值 / 總股數**
    *   總股數 = Market Cap / Current Price = 1.84億 / 1.39 ≈ 1.32億股

*   基於 P/B，SID 可能被輕微低估，合理價位約為 **1.57美元**。
*   基於 P/S 和 EV/EBITDA，SID 的倍數遠低於同行，這可能反映了市場對其盈利能力、高負債和未來前景的極度悲觀。如果市場預期其盈利能力能恢復到行業平均水平，則存在巨大的上漲空間，但這需要非常強勁的基本面改善。

### 4. 綜合結論與合理價位區間

**估值結果摘要：**

*   **DCF 模型：** 負股權價值，表明在當前假設下，公司預期現金流不足以覆蓋其債務。這是一個強烈的警示信號，暗示公司存在嚴重的財務風險或被高估。
*   **相對估值 (P/B)：** 約1.57美元，略高於當前股價。
*   **相對估值 (P/S, EV/EBITDA)：** 遠低於同行，表明市場對其未來盈利能力和前景極度悲觀。

**綜合分析：**

SID 是一家業務多元化的巴西鋼鐵生產商，但目前面臨嚴峻的財務挑戰，主要體現在高負債和近期虧損。儘管全球鋼鐵需求預計在2026-2027年觸底反彈，南美市場也有望增長，但 CSN 自身的去槓桿化壓力以及不穩定的盈利能力是其估值的關鍵不確定因素。

DCF 模型給出的負值結果，強烈暗示了公司當前的高風險和潛在的價值破壞。這可能是由於其高額債務、持續虧損以及未來現金流預期不足以彌補這些問題。

相對估值方面，P/B 顯示其可能略有低估，但 P/S 和 EV/EBITDA 的極低倍數則反映了市場的悲觀情緒。這種巨大的差異表明，市場可能認為 SID 的資產和銷售額無法有效轉化為可持續的盈利和現金流。

**合理價位區間：**

鑑於 DCF 模型的負值結果和相對估值法的巨大差異，很難給出一個精確的合理價位區間。

*   **悲觀情境：** 如果公司無法有效去槓桿化並恢復持續盈利，其價值可能遠低於當前股價，甚至可能面臨更大的下行風險。DCF 的負值結果支持這一觀點。
*   **中性情境：** 如果公司能夠成功執行去槓桿化計劃，並在鋼鐵市場復甦中逐步改善盈利能力，其股價可能會向 P/B 估值所暗示的 **1.50 - 1.60 美元** 區間靠攏。這也與分析師的目標價1.69美元接近。
*   **樂觀情境：** 如果公司能夠大幅超越市場預期，實現強勁的盈利增長和現金流產生，並顯著降低債務，那麼其估值倍數可能會向行業平均水平靠攏，從而帶來顯著的股價上漲。然而，這需要非常大的基本面轉變。

**綜合結論：**

Companhia Siderúrgica Nacional (SID) 目前的估值分析顯示，該公司面臨顯著的財務挑戰和不確定性。其高負債、近期虧損以及不穩定的股息政策使其成為一項高風險投資。儘管全球鋼鐵市場前景有所改善，但 CSN 自身的去槓桿化和盈利能力恢復是其能否實現價值增長的關鍵。

因此，對於 SID 的投資應極其謹慎。其當前股價可能已經反映了市場對其未來改善的某些預期，但潛在的下行風險仍然存在，尤其是在去槓桿化和盈利能力恢復不及預期的情況下。
