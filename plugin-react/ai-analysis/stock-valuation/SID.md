### 2. 估值方法與假設

鑑於 Gerdau S.A. 2025 年的淨利潤為負值，且巴西業務面臨挑戰，但北美業務表現強勁並產生正自由現金流，我們將主要採用折現現金流模型 (DCF) 和相對估值法。股息折現模型 (DDM) 雖然公司有派息，但由於其盈利波動和股息政策與淨利潤掛鉤，且 2025 年淨利潤為負，DDM 的適用性較低，但仍可作為參考。

**估值假設 (基於最新數據和行業趨勢):**

**A. 加權平均資本成本 (WACC) 計算**

WACC = (E/V) \* Re + (D/V) \* Rd \* (1 - Tc)

*   **股權成本 (Re) - 使用資本資產定價模型 (CAPM):**
    *   Re = Rf + Beta \* (Rm - Rf)
    *   **無風險利率 (Rf):** 考慮到 Gerdau 是一家巴西公司，但在美國上市，其業務遍及美洲，我們需要考慮巴西的無風險利率。巴西 10 年期政府債券收益率約為 13.42% (截至 2026 年 1 月)。然而，對於在美國上市的股票，通常會使用美國的無風險利率，然後加上國家風險溢價。但由於公司主要業務在巴西，使用巴西的無風險利率更為合適。我們將採用巴西 10 年期政府債券收益率作為無風險利率：**Rf = 13.42%**。
    *   **Beta:** 1.46。
    *   **市場風險溢價 (Rm - Rf):** 巴西的股權風險溢價 (ERP) 截至 2023 年 1 月 1 日為 11.13%。我們將使用此數據。
    *   Re = 13.42% + 1.46 \* 11.13% = 13.42% + 16.25% = **29.67%**

*   **債務成本 (Rd):**
    *   Gerdau 在 2025 年 10 月 30 日宣布贖回 2030 年到期、利率為 4.25% 的 5 億美元債券。這是一個美元債券。
    *   巴西的貸款風險溢價 (prime rate minus treasury bill rate) 在 2024 年為 29.05%。
    *   Gerdau 的利息支出 (Interest Expense on Debt) 在 2025 年 9 月為 BRL 358.37M。
    *   考慮到公司有美元債務，且巴西利率較高，我們需要一個綜合的債務成本。由於公司有能力贖回 4.25% 的美元債券，這表明其在國際市場的借貸成本相對較低。然而，其大部分業務在巴西，可能面臨更高的當地借貸成本。
    *   我們將使用一個估計值。考慮到巴西的高利率環境和公司較高的債務權益比 (儘管已改善)，我們假設一個較高的債務成本。
    *   根據 2025 年 9 月的利息支出 BRL 358.37M 和總債務 BRL 19.52B，我們可以粗略估計一個平均債務利率：358.37M / 19.52B = 1.83% (季度)。年化約為 7.32%。
    *   我們將採用一個保守的債務成本：**Rd = 7.5%**。

*   **稅率 (Tc):**
    *   巴西的企業所得稅率約為 34% (包括聯邦稅和社會貢獻) [未直接搜尋到最新數據，使用一般巴西企業稅率]。我們將採用 **Tc = 34%**。

*   **資本結構 (E/V 和 D/V):**
    *   市值 (E) = $8.30 billion。
    *   淨債務 (Net Debt) = 總債務 - 現金及約當現金。
    *   根據 Trading Economics，Gerdau 的總債務為 BRL 19.52B，現金及約當現金為 BRL 9.39B (截至 2025 年 9 月)。
    *   淨債務 = 19.52B - 9.39B = BRL 10.13B。
    *   假設當前匯率約為 1 USD = 5 BRL (僅為估計，實際匯率會波動)。
    *   淨債務 (USD) = 10.13B / 5 = $2.026 billion。
    *   總企業價值 (V) = E + 淨債務 = $8.30B + $2.026B = $10.326 billion。
    *   E/V = 8.30 / 10.326 = **0.804**
    *   D/V = 2.026 / 10.326 = **0.196**

*   **WACC 計算:**
    *   WACC = (0.804 \* 29.67%) + (0.196 \* 7.5% \* (1 - 0.34))
    *   WACC = 23.87% + (0.196 \* 7.5% \* 0.66)
    *   WACC = 23.87% + (0.196 \* 4.95%)
    *   WACC = 23.87% + 0.97% = **24.84%**

    這個 WACC 顯著高於一般美國公司，主要由於巴西較高的無風險利率和股權風險溢價。

### 3. 估值計算與結果

**A. 折現現金流模型 (DCF)**

**自由現金流 (FCFF) 計算公式:**
FCFF = EBIT \* (1 - Tc) + D&A - CAPEX - Change in Working Capital

**預測期 (2026-2030) 自由現金流預測 (單位：十億美元):**

| 年度 | 營收增長率 | 營收 | EBITDA | EBITDA Margin | D&A (3% of Revenue) | EBIT | 稅後 EBIT (1-34%) | CAPEX | Change in WC (5% of Rev Growth) | FCFF |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 2025 | - | 13.98 | 2.02 | 14.45% | 0.42 | 1.60 | 1.056 | 1.22 | - | 0.0788 |

*   **折現因子 (Discount Factor):** 使用 WACC = 24.84%
*   **折現後的 FCFF:**
    *   2026: 0.58 / (1 + 0.2484)^1 = 0.46
    *   2027: 0.61 / (1 + 0.2484)^2 = 0.39
    *   2028: 0.63 / (1 + 0.2484)^3 = 0.33
    *   2029: 0.66 / (1 + 0.2484)^4 = 0.28
    *   2030: 0.69 / (1 + 0.2484)^5 = 0.23
*   **預測期內折現後的 FCFF 總和 = 0.46 + 0.39 + 0.33 + 0.28 + 0.23 = $1.69 billion**

**終值 (Terminal Value) 計算:**
TV = FCFF_2031 \* (1 + g) / (WACC - g)
FCFF_2031 = FCFF_2030 \* (1 + g) = 0.69 \* (1 + 0.02) = 0.7038 billion
TV = 0.7038 / (0.2484 - 0.02) = 0.7038 / 0.2284 = $3.08 billion

**終值折現到現在:**
Discounted TV = TV / (1 + WACC)^5 = 3.08 / (1 + 0.2484)^5 = 3.08 / 3.06 = $1.01 billion

**企業價值 (Enterprise Value) = 預測期內折現後的 FCFF 總和 + 折現後的終值**
Enterprise Value = $1.69 billion + $1.01 billion = $2.70 billion

**股權價值 (Equity Value) = 企業價值 - 淨債務**
Equity Value = $2.70 billion - $2.026 billion = $0.674 billion

**每股估值 (Per Share Value) = 股權價值 / 流通股數**
流通股數 = 市值 / 股價 = $8.30 billion / $4.16 = 1.995 billion 股 (約 20 億股)
每股估值 = $0.674 billion / 1.995 billion = **$0.34**

**DCF 模型結果：**
*   **估值結果:** 每股 $0.34
*   **合理價位區間:** 考慮到高 WACC 和盈利波動，以及假設的敏感性，合理區間可能在 **$0.25 - $0.45**。

這個 DCF 估值遠低於當前股價 $4.16。這可能反映了市場對 Gerdau 未來增長和盈利能力的更高預期，或者我的 WACC 假設過於保守，或者市場對巴西的風險溢價有不同的看法。高 WACC 顯著降低了未來現金流的現值。

**C. 股息折現模型 (DDM)**

Gerdau 宣布 2025 財年股息為每股 0.10 巴西雷亞爾。假設匯率為 1 USD = 5 BRL，則每股股息為 $0.02。
MarketBeat 報告年化股息為 $0.20/股，股息收益率 4.8%。這意味著每股股息為 $0.20。我們將使用這個較高的數字作為基礎。

**DDM 公式 (戈登增長模型):**
P0 = D1 / (Re - g)
其中：
*   P0 = 當前股價
*   D1 = 下一期預期股息
*   Re = 股權成本 (29.67%)
*   g = 股息永續增長率

假設股息增長率 g = 2% (與 DCF 的長期增長率一致)。
D1 = $0.20 \* (1 + 0.02) = $0.204

P0 = $0.204 / (0.2967 - 0.02) = $0.204 / 0.2767 = **$0.74**

**DDM 模型結果：**
*   **估值結果:** 每股 $0.74
*   **合理價位區間:** 考慮到股息增長率和股權成本的敏感性，合理區間可能在 **$0.60 - $0.90**。

DDM 估值也遠低於當前股價。這再次強調了高股權成本對估值的影響，以及市場可能對 Gerdau 未來股息增長有更高的預期，或者其盈利穩定性被市場高估。
