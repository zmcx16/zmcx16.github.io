### **二、估值假設**

在進行估值之前，我們需要設定一些關鍵假設。由於 Petrobras 的國有性質和巴西的政治環境，其風險溢價會較高。

**A. 加權平均資本成本 (WACC) 計算：**

*   **無風險利率 (Risk-Free Rate):** 採用美國 10 年期國債收益率，假設為 **4.5%**。
*   **市場風險溢價 (Market Risk Premium, MRP):** 採用標準的 **5.5%**。
*   **Beta (貝塔值):** 根據網路搜尋，大型能源公司的 Beta 通常在 1.0-1.2 之間。考慮到巴西市場的波動性和政治風險，我們採用稍高的 **1.2**。
*   **股權成本 (Cost of Equity, Ke):** 採用資本資產定價模型 (CAPM)。
    *   Ke = 無風險利率 + Beta * MRP
    *   Ke = 4.5% + 1.2 * 5.5% = 4.5% + 6.6% = **11.1%**
*   **債務成本 (Cost of Debt, Kd):** 根據 Petrobras 的債券收益率和信用評級，假設為 **6.5%**。
*   **稅率 (Tax Rate, t):** 巴西企業所得稅率較高，但考慮到 Petrobras 的實際有效稅率，我們採用 **28%**。
*   **資本結構權重：**
    *   市值 (Market Cap) = $29.93B
    *   總債務 (Total Debt): 根據提供的 Debt/Eq (0.89) 和市值，我們需要估算總債務。假設市場價值股權約等於市值，則總債務約為 $29.93B * 0.89 = $26.64B。
        *   *修正：* 為了更準確，我們通常會從財報中獲取總債務和現金。假設透過搜尋，我們估計 Petrobras 的總債務約為 **$50B**，現金及約當現金約為 **$10B**。
    *   股權市場價值 (E) = $29.93B
    *   債務市場價值 (D) = $50B
    *   總資本 (V) = E + D = $29.93B + $50B = $79.93B
    *   股權權重 (We) = E / V = $29.93B / $79.93B = **37.4%**
    *   債務權重 (Wd) = D / V = $50B / $79.93B = **62.6%**
*   **WACC 計算：**
    *   WACC = (We * Ke) + (Wd * Kd * (1 - t))
    *   WACC = (0.374 * 11.1%) + (0.626 * 6.5% * (1 - 0.28))
    *   WACC = 4.15% + (0.626 * 6.5% * 0.72)
    *   WACC = 4.15% + (0.626 * 4.68%)
    *   WACC = 4.15% + 2.93% = **7.08%**
    *   *考量到 Petrobras 的政治風險，市場實際要求的折現率可能更高。為了反映市場對其高風險的定價，我們將在 DCF 模型中採用一個更保守的 WACC 範圍，例如 8% - 10%。這裡我們取 **9.0%** 作為基準 WACC。*

**B. 折現現金流模型 (DCF) 假設：**

*   **預測期間 (Explicit Forecast Period):** 5 年。
*   **起始自由現金流 (FCF0):** 根據 P/FCF = 1.99 和市值 $29.93B。
    *   FCF0 = 市值 / P/FCF = $29.93B / 1.99 = **$15.04 Billion** (這是過去 12 個月的 FCF)。
*   **FCF 增長率：**

    *   第 1 年: -10% (反映短期挑戰和政治不確定性)

    *   第 3 年: 0% (趨於穩定)

    *   第 5 年: 3%
*   **永續增長率 (Terminal Growth Rate, g):** 考慮到成熟產業、政治風險和全球能源轉型，採用保守的 **1.5%** (略低於長期通膨和全球 GDP 增長)。
*   **折現率 (Discount Rate):** 採用 WACC = **9.0%**。

**C. 股息折現模型 (DDM) 假設：**

### **三、估值模型計算與結果**

**1. 折現現金流模型 (Discounted Cash Flow, DCF)**

*   **計算預測期 FCF：**
    *   FCF0 = $15.04B
    *   FCF1 = $15.04B * (1 - 0.10) = $13.54B
    *   FCF2 = $13.54B * (1 - 0.05) = $12.86B
    *   FCF3 = $12.86B * (1 + 0.00) = $12.86B
    *   FCF4 = $12.86B * (1 + 0.02) = $13.12B
    *   FCF5 = $13.12B * (1 + 0.03) = $13.52B

*   **計算 FCF 的現值 (PV of FCFs)：** (使用 WACC = 9.0%)
    *   PV(FCF1) = $13.54B / (1.09)^1 = $12.42B
    *   PV(FCF2) = $12.86B / (1.09)^2 = $10.82B
    *   PV(FCF3) = $12.86B / (1.09)^3 = $9.93B
    *   PV(FCF4) = $13.12B / (1.09)^4 = $9.29B
    *   PV(FCF5) = $13.52B / (1.09)^5 = $8.79B
    *   **預測期 FCF 現值總和 = $12.42B + $10.82B + $9.93B + $9.29B + $8.79B = $51.25B**

*   **計算終端價值 (Terminal Value, TV)：**
    *   FCF6 = FCF5 * (1 + g) = $13.52B * (1 + 0.015) = $13.72B
    *   TV = FCF6 / (WACC - g) = $13.72B / (0.09 - 0.015) = $13.72B / 0.075 = $182.93B
    *   **終端價值現值 (PV of TV) = $182.93B / (1.09)^5 = $118.90B**

*   **企業價值 (Enterprise Value, EV)：**
    *   EV = 預測期 FCF 現值總和 + 終端價值現值
    *   EV = $51.25B + $118.90B = **$170.15B**

*   **股權價值 (Equity Value)：**
    *   Equity Value = EV + 現金 - 總債務
    *   Equity Value = $170.15B + $10B - $50B = **$130.15B**

*   **每股價值 (Value per Share)：**
    *   流通股數 = 市值 / 當前股價 = $29.93B / $10.99 = 2.723 Billion 股
    *   每股價值 = $130.15B / 2.723B 股 = **$47.80**

*   **DCF 估值結果：**
    *   **合理價位區間：$40.00 - $55.00** (考慮到 WACC 和增長率假設的敏感性，例如 WACC 提高到 10% 或增長率更保守，估值會下降，但仍遠高於現價)。

**分析與比較：**

1.  **DCF 和相對估值模型顯示 PBR-A 嚴重被低估。** 這兩個模型基於其強勁的自由現金流、盈利能力和相對於同業的低倍數，得出了遠高於當前股價的估值。例如，P/FCF 僅為 1.99，遠低於同業的 3-10 倍，這表明市場對其現金流的折現率極高。
2.  **DDM 模型結果與當前股價較為接近。** 這反映了市場對 Petrobras 股息政策的高度不確定性。由於政府干預，股息支付可能不穩定，甚至可能被削減，這使得 DDM 在此情況下可能更貼近市場的悲觀預期。
3.  **市場折價的根本原因：** 估值模型之間的巨大差異，特別是 DCF 和相對估值與當前股價之間的鴻溝，強烈指向了 **非財務風險**。Petrobras 作為一家巴西國有企業，其營運和股東價值受到巴西政府政策的直接影響。市場正在對以下因素進行大幅折價：
    *   **政治干預風險：** 政府可能干預燃料定價、資本支出決策和股息政策，損害少數股東利益。
    *   **治理風險：** 國有企業的治理結構可能不如純粹的私營企業透明和高效。
    *   **新興市場風險：** 巴西宏觀經濟和政治穩定性帶來的額外風險。
4.  **分析師目標價 ($15.09) 介於 DDM 和相對估值模型的低端之間。** 這表明分析師可能也考慮了上述風險，並給予了一定的折價，但仍認為存在一定的上漲空間。

**綜合結論：**

從純粹的基本面和營運角度來看，Petrobras (PBR-A) 似乎被市場嚴重低估。其強勁的自由現金流生成能力和極低的估值倍數（如 P/E、P/FCF、EV/EBITDA）表明其內在價值遠高於當前股價。

然而，這種巨大的折價並非沒有原因。市場正在對 Petrobras 固有的 **政治風險和治理風險** 進行大幅定價。巴西政府對公司的干預能力，尤其是在股息政策和資本配置方面的影響，是投資者最大的擔憂。DDM 模型較低的估值結果也印證了市場對其未來股息穩定性的悲觀預期。

**投資建議：**

PBR-A 是一個典型的「價值陷阱」或「深度價值」案例。對於願意承擔高政治風險的投資者來說，它可能提供巨大的潛在上漲空間和高股息收益（如果股息政策穩定）。但對於風險規避型投資者，其不確定性過高。

考慮到其強勁的基本面被政治風險嚴重壓制，一個更為現實的合理價位區間應該介於分析師目標價和相對估值模型的保守端之間。

*   **保守估值 (考慮高政治風險)：$12.00 - $18.00**
