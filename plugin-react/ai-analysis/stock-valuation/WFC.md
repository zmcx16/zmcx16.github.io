### 3. 估值方法與結果

#### 3.1. 折現現金流模型 (Discounted Cash Flow, DCF)

對於銀行業，傳統的自由現金流 (FCFF) 模型可能不完全適用，因為銀行的「資本支出」和「營運資金」概念與一般企業不同。然而，由於用戶明確要求，我們將嘗試使用自由現金流 (FCF) 數據。鑑於 WFC 的 FCF 波動較大且近期為負值，我們將基於其淨收入和 P/FCF 數據進行推算，並對未來 FCF 增長做出合理假設。

**挑戰**: 銀行業的 FCF 波動性大，且其定義與一般產業不同。我們將使用簡化的 FCF 估算方法，並強調其局限性。

**估算 FCF**:
*   2024 年淨收入 (Net Income): $186.06 億美元
*   2024 年 FCF: $30.4 億美元
*   P/FCF: 8.28 (來自提供數據)

由於 2025 年 Q3 的 FCF 為負值，且年度 FCF 波動大，直接預測 FCF 增長存在困難。我們將嘗試從淨收入角度推導 FCF，或使用更穩定的指標。

**替代方法：基於淨收入和再投資率的 FCF 估算 (簡化)**
對於銀行，更常見的是使用股息折現模型或殘餘收益模型。如果堅持 DCF，我們需要對 FCF 進行更穩定的假設。
考慮到 WFC 的淨收入相對穩定，我們可以使用以下簡化公式：
FCFE = Net Income - (1 - Payout Ratio) * Net Income = Net Income * Payout Ratio (如果假設所有未派發的利潤都用於再投資，且再投資不產生 FCF)
或者，FCFE = Net Income - Net Capital Expenditures - Change in Non-Cash Working Capital + Net Borrowing.
對於銀行，資本支出通常較低，主要現金流來自營運。

鑑於提供的數據，我們將嘗試從 P/FCF 反推 FCF，並假設其在預測期內趨於穩定增長。
*   市值 (Market Cap): $2967.1 億美元
*   P/FCF: 8.28
*   推算 TTM FCF = Market Cap / P/FCF = $2967.1 億 / 8.28 = $358.35 億美元。
    *   **注意**: 這個推算與搜尋結果中的 TTM FCF ($-142.18 億美元) 存在巨大差異。這表明 P/FCF 數據可能基於不同的 FCF 定義或時間點，或者 FCF 本身極不穩定。鑑於此，直接使用 FCF 進行 DCF 估值對於銀行業風險較高。

**由於銀行業 FCF 的特殊性及數據的矛盾，傳統 DCF 模型在此可能不適用或需要更詳細的財務報表數據。我們將轉向更適合銀行業的股息折現模型和相對估值。**

#### 3.2. 股息折現模型 (Dividend Discount Model, DDM)

DDM 適用於穩定派發股息且股息可預測的公司，Wells Fargo 符合此條件。

**假設**:
*   當前股息 (Annual Dividend): $1.80 (每股 $0.45 季度股息 * 4)
*   股息增長率 (短期 5 年): 10% (參考過去 1-3 年的增長率)
*   永續股息增長率 (長期): 2.5%
*   股權成本 (Cost of Equity, Ke): 9.55% (來自 WACC 計算中的數據)

**計算步驟**:

1.  **預測未來 5 年股息**:
    *   D0 = $1.80
    *   D1 = $1.80 * (1 + 0.10) = $1.98
    *   D2 = $1.98 * (1 + 0.10) = $2.18
    *   D3 = $2.18 * (1 + 0.10) = $2.40
    *   D4 = $2.40 * (1 + 0.10) = $2.64
    *   D5 = $2.64 * (1 + 0.10) = $2.90

2.  **計算第 5 年末的終端價值 (Terminal Value, TV)**:
    *   TV = D6 / (Ke - g_long_term)
    *   D6 = D5 * (1 + g_long_term) = $2.90 * (1 + 0.025) = $2.97
    *   TV = $2.97 / (0.0955 - 0.025) = $2.97 / 0.0705 = $42.13

3.  **將未來股息和終端價值折現回現值**:
    *   PV(D1) = $1.98 / (1 + 0.0955)^1 = $1.81
    *   PV(D2) = $2.18 / (1 + 0.0955)^2 = $1.82
    *   PV(D3) = $2.40 / (1 + 0.0955)^3 = $1.82
    *   PV(D4) = $2.64 / (1 + 0.0955)^4 = $1.82
    *   PV(D5) = $2.90 / (1 + 0.0955)^5 = $1.82
    *   PV(TV) = $42.13 / (1 + 0.0955)^5 = $26.70

4.  **DDM 估值**:
    *   DDM 估值 = PV(D1) + PV(D2) + PV(D3) + PV(D4) + PV(D5) + PV(TV)
    *   DDM 估值 = $1.81 + $1.82 + $1.82 + $1.82 + $1.82 + $26.70 = **$35.79**

**DDM 估值結果**: $35.79

**合理價位區間 (DDM)**:
考慮到股息增長率和股權成本的敏感性，我們可以設定一個區間。
*   如果短期股息增長率為 8%，長期為 2%：估值會更低。
*   如果短期股息增長率為 12%，長期為 3%：估值會更高。

DDM 估值 $35.79 遠低於當前股價 $94.31。這可能表明：
1.  市場對 WFC 的增長預期遠高於我們保守的股息增長假設。
2.  DDM 模型對於 WFC 可能存在局限性，或者其股權成本 (Ke) 需要更精確的計算。
3.  銀行業的估值通常也考慮其資產負債表健康狀況和盈利能力，而不僅僅是股息。
