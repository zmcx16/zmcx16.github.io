### 1. 業務背景與最新動態分析

*   **資產上限（Asset Cap）重大進展：** 這是 WFC 最核心的催化劑。最新消息顯示，聯準會（Fed）已初步認可富國銀行的風險管理改革，市場預期維持了 6 年之久的 1.95 兆美元資產上限有望在 2025 年內完全解除。這將釋放其資產負債表的增長潛力。
*   **財務表現：** 2024 全年 EPS 表現穩健。雖然淨利息收入（NII）受降息預期影響有所承壓，但非利息收入（手續費、投行業務）增長強勁。
*   **資本回饋：** WFC 擁有極高的資本充足率，持續進行大規模股份回購（2024 年回購約 120 億美元）並穩定提高股息。

### 2. 估值模型計算

#### 模型一：股息折現模型 (Dividend Discount Model, DDM)
對於銀行股，股息是衡量股東回報的核心。WFC 近期將季度股息提高至每股 $0.40（年化 $1.60）。

*   **假設：**
    *   **當前股息 (D0):** $1.60
    *   **股息增長率 (g):** 考慮到資產上限解除後的盈利釋放，預計前 5 年增長 8%，長期穩定增長率設為 4%。
    *   **股權成本 (Ke):** 使用 CAPM 模型。無風險利率 4.2% + Beta 1.1 * 市場風險溢價 5.5% ≈ **10.25%**。
*   **計算（兩階段 DDM）：**
    *   前 5 年現值加總 ≈ $6.50
    *   終值現值 (Terminal Value) = [$1.60 * (1.08)^5 * (1.04)] / (0.1025 - 0.04) / (1.1025)^5 ≈ $24.50 / 1.63 ≈ $15.03 (此處簡化計算)
*   **DDM 估值結果：約 $78.50 - $82.00**

#### 模型二：剩餘收益模型 (Residual Income Model)
這是銀行業最精準的估值法，因為銀行資產負債表與現金流難以用傳統 DCF 衡量。

*   **假設：**
    *   **當前每股淨資產 (BVPS):** 約 $53.15 (由 P/B 1.63 與股價 $86.64 推算)。
    *   **預期 ROE:** 12% (數據顯示為 11.85%，隨效率提升有望達 13%)。
    *   **股權成本 (Ke):** 10.25%。
    *   **長期增長率:** 3.5%。
*   **計算：**
    *   價值 = BVPS + [ (ROE - Ke) * BVPS ] / (Ke - g)
    *   價值 = 53.15 + [ (0.12 - 0.1025) * 53.15 ] / (0.1025 - 0.035)
    *   價值 = 53.15 + 0.93 / 0.0675 ≈ 53.15 + 13.78 = $66.93
    *   *註：若 ROE 提升至 14%（資產上限解除後）：價值將升至 **$82.60**。*

#### 模型三：相對估值法 (Multiples Analysis)
參考同行業（JPM, BAC, C）的估值水平：

*   **P/E 估值：** 給予 WFC 歷史平均及行業中位數約 12x 的 Forward P/E。
    *   預估 2025 EPS $8.00 (基於 Forward P/E 10.84 推算)
    *   估值 = 12 * $8.00 = **$96.00**
*   **P/B 估值：** 考慮到 ROE 改善，合理 P/B 應在 1.5x - 1.7x 之間。
    *   估值 = 1.6 * $53.15 = **$85.04**
