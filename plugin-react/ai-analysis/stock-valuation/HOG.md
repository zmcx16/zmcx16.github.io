*   股權成本 (Ke) = 11.35%。

由於公司2024年 EPS 下降，2025年營收展望不佳，股息增長存在不確定性。如果假設股息零增長 (Gordon Growth Model 的簡化版，g=0)，則：

*   每股估值 = DPS / Ke = $0.80 / 0.1135 = $7.05

如果假設股息在未來幾年內保持穩定，然後以一個較低的永續增長率增長 (例如 1%)：

*   DDM 估值 = DPS (1+g) / (Ke - g) = $0.80 * (1 + 0.01) / (0.1135 - 0.01) = $0.808 / 0.1035 = $7.81

**DDM 估值結果與合理價位區間：**

*   **DDM 估值：$7.05 - $7.81**
*   **合理價位區間：$7.00 - $8.00**

### **4. 不同模型間的估值結果比較與綜合結論**

**綜合結論：**

從上述三種估值模型來看，Harley-Davidson (HOG) 的當前股價 $17.97 似乎被顯著高估。

*   **DCF 模型**給出了最低的估值，約為 $4.03。這主要反映了公司近期業績下滑、未來增長預期較低以及較高的折現率。如果公司無法在未來幾年內顯著改善營收增長和利潤率，其內在價值將受到嚴重影響。
*   **相對估值模型**的結果範圍較廣，但即使是較高的估值 (例如基於 Forward P/E 的 $30.525，如果僅考慮 HOG 自身倍數)，也遠低於當前股價。如果我們更側重於 EV/EBITDA 和 P/B 等更穩定的倍數，估值約在 $4.00 左右，與 DCF 結果接近。這表明與同業相比，HOG 的估值倍數可能偏高，或者其盈利能力和增長前景不如同業。
*   **DDM 模型**的估值約為 $7.00 - $8.00。由於 HOG 支付穩定股息，且股息率較高，DDM 提供了一個相對較高的估值。然而，股息的可持續性取決於未來的盈利能力，而公司目前的盈利壓力可能會影響未來的股息政策。

**綜合來看，HOG 的合理價位可能落在 $4.00 - $8.00 之間。** 當前股價 $17.97 遠高於所有模型的估值結果。這可能意味著市場對 Harley-Davidson 的品牌價值、未來轉型潛力 (如 LiveWire 電動摩托車) 或其「Hardwire」策略的成功抱有更高的期望，但這些期望在目前的財務數據和保守假設下未能得到充分支持。

**風險因素：**

*   **市場環境挑戰：** 高利率和消費者信心下降持續影響可支配產品需求。
*   **人口結構變化：** 核心重型摩托車買家群體的老齡化。
*   **競爭加劇：** 來自 Indian Motorcycle、日本和歐洲製造商以及電動摩托車新興品牌的競爭。
*   **電動化轉型：** LiveWire 仍處於虧損，電動摩托車市場的成功轉型存在不確定性。
*   **關稅問題：** 貿易政策的不確定性可能影響公司盈利。

**投資建議：**

基於本次估值分析，HOG 的當前股價相對於其內在價值可能被高估。投資者應謹慎對待，並密切關注公司未來財報中營收增長、利潤率改善以及電動化戰略的執行情況。除非公司能夠展現出強勁的盈利復甦和可持續的增長，否則目前的股價風險較高。

### **5. 與同產業可比公司的估值比較**

如前所述，Harley-Davidson 在重型巡航車市場具有獨特的品牌地位。

*   **Polaris Inc. (PII):** 雖然 PII 的 P/E 較高，但其業務範圍更廣，包括 ATV、雪地摩托等動力運動產品，且 Indian Motorcycle 在巡航車市場是 HOG 的主要競爭對手。PII 的 EV/EBITDA (19.53) 顯著高於 HOG (5.47)，這可能反映了市場對 PII 更多元化業務和增長前景的更高預期。
*   **Honda (HMC) 和 Yamaha (YMHA.F):** 這兩家日本巨頭在全球摩托車市場佔據主導地位，產品線從小型通勤車到高性能運動摩托車應有盡有。它們的 EV/EBITDA 倍數 (Honda 8.4, Yamaha 6.9) 也高於 HOG，表明市場對其整體業務規模、技術實力及全球市場份額給予更高的估值。

HOG 較低的估值倍數 (特別是 EV/EBITDA) 可能反映了市場對其增長潛力、盈利穩定性以及在電動化轉型方面的擔憂。儘管 HOG 在美國 Touring 市場佔有率很高，但整體摩托車銷量下滑和高利率環境對其核心業務造成壓力。與多元化且在全球市場表現強勁的競爭對手相比，HOG 的估值顯得較為保守，這與我們 DCF 和 DDM 的結果相符。
