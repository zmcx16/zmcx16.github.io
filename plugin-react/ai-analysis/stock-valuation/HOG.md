*   債務/股權比率 (Debt/Eq): 0.97 (來自您提供的數據，與搜尋結果 0.97 一致)。
*   稅率 (Effective Tax Rate): 28% (來自 2025 年數據)。

**WACC 計算步驟:**

1.  **股權成本 (Ke):**
    *   Ke = 無風險利率 + Beta * 市場風險溢價
    *   Ke = 4.2% + 1.3 * 5.5% = 4.2% + 7.15% = 11.35%

2.  **債務成本 (Kd):**

    *   稅後債務成本 = Kd * (1 - 稅率) = 6.0% * (1 - 28%) = 6.0% * 0.72 = 4.32%

3.  **WACC:**
    *   WACC = (E / (E+D)) * Ke + (D / (E+D)) * Kd * (1 - 稅率)
    *   其中 E 為股權市值，D 為債務市值。
    *   由於 Debt/Eq = 0.97，我們可以假設 E = 1，D = 0.97。
    *   WACC = (1 / (1 + 0.97)) * 11.35% + (0.97 / (1 + 0.97)) * 4.32%
    *   WACC = (0.5076) * 11.35% + (0.4924) * 4.32%

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

**估值假設:**

*   **當前股息 (D0):** 根據 $19.46 股價和 4.66% 股息率，D0 = 19.46 * 0.0466 = $0.906。
*   **股息增長率 (g):** 考慮到 HOG 近期的營運挑戰和保守的 2026 年展望，我將採用一個較低的長期股息增長率。雖然公司有回購股票的計劃，但盈利的不確定性使得高增長率難以維持。我將假設長期股息增長率為 1.5%。
*   **股權成本 (Ke):** 11.35% (與 DCF 模型相同)。

**DDM 計算:**

*   DDM 估值 = D0 * (1 + g) / (Ke - g)
*   DDM 估值 = 0.906 * (1 + 0.015) / (0.1135 - 0.015)
*   DDM 估值 = 0.906 * 1.015 / 0.0985
*   DDM 估值 = 0.9205 / 0.0985 = $9.35

HOG 的 P/E (7.47) 遠低於 Polaris (80.78) 和 Yamaha (74.9)，這是一個顯著的差異。這可能反映了：

*   **市場預期:** 市場對 HOG 未來增長的預期可能較為悲觀，而對其競爭對手則相對樂觀。

HOG 的 P/S (0.49) 與 Polaris (0.55) 和 Yamaha (0.5) 較為接近，這表明在營收層面，市場對 HOG 的估值與其競爭對手大致相當。這可能反映了 HOG 強大的品牌價值和市場份額，即使在盈利能力受損的情況下，其營收規模仍能獲得一定的市場認可。

總體而言，HOG 在盈利倍數 (P/E, EV/EBITDA) 上顯著落後於其主要競爭對手，但在營收倍數 (P/S) 上則相對接近。這表明 HOG 的主要挑戰在於將營收轉化為穩定的利潤和現金流。
