### 2. 估值方法與假設

鑑於 HOG 的業務性質和市場狀況，我們將應用折現現金流模型 (DCF) 和相對估值法。股息折現模型 (DDM) 也將適用，因為公司有穩定的股息支付。

**共同假設：**

*   **市場風險溢價 (Market Risk Premium, MRP):** 假設為 5.5%。

*   **預測期間 (Explicit Forecast Period):** 5 年 (2026-2030)。

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**估值假設：**

*   **WACC (加權平均資本成本):**
    *   **Beta (β):** 根據 MarketBeat，HOG 的 Beta 為 1.39，Stock Analysis 顯示為 1.42。我們採用 1.4。
    *   **股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM)。
        *   Ke = 無風險利率 + Beta \* 市場風險溢價
        *   Ke = 4.2% + 1.4 \* 5.5% = 4.2% + 7.7% = 11.9%
    *   **債務成本 (Cost of Debt, Kd):** 根據 GuruFocus，HOG 的債務成本為 6.55%。
    *   **債務市值 (Market Value of Debt, D):** 根據提供的 Debt/Eq 1.46 和市值 23.7 億美元，股權市值 (E) = 23.7 億美元。D = 1.46 \* 23.7 億美元 = 34.602 億美元。
    *   **WACC 計算:**
        *   WACC = (E / (E + D)) \* Ke + (D / (E + D)) \* Kd \* (1 - 稅率)
        *   E + D = 23.7 億 + 34.602 億 = 58.302 億美元
        *   WACC = (23.7 / 58.302) \* 11.9% + (34.602 / 58.302) \* 6.55% \* (1 - 0.24)
        *   WACC = 0.4065 \* 11.9% + 0.5935 \* 6.55% \* 0.76
        *   WACC = 4.837% + 2.952% = 7.789%
        *   我們將採用約 **7.8%** 作為 WACC。這與 GuruFocus 的 7.2% 和 Stock Analysis 的 4.16% 有所不同，但考慮到當前利率環境和 HOG 的債務水平，7.8% 是一個較為合理的保守估計。
*   **收入增長率 (Revenue Growth Rate):** 考慮到公司面臨的宏觀逆風和產業挑戰，以及 2024 年 HDMC 收入預計持平至下降 9%，2025 年 HDMC 收入預計持平至下降 5%，我們將採用保守的增長預期。
    *   2026 年: -3% (延續近期下降趨勢)
    *   2027 年: -1%
    *   2028 年: 0% (趨於穩定)
    *   2029 年: 1%
    *   2030 年: 1.5%
*   **營業利潤率 (Operating Margin):** 根據提供的數據，營業利潤率為 9.78%。2023 年 HDMC 營業利潤率為 13.6%，2024 年為 6.7%，2025 年預計為 7.0% 至 8.0%。我們將採用一個逐步恢復但保守的預期。
    *   2026 年: 7.0%
    *   2027 年: 7.5%
    *   2028 年: 8.0%
    *   2029 年: 8.5%
    *   2030 年: 9.0%
*   **資本支出 (Capital Expenditures, CapEx):** 2024 年資本投資預計為 2.25 億至 2.5 億美元。我們將假設其佔收入的百分比保持相對穩定，或略有下降。
*   **折舊與攤銷 (Depreciation & Amortization, D&A):** 假設其與 CapEx 相關，或佔收入的穩定百分比。
*   **營運資本變動 (Change in Net Working Capital, NWC):** 假設其佔收入的穩定百分比。

由於缺乏詳細的歷史自由現金流數據和未來預測，這裡將提供一個簡化的 DCF 框架和估值範圍。

**簡化 DCF 估值步驟：**

1.  **計算預測期內的自由現金流 (FCFF):**
    *   FCFF = 營業收入 \* (1 - 營業利潤率) \* (1 - 稅率) + D&A - CapEx - NWC 變動
2.  **計算終值 (Terminal Value, TV):**
    *   TV = (FCFF_n \* (1 + g)) / (WACC - g)
    *   其中 FCFF_n 是預測期最後一年的 FCFF，g 是長期增長率。
3.  **計算企業價值 (Enterprise Value, EV):**
    *   EV = 預測期內 FCFF 的現值 + TV 的現值
4.  **計算股權價值 (Equity Value):**
    *   Equity Value = EV + 現金及等價物 - 債務
5.  **計算每股價值 (Intrinsic Value Per Share):**
    *   每股價值 = 股權價值 / 流通股數

**DCF 估值結果 (基於上述假設的估計範圍):**

考慮到 HOG 收入下降的趨勢和產業逆風，以及其金融服務部門的影響，DCF 估值可能相對保守。

*   **合理價位區間 (DCF):** 預計在 **18.00 美元 - 24.00 美元** 之間。

#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

HOG 支付穩定的季度股息，因此 DDM 適用。

**估值假設：**

*   **當前年度股息 (D0):** 0.18 美元/季度 \* 4 = 0.72 美元/年。
*   **股息增長率 (g):** 考慮到公司面臨的挑戰，我們將採用一個非常保守的長期股息增長率。過去 12 個月股息增長率為 4.40%，但長期來看增長不穩定。我們將假設一個較低的長期增長率。
    *   保守情境: 0.5%
    *   中性情境: 1.0%
    *   樂觀情境: 1.5%
*   **股權成本 (Ke):** 11.9% (來自 DCF 模型)。

**DDM 計算 (Gordon Growth Model):**

*   股價 = D1 / (Ke - g)
*   D1 = D0 \* (1 + g)

**DDM 估值結果：**

*   **保守情境 (g = 0.5%):**
    *   D1 = 0.72 \* (1 + 0.005) = 0.7236 美元
    *   股價 = 0.7236 / (0.119 - 0.005) = 0.7236 / 0.114 = **6.35 美元**
*   **中性情境 (g = 1.0%):**
    *   D1 = 0.72 \* (1 + 0.01) = 0.7272 美元
    *   股價 = 0.7272 / (0.119 - 0.01) = 0.7272 / 0.109 = **6.67 美元**
*   **樂觀情境 (g = 1.5%):**
    *   D1 = 0.72 \* (1 + 0.015) = 0.7308 美元
    *   股價 = 0.7308 / (0.119 - 0.015) = 0.7308 / 0.104 = **7.03 美元**

DDM 估值結果顯著低於當前股價和分析師目標價。這可能表明 DDM 在 HOG 的情況下可能不是最合適的估值方法，或者市場對其未來股息增長有更高的預期，或者其價值主要來自於非股息現金流。HOG 的派息率為 17.57%，這表明公司有能力支付更多股息，但選擇將大部分收益再投資或用於其他用途。

*   **合理價位區間 (DDM):** 預計在 **6.00 美元 - 7.50 美元** 之間。

### 3. 綜合結論與比較

**綜合分析：**

1.  **DCF 模型** 提供了一個基於公司未來現金流的內在價值估計。考慮到 HOG 面臨的收入下降和產業逆風，DCF 估值相對保守，但其範圍與當前股價接近。
2.  **相對估值** 顯示 HOG 的市盈率、市淨率和市銷率倍數相對較低，可能表明市場對其估值偏低。這與分析師的平均目標價 25.86 美元 存在一定的上漲空間相符。
3.  **DDM 模型** 的結果顯著低於其他模型和當前股價。這可能是因為 DDM 假設股息穩定增長，而 HOG 的股息增長歷史並不穩定，且其派息率較低，表明公司可能將更多現金流用於再投資或債務償還，而非股息增長。因此，對於 HOG 而言，DDM 可能無法完全捕捉其內在價值。

**綜合結論：**

綜合來看，HOG 的估值呈現出複雜的局面。儘管公司面臨宏觀經濟逆風、人口結構變化和產業競爭加劇等挑戰，但其較低的相對估值倍數（P/E、P/B、P/S）以及穩定的自由現金流產生能力（P/FCF 6.06）表明其可能被市場低估。

分析師的平均目標價為 26.44 美元，這比當前股價 20.09 美元有顯著的上漲空間。然而，公司在 2024 年和 2025 年的收入指引均為持平或下降，這反映了其業務的挑戰性。

考慮到 DCF 和相對估值模型的結果，以及分析師的普遍看法，HOG 的合理價位區間可能在 **22.00 美元至 27.00 美元** 之間。當前股價 20.09 美元位於此區間的下限附近，表明存在一定的上漲潛力，但投資者應充分考慮公司面臨的風險和挑戰。

**與同產業可比公司的估值比較：**

HOG 的 P/E (4.87) 顯著低於其主要競爭對手 Polaris Inc. (PII) 的 P/E (約 10-15)。這可能反映了市場對 HOG 增長前景的擔憂，或者其業務模式（如對傳統重型摩托車的依賴）使其估值折扣。然而，HOG 較低的 P/B (0.66) 和 P/S (0.51) 也可能表明其資產和銷售額被低估。

總體而言，HOG 似乎在當前市場中被低估，但其未來的表現將很大程度上取決於其「Hardwire」戰略的執行情況，包括新產品的推出（如 LiveWire 電動摩托車）、成本控制以及應對不斷變化的消費者偏好的能力。
