### 3. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型旨在估計公司未來自由現金流的現值。

**步驟 1: 計算加權平均資本成本 (WACC)**

*   **股權成本 (Cost of Equity, Ke):** 使用資本資產定價模型 (CAPM)。
    *   Ke = 無風險利率 + Beta × 市場風險溢價
    *   Ke = 4.5% + 1.42 × 5.5% = 4.5% + 7.81% = 12.31%
*   **債務成本 (Cost of Debt, Kd):** 假設公司平均借款利率為 6%。
*   **市場價值權重 (Market Value Weights):**
    *   市值 (Market Cap): 21.8 億美元
    *   總債務 (Total Debt): 根據提供的數據，Debt/Eq 為 0.97。因此，債務 = 0.97 * 21.8 億美元 = 21.146 億美元。
    *   總資本 = 21.8 億美元 + 21.146 億美元 = 42.946 億美元
    *   股權權重 (We) = 21.8 / 42.946 = 0.5076
    *   債務權重 (Wd) = 21.146 / 42.946 = 0.4924
*   **WACC = (We × Ke) + (Wd × Kd × (1 - 稅率))**
    *   WACC = (0.5076 × 12.31%) + (0.4924 × 6% × (1 - 20%))
    *   WACC = 6.25% + (0.4924 × 6% × 0.8)
    *   WACC = 6.25% + 2.36% = 8.61%

### 5. 股息折現模型 (Dividend Discount Model, DDM)

HOG 支付股息，因此 DDM 適用。

**假設:**

*   **當前股息 (D0):** 股息率 3.74%，股價 19.47 美元。D0 = 19.47 美元 × 0.0374 = 0.728 美元。
*   **股息增長率 (g):** 考慮到公司面臨的挑戰和重組，我們假設短期內股息增長緩慢，長期穩定增長。
    *   前 3 年 (2026-2028): 1%
    *   之後永續增長率 (g_terminal): 1.5% (與 DCF 永續增長率一致)
*   **股權成本 (Ke):** 12.31% (與 DCF 計算一致)

**步驟 1: 預測未來股息**

*   D1 (2026) = D0 × (1 + 1%) = 0.728 × 1.01 = 0.735 美元
*   D2 (2027) = D1 × (1 + 1%) = 0.735 × 1.01 = 0.742 美元
*   D3 (2028) = D2 × (1 + 1%) = 0.742 × 1.01 = 0.749 美元

**步驟 2: 計算終值 (Terminal Value of Dividends)**

*   TV_DDM = D3 × (1 + g_terminal) / (Ke - g_terminal)
*   TV_DDM = 0.749 × (1 + 1.5%) / (12.31% - 1.5%)
*   TV_DDM = 0.749 × 1.015 / 0.1081
*   TV_DDM = 0.7602 / 0.1081 = 7.03 美元 (這是 2028 年末的股息終值)

**步驟 3: 計算現值**

*   P0 = D1 / (1 + Ke)^1 + D2 / (1 + Ke)^2 + (D3 + TV_DDM) / (1 + Ke)^3
*   P0 = 0.735 / (1.1231)^1 + 0.742 / (1.1231)^2 + (0.749 + 7.03) / (1.1231)^3
*   P0 = 0.735 / 1.1231 + 0.742 / 1.2614 + 7.779 / 1.4164
*   P0 = 0.654 + 0.588 + 5.492 = 6.734 美元

**DDM 估值結果:**

*   **合理價位區間:** 6.00 - 7.50 美元 (此估值顯著低於當前股價，可能反映了市場對未來股息增長預期較低，或 DDM 模型在此情況下不完全適用，因為公司處於轉型期，現金流可能優先用於投資而非股息增長)
*   **DDM 估值:** **6.73 美元**
