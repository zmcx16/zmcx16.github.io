### 3. 估值模型計算

#### 3.1. 加權平均資本成本 (WACC) 計算

*   **無風險利率 (Rf):** 4.5% (美國 10 年期國債收益率的合理估計)。
*   **市場風險溢價 (MRP):** 5.5%。
*   **Beta (β):** 1.15。
*   **股權成本 (Ke):** Ke = Rf + β \* MRP = 4.5% + 1.15 \* 5.5% = 4.5% + 6.325% = 10.825%。

*   **債務成本 (Kd):** 假設為 6%。
*   **稅率 (Tax Rate):** 25%。
*   **稅後債務成本:** Kd \* (1 - Tax Rate) = 6% \* (1 - 0.25) = 6% \* 0.75 = 4.5%。

*   **市值 (E):** 24.4 億美元 (使用最新市值)。
*   **債務市值 (D):** 根據提供的數據，Debt/Eq 為 1.46，LT Debt/Eq 為 0.9。
    *   如果股權為 E，則總債務 D = Debt/Eq \* E = 1.46 \* 24.4 億美元 = 35.624 億美元。
    *   總資本 = E + D = 24.4 億美元 + 35.624 億美元 = 60.024 億美元。
    *   股權權重 = E / (E+D) = 24.4 / 60.024 = 0.4065。
    *   債務權重 = D / (E+D) = 35.624 / 60.024 = 0.5935。

*   **WACC:** (股權權重 \* Ke) + (債務權重 \* 稅後債務成本)
    *   WACC = (0.4065 \* 10.825%) + (0.5935 \* 4.5%)
    *   WACC = 4.400% + 2.671% = 7.071%。
    *   我們將使用約 **7.1%** 作為 WACC。

#### 3.2. 折現現金流模型 (DCF)

**挑戰:** 由於缺乏詳細的歷史財務報表（如完整的損益表、資產負債表和現金流量表），精確的 FCF 預測會非常困難。特別是 HDFS 交易對現金流的影響，使得直接從營收預測 FCF 變得複雜。

**簡化方法:** 鑑於資訊限制，我將嘗試基於營收和營業利潤率來估算 NOPAT，並對 CapEx 和 NWC 進行簡化假設。

**假設回顧:**
*   **預測期間:** 2026-2030 年 (5 年)。
*   **永續增長率 (g):** 2%。
*   **WACC:** 7.1%。
*   **2025 年預計營收:** 37.3 億美元。
*   **營收增長率:** 2026: 2%；2027-2028: 1.5%；2029-2030: 1%。
*   **營業利潤率 (Oper. Margin):** 9.78% (來自提供數據)。
*   **稅率:** 25%。
*   **CapEx 佔營收比:** 3%。
*   **D&A 佔營收比:** 2.5%。
*   **NWC 變動佔營收比:** 1%。

**計算步驟:**

1.  **預測營收 (Revenue):**
    *   2025 (Base): 37.3 億美元
    *   2026: 37.3 \* (1 + 0.02) = 38.046 億美元
    *   2027: 38.046 \* (1 + 0.015) = 38.617 億美元
    *   2028: 38.617 \* (1 + 0.015) = 39.196 億美元
    *   2029: 39.196 \* (1 + 0.01) = 39.588 億美元
    *   2030: 39.588 \* (1 + 0.01) = 39.984 億美元

2.  **預測稅後營業利潤 (NOPAT):**
    *   NOPAT = Revenue \* Operating Margin \* (1 - Tax Rate)
    *   2026: 38.046 \* 0.0978 \* (1 - 0.25) = 38.046 \* 0.0978 \* 0.75 = 2.789 億美元
    *   2027: 38.617 \* 0.0978 \* 0.75 = 2.831 億美元
    *   2028: 39.196 \* 0.0978 \* 0.75 = 2.874 億美元
    *   2029: 39.588 \* 0.0978 \* 0.75 = 2.903 億美元
    *   2030: 39.984 \* 0.0978 \* 0.75 = 2.932 億美元

3.  **預測自由現金流 (FCF):**
    *   FCF = NOPAT + D&A - CapEx - Change in NWC
    *   D&A = Revenue \* 2.5%
    *   CapEx = Revenue \* 3%
    *   Change in NWC = (Current Year Revenue - Previous Year Revenue) \* 1%

    *   **2026:**
        *   D&A: 38.046 \* 0.025 = 0.951 億美元
        *   CapEx: 38.046 \* 0.03 = 1.141 億美元
        *   Change in NWC: (38.046 - 37.3) \* 0.01 = 0.007 億美元
        *   FCF = 2.789 + 0.951 - 1.141 - 0.007 = 2.192 億美元

    *   **2027:**
        *   D&A: 38.617 \* 0.025 = 0.965 億美元
        *   CapEx: 38.617 \* 0.03 = 1.159 億美元
        *   Change in NWC: (38.617 - 38.046) \* 0.01 = 0.006 億美元
        *   FCF = 2.831 + 0.965 - 1.159 - 0.006 = 2.631 億美元

    *   **2028:**
        *   D&A: 39.196 \* 0.025 = 0.980 億美元
        *   CapEx: 39.196 \* 0.03 = 1.176 億美元
        *   Change in NWC: (39.196 - 38.617) \* 0.01 = 0.006 億美元
        *   FCF = 2.874 + 0.980 - 1.176 - 0.006 = 2.672 億美元

    *   **2029:**
        *   D&A: 39.588 \* 0.025 = 0.990 億美元
        *   CapEx: 39.588 \* 0.03 = 1.188 億美元
        *   Change in NWC: (39.588 - 39.196) \* 0.01 = 0.004 億美元
        *   FCF = 2.903 + 0.990 - 1.188 - 0.004 = 2.701 億美元

    *   **2030:**
        *   D&A: 39.984 \* 0.025 = 0.999 億美元
        *   CapEx: 39.984 \* 0.03 = 1.199 億美元
        *   Change in NWC: (39.984 - 39.588) \* 0.01 = 0.004 億美元
        *   FCF = 2.932 + 0.999 - 1.199 - 0.004 = 2.728 億美元

4.  **計算終值 (Terminal Value, TV):**
    *   TV = FCF_2030 \* (1 + g) / (WACC - g)
    *   TV = 2.728 \* (1 + 0.02) / (0.071 - 0.02)
    *   TV = 2.728 \* 1.02 / 0.051 = 2.78256 / 0.051 = 54.56 億美元

5.  **折現 FCF 和 TV:**
    *   折現因子 (DF) = 1 / (1 + WACC)^n
    *   2026 FCF 折現值: 2.192 / (1 + 0.071)^1 = 2.192 / 1.071 = 2.047 億美元
    *   2027 FCF 折現值: 2.631 / (1 + 0.071)^2 = 2.631 / 1.147 = 2.294 億美元
    *   2028 FCF 折現值: 2.672 / (1 + 0.071)^3 = 2.672 / 1.228 = 2.176 億美元
    *   2029 FCF 折現值: 2.701 / (1 + 0.071)^4 = 2.701 / 1.315 = 2.054 億美元
    *   2030 FCF 折現值: 2.728 / (1 + 0.071)^5 = 2.728 / 1.408 = 1.937 億美元
    *   TV 折現值: 54.56 / (1 + 0.071)^5 = 54.56 / 1.408 = 38.75 億美元

6.  **企業價值 (Enterprise Value, EV):**
    *   EV = 所有折現 FCF 之和 + 折現 TV

7.  **股權價值 (Equity Value):**

8.  **每股價值 (Per Share Value):**

#### 3.4. 股息折現模型 (DDM)

*   **股權成本 (Ke):** 10.825% (來自 WACC 計算)。
*   **股息增長率 (g):**
    *   公司預計明年 EPS 增長 28.78%。

    *   假設長期股息增長率為 3%。

**計算步驟 (Gordon Growth Model):**
*   DDM = D0 \* (1 + g) / (Ke - g)
*   DDM = 0.72 \* (1 + 0.03) / (0.10825 - 0.03)
*   DDM = 0.72 \* 1.03 / 0.07825
*   DDM = 0.7416 / 0.07825 = **9.48 美元/股**。

**DDM 估值結果:** 9.48 美元/股。

**合理價位區間:** 考慮到 DDM 對增長率和折現率的敏感性，以及 HOG 股息增長可能不穩定，這個估值可能偏低。如果股息增長率更高，估值會顯著上升。例如，如果股息增長率為 5%，則 DDM = 0.72 \* 1.05 / (0.10825 - 0.05) = 0.756 / 0.05825 = 12.98 美元/股。
