#### 2.3 股息折現模型 (Dividend Discount Model, DDM)

HOG有穩定的股息支付歷史。

**DDM 假設：**
*   **最新年度股息 (TTM):** 0.72美元/股
*   **股息增長率 (g)：** 過去12個月股息增長率為4.35%。 考慮到公司正在重組，未來幾年股息增長可能放緩，甚至可能維持不變或小幅下降。我將假設一個保守的長期股息增長率1%。
*   **所需報酬率 (r)：** 採用與WACC相同的10%。

**DDM 計算 (戈登增長模型):**
*   股價 = D1 / (r - g)
*   D1 = D0 * (1 + g) = 0.72 * (1 + 0.01) = 0.7272 美元
*   股價 = 0.7272 / (0.10 - 0.01) = 0.7272 / 0.09 = 8.08 美元

**DDM 估值結果：**
*   **合理價位：8.08 美元**
*   **合理價位區間：7.00 - 9.00 美元**

這個結果遠低於當前股價和DCF/相對估值，這可能表明市場預期HOG的股息增長率會更高，或者DDM模型對於HOG這種正在經歷轉型的公司可能不完全適用，因為其股息政策可能不完全反映其內在價值。此外，HOG的派息率為25.5%至26.12%，相對較低，表明公司有能力支付股息並將收益再投資於業務。
