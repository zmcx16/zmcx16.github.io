### 3. 估值模型與結果

#### 3.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型估算公司在預測期間和永續經營階段產生的自由現金流，並將其折現回現在，從而得出公司價值。

**A. 預測自由現金流 (Free Cash Flow to Firm, FCFF) - 5年預測期**

| 增長率                |              | 2.0%       | 1.5%       | 1.0%       | 0.5%       | 0.0%       |
| **EBIT**              | $754         | $769.1      | $780.6      | $788.4      | $792.4      | $792.4      |
| EBIT Margin           | 13.0%        | 13.0%      | 13.0%      | 13.0%      | 13.0%      | 13.0%      |
| 稅率                  | 22%          | 22%        | 22%        | 22%        | 22%        | 22%        |
| **NOPAT**             | $588.1       | $599.9      | $608.9      | $614.9      | $618.1      | $618.1      |

*單位為百萬美元*

**B. 終端價值 (Terminal Value, TV)**

*   預測期後第一年的 FCFF ($FCFF_{Year 6}$) = $FCFF_{Year 5} \times (1 + g) = $587.6 \times (1 + 0.018) = $598.19 百萬
*   終端價值 (TV) = $FCFF_{Year 6} / (WACC - g) = $598.19 / (0.091 - 0.018) = $598.19 / 0.073 = $8,194.38 百萬
*   終端價值現值 (PV of TV) = $TV / (1 + WACC)^5 = $8,194.38 / (1 + 0.091)^5 = $8,194.38 / 1.547 = $5,296.96 百萬

**C. 企業價值與股權價值**

*   **總企業價值 (Enterprise Value) =** 預測期 FCFF 現值總和 + 終端價值現值
    $EV = ($521.1 + $485.3 + $449.6 + $414.7 + $380.0) + $5,296.96 = $2,250.7 + $5,296.96 = $7,547.66 百萬
*   **股權價值 (Equity Value) =** 企業價值 - 淨債務 + 現金及約當現金
    (這裡假設工業部門的淨債務已包含現金，或將現金 $2.5B 單獨加回)
    若使用 **淨債務** $4.0B：
    $Equity Value = $7,547.66 - $4,000 = $3,547.66 百萬
*   **每股價值 (Per Share Value) =** 股權價值 / 流通股數
    $3,547.66 / 139 = $25.52

**DCF 估值結果：** $25.52 / 股
**合理價位區間：** $22 - $29 (考慮 WACC 及長期增長率 ±0.5% 的敏感性分析)

*備註：HOG 的工業和金融服務部門的價值分離會更精確，但這裡採用的是基於合併財務報表的簡化 DCF，並嘗試對淨債務進行了區分。*

#### 3.3 股息折現模型 (Dividend Discount Model, DDM)

HOG 過去的股息政策經歷了削減和恢復，並非穩健增長的藍籌股。因此，DDM 可能不是最適合 HOG 的估值方法。然而，我們仍將嘗試進行計算。

*   **最近12個月股息 ($D_0$):** 約 $0.68 / 股 (季度股息 $0.17 x 4$)
*   **股權成本 ($R_e$):** 12.75% (來自 WACC 計算)
*   **股息長期增長率 ($g$):** 1.5% (保守估計，假設與終端增長率相似，但需注意 HOG 股息的波動性)

**使用戈登增長模型 (Gordon Growth Model):**
$DDM = D_0 \times (1 + g) / (R_e - g)$
$DDM = $0.68 \times (1 + 0.015) / (0.1275 - 0.015)$
$DDM = $0.6902 / 0.1125 = $6.13 / 股

**DDM 估值結果：** $6.13 / 股
**合理價位區間：** $5 - $8 (該模型對 $R_e$ 和 $g$ 的微小變化極為敏感，結果不穩定，僅供參考)

**結論：** DDM 估值結果遠低於當前股價和DCF/相對估值，這再次印證 HOG 並不適用於單純的戈登增長模型，其股息並非公司價值的主要驅動因素，也未能穩定增長。公司更多依賴於營收和自由現金流增長以及策略性資本配置 (如回購、投資 LiveWire 等)。

### 4. 綜合結論與比較

| 估值方法              | 估值結果 (每股) | 合理價位區間 (每股) |
| :-------------------- | :--------------- | :------------------ |
| 折現現金流 (DCF)      | $25.52          | $22 - $29           |
| 相對估值 (P/E)        | $31.50          | $28 - $35           |
| 相對估值 (EV/EBITDA)  | $18.49          | $15 - $22           |
| 股息折現模型 (DDM)    | $6.13           | 不適用 / 不穩定     |

**分析與綜合：**

*   **DCF 估值 ($25.52):** 反映了公司在我的假設下未來現金流的潛力。這個結果考慮了公司長期的增長前景和風險，給出了一個基礎價值。它相對於當前股價略低，暗示市場可能預期了比我假設更低的長期增長或更高的風險，或者我的現金流預測相對保守。
*   **相對估值 ($18.49 - $31.50):** 存在顯著差異。
    *   **P/E 估值 ($31.50):** 接近當前股價，表明如果 HOG 的未來 EPS 能夠達到預期並以行業平均的 P/E 交易，其價值將在此區間。
    *   **EV/EBITDA 估值 ($18.49):** 顯著低於當前股價和 DCF 估值。這通常是由於 HOG 的高額債務，特別是其金融服務部門的債務，在 EV/EBITDA 計算中被放大，而金融服務業務的貢獻則未能充分體現在工業部門的 EBITDA 中。這使得 EV/EBITDA 對於 HOG 而言，是一個相對複雜且可能產生誤導的指標，除非對工業和金融部門進行嚴格分離估值。
*   **DDM 估值 ($6.13):** 極低且不穩定，證實其不適用於 HOG 這種非穩定高派息公司。

**綜合判斷：**

考慮到 DCF 的長期視角和 P/E 估值在行業中的合理性，並剔除 DDM 和 EV/EBITDA 在 HOG 案例中的局限性，我認為 **HOG 的合理股價範圍大致落在 $28 - $35 之間**。

*   **當前股價 ($32.00) 位於這個區間內。** 這可能表明市場對 HOG 的定價相對合理，平衡了其傳統業務的挑戰和轉型努力的潛力。
*   HOG 的挑戰在於其核心市場的增長停滯和客戶老化。公司正在積極應對，例如推出 LiveWire 電動摩托車、探索新的細分市場 (如冒險旅行摩托車 Pan America)，並專注於高利潤的重型摩托車業務。
*   投資者應密切關注其營收增長、EBITDA 利潤率的穩定性，以及債務管理和資本配置策略。LiveWire 的表現以及 HOG 在電動化轉型中的作用，也將是未來股價的重要影響因素。

**與同產業可比公司估值比較：**

HOG 目前的 P/E (約 6.7x) 顯著低於可比公司平均 (約 10.1x)，但 EV/EBITDA (約 9.4x) 卻高於可比公司平均 (約 7.3x)。這是一個關鍵的發現：
*   **低 P/E:** 市場對 HOG 未來盈利增長持謹慎態度，或認為其 LTM EPS 不具備可持續性。這也可能反映了其品牌和市場轉型所帶來的風險溢價。
*   **高 EV/EBITDA:** 主要由其相對較高的淨債務 (部分歸因於 HDFS 金融服務部門) 所驅動。這使得 HOG 在傳統的 EV/EBITDA 倍數下顯得估值更高，但這可能是由於其業務模式（同時經營工業和金融）的獨特性造成的。

綜合來看，HOG 面臨著較大的轉型壓力，其估值反映了這些挑戰和其獨特的業務結構。若公司能成功執行轉型戰略並提升盈利能力，其P/E倍數可能會有提升空間，從而推動股價上漲。反之，若轉型不力，則估值可能承壓。
