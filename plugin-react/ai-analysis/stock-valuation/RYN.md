## RYN (Rayonier Inc.) 股票估值分析

### 1. 公司概覽與最新動態

**Rayonier Inc. (RYN)** 是一家領先的國際房地產投資信託基金 (REIT)，主要業務是管理和開發木材地產。公司在美國東南部、太平洋西北部和紐西蘭擁有、管理或租賃約 270 萬英畝的木材地產。作為一家 REIT，RYN 必須將大部分應稅收入分配給股東，這使得股息成為其投資吸引力的重要組成部分。

    *   **住房市場影響:** 由於高利率環境，美國住房市場活動放緩，對木材需求和價格造成壓力。這是短期內影響 RYN 業績的主要因素。
    *   **永續發展與 ESG:** 作為木材地產公司，RYN 在碳固存和永續林業方面具有天然優勢，符合 ESG 投資趨勢，這可能帶來長期價值。
    *   **全球供應鏈:** 紐西蘭業務為公司提供了地理多元化和對亞洲市場的敞口，有助於緩衝美國市場的波動。
    *   **通膨與利率:** 高通膨和高利率增加了營運成本和借貸成本，對 REITs 的盈利能力構成挑戰。

### 3. 估值假設

由於 RYN 是一家 REIT，傳統的自由現金流 (FCF) 模型可能不完全適用，因為其折舊攤銷很高且資本支出模式獨特。對於 REITs，股息折現模型 (DDM) 和基於營運資金 (FFO/AFFO) 的估值方法通常更為合適。

**共同假設：**

*   **無風險利率 (Risk-Free Rate, Rf):** 採用美國 10 年期國債收益率，約 **4.5%**。
*   **股權風險溢價 (Equity Risk Premium, ERP):** 採用歷史平均值，約 **5.5%**。
*   **Beta (RYN):** 根據網路查詢，RYN 的 Beta 值約為 **0.90**。
*   **股權成本 (Cost of Equity, Ke) - CAPM 模型:**
    *   Ke = Rf + Beta * ERP = 4.5% + 0.90 * 5.5% = **9.45%**。
*   **債務成本 (Cost of Debt, Kd):** 假設稅前債務成本為 **5.5%** (基於當前利率環境和公司債務評級)。
*   **稅率 (Tax Rate, T):** 由於 RYN 是 REIT，其大部分收入無需繳納公司稅。但在計算 WACC 時，債務的稅盾效應仍需考慮。我們使用一個名義的邊際稅率 **21%** 來計算稅後債務成本，但需注意這對 REITs 的實際影響較小。
*   **債務權益比 (Debt/Equity):** 0.47 (來自提供數據)。
    *   權益佔總資本比重 (E/(E+D)) = 1 / (1 + D/E) = 1 / (1 + 0.47) = 0.68。
    *   債務佔總資本比重 (D/(E+D)) = 0.47 / (1 + 0.47) = 0.32。
*   **加權平均資本成本 (WACC):**
    *   WACC = (E/(E+D)) * Ke + (D/(E+D)) * Kd * (1-T)
    *   WACC = 0.68 * 9.45% + 0.32 * 5.5% * (1 - 0.21)
    *   WACC = 6.43% + 1.39% = **7.82%**。為簡化和保守起見，我們將使用 **8.0%** 作為 DCF 的折現率。

### 4. 估值模型

#### 4.1. 折現現金流模型 (Discounted Cash Flow, DCF) - 基於 AFFO

對於 REITs，我們通常使用調整後營運資金 (AFFO) 作為衡量可分配現金流的更好指標。

**DCF 假設：**

*   **基期 AFFO (AFFO0):** 2023 年 AFFO 為每股 $0.90。
*   **預測期:** 5 年。
*   **短期 AFFO 增長率 (g_short):** 考慮到 Q1 表現疲軟和分析師預期，我們假設未來 5 年 AFFO 增長率為 **4.0%**。
*   **長期 AFFO 增長率 (g_long):** 假設為 **2.5%** (略高於長期通膨和經濟增長)。
*   **折現率:** 採用 WACC **8.0%**。

**計算過程：**

1.  **預測期 AFFO (每股):**
    *   AFFO1 = $0.90 * (1 + 0.04) = $0.936
    *   AFFO2 = $0.936 * (1 + 0.04) = $0.973
    *   AFFO3 = $0.973 * (1 + 0.04) = $1.012
    *   AFFO4 = $1.012 * (1 + 0.04) = $1.052
    *   AFFO5 = $1.052 * (1 + 0.04) = $1.094

2.  **計算終端價值 (Terminal Value, TV):**
    *   AFFO6 = AFFO5 * (1 + g_long) = $1.094 * (1 + 0.025) = $1.121
    *   TV = AFFO6 / (WACC - g_long) = $1.121 / (0.080 - 0.025) = $1.121 / 0.055 = $20.38

3.  **折現所有現金流：**
    *   PV(AFFO1) = $0.936 / (1.08)^1 = $0.867
    *   PV(AFFO2) = $0.973 / (1.08)^2 = $0.834
    *   PV(AFFO3) = $1.012 / (1.08)^3 = $0.803
    *   PV(AFFO4) = $1.052 / (1.08)^4 = $0.773
    *   PV(AFFO5) = $1.094 / (1.08)^5 = $0.745
    *   PV(TV) = $20.38 / (1.08)^5 = $13.87

4.  **總估值 (每股):**
    *   總估值 = $0.867 + $0.834 + $0.803 + $0.773 + $0.745 + $13.87 = **$17.84**

**DCF 估值結果：**
*   **合理價位：$17.84**
*   **合理價位區間：$16.00 - $19.50** (考慮到增長率和折現率的敏感性)

#### 4.2. 股息折現模型 (Dividend Discount Model, DDM) - 多階段

RYN 作為 REIT，股息是其核心價值主張。我們採用多階段 DDM。

**DDM 假設：**

*   **當前年化股息 (D0):** $0.28/股 * 4 = $1.12/股。
*   **股權成本 (Ke):** 9.45%。
*   **短期股息增長率 (g_short):** 考慮到 2023 年 AFFO 為 $0.90，而年化股息為 $1.12，目前的派息率超過 100%，這在短期內是不可持續的。因此，我們假設短期內股息增長會非常緩慢，甚至持平，直到 AFFO 趕上。
    *   **情境一 (保守):** 前 2 年股息零增長 (0%)，之後長期增長 2.0%。
    *   **情境二 (溫和):** 前 3 年股息增長 2.0%，之後長期增長 2.5%。
    *   我們將採用情境二，因為市場可能預期 AFFO 會在未來幾年內增長以支持股息。
*   **長期股息增長率 (g_long):** 2.5%。

**計算過程 (情境二)：**

1.  **預測期股息：**
    *   D1 = $1.12 * (1 + 0.02) = $1.142
    *   D2 = $1.142 * (1 + 0.02) = $1.165
    *   D3 = $1.165 * (1 + 0.02) = $1.188

2.  **計算終端價值 (在第 3 年末):**
    *   D4 = D3 * (1 + g_long) = $1.188 * (1 + 0.025) = $1.218
    *   TV = D4 / (Ke - g_long) = $1.218 / (0.0945 - 0.025) = $1.218 / 0.0695 = $17.52

3.  **折現所有股息和終端價值：**
    *   PV(D1) = $1.142 / (1.0945)^1 = $1.043
    *   PV(D2) = $1.165 / (1.0945)^2 = $0.973
    *   PV(D3) = $1.188 / (1.0945)^3 = $0.909
    *   PV(TV) = $17.52 / (1.0945)^3 = $13.42

4.  **總估值 (每股):**
    *   總估值 = $1.043 + $0.973 + $0.909 + $13.42 = **$16.35**

**DDM 估值結果：**
*   **合理價位：$16.35**
*   **合理價位區間：$15.00 - $18.00** (考慮到股息增長率和股權成本的敏感性)

### 6. 主要風險

*   **木材市場波動性：** RYN 的收入高度依賴於木材價格和需求，受住房市場、建築活動和全球經濟狀況的影響。
*   **利率風險：** 作為 REIT，RYN 對利率變化敏感。高利率會增加借貸成本，並可能降低房地產資產的吸引力。
*   **股息可持續性：** 目前的股息派發率高於 AFFO，如果 AFFO 未能有效增長，公司可能面臨削減股息的壓力。
*   **自然災害：** 森林火災、病蟲害、風暴等自然災害可能對其木材地產造成損害。
*   **匯率風險：** 紐西蘭業務使其面臨匯率波動風險。
