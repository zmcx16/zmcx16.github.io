### **公司簡介與產業概況 (透過網路搜尋補充)**

**Civitas Resources, Inc. (CIVI)** 是一家獨立的石油和天然氣勘探與生產公司。該公司是科羅拉多州丹佛-朱爾斯堡 (DJ) 盆地最大的純油氣生產商，並透過近期一系列的策略性收購（例如 Tap Rock Resources 和 Vencer Energy），顯著擴大了其在二疊紀盆地（特拉華盆地和米德蘭盆地）的業務。

**產業趨勢與市場動態：**
*   **商品價格波動：** 石油和天然氣價格是影響 E&P 公司營收和獲利能力的最主要因素。地緣政治事件、OPEC+ 決策和全球經濟增長預期都會導致價格劇烈波動。
*   **併購整合：** 能源產業正經歷一波整合浪潮，公司透過收購來擴大規模、提高效率並鞏固市場地位。CIVI 的近期策略即是此趨勢的體現。
*   **ESG 壓力：** 儘管全球對化石燃料的需求短期內仍強勁，但 E&P 公司面臨來自環境、社會和治理 (ESG) 方面的壓力，需要投資於減排技術和可持續發展實踐。
*   **自由現金流與股東回報：** 許多 E&P 公司在經歷了高油價時期後，將重點放在生成強勁的自由現金流並透過股息和股票回購回報股東。CIVI 的高股息政策也反映了這一點。

**CIVI 近期表現與挑戰：**
*   **生產量增長：** 由於收購，CIVI 的生產量顯著增長。
*   **營收與獲利壓力：** 儘管生產量增加，但由於商品價格相較於前一時期有所下降，導致公司營收和每股盈餘 (EPS) 面臨壓力，這也解釋了其低估值倍數。
*   **股息政策：** 公司實施固定加浮動股息政策，高股息率對投資者有吸引力，但其可持續性與自由現金流生成能力緊密相關。
*   **債務與整合：** 收購帶來了債務增加和整合挑戰，管理層的重點之一是去槓桿化和實現協同效應。

### **估值假設**

在進行估值之前，我們需要設定一些關鍵假設。這些假設將基於提供的數據、網路搜尋結果以及對產業的理解。

**1. 折現率 (WACC - 加權平均資本成本)**
*   **無風險利率 (Rf)：** 採用美國 10 年期國債收益率，約 **4.5%**。
*   **市場風險溢價 (MRP)：** 歷史平均約 **5.5%**。
*   **Beta (β)：** 能源 E&P 公司對商品價格敏感，Beta 通常較高。根據網路搜尋，CIVI 的 Beta 約為 **1.6**。
*   **股權成本 (Ke) = Rf + β * MRP**
    *   Ke = 4.5% + 1.6 * 5.5% = 4.5% + 8.8% = **13.3%**
*   **債務成本 (Kd)：** 根據公司債務情況，假設平均利率為 **7.0%**。
*   **稅率 (t)：** 採用美國聯邦企業稅率 21%，加上州稅等，保守估計為 **25%**。
*   **債務權益比 (Debt/Eq)：** 0.77 (來自提供數據)
    *   權益權重 (We) = 1 / (1 + 0.77) = 0.565
    *   債務權重 (Wd) = 0.77 / (1 + 0.77) = 0.435
*   **WACC = (We * Ke) + (Wd * Kd * (1 - t))**
    *   WACC = (0.565 * 13.3%) + (0.435 * 7.0% * (1 - 0.25))
    *   WACC = 0.075145 + 0.0228375 = 0.0979825 ≈ **9.8%**

**2. 預測期間與增長率**
*   **明確預測期：** 5 年。
*   **營收增長率：** 考慮到近期營收下降和收購帶來的生產量增長，以及商品價格的不確定性，我們採取以下假設：
    *   第 1 年：-2% (反映近期商品價格壓力)
    *   第 2 年：+1% (趨於穩定，收購效益開始顯現)
    *   第 3-5 年：+3% (溫和增長，來自效率提升和潛在的商品價格復甦)
*   **長期增長率 (g)：** 終端價值增長率，假設為長期通脹率或略高，約 **2.0%**。

**3. 其他假設**
*   **初始營收 (T0)：** 根據 Market Cap / P/S = $2.31B / 0.49 = $4.71B。
*   **營運利潤率：** 23.04% (來自提供數據，假設在預測期內保持穩定)。

*   每股價值 = 股權價值 / 流通股數
*   每股價值 = $6.374B / 85.87M = **$74.23**

**DCF 模型估值結果：**
*   **合理價位：$74.23**
*   **合理價位區間：$65.00 - $85.00** (考慮到 WACC 和長期增長率的敏感性)

#### **3. 股息折現模型 (Dividend Discount Model, DDM)**

**步驟 1：計算當前每股股息 (D0)**
*   D0 = 當前股價 * 股息收益率 = $26.90 * 0.0738 = **$1.986**

**步驟 2：設定股息增長率**
*   CIVI 採用固定加浮動股息政策，且 EPS 預計下降，這使得股息增長難以預測。
*   **明確預測期 (5 年)：** 假設股息在短期內受盈利壓力影響，然後溫和增長。
    *   第 1 年：-2% (反映 EPS 下降壓力)
    *   第 2 年：0% (穩定)
    *   第 3-5 年：+1% (溫和增長)
*   **長期增長率 (g)：** 假設為 **1.5%** (比 DCF 的長期增長率更保守，因為股息增長通常更穩定)。
*   **股權成本 (Ke)：** 13.3% (來自 WACC 計算)

**步驟 3：預測股息並計算終端股息價值**

*   D1 = $1.986 * (1 - 0.02) = $1.946
*   D2 = $1.946 * (1 + 0.00) = $1.946
*   D3 = $1.946 * (1 + 0.01) = $1.966
*   D4 = $1.966 * (1 + 0.01) = $1.986
*   D5 = $1.986 * (1 + 0.01) = $2.006
*   D6 = D5 * (1 + g) = $2.006 * (1 + 0.015) = $2.036
*   **終端股息價值 (TV_Div) =** D6 / (Ke - g) = $2.036 / (0.133 - 0.015) = $2.036 / 0.118 = **$17.25**

**步驟 4：折現股息和終端股息價值至現值**

| 年度 | 股息 ($) | 折現因子 (1/(1+Ke)^n) | 股息現值 ($) |
| :--- | :------- | :-------------------- | :----------- |
| 1    | 1.946    | 0.8826                | 1.718        |
| 2    | 1.946    | 0.7790                | 1.516        |
| 3    | 1.966    | 0.6875                | 1.352        |
| 4    | 1.986    | 0.6068                | 1.206        |
| 5    | 2.006    | 0.5356                | 1.075        |
| TV_Div | 17.25    | 0.5356                | 9.234        |

*   **DDM 估值 =** 1.718 + 1.516 + 1.352 + 1.206 + 1.075 + 9.234 = **$16.10**

**DDM 模型估值結果：**
*   **合理價位：$16.10**
*   **合理價位區間：$14.00 - $18.00** (高度敏感於股息增長率和股權成本)

**DDM 模型的局限性：**
*   CIVI 的股息政策是變動的，且當前 EPS 預計下降，使得股息增長率的假設非常困難且不確定。
*   DDM 估值遠低於當前股價和分析師目標價，這可能表明市場對其未來股息增長有更高的預期，或者 DDM 並非最適合評估這類公司（其價值更多體現在自由現金流生成和資產價值上，而非單純的股息派發）。

2.  **DDM 模型**的估值顯著低於當前股價，這可能是因為其股息政策的變動性以及當前盈利下降的預期，使得保守的股息增長假設導致估值偏低。對於像 CIVI 這樣具有高資本支出、透過併購成長且股息具有變動性的 E&P 公司，DDM 可能不是最合適的估值方法。
3.  **市場對 CIVI 的低估**可能源於多重因素：商品價格波動、盈利和營收的預期下降、近期併購帶來的整合風險以及對化石燃料產業的整體負面情緒。然而，如果公司能夠有效整合新資產、管理債務並在未來穩定或改善盈利，其估值有巨大的上漲空間。
4.  **分析師目標價 $37.42** 介於當前股價和 DCF/相對估值之間，可能反映了對公司潛力的認可，但同時也考慮了短期內的風險和不確定性。

**綜合合理價位區間：**

考慮到 DCF 和相對估值模型的一致性，以及 DDM 在此案例中的局限性，我們將主要依賴 DCF 和相對估值。同時，也應考慮分析師的共識。

因此，綜合來看，CIVI 的合理價位區間應在 **$35.00 - $75.00** 之間。
