# Civitas Resources (CIVI) 深度估值分析報告

Civitas Resources (CIVI) 是美國科羅拉多州丹佛盆地（DJ Basin）最大的獨立油氣勘探與生產（E&P）公司，並於 2023 年透過一系列重大收購成功進軍二疊紀盆地（Permian Basin）。目前該公司已轉型為跨盆地的能源生產商，以高股息與強勁的自由現金流（FCF）回饋股東為核心策略。

---

#### 估值地位分析
- **折價狀態**：CIVI 目前相對於同業處於明顯的**折價 (Discount)** 狀態。其 EV/EBITDA 僅 2.8x，遠低於同業平均的 3.5x - 4.5x。
- **原因分析**：
    1. **監管風險**：市場對科羅拉多州（DJ Basin）的法規環境仍存有疑慮，儘管 CIVI 已透過 Permian 收購分散風險。
    2. **槓桿壓力**：為收購 Permian 資產，CIVI 增加了債務負擔，淨負債/EBITDA 約為 1.1x，略高於部分保守同業。
    3. **整合期**：市場正在觀察其對新收購資產的營運效率提升（Synergies）是否能如期兌現。

### 2. 折現現金流模型 (DCF)

#### 預測假設
- **預測期間**：5 年 (2025-2029)。
- **營收增長率**：預計 2025 年隨產量穩定增長 3-5%，隨後進入成熟期，假設長期油價維持在 WTI $70-$75 區間。
- **自由現金流 (FCF) 利潤率**：預計維持在 18%-22% 之間，受惠於 Permian 資產的高資本效率。
- **資本支出 (CapEx)**：假設每年約 $1.8B - $2.1B 以維持產量。

#### 折現率與終值
- **WACC 計算**：
    - 無風險利率 (10Y Treasury): 4.2%
    - Beta: 1.25 (能源股波動較高)
    - 股權風險溢價 (ERP): 5.5%
    - 稅後債務成本: ~5.5%
    - **WACC 估算: 10.5%**
- **終值增長率 (g)**：0% (考量能源轉型與資源枯竭風險，採保守假設)。

#### 估值結果
- **企業價值 (EV)**：約 $11.5B
- **減去淨負債**：約 $4.8B
- **股權價值**：約 $6.7B
- **內在價值 (Intrinsic Value)**：**$68.50 / 股**
- **合理價位區間**：**$62 - $75**

### 3. 多維度估值法

#### 相對估值法 (Relative Valuation)
參考同業平均 EV/EBITDA 3.8x。
- CIVI 2025 預期 EBITDA: ~$3.8B
- 隱含企業價值: $14.4B
- 扣除債務後股價: **$98.00** (此為消除折價後的理想目標價)。
- 考量 20% 的「科羅拉多風險折價」，目標價定為 **$78.00**。

#### 股息折現模型 (DDM)
CIVI 承諾將 100% 的自由現金流（扣除基本股息後）回饋股東。
- 預期年度總股息 (基本+變動): ~$5.50 - $6.50。
- 假設折現率 12%，增長率 0%。
- DDM 估值: $5.50 / 0.12 = **$45.83** (此為極度保守的下行支撐價)。

**最終結論：**

2. **目前股價位階**：**低估 (Undervalued)** (當前股價約 $50-$53)
3. **投資核心論點**：CIVI 目前被市場過度低估，主要源於對 DJ Basin 監管的過度擔憂。隨著 Permian 資產整合產生的協同效應顯現，以及債務透過強勁 FCF 快速去槓桿，估值倍數有望修復。其超過 10% 的預期總收益率（股息+回購）提供了極高的安全邊際，是價值投資者的優質標的。
