### III. 估值模型

#### A. 折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型旨在估計公司未來自由現金流的現值，以得出公司的內在價值。

**1. 估值假設：**

*   **預測期間：** 5年明確預測期，之後進入永續增長階段。
*   **折現率 (WACC) 計算：**
    *   **股權成本 (Cost of Equity, Ke)：** 使用資本資產定價模型 (CAPM)
        *   無風險利率 (Rf): 4.5% (美國10年期國債殖利率)
        *   Beta: 1.25
        *   市場風險溢酬 (MRP): 5.5%
        *   Ke = Rf + Beta * MRP = 4.5% + 1.25 * 5.5% = 4.5% + 6.875% = **11.375%**
    *   **債務成本 (Cost of Debt, Kd)：**
        *   總債務 (D): $4.9 Billion (來自網路查詢的最新財報數據)
        *   利息費用 (Interest Expense): 假設2023年約 $200 Million (根據財報估算)
        *   稅前債務成本 = $200M / $4.9B = 4.08% (此為歷史數據，考慮當前利率環境，假設公司新發債務或再融資成本更高，取一個更貼近市場的稅前債務成本)
        *   **調整後的稅前債務成本 (Pre-tax Kd):** 假設為 5.5% (考慮公司槓桿率和當前利率環境)
        *   有效稅率 (Tax Rate): 23%
        *   稅後債務成本 (After-tax Kd) = 5.5% * (1 - 0.23) = **4.235%**
    *   **WACC 計算：**
        *   股權市值 (E): $6.09 Billion (Market Cap)
        *   債務市值 (D): $4.9 Billion (假設帳面價值接近市場價值)
        *   總資本 (V): E + D = $6.09B + $4.9B = $10.99 Billion
        *   WACC = (E/V) * Ke + (D/V) * After-tax Kd
        *   WACC = ($6.09B / $10.99B) * 11.375% + ($4.9B / $10.99B) * 4.235%
        *   WACC = 0.554 * 11.375% + 0.446 * 4.235% = 6.30% + 1.89% = **8.19%** (取 8.2%)
*   **自由現金流 (FCF) 增長率：**
    *   基期 FCF (2023年): $520 Million (根據網路查詢的最新財報數據，營業現金流減去資本支出)
    *   預測期增長率：
        *   第1-2年 (2024-2025): 2.0% (考慮宏觀經濟逆風和公司溫和的營收增長預期)
        *   第3-4年 (2026-2027): 3.0% (預期宏觀經濟改善，公司營運效率提升)
        *   第5年 (2028): 2.5% (逐漸趨向永續增長率)
*   **永續增長率 (Terminal Growth Rate, g):** 1.5% (保守估計，低於長期GDP增長和通脹率)

**2. 計算過程：**

*   **預測期自由現金流 (FCF)：**
    *   2023 (基期): $520.0M
    *   2024: $520.0M * (1 + 0.020) = $530.4M
    *   2025: $530.4M * (1 + 0.020) = $541.0M
    *   2026: $541.0M * (1 + 0.030) = $557.2M
    *   2027: $557.2M * (1 + 0.030) = $574.0M
    *   2028: $574.0M * (1 + 0.025) = $588.4M

*   **永續價值 (Terminal Value, TV)：**
    *   FCF_2029 = $588.4M * (1 + 0.015) = $597.2M
    *   TV = FCF_2029 / (WACC - g) = $597.2M / (0.082 - 0.015) = $597.2M / 0.067 = $8,913.4M

*   **折現現金流現值：**
    *   PV(FCF 2024) = $530.4M / (1.082)^1 = $490.2M
    *   PV(FCF 2025) = $541.0M / (1.082)^2 = $460.0M
    *   PV(FCF 2026) = $557.2M / (1.082)^3 = $440.0M
    *   PV(FCF 2027) = $574.0M / (1.082)^4 = $421.0M
    *   PV(FCF 2028) = $588.4M / (1.082)^5 = $400.0M
    *   PV(TV) = $8,913.4M / (1.082)^5 = $6,050.0M

*   **企業價值 (Enterprise Value, EV)：**
    *   EV = 預測期FCF現值總和 + 永續價值現值
    *   EV = ($490.2 + $460.0 + $440.0 + $421.0 + $400.0)M + $6,050.0M = $2,211.2M + $6,050.0M = $8,261.2M

*   **股權價值 (Equity Value)：**
    *   Equity Value = EV + 現金 - 總債務
    *   Equity Value = $8,261.2M + $350M - $4,900M = $3,711.2M

*   **每股價值 (Share Price)：**
    *   每股價值 = 股權價值 / 流通股數
    *   每股價值 = $3,711.2M / 147.1M = **$25.23**
