#### A. 加權平均資本成本 (WACC) 計算

WACC 是 DCF 模型中的折現率，代表公司所有資本來源的平均成本。

*   **Beta值 (Beta):** 1.15 (來自網路搜尋)

    Ke = Rf + Beta * MRP

    Ke = 4.2% + 6.325%
    **Ke = 10.525%**

**2. 債務成本 (Cost of Debt, Kd):**
*   **利息支出 (Interest Expense):** $22 百萬美元
*   **總債務 (Total Debt):** $420 百萬美元

    Kd = 利息支出 / 總債務

    **Kd = 5.24%**

**3. 稅率 (Tax Rate):**
*   **有效稅率:** 27.5% (來自網路搜尋的財報數據)

**4. 資本結構:**

*   **債務市值 (Market Value of Debt, D):** $420 百萬美元 (假設債務帳面價值接近市場價值)
*   **總資本 (V):** E + D = $824.31M + $420M = $1244.31 百萬美元

**5. WACC 計算:**
    WACC = (E/V) * Ke + (D/V) * Kd * (1 - Tax Rate)
