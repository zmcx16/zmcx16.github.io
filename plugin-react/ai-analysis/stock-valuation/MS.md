### 2. 估值方法應用

#### 2.1. 折現現金流模型 (Discounted Cash Flow, DCF)

對於金融機構，自由現金流 (FCF) 的計算可能與一般產業公司有所不同。通常會使用調整後的淨利潤或股東自由現金流。由於 Morgan Stanley 的主要業務是金融服務，其現金流更多地與資產負債表管理和監管資本要求相關。在此，我將嘗試基於其盈利能力和再投資需求來估計 FCF。

**估值假設:**

*   **預測期間**: 5 年 (2026-2030)
*   **長期增長率 (Terminal Growth Rate)**: 2.5% (考慮到金融業的成熟性和宏觀經濟增長，略高於長期通膨率)
*   **折現率 (WACC)**:
    *   **無風險利率 (Risk-Free Rate)**: 採用美國 10 年期公債殖利率，約 4.27%。
    *   **市場風險溢酬 (Market Risk Premium, ERP)**: 歷史上通常在 4.5% - 6% 之間。我將採用 5.5%。
    *   **Beta 值**: 1.2。
    *   **股權成本 (Cost of Equity, Ke)** (使用 CAPM 模型):
        Ke = 無風險利率 + Beta * 市場風險溢酬
        Ke = 4.27% + 1.2 * 5.5% = 4.27% + 6.6% = 10.87%
    *   **債務成本 (Cost of Debt, Kd)**: 考慮到 Morgan Stanley 的信用評級，我將假設其稅前債務成本為 5.0%。
    *   **稅率 (Tax Rate)**: 採用 22.5% (2026 年預計稅率範圍的中間值)。
    *   **稅後債務成本 (After-tax Cost of Debt)**: Kd * (1 - Tax Rate) = 5.0% * (1 - 0.225) = 5.0% * 0.775 = 3.875%
    *   **股權市值 (E)**: 2862 億美元 (Market Cap)
    *   **債務市值 (D)**: 根據 Debt/Eq = 3.61，D = 3.61 * E = 3.61 * 2862 億美元 = 10331.82 億美元。
    *   **WACC (加權平均資本成本)**:
        WACC = (E / (E + D)) * Ke + (D / (E + D)) * Kd * (1 - Tax Rate)
        E / (E + D) = 2862 / (2862 + 10331.82) = 2862 / 13193.82 = 0.2169
        D / (E + D) = 10331.82 / 13193.82 = 0.7831
        WACC = 0.2169 * 10.87% + 0.7831 * 3.875%
        WACC = 2.358% + 3.035% = 5.393% ≈ 5.4%

**自由現金流 (FCF) 預測:**

對於金融機構，FCF 的預測通常更側重於調整後的淨利潤和資本配置。我將使用簡化的方法，基於淨利潤和再投資率來估計 FCF。

*   **基期淨利潤 (2025 年)**: EPS (TTM) * 總股數。總股數 = Market Cap / Close = 2862 億美元 / 180.08 美元 ≈ 15.89 億股。
    2025 年淨利潤 = 10.21 美元/股 * 15.89 億股 = 162.24 億美元。
*   **營收增長率**: 預計 2026 年營收增長 8.08%。 之後假設逐步放緩。
    *   2026: 8.0%
    *   2027: 7.0%
    *   2028: 6.0%
    *   2029: 5.0%
    *   2030: 4.0%
*   **淨利潤率**: 採用 2025 年的淨利潤率 = 162.24 億美元 / 706 億美元 ≈ 22.98%。假設未來保持穩定。
*   **再投資率 (Reinvestment Rate)**: 由於金融機構的再投資主要體現在資本配置和資產增長上，我將使用 (1 - 股息支付率) 作為再投資的近似。
    *   股息支付率 = 股息 / EPS = (180.08 * 0.0218) / 10.21 = 3.9257 / 10.21 ≈ 0.3845 (38.45%)
    *   再投資率 = 1 - 0.3845 = 0.6155 (61.55%)
    *   或者，考慮到 ROE = 16.34%，可持續增長率 = ROE * (1 - 股息支付率) = 0.1634 * (1 - 0.3845) = 0.1634 * 0.6155 = 0.1006 (10.06%)。這與預計的 EPS 增長率 10.16% 相符。
    *   因此，我將使用再投資率約 60% 作為估計。
*   **FCF = 淨利潤 * (1 - 再投資率)** (簡化處理，對於金融機構，這可以看作是可分配給股東的現金流)

*   **終端價值 (Terminal Value, TV)**:
    TV = FCF (2030) * (1 + 長期增長率) / (WACC - 長期增長率)
    TV = 86.78 億美元 * (1 + 0.025) / (0.054 - 0.025)
    TV = 86.78 億美元 * 1.025 / 0.029
    TV = 89.0045 億美元 / 0.029 = 3069.12 億美元

*   **終端價值現值 (PV of TV)**:
    PV of TV = TV / (1 + WACC)^5
    PV of TV = 3069.12 億美元 / (1.054)^5
    PV of TV = 3069.12 億美元 / 1.2959 = 2368.32 億美元

*   **公司總價值 (Enterprise Value)**:
    公司總價值 = 預測期 FCF 現值總和 + 終端價值現值
    公司總價值 = 336.22 億美元 + 2368.32 億美元 = 2704.54 億美元

*   **股權價值 (Equity Value)**: 對於金融機構，通常直接將公司總價值視為股權價值，因為其資本結構和現金流性質與非金融公司不同。
    股權價值 = 2704.54 億美元

*   **每股估值 (Intrinsic Value per Share)**:
    每股估值 = 股權價值 / 總股數
    每股估值 = 2704.54 億美元 / 15.89 億股 = 170.20 美元/股

**DCF 模型估值結果**: 170.20 美元/股
**合理價位區間**: 160 - 180 美元 (考慮到假設的敏感性)
