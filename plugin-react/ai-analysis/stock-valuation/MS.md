### 2. 估值模型計算

#### A. 股息折現模型 (Dividend Discount Model, DDM)
由於 MS 是一家成熟的金融機構，擁有穩定的派息記錄，DDM 是非常合適的工具。

*   **假設：**
    *   **當前年度股息 (D0)：** $3.70 (每季 $0.925)。
    *   **股息增長率 (g)：** 考慮到過去 5 年複合增長率較高，但長期需回歸常態，設定為 **6%** (略高於 GDP 增長，反映其財富管理市佔擴張)。
    *   **股權成本 (Ke)：** 使用 CAPM 模型。
        *   無風險利率 (Rf) ≈ 4.2% (10年美債)
        *   Beta (β) ≈ 1.2 (金融股波動略高於大盤)
        *   市場風險溢價 (ERP) ≈ 5.0%
        *   $Ke = 4.2\% + 1.2 \times 5.0\% = 10.2\%$
*   **計算：**
    *   $P = D0 \times (1 + g) / (Ke - g)$
    *   $P = 3.70 \times 1.06 / (0.102 - 0.06) = 3.922 / 0.042 = \mathbf{\$93.38}$
*   **註解：** DDM 估值通常較為保守，且未考慮股票回購帶來的價值提升。

#### B. 相對估值法 (Relative Valuation)
這是目前市場定價 MS 的主流方式。

*   **P/E 倍數法：**
    *   **預期 EPS (Forward EPS)：** 根據數據，Forward P/E 為 17.17，推算 Forward EPS 約為 $10.59。
    *   **合理 P/E 區間：** 歷史平均約 12-15x，但因業務轉型，市場目前給予 17-18x。
    *   **估值：** $10.59 \times 17.5 = \mathbf{\$185.33}$
*   **P/B 倍數法：**
    *   **當前 P/B：** 2.89 (顯著高於同行，如 GS 的 1.2-1.4x)。
    *   **理由：** MS 的 ROE (15.15%) 穩定且來自低資本消耗業務。
    *   **估值：** 若維持現有溢價，目標價約在 **$180 - $190** 區間。

#### C. 折現現金流模型 (DCF - 剩餘收益模型 Excess Return Model)
對於銀行業，傳統 FCF 較難計算，通常使用「剩餘收益模型」。

*   **假設：**
    *   **淨資產 (BVPS)：** 約 $62.93 (181.87 / 2.89)。
    *   **ROE：** 15.15%。
    *   **折現率 (Ke)：** 10.2%。
    *   **長期增長率 (g)：** 4%。
*   **計算：**
    *   $Value = BVPS + [BVPS \times (ROE - Ke) / (Ke - g)]$
    *   $Value = 62.93 + [62.93 \times (0.1515 - 0.102) / (0.102 - 0.04)]$
    *   $Value = 62.93 + [3.115 / 0.062] = 62.93 + 50.24 = \mathbf{\$113.17}$
*   **註解：** 此模型顯示，若僅看帳面價值與超額回報，目前的股價包含了極高的「品牌與特許經營權溢價」。
