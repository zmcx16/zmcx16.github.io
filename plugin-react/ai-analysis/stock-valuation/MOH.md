### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF模型需要預測未來幾年的自由現金流 (FCF)，並將其折現回現值。

**估值假設：**

*   **預測期間：** 5年 (2026-2030年)。
*   **折現率 (WACC)：** 採用GuruFocus提供的最新WACC 5.87%。
    *   **敏感性分析：** 將使用5.5%至6.5%的WACC範圍。
*   **長期增長率 (Terminal Growth Rate)：** 考慮到醫療保健行業的穩定性以及MOH的規模，我們假設長期增長率為2.5%。
*   **自由現金流 (FCF) 預測：**
    *   由於MOH的FCF波動較大，且2025年為負值，直接預測FCF增長較為困難。我們將基於營收預期和利潤率來推導FCF。

            *   2028年：3%

            *   2030年：2.5%
    *   **營業利潤率 (Operating Margin)：** 提供的數據為0.0172 (1.72%)。 2025年營業利潤率為2.97%。 考慮到2026年EPS指引大幅下降，我們將在預測期內保守估計營業利潤率，並假設其逐步改善。
        *   22026年：1.5% (考慮到2026年EPS指引的壓力)
        *   2027年：2.0%

    *   **非現金費用 (折舊與攤銷, D&A)：** 假設D&A佔營收的1.5%。

**DCF 計算步驟：**

1.  **計算預測期內的營業利潤 (EBIT)。**
2.  **計算稅後營業利潤 (NOPAT) = EBIT * (1 - 稅率)。**
3.  **計算自由現金流 (FCF) = NOPAT + D&A - CapEx - 營運資本變動。**
4.  **計算終值 (Terminal Value, TV)。**
5.  **將FCF和TV折現回現值。**

*   流通股數 (Shares Outstanding) = 市值 / 股價 = 7.87B / 151.0 = 52.12 百萬股 (近似值)

### 3. 綜合結論

我們使用了兩種主要的估值方法：折現現金流模型 (DCF) 和相對估值法。

*   **DCF模型估值：** 約 **65.33 美元/股** (範圍：55 - 75 美元)。
*   **相對估值 (P/E)：**
    *   基於TTM P/E平均值：**139.19 美元/股**
    *   基於Forward P/E平均值：**77.70 美元/股** (範圍：75 - 140 美元)

當前MOH的股價約為151.0美元。

**估值結果差異分析：**

DCF模型給出的估值顯著低於當前股價和基於TTM P/E的相對估值。這可能歸因於以下幾個因素：

1.  **FCF的波動性：** MOH的自由現金流歷史數據波動較大，2025年甚至為負值，這使得DCF模型對未來FCF的假設非常敏感。我們在DCF中對營收增長和營業利潤率的假設相對保守，特別是考慮到2026年EPS指引的大幅下降。
2.  **EPS的波動性：** MOH的EPS在2025年和2026年預期有顯著下降，這直接影響了P/E倍數的計算和Forward P/E的應用。
3.  **市場預期與管理層指引：** 2026年調整後EPS指引僅為5.00美元，遠低於2024年的23.50美元和2025年的11.03美元。這表明公司在短期內面臨較大的盈利壓力，DCF模型可能更好地捕捉了這種短期挑戰。而TTM P/E是基於過去12個月的盈利，可能未能完全反映最新的負面預期。
4.  **WACC的影響：** 5.87%的WACC相對較低，但由於FCF預測較低，仍導致較低的估值。

**綜合判斷：**

考慮到MOH管理層對2026年EPS的保守指引 (至少5.00美元)，以及自由現金流的波動和2025年的負值，DCF模型可能更為悲觀地反映了公司短期內的盈利挑戰。相對估值中，基於Forward P/E的77.70美元/股與DCF的結果更為接近，這表明市場對MOH未來盈利能力的擔憂。
