### 2. 估值方法應用

#### 2.1 折現現金流模型 (Discounted Cash Flow, DCF)

DCF 模型需要預測自由現金流 (Free Cash Flow, FCF) 並以加權平均資本成本 (WACC) 折現。由於缺乏詳細的歷史 FCF 數據和未來資本支出、營運資本變化的預測，我將基於營收和利潤率預期來估算 FCF。

**估值假設:**

*   **預測期間:** 5 年 (2026-2030)
*   **長期增長率 (Terminal Growth Rate):** 2.5% (略高於長期通脹率，反映醫療保健行業的穩定增長)
*   **加權平均資本成本 (WACC):**
    *   **股權成本 (Cost of Equity):** 使用資本資產定價模型 (CAPM)
        *   無風險利率 (Risk-Free Rate): 4.5% (參考美國 10 年期國債收益率的長期平均水平)
        *   市場風險溢價 (Market Risk Premium): 5.5% (行業標準)
        *   Beta: 0.55 (來自搜尋結果，Molina Healthcare 的 Beta 值為 0.55，低於 S&P 500 的波動性)
        *   股權成本 = 無風險利率 + Beta * 市場風險溢價 = 4.5% + 0.55 * 5.5% = 4.5% + 3.025% = 7.525%
    *   **債務成本 (Cost of Debt):**
        *   MOH 的長期債務/股權比率為 0.98。
        *   假設稅前債務成本為 6.0% (考慮當前利率環境和公司信用風險)。
        *   公司稅率 (Tax Rate): 30% (參考 2026 年預計有效稅率)
        *   稅後債務成本 = 6.0% * (1 - 30%) = 4.2%
    *   **WACC 計算:**
        *   股權權重 = 1 / (1 + 債務/股權比率) = 1 / (1 + 0.98) = 1 / 1.98 ≈ 0.505
        *   債務權重 = 0.98 / (1 + 0.98) = 0.98 / 1.98 ≈ 0.495
        *   WACC = (股權權重 * 股權成本) + (債務權重 * 稅後債務成本)
        *   WACC = (0.505 * 7.525%) + (0.495 * 4.2%) = 3.799% + 2.079% = 5.878% ≈ 5.9%

**自由現金流 (FCF) 預測 (簡化):**

由於缺乏詳細的財務報表數據，我們將使用簡化的 FCF 估算方法：FCF = 淨利潤 + 折舊攤銷 - 資本支出 - 營運資本變動。由於折舊攤銷和營運資本變動數據不易獲取，我們將基於營收和調整後淨利潤率來估算。

*   **2025 年數據:**
    *   總營收: 454.26 億美元
    *   調整後淨利潤: 5.84 億美元 (基於 11.03 美元 EPS 和 5300 萬股流通股，7.6B / 149.98 = 50.67M，取 53M 股)
    *   調整後淨利潤率: 5.84 億 / 454.26 億 = 1.28%

*   **營收增長率預測:**
    *   2026 年: -2% (公司指引)
    *   2027-2030 年: 假設每年 3% 的溫和增長 (考慮行業趨勢和公司策略調整)

*   **調整後淨利潤率預測:**
    *   2026 年: 0.6% (基於 2026 年調整後 EPS 至少 5.00 美元，總營收 420 億美元，5.00 * 53M / 42B = 0.63%)
    *   2027 年: 1.0% (預計 Medicaid 利潤率觸底後回升，MAPD 產品退出後影響減小)
    *   2028-2030 年: 1.3% (逐步恢復到 2025 年水平)

*   **資本支出 (CapEx):** 假設每年營收的 0.1% (參考 Q4 2025 數據，購買物業、廠房和設備為 100 萬美元，相對營收很小)
*   **營運資本變動:** 假設為零，簡化處理。

**終值 (Terminal Value) 計算:**

*   終值 FCF (2031 年) = 2030 年 FCF * (1 + 長期增長率) = 6.46 億 * (1 + 0.025) = 6.62 億美元
*   終值 = 終值 FCF / (WACC - 長期增長率) = 6.62 億 / (0.059 - 0.025) = 6.62 億 / 0.034 = 194.71 億美元
*   折現終值 = 終值 / (1 + WACC)^5 = 194.71 億 / (1 + 0.059)^5 = 194.71 億 / 1.333 = 146.07 億美元

**公司總價值 (Enterprise Value):**

*   折現 FCF 總和 = 2.61 + 4.05 + 5.15 + 5.00 + 4.86 = 21.67 億美元
*   公司總價值 = 折現 FCF 總和 + 折現終值 = 21.67 億 + 146.07 億 = 167.74 億美元

**股權價值 (Equity Value):**

*   股權價值 = 公司總價值 + 現金及約當現金 - 總債務
*   現金及約當現金 (截至 2025 年底): 42 億美元
*   總債務 (截至 2025 年底): 38 億美元 (長期債務)
*   股權價值 = 167.74 億 + 42 億 - 38 億 = 171.74 億美元

**每股估值:**

*   流通股數: 假設 5300 萬股 (基於 7.6 億市值 / 149.98 股價)
*   每股估值 = 171.74 億 / 5300 萬 = 324.04 美元

**DCF 估值結果:**

*   **每股估值:** 324.04 美元
*   **合理價位區間:** 280 - 370 美元 (考慮到假設的敏感性，特別是增長率和利潤率的變化)
