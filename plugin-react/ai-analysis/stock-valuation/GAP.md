#### 3. 股息折現模型 (Dividend Discount Model, DDM)

**假設：**

**計算：**

*   `D1 = D0 * (1 + g) = $0.644 * (1 + 0.03) = $0.663`

**DDM 估值結果：** **$7.37**

**合理價位區間：** DDM 估值非常低，這表明 GAP 的股息增長率相對較低，或者其股權成本較高，使得股息對其整體價值的貢獻有限。對於成長型或轉型中的公司，DDM 往往會低估其價值，因為它沒有充分考慮到未來可能再投資帶來的資本增值。因此，這個模型可能不完全適用於評估 GAP 的全部潛力。
