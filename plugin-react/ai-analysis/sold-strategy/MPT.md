### 3. 多重情境賣出策略 (Scenario Planning)

作為中長線趨勢交易者，我們應根據市場的變化靈活調整策略。

*   **獲利了結情境 (Profit Taking)**
    如果目前已獲利，考慮以下分批停利點位或訊號：
    1.  **接近52週高點或分析師目標價：** 若股價再次上攻並接近或達到 $6.47 的52週高點，或觸及分析師平均目標價 $6.67 (或 $5.83)，可以考慮分批獲利了結。這些點位可能面臨較大的賣壓。
    2.  **跌破移動停利線：** 設定一條移動停利線，例如跌破20日移動平均線 ($5.37) 或10日移動平均線 (若有明確數據)，作為分批出場的訊號。這有助於鎖定利潤並避免回吐。
    3.  **技術指標背離：** 若股價創出新高，但 RSI 或 MACD 等動能指標未能同步創出新高，形成頂部背離，這可能是上漲動能衰竭的早期訊號，應考慮減倉。
    4.  **成交量異常放大且股價下跌：** 若在股價高位出現巨量長陰線，顯示有大量資金出逃，應果斷停利。

*   **停損情境 (Stop Loss)**
    如果股價走勢不如預期，以下是關鍵支撐跌破後的具體出場機制：
    1.  **跌破50日移動平均線：** 50日移動平均線 ($5.24) 是中短期趨勢的重要參考。若股價有效跌破此線，且未能迅速收復，應考慮停損出場，以避免損失擴大。
    2.  **跌破近期盤整區間下緣：** 根據 ChartMill 的數據，近期一個月的交易區間為 $5.04 至 $6.47。若股價跌破 $5.04 這一區間下緣，則可能預示著更深的回調，應執行停損。
    3.  **跌破關鍵心理關口：** 例如 $5.00 這一整數關口，若被有效跌破，可能引發市場恐慌性賣盤。
    4.  **基本面惡化：** 若公司發布重大利空消息，如主要租戶再次出現財務問題、大幅下調財測、或槓桿率進一步惡化，應不計成本立即停損。

*   **觀望/持有 (Hold)**
    在以下條件下，可以考慮繼續持有，不急於賣出：
    1.  **股價維持在主要移動平均線之上：** 若股價能持續站穩在20日和50日移動平均線之上，顯示上升趨勢仍在延續。
    2.  **基本面持續改善：** 公司能持續成功進行租戶重組、有效降低債務、並實現穩定的現金流增長。2025年第四季度業績超出預期，以及新的租賃協議和股息增加，都是積極的信號。
    3.  **高空頭持倉引發軋空：** 雖然高空頭持倉顯示負面情緒，但若有任何利好消息或股價開始上漲，可能會引發空頭回補，推動股價進一步上漲。
    4.  **估值吸引力：** MPT 的市淨率 (P/B) 為 0.71，市銷率 (P/S) 為 3.38，均低於醫療保健 REIT 行業平均水平，且 DCF 分析顯示其被低估約 27.0%。若市場開始重新評估其價值，股價仍有上漲空間。

### 4. 綜合結論 (Final Verdict)

綜合技術面、動能與情緒指標以及基本面數據，我認為 MPT 目前處於一個**觀望與謹慎持有**的階段，但需密切關注短期動能變化。

*   **操作建議：**
    *   **現在不建議立即全部賣出。** 儘管短期動能有所減弱，但中長期均線仍呈多頭排列，且基本面近期有所改善，加上較低的估值和高空頭持倉帶來的潛在軋空機會，使得其仍具備一定的上漲潛力。
    *   **設定移動停損：** 為了保護現有獲利或控制風險，建議設定一個嚴格的移動停損點。例如，可以將停損點設在 $5.20 (略低於50日移動平均線 $5.24) 或 $5.00 心理關口。一旦股價有效跌破此點位，應果斷出場。
    *   **部分獲利了結 (若已獲利豐厚)：** 如果您的持倉已獲利豐厚，可以考慮在股價反彈至 $6.00-$6.47 區間時，分批獲利了結部分倉位，以鎖定利潤並降低風險。

*   **總結最關鍵的觀察指標：**
    1.  **50日移動平均線 ($5.24)：** 這是判斷中短期趨勢的關鍵支撐，若能守穩，則趨勢仍偏多。
    2.  **成交量變化：** 觀察股價上漲時的成交量是否能持續放大，以及下跌時的成交量是否縮小。
    3.  **RSI 和 MACD 趨勢：** 密切關注這些動能指標是否能重新轉強，或出現進一步的賣出訊號。
    4.  **公司基本面新聞：** 繼續關注 MPT 在債務削減、租戶穩定性以及未來盈利能力方面的進展。
    5.  **空頭回補情況：** 高空頭持倉可能成為股價上漲的催化劑，但若股價持續疲軟，空頭壓力也可能加劇。

總之，MPT 是一隻基本面正在改善但仍面臨挑戰的股票，其技術走勢目前處於多空拉鋸的關鍵時刻。作為中長線趨勢交易者，應保持警惕，嚴格執行交易紀律，並根據上述關鍵指標的變化靈活調整策略。
