### 3. 多重情境賣出策略 (Scenario Planning)

作為中長線趨勢交易大師，我們應根據不同情境制定靈活的賣出策略。

*   **獲利了結情境 (Profit Taking):**
    *   **情境描述:** 如果您目前持有 ILPT 股票並已獲得可觀利潤。
    *   **建議策略:**
        1.  **分批停利點位:**
            *   **第一批 (25-30%):** 當股價首次觸及或略高於 52週高點 $8.19，或接近分析師目標價區間的下限 (例如 $9.40) 時，可以考慮賣出部分持股以鎖定利潤。
            *   **第二批 (25-30%):** 如果股價在突破 $8.19 後，出現動能減弱的訊號 (例如，RSI 出現頂背離、上漲成交量縮小、K線出現長上影線或吞噬形態)，或未能有效站穩 $9.40 以上，可以考慮賣出第二批。
        2.  **移動停利線:** 設定一個移動停利線，例如跌破 20日移動平均線 (SMA20) 或 50日移動平均線 (SMA50) 時，賣出剩餘持股。這能確保在趨勢反轉時保護大部分利潤。
        3.  **乖離過大訊號:** 如果股價在短時間內與其長期移動平均線 (如 SMA200) 產生過大的正乖離，且隨後出現明顯的下跌 K線形態，這可能是短期回調的訊號，可考慮部分獲利了結。

*   **停損情境 (Stop Loss):**
    *   **情境描述:** 如果股價走勢不如預期，開始下跌。
    *   **建議策略:**
        1.  **關鍵支撐跌破:**
            *   **短期停損:** 如果股價跌破 $7.40 的近期支撐位，且伴隨較大成交量，應考慮立即停損，以避免損失擴大。
            *   **中期停損:** 如果股價跌破 $7.32 的中期支撐位 或 50日移動平均線，這可能預示著中期趨勢的改變，應果斷出場。
        2.  **百分比停損:** 設定一個固定百分比的停損點，例如從您的買入價或最近的高點下跌 5-7%。一旦觸及，無論技術形態如何，都應執行停損。
        3.  **移動停損:** 對於已獲利的部位，將停損點設定在一個動態的水平，例如最近波段低點下方，或 20日移動平均線下方。當股價上漲時，停損點也隨之上移，以保護已實現的利潤。

*   **觀望/持有 (Hold):**
    *   **情境描述:** 如果您目前持有 ILPT 股票，但尚未達到獲利了結或停損的條件。
    *   **建議條件:**
        1.  **趨勢持續:** 股價持續維持在 20日和 50日移動平均線之上，且均線保持多頭排列。
        2.  **動能不減:** RSI 和 MACD 等動能指標保持健康，未出現明顯的背離或轉弱訊號。
        3.  **成交量配合:** 上漲時成交量放大，下跌時成交量縮小，顯示買盤力量仍佔優勢。
        4.  **基本面無惡化:** 公司沒有出現重大利空消息，工業地產行業前景依然樂觀。
        5.  **成功突破壓力:** 如果股價能夠帶量突破 52週高點 $8.19，並有效站穩，則可以繼續持有，期待更高的目標價。
