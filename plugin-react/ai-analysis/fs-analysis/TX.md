# Ternium S.A. (TX) 財務報告深度分析報告

作為資深美股分析師，針對 Ternium S.A. (TX) 的財務狀況，結合提供的數據與最新財報趨勢（含 2024 年前三季表現），分析如下：

### 1. 損益表分析 (Income Statement Analysis)

*   **營收成長動能**：
    TX 最新季度營收表現穩健，主要受惠於**墨西哥市場**的強勁需求以及對巴西 **Usiminas** 的併購整合效應。雖然全球鋼價波動，但 TX 的銷量（Shipments）在 2024 年維持高檔。營收成長並非單純依賴漲價，更多來自於**垂直整合與產能擴張**。然而，阿根廷宏觀經濟的不確定性對其南美業務構成了一定程度的匯兌與需求壓力。
*   **獲利能力與利潤率**：
    *   **毛利率 (Gross Margin)**：16.13%，處於行業中等水平。受原料成本（鐵礦砂、煤炭）波動影響，利潤率較前兩年高點有所回落。
    *   **營業利益率 (Oper. Margin)**：6.46%，反映出鋼鐵業的高固定成本特性。
    *   **純益率 (Profit Margin)**：3.65%，目前處於週期性低谷。
    *   **規模經濟**：隨著墨西哥 Pesquería 新廠產能逐步開出，單位加工成本有望進一步下降，但短期內受折舊費用增加影響，利潤率擴張不明顯。
*   **費用控制**：
    S&A（銷售與管理費用）佔比穩定。TX 作為成熟的鋼鐵製造商，研發費用佔比較低，主要資本投入集中於生產工藝的環保升級與產能擴建。

### 2. 資產負債表分析 (Balance Sheet Analysis)

*   **資產品質與結構**：
    *   **現金狀況**：P/C (市現率) 為 3.1，顯示公司持有充裕現金。
    *   **周轉情況**：存貨周轉與應收帳款回收尚屬正常，未見明顯的存貨呆滯風險。在鋼價波動期，TX 展現了優異的供應鏈管理能力。
*   **負債與償債能力**：
    *   **流動性**：**流動比率 2.63** 與 **速動比率 1.53** 極為強勁，遠高於工業製造業的安全邊際。
    *   **長期風險**：**Debt/Eq 僅 0.25**，LT Debt/Eq 0.19。這在資本密集型的鋼鐵行業中屬於「堡壘型」資產負債表。淨債務（Net Debt）極低，甚至在部分季度呈現淨現金狀態，財務風險極低。
*   **無形資產與商譽**：
    商譽主要來自併購 Usiminas，目前未見減損跡象，資產負債表品質紮實。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF)**：
    TX 的 OCF 長期保持正值且與淨利趨勢一致。2024 年儘管淨利受鋼價壓制，但營運現金流依然強勁，足以支撐日常營運。
*   **自由現金流 (FCF)**：
    目前 FCF 受到較大壓力，主因是公司正處於**大規模資本支出週期**。TX 正在墨西哥投資約 32 億美元建設新的板材廠（Slab mill），這導致 CapEx 顯著上升。
*   **資本配置 (Capital Allocation)**：
    *   **股息**：Dividend % 達 4.44%，在資本支出高峰期仍維持穩定派息，顯示對現金流的信心。
    *   **併購與擴張**：資金主要投向墨西哥產能擴張，旨在利用 USMCA（美墨加協定）的關稅優勢，這是一項具備長期戰略價值的資本配置。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

*   **回報率指標**：
    *   **ROE (4.64%)**：目前較低，拆解原因在於**純益率收縮**與**低財務槓桿**。TX 並未透過舉債來放大回報，這在升息環境下是安全的，但也限制了 ROE 的表現。
    *   **ROIC (3.88%)**：略低於 WACC，反映出目前處於鋼鐵週期的下行或築底階段，以及新投資項目尚未產生貢獻。
*   **效率指標**：
    P/S 0.63 與 P/B 0.8 顯示股價低於帳面價值，具有典型的價值股特徵。資產周轉率受限於龐大的固定資產基數，需等待新產能完全釋放。

| :--- | :--- | :--- |

**最終結論：**
1. **整體評級**：**優良**
2. **核心摘要**：TX 是一家財務極其穩健的鋼鐵龍頭，擁有「堡壘級」資產負債表。儘管目前受全球鋼鐵週期影響導致獲利能力處於低位，但其在墨西哥市場的戰略佈局與極低的負債率，使其具備極強的抗風險能力與未來成長潛力。目前股價低於帳面價值（P/B 0.8），具備高度安全邊際。
3. **關鍵監測指標**：
    *   **Steel Spread (鋼鐵價差)**：監控熱軋鋼捲價格與原料成本的利差變化。
    *   **Mexico Shipment Volume**：觀察墨西哥新廠進度及在該地區的市佔率變化。
    *   **Free Cash Flow Margin**：隨著 CapEx 高峰過去，觀察 FCF 是否能快速回升以支持更高額的回購或派息。
