*   **獲利能力與利潤率**：
    *   **毛利率**：GAAP 毛利率維持在 **75.7%** 的極高水準，非公認會計準則（Non-GAAP）訂閱毛利率更是高達 **85%** 以上，顯示其軟體服務具備極強的規模經濟。
    *   **營業利益率**：這是 WDAY 近期的最大亮點。GAAP 營業利益率轉正至 **5.1%**，而 Non-GAAP 營業利益率從去年同期的 23.6% 擴張至 **24.9%**。這證明公司已度過燒錢階段，開始展現強大的營運槓桿。
*   **費用控制**：
    *   研發費用（R&D）與行銷費用（S&A）佔比正逐年優化。公司目前專注於 AI 驅動的產品升級，雖然研發投入絕對值增加，但佔營收比重趨於穩定，顯示費用控制得當。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF) 與盈餘品質**：
    *   WDAY 的 OCF 表現遠優於淨利（Net Income）。Q2 FY25 單季 OCF 達 5.7 億美元。

*   **自由現金流 (FCF)**：
