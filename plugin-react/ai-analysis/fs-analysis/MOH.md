作為一位資深美股分析師，我將針對Molina Healthcare, Inc. (MOH) 的財務狀況進行全面拆解。MOH主要提供醫療保健管理服務，業務涵蓋Medicaid、Medicare計畫以及州保險市場。

---

### 1. 損益表分析 (Income Statement Analysis)

**營收成長動能**
MOH的營收在過去幾年呈現穩健增長。2025年年度營收為454.3億美元，較2024年增長11.75%。2024年營收為406.5億美元，較2023年增長19.31%。2023年營收為340.7億美元，較2022年增長6.56%。2022年營收為319.7億美元，較2021年增長15.13%。2021年營收為277.7億美元，較2020年大幅增長42.98%。
然而，最新季度（2026年第一季度）的營收表現有所放緩。2026年第一季度保費收入約為102億美元，總營收為108億美元，較去年同期持平，甚至較上一季度下降5.1%。這顯示近期營收成長動能有所減弱。公司管理層在2026年投資者日表示，2026年全年保費收入預計約為420億美元，這意味著下半年需要更強勁的表現才能達到目標。營收增長主要來自於其核心的Medicaid和Medicare計畫，以及州保險市場的業務擴張。

**獲利能力與利潤率**
MOH的獲利能力在最新季度面臨顯著壓力。2026年第一季度，公司GAAP每股收益為0.27美元，調整後每股收益為2.35美元。淨利潤僅為1,400萬美元，較2025年第一季度的2.98億美元大幅下降95%。這導致淨利率從2025年第一季度的2.8%驟降至2026年第一季度的0.1%。
營業利益率（Operating Margin）根據提供數據為0.0116（1.16%），而利潤率（Profit Margin）為0.0042（0.42%）。這些數據顯示MOH的整體獲利能力較低。毛利率（Gross Margin）對於醫療保健服務公司而言通常不適用，Financhill報告MOH過去27年的毛利率為0.00%。
利潤率的急劇下降主要歸因於一項9,300萬美元的減值費用，該費用與公司計劃於2027年退出Medicare Advantage-Part D產品線相關。這項策略性調整雖然影響短期獲利，但旨在優化未來的業務組合，專注於Medicare的雙重資格計畫（duals）並降低市場（Marketplace）的風險敞口。

**費用控制**
2026年第一季度，MOH的銷售、一般及行政費用（G&A ratio）為7.2%，調整後G&A ratio為6.9%。這反映了營運費用的時機性。相較於其低迷的淨利率，費用控制對於維持獲利至關重要。研發費用（R&D）在醫療保健服務行業中通常佔比不高，主要費用集中在醫療服務成本和行政管理上。公司管理層在投資者日表示，2026年醫療成本趨勢好於預期，並對全年5%的趨勢假設有信心。

### 2. 資產負債表分析 (Balance Sheet Analysis)

**資產品質與結構**
截至2026年3月31日，MOH的短期資產為133億美元，超過短期負債82億美元。這表明公司擁有充足的流動資產來應對短期義務。現金與短期投資方面，截至2026年3月31日，母公司現金及投資約為2.13億美元，略低於2025年12月31日的2.23億美元。應收帳款（AR）和存貨（Inventory）周轉情況方面，醫療保健服務公司通常存貨較少。2026年3月31日的應付索賠天數（Days in claims payable）為44天，低於正常水平，這歸因於季度末支付時間的影響。這可能暗示應付帳款支付速度加快，或索賠處理效率提高。

**負債與償債能力**
*   **流動性**：根據提供數據，MOH的流動比率（Current Ratio）為1.63。Simply Wall St的數據顯示，MOH的短期資產（133億美元）超過短期負債（82億美元），這與流動比率1.63相符，表明其流動性健康。速動比率（Quick Ratio）未直接提供，但考慮到醫療服務行業存貨佔比極低，速動比率應與流動比率接近。
*   **長期風險**：MOH的債務對權益比（Debt/Equity）在過去五年中從98.3%降至92.3%，顯示公司在降低槓桿方面有所努力。截至2026年3月30日，債務為37.67億美元，股權為40.80億美元，債務對權益比為92.3%。債務佔資本比約為48%。債務對EBITDA比率為6.1倍，這表明債務負擔相對較高，且經營現金流對債務的覆蓋率不高（9.5%）。利息支付對EBIT的覆蓋倍數為2.6倍，顯示利息覆蓋能力較弱。
*   **淨現金狀況**：MOH的現金餘額高於其總債務，這是一個積極信號。管理層預計年底母公司現金將超過6億美元，並保持高於300%的RBC（風險資本）目標。

**無形資產與商譽**
雖然具體的商譽（Goodwill）數值未在搜索結果中直接列出，但作為一家通過併購擴張的醫療保健公司，MOH的資產負債表上很可能存在商譽。2026年第一季度因退出Medicare Advantage-Part D產品線而產生9,300萬美元的減值費用，這表明公司會定期評估其資產價值，並在必要時進行減值。這項減值也可能部分涉及無形資產或商譽。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

**營業現金流 (OCF)**
MOH的營運現金流在最新季度表現強勁。2026年第一季度營運現金流為10.82億美元，遠高於2025年第一季度的1.90億美元。這與其同期淨利潤1,400萬美元形成鮮明對比。
*   **盈餘品質**：營運現金流遠高於淨利潤，表明MOH的盈餘品質良好。這通常意味著公司的淨利潤並非主要由非現金項目（如折舊攤銷）或應收帳款增加所驅動，而是有實際現金流入支持。2026年第一季度的淨利潤受到一次性減值費用的嚴重影響，這進一步凸顯了營運現金流作為衡量公司核心業務產生現金能力的重要性。

**自由現金流 (FCF)**
MOH的自由現金流（FCF）在近年來波動較大。2025年年度自由現金流為-6.36億美元，較2024年下降216.91%。2024年為5.44億美元，較2023年下降65.53%。2023年為15.78億美元，較2022年增長131.38%。最新的十二個月自由現金流收益率為-8.4%。負的自由現金流表明公司在扣除資本支出後未能產生足夠的現金，這可能限制其擴張、派息或回購的能力。然而，2026年第一季度強勁的營運現金流（10.82億美元）若能持續，將有助於改善自由現金流狀況。

**資本配置 (Capital Allocation)**
MOH的資本配置策略包括戰略性地調整業務組合。公司計劃於2027年退出Medicare Advantage-Part D產品線，並將重點放在Medicare的雙重資格計畫上，同時降低市場業務的風險敞口。這項策略調整導致了2026年第一季度的減值費用。
從現金流角度看，雖然最新的營運現金流強勁，但過去一年的負自由現金流可能限制了其在擴張、派息或回購方面的靈活性。公司管理層表示，將保持穩健的資本和現金狀況，預計年底母公司現金將超過6億美元。這表明公司在資本配置上可能更傾向於維持財務彈性，以應對行業變化和潛在的戰略投資。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

**回報率指標**
*   **ROE (股東權益報酬率)**：根據提供數據，MOH的ROE為0.0448（4.48%）。這是一個相對較低的股東權益報酬率，表明公司為股東創造利潤的效率有待提高。
    *   **杜邦分析拆解ROE**：
        *   **淨利潤率 (Net Profit Margin)**：根據提供數據為0.0042（0.42%），2026年第一季度更是低至0.1%。這是一個非常低的利潤率，是拖累ROE的主要因素。
        *   **資產周轉率 (Asset Turnover)**：營收/總資產。根據2025年營收454.3億美元和Simply Wall St提供的2025年總資產約156億美元，資產周轉率約為45.43 / 15.6 ≈ 2.91。這表明公司資產利用效率尚可。
        *   **權益乘數 (Equity Multiplier)**：總資產/股東權益。根據2026年3月30日債務對權益比92.3%，可以推算出權益乘數約為 (1 + 0.923) = 1.923。
        *   **ROE = 淨利潤率 x 資產周轉率 x 權益乘數** = 0.0042 x 2.91 x 1.923 ≈ 0.0235 (2.35%)。這個計算結果與提供的ROE 4.48%有差異，可能是由於數據來源的時間點不同或計算方式略有差異。但核心結論是，低淨利潤率是限制ROE的關鍵因素。
*   **ROIC (投入資本回報率)**：根據提供數據，MOH的ROI為0.0234（2.34%）。ROIC衡量公司利用其投入資本產生利潤的效率。2.34%的ROIC相對較低，可能未能顯著超過其加權平均資本成本（WACC），這意味著公司創造股東價值的能力可能有限。

**效率指標**
*   **資產周轉率**：如上所述，約為2.91，表明公司在利用資產產生營收方面表現尚可。
*   **存貨周轉天數 (DIO)**：由於MOH是醫療服務公司，存貨通常不顯著，因此DIO不是關鍵指標。
*   **應收帳款周轉天數 (DSO)**：未直接提供，但醫療保健行業的應收帳款管理（來自政府或保險公司）至關重要。2026年第一季度應付索賠天數為44天，這是一個需要持續監測的指標。
