作為一位資深美股分析師，我將對NKE（Nike, Inc.）的財務狀況進行全面拆解，並根據最新的財報數據進行分析。我將主要參考NKE截至2026年2月28日的2026財年第三季度報告（Q3 FY2026）以及截至2025年5月31日的2025財年年報（FY2025）等公開資訊。

---

### 1. 損益表分析 (Income Statement Analysis)

*   **營收成長動能**
    NKE在最近的2026財年第三季度（截至2026年2月28日）實現了113億美元的營收，與去年同期持平（reported basis），但按貨幣中性計算則下降了3%。這顯示公司在營收增長方面面臨挑戰。
    從過去幾年的趨勢來看：
    *   FY2023營收：約512億美元
    *   FY2024營收：約514億美元 (年增約1% on a currency-neutral basis)
    *   FY2025營收：463.09億美元 (年減9.8%)
    營收下降主要受到NIKE Direct業務下滑的影響，其中NIKE Brand Digital銷售額在Q3 FY2026下降了9%，NIKE自有門店銷售額下降了5%。批發業務在Q3 FY2026則增長了5%。區域表現不一，北美地區在Q3 FY2026實現了營收增長，但EMEA（歐洲、中東和非洲）和大中華區則出現下滑。Converse品牌營收在Q3 FY2026更是大幅下降35%。營收下滑主要歸因於市場去庫存、折扣增加以及部分地區（如大中華區）的策略性調整和市場疲軟。

*   **獲利能力與利潤率**
    NKE的獲利能力在近期承壓。
    *   **毛利率 (Gross Margin)**：
        *   Q3 FY2026: 40.2%
        *   Q3 FY2025: 41.5%
        *   FY2025: 42.7%
        *   FY2024: 44.7% (Q4 FY2024數據)
        毛利率呈現下降趨勢。Q3 FY2026的毛利率為40.2%，較去年同期下降130個基點，主要原因是北美地區關稅上漲、Converse品牌獲利能力下降，部分被戰略性定價和較低的海洋運費及物流成本所抵消。FY2025的毛利率為42.7%，較FY2024的44.7%（Q4 FY2024數據）下降，主要由於折扣增加、渠道組合變化以及存貨報廢準備金增加。這表明公司在清理庫存和應對市場競爭時，犧牲了部分利潤率。
    *   **營業利益率 (Operating Margin)**：
        *   Q3 FY2026: 5.6% (EBIT margin)
        *   FY2025: 7.99%
        *   FY2024: 約12.85% (FY2024 EBIT $6.6B / Revenue $51.362B)
        營業利益率也顯著下滑。Q3 FY2026的營業利益率為5.6%，較去年同期大幅下降，反映出營收持平但毛利率下降，同時銷售和管理費用（S&A）因重組費用而增加。FY2025的營業利益率為7.99%，遠低於FY2024的水平，顯示公司在營運效率和成本控制方面面臨挑戰。
    *   **純益率 (Net Margin)**：
        *   Q3 FY2026: 4.6% (Net Income $520M / Revenue $11.3B)
        *   Q3 FY2025: 7.1% (Net Income $800M / Revenue $11.3B)
        *   FY2025: 7.0% (Net Income $3,219M / Revenue $46,309M)
        *   FY2024: 11.1% (Net Income $5,700M / Revenue $51,362M)
        純益率同樣呈現下降趨勢。Q3 FY2026的淨利潤為5.2億美元，同比下降35%，稀釋後每股收益為0.35美元，下降35%。這主要是由於毛利率壓縮和有效稅率從去年同期的5.9%上升至20.0%。FY2025的淨利潤為32.19億美元，較FY2024的57億美元下降44%。利潤率的下降表明公司在規模經濟方面可能受到挑戰，或者其成本結構在營收下滑時未能有效調整。

*   **費用控制**
    *   **銷售和管理費用 (S&A)**：
        *   Q3 FY2026: 40億美元 (佔營收約35.4%)，同比增長2%。增長主要由於員工遣散費和不利的匯率變動，部分被其他行政成本降低所抵消。
        *   FY2025: 161億美元 (佔營收約34.8%)，同比下降3%。
    *   **研發費用 (R&D)**：NKE通常將研發費用包含在S&A中，或稱為「需求創造費用 (Demand Creation Expense)」。
        *   Q3 FY2026的需求創造費用為11億美元，與去年同期持平。
        *   FY2025的需求創造費用為47億美元，同比增長9%。
    儘管營收增長停滯甚至下降，S&A費用在Q3 FY2026仍有所增加，特別是員工遣散費，這表明公司正在進行重組以優化成本結構，但短期內對利潤產生負面影響。需求創造費用的持續投入，尤其是在品牌行銷和體育行銷方面，與公司「Win Now」的成長策略相符，旨在重新點燃品牌動能和產品創新。然而，在營收疲軟的背景下，費用控制的效率將是關鍵。

### 2. 資產負債表分析 (Balance Sheet Analysis)

*   **資產品質與結構**
    *   **現金與短期投資**：
        *   截至Q3 FY2026: 81億美元
        *   截至FY2025: 91.51億美元
        現金及短期投資在Q3 FY2026較去年同期下降約23億美元，主要由於營運現金流被股息支付、債券償還、資本支出和股票回購所抵消。儘管有所下降，81億美元的現金儲備仍為公司提供了良好的流動性緩衝。
    *   **應收帳款 (AR)**：
        *   截至Q3 FY2026: 53.69億美元
        *   截至FY2025: 47.17億美元
        應收帳款在Q3 FY2026有所增加，這可能需要關注其回收效率。
    *   **存貨 (Inventory)**：
        *   截至Q3 FY2026: 74.87億美元 (同比下降1%)
        *   截至FY2025: 74.89億美元 (同比持平)
        存貨在Q3 FY2026略有下降，反映了公司在清理庫存方面的努力。儘管如此，管理層仍在努力通過增加折扣和批發合作夥伴的銷售退貨來減少市場上的某些鞋類產品供應，這對營收和毛利率產生了負面影響。存貨水平的健康對於避免未來進一步的折扣和利潤侵蝕至關重要。

*   **負債與償債能力**
    *   **流動性**：
        *   **流動比率 (Current Ratio)**：
            *   FY2025: 2.21 (總流動資產233.62億美元 / 總流動負債105.66億美元)
            *   Q3 FY2026: 2.21 (Stock Titan數據)
            流動比率2.21顯示NKE擁有足夠的短期資產來覆蓋短期負債，流動性狀況健康。
        *   **速動比率 (Quick Ratio)**：
            *   FY2025: 約1.5 (流動資產 - 存貨 / 流動負債 = (233.62 - 74.89) / 105.66 = 1.50)
            速動比率約1.5，表明即使不依賴存貨銷售，公司也能應對短期債務，流動性良好。
    *   **長期風險**：
        *   **債務對權益比 (Debt/Equity)**：
            *   Q3 FY2026: 約0.5 (長期債務70億美元 / 股東權益141億美元)
            *   FY2025: 0.60 (長期債務79.61億美元 / 股東權益132.13億美元)
            債務對權益比率約0.5至0.6，顯示公司槓桿水平適中，財務結構相對穩健。
        *   **長期債務佔資產比 (LT Debt/Assets)**：
            *   Q3 FY2026: 約18.9% (長期債務70億美元 / 總資產371億美元)
            *   FY2025: 約21.8% (長期債務79.61億美元 / 總資產365.79億美元)
            長期債務佔總資產的比重合理，表明公司資產負債表相對健康，長期償債風險可控。
    *   **淨現金 (Net Cash)**：
        *   截至Q3 FY2026，現金及短期投資為81億美元，長期債務為70億美元。公司處於淨現金狀態，表明其擁有超過債務的現金儲備，財務狀況良好。

*   **無形資產與商譽**
    *   **商譽 (Goodwill)**：
        *   截至FY2025: 2.4億美元
    NKE的商譽金額相對較低，佔總資產比重不大，因此商譽減損的風險較小。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF)**
    *   Q3 FY2026: 5.79億美元
    *   FY2025: 36.98億美元
    *   FY2024: 74.29億美元
    *   FY2023: 58.41億美元
    NKE的營業現金流在FY2025和Q3 FY2026呈現顯著下降。FY2025的OCF為36.98億美元，較FY2024的74.29億美元下降了50.22%。Q3 FY2026的OCF為5.79億美元，較去年同期下降67.7%。這表明儘管淨利潤有所下降，但營運活動產生的現金流下降幅度更大，顯示盈餘品質可能存在隱憂。Stock Titan指出，FY2025營收下降9.8%，而營運現金流下降50.2%，這表明需求疲軟對現金流產生了不成比例的影響。

*   **自由現金流 (FCF)**
    *   **計算方式**：FCF = OCF - 資本支出 (CapEx)
    *   Q3 FY2026: 2.84億美元 (OCF 5.79億美元 - CapEx 1.93億美元 = 3.86億美元，但Stock Titan報告為2.84億美元，我將採用報告數據)
    *   FY2025: 32.68億美元 (OCF 36.98億美元 - CapEx 4.3億美元)
    *   FY2024: 66.17億美元 (OCF 74.29億美元 - CapEx 8.12億美元)
    NKE的自由現金流在FY2025和Q3 FY2026也大幅下降。FY2025的FCF為32.68億美元，較FY2024的66.17億美元顯著減少。Q3 FY2026的FCF為2.84億美元，較上一季度下降26.4%。儘管FCF仍為正值，但其大幅下降表明公司用於擴張、派息或回購的現金能力正在減弱。FY2025的FCF保持正值部分歸因於資本支出的大幅削減。

*   **資本配置 (Capital Allocation)**
    *   **股息支付**：
        *   Q3 FY2026: 6.09億美元 (同比增長3%)
        *   FY2025: 23億美元 (同比增長6%)
    *   **股份回購**：
        *   Q3 FY2026: 0美元 (Stock Titan報告)
        *   FY2025: 30億美元 (回購3760萬股)
    NKE在過去幾年持續通過股息和股票回購向股東返還資本。公司有24年連續增加股息支付的記錄。然而，FY2025的資本回報（股息23億美元 + 回購30億美元 = 53億美元）超過了其自由現金流（32.68億美元），這解釋了現金餘額的下降。Q3 FY2026沒有進行股票回購，這可能反映了公司在現金流壓力下對資本配置策略的調整。資本配置的效率在營收和現金流下降的環境下，需要更加謹慎。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

*   **回報率指標**
    *   **ROE (股東權益報酬率)**：
        *   Q3 FY2026: 3.7%
        *   FY2025: 24.4%
        *   FY2024: 39.5%
        ROE從FY2024的39.5%大幅下降至FY2025的24.4%，並在Q3 FY2026進一步降至3.7%。這表明公司為股東創造利潤的效率顯著降低。
        **杜邦分析 (DuPont Analysis) 拆解 ROE (FY2025數據)**：
        *   **淨利潤率 (Net Profit Margin)** = 淨利潤 / 營收 = 32.19億美元 / 463.09億美元 = 6.95%
        *   **資產周轉率 (Asset Turnover)** = 營收 / 總資產 = 463.09億美元 / 365.79億美元 = 1.26
        *   **權益乘數 (Equity Multiplier)** = 總資產 / 股東權益 = 365.79億美元 / 132.13億美元 = 2.77
        *   **ROE** = 淨利潤率 x 資產周轉率 x 權益乘數 = 6.95% x 1.26 x 2.77 = 24.25% (與報告的24.4%接近)
        ROE的下降主要歸因於淨利潤率和資產周轉率的下降。淨利潤率的壓縮反映了毛利率下降和費用控制的挑戰。資產周轉率的下降則表明公司利用資產產生營收的效率降低，可能與營收下滑和庫存管理有關。
    *   **ROIC (投入資本回報率)**：
        *   FY2025: 20.2%
        *   FY2024: 34.9%
        ROIC從FY2024的34.9%下降至FY2025的20.2%。雖然20.2%仍是一個不錯的回報率，但顯著的下降表明公司投資資本的效益正在減弱。如果公司的加權平均資本成本 (WACC) 保持穩定或上升，ROIC的下降可能意味著創造經濟價值的能力正在減弱。

*   **效率指標**
    *   **資產周轉率 (Asset Turnover)**：
        *   FY2025: 1.26 (如上述杜邦分析)
        資產周轉率的下降表明公司資產利用效率降低，這與營收下滑和庫存管理挑戰相符。
    *   **存貨周轉天數 (DIO)**：
        *   FY2025: (存貨74.89億美元 / 銷貨成本265.2億美元) * 365天 = 約103天 (FY2025銷貨成本 = 營收463.09億美元 * (1 - 毛利率42.7%) = 265.2億美元)
        *   Q3 FY2026: (存貨74.87億美元 / 銷貨成本67.5億美元 (Q3 FY2026)) * 90天 = 約99.8天 (Q3 FY2026銷貨成本 = 營收113億美元 * (1 - 毛利率40.2%) = 67.5億美元)
        存貨周轉天數約為100天左右，雖然Q3 FY2026略有改善，但仍需密切關注存貨水平，以避免過度積壓導致的減值風險。
    *   **應收帳款周轉天數 (DSO)**：
        *   FY2025: (應收帳款47.17億美元 / 營收463.09億美元) * 365天 = 約37天
        *   Q3 FY2026: (應收帳款53.69億美元 / 營收113億美元) * 90天 = 約42.7天
        應收帳款周轉天數在Q3 FY2026有所增加，這可能暗示應收帳款回收速度變慢，需要關注潛在的壞帳風險。

**最終結論：**
1.  **整體評級**：存在隱憂
2.  **核心摘要**：NKE的財務健康狀況近期呈現惡化趨勢。儘管資產負債表依然穩健，擁有充足的現金和合理的債務水平，但營收增長停滯、獲利能力（毛利率、營業利益率、純益率）顯著下滑，以及營業現金流和自由現金流大幅萎縮，均表明公司營運面臨挑戰。資本效率指標如ROE和ROIC的下降，也反映出其為股東創造價值的能力正在減弱。公司正處於轉型期，重組費用和市場去庫存對短期業績造成壓力。
3.  **關鍵監測指標**：
    *   **毛利率 (Gross Margin)**：關注其能否在未來幾個季度止跌回升，特別是考慮到關稅影響和去庫存策略的進展。
    *   **自由現金流利潤率 (Free Cash Flow Margin)**：監測公司產生足夠現金以支持營運、投資和股東回報的能力。
    *   **大中華區營收增長 (Greater China Revenue Growth)**：該地區的戰略調整和市場表現將是NKE整體營收復甦的關鍵驅動力。
