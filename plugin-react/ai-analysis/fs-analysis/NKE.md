作為一名資深美股分析師，我將對NKE（Nike, Inc.）的財務狀況進行全面拆解，並根據最新的財報數據進行分析。

### 1. 損益表分析 (Income Statement Analysis)

**營收成長動能**
NKE在最近幾個季度面臨營收成長放緩的挑戰。截至2026財年第三季度（截至2026年2月28日），公司營收為113億美元，按報告基準持平，按貨幣中性基準則下降3%。這顯示出公司在整體營收上缺乏成長動能。

細分來看，批發業務（Wholesale）表現相對較好，第三季度營收為65億美元，按報告基準增長5%，按貨幣中性基準增長1%。然而，NIKE Direct（包括自有零售店和數位平台）的營收為45億美元，按報告基準下降4%，按貨幣中性基準下降7%。其中，NIKE品牌數位業務下降9%，NIKE自有門店下降5%。這表明NIKE Direct，特別是數位銷售，是當前營收下滑的主要拖累。Converse品牌的營收也大幅下滑35%至2.64億美元。

營收下滑主要歸因於NIKE Direct的疲軟以及部分地區（如大中華區和EMEA）的挑戰。公司正在透過「Win Now」策略調整市場策略，包括重新平衡與批發夥伴的關係，並將NIKE Digital重新定位為全價業務。

**獲利能力與利潤率**
NKE的獲利能力在近期承壓。2026財年第三季度的毛利率為40.2%，較去年同期下降130個基點。毛利率下降的主要原因是北美地區較高的關稅、Converse獲利能力下降，部分被較好的定價、產品組合以及較低的物流和過時庫存成本所抵消。在2026財年第一季度和第二季度，毛利率也分別下降了320個基點至42.2%和300個基點至40.6%。這顯示出利潤率持續受到壓力，且尚未穩定或擴張。

營業利益率（Operating Margin）方面，雖然最新季度數據未直接提供，但淨利潤的顯著下降暗示營業利益率也受到影響。第三季度淨利潤為5億美元，同比下降35%。稀釋後每股收益為0.35美元，同樣下降35%。這表明公司在實現規模經濟方面面臨挑戰，成本壓力未能有效轉嫁或抵消。

**費用控制**
銷售和管理費用（Selling and administrative expense）在2026財年第三季度增長2%至40億美元。這主要是由於本季度產生了員工遣散費。公司正在進行成本調整，預計這些措施的效益將在2027財年開始顯現。研發費用（R&D）的具體佔比未在最新季度報告中詳細說明，但行銷費用（Demand creation expense）在2026財年第二季度增長13%至13億美元，主要由於品牌行銷和體育行銷費用增加。這表明公司仍在投入資源於品牌建設和市場推廣，以支持其成長策略，但費用增長與營收停滯形成對比，影響了獲利能力。

### 2. 資產負債表分析 (Balance Sheet Analysis)

**資產品質與結構**
截至2026年2月28日，NKE的庫存為75億美元，同比下降1%。這主要反映了單位數量和產品組合的變化，部分被產品成本增加（主要是北美地區較高的關稅）所抵消。儘管庫存金額略有下降，但公司在某些地區（如大中華區和APLA）仍存在庫存過剩問題，需要透過促銷活動來清理。應收帳款的周轉情況未在最新季度報告中詳細說明，但庫存管理仍是公司需要持續關注的重點，以避免存貨堆積風險。

現金及約當現金和短期投資為81億美元，較去年同期減少約23億美元。這表明公司現金儲備有所減少，主要原因是營運產生的現金被現金股利、債券償還、資本支出和股票回購所抵消。

**負債與償債能力**
*   **流動性**：根據提供的基本面數據，NKE的流動比率（Current Ratio）為2.14，速動比率（Quick Ratio）為1.45。這些比率顯示公司擁有健康的短期償債能力，能夠覆蓋其短期負債。
*   **長期風險**：債務對權益比（Debt/Equity）為0.79，長期債務對權益比（LT Debt/Eq）為0.69。這些比率表明NKE的債務水平處於中等偏高水平，但仍在可控範圍內。公司在2026財年第三季度進行了債券償還，顯示其積極管理債務。
*   **淨現金狀況**：由於現金及約當現金和短期投資下降，且公司有債務，其淨現金狀況可能不如以往充裕，需要進一步分析其總債務水平來確定。

**無形資產與商譽**
最新財報中未提供商譽（Goodwill）的具體金額及佔比，因此無法直接評估其是否過高或存在減損風險。然而，作為一個品牌驅動型公司，品牌價值（無形資產）對NKE至關重要。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

**營業現金流 (OCF)**
最新季度報告指出，營運產生的現金被現金股利、債券償還、資本支出和股票回購所抵消。這暗示營業現金流可能不足以完全覆蓋這些資金流出，或者其增長速度未能與淨利潤同步。由於淨利潤在第三季度下降35%，預計營業現金流的表現也可能不佳。若OCF長期低於淨利，可能意味著公司盈餘品質不佳，例如應收帳款或存貨的增加導致現金流出，或非現金費用佔比較高。

**自由現金流 (FCF)**
自由現金流（FCF）是營業現金流減去資本支出（CapEx）。雖然具體數字未在季度報告中直接提供，但現金及約當現金和短期投資下降約23億美元，且被多項資金流出抵消，這表明FCF可能受到壓力。公司需要足夠的FCF來支持其擴張、派息和回購計畫。

**資本配置 (Capital Allocation)**
NKE在資本配置方面，過去幾年持續透過股息和股票回購向股東回報價值。在2025財年，公司向股東返還了約53億美元，其中包括23億美元的股息（增長6%）和30億美元的股票回購。在2026財年第三季度，公司透過股息向股東返還了約6.09億美元，同比增長3%。同時，公司也進行了債券償還和資本支出。

然而，隨著營收成長放緩、獲利能力下降以及現金儲備減少，公司在未來的資本配置上可能需要更加謹慎。資本配置的效率將取決於這些投資能否帶來預期的回報，尤其是在當前充滿挑戰的營運環境下。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

**回報率指標**
*   **ROE (股東權益報酬率)**：提供的基本面數據顯示ROE為0.1602。結合杜邦分析，ROE = 淨利率 x 資產周轉率 x 權益乘數。
    *   **淨利率 (Net Profit Margin)**：根據提供的數據為0.0483。最新季度淨利潤下降35%，表明淨利率正在惡化。
    *   **資產周轉率 (Asset Turnover)**：營收成長停滯，甚至下降，可能導致資產周轉率下降。
    *   **財務槓桿 (Equity Multiplier)**：債務對權益比為0.79，顯示公司有一定程度的財務槓桿。
    綜合來看，NKE的ROE可能受到淨利率下降的負面影響，即使資產周轉率和財務槓桿保持穩定，整體回報率也將承壓。
*   **ROIC (投入資本回報率)**：2025財年的ROIC為20.2%，較2024財年的34.9%大幅下降。這表明公司投資資本的效益正在顯著惡化，可能未能超過其加權平均資本成本（WACC），這是一個重要的警訊。

**效率指標**
*   **資產周轉率 (Asset Turnover)**：由於營收成長停滯，資產周轉率可能面臨壓力。
*   **存貨周轉天數 (DIO)**：儘管庫存金額略有下降，但公司在某些地區仍有庫存過剩問題，這可能導致存貨周轉天數延長，影響營運效率。
*   **應收帳款周轉天數 (DSO)**：最新財報未提供具體數據，但若營收品質下降，DSO也可能惡化。
