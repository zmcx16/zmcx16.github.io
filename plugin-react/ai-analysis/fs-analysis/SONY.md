*   **獲利能力與利潤率**：
    *   **毛利率 (Gross Margin)**：維持在 **30.89%**，表現穩定。
    *   **營業利益率 (Operating Margin)**：約 **12.24%**。Q1 營業利益年增 10%，主要受惠於影像感測器業務的強勁復甦（該部門利潤翻倍）。

*   **費用控制**：
