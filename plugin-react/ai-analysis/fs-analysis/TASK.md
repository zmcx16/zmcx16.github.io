        *   權益乘數 (Equity Multiplier)：總資產 / 股東權益。由於 Debt/Equity 為 0.5，這意味著股東權益約為總資產的 2/3 (Equity = Assets / (1 + Debt/Equity) = Assets / 1.5)，所以權益乘數約為 1.5。
