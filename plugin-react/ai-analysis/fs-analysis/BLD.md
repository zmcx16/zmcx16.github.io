    *   **結構：** 安裝業務（Installation）與特種分銷（Specialty Distribution）雙引擎驅動，後者在商業與工業市場的滲透彌補了住宅市場的波動。
*   **獲利能力與利潤率：**
    *   **毛利率 (Gross Margin)：** 約 **29.01%**（最新財報顯示調整後毛利接近 31%）。這在建築材料行業屬於高水準，顯示其對上游供應商有強大的議價能力。
    *   **營業利益率 (Oper. Margin)：** **15.18%**。受益於營運效率提升與數位化管理，利潤率呈現擴張趨勢。

*   **費用控制：**
    *   SG&A（銷售與管理費用）佔比穩定。公司專注於整合併購後的重疊職能，費用控制與其「輕資產、高效率」的策略相符。

### 2. 資產負債表分析 (Balance Sheet Analysis)

*   **資產品質與結構：**
    *   **存貨與應收帳款：** 存貨周轉天數保持在行業領先水準。由於其服務屬性強，存貨堆積風險較低。
    *   **商譽 (Goodwill)：** 由於長期採取併購策略，商譽佔總資產比例較高（約 40% 以上）。**這是潛在風險點**，若收購標的表現不如預期，可能面臨減損壓力。
*   **負債與償債能力：**
    *   **流動性：** 流動比率 **1.94**，速動比率 **1.34**，顯示短期償債能力極佳，無流動性危機。
    *   **債務結構：** 債務對權益比 (Debt/Equity) 為 **1.36**，長期債務佔比高。雖然槓桿率看似稍高，但考慮到其強大的現金流產生能力，利息保障倍數（Interest Coverage Ratio）依然健康。
    *   **淨現金狀況：** 雖然不是淨現金公司，但其債務多為固定利率長債，受短期利率波動影響有限。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF)：**
    *   BLD 的盈餘品質極高。OCF 長期高於淨利（Net Income），顯示利潤確實轉化為現金，而非僅是會計帳面數字。
*   **自由現金流 (FCF)：**
    *   **P/FCF 為 17.8**，這對於一家成長型工業股來說非常具吸引力。
    *   FCF 轉化率（FCF/Adjusted Net Income）通常超過 100%，這為其資本配置提供了充足彈藥。
*   **資本配置 (Capital Allocation)：**
    *   **併購：** 這是 BLD 的核心成長邏輯。公司持續收購地方性小型安裝商，實現行業整合。
    *   **股份回購：** 公司極其重視股東回報。2024 年持續執行大規模回購計畫，這也是支撐股價與 EPS 成長（儘管營收成長放緩）的主因。
    *   **股息：** 目前不發放股息，資金優先用於回購與併購，這符合高成長階段的資本效率原則。

*   **效率指標：**
