*   **獲利能力與利潤率**：

    *   **營業利益率 (Oper. Margin)**：目前約 **8.05%**。雖然毛利高，但營業費用（主要是技術開發與行銷）佔比較大。
    *   **純益率 (Profit Margin)**：約 **6.26%**。獲利能力穩定，但受限於規模效應尚未完全爆發，利潤率擴張速度較為溫和。
