# Target Corp (TGT) 財務報表深度分析報告

作為資深美股分析師，針對 Target (TGT) 最新公佈的 2024 財年第三季（Q3）財報及歷史數據，進行以下專業拆解。

### 1. 損益表分析 (Income Statement Analysis)

*   **營收成長動能**：
    Target 最新 Q3 營收為 256.7 億美元，年增僅 1.1%，低於市場預期。核心指標「同店銷售額 (Comparable Sales)」僅增長 0.3%，顯示成長動能極其疲軟。成長主要由數位通路（Digital Sales 年增 10.8%）驅動，但實體店面流量下滑，反映消費者在非必需品（如家居、電子產品）的支出依然謹慎。
*   **獲利能力與利潤率**：
    *   **毛利率 (Gross Margin)**：約 27.2%，雖受惠於成本控制，但被高昂的物流與庫存管理成本抵銷。
    *   **營業利益率 (Operating Margin)**：從去年同期的 5.2% 下滑至 **4.6%**。這是一個警訊，顯示公司在維持營收的同時，必須付出更高的促銷成本與供應鏈支出。
    *   **純益率 (Net Margin)**：目前約 3.24%，處於零售業中段，但相較於 Walmart (WMT) 的規模經濟優勢，Target 的利潤空間受通膨與薪資上漲擠壓更為明顯。
*   **費用控制**：
    SG&A 費用佔比上升，主要源於薪資調漲及為了應對供應鏈不確定性而提前備貨產生的倉儲費用。

### 2. 資產負債表分析 (Balance Sheet Analysis)

*   **資產品質與結構**：
    *   **存貨 (Inventory)**：Q3 存貨金額達 152 億美元，年增約 10%。在營收僅增長 1.1% 的背景下，存貨增速遠超營收增速，暗示未來 1-2 季可能面臨「清庫存」的毛利稀釋風險。
*   **負債與償債能力**：
    *   **流動性**：流動比率 (Current Ratio) 為 0.93，速動比率 (Quick Ratio) 僅 0.3。這在零售業雖非罕見，但反映其短期資金調度極度依賴存貨變現。
    *   **債務結構**：債務對權益比 (Debt/Equity) 為 1.17，長期債務穩定，但利息保障倍數因營業利益下滑而承壓。
*   **無形資產**：商譽佔比穩定，目前無重大減損風險。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF)**：
    YTD（年初至今）營業現金流表現尚可，但 Q3 單季因存貨增加導致現金流入放緩。盈餘品質（OCF/Net Income）仍大於 1，顯示利潤具備現金支撐。
*   **自由現金流 (FCF)**：
    FCF 受資本支出 (CapEx) 影響較大。Target 持續投資於門市翻新與配送中心自動化。雖然目前 FCF 足以支付股息，但回購股份的空間已因利潤指引下修而收窄。
*   **資本配置**：
    Target 是著名的「股息國王」，已連續 50 多年增加股息。Q3 支付了 5.16 億美元股息，並回購了約 3.5 億美元股份。然而，在獲利預警下，過度的股份回購可能削弱財務韌性。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

*   **杜邦分析 (ROE 拆解)**：
    *   **ROE (22.02%)** = 純益率 (3.24%) × 資產周轉率 (約 1.8) × 財務槓桿 (約 3.8)。
    *   Target 的高 ROE 高度依賴**財務槓桿**與**資產周轉**。當純益率與周轉率（受存貨堆積影響）雙降時，ROE 將面臨劇烈修正。
*   **效率指標**：
    存貨周轉天數 (DIO) 有拉長趨勢，這與 Q3 財報中提到的「提前備貨以應對港口罷工」有關，但若 Q4 旺季銷售不如預期，將演變成財務包袱。

**最終結論：**
1. **整體評級**：**[存在隱憂]**
2. **核心摘要**：Target 目前正處於獲利修正期。雖然營收微幅增長，但為了應對供應鏈風險而堆積的存貨，以及非必需品類別的疲軟，導致營業利益率嚴重受壓。公司雖具備強大的股息支付能力，但短期內缺乏強勁的成長催化劑，且財務槓桿在利潤萎縮時會放大風險。
3. **關鍵監測指標**：
    *   **Inventory-to-Sales Ratio**：觀察存貨增速是否回落至營收增速之下。
    *   **Operating Margin**：能否回升至 5% 以上的歷史正常水平。
    *   **Comparable Sales Growth**：實體店面客流量是否恢復正成長。
