# CAMT 財務狀況深度拆解報告

## 1. 損益表分析 (Income Statement Analysis)

*   **營收成長動能：強勁且具備 AI 題材支撐**
    *   **趨勢**：CAMT 2024 年 Q1 營收達到 9,700 萬美元，年增 34%，創下單季歷史新高。過去 3 年複合成長率（CAGR）表現優異。
    *   **驅動因素**：成長主要來自**銷量增加**，特別是高效能運算（HPC）和 AI 晶片所需的 **HBM（高頻寬記憶體）** 與 **Chiplets（小晶片）** 的先進封裝檢測需求。這並非單純漲價，而是受惠於半導體製程轉向先進封裝的結構性紅利。
*   **獲利能力與利潤率：高位穩定**
    *   **毛利率 (Gross Margin)**：目前維持在 **50.5%** 左右。雖然面臨供應鏈成本壓力，但因產品組合轉向高階檢測設備，毛利率保持穩定。
    *   **營業利益率 (Operating Margin)**：約 **25.8%**。這顯示公司在規模擴張的同時，行政與銷售費用控制良好，具備明顯的規模經濟效應。
    *   **純益率 (Net Margin)**：約 **10.2%**（GAAP），但若觀察 Non-GAAP 數據則更高。目前的 P/E 較高（209x）主因是過去一年有較多的一次性支出或會計調整，Forward P/E 44x 更能反映其未來的獲利預期。
*   **費用控制**：
    *   **研發費用 (R&D)**：持續佔營收約 13-15%，這對於半導體設備商至關重要，確保其在光學檢測技術上的領先地位。

## 2. 資產負債表分析 (Balance Sheet Analysis)

*   **資產品質與結構**：
    *   **現金水平**：截至 2024 年 Q1，公司擁有約 **4.48 億美元** 的現金及等價物。這在 88 億市值的公司中屬於極其充裕的水平。
    *   **存貨與應收帳款**：存貨周轉天數略有增加，這是為了應對 2024 下半年強勁的訂單預期（HBM 產能擴張），屬於健康的戰略性囤貨。
*   **負債與償債能力**：
    *   **流動性極佳**：流動比率 **8.35**，速動比率 **7.31**。這意味著公司每 1 元的短期負債，有超過 7 元的現金或約當現金支撐，完全沒有短期償債風險。
    *   **債務結構**：Debt/Equity 為 **0.86**。主要債務來自於 2023 年底發行的可轉換公司債（Convertible Notes），票面利率極低，主要是為了併購（如收購 FRT Metrology）及未來擴張儲備資金。
*   **無形資產與商譽**：
    *   收購 FRT 後商譽有所增加，但目前無減損跡象，因為該業務與 CAMT 原有產品線整合良好，產生了交叉銷售的綜效。

## 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF)**：
    *   CAMT 的 OCF 長期為正。2024 Q1 營運現金流表現穩健，與淨利增長趨勢一致。盈餘品質高，並未出現「有獲利、無現金」的窘境。
*   **自由現金流 (FCF)**：
    *   **FCF = OCF - CapEx**。由於 CAMT 屬於輕資產（Asset-light）組裝模式，資本支出（CapEx）佔營收比例不高，因此 FCF 轉化率極強。這為公司提供了大量的戰略靈活性。
*   **資本配置 (Capital Allocation)**：
    *   **併購 (M&A)**：2023 年收購 FRT 是關鍵一步，擴展了其在混合鍵合（Hybrid Bonding）市場的檢測能力。
    *   **回購與派息**：目前公司傾向於保留現金進行研發與潛在併購，而非大規模派息，這符合成長型科技股的特徵。

## 4. 核心財務指標與杜邦分析 (DuPont Analysis)

*   **ROE (8.7%) 拆解**：
    1.  **淨利率 (10.2%)**：中規中矩，受研發投入影響。
    2.  **資產周轉率 (0.46)**：偏低，主因是資產負債表上躺了過多現金（現金是資產但不直接產生營收）。
    3.  **財務槓桿 (1.8)**：適中，主要由可轉債構成。
    *   *分析師觀點*：若扣除超額現金，CAMT 的 **核心 ROIC（投入資本回報率）** 實際上遠高於 ROE，顯示其核心業務極具賺錢效率。
*   **效率指標**：
    *   **DSO (應收帳款周轉天數)**：維持在 70-80 天，符合半導體設備行業慣例。
