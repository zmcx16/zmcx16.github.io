身為資深美股分析師，我將針對Salesforce (CRM) 的財務狀況進行全面拆解，並根據最新的財報數據進行分析。

### 1. 損益表分析 (Income Statement Analysis)

**營收成長動能**
Salesforce展現了穩健的營收成長。截至2026年4月30日的第一季度（FY27 Q1），公司營收達到111.3億美元，年增13%（以固定匯率計算為12%）。其中，Informatica的貢獻為4.44億美元。在截至2026年1月31日的2026財年，Salesforce的年度營收為415億美元，年增10%。過去五年的營收複合年增長率（CAGR）為13.88%。

營收成長主要來自於其核心的訂閱與支援服務，FY27 Q1該部分營收為106億美元，年增14%（以固定匯率計算為12%），其中包含Informatica的4.28億美元貢獻。公司積極推動AI相關產品，如Agentforce和Data Cloud，Agentforce的年化經常性收入（ARR）已超過10億美元，AI和數據相關ARR達到34億美元，其中50%來自現有客戶的擴展。這顯示其成長動能主要來自於產品創新、現有客戶的擴展以及策略性併購（如Informatica）。然而，部分地區（如EMEA）的成長受限，以及分銷能力持平，可能對整體成長速度構成挑戰。

**獲利能力與利潤率**
Salesforce的獲利能力顯著提升。截至2026年4月30日的第一季度，非GAAP營業利益率達到34.8%，年增250個基點，創下歷史新高。GAAP營業利益率也達到21.1%，年增130個基點。

從2021財年到2026財年，毛利率呈現穩定且略微上升的趨勢，從74.41%增長至77.68%。營業利益率則有顯著改善，從2021財年的2.14%大幅提升至2026財年的20.06%。淨利率也從2023財年的0.66%急劇上升至2026財年的17.96%。這表明公司在營運效率和成本控制方面取得了顯著進展，並可能開始展現規模經濟效應。

**費用控制**
雖然具體的研發費用（R&D）和行銷費用（S&A）佔比未直接提供，但營業利益率的顯著提升（從2021年的2.14%到2026年的20.06%） 表明公司在費用控制方面表現良好。這與公司從「不惜一切代價成長」轉向「積極關注營運效率和資本回報」的策略轉變相符。公司正透過Agentforce AI框架推動有機利潤率擴張。

### 2. 資產負債表分析 (Balance Sheet Analysis)

**資產品質與結構**
截至2026財年，Salesforce的總資產為1123.05億美元。由於公司主要提供SaaS服務，其資產結構中無形資產和商譽佔比較高，這通常是科技公司併購策略的結果。應收帳款和存貨的具體周轉情況未直接提供，但其流動比率和速動比率的趨勢可以間接反映資產品質。

**負債與償債能力**
*   **流動性**：Salesforce的流動性指標顯示出潛在的壓力。截至2026年5月（TTM），流動比率為0.76，而速動比率為0.6x。這兩項比率均低於1，表示公司可能難以僅憑流動資產來覆蓋短期負債。從2021年到2026年，流動比率從1.23下降至0.76，速動比率從1.11下降至0.64，顯示流動性持續惡化。這可能需要進一步關注其營運資金管理。
*   **長期風險**：截至2026年4月30日，債務對權益比（Debt/Equity）為1.22。這遠高於其10年平均中位數0.23，顯示公司在融資成長方面對債務的依賴程度顯著增加。截至2026年4月，短期債務和資本租賃義務為5.57億美元，長期債務和資本租賃義務為413.27億美元。儘管債務水平較高，但其利息覆蓋率為41.45，表明公司仍有強大的能力償付債務利息。
*   **淨現金狀況**：雖然未直接提供淨現金數據，但高額的長期債務和較低的流動比率表明公司可能不處於淨現金狀態，而是有相當的淨負債。

**無形資產與商譽**
Salesforce透過多項大型併購（如Tableau、Slack、MuleSoft和最近的Informatica）擴展其技術生態系統。這些併購通常會產生大量的商譽和無形資產。雖然具體商譽金額未提供，但過去的併購策略可能導致商譽佔總資產比重較高，這需要持續監測是否存在減損風險。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

**營業現金流 (OCF)**
Salesforce的營業現金流表現強勁。截至2026年4月30日的第一季度，營業現金流為67億美元，年增3%。這表明公司營運產生的現金與淨利潤同步增長，甚至可能更高，顯示其盈餘品質良好。

**自由現金流 (FCF)**
自由現金流（FCF）也表現出色。截至2026年4月30日的第一季度，自由現金流為66億美元，年增4%。截至2026年4月30日的過去十二個月（TTM），自由現金流為146.61億美元，年增15.93%。2026財年的年度自由現金流為144億美元，年增15.83%。過去五年的自由現金流複合年增長率為21.16%。這顯示公司擁有充足的現金用於擴張、派息或回購。

**資本配置 (Capital Allocation)**
Salesforce的資本配置策略已發生重大轉變。過去，公司以「不惜一切代價成長」的理念，透過股權稀釋進行大規模併購（如Slack、Tableau、MuleSoft）。然而，在激進投資者的壓力下，公司已將重點轉向營運效率和大規模資本回報。

公司董事會已授權500億美元的股票回購計畫，並在FY27 Q1啟動了250億美元的加速股票回購（ASR）計畫，這使得稀釋後的流通股數量年減10%。此外，公司還將季度股息提高了5.8%至每股0.44美元。這表明公司正積極透過回購和派息來回饋股東，並提升每股盈餘能力。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

**回報率指標**
*   **ROE (股東權益報酬率)**：截至2026年5月26日，ROE為12.61%。其5年平均ROE為6.34%，顯示近期ROE有顯著改善。杜邦分析拆解ROE（利潤率 x 周轉率 x 財務槓桿）來看，淨利率的提升（從2023年的0.66%到2026年的17.96%） 是推動ROE增長的主要因素。然而，高債務對權益比（1.22） 也增加了財務槓桿，這在提升ROE的同時也增加了財務風險。
*   **ROIC (投入資本回報率)**：截至2026年4月，ROIC約為6.87%至7.9%。而其加權平均資本成本（WACC）為9.05%。由於ROIC（6.87%）低於WACC（9.05%），這表示Salesforce目前的投資資本效益未能超過其資本成本，從長遠來看，這可能會損害股東價值。這與過去透過股權稀釋進行高調併購導致ROIC受壓的歷史相符。

**效率指標**
*   **資產周轉率**：雖然未直接提供資產周轉率，但淨固定資產周轉率從2021年的3.75上升到2026年的8.11，表明公司在利用固定資產產生收入方面的效率正在提高。
*   **存貨周轉天數 (DIO) 與應收帳款周轉天數 (DSO)**：由於Salesforce是SaaS公司，存貨通常不是關鍵指標。應收帳款周轉天數未直接提供，但流動比率和速動比率的下降可能暗示應收帳款回收效率需要關注。

**最終結論：**

2.  **核心摘要**：Salesforce在營收成長和獲利能力方面表現出色，特別是近期利潤率的顯著擴張和強勁的自由現金流。公司積極透過股票回購和股息回饋股東，顯示其資本配置策略的轉變。然而，流動性指標的惡化和高債務對權益比帶來償債安全性隱憂，且ROIC低於WACC，表明資本效率仍有提升空間。
