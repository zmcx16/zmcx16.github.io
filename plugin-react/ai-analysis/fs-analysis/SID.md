Companhia Siderúrgica Nacional (NYSE: SID) 是一家巴西綜合鋼鐵生產商，業務涵蓋鋼鐵、礦業、水泥、能源和物流等多個領域。根據最新的財務報告，SID 在2025財年及2026年第一季度面臨顯著的獲利能力挑戰和高槓桿風險。

### 1. 損益表分析 (Income Statement Analysis)

*   **營收成長動能**：
    SID 在2025財年實現了 R$44.80 億的總營收。雖然物流業務在2025年營收達到 R$43.742 億，且鋼鐵業務的調整後 EBITDA 在2025年同比增長36.9%，但整體營收的具體同比增長率在可獲取的最新資訊中未明確指出。然而，從獲利能力來看，營收成長並未轉化為正向的淨利潤。

*   **獲利能力與利潤率**：
    SID 在2025財年的獲利能力表現不佳。毛利率為 26.14%，但由於高額的「其他費用」（R$13.7 億），導致營業利潤為負 R$2.0 億，營業利益率約為 -4.46%。最終，公司在2025財年錄得 R$2.0 億的淨虧損，淨益率為 -4.47%。這顯示公司在控制營運成本和費用方面存在挑戰，未能實現規模經濟效益，且核心業務的獲利能力受到侵蝕。

*   **費用控制**：
    2025財年 R$13.7 億的「其他費用」 顯著侵蝕了毛利，並導致營業虧損。由於缺乏研發費用 (R&D) 和銷售與管理費用 (S&A) 的具體細項，難以評估其合理性及與公司成長策略的匹配度。然而，如此龐大的費用支出導致營業虧損，表明費用控制可能存在問題或公司面臨較高的營運成本壓力。

### 2. 資產負債表分析 (Balance Sheet Analysis)

*   **資產品質與結構**：
    截至2025年12月31日，SID 持有 R$160 億的現金及現金等價物，但在2026年第一季度末降至 R$146 億。雖然現金儲備相對充足，足以覆蓋短期債務，但其下降趨勢值得關注。目前缺乏應收帳款和存貨周轉情況的詳細數據，無法判斷是否存在存貨堆積或應收帳款回收過慢的風險。

*   **負債與償債能力**：
    *   **流動性**：流動比率為 1.08，速動比率為 0.71。流動比率略高於 1，表明短期償債能力尚可，但速動比率低於 1，顯示公司對存貨的依賴性較高，若存貨變現能力不佳，可能面臨流動性壓力。
    *   **長期風險**：截至2025年12月31日，公司合併淨負債高達 R$412.176 億，2026年第一季度略降至 R$405.05 億。淨負債對過去十二個月 EBITDA 比率在2025年底為 3.47 倍，2026年第一季度為 3.36 倍，顯示公司槓桿水平非常高。債務對權益比率在2025年底為 336.3% (即 3.36 倍)，遠高於健康水平，表明公司資本結構嚴重依賴債務融資，存在較大的長期償債風險。公司正積極通過資產剝離（計劃 R$160-180 億）和債務再融資（已安排 12-14 億美元擔保貸款）來降低槓桿。
    *   **淨現金 (Net Cash)**：鑑於龐大的淨負債，SID 處於淨負債狀態。

*   **無形資產與商譽**：目前可獲取的資訊中未提供無形資產和商譽的具體數據，因此無法評估其減損風險。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF)**：
    2025年第四季度調整後現金流為負 R$2.823 億，儘管較2025年早些時候有所改善，但仍為負值。2026年第一季度自由現金流為負 R$15.977 億，主要受季節性營運疲軟、營運資金消耗、高財務費用和大量債務攤銷影響。這表明公司的營運現金流可能不足以覆蓋其營運和投資需求，盈餘品質存在隱憂。

*   **自由現金流 (FCF)**：
    2026年第一季度自由現金流為負 R$15.977 億，這是一個重大的警訊。負的自由現金流意味著公司未能從其核心業務中產生足夠的現金來支付資本支出，更遑論用於擴張、派息或回購。這將迫使公司依賴外部融資或資產出售來維持運營和償債。

*   **資本配置 (Capital Allocation)**：
    2025年資本支出 (CapEx) 達到 R$59.36 億，高於2024年的 R$55.25 億，主要用於 P15 項目以及物流和能源資產。這表明公司仍在進行大規模投資以支持未來成長。然而，在自由現金流為負的情況下進行大規模資本支出，凸顯了公司對外部資金的依賴。公司也積極通過出售資產（如2025年出售 MRS Logística S.A. 11% 股權獲得 R$33.5 億）來優化資本結構和降低槓桿。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

*   **回報率指標**：
    股東權益報酬率 (ROE) 為 -0.1663，資產報酬率 (ROA) 為 -0.0205。負值的 ROE 和 ROA 與2025財年的淨虧損一致，表明公司未能為股東創造價值，且資產運用效率低下。
    **杜邦分析 (ROE 拆解)**：
    ROE = 淨利率 x 資產周轉率 x 權益乘數。
    SID 的淨利率為 -4.47%，這是導致 ROE 為負的主要原因。儘管公司擁有高達 3.36 倍 的權益乘數（高財務槓桿），但由於核心獲利能力為負，高槓桿反而放大了虧損，導致 ROE 嚴重惡化。

*   **效率指標**：
    目前缺乏資產周轉率、存貨周轉天數 (DIO) 和應收帳款周轉天數 (DSO) 的具體數據。然而，2025年第四季度營運資金的釋放 和2026年第一季度營運資金的消耗 表明營運資金管理存在波動。

**最終結論：**
1.  **整體評級**：具重大風險
2.  **核心摘要**：Companhia Siderúrgica Nacional (SID) 的財務健康狀況令人擔憂。公司在2025財年錄得顯著虧損，且2026年第一季度自由現金流為負，顯示其核心業務獲利能力和現金生成能力嚴重不足。高達 3.36 倍的債務對權益比率和超過 3 倍的淨負債對 EBITDA 比率，使其面臨巨大的償債壓力。儘管管理層正積極通過資產剝離和債務再融資來應對，但短期內財務風險依然嚴峻。
3.  **關鍵監測指標**：
    *   **Free Cash Flow Margin (自由現金流利潤率)**：密切關注公司能否扭轉負向自由現金流，實現正向現金生成。
    *   **Net Debt/EBITDA (淨負債對 EBITDA 比率)**：監測公司去槓桿化策略的進展，目標是將此比率顯著降低。
    *   **Operating Margin (營業利益率)**：觀察公司能否改善核心業務的營運效率，將營業利益率轉為正值並持續擴張。
