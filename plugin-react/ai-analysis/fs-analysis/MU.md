# 美光科技 (MU) 財務深度分析報告

### 1. 損益表分析 (Income Statement Analysis)
美光正處於記憶體週期的強勁復甦階段，AI 需求是核心驅動力。

*   **營收成長動能**：
    *   **最新季度表現**：Q3 營收達 **68.1 億美元**，較去年同期的 37.5 億美元大幅增長 **82%**，環比增長 17%。
    *   **驅動因素**：成長主要來自 **HBM3E（高頻寬記憶體）** 的出貨以及資料中心對 DDR5 的強勁需求。這屬於「量價齊揚」：AI 伺服器導致高端產品供不應求，帶動平均售價 (ASP) 上升。
*   **獲利能力與利潤率**：
    *   **毛利率 (Gross Margin)**：從去年同期的 -17.8% 驚人地回升至 **28.1% (Non-GAAP)**。這顯示記憶體價格已走出谷底，且高毛利的 HBM 產品佔比提升。
    *   **營業利益率**：本季轉虧為盈，營業利益達 9.4 億美元。
    *   **規模經濟**：隨著產能利用率回升，固定成本被攤提，利潤率正處於擴張週期。
*   **費用控制**：
    *   研發費用 (R&D) 維持在約 10 億美元/季，佔營收比重隨營收擴大而下降，顯示研發效率提升。公司專注於 1-beta DRAM 與 232 層 NAND 技術，維持技術領先。

### 2. 資產負債表分析 (Balance Sheet Analysis)
美光的資產負債表極為穩健，足以支撐其高額的資本支出。

*   **資產品質與結構**：
    *   **現金狀況**：持有現金及短期投資約 **92 億美元**。
    *   **存貨周轉**：存貨金額約 56 億美元。雖然金額隨營收增長，但存貨天數 (DIO) 已從高點回落，顯示去庫存週期結束，進入補庫存階段。
*   **負債與償債能力**：
    *   **流動性**：流動比率 (Current Ratio) 約 **2.5x - 3.0x**，速動比率超過 **2.0x**，短期償債能力極佳。
    *   **債務結構**：總債務約 134 億美元，**淨債務 (Net Debt)** 僅約 42 億美元。債務/權益比 (D/E) 處於 0.3 左右的低位，財務槓桿安全。
*   **無形資產**：美光以自有技術研發為主，商譽佔比極低，不存在重大的資產減損風險。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)
現金流是美光本季最大的亮點，正式宣告走出「燒錢」陰霾。

*   **營業現金流 (OCF)**：本季 OCF 達 **24.8 億美元**，較上一季的 12.2 億大幅翻倍，且與淨利同步增長，盈餘品質極高。
*   **自由現金流 (FCF)**：
    *   計算：OCF (24.8億) - CapEx (20.6億) = **4.2 億美元**。
    *   這是美光在經歷連續數季負 FCF 後，**首次轉正**。這對於資本密集型的半導體公司是極其重要的轉折點。
*   **資本配置 (Capital Allocation)**：
    *   **資本支出 (CapEx)**：預計 2024 財年約 80 億美元，2025 財年將顯著增加（用於愛達荷州與紐約州的新廠）。
    *   **股東回報**：儘管處於擴張期，美光仍維持每季 0.115 美元的派息，顯示對長期現金流的信心。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)
*   **ROE 拆解**：
    *   **利潤率**：正在快速修復（從負值轉向雙位數）。
    *   **資產周轉率**：隨著 AI 伺服器需求爆發，資產周轉效率正在提升。
    *   **財務槓桿**：保持穩定，ROE 的增長主要由「獲利能力」驅動，而非過度借債。
*   **ROIC (投入資本回報率)**：目前正處於回升軌道。隨著 HBM3E 產能爬坡，預計 2025 年 ROIC 將超越 WACC（加權平均資本成本），創造經濟價值。
*   **效率指標**：應收帳款周轉天數 (DSO) 保持在 50 天左右，回款速度穩定。
