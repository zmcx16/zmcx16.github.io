# Teradata Corp (TDC) 財務報表深度分析報告

作為資深美股分析師，針對 Teradata (TDC) 的最新財務數據（結合 2024 年第三季財報與提供之指標）進行全面拆解。Teradata 目前正處於從傳統地端授權轉向雲端訂閱制（SaaS）的轉型關鍵期，這反映在其波動的損益表與強勁的現金流表現上。

---

### 1. 損益表分析 (Income Statement Analysis)

*   **營收成長動能**：
    *   **轉型陣痛期**：最新季度營收約 4.4 億美元，同比持平或微跌。成長動能完全來自於 **公有雲 ARR (年度經常性收入)**，該指標同比增長達 26%（固定匯率下），達到 6.1 億美元。
    *   **結構位移**：傳統維護與諮詢業務持續萎縮，這解釋了總營收成長乏力的原因。成長並非來自銷量暴增，而是來自現有客戶向雲端遷移及訂閱單價的提升。
*   **獲利能力與利潤率**：
    *   **毛利率 (Gross Margin)**：維持在 **60.6%** 的高水準，顯示其數據倉庫技術仍具競爭力。
    *   **營業利益率 (Oper. Margin)**：GAAP 營業利益率僅 **6.63%**，主因是轉型期間需同時維持雲端研發與地端支援，且雲端基礎設施成本較高。
    *   **純益率 (Profit Margin)**：數據顯示為 **24.93%**，顯著高於營業利益率。這主要源於 2024 年度的**稅務利益（Tax Benefit）**一次性沖回，並非核心業務獲利能力的常態表現。
*   **費用控制**：
    *   研發 (R&D) 與行銷 (S&A) 佔比穩定。公司正積極優化雲端架構以降低雲端營運成本，試圖實現規模經濟。

### 2. 資產負債表分析 (Balance Sheet Analysis)

*   **資產品質與結構**：
    *   **現金狀況**：持有現金及等價物約 3.2 億美元。
    *   **周轉效率**：應收帳款周轉天數（DSO）維持在軟體產業合理區間，無明顯存貨風險（軟體業特性）。
*   **負債與償債能力**：
    *   **流動性**：流動比率 **1.3**，速動比率 **1.29**，短期償債能力無虞。
    *   **債務結構**：總債務對權益比 (Debt/Equity) 為 **0.99**，長期債務佔比 **0.86**。雖然槓桿看似不低，但考慮到其穩定的經常性收入，債務壓力尚在可控範圍。
    *   **淨現金狀況**：由於持續進行股份回購，股東權益被壓縮，導致 Debt/Equity 比例顯得較高，但實際利息保障倍數仍屬健康。
*   **無形資產與商譽**：商譽佔比穩定，近期無重大併購，減損風險較低。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

*   **營業現金流 (OCF)**：這是 TDC 最強勁的部分。儘管 GAAP 淨利受轉型影響波動，但 OCF 表現優異，主要受益於預收訂閱款項。
*   **自由現金流 (FCF)**：
    *   **P/FCF 僅 4.63**，顯示公司相對於其產生的現金流被嚴重低估。
    *   FCF 轉換率極高，公司預計全年 FCF 可達 3.4 億至 3.8 億美元區間。
*   **資本配置 (Capital Allocation)**：
    *   **股份回購**：TDC 極其激進地進行股份回購，過去一年回購金額幾乎佔據了大部分 FCF。這也是 ROE 被推升至 **117.76%** 的主因（分母權益因回購而極小）。
    *   **不發放股息**：資金集中於回購與雲端技術研發。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

*   **回報率指標**：
    *   **ROE (117.76%)**：此數字具誤導性，主要是財務槓桿與股份回購導致的權益縮減，而非單純的獲利暴增。
    *   **ROIC (40.72%)**：這是一個更真實的指標，顯示 TDC 在投入資本上的獲利效率極高，遠超其 WACC（加權平均資本成本）。
*   **效率指標**：
    *   資產周轉率穩定，顯示公司在轉型過程中仍能有效利用現有資產產生營收。

### 5. 財務健康總結

| 維度 | 評等 (1-5★) | 關鍵評語 (簡短扼要) |
| :--- | :--- | :--- |
| **獲利能力** | ★★★ | 雲端轉型壓低營業利潤，純益受稅務因素美化。 |
| **償債安全性** | ★★★★ | 現金流充沛，足以覆蓋債務利息，流動性穩健。 |
| **現金流品質** | ★★★★★ | FCF 極其強勁，P/FCF 顯示估值極具吸引力。 |
| **資本效率** | ★★★★ | ROIC 表現優異，股份回購積極提升股東價值。 |

**最終結論：**
1. **整體評級**：**穩健**（具備價值投資屬性）
2. **核心摘要**：Teradata 正處於「低估值、高現金流」的轉型期。雖然總營收成長緩慢，但高毛利的雲端 ARR 佔比持續提升。極低的 P/FCF (4.63) 與積極的回購政策為股價提供了強大支撐，但市場仍在等待其營業利益率隨規模經濟擴張而回升。
3. **關鍵監測指標**：
    *   **Public Cloud ARR Growth**：能否維持 25% 以上的增長以抵銷地端流失。
    *   **Non-GAAP Operating Margin**：觀察雲端業務規模化後，利潤率是否能重回 20% 以上。
    *   **Free Cash Flow Margin**：確認現金流產生能力是否因競爭加劇而受損。
