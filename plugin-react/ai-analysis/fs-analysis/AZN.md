# AZN (AstraZeneca) 財務報表深度分析報告

作為資深美股分析師，針對阿斯特捷利康（AstraZeneca, AZN）2024 年上半年（H1）及最新財報數據，進行以下專業拆解：

### 1. 損益表分析 (Income Statement Analysis)
*   **營收成長動能**：AZN 在 2024 年上半年表現強勁，總營收達 256.17 億美元，按固定匯率（CER）計算增長 18%。成長主要由**銷量驅動**而非單純漲價。核心支柱包括：
    *   **腫瘤治療（Oncology）**：增長 22%，其中 Tagrisso (+12%) 與 Enhertu (+49%) 表現極佳。
    *   **生物製藥（CVRM）**：Farxiga 營收達 37.8 億美元 (+36%)，展現強大的市場滲透力。
    *   **罕見疾病**：Ultomiris 增長 34%，成功承接了 Soliris 的專利到期壓力。
*   **獲利能力與利潤率**：
    *   **毛利率**：GAAP 毛利率約 74.3%，核心（Core）毛利率則高達 82.4%。利潤率保持穩定，反映出高價值生物藥物佔比提升。
    *   **營業利益率**：GAAP 營業利益率為 23.2%，核心營業利益率則提升至 34% 左右。規模經濟效應顯著，隨著營收擴張，固定成本被有效攤提。
*   **費用控制**：研發費用（R&D）佔營收比重約 22-24%（H1 投入 57.9 億美元），這在製藥業屬於高水位，符合其「科學領先」的擴張策略。銷售與行政費用（SG&A）佔比隨營收增長而相對下降，顯示管理效率提升。

### 2. 資產負債表分析 (Balance Sheet Analysis)
*   **資產品質與結構**：
    *   **現金狀況**：截至 2024 年中，現金及等價物約 80 億美元。應收帳款周轉天數保持穩定，無明顯回收風險。
    *   **存貨管理**：存貨隨營收規模擴大略有增加，但周轉率維持在行業平均水平，未見堆積跡象。
*   **負債與償債能力**：
    *   **流動性**：流動比率 0.91 與速動比率 0.71 偏低，反映出公司利用短期負債進行營運資助的習慣，但在大型藥企中，憑藉強大的現金流產生能力，此數值尚在可接受範圍。
    *   **長期風險**：債務對權益比（Debt/Equity）為 0.72，長期債務約 248 億美元。淨債務/EBITDA 比率持續下降，顯示償債壓力隨獲利增長而減輕。
*   **無形資產與商譽**：商譽高達 190 億美元以上（主要來自 Alexion 收購）。目前 Ultomiris 等產品表現優異，短期內無商譽減損風險。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)
*   **營業現金流 (OCF)**：2024 年上半年 OCF 為 51.2 億美元。雖然淨利增長，但 OCF 受營運資本變動（如稅款支付時間與庫存增加）影響，略低於去年同期，但盈餘品質（OCF/Net Income）仍維持在 1.0 以上的健康區間。
*   **自由現金流 (FCF)**：計算 H1 FCF 約為 40 億美元（OCF 減去約 11 億美元的資本支出）。FCF 足以支撐其 1.72% 的股息發放。
*   **資本配置 (Capital Allocation)**：
    *   **併購與研發**：AZN 展現極強的擴張意圖，近期收購 Amolyt Pharma 與 Fusion Pharmaceuticals，旨在強化罕見病與放射性藥物管線。
    *   **股息**：公司近期宣佈將 2024 全年股息提高 0.20 美元至 3.10 美元，顯示對未來現金流的信心。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)
*   **ROE 拆解**：
    *   **淨利率 (17.2%)** x **資產周轉率 (0.55)** x **財務槓桿 (2.5)** = **ROE (23.5%)**。
    *   高 ROE 主要來自於優異的產品利潤率與適度的財務槓桿。
*   **ROIC**：投入資本回報率約 14.2%，遠高於其加權平均資本成本（WACC，約 7-8%），顯示公司每投入一塊錢都能創造超額價值。
*   **效率指標**：資產周轉率穩定，顯示收購 Alexion 後的資產整合已進入收割期。

**最終結論：**
1. **整體評級**：**優良**
2. **核心摘要**：AZN 正處於獲利爆發期，憑藉腫瘤藥物的領先地位與罕見疾病領域的成功轉型，營收與核心 EPS 均實現雙位數增長。儘管流動比率略低，但強大的 OCF 與持續改善的資產負債表使其具備極高的財務韌性，足以應對高額研發投入與戰略併購。
3. **關鍵監測指標**：
    *   **Core Operating Margin**：觀察是否能維持在 30% 以上。
    *   **Net Debt/EBITDA**：監控槓桿下降速度，確保財務靈活性。
    *   **Pipeline Success Rate**：特別是新收購項目的臨床進展，這決定了長期資產周轉率。
