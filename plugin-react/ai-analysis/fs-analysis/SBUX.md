## 星巴克 (SBUX) 財務狀況全面拆解與健康評等

作為資深美股分析師，我將對星巴克 (SBUX) 的財務報表進行深入分析，以評估其獲利能力、償債能力、現金流品質及資本效率，並給出最終的財務健康評等。

### 1. 損益表分析 (Income Statement Analysis)

#### 營收成長動能
星巴克在過去五年展現了穩定的營收成長。從財年2021年的290.6億美元增長至財年2025年的371.84億美元。
*   **財年2021至2025年營收趨勢 (單位：百萬美元)**
    *   FY2021: $29,060
    *   FY2022: $32,250 (年增 10.98%)
    *   FY2023: $35,976 (年增 11.55%)
    *   FY2024: $36,176 (年增 0.56%)
    *   FY2025: $37,184 (年增 2.79%)
*   **最新季度 (2026財年第二季度) 營收**：截至2026年3月29日的第二季度，星巴克實現淨營收95億美元，較2025財年第二季度的88億美元增長8.8%。
*   **成長來源**：2026財年第二季度的營收增長主要來自於全球同店銷售額增長6.2%，其中交易量增長3.8%，平均客單價增長2.3%。北美地區同店銷售額增長7.1%，國際市場同店銷售額增長2.6%。這表明營收成長主要來自於銷量增加和平均客單價的提升，而非大規模併購。公司也提到「Back to Starbucks」計畫推動了營收和利潤的增長。

#### 獲利能力與利潤率
*   **毛利率 (Gross Margin)**：
    *   2026財年第二季度為20.08%。
    *   過去五年毛利率呈現波動：FY2021為29.0%，FY2022為26.0%，FY2023為27.4%，FY2024為26.9%，FY2025為23.0%。
    *   毛利率在2025財年和2026財年第二季度有所下降，顯示產品成本或定價策略面臨壓力。
*   **營業利益率 (Operating Margin)**：
    *   2026財年第二季度GAAP營業利益率為8.7%，較2025財年第二季度的6.9%有所擴張。非GAAP營業利益率為9.4%。
    *   過去五年GAAP營業利益率趨勢：FY2021為16.77%，FY2022為14.32%，FY2023為16.32%，FY2024為14.95%，FY2025為7.9%。
    *   2025財年營業利益率大幅下滑至7.9%，主要受重組成本、槓桿效應、對「Back to Starbucks」計畫的勞動力投資以及通膨影響。2026財年第二季度營業利益率的回升，表明公司在成本控制和銷售槓桿方面取得初步成效。
*   **純益率 (Net Margin)**：
    *   2026財年第二季度為5.4%。
    *   2025財年為5.0% (18.567億美元淨利 / 371.84億美元營收)。
    *   純益率在2025財年和2026財年第二季度較前幾年有所下降，反映了營業成本和稅務的影響。

#### 費用控制
*   **研發費用 (R&D)**：星巴克並未報告有意義的研發費用，這符合其作為咖啡零售商的商業模式。
*   **行銷費用 (S&A)**：
    *   2025財年一般及行政費用為26.172億美元，佔總營收的7.0%。
    *   2024財年為25.233億美元，佔總營收的7.0%。
    *   2023財年為24.41億美元，佔總營收的6.8% (24.41億美元 / 359.76億美元)。
    *   S&A費用佔比相對穩定，但2025財年因「Leadership Experience 2025」活動導致一般及行政費用增加9400萬美元。公司正透過「Back to Starbucks」計畫進行組織簡化和成本節約，目標在2028財年前實現20億美元的總成本節約。

### 2. 資產負債表分析 (Balance Sheet Analysis)

#### 資產品質與結構
*   **現金與短期投資佔比**：
    *   截至2026年3月29日，現金及現金等價物為15.32億美元，短期投資為1.683億美元。總計17.003億美元，佔總資產的5.56% (17.003億美元 / 305.575億美元)。
    *   相較於2025年9月28日的32.2億美元現金及現金等價物，現金儲備有所下降。
*   **應收帳款 (AR) 與存貨 (Inventory) 的周轉情況**：
    *   截至2026年3月29日，應收帳款淨額為12.889億美元，存貨為21.578億美元。
    *   截至2025年3月30日，應收帳款淨額為11.547億美元，存貨為20.473億美元。
    *   應收帳款和存貨均呈現小幅增長，需要進一步觀察其周轉天數，以判斷是否存在堆積或回收過慢的風險。目前數據不足以計算精確周轉天數，但從絕對值來看，增長幅度與營收增長大致相符。

#### 負債與償債能力
*   **流動性**：
    *   **流動比率 (Current Ratio)**：
        *   2026財年第二季度：總流動資產105.592億美元 / 總流動負債114.494億美元 = 0.92。
        *   2025財年第二季度：總流動資產67.137億美元 / 總流動負債104.293億美元 = 0.64。
        *   流動比率低於1，表明流動資產不足以完全覆蓋流動負債，流動性存在一定壓力。然而，對於像星巴克這樣擁有穩定現金流的零售企業，低於1的流動比率並不一定代表重大風險，但仍需密切關注。
    *   **速動比率 (Quick Ratio)**：
        *   2026財年第二季度：(現金及現金等價物15.32億美元 + 短期投資1.683億美元 + 應收帳款12.889億美元) / 總流動負債114.494億美元 = 0.26。
        *   速動比率遠低於1，顯示公司對存貨的依賴較高，若存貨變現能力下降，可能影響短期償債能力。
*   **長期風險**：
    *   **債務佔資產比 (Total Debt/Assets)**：
        *   2026財年第二季度：總債務 (長期債務130.842億美元 + 流動負債中的長期債務部分19.977億美元) 150.819億美元 / 總資產305.575億美元 = 0.49。
        *   債務佔資產比約為49%，顯示公司資產中約一半由債務融資。
    *   **債務對權益比 (Debt/Equity)**：
        *   由於星巴克在2026財年第二季度和2025財年第二季度均呈現股東權益赤字 (負值)，債務對權益比率不具備傳統意義上的參考價值。
    *   **債務結構**：截至2026年3月29日，長期債務為130.842億美元，流動負債中的長期債務部分為19.977億美元。長期債務佔比較高，顯示公司主要依賴長期融資。
*   **淨現金 (Net Cash) 狀況**：
    *   2026財年第二季度：現金及現金等價物15.32億美元 - 總債務150.819億美元 = -135.499億美元。星巴克處於淨負債狀態。

#### 無形資產與商譽
*   **商譽 (Goodwill)**：
    *   截至2026年3月29日，商譽為12.951億美元。
    *   相較於2025年3月30日的33.247億美元和2025年9月28日的33.69億美元，商譽大幅減少。這主要是由於公司將中國星巴克零售業務的資產歸類為「待售資產」所致，導致相關商譽和折舊攤銷成本降低。這表明公司進行了資產重組，並可能涉及商譽減損。

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

#### 營業現金流 (OCF)
*   **營運產生的現金是否與淨利同步增長？**
    *   2026財年第二季度營業現金流為3.645億美元。同期淨利為5.109億美元。營業現金流低於淨利。
    *   過去五年年度營業現金流趨勢 (單位：百萬美元)：
        *   FY2021: $5,989
        *   FY2022: $4,397
        *   FY2023: $6,008.7
        *   FY2024: $6,095.6
        *   FY2025: $4,747.5
    *   2025財年營業現金流較2024財年有所下降，但仍保持在較高水平。
*   **盈餘品質**：2026財年第二季度營業現金流低於淨利，這可能受到營運資本變動的影響。例如，指出2026財年第二季度營業現金流大幅減弱至3.645億美元，主要由於12.8億美元的營運資本大幅負向變動。這表明部分淨利潤可能尚未轉化為實際現金，盈餘品質短期內存在一定壓力。

#### 自由現金流 (FCF)
*   **計算並分析 FCF (OCF - CapEx)**：
    *   2026財年第二季度自由現金流為9180萬美元 (營業現金流3.645億美元 - 資本支出2.727億美元)。
    *   過去五年年度自由現金流趨勢 (單位：百萬美元)：
        *   FY2021: $4,519
        *   FY2022: $2,556
        *   FY2023: $3,785
        *   FY2024: $3,317.6
        *   FY2025: $2,441.5
    *   自由現金流在2025財年和2026財年第二季度呈現下降趨勢。2026財年第二季度的自由現金流僅為9180萬美元，顯示公司當前用於擴張、派息或回購的現金較為緊張。
*   **公司是否有足夠的現金用於擴張、派息或回購？**
    *   2025財年，股息支付消耗了28億美元，而自由現金流為24.4億美元。這表明現金分配超出了內部產生的自由現金流。雖然公司已暫停股票回購，但高額的股息支付在自由現金流下降的情況下，可能會對公司的財務靈活性造成壓力。

#### 資本配置 (Capital Allocation)
*   **過去幾年資金主要投向哪裡？**
    *   **資本支出 (CapEx)**：過去幾年資本支出持續較高，FY2025為23.06億美元，FY2024為27.78億美元，FY2023為23.34億美元。這反映了公司在門店擴張和基礎設施方面的持續投資。
    *   **發放股息**：星巴克有連續64個季度派發股息的歷史，並承諾持續為股東創造價值。2026財年第二季度宣布派發每股0.62美元的現金股息。
    *   **股份回購**：2025財年第四季度，星巴克在股票回購上花費為0，顯示公司已暫停回購以保留現金。
    *   **償債**：公司持續管理其債務，但淨負債狀況表明債務負擔較重。
*   **資本配置的效率如何？**
    *   在自由現金流下降的情況下，高額的股息支付和持續的資本支出，使得資本配置的效率面臨挑戰。公司需要平衡成長投資與股東回報，尤其是在盈利能力和現金流面臨壓力的時期。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

#### 回報率指標
*   **ROE (股東權益報酬率)**：
    *   由於星巴克目前股東權益為負值 (股東權益赤字)，傳統的ROE計算結果會失真或無意義。這表明公司的負債遠超資產，股東權益為負。在這種情況下，應更關注ROA和ROIC等指標。
*   **ROIC (投入資本回報率)**：
    *   提供的ROIC為0.1184 (11.84%)。這表示公司每投入1美元資本，能產生約0.1184美元的稅後營業利潤。
    *   要判斷ROIC是否超過WACC，需要WACC數據。若ROIC高於WACC，則表示公司創造了股東價值。考慮到星巴克較高的債務水平，其WACC可能相對較高。11.84%的ROIC是一個不錯的回報，但需要與行業平均和WACC進行比較。

#### 效率指標
*   **資產周轉率 (Asset Turnover)**：
    *   2025財年：營收371.84億美元 / 總資產320.2億美元 = 1.16次。
    *   資產周轉率顯示公司利用資產產生營收的效率。1.16次表明資產利用效率尚可。
*   **存貨周轉天數 (DIO)**：
    *   2026財年第二季度存貨21.578億美元。假設季度銷貨成本為營收的 (1-毛利率) = 95億美元 * (1-0.2008) = 75.924億美元。
    *   季度存貨周轉率 = 銷貨成本 / 存貨 = 75.924億美元 / 21.578億美元 = 3.52次。
    *   存貨周轉天數 = 90天 / 3.52 = 25.6天。
    *   這是一個相對健康的水平，表明存貨管理效率較高，沒有明顯的存貨堆積風險。
*   **應收帳款周轉天數 (DSO)**：
    *   2026財年第二季度應收帳款12.889億美元。假設季度營收為95億美元。
    *   季度應收帳款周轉率 = 營收 / 應收帳款 = 95億美元 / 12.889億美元 = 7.37次。
    *   應收帳款周轉天數 = 90天 / 7.37 = 12.2天。
    *   應收帳款周轉天數較短，表明公司應收帳款回收效率高，壞帳風險較低。

**最終結論：**
1.  **整體評級**：**存在隱憂**
2.  **核心摘要**：星巴克在2026財年第二季度展現了營收和營業利益率的回升，顯示「Back to Starbucks」轉型計畫初見成效。然而，公司仍面臨較高的債務負擔和負股東權益的挑戰，流動性指標偏弱。自由現金流的下降以及股息支付超出自由現金流的情況，對其財務靈活性構成壓力。儘管營運效率良好，但整體財務結構仍需加強。
3.  **關鍵監測指標**：
    *   **自由現金流利潤率 (Free Cash Flow Margin)**：監測公司將營收轉化為自由現金的能力，以評估其內生增長和股東回報的可持續性。
    *   **淨債務/EBITDA (Net Debt/EBITDA)**：評估公司償還債務的能力，尤其是在其淨負債和負股東權益的背景下。
    *   **同店銷售額增長 (Comparable Store Sales Growth)**：作為核心營收驅動力，持續監測其增長趨勢，特別是交易量和平均客單價的構成，以評估「Back to Starbucks」計畫的長期有效性。
