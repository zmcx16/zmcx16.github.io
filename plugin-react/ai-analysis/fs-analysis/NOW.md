作為一位資深美股分析師，我將對ServiceNow (NOW) 的財務狀況進行全面拆解。

### 1. 損益表分析 (Income Statement Analysis)

-   **營收成長動能**：
    ServiceNow展現出強勁且持續的營收成長動能。根據最新數據，2024年第一季總營收達到26.03億美元，年增長24%。其中，訂閱營收為25.23億美元，年增長25%。截至2026年3月31日的過去十二個月(TTM)營收為140億美元，年增長21.72%。從過去五年的趨勢來看，ServiceNow的年營收從2021年的58.96億美元穩步增長至2025年的132.78億美元，複合年增長率(CAGR)達24.1%。這種成長主要來自於其數位工作流程解決方案的強勁需求、對AI的策略性投資，以及不斷擴大的客戶基礎。截至2024年第一季，擁有超過100萬美元年度合約價值(ACV)的客戶數量增長了15%，達到1,933家。此外，公司擁有高達98%的客戶續約率，顯示其產品黏性極高。其Now Assist AI產品在2025年ACV已超過6億美元，並有望在2026年突破10億美元。

-   **獲利能力與利潤率**：
    ServiceNow的獲利能力表現穩健。截至2026年3月31日的TTM毛利率為76.6%，而2024年第一季的訂閱毛利率更高達83%，這得益於其可擴展的雲端服務模式和高效的基礎設施管理。營業利益率方面，截至2026年3月31日的TTM為13.74%，並呈現持續上升趨勢，從2022年3月的3.95%增長至2026年3月的13.44%，顯示公司在營運效率和費用控制方面有所改善，並存在規模經濟效應。淨利率方面，截至2026年3月31日的TTM為12.59%。雖然淨利率在2024年3月曾達到20.34%的峰值，隨後穩定在12%至14%的區間，但整體而言，公司的獲利能力保持在健康水平。

-   **費用控制**：
    ServiceNow的費用結構反映了其處於成長階段的策略。2024財年，銷售與行銷費用佔營收的35.1%，研發費用佔營收的23.2%，而一般及行政費用佔營收的8.5%。研發費用持續增長，截至2026年3月31日的TTM為30.8億美元，年增16.67%，這表明公司持續投入大量資源於平台技術和AI創新，以維持競爭優勢和推動未來成長。銷售與行銷費用的高佔比則是用於積極的客戶獲取和留存策略。

### 2. 資產負債表分析 (Balance Sheet Analysis)

-   **資產品質與結構**：
    ServiceNow的資產結構以現金和短期投資為主，顯示良好的流動性。截至2024年第一季，公司持有20.56億美元的現金及約當現金。作為一家軟體服務公司，ServiceNow幾乎沒有存貨風險。應收帳款周轉情況雖未直接提供，但其高達84.5億美元的當前剩餘履約義務(cRPO)（截至2024年第一季，年增21%）表明其合約健康且未來營收可預期，間接反映了應收帳款的良好管理。

-   **負債與償債能力**：
    -   **流動性**：ServiceNow的流動比率(Current Ratio)和速動比率(Quick Ratio)相對較低。截至2026年5月TTM的流動比率為0.85，而截至2026年6月的速動比率為0.84。這些比率低於1，通常會被視為潛在風險。然而，對於SaaS公司而言，由於其商業模式涉及大量的遞延收入（預收客戶款項，計入流動負債），這會導致流動比率和速動比率在會計上顯得較低，但實際上並不代表償債能力不足。遞延收入是ServiceNow流動負債的重要組成部分。
    -   **長期風險**：公司的債務水平非常健康。截至2026年6月，債務對權益比(Debt/Equity)為0.13，遠低於0.21的提供數據，顯示公司對債務融資的依賴程度極低。長期債務對權益比為0.2。此外，財務槓桿從2021年的2.92持續下降至2025年的2.01，進一步印證了公司去槓桿化的趨勢。
    -   **淨現金狀況**：憑藉20.56億美元的現金及約當現金和極低的債務水平，ServiceNow擁有穩健的淨現金狀況。

-   **無形資產與商譽**：

### 3. 現金流量表分析 (Cash Flow Statement Analysis)

-   **營業現金流 (OCF)**：
    ServiceNow的營運現金流表現強勁。雖然未直接提供單一季度OCF數據，但其強勁的自由現金流(FCF)表現間接證明了健康的營運現金流。對於SaaS公司而言，營運現金流通常會高於淨利，因為遞延收入的會計處理。

-   **自由現金流 (FCF)**：
    ServiceNow展現出卓越的自由現金流生成能力。截至2026年3月31日的TTM自由現金流高達46.24億美元。每股自由現金流在2025年達到4.37美元，年增長33.4%，三年複合年增長率為27.0%。過去十二個月的平均每股自由現金流增長率為27.60%。以TTM營收140億美元和TTM FCF 46.24億美元計算，自由現金流利潤率約為33%，這是一個非常高的水平，表明公司擁有充裕的現金用於業務擴張、研發投入以及股東回報。

-   **資本配置 (Capital Allocation)**：
    ServiceNow的資本配置策略主要集中於支持其成長。公司將大量資金投入研發（用於平台和AI創新）和銷售與行銷（用於客戶獲取和留存）。此外，公司也透過股票回購計劃來管理股權稀釋，例如在2024年第一季回購了價值1.75億美元的股票。ServiceNow內部設有「投資資金」(Investment Funding)應用程式，用於規劃和管理對業務部門、產品和團隊的投資，並根據業務需求和策略目標進行優先排序，顯示其資本配置具備結構化和策略性。

### 4. 核心財務指標與杜邦分析 (Key Ratios & DuPont Analysis)

-   **回報率指標**：
    -   **ROE (股東權益報酬率)**：截至2026年3月31日的TTM ROE為14.98%。杜邦分析顯示，ROE在2023年曾顯著增長至22.69%，主要受淨利率改善的驅動，儘管資產周轉率和財務槓桿有所下降。隨後，ROE在2024年和2025年有所回落，分別為14.83%和13.48%，但仍保持在健康水平。
    -   **ROIC (投入資本回報率)**：截至2026年3月31日的ROIC為8.04%。ROIC在過去三年從4.3%提升至8.4%，並在2024年達到23.07%的峰值，隨後在2025年下降至19.66%。雖然目前的ROIC略低於其三年中位數10.3%，但整體而言，公司在利用投入資本產生利潤方面仍具備效率。ROIC的變化受營業利益率和資本周轉率的共同影響。

-   **效率指標**：
    -   **資產周轉率**：資產周轉率呈現輕微下降趨勢，從2021年的0.55降至2025年的0.51，這表明資產利用效率略有降低。資本周轉率在2025年也出現下降。
    -   **存貨周轉天數 (DIO)**：作為軟體公司，ServiceNow幾乎沒有存貨，因此此指標不適用。
    -   **應收帳款周轉天數 (DSO)**：雖然未直接提供DSO數據，但強勁的cRPO增長和高續約率表明公司在應收帳款管理方面應是高效的。
