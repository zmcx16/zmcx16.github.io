# ZIM 投資評估：決策樹分析與期望值分析

## 核心假設

1.  **市場動態**：貨櫃航運業高度週期性，受全球貿易量、地緣政治事件（如紅海危機）及供需平衡影響。
2.  **紅海危機**：紅海航運中斷的持續時間和強度至關重要。危機延長有利於 ZIM 維持高運費並收緊運力；若恢復正常則運費可能下降。紅海危機在 2023 年底至 2024 年對 ZIM 的業績產生了顯著的正面影響。
3.  **運力過剩**：全球船隊新船交付量大增，構成顯著阻力。預計 2024 年全球貨櫃船隊運力將增長約 7.09% 至 9.5%，而需求增長約 3.7% 至 4.5%，可能導致運力過剩並對運費造成下行壓力。
4.  **ZIM 營運策略**：ZIM 主要採用租賃船隊模式，使其對即期運費高度敏感。公司已投資於新的液化天然氣（LNG）動力船舶，以提高效率並符合環保法規，截至 2025 年 3 月，其船隊中 40% 為 LNG 動力船。
5.  **併購案**：Hapag-Lloyd 於 2026 年 2 月提出以每股 35 美元的現金價格收購 ZIM，總價約 42 億美元。此交易預計於 2026 年底前完成，但需獲得監管批准。
6.  **股息政策**：ZIM 的股息支付具有高度變動性，直接與其淨利潤掛鉤，反映了業務的週期性。其政策為支付季度淨利潤的 30% (Q1-Q3)，並在 Q4 支付「補足」股息，使全年總派息達到年度淨利潤的 30% 至 50%。

## 期望值分析 (Expected Value Analysis)

### 計算過程

1.  **情境 1: Hapag-Lloyd 併購案完成**
    *   機率 (P1) = 0.70
    *   預期報酬 (R1) = 33.08%
    *   期望值 (EV1) = P1 \* R1 = 0.70 \* 0.3308 = 0.23156

2.  **情境 2: Hapag-Lloyd 併購案失敗**
    *   機率 (P2) = 0.30
    *   此情境下需進一步計算其子情境的期望值：
        *   **子情境 2.1: 市場狀況有利**
            *   條件機率 (P2.1|P2) = 0.40
            *   總機率 (P2.1) = P2 \* P2.1|P2 = 0.30 \* 0.40 = 0.12
            *   預期報酬 (R2.1) = 40%
            *   期望值 (EV2.1) = P2.1 \* R2.1 = 0.12 \* 0.40 = 0.048
        *   **子情境 2.2: 市場狀況中性**
            *   條件機率 (P2.2|P2) = 0.40
            *   總機率 (P2.2) = P2 \* P2.2|P2 = 0.30 \* 0.40 = 0.12
            *   預期報酬 (R2.2) = -30%
            *   期望值 (EV2.2) = P2.2 \* R2.2 = 0.12 \* (-0.30) = -0.036
        *   **子情境 2.3: 市場狀況不利**
            *   條件機率 (P2.3|P2) = 0.20
            *   總機率 (P2.3) = P2 \* P2.3|P2 = 0.30 \* 0.20 = 0.06
            *   預期報酬 (R2.3) = -50%
            *   期望值 (EV2.3) = P2.3 \* R2.3 = 0.06 \* (-0.50) = -0.030
    *   **情境 2 的總期望值 (EV2)** = EV2.1 + EV2.2 + EV2.3 = 0.048 - 0.036 - 0.030 = -0.018

3.  **整體期望值 (Overall Expected Value)**
    *   整體期望值 = EV1 + EV2 = 0.23156 + (-0.018) = 0.21356

    因此，投資 ZIM 的整體期望值為 **21.36%**。

## 最終結論

根據決策樹分析和期望值計算，投資 ZIM 股票的整體期望值為 **21.36%**。

**適合投資。**

**理由：**

儘管貨櫃航運業具有高度週期性和運力過剩的潛在風險，但 Hapag-Lloyd 以每股 35 美元的價格收購 ZIM 的要約，為投資者提供了一個明確且具吸引力的上行空間（約 33.08% 的潛在報酬）。由於該併購案預計將於 2026 年底前完成，且我們賦予其 70% 的較高成功機率，這成為了整體期望值的主要驅動力。

即使併購案失敗（30% 機率），ZIM 的基本面在紅海危機持續的情況下仍可能表現良好，如 2024 年的強勁復甦所示，當年淨利潤達 21.5 億美元，股息總額為每股 7.98 美元。雖然市場狀況轉差可能導致顯著虧損，但併購案的潛在收益足以抵消這些風險，使整體投資呈現正向且可觀的期望報酬。因此，在當前股價下，ZIM 股票適合投資，主要基於併購案帶來的確定性溢價。
