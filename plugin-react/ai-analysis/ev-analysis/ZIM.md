    *   **全球經濟成長 (Global Economic Growth):** 假設全球經濟將保持溫和成長，並受惠於預期的降息，對貨櫃運輸需求提供一定支撐。
    *   **供給過剩 (Oversupply):** 假設大量新造船隻的交付將在2024年下半年至2026年對全球貨櫃航運市場造成顯著的供給壓力，長期可能導致運費承壓。

2.  **財務假設 (Financial Assumptions):**
    *   **ZIM 獲利能力 (ZIM Profitability):** 假設 ZIM 在紅海危機的短期利好下，2024年將維持獲利，並可能支付可觀股息。ZIM 在2024年第一季度實現淨利潤9200萬美元，並上調了全年業績指引。然而，分析師預計2026年和2027年可能面臨負每股盈餘 (EPS)，這將影響未來的股息支付能力。
    *   **股息政策 (Dividend Policy):** 假設 ZIM 將繼續遵循其股息政策（季度淨利的30%，全年最高50%），但未來股息金額將高度依賴其淨利表現，且存在不確定性。
    *   **資產負債表 (Balance Sheet):** 假設 ZIM 透過靈活的租船策略，能夠在市場變化時調整運力，以應對供給過剩的挑戰。

3.  **產業趨勢假設 (Industry Trend Assumptions):**
    *   **運費波動 (Freight Rate Volatility):** 貨櫃航運業將繼續高度週期性，運費受地緣政治、供需平衡和全球貿易量影響劇烈。
    *   **環保法規 (Environmental Regulations):** 新的環保法規可能增加營運成本，但 ZIM 的新造船隊（包括 LNG 動力船）有助於提升效率和符合法規.
    *   **併購活動 (M&A Activity):** ZIM 目前面臨的收購戰（Hapag-Lloyd 與 Sakal Group）可能為股東帶來溢價，但也可能改變公司的長期策略和資本回報政策。

### 計算過程 (Calculation Process)

1.  **情境機率計算：** 各市場情境的機率 (P) 相加為 1 (0.40 + 0.35 + 0.25 = 1)。每個 ZIM 表現情境的機率是其所屬市場情境機率與自身機率的乘積。

2.  **預期總報酬計算：**

    *   **預期總報酬 = 資本利得/損失 + 股息收益。**
    *   例如：在「運費持續高檔」且「ZIM 表現強勁」情境下：

3.  **期望值計算：** 每個最終節點的期望值為該情境的機率乘以該情境下的預期總報酬。

儘管計算出的整體期望值為正，但 +2.78% 的預期報酬對於 ZIM 這種高度波動且風險較高的股票而言，顯得非常微薄。
