### 3. 明確列出所有計算過程

我們使用反向歸納法（Backward Induction）來計算期望值。

**步驟1：計算各「鐵礦石價格情境」下的 VALE 營運期望值 (第二層機會節點)**

*   **情境1: 鐵礦石價格高漲 (機率: 25%)**
    *   預期報酬 (EV_1) = (營運優異報酬 \* 機率) + (營運持平報酬 \* 機率) + (營運不佳報酬 \* 機率)
    *   EV_1 = (40% \* +40%) + (40% \* +25%) + (20% \* +10%)
    *   EV_1 = 16.0% + 10.0% + 2.0%
    *   **EV_1 = +28.00%**

*   **情境2: 鐵礦石價格持平 (機率: 50%)**
    *   預期報酬 (EV_2) = (營運優異報酬 \* 機率) + (營運持平報酬 \* 機率) + (營運不佳報酬 \* 機率)
    *   EV_2 = (30% \* +15%) + (50% \* +5%) + (20% \* -10%)
    *   EV_2 = 4.5% + 2.5% - 2.0%
    *   **EV_2 = +5.00%**

*   **情境3: 鐵礦石價格下跌 (機率: 25%)**
    *   預期報酬 (EV_3) = (營運優異報酬 \* 機率) + (營運持平報酬 \* 機率) + (營運不佳報酬 \* 機率)
    *   EV_3 = (10% \* -5%) + (40% \* -20%) + (50% \* -40%)
    *   EV_3 = -0.5% - 8.0% - 20.0%
    *   **EV_3 = -28.50%**

**步驟2：計算投資 VALE 的總體期望值 (第一層機會節點)**

*   總體期望值 (Overall EV) = (鐵礦石高漲機率 \* EV_1) + (鐵礦石持平機率 \* EV_2) + (鐵礦石下跌機率 \* EV_3)
*   Overall EV = (25% \* +28.00%) + (50% \* +5.00%) + (25% \* -28.50%)
*   Overall EV = 7.00% + 2.50% - 7.125%
*   **Overall EV = +2.375%**
