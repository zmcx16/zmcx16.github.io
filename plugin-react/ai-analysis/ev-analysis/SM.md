*   **EV_P1_SM_Perf** = 49% + 12% = **61%**

**1.2. 溫和成長與市場盤整情境下的 SM 表現 (EV_P2_SM_Perf)**

*   SM 符合預期表現: 60% * (+15%) = 9%
*   SM 低於預期表現: 20% * (-10%) = -2%
*   **EV_P2_SM_Perf** = 6% + 9% - 2% = **13%**

**1.3. AI 泡沫破裂或經濟衰退擔憂情境下的 SM 表現 (EV_P3_SM_Perf)**

*   SM 嚴重受挫: 70% * (-30%) = -21%

*   **EV_Invest_SM** = 21.35% + 5.85% - 4.5% = **22.7%**
