#### 2. 計算公式
$$EV = \sum (Probability_i \times Return_i)$$

*   **牛市期望貢獻**：$30\% \times 35\% = 10.5\%$
*   **基準期望貢獻**：$50\% \times 15\% = 7.5\%$
*   **熊市期望貢獻**：$20\% \times (-20\%) = -4.0\%$

#### 3. 總體期望報酬率
$$EV = 10.5\% + 7.5\% - 4.0\% = 14.0\%$$
