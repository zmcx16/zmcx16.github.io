### 公司概況與最新資訊補充

Companhia Siderúrgica Nacional (SID) 是一家巴西綜合鋼鐵生產商，業務涵蓋鋼鐵、礦業、物流、能源和水泥五大板塊。其中，礦業被視為重要的增長引擎，並有擴張計劃。公司正推動戰略計劃，包括可能出售水泥業務的控制權、剝離基礎設施的少數股權以及在鋼鐵領域尋求合作夥伴，以期降低槓桿率。

*   **淨負債:** 截至2025年12月31日，綜合淨負債為412.176億巴西雷亞爾，淨負債與過去12個月EBITDA比率為3.47倍，槓桿率仍高。
*   **現金狀況:** 截至2025年12月31日，現金及現金等價物為160億巴西雷亞爾。
*   **EBITDA (2025財年):** 鋼鐵業務調整後EBITDA同比增長36.9%至21.942億巴西雷亞爾，利潤率為10.0%。物流業務的收入和EBITDA也有所增長。全年EBITDA達到118億巴西雷亞爾，較2024年增長15%。
*   **營收 (2025財年):** 達到447.979億巴西雷亞爾，較2024年增長2.5%，主要受礦業、物流和能源業務的推動。
*   **盈利能力:** 公司目前處於虧損狀態，過去五年虧損以每年平均73.7%的速度增長。股東權益報酬率 (ROE) 為負9.57%。

*   **市淨率 (P/B):** 0.73，可能暗示基於賬面價值的低估。
*   **目標價:** 分析師目標價介於$1.40至$1.94之間。
*   **股價表現:** 股價呈下跌趨勢，接近52週低點。

**市場與產業趨勢 (巴西鋼鐵業):**
*   巴西鋼鐵市場預計在2025年至2030年間以3.2%的複合年增長率增長，到2030年達到304.362億美元。
*   主要受基礎設施、汽車和建築等國內強勁需求的推動。

**判斷：不適合投資**

**理由：**
儘管計算出的期望值略高於當前股價，且預期報酬率為正，但考慮到以下因素，目前投資SID股票仍被認為 **不適合**：
1.  **高風險與高槓桿：** SID目前處於虧損狀態，且負債權益比高達4.2，長期負債權益比為3.37，顯示公司財務結構高度槓桿化，存在顯著的財務風險。
2.  **盈利能力持續惡化：** 公司過去五年虧損持續擴大，且分析師預期明年EPS將進一步下降，顯示其核心盈利能力面臨嚴峻挑戰。
3.  **產業逆風：** 巴西鋼鐵業面臨全球產能過剩、國內產量和銷售額下降以及低價進口鋼材競爭加劇的壓力。儘管政府有保護措施，但短期內難以完全抵消這些負面影響。
4.  **分析師意見分歧且偏負面：** 雖然有部分分析師給出目標價，但整體推薦評級偏向「中性至賣出」，甚至有「強烈賣出」的建議。這反映了市場對其前景的普遍擔憂。
5.  **預期報酬率不足以彌補風險：** 8.53%的預期報酬率對於一家高負債、虧損且面臨產業逆風的公司而言，可能不足以彌補其所承擔的巨大風險。投資者應尋求在類似風險水平下具有更高潛在回報的機會。

綜上所述，儘管公司在多元化業務（如礦業）和戰略轉型方面存在潛在亮點，但其當前的財務困境、產業挑戰以及高度不確定的前景，使得SID股票在目前階段不適合投資。
