### 三、 計算過程與核心假設

#### 2. 期望值 (Expected Value) 計算
*   **期望報酬率** = $(0.30 \times 68.1\%) + (0.50 \times 29.3\%) + (0.20 \times -28.8\%)$
*   **期望報酬率** = $20.43\% + 14.65\% - 5.76\% = \mathbf{29.32\%}$
