## 美股公司 HMC 投資評估：決策樹與期望值分析

### 核心假設 (Core Assumptions)

本分析基於以下核心假設，這些假設將影響決策樹中的機率與預期報酬：

1.  **投資期限：** 1年。所有報酬率均為年化預期報酬率。
2.  **市場趨勢：** 全球經濟與汽車產業的表現是影響 HMC 銷售量和獲利能力的關鍵因素。
    *   **正向市場：** 全球經濟穩健成長，汽車需求強勁，供應鏈瓶頸大幅緩解。
    *   **穩定市場：** 全球經濟溫和成長，汽車需求持平或略有增長，供應鏈仍存在一定挑戰。
    *   **負向市場：** 全球經濟衰退或增速放緩，高通膨、高利率抑制消費需求，供應鏈問題惡化。
3.  **HMC 營運與產業趨勢：**
    *   **電動車轉型：** HMC 在電動車（EV）領域的發展速度和市場接受度將直接影響其長期競爭力。
    *   **成本控制與產品組合：** HMC 在不同市場的定價能力、成本管理以及產品組合（汽油車、混合動力車、電動車、摩托車、動力設備）的效益。
    *   **日圓匯率：** 作為一家日本公司，日圓兌美元的匯率波動會顯著影響其海外收入換算回日圓後的價值。本分析假設匯率波動已納入預期報酬考量。
4.  **競爭環境：** 假設汽車產業競爭激烈，但 HMC 憑藉品牌力、技術積累仍能維持一定市場地位。
5.  **不投資選項：** 若不投資 HMC，假設資金配置於無風險資產或僅維持現金部位，其預期報酬設定為 0% (僅計算投資 HMC 的超額報酬)。

### 期望值分析 (Expected Value Analysis) 與計算過程

我們將從決策樹的末端節點開始，逐步計算每個機會節點和決策節點的期望值。

#### 1. 終端節點期望值 (Terminal Node Expected Values)

終端期望值 = 情境發生機率 (絕對機率) × 預期報酬率

*   **正向市場展望下 HMC 營運表現：**
    *   情境 1A: 卓越營運與成功 EV 轉型
        *   絕對機率 = P(正向市場) × P(卓越營運 | 正向市場) = 30% × 60% = 18%
        *   預期報酬 = +30%
        *   **終端期望值 (1A)** = 18% × (+30%) = **+5.40%**
    *   情境 1B: 良好營運與穩健 EV 進展
        *   絕對機率 = P(正向市場) × P(良好營運 | 正向市場) = 30% × 40% = 12%
        *   預期報酬 = +15%
        *   **終端期望值 (1B)** = 12% × (+15%) = **+1.80%**

*   **穩定市場展望下 HMC 營運表現：**
    *   情境 2A: 良好營運與穩健 EV 進展
        *   絕對機率 = P(穩定市場) × P(良好營運 | 穩定市場) = 50% × 60% = 30%
        *   預期報酬 = +10%
        *   **終端期望值 (2A)** = 30% × (+10%) = **+3.00%**
    *   情境 2B: 一般營運與緩慢 EV 進展
        *   絕對機率 = P(穩定市場) × P(一般營運 | 穩定市場) = 50% × 40% = 20%
        *   預期報酬 = +2%
        *   **終端期望值 (2B)** = 20% × (+2%) = **+0.40%**

*   **負向市場展望下 HMC 營運表現：**
    *   情境 3A: 具韌性表現，市場份額穩定
        *   絕對機率 = P(負向市場) × P(具韌性 | 負向市場) = 20% × 30% = 6%
        *   預期報酬 = -5%
        *   **終端期望值 (3A)** = 6% × (-5%) = **-0.30%**
    *   情境 3B: 疲弱營運，市場份額下降
        *   絕對機率 = P(負向市場) × P(疲弱營運 | 負向市場) = 20% × 70% = 14%
        *   預期報酬 = -20%
        *   **終端期望值 (3B)** = 14% × (-20%) = **-2.80%**

*   **不投資 HMC：**
    *   情境 4A: 資金配置於無風險資產或其他
        *   絕對機率 = 100%
        *   預期報酬 = 0%
        *   **終端期望值 (4A)** = 100% × (0%) = **0.00%**

#### 2. 中間節點期望值 (Intermediate Node Expected Values)

*   **期望值 (HMC 營運與 EV 轉型表現 | 正向市場)**
    = (60% × +30%) + (40% × +15%)
    = 18.00% + 6.00% = **+24.00%**

*   **期望值 (HMC 營運與 EV 轉型表現 | 穩定市場)**
    = (60% × +10%) + (40% × +2%)
    = 6.00% + 0.80% = **+6.80%**

*   **期望值 (HMC 營運與 EV 轉型表現 | 負向市場)**
    = (30% × -5%) + (70% × -20%)
    = -1.50% - 14.00% = **-15.50%**

#### 3. 「投資 HMC」選項的總期望值

期望值 (投資 HMC)
= (P(正向市場) × 期望值(HMC 營運表現 | 正向市場))
  + (P(穩定市場) × 期望值(HMC 營運表現 | 穩定市場))
  + (P(負向市場) × 期望值(HMC 營運表現 | 負向市場))

= (30% × +24.00%) + (50% × +6.80%) + (20% × -15.50%)
= 7.20% + 3.40% - 3.10%
= **+7.50%**

#### 4. 「不投資 HMC」選項的總期望值

期望值 (不投資 HMC) = **0.00%** (根據假設，視為基準報酬)

### 最終結論 (Final Conclusion)

根據上述決策樹與期望值分析，對比兩個主要選項：

*   **投資 HMC 的整體期望值 = +7.50%**
*   **不投資 HMC 的整體期望值 = 0.00%**

由於「投資 HMC」的整體期望值為 **正數 (+7.50%)**，且高於「不投資 HMC」的期望值，因此根據本模型和所作假設，**目前適合投資 HMC**。

**簡短理由：**
HMC 在全球汽車市場中，即使在穩定或部分負向情境下，仍有機會展現其營運韌性與穩健的預期報酬。儘管存在市場下行及公司 EV 轉型進度不如預期的風險，但綜合各情境及其機率分佈，預期投資 HMC 在未來一年內能帶來正向的超額收益。然而，投資者仍需留意分析中的假設，尤其是市場情境與 HMC 營運表現的機率和報酬估計，這些都可能隨實際情況變化而調整。
