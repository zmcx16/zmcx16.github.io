### 期望值計算過程

**整體期望值 (EV) = (樂觀情境報酬 * 樂觀情境機率) + (中性情境報酬 * 中性情境機率) + (悲觀情境報酬 * 悲觀情境機率)**

*   **樂觀情境期望值：** $74.74 * 0.30 = $22.422
*   **中性情境期望值：** $64.96 * 0.50 = $32.480
*   **悲觀情境期望值：** $50.04 * 0.20 = $10.008

**整體期望值 (EV) = $22.422 + $32.480 + $10.008 = $64.91**
