# 決策樹分析：評估美股公司 NVO 投資機會

## 核心假設

*   **市場趨勢：** 全球糖尿病與肥胖症藥物市場持續增長，GLP-1 藥物需求旺盛。然而，競爭日益激烈，特別是來自 Eli Lilly 的新產品，可能導致市場份額和定價壓力。
*   **財務狀況：** NVO 擁有強勁的歷史盈利能力和資產負債表，但 2026 年面臨銷售增長放緩和定價壓力，以及研發支出增加和債務上升的挑戰。
*   **產業趨勢：** 藥品定價壓力、監管審批、新藥研發成功率以及專利到期是關鍵因素。遠程醫療合作夥伴關係和新劑型（口服 Wegovy）是潛在增長點，有助於擴大市場准入。

## 期望值計算

**投資 NVO 的總期望值 (Expected Value of Investing in NVO):**
= (情境 1 節點期望值) + (情境 2 節點期望值) + (情境 3 節點期望值)
= 0.24627 + 0.15201 + (-0.02555)
= **0.37273 (或 37.27%)**

## 最終結論

根據決策樹分析和期望值計算，投資 NVO 的總期望值為 **37.27%**。

**判斷：適合投資**

**理由：**
儘管 Novo Nordisk 面臨來自 Eli Lilly 等競爭對手的激烈競爭、定價壓力以及 2026 年銷售增長放緩的預期，但其在糖尿病和肥胖症治療領域的領導地位依然穩固，擁有超過 45% 的全球胰島素市場和約 63-69% 的 GLP-1 市場份額。公司核心產品 Ozempic 和 Wegovy 的需求持續強勁，且正在積極擴展市場（例如透過遠程醫療訂閱服務和口服劑型）。

此外，NVO 目前的估值顯著低於其歷史平均水平和行業平均水平，P/E 比率僅為 10.39，遠低於其五年平均的 25-30 倍，顯示其被市場嚴重低估。公司擁有強勁的盈利能力（ROE 0.6125, ROA 0.2059, ROI 0.3138, Profit Margin 0.3303）和健康的股息收益率（4.79%）。分析師的共識目標價也顯示出可觀的潛在上漲空間。

綜合考量，儘管存在風險，但 NVO 目前的低估值、強勁的產品組合、市場領導地位以及未來增長潛力，使得其在風險調整後的期望報酬為正且具吸引力，因此適合投資。
