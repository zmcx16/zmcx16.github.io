## 核心假設 (Core Assumptions)

在進行決策樹分析之前，我們需要建立一些關於市場、財務和產業趨勢的核心假設。這些假設將影響情境機率和預期報酬的設定。

### 1. 市場假設 (Market Assumptions)
*   **GLP-1 藥物需求持續強勁**：全球對糖尿病和肥胖症治療的需求將持續增長，尤其是在體重管理方面。GLP-1 類藥物（如 Ozempic 和 Wegovy）將在未來數年內維持其市場領導地位。
*   **美國及主要國際市場的支付能力**：儘管存在價格壓力，但主要的醫療保險和政府支付體系仍將為 GLP-1 藥物提供一定程度的覆蓋。

### 2. 財務假設 (Financial Assumptions)
*   **NVO 營收成長**：受 GLP-1 藥物銷量和潛在的新產品帶動，NVO 的營收將保持高增長。
*   **毛利率和營運效率**：NVO 預計能維持其高毛利率，並透過規模經濟和生產效率提升，有效管理營運費用。
*   **研發投入**：NVO 將持續投入大量資金於研發，以擴展現有產品線（如口服 GLP-1 藥物）和開發新的治療領域。

### 3. 產業趨勢假設 (Industry Trend Assumptions)
*   **競爭格局**：禮來 (Eli Lilly) 及其 Mounjaro/Zepbound 將是 NVO 的主要競爭對手。此外，其他製藥公司也將陸續進入 GLP-1 市場或開發替代療法。
*   **監管環境**：主要市場（如美國 FDA 和歐洲 EMA）的藥品審批流程將保持相對穩定，但對藥品定價和市場壟斷的審查可能加劇。
*   **供應鏈穩定性**：NVO 能夠逐步解決其 GLP-1 藥物的供應限制，以滿足日益增長的需求。
*   **潛在風險**：儘管機率較低，但長期的不良副作用報告、嚴格的藥價管制或專利懸崖仍是產業面臨的潛在風險。

## 計算過程 (Calculation Process)

我們將從決策樹的末端節點開始，計算每個分支的期望值，然後回溯到初始決策點。

### 1. 計算次情境的期望值 (Expected Value for Sub-Scenarios)

這些是葉子節點的預期報酬，已在圖中標示。

*   **次情境1.1 (持續高需求與有效管線擴張):** +50%
*   **次情境1.2 (強勁需求但面臨新興競爭壓力):** +30%
*   **次情境2.1 (穩定市場份額與持續創新):** +20%
*   **次情境2.2 (競爭加劇與價格審查):** +10%
*   **次情境3.1 (重大競爭侵蝕與市場飽和):** -10%
*   **次情境3.2 (重大不良事件 / 嚴格監管打擊):** -40%

### 2. 計算主要情境節點的期望值 (Expected Value for Main Scenario Nodes)

這是將各次情境的機率乘以其預期報酬的總和。

*   **情境1: 強勁的 GLP-1 市場主導地位與成長 (P=0.40)**
    *   節點期望值 = (次情境1.1機率 × 預期報酬) + (次情境1.2機率 × 預期報酬)
    *   節點期望值 = (0.60 × 50%) + (0.40 × 30%)
    *   節點期望值 = 30% + 12% = **42.0%**

*   **情境2: 適度的 GLP-1 市場成長 (P=0.45)**
    *   節點期望值 = (次情境2.1機率 × 預期報酬) + (次情境2.2機率 × 預期報酬)
    *   節點期望值 = (0.70 × 20%) + (0.30 × 10%)
    *   節點期望值 = 14% + 3% = **17.0%**

*   **情境3: GLP-1 市場成長放緩 / 負面事件 (P=0.15)**
    *   節點期望值 = (次情境3.1機率 × 預期報酬) + (次情境3.2機率 × 預期報酬)
    *   節點期望值 = (0.60 × -10%) + (0.40 × -40%)
    *   節點期望值 = -6% + (-16%) = **-22.0%**

### 3. 計算整體投資的期望值 (Overall Expected Value for "Invest in NVO")

這是將各主要情境的機率乘以其節點期望值的總和。

*   整體期望值 = (情境1機率 × 情境1節點期望值) + (情境2機率 × 情境2節點期望值) + (情境3機率 × 情境3節點期望值)
*   整體期望值 = (0.40 × 42.0%) + (0.45 × 17.0%) + (0.15 × -22.0%)
*   整體期望值 = 16.8% + 7.65% - 3.3%
*   整體期望值 = **21.15%**
