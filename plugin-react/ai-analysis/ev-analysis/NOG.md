### 三、 期望值分析 (Expected Value Analysis)

*   **牛市期望值**：$0.25 \times 30\% = 7.5\%$

**總期望報酬率 (Total EV)** = $7.5\% + 6.6\% - 5.0\% = \mathbf{9.1\%}$
