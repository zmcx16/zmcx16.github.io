### 三、 期望值分析 (Expected Value Analysis)

#### 1. 計算過程
我們以目前股價 **$80.00** 為基準，計算未來一年的預期報酬率期望值：

*   **牛市期望值**：$31.25\% \times 0.30 = 9.375\%$

**總體期望報酬率 (Total EV)** = $9.375\% + 6.25\% - 3.00\% = \mathbf{12.625\%}$
