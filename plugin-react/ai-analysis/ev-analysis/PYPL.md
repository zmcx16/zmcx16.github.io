**整體期望報酬率 = 12.4575% + 6.1215% - 4.406% = 14.173%**

**預期未來股價 = 目前股價 * (1 + 整體期望報酬率)**
預期未來股價 = $45.24 * (1 + 0.14173) = $45.24 * 1.14173 = **$51.65**
