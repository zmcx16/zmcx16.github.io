#### 2. 明確列出所有計算過程

**起始投資：$130/股**

**A. 計算終端節點的最終價值：**
每個終端節點的最終價值 = 起始股價 * (1 + 預期報酬率)

*   **情境1.1 (強勁復甦 & 營運強勁):** $130 * (1 + 0.50) = $195
*   **情境1.2 (強勁復甦 & 營運平庸):** $130 * (1 + 0.30) = $169
*   **情境1.3 (強勁復甦 & 營運不佳):** $130 * (1 + 0.15) = $149.5
*   **情境2.1 (溫和成長 & 營運強勁):** $130 * (1 + 0.20) = $156
*   **情境2.2 (溫和成長 & 營運平庸):** $130 * (1 + 0.05) = $136.5
*   **情境2.3 (溫和成長 & 營運不佳):** $130 * (1 - 0.10) = $117
*   **情境3.1 (經濟放緩 & 營運強勁):** $130 * (1 - 0.05) = $123.5
*   **情境3.2 (經濟放緩 & 營運平庸):** $130 * (1 - 0.20) = $104
*   **情境3.3 (經濟放緩 & 營運不佳):** $130 * (1 - 0.40) = $78

**B. 計算第二層機率節點的期望值 (DG營運表現的期望值)：**
節點期望值 = Σ (分支機率 * 該分支最終價值)

*   **子節點 EV (強勁經濟復甦):**
    *   (0.60 * $195) + (0.30 * $169) + (0.10 * $149.5)
    *   = $117 + $50.7 + $14.95 = **$182.65**

*   **子節點 EV (溫和經濟成長):**
    *   (0.30 * $156) + (0.50 * $136.5) + (0.20 * $117)
    *   = $46.8 + $68.25 + $23.4 = **$138.45**

*   **子節點 EV (經濟放緩):**

**C. 計算投資DG的總期望值：**
總期望值 = Σ (主要情境機率 * 該情境節點期望值)

*   **總 EV (投資DG):**
    *   (0.20 * $182.65) + (0.50 * $138.45) + (0.30 * $92.95)

### 3. 最終結論

根據上述計算，投資DG的整體期望值為 **$133.64**。考慮到起始投資為 $130，這意味著在未來12個月，每股的預期收益為 $133.64 - $130 = $3.64。

預期報酬率 = ($3.64 / $130) * 100% = **約 2.8%**

**判斷：不適合投資**

**簡短理由：**
儘管投資DG的期望值略高於其當前股價，預期報酬率約為2.8%。然而，對於一項股票投資而言，這是一個相對較低的預期報酬。與市場平均報酬（例如，歷史上S&P 500年化報酬率約7-10%）或當前無風險利率（例如，美國公債殖利率可能更高）相比，2.8%的預期報酬並不足以彌補投資DG所涉及的固有市場風險和公司特定風險。在風險調整後，這一投資選項的吸引力不足。因此，根據此分析，DG目前不適合投資。
