#### 2. 計算過程
將各情境的「機率」乘以「預期報酬」：

*   **樂觀情境 EV** = $25\% \times 45\% = 11.25\%$

**總體期望報酬率 (Total Expected Return)**：
$$11.25\% + 10.5\% - 3.75\% = \mathbf{18.0\%}$$
