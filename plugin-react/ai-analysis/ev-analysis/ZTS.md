### 3. 期望值計算 (Expected Value Calculation)

我們根據上述情境計算一年後的**預期股價期望值 (EV)**：

$$EV = (P_{Bull} \times V_{Bull}) + (P_{Base} \times V_{Base}) + (P_{Bear} \times V_{Bear})$$

*   **計算過程：**
    *   樂觀：$0.30 \times 170 = 51.00$
    *   基準：$0.50 \times 151 = 75.50$
    *   悲觀：$0.20 \times 110 = 22.00$
*   **總期望值 (Expected Price)：** $51.00 + 75.50 + 22.00 = \mathbf{\$148.50}$

**預期報酬率分析：**
*   現價：$128.66
*   期望值：$148.50
*   **預期總報酬率：** $\frac{148.50 - 128.66}{128.66} \approx \mathbf{15.42\%}$ (不含股息)
*   **含息總報酬 (Dividend 1.58%)：** $\approx \mathbf{17.0\%}$

### 4. 綜合評估與最終結論

#### 基本面數據支持點：
*   **極強的獲利能力：** ROE 65.99% 與 Operating Margin 38% 顯示其在動物藥品市場擁有極強的護城河與定價權。
*   **估值吸引力：** Forward P/E 17.04x 對於一家長期增長接近雙位數的龍頭公司來說，處於歷史低位區間。
*   **財務穩健：** Current Ratio 3.03 顯示短期償債能力極佳，儘管 Debt/Eq 較高，但其穩定的現金流（P/FCF 23.79）足以支撐債務。

#### 風險提示：
*   **技術面壓力：** SMA200 為 -9.44%，顯示長期趨勢尚未完全反轉，短期內可能仍有震盪。
*   **情緒面：** 內部人交易 (Insider Trans -2.43%) 與機構交易 (Inst Trans -1.71%) 略微偏向賣方，顯示大戶仍在觀望安全性爭議的後續。

#### **最終結論：適合投資 (Buy / Overweight)**

**理由：**
1.  **期望值為正：** 經過風險加權後的期望報酬率高達 **17%**，遠高於無風險利率與標普 500 的平均預期回報。
2.  **安全邊際：** 目前股價 ($128.66) 接近 52 週低點，且 P/E 已大幅修正，下行空間相對有限（悲觀情境僅預估 -14.5%）。
3.  **產業龍頭地位：** Zoetis 在寵物與經濟動物領域的多元化佈局能有效抵禦單一產品風險。

**建議策略：**
考慮到目前股價處於低位震盪，建議採取**分批買入（Dollar Cost Averaging）**策略，以應對短期內可能因 Librela 相關新聞引起的波動，長期持有以獲取估值修復與業績增長的雙重紅利。
