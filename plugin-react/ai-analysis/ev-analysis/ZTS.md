# ZTS (Zoetis Inc.) 量化投資分析報告

作為量化投資分析師，針對 Zoetis (ZTS) 目前的市場表現與基本面數據，我將透過機率思考框架，拆解其未來 12 個月的期望值（Expected Value, EV）。

---

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
1.  **Librela 與 Solensia 的市場滲透與安全性澄清**：這兩款用於犬貓骨關節炎（OA）疼痛的單株抗體藥物是 ZTS 的增長引擎。若未來 6-12 個月內臨床數據或市場反饋能有效澄清近期關於副作用的疑慮，將引發強勁的估值修復。
2.  **伴侶動物業務的韌性與利潤率擴張**：ZTS 擁有高達 70.4% 的毛利率與 37.6% 的營業利益率。隨著供應鏈優化與高毛利產品（如 Simparica Trio）佔比提升，財報超預期的可能性增加。
3.  **估值均值回歸 (Mean Reversion)**：目前 Forward P/E 僅 10.74 倍，遠低於其歷史平均水準（通常在 30-40 倍）。一旦市場情緒轉向，極低的 PEG (1.38) 將吸引價值投資者進場。

#### **主要風險點 (Risks)**
1.  **競爭壓力加劇**：競爭對手 Elanco (ELAN) 的新產品（如 Zenrelia, Credelio Quattro）若獲批並展現更佳的安全性或價格優勢，將直接威脅 ZTS 的市場份額。
2.  **監管與安全性負面新聞**：FDA 或 EMA 若對 Librela 等核心產品發布重大安全警告，將導致銷量斷崖式下跌。
3.  **宏觀經濟對寵物支出影響**：雖然寵物醫療具備防禦性，但若經濟衰退加深，非必要的預防性醫療支出可能減少。

### 2. 情境設定與機率賦予 (Scenario Modeling)

基於當前股價 $79.44 與分析師目標價 $120.57，設定以下三種互斥情境：

#### **樂觀情境 (Bull Case)**
*   **發生條件**：Librela 安全性疑慮完全消除，銷量超預期；Elanco 競品上市進度推遲；市場重新給予 ZTS 成長股估值（P/E 回升至 25x）。
*   **預估機率**：25%
*   **目標價格與預期回報**：$120.00 (+51.1%)

#### **基準情境 (Base Case)**
*   **發生條件**：業務維持穩定增長（EPS 增長 ~7%），Librela 銷量穩健但受競爭干擾；估值適度修復至 Forward P/E 15x-18x。
*   **預估機率**：50%
*   **目標價格與預期回報**：$100.00 (+25.9%)

#### **悲觀情境 (Bear Case)**
*   **發生條件**：核心產品面臨重大監管限制或大規模召回；競爭對手全面奪取市場份額；股價跌破 52 週低點尋找支撐。
*   **預估機率**：25%
*   **目標價格與預期回報**：$65.00 (-18.2%)

### 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

#### **決策樹結構**
```text
[ZTS 投資決策]
    ├── 樂觀 (25%): 估值修復 + 產品爆發 -> 回報 +51.1%
    ├── 基準 (50%): 穩健增長 + 均值回歸 -> 回報 +25.9%
    └── 悲觀 (25%): 監管風險 + 競爭加劇 -> 回報 -18.2%
```

#### **總期望值計算**
*   `EV = (0.25 * 0.511) + (0.50 * 0.259) + (0.25 * -0.182)`
*   `EV = 0.12775 + 0.1295 - 0.0455 = 0.21175`
*   **總期望報酬率：21.18%**

#### **風險回報比分析**
*   **上行潛力**：平均約 +38.5% (Bull & Base)
*   **下行空間**：-18.2%
*   **風險回報比 (Risk/Reward Ratio)**：1 : 2.11。這顯示每承擔 1 單位的風險，預期可獲得 2.11 單位的回報，具備高度不對稱的贏面。
