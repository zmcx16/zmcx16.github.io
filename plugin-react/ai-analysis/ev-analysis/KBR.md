### 3. 期望值計算 (Expected Value Calculation)

**總期望報酬率 (Expected Return) 計算公式：**
$E(R) = \sum (P_i \times R_i)$

*   $P_i$ = 該情境發生的機率
*   $R_i$ = 該情境的預期報酬率

**計算過程：**
1.  樂觀：$0.30 \times 59.8\% = 17.94\%$
2.  中性：$0.50 \times 36.8\% = 18.40\%$
3.  悲觀：$0.20 \times (-14.7\%) = -2.94\%$

**總期望報酬率 = $17.94\% + 18.40\% - 2.94\% = 33.4\%$**
