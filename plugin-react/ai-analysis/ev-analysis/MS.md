### 三、 期望值分析 (Expected Value Analysis)

我們將各情境的機率與預期報酬相乘，計算總體期望值。

#### 2. 總體期望報酬率 (Total EV)：
$$EV = 7.5\% + 6.0\% - 3.0\% = 10.5\%$$
