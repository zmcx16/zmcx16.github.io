# ACMR 投資評估：決策樹與期望值分析

本分析旨在評估美股公司 **ACMR** 目前是否適合投資，透過決策樹分析與期望值分析，量化不同市場情境下的潛在報酬與風險。

## 核心假設 (Core Assumptions)

本分析基於以下核心假設：

1.  **時間週期 (Time Horizon)**：12-18 個月。這是評估短期至中期投資潛力的合理範圍。
2.  **投資門檻 (Investment Threshold)**：為補償半導體產業的週期性、地緣政治風險（尤其是美中科技競爭對 ACMR 業務的影響），以及一般股票投資的波動性，我們預期投資 ACMR 的期望報酬應顯著高於無風險利率，且應超過市場平均報酬（例如 S&P 500 的長期年化報酬率約為 8-10%）。我們將設定至少需達到 **10% 以上的期望報酬**才能視為「適合投資」。
3.  **主要不確定性 (Key Uncertainties)**：
    *   **半導體市場景氣 (Semiconductor Market Conditions)**：這是 ACMR 業務最主要的驅動因素，直接影響晶圓設備的需求。
    *   **美中科技關係 (US-China Tech Relations)**：ACMR 在中國市場有 significant 的業務，地緣政治緊張關係可能導致出口管制、供應鏈中斷或客戶關係變化。
4.  **報酬率定義 (Return Definition)**：為預期在時間週期內的股價漲跌幅百分比。這些報酬率是基於上述假設情境下的預期表現。

### 計算過程詳述

**機率假設 (Probabilities):**

*   **半導體市場景氣：**
    *   P(強勁成長) = 0.3 (例如：AI 驅動需求持續爆發，庫存去化加速)
    *   P(溫和成長) = 0.5 (例如：市場穩健復甦，但增長速度放緩)
    *   P(疲軟/衰退) = 0.2 (例如：全球經濟衰退，終端需求不振，半導體週期下行)
*   **美中科技關係 (條件機率)：** 假設地緣政治緊張程度與市場景氣有一定關聯性，市場好時可能緩和，市場差時可能惡化。
    *   若市場「強勁成長」：P(關係穩定/改善) = 0.6, P(關係惡化/趨緊) = 0.4
    *   若市場「溫和成長」：P(關係穩定/改善) = 0.5, P(關係惡化/趨緊) = 0.5
    *   若市場「疲軟/衰退」：P(關係穩定/改善) = 0.4, P(關係惡化/趨緊) = 0.6

**預期報酬假設 (Expected Returns for each terminal node):** 這些是基於各情境下 ACMR 股價在 12-18 個月內的預期漲跌幅。

*   **強勁成長 + 關係穩定/改善** (最佳情境): +35% (ACMR 受益於市場擴張，無重大地緣政治阻力)
*   **強勁成長 + 關係惡化/趨緊**: +10% (市場成長抵消部分地緣政治壓力，但仍限制其潛力)
*   **溫和成長 + 關係穩定/改善**: +15% (市場表現平平，ACMR 穩健經營)
*   **溫和成長 + 關係惡化/趨緊**: -5% (市場平淡，地緣政治壓力導致獲利承壓甚至微虧)
*   **疲軟/衰退 + 關係穩定/改善**: -15% (市場嚴重下滑，即使地緣政治無惡化也難以倖免)
*   **疲軟/衰退 + 關係惡化/趨緊** (最差情境): -30% (市場崩盤疊加地緣政治打擊，公司面臨巨大挑戰)

---

**節點期望值計算 (Expected Value Calculation per Node):**

1.  **計算「強勁成長」情境下的期望值 (Expected Value for "Strong Growth" Scenario):**
    *   EV(強勁成長) = [P(關係穩定 | 強勁成長) * 報酬(關係穩定)] + [P(關係惡化 | 強勁成長) * 報酬(關係惡化)]
    *   EV(強勁成長) = (0.6 * +35%) + (0.4 * +10%)
    *   EV(強勁成長) = 21% + 4% = **+25%**

2.  **計算「溫和成長」情境下的期望值 (Expected Value for "Moderate Growth" Scenario):**
    *   EV(溫和成長) = [P(關係穩定 | 溫和成長) * 報酬(關係穩定)] + [P(關係惡化 | 溫和成長) * 報酬(關係惡化)]
    *   EV(溫和成長) = (0.5 * +15%) + (0.5 * -5%)
    *   EV(溫和成長) = 7.5% - 2.5% = **+5%**

3.  **計算「疲軟/衰退」情境下的期望值 (Expected Value for "Weak/Recession" Scenario):**
    *   EV(疲軟/衰退) = [P(關係穩定 | 疲軟/衰退) * 報酬(關係穩定)] + [P(關係惡化 | 疲軟/衰退) * 報酬(關係惡化)]
    *   EV(疲軟/衰退) = (0.4 * -15%) + (0.6 * -30%)
    *   EV(疲軟/衰退) = -6% - 18% = **-24%**

4.  **計算整體投資 ACMR 的期望值 (Overall Expected Value for Investing in ACMR):**
    *   EV(ACMR 投資) = [P(強勁成長) * EV(強勁成長)] + [P(溫和成長) * EV(溫和成長)] + [P(疲軟/衰退) * EV(疲軟/衰退)]
    *   EV(ACMR 投資) = (0.3 * +25%) + (0.5 * +5%) + (0.2 * -24%)
    *   EV(ACMR 投資) = 7.5% + 2.5% - 4.8%
    *   EV(ACMR 投資) = 10% - 4.8% = **+5.2%**

## 最終結論 (Final Conclusion)

根據我們的決策樹分析與期望值計算，投資 ACMR 在未來 12-18 個月內的**整體期望報酬為 +5.2%**。

**判斷：不適合投資**

**理由：**
儘管計算出的期望值為正，但 +5.2% 的期望報酬率對於像 ACMR 這樣具有高波動性、高度受半導體產業週期性影響，且面臨顯著地緣政治風險的公司而言，並不足以充分補償其所承擔的風險。根據我們的核心假設，為使其成為「適合投資」的標的，其期望報酬應至少達到 10% 以上，以提供足夠的風險溢價。

考慮到存在 -30% 的嚴重虧損情境，+5.2% 的預期報酬率顯得相對微薄，無法提供足夠的風險回報。因此，在當前情境假設下，投資 ACMR 並不具備吸引力，風險報酬比不佳。

**重要提示：**
本分析基於一系列假設（機率和預期報酬），這些假設本身具有高度主觀性，且市場環境可能迅速變化。任何投資決策均應基於個人研究、風險承受能力和專業財務建議。
