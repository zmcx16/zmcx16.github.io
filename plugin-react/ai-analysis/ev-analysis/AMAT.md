### 期望值計算過程

**整體期望報酬率 (Expected Value of Return) =**
(樂觀情境機率 \* 樂觀情境報酬率) + (中性情境機率 \* 中性情境報酬率) + (悲觀情境機率 \* 悲觀情境報酬率)

整體期望報酬率 = (0.35 \* 0.3250) + (0.45 \* 0.0434) + (0.20 \* -0.2716)
整體期望報酬率 = 0.11375 + 0.01953 - 0.05432
**整體期望報酬率 = 0.07896 (約 7.90%)**

**整體期望值 (Expected Value of Investment) =**
當前股價 \* (1 + 整體期望報酬率)

整體期望值 = 301.89 \* (1 + 0.07896)
整體期望值 = 301.89 \* 1.07896
**整體期望值 = 325.70 美元**
