2.  **「投資 ACLS」的整體期望值 (EV_Invest)**
    *   這是一個加權平均的計算，將每個情境的預期報酬乘以其對應的機率，然後將結果加總。
    *   **EV_Invest = (P_強勁成長 * ER_強勁成長) + (P_溫和成長 * ER_溫和成長) + (P_停滯/挫折 * ER_停滯/挫折)**

    *   EV_Invest = 0.21 + 0.10 - 0.08
    *   EV_Invest = 0.31 - 0.08
