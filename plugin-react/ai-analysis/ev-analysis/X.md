#### 2. 計算過程
期望值（EV）的計算公式為：
$$EV = \sum (機率 \times 預期報酬)$$

*   **牛市貢獻**：$40\% \times 45\% = 18\%$
*   **基準貢獻**：$40\% \times 15\% = 6\%$
*   **熊市貢獻**：$20\% \times (-30\%) = -6\%$

**總體期望報酬率**：
$$18\% + 6\% + (-6\%) = 18\%$$
