### 三、 期望值計算 (Expected Value Calculation)

期望值（EV）計算公式：
$EV = (P_{Bull} \times R_{Bull}) + (P_{Base} \times R_{Base}) + (P_{Bear} \times R_{Bear})$

**計算過程：**
1.  **樂觀貢獻**：$0.3 \times 35\% = 10.5\%$
2.  **中性貢獻**：$0.5 \times 10\% = 5.0\%$
3.  **悲觀貢獻**：$0.2 \times (-25\%) = -5.0\%$

**總期望報酬率：**
$10.5\% + 5.0\% - 5.0\% = \mathbf{10.5\%}$
