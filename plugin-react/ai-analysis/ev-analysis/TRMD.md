# TRMD (Torm plc) 量化投資分析報告

作為量化投資分析師，針對成品油輪（Product Tanker）龍頭 **Torm plc (TRMD)**，我們必須理解其高度週期性與地緣政治敏感性的本質。以下基於當前市場數據、運費環境（Spot Rates）及宏觀背景進行期望值決策分析。

---

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
1.  **噸海里（Ton-mile）需求持續擴張**：受紅海危機影響，船舶繞行好望角已常態化，顯著拉長航程。同時，全球煉油重心東移（中東、亞洲）與消費地（歐美）分離，結構性支撐成品油輪需求。
2.  **極低的供給增速**：成品油輪新船訂單量雖有回升，但 2024-2025 年實際交付量仍處於歷史低位。加上環保法規（CII/EEXI）迫使老舊船舶減速或報廢，有效供給受限。
3.  **高額股息政策**：TRMD 擁有極具吸引力的派息政策（通常將大部分自由現金流分配給股東），在當前高運費環境下，年化殖利率具備極強的下行支撐。

#### **主要風險點 (Risks)**
1.  **地緣政治局勢緩解**：若紅海局勢突然和平解決，船舶回歸蘇伊士運河，將釋放約 5-10% 的有效運力，導致運費（TCE）迅速回落。
2.  **全球經濟衰退導致需求萎縮**：成品油（汽油、航空煤油、柴油）需求與經濟活動高度相關，若歐美進入深度衰退，將直接打擊運費。
3.  **OPEC+ 減產延長**：原油產量減少會間接影響成品油的貿易總量，減少油輪租賃需求。

### 2. 情境設定與機率賦予 (Scenario Modeling)
*基準價格參考：$26.00 USD (假設當前價位)；預期持有期：12 個月；回報率包含預期股息。*

#### **樂觀情境 (Bull Case)**
-   **發生條件**：紅海封鎖持續全年，全球補庫存需求強勁，運費維持在 $45,000/天以上。
-   **預估機率**：25%
-   **目標價格與預期回報**：目標價 $36.00，加計約 $5.00 股息，總回報率約 **+58%**。
-   **估值邏輯**：P/NAV 溢價交易，反映極高的現金流產生能力。

#### **基準情境 (Base Case)**
-   **發生條件**：地緣政治維持現狀，運費受季節性波動影響但均值維持在 $35,000/天。
-   **預估機率**：50%
-   **目標價格與預期回報**：目標價 $28.00，加計約 $4.00 股息，總回報率約 **+23%**。
-   **估值邏輯**：基於歷史平均 P/E 與穩定的股息發放。

#### **悲觀情境 (Bear Case)**
-   **發生條件**：紅海危機解除，全球經濟衰退導致成品油需求下降，運費跌破 $20,000/天。
-   **預估機率**：25%
-   **目標價格與預期回報**：目標價 $18.00，加計約 $1.50 股息，總回報率約 **-25%**。
-   **估值邏輯**：股價回落至資產清算價值（NAV）折價區間。

### 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

#### **決策樹結構**
```text
[TRMD Investment]
├── (25%) 樂觀情境: +58% Return
├── (50%) 基準情境: +23% Return
└── (25%) 悲觀情境: -25% Return
```

#### **總期望值計算**
-   `EV = (0.25 * 0.58) + (0.50 * 0.23) + (0.25 * -0.25)`
-   `EV = 0.145 + 0.115 - 0.0625 = 0.1975`
-   **總期望報酬率：19.75%**

#### **風險回報比分析**
-   **上行潛力 (Upside)**：樂觀情境下的 58% 漲幅。
-   **下行空間 (Downside)**：悲觀情境下的 25% 跌幅。
-   **風險回報比 (Risk/Reward Ratio)**：1 : 2.32。這顯示每承擔 1 單位的風險，預期可獲得 2.32 單位的回報，具備正向不對稱性。
