### 一、 核心假設與市場背景分析

### 三、 期望值計算 (Expected Value Calculation)

期望值 (EV) = Σ (各情境機率 × 各情境報酬率)

*   **樂觀情境貢獻**：0.25 × 32.8% = **8.2%**
*   **中性情境貢獻**：0.50 × 12.0% = **6.0%**
*   **悲觀情境貢獻**：0.25 × (-15.0%) = **-3.75%**

**總體期望報酬率 (Total EV) = 8.2% + 6.0% - 3.75% = 10.45%**
