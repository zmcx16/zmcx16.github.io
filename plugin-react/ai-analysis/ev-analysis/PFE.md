### 三、 期望值分析 (Expected Value Analysis)

根據上述情境，我們計算投資 PFE 的總體期望報酬率：

#### 1. 計算公式：
$$EV = \sum (Probability_i \times Return_i)$$

#### 2. 計算過程：
*   **牛市貢獻**：$0.25 \times 35\% = 8.75\%$
*   **基準貢獻**：$0.50 \times 15\% = 7.5\%$
*   **熊市貢獻**：$0.25 \times (-10\%) = -2.5\%$

#### 3. 總期望值：
$$EV = 8.75\% + 7.5\% - 2.5\% = 13.75\%$$
