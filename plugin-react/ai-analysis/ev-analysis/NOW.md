### **3. 整體期望值計算**

**投資 NOW 的整體期望報酬率：**
= (情境一報酬率 * 情境一機率) + (情境二報酬率 * 情境二機率) + (情境三報酬率 * 情境三機率)
= (0.9436 * 0.40) + (0.6582 * 0.45) + (-0.1607 * 0.15)
= 0.37744 + 0.29619 - 0.024105
= 0.649525
= **64.95%**

**投資 NOW 的整體期望值 (以每股當前價格 $113.19 計算)：**
= 當前股價 * (1 + 整體期望報酬率)
= $113.19 * (1 + 0.649525)
= $113.19 * 1.649525
= **$186.71**
