### 計算過程

**1. 終端節點期望值計算**
每個最末端的「預期報酬」是該子情境發生時的直接結果。其「節點期望值」為該「預期報酬」乘以其「機率」。

*   **情境 1.1:** 35% * 0.6 = 21.0%
*   **情境 1.2:** 20% * 0.4 = 8.0%
*   **情境 2.1:** 10% * 0.7 = 7.0%
*   **情境 2.2:** -5% * 0.3 = -1.5%
*   **情境 3.1:** -25% * 0.6 = -15.0%
*   **情境 3.2:** -10% * 0.4 = -4.0%

**2. 中間節點（情境）期望值計算**
每個主要情境的「總期望值」是其所有子情境的「節點期望值」之和。

*   **情境 1 總期望值 (全球經濟復甦強勁):**
    *   EV_情境1 = (預期報酬_1.1 * P_1.1) + (預期報酬_1.2 * P_1.2)
    *   EV_情境1 = (35% * 0.6) + (20% * 0.4) = 21.0% + 8.0% = **29.0%**

*   **情境 2 總期望值 (全球經濟溫和成長):**
    *   EV_情境2 = (預期報酬_2.1 * P_2.1) + (預期報酬_2.2 * P_2.2)
    *   EV_情境2 = (10% * 0.7) + (-5% * 0.3) = 7.0% - 1.5% = **5.5%**

*   **情境 3 總期望值 (全球經濟衰退):**
    *   EV_情境3 = (預期報酬_3.1 * P_3.1) + (預期報酬_3.2 * P_3.2)
    *   EV_情境3 = (-25% * 0.6) + (-10% * 0.4) = -15.0% - 4.0% = **-19.0%**

**3. 最終期望值 (整體投資 GGB 的期望值) 計算**
整體投資 GGB 的期望值是將每個主要情境的「總期望值」乘以該情境發生的「機率」，然後加總。

*   EV_GGB = (EV_情境1 * P_情境1) + (EV_情境2 * P_情境2) + (EV_情境3 * P_情境3)
*   EV_GGB = (29.0% * 0.35) + (5.5% * 0.45) + (-19.0% * 0.20)
*   EV_GGB = 10.15% + 2.475% - 3.8%
*   **EV_GGB = 8.825%**
