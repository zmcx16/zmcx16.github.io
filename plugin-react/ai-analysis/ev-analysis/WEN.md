# Wendy's (WEN) 投資價值定量分析報告

作為量化投資分析師，針對 Wendy's (WEN) 未來 6-12 個月的表現，我將透過概率模型與期望值框架進行深度拆解。目前 WEN 的股價約在 $18.5 - $19.0 區間波動，股息殖利率約為 5.3%，這為投資組合提供了顯著的現金流墊高。

---

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
1.  **早餐時段與數位轉型增長**：Wendy's 持續投入早餐時段的行銷與數位 App 忠誠度計畫。數位銷售佔比已提升至約 15%，若能透過數據分析提高客單價與回購率，將顯著提升利潤率。
2.  **全球門店擴張 (Global Next Gen)**：公司推動新型態店面設計，降低建設成本並提升營運效率。預計 2024-2025 年全球門店增長率將加速，特別是在國際市場（如英國、墨西哥）的滲透。
3.  **高股息防禦屬性**：在市場波動期，WEN 超過 5% 的股息殖利率與穩定的現金流產生能力，使其成為價值型投資者的避風港。

#### **主要風險點 (Risks)**
1.  **低收入消費者支出放緩**：通膨壓力導致低收入族群減少外食，麥當勞等競爭對手發起價格戰（如 $5 超值餐），可能迫使 Wendy's 犧牲毛利以維持市佔。
2.  **成本通膨壓力**：勞動力成本持續上升，加上牛肉等核心食材價格波動，若無法完全轉嫁給消費者，將壓抑營運利潤率。
3.  **高槓桿財務結構**：Wendy's 的債務水平相對較高，在長期高利率環境下，再融資成本的上升可能侵蝕淨利。

### 2. 情境設定與機率賦予 (Scenario Modeling)

以當前股價 **$18.80** 為基準，預估未來 12 個月含息總報酬（Total Return）：

#### **樂觀情境 (Bull Case)**
*   **發生條件**：早餐時段市佔顯著提升；數位銷售佔比突破 20%；國際擴張超預期；通膨降溫帶動利潤率回升。
*   **預估機率**：25%
*   **目標價格與預期回報**：目標價 $23.0 (18x EV/EBITDA)，含息總報酬率約 **+27%**。

#### **基準情境 (Base Case)**
*   **發生條件**：同店銷售額（SSS）維持 2-3% 穩定增長；數位化進程符合預期；股息維持發放；成本壓力與價格調漲達成平衡。
*   **預估機率**：50%
*   **目標價格與預期回報**：目標價 $20.0 (16x EV/EBITDA)，含息總報酬率約 **+11%**。

#### **悲觀情境 (Bear Case)**
*   **發生條件**：美國經濟進入深度衰退；價格戰導致毛利大幅萎縮；門店擴張計畫因融資成本過高而停滯。
*   **預估機率**：25%
*   **目標價格與預期回報**：目標價 $16.0 (13x EV/EBITDA)，含息總報酬率約 **-10%**。

### 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

#### **決策樹結構**
```text
[WEN 投資決策]
├── 樂觀 (25%) ──> 預期回報 +27% (權重貢獻: 6.75%)
├── 基準 (50%) ──> 預期回報 +11% (權重貢獻: 5.50%)
└── 悲觀 (25%) ──> 預期回報 -10% (權重貢獻: -2.50%)
```

#### **總期望值計算**
`EV = (0.25 * 27%) + (0.50 * 11%) + (0.25 * -10%)`
`EV = 6.75% + 5.5% - 2.5% = 9.75%`

#### **風險回報比分析**
*   **上行潛力 (Upside)**：$23.0 - $18.8 = +$4.2
*   **下行空間 (Downside)**：$18.8 - $16.0 = -$2.8
*   **風險回報比 (Risk/Reward Ratio)**：1 : 1.5
*   **分析**：期望值為正（9.75%），且上行空間大於下行空間，顯示該標的具備統計學上的贏面，主要由高股息提供的安全邊際支撐。

**最終結論：**
1. **投資建議**：**持有 (Hold) / 逢低買入 (Buy on Dips)**
2. **核心逻辑**：WEN 的期望值回報約 9.75%，略高於市場長期平均，但其核心吸引力在於「不對稱的下行保護」。5.3% 的股息率為股價提供了強大的底部支撐，即使在悲觀情境下，總虧損也相對可控。目前估值處於歷史中值偏下，適合追求穩定現金流與中等增長的防禦型配置。
