### 明確列出所有計算過程

#### 1. 計算各市場情境下 EPAM 的期望值 (Expected Value for EPAM's Performance within each Market Scenario)

*   **情境 1：強勁復甦 (Strong Recovery)**
    *   計算：(表現優異的機率 * 預期報酬) + (表現良好的機率 * 預期報酬)
    *   EV_Strong = (0.60 * +$5,000) + (0.40 * +$2,500)
    *   EV_Strong = $3,000 + $1,000 = **+$4,000**

*   **情境 2：溫和增長/持平 (Moderate Growth/Stagnation)**
    *   計算：(表現穩定的機率 * 預期報酬) + (表現不佳的機率 * 預期報酬)
    *   EV_Moderate = (0.70 * +$500) + (0.30 * -$1,000)
    *   EV_Moderate = $350 - $300 = **+$50**

*   **情境 3：衰退/熊市 (Recession/Bear Market)**
    *   計算：(具備韌性的機率 * 預期報酬) + (顯著下滑的機率 * 預期報酬)
    *   EV_Recession = (0.50 * -$2,000) + (0.50 * -$4,000)
    *   EV_Recession = -$1,000 - $2,000 = **-$3,000**

#### 2. 計算投資 EPAM 的整體期望值 (Overall Expected Value of Investing in EPAM)

*   計算：(強勁復甦的機率 * EV_Strong) + (溫和增長/持平的機率 * EV_Moderate) + (衰退/熊市的機率 * EV_Recession)
*   Overall EV = (0.30 * +$4,000) + (0.40 * +$50) + (0.30 * -$3,000)
*   Overall EV = $1,200 + $20 - $900
*   Overall EV = **+$320**

根據我們的決策樹分析，投資 EPAM 的整體期望值為 **+$320** (基於 $10,000 的假想投資，相當於 **+3.2% 的預期報酬**)。

**判斷：不適合投資**

**簡短理由：**
儘管計算出的整體期望值為正數，但 +3.2% 的預期報酬對於一檔具有顯著地緣政治風險、市場波動性較高的科技成長股來說，吸引力**不足以彌補其潛在風險**。在同樣的 12-18 個月時間範圍內，投資者可以尋求波動性更低、或風險收益比更具吸引力的替代品（例如：整體市場指數型基金，其歷史平均年化報酬率更高，且分散了單一股票風險；或當前利率環境下的固定收益產品，提供相對安全的正報酬）。

因此，基於本模型所設定的機率和報酬假設，EPAM 目前不被視為一個適合投資的標的，因為其潛在報酬未能充分補償其固有的風險。投資者可能需要更高的預期報酬，或更低的不利情境機率，才能使其成為有吸引力的投資。
