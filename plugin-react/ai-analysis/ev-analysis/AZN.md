### 三、 期望值分析 (Expected Value Analysis)

#### 1. 參數設定
*   **當前股價 ($P_0$):** 204.80
*   **情境 1：樂觀 (Bull Case)**
    *   機率：30%
    *   預期報酬：+25% (基於 2030 願景提前發酵與新藥獲批)
    *   預期股價：$204.8 \times 1.25 = 256.0$
*   **情境 2：基準 (Base Case)**
    *   機率：50%
    *   預期報酬：+8.3% (參考分析師平均目標價 221.82)
    *   預期股價：221.82
*   **情境 3：悲觀 (Bear Case)**
    *   機率：20%
    *   預期報酬：-15% (考慮專利到期與政策風險)
    *   預期股價：$204.8 \times 0.85 = 174.08$

#### 2. 期望值計算過程
$$EV = (P_{Bull} \times Prob_{Bull}) + (P_{Base} \times Prob_{Base}) + (P_{Bear} \times Prob_{Bear})$$

*   $EV = (256.0 \times 0.30) + (221.82 \times 0.50) + (174.08 \times 0.20)$
*   $EV = 76.8 + 110.91 + 34.82$
*   **$EV = 222.53$**

#### 3. 預期報酬率計算
*   **總期望報酬率** = $(222.53 - 204.80) / 204.80 \approx 8.66\%$
*   **加上股息收益率** = $8.66\% + 1.56\% = 10.22\%$
