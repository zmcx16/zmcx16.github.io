### **核心假設 (Core Assumptions)**

為了進行分析，我們需要建立以下核心假設，這些假設將作為情境分析的基礎：

1.  **時間範圍 (Time Horizon)**：12-18個月。這是評估製藥公司短期至中期營運表現和新藥進展的合理期間。
2.  **市場環境 (Market Environment)**：
    *   整體股市維持穩定或溫和成長。
    *   醫療保健板塊因其防禦性及創新需求，仍受投資者關注。
    *   全球經濟沒有出現重大衰退，對藥品需求影響不大。
3.  **產業趨勢 (Industry Trends)**：
    *   製藥產業持續受研發創新驅動，特別是在腫瘤、罕見疾病、心血管與呼吸道疾病領域。
    *   專利懸崖（Patent Cliff）和學名藥（Generics）競爭是持續存在的風險，但AZN在創新藥物的推出上表現積極。
    *   藥品監管環境保持相對穩定，未有大規模的藥價管制壓力。
4.  **AZN 特定假設 (AZN-Specific Assumptions)**：
    *   AZN的研發管線（Pipeline）被視為穩健且多元，特別是其在腫瘤學領域的領先地位。
    *   公司財務狀況良好，有能力支持研發和市場擴張。
    *   管理層維持現有策略，持續專注於高增長領域。
5.  **報酬估計基礎 (Return Estimation Basis)**：以下情境中的預期報酬率為總報酬（股價漲跌幅加上可能的股息收益），這些數值是基於對大型製藥公司在不同營運情境下的歷史表現和市場預期的**假設性估計**。它們不代表實際未來表現，僅供模型分析使用。

### **決策樹分析 (Decision Tree Analysis)**

我們的決策點是「投資AZN」。根據上述假設，我們將情境分為三種可能性：強勁成長與管線成功、溫和表現與穩定成長、以及重大挫折與表現不佳。

#### **計算過程詳述**

**1. 決策點：考慮投資AZN**
   *   這是我們的初始決策點。

**2. 機會節點：AZN未來表現**
   *   此節點代表投資AZN後可能面臨的三種不同市場情境。
   *   我們將計算每種情境的期望值，並加總以得出整體投資AZN的期望值。

    *   **情境一：強勁成長與管線成功**
        *   **情境描述**：AZN的關鍵晚期研發管線藥物（特別是腫瘤學或罕見疾病領域）臨床試驗取得超預期成功或提前獲得監管批准。現有暢銷藥物銷售表現強勁，市場份額擴大，公司業績大幅超越分析師預期。市場給予高評價。
        *   **機率 (Probability)**：35%
        *   **預期報酬 (Expected Return)**：+25% (基於股價上漲與股息)
        *   **期望值 (Expected Value for this scenario)**：
            0.35 (機率) $\times$ 0.25 (預期報酬) = **+0.0875 或 +8.75%**

    *   **情境二：溫和表現與穩定成長**
        *   **情境描述**：AZN的核心產品銷售保持穩定增長，研發管線進展符合預期，沒有重大突破但也沒有重大挫折。公司業績符合或略高於分析師預期。部分競爭壓力或小型試驗延遲被其他利好消息抵消。
        *   **機率 (Probability)**：50%
        *   **預期報酬 (Expected Return)**：+8% (基於穩定股息收入和溫和的股價上漲)
        *   **期望值 (Expected Value for this scenario)**：
            0.50 (機率) $\times$ 0.08 (預期報酬) = **+0.0400 或 +4.00%**

    *   **情境三：重大挫折與表現不佳**
        *   **情境描述**：AZN的一個或多個關鍵晚期研發管線藥物臨床試驗失敗，或遭遇嚴重的監管阻礙、拒絕批准。現有暢銷藥物面臨意想不到的專利到期影響、強烈學名藥競爭或負面安全事件，導致銷售大幅下滑。公司業績顯著低於分析師預期，市場信心受損。
        *   **機率 (Probability)**：15%
        *   **預期報酬 (Expected Return)**：-15% (基於股價下跌)
        *   **期望值 (Expected Value for this scenario)**：
            0.15 (機率) $\times$ (-0.15) (預期報酬) = **-0.0225 或 -2.25%**

**3. 整體投資AZN的期望值 (Overall Expected Value of Investing in AZN)**

將所有情境的期望值加總：
整體期望值 = (情境一期望值) + (情境二期望值) + (情境三期望值)
整體期望值 = +0.0875 + +0.0400 + -0.0225
整體期望值 = **+0.1050 或 +10.50%**

### **最終結論 (Final Conclusion)**

根據上述決策樹分析和期望值計算，投資AZN的整體期望報酬為 **+10.50%**。

*   **理由**：

    *   儘管存在下行風險（-2.25%的期望值貢獻），但AZN強勁成長和穩定成長情境下的正向報酬貢獻（總計+12.75%）足以抵消並超越負面情境的影響。這反映了AZN作為一家大型製藥公司，在研發創新和市場執行方面具備較強的韌性和潛力。
    *   對於一個12-18個月的投資期，10.50%的預期報酬率是相對具吸引力的，尤其考慮到大型製藥股相對穩定的特性。
