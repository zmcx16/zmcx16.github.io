### 三、 期望值計算過程

**1. 總期望報酬率 (Expected Return) 計算：**
$$E(R) = (P_{Bull} \times R_{Bull}) + (P_{Base} \times R_{Base}) + (P_{Bear} \times R_{Bear})$$
$$E(R) = (0.30 \times 0.182) + (0.50 \times 0.10) + (0.20 \times -0.15)$$
$$E(R) = 0.0546 + 0.0500 - 0.0300 = 0.0746$$
**總期望報酬率 = 7.46%**
