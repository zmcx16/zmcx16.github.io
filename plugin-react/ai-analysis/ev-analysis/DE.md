### 三、 期望值計算 (Expected Value Analysis)

根據上述決策樹，我們計算投資 DE 的總體期望報酬率：

$$EV = (P_{Bull} \times R_{Bull}) + (P_{Base} \times R_{Base}) + (P_{Bear} \times R_{Bear})$$

*   **計算過程**：
    *   樂觀：$0.25 \times 20\% = 5.0\%$
    *   基準：$0.50 \times 5\% = 2.5\%$
    *   悲觀：$0.25 \times (-15\%) = -3.75\%$
*   **總期望報酬率**：
    $$5.0\% + 2.5\% - 3.75\% = 3.75\%$$
