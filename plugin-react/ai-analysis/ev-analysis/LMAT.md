### 3. 期望值計算過程 (Expected Value Calculation)

我們計算持有該股票一年後的**預期價值 (Expected Value, EV)** 與 **期望報酬率**。

#### A. 預期股價計算：
$$EV_{Price} = (P_{Bull} \times Price_{Bull}) + (P_{Base} \times Price_{Base}) + (P_{Bear} \times Price_{Bear})$$
$$EV_{Price} = (0.30 \times 135) + (0.50 \times 115) + (0.20 \times 85)$$
$$EV_{Price} = 40.5 + 57.5 + 17 = \mathbf{\$115.00}$$

#### B. 預期報酬率計算：
*   **資本利得報酬**：$(\$115.00 - \$107.86) / \$107.86 = 6.62\%$
*   **股息收益**：$0.79\%$ (根據數據中的 Dividend %)
*   **總期望報酬率**：$6.62\% + 0.79\% = \mathbf{7.41\%}$
