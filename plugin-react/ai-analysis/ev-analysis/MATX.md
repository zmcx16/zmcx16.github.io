#### B. 期望值 (Expected Value, EV) 計算
期望值計算公式：
$EV = (P_{Bull} \times R_{Bull}) + (P_{Base} \times R_{Base}) + (P_{Bear} \times R_{Bear})$

*   **樂觀貢獻**：$0.30 \times 230 = 69.0$
*   **中性貢獻**：$0.50 \times 213 = 106.5$
*   **悲觀貢獻**：$0.20 \times 130 = 26.0$

**預期一年後股價 (EV Price)** = $69.0 + 106.5 + 26.0 = \mathbf{\$201.5}$

**預期報酬率 (Expected Return)**:
$\frac{201.5 - 164.3}{164.3} \times 100\% \approx \mathbf{22.64\%}$
