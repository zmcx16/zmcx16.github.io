# 1. 核心驅動因素與風險 (Drivers & Risks)

### 關鍵催化劑 (Catalysts)
1.  **墨西哥近岸外包 (Nearshoring) 紅利與產能擴張**：Ternium 是墨西哥最大的鋼鐵生產商。隨著北美供應鏈轉移至墨西哥，汽車與工業用鋼需求持續增長。公司在 Pesquería 的新熱軋廠與下游產能預計於 2024-2025 年逐步投產，將顯著提升高附加價值產品的市佔率。
2.  **鋼鐵價格週期性反彈**：目前全球鋼價處於相對低位。若中國房地產政策刺激奏效或全球製造業 PMI 回升，鋼價利差 (Steel Spreads) 擴大將直接帶動 TX 的利潤率彈升。
3.  **高股息與資本回報**：TX 擁有強勁的資產負債表，歷史上維持穩定的股息發放。在當前價值股受青睞的環境下，其超過 5% 的預期股息率具備強大的下行支撐力。

### 主要風險點 (Risks)
1.  **阿根廷宏觀經濟動盪**：TX 在阿根廷擁有大量業務（Ternium Argentina）。阿根廷的惡性通貨膨脹、匯率大幅貶值以及資本管制，可能導致合併報表出現匯兌損失或營運利潤縮減。
2.  **貿易政策與關稅風險**：美國對墨西哥鋼鐵出口的監測加強，若未來 USMCA 重新談判或美國加徵關稅，將衝擊其核心出口市場。
3.  **中國產能過剩外溢**：中國國內需求疲軟導致廉價鋼材大量出口至拉美市場，壓低區域鋼價，擠壓 TX 的利潤空間。

# 2. 情境設定與機率賦予 (Scenario Modeling)

### 樂觀情境 (Bull Case)
-   **發生條件**：墨西哥近岸外包需求超預期，Pesquería 新產能提前達產；阿根廷經濟改革見效，匯率趨穩；全球鋼價進入上升週期。
-   **預估機率**：25%
-   **目標價格與預期回報**：**$56.00 (+40%)**。基於 EV/EBITDA 估值從目前的 3.5x 修復至歷史均值 5.0x，且 EBITDA 因利差擴大增長 15%。

### 基準情境 (Base Case)
-   **發生條件**：墨西哥需求保持穩健增長；阿根廷業務維持現狀，損益兩平或小幅貢獻；鋼價維持在當前區間震盪。
-   **預估機率**：50%
-   **目標價格與預期回報**：**$46.00 (+15%)**。基於當前估值倍數維持不變，主要回報來自於盈利增長與股息發放。

### 悲觀情境 (Bear Case)
-   **發生條件**：全球經濟衰退導致鋼鐵需求萎縮；阿根廷發生極端貨幣貶值；美國對墨鋼鐵實施嚴厲關稅。
-   **預估機率**：25%
-   **目標價格與預期回報**：**$30.00 (-25%)**。股價跌至帳面價值 (P/B) 約 0.5x 的極端安全邊際位置。

# 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

### 決策樹結構
```text
[TX 投資決策]
    |
    |-- 樂觀 (25%): 回報 +40% --> 貢獻 +10.0%
    |
    |-- 基準 (50%): 回報 +15% --> 貢獻 +7.5%
    |
    |-- 悲觀 (25%): 回報 -25% --> 貢獻 -6.25%
```

### 總期望值計算
-   **EV** = (0.25 * 40%) + (0.50 * 15%) + (0.25 * -25%)
-   **EV** = 10% + 7.5% - 6.25% = **11.25%**

### 風險回報比分析
-   **上行潛力 (Upside)**：40% (樂觀)
-   **下行空間 (Downside)**：25% (悲觀)
-   **風險回報比 (Risk/Reward Ratio)**：1 : 1.6。
-   **分析**：期望值為正（11.25%），且上行空間明顯大於下行空間。考慮到 TX 極低的估值倍數（P/E 約 8x），市場已部分定價了阿根廷風險，具備不對稱的博弈價值。
