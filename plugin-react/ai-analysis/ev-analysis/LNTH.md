#### 2. 計算過程
期望值 (EV) = Σ (各情境報酬率 × 對應機率)

*   **樂觀貢獻:** $51.8\% \times 0.30 = 15.54\%$
*   **中性貢獻:** $20.6\% \times 0.50 = 10.30\%$

**總期望報酬率 (Total EV) = 15.54% + 10.30% - 5.48% = 20.36%**

*   **PEG 1.53：** 考慮到其高成長性，估值尚屬合理。
