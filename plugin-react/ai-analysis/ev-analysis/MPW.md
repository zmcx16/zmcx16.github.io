# MPW 投資決策樹與期望值分析

## 核心假設

為了進行本分析，我們基於當前市場對 MPW 的主要關注點做出以下核心假設：

1.  **Steward Health Care 的風險是核心驅動因素**：MPW 的股價表現主要受其最大租戶 Steward Health Care 的財務狀況和租金支付能力影響。Steward 的任何解決方案（穩定、重組、破產）都將對 MPW 產生重大影響。
2.  **市場對風險的敏感度高**：由於 MPW 過去的股價波動和不確定性，市場對任何正面或負面消息都可能產生放大效應。
3.  **其他租戶和資產表現穩定**：假設除了 Steward 之外，MPW 的其他租戶和資產組合大致維持穩定，沒有出現其他大規模違約或資產價值大幅貶值的情況。
4.  **利率環境適度**：假設未來的利率環境不會出現極端變化，避免對 REITs 造成額外的系統性壓力。
5.  **預期報酬為中短期（1-3年）潛在報酬**：情境中的預期報酬是基於在此時間框架內股票的潛在漲跌幅，不考慮長期的股息增長等因素（因目前股息已暫停）。
6.  **初始投資額為 $1000**：便於計算期望值和報酬率。

## 決策樹分析

### 節點說明
*   **決策節點 (Decision Node)**：選擇是否投資。
*   **機會節點 (Chance Node)**：代表不確定性，有不同的情境和機率。
*   **終端節點 (Terminal Node)**：代表每個情境的最終結果或預期報酬。


## 期望值分析

### 所有計算過程

1.  **情境 A 的期望值計算**：
    *   情境名稱：Steward 成功重組/穩定，租金支付正常化
    *   預期報酬：+150%
    *   對應最終價值 (假設初始投資 $1000)：`$1000 * (1 + 1.50) = $2500`
    *   該情境對總期望值的貢獻：`$2500 * 0.20 = $500`

2.  **情境 B 的期望值計算**：
    *   情境名稱：Steward 部分重組/資產出售，租金部分收回/新租戶過渡
    *   *   預期報酬：+20%
    *   對應最終價值 (假設初始投資 $1000)：`$1000 * (1 + 0.20) = $1200`
    *   該情境對總期望值的貢獻：`$1200 * 0.50 = $600`

3.  **情境 C 的期望值計算**：
    *   情境名稱：Steward 破產/大規模違約，MPW 須收回資產/大量減值
    *   預期報酬：-60%
    *   對應最終價值 (假設初始投資 $1000)：`$1000 * (1 - 0.60) = $400`
    *   該情境對總期望值的貢獻：`$400 * 0.30 = $120`

4.  **整體期望值 (Overall Expected Value) 計算**：
    *   整體期望值 = (情境 A 的貢獻) + (情境 B 的貢獻) + (情境 C 的貢獻)
    *   整體期望值 = `$500 + $600 + $120 = $1220`

5.  **預期投資報酬率計算**：
    *   預期淨利潤 = 整體期望值 - 初始投資額 = `$1220 - $1000 = $220`
    *   預期投資報酬率 = `($220 / $1000) * 100% = 22%`

## 最終結論

根據上述決策樹分析和期望值計算，投資 MPW 的 **整體期望值為 $1220** (基於 $1000 的初始投資)。這意味著 **預期投資報酬率為 22%**。

因此，根據本分析模型，**MPW 目前適合投資**。

**簡短理由**：
儘管 MPW 面臨較高的風險（特別是來自 Steward Health Care 的挑戰，導致了 30% 的顯著虧損機率），但市場對其成功解決問題的情境給予了極高的潛在回報（150%）。同時，即使是部分解決問題的溫和情境，也佔據了最大的機率（50%）並帶來了正向回報（20%）。這些較高回報的情境與其相對較高的機率結合，抵消了最差情境的負面影響，使得整體期望值為正，並提供了一個具吸引力的預期回報率。這表明在當前價格下，市場可能已經充分反映了大部分負面消息，為潛在的反彈留下了空間。
