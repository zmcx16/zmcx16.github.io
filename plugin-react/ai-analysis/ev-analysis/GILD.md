# GILD 量化投資分析報告：基於期望值模型與決策樹分析

作為量化投資分析師，針對 **Gilead Sciences (GILD)** 的當前數據與市場動態，我將複雜的變數拆解為機率模型。目前 GILD 正處於從傳統 HIV 龍頭轉型為「HIV 預防 + 腫瘤學」雙引擎驅動的關鍵期。

---

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
1.  **Lenacapavir (HIV PrEP) 的商業化進程**：Lenacapavir 在 PURPOSE 1 & 2 臨床試驗中展現了 100% 的預防效力。未來 6-12 個月內，FDA 的審批進度與市場對其「半年一次」長效針劑的滲透率預期，將是股價突破 52 週高點的核心動力。
2.  **腫瘤學業務 (Trodelvy) 的增長與適應症擴張**：Trodelvy 在三陰性乳腺癌（TNBC）的穩健表現外，若在其他實體瘤（如肺癌）的後續數據超預期，將顯著提升市場對其非 HIV 業務的估值倍數。
3.  **Seladelpar (Livdelzi) 的貢獻**：近期獲批用於治療原發性膽汁性膽管炎（PBC），其市場份額的快速獲取將改善現金流結構。

#### **主要風險點 (Risks)**
1.  **藥價談判風險 (IRA)**：美國《通膨削減法案》對 Medicare 藥價的談判可能壓低 Biktarvy 等核心產品的利潤率。
2.  **臨床試驗波動**：Trodelvy 在非小細胞肺癌（NSCLC）的部分試驗數據曾出現反覆，任何後續數據的挫敗都會引發市場對其收購溢價的質疑。
3.  **專利懸崖擔憂**：儘管 Lenacapavir 延續了生命週期，但市場仍對 2020 年代末期 Biktarvy 專利到期後的增長缺口保持警惕。

---

### 2. 情境設定與機率賦予 (Scenario Modeling)

基於當前股價 $134.43 與分析師平均目標價 $160.73，設定以下三種互斥情境：

#### **樂觀情境 (Bull Case)**
*   **發生條件**：Lenacapavir 獲 FDA 提前批准並被視為「重磅炸彈」；Trodelvy 銷售額同比增長 >30%；整體市場情緒轉向防禦性高增長股。
*   **預估機率**：25%
*   **目標價格與預期回報**：$175 (Forward P/E 擴張至 16x)；**回報率：+30.2%** (含股息)。

#### **基準情境 (Base Case)**
*   **發生條件**：HIV 業務維持個位數增長；Lenacapavir 按部就班推進；腫瘤業務符合預期；EPS 下一年增長 35.7% 的預期達成。
*   **預估機率**：50%
*   **目標價格與預期回報**：$155 (接近分析師共識)；**回報率：+15.3%** (含股息)。

#### **悲觀情境 (Bear Case)**
*   **發生條件**：藥價政策超預期嚴厲；Trodelvy 競爭加劇導致市佔率下滑；大盤出現系統性回撤，回測 200 日均線支撐。
*   **預估機率**：25%
*   **目標價格與預期回報**：$110 (安全邊際支撐位)；**回報率：-18.2%** (含股息)。

### 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

#### **決策樹結構**
```text
[GILD 投資決策]
├── 樂觀 (25%): +30.2% (Lenacapavir 爆發 + 估值修復)
├── 基準 (50%): +15.3% (業務穩健 + EPS 增長兌現)
└── 悲觀 (25%): -18.2% (政策壓力 + 臨床挫敗)
```

#### **總期望值計算**
*   `EV = (0.25 * 0.302) + (0.50 * 0.153) + (0.25 * -0.182)`
*   `EV = 0.0755 + 0.0765 - 0.0455 = 0.1065`
*   **加權平均預期回報：10.65%**

#### **風險回報比分析**
*   **上行潛力 (Upside)**：$175 - $134.43 = $40.57
*   **下行空間 (Downside)**：$134.43 - $110 = $24.43
*   **風險回報比 (Risk/Reward Ratio)**：1 : 1.66。這顯示每承擔 1 單位的風險，可獲得 1.66 單位的潛在回報，具備正向不對稱性。

### 4. 決策總結 (Decision Summary)

| 情境 | 發生機率 (%) | 預期報酬率 (%) | 關鍵驅動/觸發因素 |
| :--- | :--- | :--- | :--- |
| **樂觀 (Bull)** | 25% | +30.2% | Lenacapavir 獲批且滲透超預期，腫瘤業務多點開花 |
| **基準 (Base)** | 50% | +15.3% | HIV 核心業務穩健，EPS 增長 35% 目標達成 |
| **悲觀 (Bear)** | 25% | -18.2% | 藥價談判利空，Trodelvy 臨床數據不及預期 |
| **整體期望值** | **100%** | **10.65%** | **加權平均預期回報 (優於無風險利率)** |

**最終結論：**
1. **投資建議**：**買入 (Buy)**
2. **核心逻辑**：GILD 的期望值回報率 (10.65%) 顯著高於當前無風險利率，且風險回報比優異 (1:1.66)。儘管近期股價有所反彈，但 Forward P/E 僅 13.96x，相對於其 35% 的 EPS 增長預期（PEG 1.55），估值仍具吸引力。Lenacapavir 的潛力尚未被完全定價。
3. **風控建議**：若股價跌破 $120 (SMA200 附近) 或 Lenacapavir 遭遇 FDA 拒絕信 (CRL)，應視為悲觀情境觸發訊號，建議執行止損或減倉以保護資本。
