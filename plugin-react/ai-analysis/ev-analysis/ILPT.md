### 最新資訊與產業趨勢補充

1.  **產業趨勢:** 工業和物流地產市場受電子商務持續增長和供應鏈轉移的推動，需求強勁，預計將維持高入住率和租金增長。ILPT 的物業組合位於關鍵分銷市場和交通樞紐，受益於這些趨勢。
2.  **營運表現:** ILPT 擁有高入住率（截至2025年12月31日為94.5%，2024年第一季度為99%），租賃期限長（加權平均租賃期為7.6年），且約76-77%的年化租金收入來自投資級租戶或夏威夷土地租賃。近期租賃活動顯示租金大幅上漲（例如，2024年第一季度GAAP和現金租賃價差分別為38.3%和25%，2025年第四季度租金平均增長25.7%）。
3.  **財務狀況與債務管理:**
    *   ILPT 在2025年6月成功完成了11.6億美元的五年期固定利率抵押貸款融資，用於償還2025年10月到期的12.35億美元浮動利率抵押貸款。這項再融資預計每年可節省約850萬美元的利息支出（每股0.13美元），並將100%的自有債務轉換為固定利率（6.399%）。
    *   儘管如此，截至2025年7月，公司總債務仍高達43億美元。此外，一筆14億美元的合資企業債務已在2025年3月延期至2026年3月到期，這是一個迫在眉睫的風險。
    *   公司持續報告淨虧損（例如，2025年第四季度EPS為-0.03美元，2025年第一季度每股虧損0.33美元），儘管調整後的營運資金（Normalized FFO）有所改善（2025年第四季度同比增長113%）。
4.  **分析師評級:** 分析師對 ILPT 的看法不一，部分給予「買入」或「強烈買入」評級，平均目標價為7.70美元，但也有共識評級為「持有」的報告。
5.  **管理層變動:** Yael Duffy 於2026年1月1日正式成為執行受託人兼總裁及首席執行官。
6.  **法律調查:** 有消息指出 Kaskela 律師事務所正在對 ILPT 進行調查，具體細節不明。
