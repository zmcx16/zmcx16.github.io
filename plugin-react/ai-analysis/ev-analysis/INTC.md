# Intel (INTC) 投資價值多情境分析報告

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
1.  **Intel 18A 製程量產與外部客戶簽約**：18A 是 Intel 重返領導地位的關鍵。若 2025 年能順利量產並宣佈如 Apple、NVIDIA 或 Amazon 等重量級外部代工客戶，將引發估值重估（Re-rating）。
2.  **資產剝離與結構性改革**：將晶圓代工業務（Foundry）轉為獨立子公司，並可能引入外部投資或分拆，有助於減輕集團現金流壓力並提升透明度。
3.  **CHIPS Act 補貼落地**：美國政府高達 85 億美元的直接補貼與 110 億美元的貸款撥款進度，是緩解其資本支出（CapEx）壓力的重要強心針。

#### **主要風險點 (Risks)**
1.  **代工業務執行風險**：若 18A 良率不及預期或量產推遲，Intel 將徹底失去與 TSM 競爭的機會，導致 Foundry 業務成為永久的現金黑洞。
2.  **數據中心市佔流失**：在 AI 伺服器領域，Intel 的 Gaudi 系列目前難以撼動 NVIDIA；在通用 CPU 領域，則面臨 AMD 與 ARM 架構（如 Ampere, Graviton）的雙重夾擊。
3.  **財務流動性壓力**：目前毛利率處於歷史低位（約 35%），且暫停派息，若營收增長停滯，高額的研發與建廠支出將面臨融資困境。
