3.  **情境 3 期望值 (EV3):**

    *   EV3 = P3 * R3 = 0.15 * -0.12 = **-0.018**

4.  **整體期望值 (EV_Total):**
    *   EV_Total = EV1 + EV2 + EV3
    *   EV_Total = 0.072 + 0.0315 + (-0.018)

因此，投資 AAPL 的整體期望值為 **8.55%**。
