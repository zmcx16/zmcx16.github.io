**計算過程：**
$EV = (26.4\% \times 0.3) + (10\% \times 0.5) + (-25\% \times 0.2)$
$EV = 7.92\% + 5\% - 5\% = 7.92\%$

**考慮股息收益：**
數據顯示 `Dividend %` 為 **0.87%**。
**總期望回報 = 7.92% + 0.87% = 8.79%**
