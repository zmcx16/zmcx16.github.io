### 3. 期望值分析 (Expected Value Analysis)

#### B. 計算過程
期望值 (EV) = (機率1 × 預期股價1) + (機率2 × 預期股價2) + (機率3 × 預期股價3)

*   **EV** = (0.30 × $300) + (0.50 × $244.37) + (0.20 × $180)
*   **EV** = $90 + $122.185 + $36
*   **EV = $248.185**

#### C. 預期報酬率計算
*   **預期總報酬率** = (期望值 - 買入價格) / 買入價格
*   **預期總報酬率** = ($248.185 - $130.23) / $130.23 ≈ **90.58%**
