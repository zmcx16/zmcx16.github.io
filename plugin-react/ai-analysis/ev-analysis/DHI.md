### 3. 期望值計算過程 (Expected Value Calculation)

期望值 (EV) 的計算公式為：
$$EV = \sum (Probability_i \times Return_i)$$

**計算步驟：**
1.  **牛市貢獻：** $0.35 \times 25\% = 8.75\%$
2.  **基本情境貢獻：** $0.50 \times 10\% = 5.0\%$
3.  **熊市貢獻：** $0.15 \times (-20\%) = -3.0\%$

**總期望報酬率：**
$$8.75\% + 5.0\% - 3.0\% = 10.75\%$$
