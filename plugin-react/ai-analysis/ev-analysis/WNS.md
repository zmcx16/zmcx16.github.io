#### 2. 明確列出所有計算過程

**基準股價 (Current Stock Price):** $62.50

**A. 投資 WNS 路徑的期望值計算**

我們將根據不同的情境，設定其發生的機率 (Probability) 和預期的股價 (Expected Stock Price)，進而計算每個情境下的預期報酬率 (Expected Return)。

*   **情境 1: 樂觀情境**
    *   **預測情境名稱:** 強勁增長與成功轉型

    *   **情境期望值 (Scenario Expected Value):** 30% * 44.00% = **+13.20%**

*   **情境 2: 中性情境**

    *   **情境期望值 (Scenario Expected Value):** 45% * 20.00% = **+9.00%**

*   **情境 3: 悲觀情境**
    *   **預測情境名稱:** 表現不佳與挑戰
