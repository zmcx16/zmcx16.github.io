### 三、 期望值分析 (Expected Value Analysis)

#### 2. 計算過程
期望值 (EV) = Σ (各情境報酬 × 對應機率)

*   **樂觀情境報酬**：$9.80 \times (1 + 45\%) = \$14.21$
*   **中性情境報酬**：$9.80 \times (1 + 25\%) = \$12.25$

**期望值計算：**
$$EV = (45\% \times 0.3) + (25\% \times 0.5) + (-15\% \times 0.2)$$
$$EV = 13.5\% + 12.5\% - 3\%$$
$$EV = 23\%$$

**預期一年後股價 + 股息的綜合期望報酬率為 23%。**
