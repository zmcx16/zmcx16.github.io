**1. 時間框架 (Time Horizon):** 12-18 個月，評估短中期投資潛力。

**2. 市場趨勢假設:**
   *   **經濟強勁成長 (Strong Economic Growth):** 全球經濟穩健擴張，消費者信心高漲，可支配所得增加。有利於高端酒品消費和外出消費。
   *   **經濟溫和成長 (Moderate Economic Growth):** 經濟維持穩定但增長放緩，消費者行為趨於理性，對必需品消費較為穩定。
   *   **經濟衰退/疲弱 (Weak Economic Growth / Recession):** 經濟活動收縮，消費者信心低迷，可支配所得減少，可能導致酒品消費降級或減少。

**3. 產業趨勢假設:**
   *   **高端化趨勢 (Premiumization):** 消費者持續傾向購買高品質、高價位的酒品，有利於 DEO 的產品組合。
   *   **即飲趨勢 (Ready-to-Drink, RTD):** RTD 飲料市場持續增長，DEO 在此領域有投入。
   *   **健康意識 (Health & Wellness):** 對於酒精攝入量的健康擔憂可能對整體市場構成壓力，但 DEO 可通過無酒精或低酒精產品應對。
   *   **監管環境 (Regulatory Environment):** 全球各地區的酒精稅收、廣告限制等法規可能影響營運。

**4. 公司特定假設 (DEO Specifics):**
   *   **品牌力量 (Brand Power):** DEO 擁有強大的全球品牌組合（Johnnie Walker, Smirnoff, Guinness, Tanqueray 等），具備定價能力和市場份額。
   *   **營運效率 (Operational Efficiency):** 公司持續進行成本控制和供應鏈優化。
   *   **新興市場潛力 (Emerging Markets):** 尤其在亞太地區，中產階級崛起帶來增長機會。
   *   **股息政策 (Dividend Policy):** 作為消費必需品公司，DEO 通常維持穩定的股息發放，為投資提供一定底部支撐。

**5. 預期報酬率假設 (Expected Return Assumptions):**
   *   這些報酬率是基於股價上漲潛力與股息收益的總和，並考量了風險。
   *   **優異表現 (Excellent Performance):** DEO 超出預期，市佔率提升，新產品成功，利潤率擴大。
   *   **平均表現 (Average Performance):** DEO 表現符合市場預期，保持穩定增長。
   *   **不佳表現 (Weak Performance):** DEO 營運不及預期，市場份額流失，成本壓力增加。

### 2. 明確列出所有計算過程

我們從決策樹的末端開始，向上計算每個機會點的期望值。

**符號說明:**
*   `P_X`: 情境 X 發生的機率。
*   `Return_XY`: 在情境 X 下，DEO 表現為 Y 的預期報酬。
*   `EV_X`: 情境 X 的期望值。
*   `Overall EV`: 最終投資決策的總體期望值。

#### **步驟 1: 計算 DEO 公司表現機會點的期望值**

**1.1 經濟強勁成長 (情境 A) 下的 DEO 表現期望值 (EV_A):**
*   `EV_A = (P_A1 * Return_A1) + (P_A2 * Return_A2) + (P_A3 * Return_A3)`
*   `EV_A = (0.60 * 20%) + (0.30 * 10%) + (0.10 * 5%)`
*   `EV_A = 12.00% + 3.00% + 0.50%`
*   `EV_A = 15.50%`

**1.2 經濟溫和成長 (情境 B) 下的 DEO 表現期望值 (EV_B):**
*   `EV_B = (P_B1 * Return_B1) + (P_B2 * Return_B2) + (P_B3 * Return_B3)`
*   `EV_B = (0.30 * 12%) + (0.50 * 7%) + (0.20 * 2%)`
*   `EV_B = 3.60% + 3.50% + 0.40%`
*   `EV_B = 7.50%`

**1.3 經濟衰退/疲弱 (情境 C) 下的 DEO 表現期望值 (EV_C):**
*   `EV_C = (P_C1 * Return_C1) + (P_C2 * Return_C2) + (P_C3 * Return_C3)`
*   `EV_C = (0.10 * 5%) + (0.40 * -5%) + (0.50 * -15%)`
*   `EV_C = 0.50% - 2.00% - 7.50%`
*   `EV_C = -9.00%`

#### **步驟 2: 計算投資 DEO 的總體期望值 (Overall EV)**

*   `Overall EV = (P_A * EV_A) + (P_B * EV_B) + (P_C * EV_C)`
*   `Overall EV = (0.30 * 15.50%) + (0.50 * 7.50%) + (0.20 * -9.00%)`
*   `Overall EV = 4.65% + 3.75% - 1.80%`
*   `Overall EV = 6.60%`
