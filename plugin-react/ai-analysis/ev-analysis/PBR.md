Petrobras (PBR) 是一家巴西國有控制的綜合性石油和天然氣公司，業務涵蓋石油和天然氣的勘探、生產、煉油、運輸和銷售。由於其國有性質，公司營運和股東回報（尤其是股息政策）極易受到巴西政府政策和政治環境的影響。此外，作為能源公司，其盈利能力與全球原油價格高度相關。

        *   **看漲情境 (Bullish Oil):** 由於地緣政治緊張、OPEC+ 持續減產或全球經濟復甦超預期，原油價格維持在 $90/桶以上。

        *   **不利治理 (Unfavorable Governance):** 政府強烈干預燃料定價、大幅削減股息以資助其他項目，或進行不利於股東的資本支出決策，BRL 可能因政策不確定性而走弱。
