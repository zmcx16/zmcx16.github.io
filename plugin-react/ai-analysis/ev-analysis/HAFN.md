### 核心假設 (Core Assumptions)

在構建決策樹和進行期望值計算之前，我們需要設定一系列核心假設。這些假設是分析的基礎，其變動將直接影響最終結論。

1.  **投資期限（Investment Horizon）**：假設為未來 12-24 個月。
2.  **市場趨勢（Market Trends）**：
    *   **全球經濟增長**：影響成品油消費與貿易量。
    *   **地緣政治穩定性**：例如紅海衝突、俄烏戰爭等，這些事件可能導致航線繞道，增加成品油輪的周轉時間（ton-mile demand），從而推高運費。
    *   **利率環境**：影響 HAFN 的借貸成本和整體資本市場情緒。
3.  **產業趨勢（Industry Trends - 成品油輪特定）**：
    *   **新船訂單與船隊供給**：目前成品油輪的新船訂單簿相對較低，有助於維持供給緊張。然而，若訂單量顯著增加，未來供給可能過剩。
    *   **拆船率**：老舊船舶的拆解速度影響船隊供給。
    *   **燃油價格（Bunker Fuel Prices）**：成品油輪的主要營運成本之一。
4.  **公司財務與營運（Company-Specific Factors - HAFN）**：
    *   **營運效率與成本控制**：假設 HAFN 能維持其現有的高效營運模式。
    *   **船隊規模與結構**：HAFN 擁有大型、現代化的船隊，有助於其市場競爭力。
    *   **財務槓桿**：HAFN 的債務水平和現金流表現。
    *   **股息政策**：成品油輪公司在高運費環境下通常會派發較高股息。

#### 1. 預測情境與機率（Scenario Probability）

基於當前市場環境和上述假設，我們設定以下三種市場情境及其機率：

*   **看漲情境 (Bullish Market)**：
    *   **描述**：全球經濟表現強勁，成品油需求持續增長。地緣政治緊張局勢（如紅海問題）長期化，導致航運路線持續延長，成品油輪運力持續吃緊。新船訂單量維持低位，未有大量新船交付。高運費（TCE rates）維持高檔。
    *   **機率**：35%
    *   **預期報酬**：+30% (基於高運費帶來的強勁盈利、高股息及股價上漲潛力)

*   **中性情境 (Neutral Market)**：
    *   **描述**：全球經濟溫和增長，成品油需求穩定。地緣政治緊張局勢有所緩解但未完全解決，航運效率部分恢復，但仍高於常態。新船交付量適度增加。運費水平從高點略有回落，但仍保持在健康水平。
    *   **機率**：45%
    *   **預期報酬**：+10% (基於穩定的盈利、常規股息及溫和股價增長)

*   **看跌情境 (Bearish Market)**：
    *   **描述**：全球經濟顯著放緩，甚至陷入衰退，成品油需求大幅下降。地緣政治緊張局勢完全解決，航運路線恢復正常，運力過剩顯現。大量新船交付導致市場供給過剩。運費水平大幅下降，公司盈利能力受損。
    *   **機率**：20%
    *   **預期報酬**：-15% (基於盈利壓力、股息減少甚至取消，及股價下跌)

### 期望值分析 (Expected Value Analysis) 與計算過程

期望值分析將計算在各種情境下的加權平均報酬，以得出每個決策節點的總體期望值。

#### 1. 計算「投資 HAFN」選項的期望值

*   **步驟一：計算各情境的期望報酬**
    *   **看漲市場情境**：
        *   期望報酬 = 機率 × 預期報酬

    *   **中性市場情境**：
        *   期望報酬 = 機率 × 預期報酬
        *   期望報酬 = 0.45 × (+0.10) = +0.045 或 +4.5%

    *   **看跌市場情境**：
        *   期望報酬 = 機率 × 預期報酬
        *   期望報酬 = 0.20 × (-0.15) = -0.030 或 -3.0%

*   **步驟二：計算「市場情境」事件節點的期望值**
    *   此節點的期望值是所有情境期望報酬的總和。
    *   `EV_市場情境` = (+0.105) + (+0.045) + (-0.030)

*   **步驟三：計算「投資 HAFN」決策選項的期望值**
    *   由於「投資 HAFN」選項直接連結到「市場情境」事件節點，因此其期望值即為 `EV_市場情境`。
    *   `EV_投資HAFN` = +12.0%

*   `EV_不投資HAFN` = 0.00%
