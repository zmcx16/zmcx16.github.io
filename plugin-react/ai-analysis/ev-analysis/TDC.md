$$ER = \sum (Probability_i \times Return_i)$$

$$ER = (0.3 \times 65.94\%) + (0.5 \times 22.48\%) + (0.2 \times -28.88\%)$$
$$ER = 19.78\% + 11.24\% - 5.78\% = 25.24\%$$

**期望價值 (Expected Value, EV)：**
