# Teradata (TDC) 量化投資分析報告

作為量化投資分析師，針對 Teradata (TDC) 的當前數據與市場動態，我將透過機率模型與期望值（Expected Value, EV）框架進行深度拆解。

---

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
1.  **雲端 ARR 增長與轉型加速**：Teradata 正從傳統地端轉向 VantageCloud。若 Cloud ARR（年度經常性收入）增長率超過市場預期的 30% 以上，將引發估值倍數（Re-rating）從價值股向成長股修復。
2.  **AI 與數據編排需求**：隨著企業對 AI 數據準備（Data Prep）的需求增加，Teradata 的 AI Unlimited 與微軟 Azure 的深度整合若能轉化為具體訂單，將成為股價突破 52 週高點的動力。
3.  **空頭擠壓 (Short Squeeze) 潛力**：目前 Short Float 高達 17.24%，在 P/E 僅 8.37 倍的低估值背景下，任何超預期的財報利多都極易引發空頭回補的連鎖反應。

#### **主要風險點 (Risks)**
1.  **競爭加劇與市佔流失**：面對 Snowflake、Databricks 及雲端巨頭（AWS Redshift, Google BigQuery）的直接競爭，若舊有客戶流失速度超過新雲端客戶增長速度，營收將萎縮。
2.  **宏觀 IT 支出放緩**：企業在經濟不確定性下可能推遲大型數據平台的遷移計畫，這對 TDC 這種高客單價的 B2B 業務是直接打擊。
3.  **高槓桿與財務壓力**：Debt/Eq 為 0.99，雖然 P/FCF 僅 5.13 顯示現金流強勁，但在高利率環境下，債務再融資成本可能侵蝕利潤。

### 2. 情境設定與機率賦予 (Scenario Modeling)

基於當前股價 $34.97 及分析師平均目標價 $34.14，我們設定未來 12 個月的三種情境：

#### **樂觀情境 (Bull Case)**
*   **發生條件**：Cloud ARR 增長 >35%，AI 相關營收貢獻顯著，且發生空頭擠壓。
*   **預估機率**：25%
*   **目標價格與預期回報**：**$50.00 (+43%)**。基於 Forward P/E 回升至 18x（接近行業平均）或 P/FCF 回升至 8x。

#### **基準情境 (Base Case)**
*   **發生條件**：雲端轉型符合預期，流失率穩定，維持現有的獲利能力。
*   **預估機率**：50%
*   **目標價格與預期回報**：**$38.00 (+8.6%)**。基於 Forward P/E 13.5x，反映穩定的經常性收入增長。

#### **悲觀情境 (Bear Case)**
*   **發生條件**：地端客戶大幅流失，雲端增長停滯，或宏觀衰退導致 IT 預算砍半。
*   **預估機率**：25%
*   **目標價格與預期回報**：**$24.00 (-31%)**。基於 P/S 降至 1.5x，回測 52 週低點附近的支撐位。

### 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

#### **決策樹結構**
```text
[投資 TDC]
    |
    |-- 樂觀 (25%): +43% 回報 --> EV 貢獻: 10.75%
    |-- 基準 (50%): +8.6% 回報 --> EV 貢獻: 4.30%
    |-- 悲觀 (25%): -31% 回報 --> EV 貢獻: -7.75%
```

#### **總期望值計算**
`EV = (0.25 * 43%) + (0.50 * 8.6%) + (0.25 * -31%)`
`EV = 10.75% + 4.30% - 7.75% = 7.30%`

#### **風險回報比分析**
*   **上行潛力 (Upside)**：$50.00 - $34.97 = $15.03
*   **下行空間 (Downside)**：$34.97 - $24.00 = $10.97
*   **風險回報比 (Risk/Reward Ratio)**：1 : 1.37。這顯示該筆投資具備正向不對稱性，但優勢並非壓倒性。
