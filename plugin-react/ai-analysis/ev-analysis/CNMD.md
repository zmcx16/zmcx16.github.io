### 3. 期望值計算過程 (Expected Value Calculation)

#### A. 核心假設
*   **當前股價：** $40.60
*   **股息收益：** 1.48% (約 $0.60)
*   **計算公式：** $EV = \sum (機率 \times 預期報酬率) + 股息收益率$

#### B. 計算步驟
1.  **牛市貢獻：** $0.25 \times 35\% = 8.75\%$
2.  **基本情境貢獻：** $0.50 \times 10\% = 5.0\%$
3.  **熊市貢獻：** $0.25 \times (-20\%) = -5.0\%$
4.  **加總期望報酬率：** $8.75\% + 5.0\% - 5.0\% = 8.75\%$
5.  **總期望值 (含股息)：** $8.75\% + 1.48\% = 10.23\%$
