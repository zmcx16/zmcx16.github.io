   *   衰退/挑戰市場 -> 嚴重下滑: $0.20 \times (-30\%) = -6.0\%$

**2. 計算各市場情境下的節點期望值 (Expected Value at Market Scenario Nodes):**

**3. 計算整體投資期望值 (Overall Expected Value for Investing in CNMD):**

   $EV_{投資CNMD} = \textbf{11.71\%}$
