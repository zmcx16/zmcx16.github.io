### 3. 期望值計算過程 (Expected Value Calculation)

期望值 (EV) 的計算公式為：
$$EV = \sum (Probability_i \times Return_i)$$

**計算步驟：**
1.  **樂觀貢獻：** $0.30 \times 18.5\% = 5.55\%$
2.  **中性貢獻：** $0.50 \times 8.5\% = 4.25\%$
3.  **悲觀貢獻：** $0.20 \times (-12.0\%) = -2.40\%$

**總期望報酬率：**
$$5.55\% + 4.25\% - 2.40\% = 7.40\%$$
