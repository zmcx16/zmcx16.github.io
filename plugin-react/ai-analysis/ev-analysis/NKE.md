### **期望值分析 (Expected Value Analysis) - 計算過程**

期望值 = (樂觀情境機率 * 樂觀情境預期報酬) + (中性情境機率 * 中性情境預期報酬) + (悲觀情境機率 * 悲觀情境預期報酬)

*   **樂觀情境期望值:** 0.30 * $15.30 = $4.59
*   **中性情境期望值:** 0.50 * $2.47 = $1.235

**總期望值 (EV_NKE) = $4.59 + $1.235 - $2.02 = $3.805**

這表示每投資一股 NKE (當前價格 $62.24)，預期在一年後可獲得 $3.805 的收益。
