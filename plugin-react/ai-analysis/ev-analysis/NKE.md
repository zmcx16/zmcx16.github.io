## NKE 投資評估：決策樹與期望值分析

### 核心假設

在進行決策樹分析之前，我們需要設定一系列核心假設，這些假設將影響各情境的機率與預期報酬。

1.  **投資期限：** 1 年。
2.  **市場環境假設：**
    *   **經濟週期：** 假設未來一年全球經濟可能呈現多種狀況，從強勁增長到溫和衰退。
    *   **消費者支出：** 與整體經濟狀況高度相關，NKE 作為非必需消費品，受此影響較大。
    *   **利率與通膨：** 假設通膨可能逐漸受控，但利率維持在相對高位一段時間。
3.  **NKE 公司特定假設：**
    *   **品牌力：** NKE 作為全球領先的運動品牌，其品牌護城河依然穩固。
    *   **直營（DTC）策略：** 持續推進 DTC 轉型，目標是提高毛利率和客戶黏著度。
    *   **庫存管理：** 預計庫存壓力在短期內可能持續存在，但中長期會有所改善。
    *   **中國市場：** 假設中國市場在未來一年可能經歷溫和復甦，但存在不確定性。
    *   **競爭格局：** 面臨 Adidas、Lululemon 等品牌的激烈競爭，以及時尚潮流的快速變化。
    *   **產品創新：** 假設 NKE 能持續推出具吸引力的新產品和科技。
4.  **預期報酬估計基礎：**
    *   基於當前股價、歷史股價波動、分析師共識預期、以及對公司基本面的判斷。所有報酬率均為扣除潛在交易成本前的預估。
    *   情境機率設定基於對當前宏觀經濟和產業趨勢的主觀判斷，所有機率總和為 1。

### 2. 明確列出所有計算過程

我們將從最末端的預期報酬節點開始，逆向計算各機率節點的期望值，直到初始決策點。

**a. 計算 NKE 在不同市場情境下的期望值**

*   **情境1：市場表現強勁 (P=0.40)**
    *   預期報酬：
        *   NKE 超越市場預期 (+30%)，機率 P=0.40
        *   NKE 符合市場預期 (+15%)，機率 P=0.40
        *   NKE 低於市場預期 (+5%)，機率 P=0.20
    *   **EV_市場強勁** = (0.40 * 30%) + (0.40 * 15%) + (0.20 * 5%)
        = 12.0% + 6.0% + 1.0% = **19.0%**
        *(註：圖中為 18.5%，此處依據計算結果修正為 19.0%，此為計算細節調整)*

*   **情境2：市場表現溫和 (P=0.40)**
    *   預期報酬：
        *   NKE 超越市場預期 (+15%)，機率 P=0.30
        *   NKE 符合市場預期 (+8%)，機率 P=0.40
        *   NKE 低於市場預期 (-5%)，機率 P=0.30
    *   **EV_市場溫和** = (0.30 * 15%) + (0.40 * 8%) + (0.30 * -5%)
        = 4.5% + 3.2% - 1.5% = **6.2%**
        *(註：圖中為 5.5%，此處依據計算結果修正為 6.2%，此為計算細節調整)*

*   **情境3：市場表現不佳 (P=0.20)**
    *   預期報酬：
        *   NKE 展現韌性 (+5%)，機率 P=0.20
        *   NKE 符合市場預期 (-10%)，機率 P=0.40
        *   NKE 低於市場預期 (-20%)，機率 P=0.40
    *   **EV_市場不佳** = (0.20 * 5%) + (0.40 * -10%) + (0.40 * -20%)
        = 1.0% - 4.0% - 8.0% = **-11.0%**

**b. 計算「投資 NKE」的整體期望值**

*   基於以上三個市場情境的期望值及其對應的機率：
    *   市場強勁 (EV=19.0%)，機率 P=0.40
    *   市場溫和 (EV=6.2%)，機率 P=0.40
    *   市場不佳 (EV=-11.0%)，機率 P=0.20
*   **整體期望值 (EV_投資 NKE)** = (0.40 * 19.0%) + (0.40 * 6.2%) + (0.20 * -11.0%)
    = 7.6% + 2.48% - 2.2%
    = **7.88%**
    *(註：圖中為 7.11%，此處依據計算結果修正為 7.88%，此為計算細節調整)*

**c. 計算「不投資 NKE」的期望值**

*   如果選擇不投資 NKE，假設將資金保留現金或投資於無風險資產，為簡化分析，我們設定其期望值為 **0%** (或可考慮扣除通膨後的實質零報酬)。

### 3. 最終結論

根據上述計算，
*   **投資 NKE 的整體期望值為：7.88%**
*   不投資 NKE 的整體期望值為：0%

**判斷：適合投資**

**簡短理由：**

根據期望值分析，投資 NKE 的預期報酬率為 7.88%，顯著高於不投資（0%）的選項。這表示在我們設定的市場及公司情境機率和預期報酬下，NKE 作為一項投資具有正向的期望價值。儘管存在市場表現不佳和公司業績低於預期的風險情境，但其強勁品牌力、DTC 策略的推進以及在有利市場情境下的潛在增長，足以讓其整體期望值為正。因此，從期望值分析的角度看，NKE 目前適合投資。

**風險提示：**

此分析高度依賴於對各情境機率和預期報酬的主觀判斷，實際市場狀況和公司表現可能與假設存在差異。投資者仍需綜合考慮個人風險承受能力、市場流動性、NKE 最新財報及產業動態等因素。
