#### B. 期望值 (Expected Value, EV) 計算
期望值計算公式：
$$EV = \sum (機率 \times 預期股價)$$

*   **樂觀節點**：$135 \times 0.30 = 40.5$
*   **中性節點**：$116 \times 0.50 = 58.0$
*   **悲觀節點**：$75 \times 0.20 = 15.0$

**總期望股價 = $40.5 + $58.0 + $15.0 = $113.5**

#### C. 預期報酬率計算
*   目前股價：$97.70
*   期望股價：$113.5
*   **預期報酬率 = ($113.5 - $97.70) / $97.70 = +16.17%**
