### 3. 期望值計算 (Expected Value Analysis)

期望值 (EV) 計算公式：
$EV = (P_{Bull} \times R_{Bull}) + (P_{Base} \times R_{Base}) + (P_{Bear} \times R_{Bear})$

**計算過程：**
1.  **牛市貢獻：** $0.30 \times 30\% = 9\%$
2.  **基準貢獻：** $0.50 \times 10\% = 5\%$
3.  **熊市貢獻：** $0.20 \times (-20\%) = -4\%$

**總期望報酬率：**
$9\% + 5\% - 4\% = \mathbf{10\%}$

**期望股價 (12個月)：**
$76.20 \times (1 + 10\%) = \mathbf{83.82}$
