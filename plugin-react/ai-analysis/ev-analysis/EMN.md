### 計算過程 (Calculation Process)

我們將從決策樹的末端開始，向後計算每個節點的期望值。

**1. 計算第二層機會節點的期望值：**

*   **情境 A: 強勁全球經濟復甦 (機率 20%)**
    *   預期報酬率 (EV_A) = (EMN 表現優異機率 \* 報酬率) + (EMN 表現一般機率 \* 報酬率)
    *   EV_A = (0.70 \* +35%) + (0.30 \* +20%)
    *   EV_A = 24.5% + 6.0%
    *   **EV_A = 30.5%**
    *   *此情境下投資 EMN 的期望報酬為 30.5%。*

*   **情境 B: 溫和經濟增長/穩定 (機率 50%)**
    *   預期報酬率 (EV_B) = (EMN 符合預期機率 \* 報酬率) + (EMN 未達預期機率 \* 報酬率)
    *   EV_B = (0.60 \* +10%) + (0.40 \* -5%)
    *   EV_B = 6.0% - 2.0%
    *   **EV_B = 4.0%**
    *   *此情境下投資 EMN 的期望報酬為 4.0%。*

*   **情境 C: 經濟衰退/放緩 (機率 30%)**
    *   預期報酬率 (EV_C) = (EMN 表現具韌性機率 \* 報酬率) + (EMN 表現疲軟機率 \* 報酬率)
    *   EV_C = (0.40 \* -10%) + (0.60 \* -30%)
    *   EV_C = -4.0% - 18.0%
    *   **EV_C = -22.0%**
    *   *此情境下投資 EMN 的期望報酬為 -22.0%。*

**2. 計算決策節點「投資 EMN 嗎？」的整體期望值：**

*   整體期望值 (EV_Overall) = (強勁復甦情境機率 \* EV_A) + (溫和增長情境機率 \* EV_B) + (經濟衰退情境機率 \* EV_C)
*   EV_Overall = (0.20 \* 30.5%) + (0.50 \* 4.0%) + (0.30 \* -22.0%)
*   EV_Overall = 6.1% + 2.0% - 6.6%
*   **EV_Overall = 1.5%**

### 最終結論 (Final Conclusion)

根據上述決策樹與期望值分析，美股公司 EMN 目前 **不適合投資**。

**理由：**

計算出的整體期望報酬率僅為 **1.5%**。在考慮到投資單一股票所固有的較高風險（相對於多元化投資組合或無風險資產），以及當前通脹和市場利率水平，1.5% 的預期報酬率顯著過低。這樣的報酬率甚至可能無法覆蓋投資的機會成本或通脹侵蝕。

雖然在「強勁全球經濟復甦」情境下，EMN 有機會帶來較高回報，但該情境的機率較低。而公司在經濟衰退情境下表現疲軟的可能性（30% 機率下的 60%，即總體 18% 機率）將導致顯著的負回報，極大地拉低了整體期望值。

因此，從純粹的期望值角度來看，投資 EMN 並不具有吸引力。投資者應尋求具有更高風險調整後預期報酬的機會。
