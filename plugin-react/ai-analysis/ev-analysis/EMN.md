#### 2. 期望值 (Expected Value, EV) 計算
期望值計算公式：
$$EV = \sum (機率 \times 預期報酬)$$

*   **樂觀情境貢獻**：$30\% \times 25\% = 7.5\%$
*   **中性情境貢獻**：$50\% \times 12\% = 6.0\%$
*   **悲觀情境貢獻**：$20\% \times (-15\%) = -3.0\%$

**總體期望報酬率 = 7.5% + 6.0% - 3.0% = 10.5%**
