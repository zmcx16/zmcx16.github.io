# Eastman Chemical (EMN) 投資價值量化分析報告

作為量化投資分析師，我將透過期望值模型（Expected Value Model）與決策樹分析，拆解 Eastman Chemical (EMN) 在未來 6-12 個月的投資勝率。

---

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
1.  **循環經濟轉型與 Kingsport 工廠放量**：EMN 位於田納西州 Kingsport 的全球最大分子回收工廠已開始運營。隨著產能利用率提升，該技術（聚酯再生技術）將帶動高毛利特種塑料業務增長，並吸引 ESG 溢價資金。
2.  **去庫存週期結束與銷量回升**：經歷了 2023 年的劇烈去庫存，2024 年起終端市場（如包裝、建築、汽車）需求回溫。Q2 財報已顯示銷量轉正，預計未來兩季將持續受益於補庫存需求。
3.  **降息環境下的估值修復**：EMN 屬於資本密集型且與終端消費掛鉤的特種化學品公司。聯準會進入降息週期將降低其債務成本（Debt/Eq 0.87），並刺激下游建築與汽車市場需求。

#### **主要風險點 (Risks)**
1.  **全球宏觀經濟疲軟**：特別是歐洲與中國市場的復甦慢於預期，可能抵消北美市場的增長。
2.  **原材料與能源價格波動**：作為化學品製造商，天然氣與原油衍生品價格上漲將直接壓縮毛利率（目前 Gross Margin 為 19.96%，處於歷史中低位）。
3.  **新技術執行風險**：法國與德州後續回收工廠的資本支出巨大，若 Kingsport 運營效率未達標，將引發市場對其「循環經濟」故事的質疑。

---

### 2. 情境設定與機率賦予 (Scenario Modeling)

基於當前股價 $76.36、Forward P/E 10.87x 以及 4.4% 的股息率，設定以下三種互斥情境：

#### **樂觀情境 (Bull Case)**
*   **發生條件**：Kingsport 工廠提前達到滿產，特種塑料需求因環保法規超預期增長；聯準會降息節奏快，帶動房地產與汽車銷量大增。
*   **預估機率**：25%
*   **目標價格與預期回報**：$100 (基於 Forward P/E 14x)。預期總回報（含股息）：**+35%**。

#### **基準情境 (Base Case)**
*   **發生條件**：銷量溫和復甦，循環經濟業務按計畫貢獻利潤；利潤率因成本控制改善。市場共識 EPS 達成。
*   **預估機率**：55%
*   **目標價格與預期回報**：$85 (接近分析師目標價 $83)。預期總回報（含股息）：**+16%**。

#### **悲觀情境 (Bear Case)**
*   **發生條件**：全球經濟衰退，終端需求萎縮；新工廠出現重大技術故障導致停工；能源成本飆升。
*   **預估機率**：20%
*   **目標價格與預期回報**：$62 (回測 52W 低點，P/B 接近 1.1x)。預期總回報（含股息）：**-14%**。

### 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

#### **決策樹結構**
```text
[投資 EMN]
    |
    |---- 樂觀 (25%): 回報 +35%  => 貢獻 EV: 8.75%
    |
    |---- 基準 (55%): 回報 +16%  => 貢獻 EV: 8.80%
    |
    |---- 悲觀 (20%): 回報 -14%  => 貢獻 EV: -2.80%
```

#### **總期望值計算**
*   **EV** = (0.25 * 35%) + (0.55 * 16%) + (0.20 * -14%)
*   **EV** = 8.75% + 8.80% - 2.80% = **14.75%**

#### **風險回報比分析**
*   **上行潛力 (Upside)**：+16% ~ +35%
*   **下行空間 (Downside)**：-14%
*   **不對稱性**：期望值為正（14.75%），且基準情境下的回報已具備吸引力。風險回報比約為 1 : 1.8（以基準+樂觀對比悲觀），顯示贏面較大。

### 4. 決策總結 (Decision Summary)

| 情境 | 發生機率 (%) | 預期報酬率 (%) | 關鍵驅動/觸發因素 |
| :--- | :--- | :--- | :--- |
| **樂觀 (Bull)** | 25% | +35% | 分子回收技術全面成功，ESG 溢價與降息紅利疊加。 |
| **基準 (Base)** | 55% | +16% | 補庫存需求穩定，Kingsport 工廠按進度貢獻利潤。 |
| **悲觀 (Bear)** | 20% | -14% | 全球經濟衰退，能源成本失控，新技術執行受阻。 |
| **整體期望值** | **100%** | **14.75%** | **加權平均預期回報（含 4.4% 股息保護）** |

**最終結論：**
1.  **投資建議**：**買入 (Buy)**
2.  **核心逻辑**：EMN 目前估值極具吸引力（Forward P/E 僅 10.8x），遠低於歷史均值。4.4% 的高股息提供了強大的下行保護（Safety Margin）。隨著 Kingsport 工廠這一關鍵催化劑發酵，公司正從傳統週期股轉型為具備成長性的循環經濟龍頭，期望值 14.75% 顯著優於標普 500 歷史平均。
3.  **風控建議**：若股價跌破 $68（SMA200 支撐位附近），需重新評估 Kingsport 工廠的運營數據；若毛利率連續兩季下滑超過 200 個基點，則視為悲觀情境觸發，應考慮減碼。
