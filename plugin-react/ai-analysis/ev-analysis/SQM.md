## SQM 投資評估：決策樹與期望值分析

### 核心假設

本分析基於以下核心假設，這些假設將影響情境的機率與預期報酬：

1.  **投資期限**：本評估的預期報酬是基於未來 **12-18 個月** 的潛在投資回報。
2.  **SQM 業務重點**：SQM 的主要獲利動能來自鋰（Lithium）業務，碘、硝酸鹽、鉀肥等業務亦有貢獻，但鋰市場的波動性對公司影響最大。
3.  **鋰市場趨勢**：
    *   **需求面**：全球電動車（EV）產業的成長速度、儲能應用擴張、以及電池技術演進（例如：鈉離子電池的商業化進程）。
    *   **供給面**：全球鋰礦產能的擴張速度（澳洲、阿根廷、中國新產能）、現有供應商的擴產進度。
    *   **價格**：鋰價的波動性極高，受供需失衡、庫存水準及市場情緒影響。
4.  **智利政府政策**：
    *   智利政府對鋰產業的國家化政策（Public-Private Partnership Model）及其執行細節，將影響 SQM 在 Atacama 鹽湖的租賃合約續簽及未來營運模式。
    *   潛在的稅率或特許權使用費（Royalty）調整。
5.  **財務表現**：SQM 自身的生產成本、擴張計劃的執行效率、資本支出和現金流管理能力。
6.  **宏觀經濟**：全球經濟成長、利率環境、通膨壓力對終端需求及企業融資成本的影響。
7.  **ESG 因素**：環境保護、水資源利用、社區關係等因素可能帶來營運風險或監管壓力。

### 決策樹分析

我們將從「投資 SQM」的決策點開始，分析不同市場情境下 SQM 的潛在表現及其對應的期望值。

**決策點：** 投資 SQM

```
[投資 SQM]
  └─┐
    ├─── [情境 1: 全球鋰市場 - 強勁成長 (高鋰價)] (機率: 30%)
    │     ├─── [結果 1.1: SQM 成功擴產並受惠] (機率: 70%) - 預期報酬: +40%
    │     └─── [結果 1.2: SQM 遇小幅阻礙 (如監管、成本增)] (機率: 30%) - 預期報酬: +15%
    │
    ├─── [情境 2: 全球鋰市場 - 適度成長 (穩定鋰價)] (機率: 45%)
    │     ├─── [結果 2.1: SQM 營運符合預期] (機率: 60%) - 預期報酬: +10%
    │     └─── [結果 2.2: SQM 面臨競爭或小型挑戰] (機率: 40%) - 預期報酬: +3%
    │
    └─── [情境 3: 全球鋰市場 - 供應過剩/需求放緩 (低鋰價)] (機率: 25%)
          ├─── [結果 3.1: SQM 獲利受重創] (機率: 60%) - 預期報酬: -25%
          └─── [結果 3.2: SQM 憑成本優勢較具韌性] (機率: 40%) - 預期報酬: -10%
```

### 計算過程

#### 1. 計算各情境分支的期望值

*   **情境 1: 全球鋰市場 - 強勁成長 (高鋰價)**
    *   **情境描述**：電動車銷量超預期，新供應上線速度慢於需求，鋰價重回高位。SQM 憑藉其在智利和澳洲的擴產能力和成本優勢，大幅受益。
    *   **機率**：30%
    *   **結果 1.1: SQM 成功擴產並受惠**
        *   機率 (佔情境 1)：70%
        *   預期報酬：+40%
    *   **結果 1.2: SQM 遇小幅阻礙**
        *   機率 (佔情境 1)：30%
        *   預期報酬：+15%
    *   **情境 1 期望值 (EV_情境1)**：
        EV_情境1 = (0.70 * 40%) + (0.30 * 15%)
        EV_情境1 = 28% + 4.5% = **32.5%**

*   **情境 2: 全球鋰市場 - 適度成長 (穩定鋰價)**
    *   **情境描述**：電動車銷量持續增長但速度放緩，新供應逐漸滿足需求，鋰價波動但維持在相對穩定且可獲利的區間。SQM 營運穩定。
    *   **機率**：45%
    *   **結果 2.1: SQM 營運符合預期**
        *   機率 (佔情境 2)：60%
        *   預期報酬：+10%
    *   **結果 2.2: SQM 面臨競爭或小型挑戰**
        *   機率 (佔情境 2)：40%
        *   預期報酬：+3%
    *   **情境 2 期望值 (EV_情境2)**：
        EV_情境2 = (0.60 * 10%) + (0.40 * 3%)
        EV_情境2 = 6% + 1.2% = **7.2%**

*   **情境 3: 全球鋰市場 - 供應過剩/需求放緩 (低鋰價)**
    *   **情境描述**：全球經濟衰退影響電動車銷售，或大量新鋰礦產能集中釋放導致市場嚴重供過於求，鋰價持續下跌至低谷。SQM 獲利能力大幅受損。
    *   **機率**：25%
    *   **結果 3.1: SQM 獲利受重創**
        *   機率 (佔情境 3)：60%
        *   預期報酬：-25%
    *   **結果 3.2: SQM 憑成本優勢較具韌性**
        *   機率 (佔情境 3)：40%
        *   預期報酬：-10%
    *   **情境 3 期望值 (EV_情境3)**：
        EV_情境3 = (0.60 * -25%) + (0.40 * -10%)
        EV_情境3 = -15% + -4% = **-19%**

#### 2. 計算「投資 SQM」的總體期望值

總體期望值是將各主要情境的期望值與其對應的機率相乘後加總。

總體期望值 (EV_總體) = (EV_情境1 * 情境1機率) + (EV_情境2 * 情境2機率) + (EV_情境3 * 情境3機率)

EV_總體 = (32.5% * 0.30) + (7.2% * 0.45) + (-19% * 0.25)
EV_總體 = 9.75% + 3.24% - 4.75%
EV_總體 = **8.24%**

### 最終結論

根據上述決策樹分析與期望值計算，投資 SQM 的總體期望報酬率為 **8.24%**。

**判斷：** 適合投資

**簡短理由：**
儘管鋰市場存在高度波動性且智利政策帶來不確定性，但基於本分析中的情境假設和機率分配，SQM 在未來 12-18 個月內呈現正向的期望值（8.24%）。這表明在考慮所有潛在結果及其可能性後，投資 SQM 具有預期的正回報。SQM 作為鋰產業的領導者之一，若鋰市場能維持適度或強勁成長，其較低的生產成本和擴產能力有望使其受惠。然而，投資者應充分理解其潛在的高波動性風險，並將此8.24%的預期報酬與自身的風險承受能力及其他投資機會進行比較。
