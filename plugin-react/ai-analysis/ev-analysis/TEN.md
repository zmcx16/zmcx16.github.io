#### 2. 計算過程
期望值 (EV) = $\sum (機率 \times 預期股價)$

*   **預期股價計算**：
    *   $EV_{price} = (0.30 \times 55.00) + (0.50 \times 46.00) + (0.20 \times 30.00)$

*   **預期總報酬率 (Expected Return)**：
    *   $Return_{capital} = (45.50 - 38.14) / 38.14 = 19.3\%$
    *   $Return_{total} = 19.3\% (\text{價差}) + 4.19\% (\text{股息}) = \mathbf{23.49\%}$
