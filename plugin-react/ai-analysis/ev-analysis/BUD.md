#### **2. 明確列出所有計算過程**

我們將根據上述決策樹節點的機率和預期報酬來計算期望值。

**a. 節點的期望值計算方式**

期望值 (Expected Value, EV) 的計算公式為：
$EV = \sum (P_i \times R_i)$
其中，$P_i$ 是情境 $i$ 發生的機率，$R_i$ 是情境 $i$ 的預期報酬。

**b. 計算「投資 BUD」的整體期望值**

*   **情境1: 強勁復甦與成長**
    *   機率 (P1) = 0.30 (30%)
    *   預期報酬 (R1) = +0.25 (25%)
    *   期望值貢獻 = $0.30 \times 0.25 = 0.075$

*   **情境2: 適度復甦與穩定**
    *   機率 (P2) = 0.50 (50%)
    *   預期報酬 (R2) = +0.08 (8%)
    *   期望值貢獻 = $0.50 \times 0.08 = 0.040$

*   **情境3: 持續逆風與表現不佳**
    *   機率 (P3) = 0.20 (20%)
    *   預期報酬 (R3) = -0.15 (-15%)
    *   期望值貢獻 = $0.20 \times (-0.15) = -0.030$

**總計「投資 BUD」的整體期望值 (EV_Invest)**
$EV_{Invest} = 0.075 + 0.040 - 0.030 = 0.085$
**因此，投資 BUD 的整體期望報酬為 +8.5%。**

**c. 計算「不投資 BUD」的整體期望值**

*   選擇不投資 BUD，我們假設將資金維持現金或投資於無風險資產，其相對投資 BUD 的預期報酬為 0%。
$EV_{Do Not Invest} = 0\%$
