### **2. 明確列出所有計算過程**

#### **2.1. 核心假設 (Core Assumptions)**

*   **市場假設：**
    *   **全球經濟增長：** 預計在分析期間（未來12-18個月）維持溫和增長趨勢，但存在潛在的衰退風險和通脹壓力。
    *   **利率環境：** 假設央行貨幣政策趨於穩定，利率可能維持在高位或有小幅下調空間，但資金成本仍是企業考量因素。
    *   **地緣政治：** 假設地緣政治風險維持在當前水平，未發生對全球貿易和供應鏈產生重大顛覆的事件。
*   **財務假設 (針對 CE)：**
    *   **估值：** 假設 CE 目前股價反映了市場對其當前盈利和增長前景的預期，但未完全計入所有潛在的利好或利空因素。
    *   **盈利能力：** 預計 CE 能維持其在特種材料領域的競爭優勢，並通過成本控制和產品組合優化來維持毛利率。
    *   **現金流：** 預計公司能產生穩定的自由現金流，支持股息支付和債務償還。
    *   **負債水平：** 考慮到 CE 過去的收購活動，假設其負債水平可控，不會對其運營產生過度壓力。
*   **產業趨勢假設 (針對特種材料/化工產業)：**
    *   **需求端：** 汽車、電子、醫療保健、消費品等終端市場對高性能材料的需求持續增長，是 CE 的主要驅動力。
    *   **供應鏈：** 假設供應鏈問題逐步緩解，但原材料（如石油和天然氣衍生品）價格波動仍是主要風險。
    *   **競爭格局：** 產業內競爭激烈，技術創新和差異化是維持優勢的關鍵。
    *   **環保法規：** 假設環保政策趨嚴，可能增加生產成本，但也可能催生對綠色材料的新需求。

#### **2.2. 節點的期望值計算方式**

*   **投資 CE 的期望值 (EV_Invest_CE)：**
    這是計算所有情境下的預期報酬率的加權平均值。
    $EV_{Invest\_CE} = (P_{Optimistic} \times R_{Optimistic}) + (P_{Moderate} \times R_{Moderate}) + (P_{Pessimistic} \times R_{Pessimistic})$
    $EV_{Invest\_CE} = (0.30 \times 0.35) + (0.45 \times 0.10) + (0.25 \times -0.20)$
    $EV_{Invest\_CE} = 0.105 + 0.045 - 0.050$
    $EV_{Invest\_CE} = 0.100 = \textbf{+10.0%}$

*   **不投資 CE 的期望值 (EV_DoNotInvest)：**
    這是替代投資的預期報酬率。
    $EV_{DoNotInvest} = P_{Alternative} \times R_{Alternative}$
    $EV_{DoNotInvest} = 1.00 \times 0.04$
    $EV_{DoNotInvest} = \textbf{+4.0%}$
