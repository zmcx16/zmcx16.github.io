### 三、 期望值計算過程 (Expected Value Calculation)

期望值（EV）的計算公式為：
$$EV = \sum (機率 \times 預期報酬)$$

**計算步驟：**
1.  **樂觀貢獻**：$0.30 \times 35\% = 10.5\%$
2.  **中性貢獻**：$0.50 \times 15\% = 7.5\%$
3.  **悲觀貢獻**：$0.20 \times (-20\%) = -4.0\%$

**總期望報酬率：**
$$10.5\% + 7.5\% - 4.0\% = 14.0\%$$

**換算為預期股價：**
目前股價 $111.20 \times (1 + 14.0\%) \approx \mathbf{\$126.77}$
