#### B. 期望值 (Expected Value, EV) 計算
我們根據各情境的報酬率與機率進行加權計算：

*   **現價 (Current Price)**: $29.94
*   **牛市報酬**: `($43.00 - $29.94) / $29.94 = +43.6%`
*   **基準報酬**: `($35.00 - $29.94) / $29.94 = +16.9%`
*   **熊市報酬**: `($22.00 - $29.94) / $29.94 = -26.5%`

$EV = (0.30 \times 43.6\%) + (0.50 \times 16.9\%) + (0.20 \times -26.5\%)$
