### **計算過程**

1.  **計算各情境的期望值貢獻 (Expected Value Contribution for each scenario)：**

        `EV_ModerateGrowth = 機率 * 預期報酬 = 0.40 * (+0.15) = +0.06` (即 +6.0%)

        `EV_SevereDownturn = 機率 * 預期報酬 = 0.10 * (-0.30) = -0.03` (即 -3.0%)
