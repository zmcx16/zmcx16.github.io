2.  **公司財務與策略：** EL 公司正在執行「利潤復甦與增長計畫 (PRGP)」，該計畫已開始產生正面效益，有助於毛利率擴張和營運利潤改善。公司積極投資研發、數位轉型（包括AI應用）及新產品發布，並透過收購DECIEM（The Ordinary）等品牌來擴大護膚品組合並吸引年輕客群。
3.  **產業挑戰：** 中國市場和旅遊零售業務的疲軟是 EL 面臨的主要逆風，消費者情緒低迷導致銷售額下降。此外，公司也面臨來自利基品牌日益激烈的競爭，以及原材料成本上升和消費者對價格敏感的壓力。近期公司也捲入商標侵權訴訟。
4.  **分析師預期：** 分析師對 EL 的共識評級為「持有」或「溫和買入」，平均目標價介於 $104.43 至 $112.81 之間，相較於目前股價（約 $87.88）有約 18.9% 至 25.80% 的上漲空間。
