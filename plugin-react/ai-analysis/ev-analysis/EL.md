### 核心假設

1.  **市場趨勢：** 全球高端美妝市場預計將持續增長，主要受個性化、可持續性、科技整合（AI/AR）以及數位化行銷的推動。然而，全球宏觀經濟狀況，特別是中國市場和旅遊零售渠道，仍存在一定波動性。
2.  **公司財務：** EL 在2025財年表現疲軟，錄得淨虧損，但2026財年第一季度已顯示出復甦跡象，尤其是在護膚品業務和亞洲旅遊零售方面。公司正積極執行「美妝重塑」（Beauty Reimagined）戰略，包括數位化轉型、AI應用、產品創新和資產組合優化（剝離表現不佳品牌）。
3.  **分析師預期：** 分析師對EL的目標價存在較大分歧，平均目標價略低於當前股價，但近期有多家機構上調評級，並給出較高的目標價。這反映了市場對公司轉型潛力的觀望態度。
4.  **估值：** EL目前的股價已反映了市場對其未來轉型成功的部分樂觀預期，其遠期本益比（Forward P/E）較高，表明市場對其未來盈利增長有較高要求。
