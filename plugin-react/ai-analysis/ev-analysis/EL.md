### 2. 計算過程與核心假設

#### 2.1 節點的期望值計算方式

期望值 (Expected Value, EV) 的計算公式為：

$$EV = \sum_{i=1}^{n} (P_i \times R_i)$$

其中：
*   $P_i$ 是第 i 個結果情境的機率。
*   $R_i$ 是第 i 個結果情境的預期報酬（或損失）。

**計算「投資 EL」選項的期望值：**

*   **情境 1.1 (市場樂觀)：**
    *   機率 ($P_{1.1}$) = 40% = 0.4
    *   預期報酬 ($R_{1.1}$) = +20% = 0.20
*   **情境 1.2 (市場悲觀)：**
    *   機率 ($P_{1.2}$) = 60% = 0.6
    *   預期報酬 ($R_{1.2}$) = -20% = -0.20

$$EV_{投資EL} = (P_{1.1} \times R_{1.1}) + (P_{1.2} \times R_{1.2})$$
$$EV_{投資EL} = (0.4 \times 0.20) + (0.6 \times -0.20)$$
$$EV_{投資EL} = 0.08 + (-0.12)$$
$$EV_{投資EL} = -0.04$$

**所以，「投資 EL」選項的整體期望值為 -4%。**

**計算「不投資 EL」選項的期望值：**

*   **情境 2.1 (市場樂觀)：**
    *   機率 ($P_{2.1}$) = 40% = 0.4
    *   預期報酬 ($R_{2.1}$) = +2% = 0.02
*   **情境 2.2 (市場悲觀)：**
    *   機率 ($P_{2.2}$) = 60% = 0.6
    *   預期報酬 ($R_{2.2}$) = +2% = 0.02 (無風險報酬)

$$EV_{不投資EL} = (P_{2.1} \times R_{2.1}) + (P_{2.2} \times R_{2.2})$$
$$EV_{不投資EL} = (0.4 \times 0.02) + (0.6 \times 0.02)$$
$$EV_{不投資EL} = 0.008 + 0.0012$$
$$EV_{不投資EL} = 0.02$$

**所以，「不投資 EL」選項的整體期望值為 +2%。**
