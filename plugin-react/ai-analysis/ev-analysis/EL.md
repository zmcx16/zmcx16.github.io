#### 2. 計算過程

**當前股價 (P0):** $86.20

**情境一：樂觀情境 (Strong Recovery & Growth)**
*   **預測情境名稱：** 公司「利潤復甦與增長計劃 (PRGP)」成功大幅提升營運效率和成本節約。數位轉型和品牌擴張（如與 Shopify 合作、投資 111SKIN）取得顯著成效。全球高端美妝市場強勁復甦，Estée Lauder 在香水和護膚等關鍵品類及中國市場份額持續增長。宏觀經濟環境改善，消費者信心增強。
*   **對應的機率 (Probability)：** 30%
*   **預期報酬 / 期望值 (Expected Value) 計算：**
    *   我們參考分析師的最高目標價 ($130) 及公司積極的轉型策略，設定此情境下的目標價為 $125。
    *   潛在報酬 = 目標價 - 當前股價 = $125 - $86.20 = $38.80
    *   期望值 (EV_Opt) = 潛在報酬 × 機率 = $38.80 × 0.30 = $11.64

**情境二：中性情境 (Steady Progress & Stabilization)**
*   **預測情境名稱：** PRGP 計劃按預期推進，帶來穩定的成本節約和效率提升，但增長速度並非爆發性。公司有效應對中東衝突、關稅和市場競爭等挑戰，但這些因素仍對業績構成一定壓力。數位化和品牌策略貢獻穩定表現。股價趨向分析師的平均目標價。
*   **對應的機率 (Probability)：** 50%
*   **預期報酬 / 期望值 (Expected Value) 計算：**
    *   參考分析師的平均目標價 ($98.80 - $101.00)，設定此情境下的目標價為 $100。
    *   潛在報酬 = 目標價 - 當前股價 = $100 - $86.20 = $13.80
    *   期望值 (EV_Mod) = 潛在報酬 × 機率 = $13.80 × 0.50 = $6.90

**情境三：悲觀情境 (Challenges Persist & Underperformance)**
*   **預測情境名稱：** 重組成本和營運中斷超出預期，導致盈利能力改善緩慢。宏觀經濟逆風加劇，消費者對高端美妝產品的支出減少。激烈的市場競爭或不可預見的事件（如進一步的地緣政治衝突、供應鏈問題）嚴重影響銷售和利潤率。公司未能充分利用數位化和市場趨勢。
*   **對應的機率 (Probability)：** 20%
*   **預期報酬 / 期望值 (Expected Value) 計算：**
    *   參考分析師的最低目標價 ($70 - $75)，設定此情境下的目標價為 $70。
    *   潛在報酬 = 目標價 - 當前股價 = $70 - $86.20 = -$16.20 (即潛在損失)
    *   期望值 (EV_Pes) = 潛在報酬 × 機率 = -$16.20 × 0.20 = -$3.24

**整體期望值 (Overall Expected Value):**
整體期望值 = EV_Opt + EV_Mod + EV_Pes
整體期望值 = $11.64 + $6.90 - $3.24 = $15.30

這表示每股預期有 $15.30 的潛在收益。
預期的未來股價 = 當前股價 + 整體期望值 = $86.20 + $15.30 = $101.50。

根據決策樹分析和期望值計算，Estée Lauder (EL) 的整體期望值為 **$15.30**。這意味著在考慮了不同情境及其機率後，預期每股有 $15.30 的正向收益。預期的未來股價為 $101.50，高於當前股價 $86.20。
