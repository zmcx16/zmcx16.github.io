### 一、 核心假設與市場背景分析

在構建決策樹前，我們必須考慮以下關鍵變數：

1.  **新任 CEO 效應（The Niccol Effect）：** 前 Chipotle 執行長 Brian Niccol 被寄予厚望，其核心策略為「回到星巴克（Back to Starbucks）」，專注於改善門市體驗、簡化菜單及優化早晨尖峰時段效率。
2.  **中國市場挑戰：** 面對瑞幸咖啡（Luckin Coffee）等本土品牌的價格戰，星巴克在中國的同店銷售額（SSS）持續承壓。
3.  **美國消費力：** 通膨雖放緩，但消費者對高價咖啡的支出變得謹慎。
4.  **估值水平：** 目前 Forward P/E 約 28.3x，相較於歷史高點已有所回落，但仍高於餐飲業平均。

### 三、 期望值分析與計算過程

#### 1. 各情境參數設定理由：
*   **樂觀情境 (30%)：** Brian Niccol 成功複製 Chipotle 奇蹟，解決美國門市排隊問題，且中國市場透過策略結盟止跌回升。目標價參考 52 週高點附近。
*   **基準情境 (50%)：** 營運效率有所提升，但中國市場競爭依劇，營收維持個位數增長。目標價參考分析師平均目標價 ($94.62)。
*   **悲觀情境 (20%)：** 美國經濟陷入衰退導致消費降級，且中國市場份額持續被蠶食。目標價參考 52 週低點 ($75.50) 以下的支撐位。

#### 2. 期望值 (Expected Value, EV) 計算：
我們計算一年後的「預期股價」與「預期總報酬（含股息）」。
*   **目前股價 ($P_0$):** $85.08
*   **年度股息 (Dividend):** $85.08 \times 2.88\% \approx \$2.45$

**預期股價計算：**
$$E(Price) = (115 \times 0.30) + (95 \times 0.50) + (72 \times 0.20)$$
$$E(Price) = 34.5 + 47.5 + 14.4 = \$96.4$$

**預期總報酬率計算：**
$$E(Return) = \frac{E(Price) + Dividend - P_0}{P_0}$$
$$E(Return) = \frac{96.4 + 2.45 - 85.08}{85.08} = \frac{13.77}{85.08} \approx 16.18\%$$

#### 1. 財務數據亮點與隱憂：

*   **劣勢：** 負債比（Debt/Eq）未顯示但通常較高；Current Ratio (0.72) 偏低，流動性略顯緊張；EPS Q/Q (-85.4%) 顯示近期獲利能力受到重創。

#### 2. 最終判斷：**適合投資 (建議：分批買入)**

#### 3. 理由：
1.  **期望值為正且具吸引力：** 16.18% 的預期總報酬率優於標普 500 的長期平均回報。
2.  **管理層紅利：** Brian Niccol 是餐飲業公認的轉型專家，市場目前正處於「信任投票」階段，股價在 $85 附近具備較強支撐。
3.  **股息防禦：** 2.88% 的股息率在餐飲龍頭中極具競爭力，能為下行風險提供緩衝。
4.  **技術面修復：** 股價已從低點回升，目前在 SMA50 (0.32%) 之上震盪，顯示短期趨勢已轉為中性偏多。
