### **3. 計算過程 (Calculation Process)**

2.  **中性情境期望值**

    *   預期報酬 = +5% = +0.05
    *   計算：$0.50 \times 0.05 = +0.025$ (或 +2.5%)

3.  **悲觀情境期望值**

    *   預期報酬 = -30% = -0.30
    *   計算：$0.25 \times (-0.30) = -0.075$ (或 -7.5%)

**B. 計算「投資 GT」選項的整體期望值：**

*   **公式**：$\sum (\text{各情境機率} \times \text{各情境預期報酬})$

*   **計算**：$100\% \times 0\% = 0\%$
*   **結論**：「不投資 GT」的整體期望值為 **0.0%**。
