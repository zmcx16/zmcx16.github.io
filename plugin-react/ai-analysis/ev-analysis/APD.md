### 3. 期望值計算 (Expected Value Calculation)

我們根據上述情境與機率計算一年期的預期收益率（Expected Return, ER）：

**計算公式：**
$$ER = \sum (Probability_i \times Return_i)$$

**計算過程：**
*   **樂觀情境貢獻**：$0.35 \times 20\% = 7.0\%$
*   **中性情境貢獻**：$0.45 \times 8\% = 3.6\%$
*   **悲觀情境貢獻**：$0.20 \times (-15\%) = -3.0\%$

**總期望值 (EV)：**
$$7.0\% + 3.6\% - 3.0\% = 7.6\%$$
