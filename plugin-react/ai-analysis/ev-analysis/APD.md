1.  **分析時間範圍 (Time Horizon)**：未來1-3年內的投資回報。
2.  **市場前景 (Market Outlook)**：
    *   **工業氣體需求穩定性**：APD作為工業氣體巨頭，業務具備一定程度的防禦性，尤其是有長期合約支撐。
    *   **能源轉型趨勢**：綠色氫能和碳捕捉等新興技術是APD的潛在成長動力，但也存在執行風險和市場接受度不確定性。
    *   **全球經濟景氣**：直接影響工業生產活動和對氣體的需求。
3.  **APD 公司特定假設 (Company-Specific Assumptions)**：
    *   **營運效率與專案執行**：大型資本支出專案（如大型工業氣體設施）的準時交付和成本控制是關鍵。

    *   **競爭環境**：來自Linde、Air Liquide等同業的競爭壓力。
4.  **期望報酬率 (Expected Returns)**：以下情境中的報酬率為在該情境下，對APD股票的預期總回報（包含股價漲跌與股息，以年化百分比表示）。這些是基於APD過去表現、行業特性和假設情境下的綜合判斷，具有主觀性。
5.  **機率分配 (Probability Distribution)**：機率分配是根據對當前宏觀經濟和公司特定因素的判斷所賦予的，具有主觀性。

#### 2. 計算過程

我們將從決策樹的最右端向左計算各節點的期望值。

**步驟 1: 計算各「APD 營運表現」機會節點的期望值**

這些期望報酬率的假設是基於宏觀經濟前景和APD自身營運能力結合後的結果。

*   **情境 A: 經濟強勁成長 (P=0.30)**

        *   情境 A.1 (卓越營運, P=0.30): +25%
        *   情境 A.2 (平均營運, P=0.50): +20%
        *   情境 A.3 (欠佳營運, P=0.20): +10%
    *   **期望值 A 的計算**：
        $EV_A = (0.30 \times 25\%) + (0.50 \times 20\%) + (0.20 \times 10\%)$
        $EV_A = 7.5\% + 10\% + 2\% = 19.5\%$

*   **情境 B: 經濟溫和成長 (P=0.50)**
    *   **預期報酬假設**：在溫和經濟下，APD保持穩定，但成長性受限。
        *   情境 B.1 (卓越營運, P=0.30): +15%
        *   情境 B.2 (平均營運, P=0.50): +10%
        *   情境 B.3 (欠佳營運, P=0.20): +0% (可能僅能保本或微幅虧損)
    *   **期望值 B 的計算**：
        $EV_B = (0.30 \times 15\%) + (0.50 \times 10\%) + (0.20 \times 0\%)$
        $EV_B = 4.5\% + 5\% + 0\% = 9.5\%$

*   **情境 C: 經濟衰退/放緩 (P=0.20)**
    *   **預期報酬假設**：在衰退中，即使APD業務具備防禦性，仍會面臨需求下降和專案延遲的壓力。
        *   情境 C.1 (卓越營運, P=0.30): -10% (虧損較少)
        *   情境 C.2 (平均營運, P=0.50): -15%
        *   情境 C.3 (欠佳營運, P=0.20): -25% (虧損較多)
    *   **期望值 C 的計算**：
        $EV_C = (0.30 \times -10\%) + (0.50 \times -15\%) + (0.20 \times -25\%)$
        $EV_C = -3\% - 7.5\% - 5\% = -15.5\%$

**步驟 2: 計算「投資 APD」的整體期望值**

現在我們將上述計算出的各經濟情境的期望值，乘以其對應的機率，加總得出投資APD的整體期望值。

*   **整體期望值 (EV_Invest)**：
    $EV_{Invest} = (P_{Strong Growth} \times EV_A) + (P_{Moderate Growth} \times EV_B) + (P_{Slowdown/Recession} \times EV_C)$
    $EV_{Invest} = (0.30 \times 19.5\%) + (0.50 \times 9.5\%) + (0.20 \times -15.5\%)$
    $EV_{Invest} = 5.85\% + 4.75\% - 3.1\%$
    $EV_{Invest} = 7.5\%$
