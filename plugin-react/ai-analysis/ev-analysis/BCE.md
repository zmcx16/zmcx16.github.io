### 3. 期望值分析 (Expected Value Analysis)

期望值 (EV) = $\sum (機率 \times 預期報酬)$

*   **牛市節點**: $0.20 \times 25\% = 5.0\%$
*   **基準節點**: $0.50 \times 5\% = 2.5\%$
*   **熊市節點**: $0.30 \times (-20\%) = -6.0\%$

**總期望報酬率 (Total EV) = 5.0% + 2.5% - 6.0% = 1.5%**
