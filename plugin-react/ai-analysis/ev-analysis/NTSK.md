### 計算過程

**1. 預期報酬計算 (基於當前股價 $15.50)：**

*   **情境 1: 樂觀 (高增長與盈利)**
    *   預期股價 (P1_out): $30.00 (參考分析師高目標價)
    *   預期報酬 (R1): ($30.00 - $15.50) / $15.50 = 0.9355 或 93.55%
    *   機率 (P1): 30%

*   **情境 2: 中性 (穩定增長與財務改善)**
    *   預期股價 (P2_out): $26.50 (參考分析師平均目標價)
    *   預期報酬 (R2): ($26.50 - $15.50) / $15.50 = 0.7097 或 70.97%
    *   機率 (P2): 45%

*   **情境 3: 停滯/表現不佳 (增長放緩與財務挑戰)**
    *   預期股價 (P3_out): $18.00 (略高於當前低點，但遠低於平均目標價)
    *   預期報酬 (R3): ($18.00 - $15.50) / $15.50 = 0.1613 或 16.13%
    *   機率 (P3): 20%

*   **情境 4: 負面 (顯著下跌)**
    *   預期股價 (P4_out): $10.00 (低於 52 週低點及 InvestingPro 的公允價值)
    *   預期報酬 (R4): ($10.00 - $15.50) / $15.50 = -0.3548 或 -35.48%
    *   機率 (P4): 5%

**2. 整體期望值 (Expected Value) 計算：**

期望值 (EV) = (P1 * R1) + (P2 * R2) + (P3 * R3) + (P4 * R4)
EV = (0.30 * 0.9355) + (0.45 * 0.7097) + (0.20 * 0.1613) + (0.05 * -0.3548)
EV = 0.28065 + 0.319365 + 0.03226 - 0.01774
**EV = 0.614535 或 61.45%**

**3. 預期股價 (Expected Price) 計算：**

預期股價 (EP) = (P1 * P1_out) + (P2 * P2_out) + (P3 * P3_out) + (P4 * P4_out)
EP = (0.30 * $30.00) + (0.45 * $26.50) + (0.20 * $18.00) + (0.05 * $10.00)
EP = $9.00 + $11.925 + $3.60 + $0.50
**EP = $25.025**
