# OMF (OneMain Holdings, Inc.) 量化投資分析報告

作為量化投資分析師，針對 OneMain Financial (OMF) 的評估核心在於其**信貸週期位置**與**利率環境的敏感度**。OMF 目前處於低估值（P/E 8.15x, Forward P/E 6.29x）與高股息（7.65%）的防禦性架構中，以下是基於概率思維的深度拆解。

---

### 1. 核心驅動因素與風險 (Drivers & Risks)

#### **關鍵催化劑 (Catalysts)**
*   **聯準會降息週期啟動**：OMF 作為非投資級借款人的主要債權人，其資金成本（Funding Cost）隨基準利率下降而減輕。更重要的是，降息能緩解其目標客群（中低收入戶）的債務壓力，降低違約率。
*   **信貸損失趨於穩定 (Credit Normalization)**：最新財報顯示其淨銷帳率（NCOs）已出現見頂跡象。若未來兩季 NCO 數據優於市場預期，將引發估值倍數（Multiple Re-rating）的修復。
*   **業務多元化與 Foursight 整合**：收購 Foursight Capital 進入汽車貸款領域，有助於優化資產組合並提升資本效率（ROE 高達 23.9%）。

#### **主要風險點 (Risks)**
*   **失業率超預期上升**：OMF 的客戶對就業市場極度敏感。若美國失業率突破 4.5%-5%，違約率將大幅飆升，侵蝕利潤。
*   **監管壓力 (Regulatory Risk)**：CFPB 對於高利貸利率上限或滯納金（Late Fees）的限制政策，可能直接衝擊其營收結構。
*   **融資成本滯後性**：雖然利率下降，但 OMF 存量債務的置換需要時間，短期內淨利差（NIM）可能受壓。

---

### 2. 情境設定與機率賦予 (Scenario Modeling)

#### **樂觀情境 (Bull Case)**
*   **發生條件**：美國經濟實現「軟著陸」，失業率維持低位，聯準會降息節奏明確（12個月內降息 100bps+），NCO 低於 7%。
*   **預估機率**：25%
*   **目標價格與預期回報**：Forward P/E 回升至 9x。目標價 **$78**（含息總回報約 **+51%**）。

#### **基準情境 (Base Case)**
*   **發生條件**：經濟增長放緩但未衰退，信貸損失符合預期（NCO 穩定在 7.5%-8%），股息維持發放。
*   **預估機率**：50%
*   **目標價格與預期回報**：回歸分析師平均目標價 **$67.5**（含息總回報約 **+31%**）。

#### **悲觀情境 (Bear Case)**
*   **發生條件**：美國進入中度衰退，失業率飆升，信貸損失率突破 10%，公司被迫削減股息以保留資本。
*   **預估機率**：25%
*   **目標價格與預期回報**：股價回測 52 週低點並跌破，P/E 壓縮至 5x。目標價 **$40**（含息總回報約 **-19%**）。

---

### 3. 期望值計算與決策樹 (EV Calculation & Decision Tree)

#### **決策樹結構**
```text
[OMF Investment]
├── (25%) 樂觀情境: +51% Total Return
├── (50%) 基準情境: +31% Total Return
└── (25%) 悲觀情境: -19% Total Return
```

#### **總期望值計算**
*   `EV = (0.25 * 0.51) + (0.50 * 0.31) + (0.25 * -0.19)`
*   `EV = 0.1275 + 0.155 - 0.0475 = 0.235`
*   **預期總報酬率：23.5%**

#### **風險回報比分析**
*   **上行潛力**：平均約 +36% (Bull & Base)
*   **下行空間**：-19%
*   **不對稱性**：風險回報比約為 **1 : 1.9**。這顯示在目前的估值水平下，贏面顯著大於輸面，具備高度的投資吸引力。

### 4. 決策總結 (Decision Summary)

| 情境 | 發生機率 (%) | 預期報酬率 (%) | 關鍵驅動/觸發因素 |
| :--- | :--- | :--- | :--- |

| **基準 (Base)** | 50% | +31% | 經濟平穩 + 股息穩定 + 估值回歸共識 |
| **悲觀 (Bear)** | 25% | -19% | 硬著陸 + 失業率飆升 + 壞帳失控 |
| **整體期望值** | **100%** | **+23.5%** | **加權平均預期回報（含 7.6% 股息）** |

**最終結論：**
1. **投資建議**：**買入 (Buy)**
2. **核心逻辑**：OMF 目前的 Forward P/E 僅 6.29x，已定價了大部分的信貸風險。23.5% 的期望值遠高於市場平均水平，且 7.6% 的股息提供了強大的下行緩衝（Margin of Safety）。在降息週期中，該股具備「估值修復」與「利差擴大」的雙重紅利。
3. **風控建議**：若美國失業率連續兩個月超過 4.6%，或公司宣佈削減股息，視為悲觀情境觸發訊號，應立即重新評估或執行止損（建議止損位設於 $45，即 52 週低點附近）。
