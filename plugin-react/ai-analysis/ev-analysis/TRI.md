*   **中性增長**：參考 Forward P/E 與 EPS next Y (13.79%)，設定為 **+15%**（含股息）。

#### 2. 期望值 (EV) 計算公式
$$EV = (P_{Bull} \times R_{Bull}) + (P_{Base} \times R_{Base}) + (P_{Bear} \times R_{Bear})$$

#### 3. 代入數值
*   $EV = (0.3 \times 0.483) + (0.5 \times 0.15) + (0.2 \times -0.105)$
