### 三、 期望值計算 (Expected Value Calculation)

期望值 (EV) = Σ (各情境機率 × 各情境總報酬)

*   **樂觀節點**：$0.25 \times 18.4\% = 4.6\%$
*   **中性節點**：$0.50 \times 8.4\% = 4.2\%$
*   **悲觀節點**：$0.25 \times (-6.6\%) = -1.65\%$

**總期望報酬率 (Total Expected Return) = 4.6% + 4.2% - 1.65% = 7.15%**
