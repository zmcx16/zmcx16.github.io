### 三、 期望值分析 (Expected Value Analysis)

| 熊市情境 | 0.20 | -17.4% | -3.48% |
| **總計期望值** | **1.00** | | **+15.02%** |

#### 計算過程：
$EV = (0.30 \times 35.5\%) + (0.50 \times 15.7\%) + (0.20 \times -17.4\%)$
$EV = 10.65\% + 7.85\% - 3.48\% = 15.02\%$

**風險提示：**
