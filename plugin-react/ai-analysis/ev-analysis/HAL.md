### 三、 期望值計算 (Expected Value Analysis)

我們將各情境的機率與預期報酬相乘，得出整體期望報酬率：

| 情境 | 機率 (P) | 預期報酬 (R) | 期望值 (P * R) |
| :--- | :--- | :--- | :--- |
| **樂觀情境** | 0.25 | +30% | +7.5% |
| **基準情境** | 0.50 | +10% | +5.0% |
| **悲觀情境** | 0.25 | -25% | -6.25% |
| **總計期望報酬** | **1.00** | | **+6.25%** |

**計算過程：**
$EV = (0.25 \times 0.30) + (0.50 \times 0.10) + (0.25 \times -0.25)$
$EV = 0.075 + 0.05 - 0.0625 = 0.0625$ (即 **6.25%**)
