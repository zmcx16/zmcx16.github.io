## 3. 風險評估 (Pre-Mortem)
**假設一年後 AZN 股價腰斬 (跌 50%)，最可能的原因是：**

1.  **核心藥物研發失敗或監管重擊：** 某款貢獻巨大營收的腫瘤藥物（如 Tagrisso 或 Enhertu）在關鍵臨床試驗中出現安全性問題，或被 FDA 撤銷加速批准。
2.  **中國市場劇變：** 作為 AZN 的重要增長引擎，若中國加強藥品集中採購 (VBP) 壓力，或因地緣政治導致業務受阻，將直接重創其營收增速。
3.  **利率環境與估值修正：** 若通膨回升導致利率長期維持高位，市場對高 P/E 的醫療成長股容忍度降低，加上其流動比率偏低，可能引發資金撤出。
