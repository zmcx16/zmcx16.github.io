### 3. 風險評估 (Pre-Mortem)

假設一年後 SEE 股價腰斬，最可能的導火線是：

1.  **債務危機與信用評級下調**: 在高利率環境下，SEE 無法有效再融資其巨額債務，導致利息支出暴增，被迫削減股息，引發機構投資者恐慌性拋售。
2.  **ESG 與替代材料的衝擊**: 全球對一次性塑料的監管超預期收緊，而 SEE 的紙質或生物可降解材料轉型進度落後於競爭對手（如 Graphic Packaging），導致核心客戶流失。
3.  **自動化轉型失敗**: 公司投入大量資本研發的自動化包裝系統未能獲得客戶認可，導致資本支出 (CAPEX) 浪費，且無法抵消人工成本的上升。
