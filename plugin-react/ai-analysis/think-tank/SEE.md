### 3. 風險評估 (Pre-Mortem)

3.  **宏觀消費疲軟：** 全球經濟進入深度衰退，消費者減少昂貴蛋白質（如牛肉）的攝取，直接打擊 SEE 最賺錢的 Cryovac 食品包裝業務。
